### <a name="wpf-aero2-and-aerolite-themes-toggle-button-doesnt-show-any-state"></a>WPF Aero2 和 AeroLite 佈景主題切換按鈕不會顯示任何狀態

|   |   |
|---|---|
|詳細資料|在 .NET Framework 4.5 中，WPF 的 Aero2 和 AeroLite 佈景主題不會顯示切換按鈕之「開啟」與「關閉」狀態間的差異。|
|建議|此問題在服務更新中已修正。 請更新 .NET Framework 4.5，或是升級至 .NET Framework 4.5.1 或更新版本，以修正此問題。 或者，可能會明確地設定切換按鈕的樣式來解決此問題。|
