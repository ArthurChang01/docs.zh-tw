### <a name="serializing-with-an-xml-declaration"></a><span data-ttu-id="e7688-108">使用 XML 宣告進行序列化</span><span class="sxs-lookup"><span data-stu-id="e7688-108">Serializing with an XML Declaration</span></span>  
 <span data-ttu-id="e7688-109">下列範例會建立 <xref:System.Xml.Linq.XElement>、將文件儲存為檔案，然後將檔案列印到主控台：</span><span class="sxs-lookup"><span data-stu-id="e7688-109">The following example creates an <xref:System.Xml.Linq.XElement>, saves the document to a file, and then prints the file to the console:</span></span>  
  
```csharp  
XElement root = new XElement("Root",  
    new XElement("Child", "child content")  
);  
root.Save("Root.xml");  
string str = File.ReadAllText("Root.xml");  
Console.WriteLine(str);  
```  
  
 <span data-ttu-id="e7688-110">這個範例會產生下列輸出：</span><span class="sxs-lookup"><span data-stu-id="e7688-110">This example produces the following output:</span></span>  
  
```xml  
<?xml version="1.0" encoding="utf-8"?>  
<Root>  
  <Child>child content</Child>  
</Root>  
```  
  
### <a name="serializing-without-an-xml-declaration"></a><span data-ttu-id="e7688-111">不使用 XML 宣告序列化</span><span class="sxs-lookup"><span data-stu-id="e7688-111">Serializing without an XML Declaration</span></span>  
 <span data-ttu-id="e7688-112">下列範例顯示如何將 <xref:System.Xml.Linq.XElement> 儲存為 <xref:System.Xml.XmlWriter>。</span><span class="sxs-lookup"><span data-stu-id="e7688-112">The following example shows how to save an <xref:System.Xml.Linq.XElement> to an <xref:System.Xml.XmlWriter>.</span></span>  
  
```csharp  
StringBuilder sb = new StringBuilder();  
XmlWriterSettings xws = new XmlWriterSettings();  
xws.OmitXmlDeclaration = true;  
  
using (XmlWriter xw = XmlWriter.Create(sb, xws)) {  
    XElement root = new XElement("Root",  
        new XElement("Child", "child content")  
    );  
    root.Save(xw);  
}  
Console.WriteLine(sb.ToString());  
```  
  
 <span data-ttu-id="e7688-113">這個範例會產生下列輸出：</span><span class="sxs-lookup"><span data-stu-id="e7688-113">This example produces the following output:</span></span>  
  
```xml  
<Root><Child>child content</Child></Root>  
```
