# <a name="generics-in-the-net-framework-class-library-c-programming-guide"></a>.NET Framework 類別庫中的泛型 (C# 程式設計手冊)
.NET Framework Class Library 2.0 版提供了新的命名空間 <xref:System.Collections.Generic>，其中包含數個立即可用的泛型集合類別和關聯介面。 其他命名空間 (例如 <xref:System>) 也提供新的泛型介面 (例如 <xref:System.IComparable%601>)。 與舊版 .NET Framework 中提供的非泛型集合類別相較，這些類別和介面更有效率且為型別安全。 在設計和實作您自己的自訂集合類別之前，請考慮是否可以使用 .NET Framework Class Library 所提供的其中一個類別，或從中衍生類別。
