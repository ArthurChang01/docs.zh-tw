# <a name="para-c-programming-guide"></a>\<para> (C# 程式設計指南)
## <a name="syntax"></a>語法  
  
```xml  
<para>content</para>  
```  
  
#### <a name="parameters"></a>參數  
 `content`  
 段落的文字。  
  
## <a name="remarks"></a>備註  
 \<para> 標記是用於 [\<summary>](../../../csharp/programming-guide/xmldoc/summary.md)、[\<remarks>](../../../csharp/programming-guide/xmldoc/remarks.md) 或 [\<returns>](../../../csharp/programming-guide/xmldoc/returns.md) 等標記內，並可讓您新增文字的結構。  
  
 使用 [/doc](../../../csharp/language-reference/compiler-options/doc-compiler-option.md) 編譯，可處理檔案的文件註解。  
  
## <a name="example"></a>範例  
 如需使用 \<para> 的範例，請參閱 [\<summary>](../../../csharp/programming-guide/xmldoc/summary.md)。
