# <a name="para-c-programming-guide"></a>\<段落 > （C#程式設計手冊）

## <a name="syntax"></a>語法

```xml
<para>content</para>
```

## <a name="parameters"></a>參數

- `content`

  段落的文字。

## <a name="remarks"></a>備註

\<para> 標記是用於 [\<summary>](./summary.md)、[\<remarks>](./remarks.md) 或 [\<returns>](./returns.md) 等標記內，並可讓您新增文字的結構。

使用 [-doc](../../language-reference/compiler-options/doc-compiler-option.md) 編譯可處理檔案的文件註解。

## <a name="example"></a>範例

如需使用 \<para> 的範例，請參閱 [\<summary>](./summary.md)。
