# &lt;returns&gt; (C# 程式設計手冊)
## 語法  
  
```  
<returns>description</returns>  
```  
  
#### 參數  
 `description`  
 為傳回值的描述。  
  
## 備註  
 \<returns\> 標記應使用於方法宣告的註解來描述傳回值。  
  
 使用 [\/doc](../../../csharp/language-reference/compiler-options/doc-compiler-option.md) 進行編譯，將文件註解處理為檔案。
