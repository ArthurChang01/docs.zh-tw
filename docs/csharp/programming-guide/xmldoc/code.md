# <a name="code-c-programming-guide"></a>\<代碼>（C# 程式設計指南）

## <a name="syntax"></a>語法

```xml
<code>content</code>
```

## <a name="parameters"></a>參數

- `content`

  您要標記為程式碼的文字。

## <a name="remarks"></a>備註

\<code> 標記可讓您將多行指定為程式碼。 使用[\<c>](./code-inline.md)指示描述中的文本應標記為代碼。

使用[-doc](../../language-reference/compiler-options/doc-compiler-option.md)編譯，以處理檔的文檔注釋。
