# <a name="code-c-programming-guide"></a>\<code> (C# 程式設計手冊)
## <a name="syntax"></a>語法  
  
```xml  
<code>content</code>  
```  
  
## <a name="parameters"></a>參數  
 `content`  
 您要標記為程式碼的文字。  
  
## <a name="remarks"></a>備註  
 \<code> 標記可讓您將多行指定為程式碼。 [\<c>](./code-inline.md) 則用來指出在一段描述中應該標記為程式碼的文字。  
  
 使用 [-doc](../../language-reference/compiler-options/doc-compiler-option.md) 編譯可處理檔案的文件註解。

 如需如何使用 \<code> 標記的範例，請參閱 [\<example>>](./example.md) 主題。
