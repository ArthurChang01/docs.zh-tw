|C# 類型關鍵字|.NET 類型|
|--------------|-------------------------|
|[`object`](reference-types.md#the-object-type)|<xref:System.Object?displayProperty=nameWithType>|

在前面表中，左列中的每個 C# 類型關鍵字是相應 .NET 類型的別名。 它們是可互換的。 例如，下列宣告會宣告相同型別的變數：

```csharp
int a = 123;
System.Int32 b = 123;
```
