# #endregion (C# 參考)
`#endregion` 標記 [\#region](../../../csharp/language-reference/preprocessor-directives/preprocessor-region.md) 區塊的結尾。  例如：  
  
```  
  
      #region MyClass definition  
class MyClass   
{  
    static void Main()   
    {  
    }  
}  
#endregion  
```
