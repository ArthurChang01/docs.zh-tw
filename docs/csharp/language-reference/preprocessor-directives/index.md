 [#define](../../../csharp/language-reference/preprocessor-directives/preprocessor-define.md)

 如需詳細資訊和範例，請參閱個別主題。  
  
 雖然編譯器沒有個別的前置處理器，但處理本節中所述的指示詞時會像是有一個前置處理器一樣。 這些指示詞可用於協助條件式編譯。 不同於 C 和 C++ 指示詞，您無法使用這些指示詞來建立巨集。  
  
 每個前置處理器指示詞都必須是該行中的唯一指令。
