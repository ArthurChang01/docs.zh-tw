# <a name="ascending-c-reference"></a>ascending (C# 參考)
在查詢運算式的 [orderby 子句](../../../csharp/language-reference/keywords/orderby-clause.md) `ascending`中使用內容關鍵字，指定由小到大的排序次序。 因為 `ascending` 是預設的排序次序，所以不必指定它。
