# <a name="protected-c-reference"></a>protected (C# 參考)
`protected` 關鍵字是成員存取修飾詞。 受保護的成員可在其類別內由衍生類別執行個體存取。 如需 `protected` 和其他存取修飾詞的比較，請參閱[存取範圍層級](../../../csharp/language-reference/keywords/accessibility-levels.md)。

 結構成員不可以是受保護的成員，因為無法繼承結構。
