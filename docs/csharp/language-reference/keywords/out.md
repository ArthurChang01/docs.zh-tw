# <a name="out-c-reference"></a>out (C# 參考)

您可以在兩個內容中使用 `out` 關鍵字︰

- 用作[參數修飾詞](out-parameter-modifier.md)，讓您以傳址方式而非傳值方式將引數傳遞到方法。

- 在介面及委派的[泛型型別參數宣告](out-generic-modifier.md)中，指定型別參數是 Covariant。
