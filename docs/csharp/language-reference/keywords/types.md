# <a name="types-c-reference"></a>類型 (C# 參考)

C# 類型系統包含下列類別：

- [實值型別](value-types.md)

- [參考型別](reference-types.md)

- [指標型別](../../programming-guide/unsafe-code-pointers/pointer-types.md)

 為實值型別的變數可儲存資料，而為參考型別的變數可儲存實際資料的參考。 參考型別的執行個體也稱為物件。 指標類型只能用於 [unsafe](unsafe.md) 模式中。

 您可以使用 [boxing 和 unboxing](../../../csharp/programming-guide/types/boxing-and-unboxing.md)，將實值型別轉換成參考型別，然後再轉換回實值型別。 除了 boxed 實值型別之外，您無法將參考型別轉換成實值型別。

 本節也會介紹 [void](void.md)。
