# <a name="iteration-statements-c-reference"></a>反覆運算陳述式 (C# 參考)

您可以使用反覆運算陳述式來建立迴圈。 反覆運算陳述式會導致內嵌陳述式多次執行 (依據迴圈終止準則而定)。 除非出現[跳躍陳述式](jump-statements.md)，否則這些陳述式均會照順序執行。

反覆運算陳述式中會使用下列關鍵字：

- [do](do.md)

- [for](for.md)

- [foreach、in](foreach-in.md)

- [while](while.md)
