# <a name="this-c-reference"></a>this (C# 參考)

`this` 關鍵字指的是類別的目前執行個體，也用作擴充方法第一個參數的修飾詞。

> [!NOTE]
> 本文討論如何搭配使用 `this` 與類別執行個體。 如需其在擴充方法中之用法的詳細資訊，請參閱[擴充方法](../../programming-guide/classes-and-structs/extension-methods.md)。

- 限定透過類似名稱所隱藏的成員，例如︰

- 傳遞物件作為其他方法的參數，例如：

  ```csharp
  CalcTax(this);
  ```
