# <a name="access-modifiers-c-reference"></a>存取修飾詞 (C# 參考)
存取修飾詞是用來指定成員或類型的宣告存取範圍的關鍵字。 本節將介紹四種存取修飾詞：  
  
-   `public`
-   `protected`
-   `internal`
-   `private`
  
 您可以使用存取修飾詞，指定下列六個存取層級：  
  
- [`public`](public.md)：未限制存取。  
  
- [`protected`](protected.md)：存取限於包含類別或衍生自包含類別的型別。  
  
- [`internal`](internal.md)：存取限於目前組件。  
  
- [`protected internal`](protected-internal.md)：存取限於目前組件或衍生自包含類別的型別。  
  
- [`private`](private.md)：存取限於包含型別。  

- [`private protected`](private-protected.md)：存取限於目前組件內包含類別或衍生自包含類別的型別。  
  
 本節也會介紹下列項目：  
  
-   [存取層級](../../../csharp/language-reference/keywords/accessibility-levels.md)：使用四種存取修飾詞來宣告六個存取層級。  
  
-   [存取範圍定義域](../../../csharp/language-reference/keywords/accessibility-domain.md)：指定可以參考成員的程式區段。  
  
-   [使用存取層級的限制](../../../csharp/language-reference/keywords/restrictions-on-using-accessibility-levels.md)：使用已宣告存取層級的限制摘要。
