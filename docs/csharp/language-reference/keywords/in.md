# <a name="in-c-reference"></a>in (C# 參考)

`in` 關鍵字用在四項內容中：  
  
- 泛型介面和委派的[泛型型別參數](in-generic-modifier.md)。
- 用作[參數修飾詞](in-parameter-modifier.md)，讓您以傳址方式而非傳值方式將引數傳遞到方法。
- [foreach](foreach-in.md) 陳述式。
- LINQ 查詢運算式中的 [join 子句](join-clause.md)。
