# <a name="query-keywords-c-reference"></a>查詢關鍵字 (C# 參考)

本節包含查詢運算式中所使用的內容關鍵字。

## <a name="in-this-section"></a>本節內容

|子句|描述|
|------------|-----------------|
|[從](from-clause.md)|指定資料來源和範圍變數 (與反覆運算變數類似)。|
|[其中](where-clause.md)|根據以邏輯 AND 和 OR 運算子區隔的一或多個布林運算式，以篩選來源項目 (`&&` 或 <code>&#124;&#124;</code>)。|
|[選擇](select-clause.md)|指定在執行查詢時，所傳回序列中的項目會有的類型和形狀。|
|[群組](group-clause.md)|根據指定的索引鍵值來群組查詢結果。|
|[into](into.md)|提供可作為 join、group 或 select 子句結果參考的識別碼。|
|[orderby](orderby-clause.md)|根據項目類型的預設比較子，以遞增或遞減順序來排序查詢結果。|
|[加入](join-clause.md)|根據兩個指定比對準則的相等比較，以聯結兩個資料來源。|
|[讓](let-clause.md)|引進範圍變數以儲存查詢運算式中的子運算式結果。|
|[在](in.md)|[join](join-clause.md) 子句中的內容關鍵字。|
|[上](on.md)|[join](join-clause.md) 子句中的內容關鍵字。|
|[等於](equals.md)|[join](join-clause.md) 子句中的內容關鍵字。|
|[由](by.md)|[group](group-clause.md) 子句中的內容關鍵字。|
|[上升](ascending.md)|[orderby](orderby-clause.md) 子句中的內容關鍵字。|
|[降](descending.md)|[orderby](orderby-clause.md) 子句中的內容關鍵字。|
