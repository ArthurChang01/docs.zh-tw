# <a name="operator-keywords-c-reference"></a>運算子關鍵字 (C# 參考)

用來執行其他動作，例如建立物件、檢查物件的執行階段類型，取得類型的大小和其他動作。 本節將介紹下列關鍵字：

- [as](as.md) 將物件轉換為相容類型。

- [await](await.md) 暫止 [async](async.md) 方法，直到等候的工作完成為止。

- [is](is.md) 檢查物件的執行階段類型，或 (從 C# 7.0 開始) 根據模式來測試運算式。

- [new](new.md)

  - [new 運算子](new-operator.md) 建立物件。

  - [new 修飾詞](new-modifier.md) 隱藏繼承的成員。

  - [new 條件約束](new-constraint.md) 限定型別參數。

- [nameof](nameof.md) 取得變數、類型或成員的簡單 (不完整) 字串名稱。

- [sizeof](sizeof.md)取得 Unmanaged 類型的大小。  

- [typeof](typeof.md) 取得類型的 <xref:System.Type?displayProperty=nameWithType> 物件。  

- [true](true.md)  

  - [true 運算子](true-false-operators.md) 傳回 [bool](bool.md) 值 `true` 以指出運算元必然為 true。

  - [true 常值](true-literal.md) 代表 [bool](bool.md) 值 `true`。

- [false](false.md)  

  - [false 運算子](true-false-operators.md) 傳回 [bool](bool.md) 值 `true` 以指出運算元必然為 false。

  - [false 常值](false-literal.md) 代表 [bool](bool.md) 值 `false`。

- [stackalloc](stackalloc.md) 配置堆疊上的記憶體區塊。  

[陳述式](statement-keywords.md)一節涵蓋了可作為運算子和陳述式的下列關鍵字：

- [checked](checked.md) 指定 checked 內容。  

- [unchecked](unchecked.md) 指定 unchecked 內容。
