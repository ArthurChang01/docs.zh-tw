# null (C# 參考)
`null` 關鍵字是表示不參考任何物件之 Null 參考的常值。  `null` 是參考型別變數的預設值。  一般實值型別 \(Value Type\) 不可以為 null。  不過，C\# 2.0 中增加了可為 null 的實值型別。  請參閱 [可為 Null 的類型](../../../csharp/programming-guide/nullable-types/index.md)。
