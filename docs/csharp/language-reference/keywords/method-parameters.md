# <a name="method-parameters-c-reference"></a>方法參數 (C# 參考)
如果針對沒有 [ref](../../../csharp/language-reference/keywords/ref.md) 或 [out](../../../csharp/language-reference/keywords/out.md) 的方法宣告參數，參數可以有與其建立關聯的值。 該值可以在方法中進行變更，但控制權傳回給呼叫程序時，不會保留已變更值。 使用方法參數關鍵字，即可變更此行為。  
  
 本節描述在您宣告方法參數時可以使用的關鍵字：  
  
-   [params](../../../csharp/language-reference/keywords/params.md)  
  
-   [ref](../../../csharp/language-reference/keywords/ref.md)  
  
-   [out](../../../csharp/language-reference/keywords/out.md)
