## <a name="remarks"></a><span data-ttu-id="50f5c-122">備註</span><span class="sxs-lookup"><span data-stu-id="50f5c-122">Remarks</span></span>

<span data-ttu-id="50f5c-123">表中除了 `object` 和 `string` 以外的所有型別，都視為簡單型別。</span><span class="sxs-lookup"><span data-stu-id="50f5c-123">All of the types in the table, except `object` and `string`, are referred to as simple types.</span></span>

<span data-ttu-id="50f5c-124">.NET 型別與其 C# 型別關鍵字別名是可互換的。</span><span class="sxs-lookup"><span data-stu-id="50f5c-124">The .NET types and their C# type keyword aliases are interchangeable.</span></span> <span data-ttu-id="50f5c-125">例如，您可以使用下列任一個宣告來宣告整數變數：</span><span class="sxs-lookup"><span data-stu-id="50f5c-125">For example, you can declare an integer variable by using either of the following declarations:</span></span>

```csharp
int x = 123;
System.Int32 y = 123;
```

<span data-ttu-id="50f5c-126">使用 [typeof](../operators/type-testing-and-cast.md#typeof-operator) 運算子，取得表示指定類型的 <xref:System.Type?displayProperty=nameWithType> 執行個體：</span><span class="sxs-lookup"><span data-stu-id="50f5c-126">Use the [typeof](../operators/type-testing-and-cast.md#typeof-operator) operator to get the <xref:System.Type?displayProperty=nameWithType> instance that represents the specified type:</span></span>

```csharp
Type stringType = typeof(string);
Console.WriteLine(stringType.FullName);

Type doubleType = typeof(System.Double);
Console.WriteLine(doubleType.FullName);

// Output:
// System.String
// System.Double
```
