# <a name="selection-statements-c-reference"></a>選擇陳述式 (C# 參考)

選擇陳述式可依據特定條件是否為 `true`，將程式控制傳送給特定的流程。

選擇陳述式中會使用下列關鍵字：

- [if](if-else.md)

- [else](if-else.md)

- [switch](switch.md)

- [case](switch.md)

- [default](switch.md)
