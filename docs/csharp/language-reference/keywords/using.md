# <a name="using-c-reference"></a>using (C# 參考)

`using` 關鍵字有三個主要用途：

- [using 陳述式](using-statement.md)能定義一個範圍，系統會在範圍結尾處對物件進行處置。
- [using 指示詞](using-directive.md)能為命名空間建立別名，或匯入在其他命名空間中定義的類型。
- [using static 指示詞](using-static.md)會匯入單一類別的成員。
