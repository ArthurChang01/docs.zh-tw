# <a name="default-c-reference"></a>default (C# 參考)

`default` 關鍵字。

`default` 關鍵字可用於 `switch` 陳述式或預設值運算式中：

- [switch 陳述式](switch.md)：指定預設標籤。

- [預設值運算式](../../programming-guide/statements-expressions-operators/default-value-expressions.md)：為型別產生預設值。 參考型別為 null，實值型別為零，結構則為 0 位元模式。
