# <a name="default-c-reference"></a>default (C# 參考)

`default` 關鍵字可用於 `switch` 陳述式或預設值運算式中：

- [switch 陳述式](switch.md)：指定預設標籤。

- [預設值運算式](../../programming-guide/statements-expressions-operators/default-value-expressions.md)：產生類型的預設值。
