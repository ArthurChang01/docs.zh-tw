# <a name="return-c-reference"></a>return (C# 參考)
`return` 陳述式會終止執行在其中出現的方法，並且將控制權傳回給呼叫方法。 它也可以傳回選擇性值。 如果方法是 `void` 類型，則可以省略 `return` 陳述式。  
  
 如果 return 陳述式位在 `try` 區塊內，則會先執行 `finally` 區塊 (如果存在的話)，控制權才會傳回到呼叫方法。
