 <span data-ttu-id="1606b-104">如果變數是 [const](../keywords/const.md)，它也會是 [static](../keywords/static.md)。</span><span class="sxs-lookup"><span data-stu-id="1606b-104">If a variable is [const](../keywords/const.md), it is also [static](../keywords/static.md).</span></span> <span data-ttu-id="1606b-105">如果您想要 **const** 和 **static** 變數，只要將該變數宣告為 **const** 即可；如果您想要 **static** 變數，只要將它標記成 **static** 即可。</span><span class="sxs-lookup"><span data-stu-id="1606b-105">If you want a **const** and **static** variable, just declare that variable as **const**; if all you want is a **static** variable, just mark it **static**.</span></span>  
  
 <span data-ttu-id="1606b-106">下列範例會產生 CS0504：</span><span class="sxs-lookup"><span data-stu-id="1606b-106">The following sample generates CS0504:</span></span>

```csharp  
      static const int i = 0;   // CS0504, cannot be both static and const  
      abstract public void f();  
   }  
}  
```
