# <a name="compiler-error-cs0504"></a><span data-ttu-id="84e5a-102">編譯器錯誤 CS0504</span><span class="sxs-lookup"><span data-stu-id="84e5a-102">Compiler Error CS0504</span></span>
<span data-ttu-id="84e5a-103">無法將常數 'variable' 標記成靜態</span><span class="sxs-lookup"><span data-stu-id="84e5a-103">The constant 'variable' cannot be marked static</span></span>  
  
 <span data-ttu-id="84e5a-104">如果變數是 [const](../../../csharp/language-reference/keywords/const.md)，它也會是 [static](../../../csharp/language-reference/keywords/static.md)。</span><span class="sxs-lookup"><span data-stu-id="84e5a-104">If a variable is [const](../../../csharp/language-reference/keywords/const.md), it is also [static](../../../csharp/language-reference/keywords/static.md).</span></span> <span data-ttu-id="84e5a-105">如果您想要 **const** 和 **static** 變數，只要將該變數宣告為 **const** 即可；如果您想要 **static** 變數，只要將它標記成 **static** 即可。</span><span class="sxs-lookup"><span data-stu-id="84e5a-105">If you want a **const** and **static** variable, just declare that variable as **const**; if all you want is a **static** variable, just mark it **static**.</span></span>  
  
 <span data-ttu-id="84e5a-106">下列範例會產生 CS0504：</span><span class="sxs-lookup"><span data-stu-id="84e5a-106">The following sample generates CS0504:</span></span>  
  
```csharp  
// CS0504.cs  
namespace x  
{  
   abstract public class clx  
   {  
      static const int i = 0;   // CS0504, cannot be both static and const  
      abstract public void f();  
   }  
}  
```
