<span data-ttu-id="4055a-103">最符合 'method' 的多載方法，沒有正確的初始設定式項目簽章。</span><span class="sxs-lookup"><span data-stu-id="4055a-103">The best overloaded method match for 'method' has wrong signature for the initializer element.</span></span> <span data-ttu-id="4055a-104">可初始化的 Add 必須是可存取的執行個體方法。</span><span class="sxs-lookup"><span data-stu-id="4055a-104">The initializable Add must be an accessible instance method.</span></span>  
  
 <span data-ttu-id="4055a-105">嘗試搭配使用集合初始設定式與沒有公用非靜態 `Add` 方法的類別時，就會產生此錯誤。</span><span class="sxs-lookup"><span data-stu-id="4055a-105">This error is generated when you try to use a collection initializer with a class that has no public non-static `Add` method.</span></span> <span data-ttu-id="4055a-106">如果 `Add` 方法的保護層級 (`private`、`protected`、`internal`) 導致無法對其進行存取，則會收到 CS0122，因此這個錯誤可能表示方法已定義為 `static`。</span><span class="sxs-lookup"><span data-stu-id="4055a-106">If the `Add` method is not accessible because of its protection level (`private`, `protected`, `internal`) then you will get CS0122, so that this error probably means that the method is defined as `static`.</span></span>
