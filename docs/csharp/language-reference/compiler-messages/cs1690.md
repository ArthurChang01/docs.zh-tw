# <a name="compiler-warning-level-1-cs1690"></a><span data-ttu-id="490cc-102">編譯器警告 (層級 1) CS1690</span><span class="sxs-lookup"><span data-stu-id="490cc-102">Compiler Warning (level 1) CS1690</span></span>
<span data-ttu-id="490cc-103">存取 'member' 上的成員可能會造成執行階段例外狀況，因為其為 marshal-by-reference 類別的欄位</span><span class="sxs-lookup"><span data-stu-id="490cc-103">Accessing a member on 'member' may cause a runtime exception because it is a field of a marshal-by-reference class</span></span>  
  
 <span data-ttu-id="490cc-104">當您嘗試在衍生自 <xref:System.MarshalByRefObject> 的類別成員上呼叫方法、屬性或索引子，而且此成員為實值型別時，就會發生此警告。</span><span class="sxs-lookup"><span data-stu-id="490cc-104">This warning occurs when you try to call a method, property, or indexer on a member of a class that derives from <xref:System.MarshalByRefObject>, and the member is a value type.</span></span> <span data-ttu-id="490cc-105">繼承自 `MarshalByRefObject` 的物件通常用於以傳址方式跨應用程式定義域進行封送處理。</span><span class="sxs-lookup"><span data-stu-id="490cc-105">Objects that inherit from `MarshalByRefObject` are typically intended to be marshaled by reference across an application domain.</span></span> <span data-ttu-id="490cc-106">如果有任何程式碼嘗試跨應用程式定義域直接存取這類物件的實值型別成員，就會發生執行階段例外狀況。</span><span class="sxs-lookup"><span data-stu-id="490cc-106">If any code ever attempts to directly access the value-type member of such an object across an application domain, a runtime exception will occur.</span></span> <span data-ttu-id="490cc-107">若要解決這個警告，請先將成員複製到區域變數，再於該變數上呼叫方法。</span><span class="sxs-lookup"><span data-stu-id="490cc-107">To resolve the warning, first copy the member into a local variable and call the method on that variable.</span></span>  
  
 <span data-ttu-id="490cc-108">下列範例會產生 CS1690：</span><span class="sxs-lookup"><span data-stu-id="490cc-108">The following sample generates CS1690:</span></span>  
  
```csharp  
// CS1690.cs  
using System;  
  
class WarningCS1690: MarshalByRefObject  
{  
   int i = 5;  
  
   public static void Main()   
   {  
     WarningCS1690 e = new WarningCS1690();  
     e.i.ToString();   // CS1690  
  
     // OK  
     int i = e.i;  
     i.ToString();  
     e.i = i;  
   }  
}  
```
