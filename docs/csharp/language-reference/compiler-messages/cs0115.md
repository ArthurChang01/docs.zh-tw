## <a name="example"></a><span data-ttu-id="2f78d-106">範例</span><span class="sxs-lookup"><span data-stu-id="2f78d-106">Example</span></span>  
 <span data-ttu-id="2f78d-107">下列範例會產生 CS0115。</span><span class="sxs-lookup"><span data-stu-id="2f78d-107">The following sample generates CS0115.</span></span> <span data-ttu-id="2f78d-108">您可以透過下列兩個方法之一來解決 CS0115：</span><span class="sxs-lookup"><span data-stu-id="2f78d-108">You can resolve CS0115 in one of two ways:</span></span>

```csharp  
// CS0115.cs  
namespace MyNamespace  
{  
    abstract public class MyClass1  
    {  
        public abstract int f();  
    }  
  
    abstract public class MyClass2  
    {  
        public override int f()   // CS0115  
        {  
            return 0;  
        }  
  
        public static void Main()  
        {  
        }  
    }  
}  
```
