# <a name="compiler-error-cs0052"></a><span data-ttu-id="c799a-102">編譯器錯誤 CS0052</span><span class="sxs-lookup"><span data-stu-id="c799a-102">Compiler Error CS0052</span></span>
<span data-ttu-id="c799a-103">不一致的存取範圍：欄位類型 'type' 比欄位 'field' 的存取範圍小</span><span class="sxs-lookup"><span data-stu-id="c799a-103">Inconsistent accessibility: field type 'type' is less accessible than field 'field'</span></span>  
  
 <span data-ttu-id="c799a-104">欄位類型的存取範圍不可以小於欄位本身，因為所有的公用建構都必須傳回可公開存取的物件。</span><span class="sxs-lookup"><span data-stu-id="c799a-104">The type of a field cannot be less accessible than the field itself because all public constructs must return a publicly accessible object.</span></span>  
  
## <a name="example"></a><span data-ttu-id="c799a-105">範例</span><span class="sxs-lookup"><span data-stu-id="c799a-105">Example</span></span>  
 <span data-ttu-id="c799a-106">下列範例會產生 CS0052：</span><span class="sxs-lookup"><span data-stu-id="c799a-106">The following sample generates CS0052:</span></span>  
  
```csharp  
// CS0052.cs  
public class MyClass2  
{  
    // The following line causes an error because the field, M, is declared  
    // as public, but the type, MyClass, is private. Therefore the type is   
    // less accessible than the field.  
    public MyClass M;   // CS0052  
  
    private class MyClass  
    {  
    }  
    // One way to resolve the error is to change the accessibility of the type  
    // to public.   
    //public class MyClass  
    // Another solution is to change the accessibility of the field to private.  
    // private MyClass M;  
}  
  
    public class MainClass  
    {  
        public static void Main()  
        {  
        }  
    }  
```
