# <a name="compiler-error-cs0151"></a>編譯器錯誤 CS0151

必須是整數型別的值

在需要整數資料型別的情況下使用了變數。 如需詳細資訊，請參閱[類型](../../../csharp/programming-guide/types/index.md)。

## <a name="example-of-ambiguous-conversion"></a>模稜兩可的轉換範例

如果沒有轉換，或是可用的隱含轉換導致模稜兩可的情況，就會發生此錯誤。 下列範例會產生 CS0151：

```csharp
public class MyClass
{
   public static implicit operator int (MyClass aa)
   {
      return 0;
   }

   public static implicit operator long (MyClass aa)
   {
      return 0;
   }

   public static void Main()
   {
      MyClass a = new MyClass();

      // Compiler cannot choose between int and long.
      switch (a)   // CS0151
      // try the following line instead
      // switch ((int)a)
      {
         case 1:
            break;
      }
   }
}
```

## <a name="example-of-void-method"></a>void 方法範例

[switch](../keywords/switch.md) 比對運算式中的 [void](../../../csharp/language-reference/keywords/void.md) 方法引動過程會產生 CS0151。 您可以改為呼叫傳回整數型別 (例如 [int](../../../csharp/language-reference/keywords/int.md) 或 [long](../../../csharp/language-reference/keywords/long.md)) 的方法，來修正此錯誤。

```csharp
class C
{
    static void Main()
    {
        switch (M()) // CS0151
        {
            default:
                break;
        }
    }

    static void M()
    {
    }
}
```
