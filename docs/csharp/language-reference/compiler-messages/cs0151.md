# <a name="compiler-error-cs0151"></a><span data-ttu-id="ab5ed-102">編譯器錯誤 CS0151</span><span class="sxs-lookup"><span data-stu-id="ab5ed-102">Compiler Error CS0151</span></span>

<span data-ttu-id="ab5ed-103">必須是整數型別的值</span><span class="sxs-lookup"><span data-stu-id="ab5ed-103">A value of an integral type expected</span></span>

<span data-ttu-id="ab5ed-104">在需要整數資料型別的情況下使用了變數。</span><span class="sxs-lookup"><span data-stu-id="ab5ed-104">A variable was used in a situation where an integral data type was required.</span></span> <span data-ttu-id="ab5ed-105">如需詳細資訊，請參閱[類型](../../programming-guide/types/index.md)。</span><span class="sxs-lookup"><span data-stu-id="ab5ed-105">For more information, see [Types](../../programming-guide/types/index.md).</span></span>

## <a name="example-of-ambiguous-conversion"></a><span data-ttu-id="ab5ed-106">模稜兩可的轉換範例</span><span class="sxs-lookup"><span data-stu-id="ab5ed-106">Example of ambiguous conversion</span></span>

<span data-ttu-id="ab5ed-107">如果沒有轉換，或是可用的隱含轉換導致模稜兩可的情況，就會發生此錯誤。</span><span class="sxs-lookup"><span data-stu-id="ab5ed-107">This error can occur when there is no conversion or if the available implicit conversions result in an ambiguous situation.</span></span> <span data-ttu-id="ab5ed-108">下列範例會產生 CS0151：</span><span class="sxs-lookup"><span data-stu-id="ab5ed-108">The following example generates CS0151:</span></span>

```csharp
public class MyClass
{
   public static implicit operator int (MyClass aa)
   {
      return 0;
   }

   public static implicit operator long (MyClass aa)
   {
      return 0;
   }

   public static void Main()
   {
      MyClass a = new MyClass();

      // Compiler cannot choose between int and long.
      switch (a)   // CS0151
      // try the following line instead
      // switch ((int)a)
      {
         case 1:
            break;
      }
   }
}
```

## <a name="example-of-void-method"></a><span data-ttu-id="ab5ed-109">void 方法範例</span><span class="sxs-lookup"><span data-stu-id="ab5ed-109">Example of void method</span></span>

<span data-ttu-id="ab5ed-110">[switch](../keywords/switch.md) 比對運算式中的 [void](../keywords/void.md) 方法引動過程會產生 CS0151。</span><span class="sxs-lookup"><span data-stu-id="ab5ed-110">A [void](../keywords/void.md) method invocation in a [switch](../keywords/switch.md) match expression generates CS0151.</span></span> <span data-ttu-id="ab5ed-111">您可以改為呼叫傳回整數型別 (例如 [int](../builtin-types/integral-numeric-types.md) 或 [long](../builtin-types/integral-numeric-types.md)) 的方法，來修正此錯誤。</span><span class="sxs-lookup"><span data-stu-id="ab5ed-111">You can fix the error by calling a method that returns an integral type such as [int](../builtin-types/integral-numeric-types.md) or [long](../builtin-types/integral-numeric-types.md) instead.</span></span>

```csharp
class C
{
    static void Main()
    {
        switch (M()) // CS0151
        {
            default:
                break;
        }
    }

    static void M()
    {
    }
}
```
