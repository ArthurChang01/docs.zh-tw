## <a name="example-of-ambiguous-conversion"></a><span data-ttu-id="64f86-106">模稜兩可的轉換範例</span><span class="sxs-lookup"><span data-stu-id="64f86-106">Example of ambiguous conversion</span></span>

<span data-ttu-id="64f86-107">如果沒有轉換，或是可用的隱含轉換導致模稜兩可的情況，就會發生此錯誤。</span><span class="sxs-lookup"><span data-stu-id="64f86-107">This error can occur when there is no conversion or if the available implicit conversions result in an ambiguous situation.</span></span> <span data-ttu-id="64f86-108">下列範例會產生 CS0151：</span><span class="sxs-lookup"><span data-stu-id="64f86-108">The following example generates CS0151:</span></span>

```csharp
public class MyClass
{
   public static implicit operator int (MyClass aa)
   {
      return 0;
   }

   public static implicit operator long (MyClass aa)
   {
      return 0;
   }

   public static void Main()
   {
      MyClass a = new MyClass();

      // Compiler cannot choose between int and long.
      switch (a)   // CS0151
      // try the following line instead
      // switch ((int)a)
      {
         case 1:
            break;
      }
   }
}
```

## <a name="example-of-void-method"></a><span data-ttu-id="64f86-109">void 方法範例</span><span class="sxs-lookup"><span data-stu-id="64f86-109">Example of void method</span></span>

```csharp

```
