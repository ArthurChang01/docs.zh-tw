# <a name="compiler-error-cs0001"></a>編譯器錯誤 CS0001
內部編譯器錯誤

 請嘗試判斷編譯器失敗的原因是否為無法剖析未預期的語法。 如果您重複收到此錯誤，請連絡 Microsoft。
