## <a name="example"></a><span data-ttu-id="26ccb-108">範例</span><span class="sxs-lookup"><span data-stu-id="26ccb-108">Example</span></span>

<span data-ttu-id="26ccb-109">因為要求 `equals` 運算子比較 `int` 與 `string`，所以下列程式碼會產生 CS1941。</span><span class="sxs-lookup"><span data-stu-id="26ccb-109">The following code generates CS1941 because the `equals` operator is being asked to compare an `int` to a `string`.</span></span>  

```csharp
// cs1941.cs  
using System.Collections;  
using System.Linq;  
class Test  
{  
    static int Main()  
    {  
        var nums = new[] { 1, 2, 3, 4, 5, 6 };  
        var words = new string[] { "lake", "mountain", "sky" };  
        IEnumerable e = from n in nums  
                        join w in words on n equals w // CS1941  
                        select w;  
        return 0;  
    }  
}  
```

<span data-ttu-id="26ccb-110">型別推斷失敗的方法就是在編譯時間轉譯查詢子句的方法。</span><span class="sxs-lookup"><span data-stu-id="26ccb-110">The method in which type inference fails is the method that the query clause is translated to at compile time.</span></span>
