# <a name="compiler-error-cs1943"></a><span data-ttu-id="7233b-102">編譯器錯誤 CS1943</span><span class="sxs-lookup"><span data-stu-id="7233b-102">Compiler Error CS1943</span></span>
<span data-ttu-id="7233b-103">在具有來源類型 'type' 之查詢運算式內後續的 from 子句中不允許有類型 'type' 的運算式。</span><span class="sxs-lookup"><span data-stu-id="7233b-103">An expression of type 'type' is not allowed in a subsequent from clause in a query expression with source type 'type'.</span></span> <span data-ttu-id="7233b-104">呼叫 'method' 時發生型別推斷失敗。</span><span class="sxs-lookup"><span data-stu-id="7233b-104">Type inference failed in the call to 'method'.</span></span>  
  
 <span data-ttu-id="7233b-105">所有範圍變數必須代表可查詢型別。</span><span class="sxs-lookup"><span data-stu-id="7233b-105">All range variables must represent queryable types.</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="7233b-106">更正這個錯誤</span><span class="sxs-lookup"><span data-stu-id="7233b-106">To correct this error</span></span>  
  
1.  <span data-ttu-id="7233b-107">確定類型是實作 `IEnumerable`、`IEnumerable<T>` 或衍生介面的可查詢型別，或是已定義查詢模式的任何其他類型。</span><span class="sxs-lookup"><span data-stu-id="7233b-107">Make sure that the type is a queryable type that implements `IEnumerable`, `IEnumerable<T>` or a derived interface, or any other type which has a query pattern defined for it.</span></span>  
  
2.  <span data-ttu-id="7233b-108">如果類型是非泛型 `IEnumerable`，請在範圍變數上提供明確類型。</span><span class="sxs-lookup"><span data-stu-id="7233b-108">If the type is a non-generic `IEnumerable`, provide an explicit type on the range variable.</span></span>  
  
## <a name="example"></a><span data-ttu-id="7233b-109">範例</span><span class="sxs-lookup"><span data-stu-id="7233b-109">Example</span></span>  
 <span data-ttu-id="7233b-110">下列程式碼會產生 CS1943：</span><span class="sxs-lookup"><span data-stu-id="7233b-110">The following code generates CS1943:</span></span>  
  
```csharp  
// cs1943.cs  
using System.Linq;  
class Test  
{  
    class TestClass  
    { }  
    static void Main()  
    {  
        int[] nums = { 0, 1, 2, 3, 4, 5 };  
        TestClass tc = new TestClass();  
  
        var x = from n in nums  
                from s in tc // CS1943  
                select n + s;  
    }  
}  
```
