# Compiler Error CS0006
找不到中繼資料檔 'dll\_name'  
  
 程式已編譯，並明確地傳遞包含中繼資料 \(Metadata\) 的檔案名稱；不過，仍找不到該 .dll。  如需詳細資訊，請參閱 [\/reference \(Import Metadata\)](../../../csharp/language-reference/compiler-options/reference-compiler-option.md)。
