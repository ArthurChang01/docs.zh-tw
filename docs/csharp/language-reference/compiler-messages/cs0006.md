# <a name="compiler-error-cs0006"></a>編譯器錯誤 CS0006

找不到中繼資料檔 'dll_name'
  
 程式已經過編譯，並已對其明確傳遞包含中繼資料之檔案的名稱，但找不到此 .dll。 如需詳細資訊，請參閱 [/reference (C# 編譯器選項)](../compiler-options/reference-compiler-option.md)。
