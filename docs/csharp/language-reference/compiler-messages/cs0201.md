# <a name="compiler-error-cs0201"></a>編譯器錯誤 CS0201

只有指派、呼叫、遞增、遞減和新增物件運算式可以當做陳述式使用  
  
 編譯器在遇到無效的陳述式時會產生錯誤。 無效語句是以分號結尾的任何行或行系列，不表示賦值[=](../operators/assignment-operator.md)（）、方法調用[（）、](../operators/member-access-operators.md#invocation-operator-)[新](../operators/new-operator.md)或[--](../operators/arithmetic-operators.md#decrement-operator---)[++](../operators/arithmetic-operators.md#increment-operator-)操作。 如需詳細資訊，請參閱[陳述式、運算式和運算子](../../programming-guide/statements-expressions-operators/index.md)。  
  
## <a name="example"></a>範例

 下列範例會產生 CS0201，因為 2 * 3 是運算式，而不是陳述式。 若要編譯程式碼，請嘗試將運算式的值指派給變數。  

```csharp
// CS0201.cs  
public class MainClass  
{  
   public static void Main()  
   {  
      2 * 3;   // CS0201  
      // Try the following line instead.  
      //   int i = 2 * 3;  
   }  
}  
```

## <a name="example"></a>範例

 下列範例會產生 CS0201，因為 checked 本身不是陳述式，即使透過遞增運算加以參數化也一樣。  

```csharp
// CS0201_b.cs  
// compile with: /target:library  
public class MyList<T>
{  
   public void Add(T x)  
   {  
      int i = 0;  
      if ( (object)x == null)  
      {  
         checked(i++);   // CS0201  
  
         // OK  
         checked {  
            i++;
         }  
      }  
   }  
}  
```
