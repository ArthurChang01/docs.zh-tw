## <a name="example"></a><span data-ttu-id="a7632-110">範例</span><span class="sxs-lookup"><span data-stu-id="a7632-110">Example</span></span>

<span data-ttu-id="a7632-111">下例會產生 CS1936：</span><span class="sxs-lookup"><span data-stu-id="a7632-111">The following example produces CS1936:</span></span>  

```csharp
// cs1936.cs  
using System.Collections;  
using System.Linq;  
class Test  
{  
    static int Main()  
    {  
        object obj;  
        IEnumerable e = from x in obj // CS1936  
                        select x;  
        return 0;  
    }  
}  
```

<span data-ttu-id="a7632-112">當您不小心嘗試查詢某類型的一個物件，而非一組該類型的物件時，通常會發生此錯誤。</span><span class="sxs-lookup"><span data-stu-id="a7632-112">This error typically occurs when you accidentally try to query an object of some type instead of a collection of those objects.</span></span>
