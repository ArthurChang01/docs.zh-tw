# <a name="compiler-warning-level-2-cs0618"></a><span data-ttu-id="12148-102">編譯器警告 (層級 2) CS0618</span><span class="sxs-lookup"><span data-stu-id="12148-102">Compiler Warning (level 2) CS0618</span></span>
<span data-ttu-id="12148-103">'member' 已過時: 'text'</span><span class="sxs-lookup"><span data-stu-id="12148-103">'member' is obsolete: 'text'</span></span>  
  
 <span data-ttu-id="12148-104">類別成員已標上 `Obsolete` 屬性；因此，參考類別成員時，會發出警告。</span><span class="sxs-lookup"><span data-stu-id="12148-104">A class member was marked with the `Obsolete` attribute, such that a warning will be issued when the class member is referenced.</span></span> <span data-ttu-id="12148-105">如需詳細資訊，請參閱[常見屬性](../../programming-guide/concepts/attributes/common-attributes.md)。</span><span class="sxs-lookup"><span data-stu-id="12148-105">For more information, see [Common Attributes](../../programming-guide/concepts/attributes/common-attributes.md).</span></span>  
  
 <span data-ttu-id="12148-106">下列範例會產生 CS0618：</span><span class="sxs-lookup"><span data-stu-id="12148-106">The following sample generates CS0618:</span></span>  
  
```csharp  
// CS0618.cs  
// compile with: /W:2  
using System;  
  
public class C  
{  
   [Obsolete("Use newMethod instead", false)]   // warn if referenced  
   public static void m2()  
   {  
   }  
  
   public static void newMethod()  
   {  
   }  
}  
  
class MyClass  
{  
   public static void Main()  
   {  
      C.m2();  // CS0618  
   }  
}  
```
