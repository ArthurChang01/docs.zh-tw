# <a name="compiler-error-cs0703"></a><span data-ttu-id="a67bd-102">編譯器錯誤 CS0703</span><span class="sxs-lookup"><span data-stu-id="a67bd-102">Compiler Error CS0703</span></span>
<span data-ttu-id="a67bd-103">不一致的存取範圍：條件約束類型 'identifier' 比 'identifier' 的存取範圍小</span><span class="sxs-lookup"><span data-stu-id="a67bd-103">Inconsistent accessibility: constraint type 'identifier' is less accessible than 'identifier'</span></span>  
  
 <span data-ttu-id="a67bd-104">條件約束不能強制泛型參數的存取範圍小於泛型類別本身。</span><span class="sxs-lookup"><span data-stu-id="a67bd-104">A constraint may not force the generic parameter to be less accessible than the generic class itself.</span></span> <span data-ttu-id="a67bd-105">在下列範例中，雖然泛型類別 C\<T> 宣告為 public，但條件約束會嘗試強制 T 實作內部介面。</span><span class="sxs-lookup"><span data-stu-id="a67bd-105">In the following example, while the generic class C\<T> is declared public, the constraint attempts to force T to implement an internal interface.</span></span> <span data-ttu-id="a67bd-106">即使允許這樣做，只有具內部存取權的用戶端才能建立此類別的參數，因此實際上，只有具內部存取權的用戶端才能夠使用此類別。</span><span class="sxs-lookup"><span data-stu-id="a67bd-106">Even if this were allowed, only clients with internal access would be able to create the parameter for the class, so in effect, the class could be used only by clients with internal access.</span></span>  
  
 <span data-ttu-id="a67bd-107">若要解決這個錯誤，請確定泛型類別的存取層級，比出現在界限內的任何類別或介面都還要嚴格。</span><span class="sxs-lookup"><span data-stu-id="a67bd-107">To get rid of this error, make sure the access level of the generic class is not less restrictive than any classes or interfaces that appear in the bounds.</span></span>  
  
 <span data-ttu-id="a67bd-108">下列範例會產生 CS0703：</span><span class="sxs-lookup"><span data-stu-id="a67bd-108">The following sample generates CS0703:</span></span>  
  
```csharp  
// CS0703.cs  
internal interface I {}  
public class C<T> where T : I  // CS0703 – I is internal; C<T> is public  
{  
}  
```
