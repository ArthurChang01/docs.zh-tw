## <a name="example"></a><span data-ttu-id="eec2a-108">範例</span><span class="sxs-lookup"><span data-stu-id="eec2a-108">Example</span></span>

 <span data-ttu-id="eec2a-109">在下列範例中，因為 C# 中的 [bool](../../../csharp/language-reference/keywords/bool.md) 無法轉換成 [int](../../../csharp/language-reference/builtin-types/integral-numeric-types.md)，所以會在兩個位置產生 CS0019。將減法運算子套用至字串時，也會產生 CS0019。</span><span class="sxs-lookup"><span data-stu-id="eec2a-109">In the following example, CS0019 is generated in two places because [bool](../../../csharp/language-reference/keywords/bool.md) in C# is not convertible to [int](../../../csharp/language-reference/builtin-types/integral-numeric-types.md). CS0019 also is generated when the subtraction operator is applied to a string.</span></span> <span data-ttu-id="eec2a-110">加法運算子 (+) 可搭配字串運算元使用，因為 `String` 類別會多載該運算子來執行字串串連。</span><span class="sxs-lookup"><span data-stu-id="eec2a-110">The addition operator (+) can be used with string operands because that operator is overloaded by the `String` class to perform string concatenation.</span></span>  

```csharp
static void Main()  
{  
    bool result = true;  
    if (result > 0) //CS0019  
    {  
        // Do something.  
    }  
  
    int i = 1;  
    // You cannot compare an integer and a boolean value.  
    if (i == true) //CS0019  
    {  
        //Do something...  
    }  
  
    // The following use of == causes no error. It is the comparison of
    // an integer and a boolean value that causes the error in the
    // previous if statement.  
    if (result == true)  
    {  
        //Do something...  
    }  
  
    string s = "Just try to subtract me.";  
    float f = 100 - s; // CS0019  
}  
```

## <a name="example"></a><span data-ttu-id="eec2a-111">範例</span><span class="sxs-lookup"><span data-stu-id="eec2a-111">Example</span></span>

 <span data-ttu-id="eec2a-114">下列範例會產生 CS0019。</span><span class="sxs-lookup"><span data-stu-id="eec2a-114">The following sample generates CS0019.</span></span>  

```csharp

{

```
