## <a name="example"></a><span data-ttu-id="6356e-108">範例</span><span class="sxs-lookup"><span data-stu-id="6356e-108">Example</span></span>

 <span data-ttu-id="6356e-109">在下面的示例中，CS0019 在兩個位置生成，因為 C# 中的[bool](../builtin-types/bool.md)不可轉換為[int](../builtin-types/integral-numeric-types.md)。當減法運算子`-`應用於字串時，也會生成 CS0019。</span><span class="sxs-lookup"><span data-stu-id="6356e-109">In the following example, CS0019 is generated in two places because [bool](../builtin-types/bool.md) in C# is not convertible to [int](../builtin-types/integral-numeric-types.md). CS0019 also is generated when the subtraction operator `-` is applied to a string.</span></span> <span data-ttu-id="6356e-110">添加運算子`+`可用於字串運算元，因為`String`該運算子被類重載以執行字串串聯。</span><span class="sxs-lookup"><span data-stu-id="6356e-110">The addition operator `+` can be used with string operands because that operator is overloaded by the `String` class to perform string concatenation.</span></span>

```csharp
static void Main()
{
    bool result = true;
    if (result > 0) //CS0019
    {
        // Do something.
    }

    int i = 1;
    // You cannot compare an integer and a boolean value.
    if (i == true) //CS0019
    {
        //Do something...
    }

    string s = "Just try to subtract me.";
    float f = 100 - s; // CS0019
}
```
