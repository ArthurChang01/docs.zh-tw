# <a name="compiler-error-cs1519"></a><span data-ttu-id="b77b6-102">編譯器錯誤 CS1519</span><span class="sxs-lookup"><span data-stu-id="b77b6-102">Compiler Error CS1519</span></span>
<span data-ttu-id="b77b6-103">類別、結構或介面成員宣告中的無效權杖 'token'</span><span class="sxs-lookup"><span data-stu-id="b77b6-103">Invalid token 'token' in class, struct, or interface member declaration</span></span>  
  
 <span data-ttu-id="b77b6-104">每當在非權杖所屬位置中發現權杖時，就會產生此錯誤。</span><span class="sxs-lookup"><span data-stu-id="b77b6-104">This error is generated whenever a token is encountered in a location where it does not belong.</span></span> <span data-ttu-id="b77b6-105">「權杖」是關鍵字；是識別碼 (類別、結構、方法等等的名稱)；是 108、"Hello" 或 'A' 等字串、字元或數值常值；或者是運算子或標點符號，例如 `==` 或 `;`。</span><span class="sxs-lookup"><span data-stu-id="b77b6-105">A *token* is a keyword; an identifier (the name of a class, struct, method, and so on); a string, character, or numeric literal value such as 108, "Hello", or 'A'; or an operator or punctuator such as `==` or `;`.</span></span>  
  
 <span data-ttu-id="b77b6-106">類型產生此錯誤前，任何包含無效修飾詞的[類別](../../../csharp/language-reference/keywords/class.md)、結構或介面成員。</span><span class="sxs-lookup"><span data-stu-id="b77b6-106">Any [class](../../../csharp/language-reference/keywords/class.md), struct, or interface member declaration that contains invalid modifiers before the type will generate this error.</span></span> <span data-ttu-id="b77b6-107">若要修正錯誤，請移除無效的修飾詞。</span><span class="sxs-lookup"><span data-stu-id="b77b6-107">To fix the error, remove the invalid modifiers.</span></span>  
  
 <span data-ttu-id="b77b6-108">下列範例會在五個位置中產生 CS1519，因為權杖放置在它們無效的位置中︰</span><span class="sxs-lookup"><span data-stu-id="b77b6-108">The following sample generates CS1519 in five places because tokens are placed in locations where they are not valid:</span></span>  
  
```csharp  
// CS1519.cs  
// Generates CS1519 because a class name cannot be a number:  
class Test 42   
{  
// Generates CS1519 because of 'j' following 'I'  
// with no comma between them:  
    int i j;   
// Generates CS1519 because of "checked" on void method:  
    checked void f4();     
  
// Generates CS1519 because of "num":  
    void f5(int a num){}        
  
// Generates CS1519 because of namespace inside class:  
    namespace;             
  
}  
```
