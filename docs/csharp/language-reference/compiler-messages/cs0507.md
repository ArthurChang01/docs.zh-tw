# <a name="compiler-error-cs0507"></a><span data-ttu-id="815bc-102">編譯器錯誤 CS0507</span><span class="sxs-lookup"><span data-stu-id="815bc-102">Compiler Error CS0507</span></span>
<span data-ttu-id="815bc-103">'function1'：覆寫 'access' 繼承的成員 'function2' 時無法變更存取修飾詞</span><span class="sxs-lookup"><span data-stu-id="815bc-103">'function1' : cannot change access modifiers when overriding 'access' inherited member 'function2'</span></span>  
  
 <span data-ttu-id="815bc-104">嘗試變更方法覆寫中的存取規格。</span><span class="sxs-lookup"><span data-stu-id="815bc-104">An attempt was made to change the access specification in a method override.</span></span>  
  
## <a name="example"></a><span data-ttu-id="815bc-105">範例</span><span class="sxs-lookup"><span data-stu-id="815bc-105">Example</span></span>  
 <span data-ttu-id="815bc-106">下列範例會產生 CS0507。</span><span class="sxs-lookup"><span data-stu-id="815bc-106">The following sample generates CS0507.</span></span>  
  
```csharp  
// CS0507.cs  
abstract public class clx  
{  
   virtual protected void f() {}  
}  
  
public class cly : clx  
{  
   public override void f() {}   // CS0507  
   public static void Main() {}  
}  
```  
  
## <a name="example"></a><span data-ttu-id="815bc-107">範例</span><span class="sxs-lookup"><span data-stu-id="815bc-107">Example</span></span>  
 <span data-ttu-id="815bc-108">如果類別嘗試覆寫的方法已標記為 `protected internal` 並定義於所參考的中繼資料中，也可能會發生 CS0507。</span><span class="sxs-lookup"><span data-stu-id="815bc-108">CS0507 can also occur if a class attempts to override a method marked as `protected internal` defined in referenced metadata.</span></span> <span data-ttu-id="815bc-109">在此情況下，覆寫方法應該標記為 `protected`。</span><span class="sxs-lookup"><span data-stu-id="815bc-109">In this situation, the overriding method should be marked as `protected`.</span></span>  
  
```csharp  
// CS0507_b.cs  
// compile with: /target:library  
abstract public class clx  
{  
   virtual protected internal void f() {}  
}  
```  
  
## <a name="example"></a><span data-ttu-id="815bc-110">範例</span><span class="sxs-lookup"><span data-stu-id="815bc-110">Example</span></span>  
 <span data-ttu-id="815bc-111">下列範例會產生 CS0507。</span><span class="sxs-lookup"><span data-stu-id="815bc-111">The following sample generates CS0507.</span></span>

