# <a name="compiler-error-cs0311"></a><span data-ttu-id="3fef8-102">編譯器錯誤 CS0311</span><span class="sxs-lookup"><span data-stu-id="3fef8-102">Compiler Error CS0311</span></span>
<span data-ttu-id="3fef8-103">'type1' 類型不能作為泛型類型或 '\<name>' 方法中的類型參數 'T' 使用。</span><span class="sxs-lookup"><span data-stu-id="3fef8-103">The type 'type1' cannot be used as type parameter 'T' in the generic type or method '\<name>'.</span></span> <span data-ttu-id="3fef8-104">從 'type1' 到 'type2' 沒有隱含參考轉換。</span><span class="sxs-lookup"><span data-stu-id="3fef8-104">There is no implicit reference conversion from 'type1' to 'type2'.</span></span>  
  
 <span data-ttu-id="3fef8-105">將條件約束套用至泛型類型參數時，從實體引數到條件約束的類型必須存在隱含識別或參考轉換。</span><span class="sxs-lookup"><span data-stu-id="3fef8-105">When a constraint is applied to a generic type parameter, an implicit identity or reference conversion must exist from the concrete argument to the type of the constraint.</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="3fef8-106">更正這個錯誤</span><span class="sxs-lookup"><span data-stu-id="3fef8-106">To correct this error</span></span>  
  
1.  <span data-ttu-id="3fef8-107">變更您用以建立此類別的引數。</span><span class="sxs-lookup"><span data-stu-id="3fef8-107">Change the argument you are using to create the class.</span></span>  
  
2.  <span data-ttu-id="3fef8-108">如果您擁有該類別，可以移除條件約束，或是執行可啟用隱含參考或識別轉換的動作。</span><span class="sxs-lookup"><span data-stu-id="3fef8-108">If you own the class, you can remove the constraint or else do something to enable an implicit reference or identity conversion.</span></span> <span data-ttu-id="3fef8-109">例如，您可以讓第二個類型繼承自第一個類型。</span><span class="sxs-lookup"><span data-stu-id="3fef8-109">For example, you can make the second type inherit from the first.</span></span>  
  
## <a name="example"></a><span data-ttu-id="3fef8-110">範例</span><span class="sxs-lookup"><span data-stu-id="3fef8-110">Example</span></span>  
  
```csharp  
// cs0311.cs  
class B{}  
class C{}  
class Test<T> where T : C  
{ }  
  
class Program  
{  
    static void Main()  
    {  
        Test<B> test = new Test<B>(); //CS0311  
    }  
}  
```  
  
 <span data-ttu-id="3fef8-111">如果在嘗試使用實值類型引數時發生此錯誤，請注意，隱含數值轉換 (例如從 `short` 到 `int`) 並無法滿足泛型類型參數。</span><span class="sxs-lookup"><span data-stu-id="3fef8-111">If this error occurs when trying to use a value-type argument, notice that an implicit numeric conversion, for example from `short` to `int`, does not satisfy a generic type parameter.</span></span>
