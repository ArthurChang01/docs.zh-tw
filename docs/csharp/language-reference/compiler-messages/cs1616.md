 <span data-ttu-id="be1b4-104">如果原始檔中的組件屬性 <xref:System.Reflection.AssemblyKeyFileAttribute> 或 <xref:System.Reflection.AssemblyKeyNameAttribute>，與 [/keyfile](../compiler-options/keyfile-compiler-option.md) 或 [/keycontainer](../compiler-options/keycontainer-compiler-option.md) 命令列選項，或是 [專案屬性] 中指定的金鑰檔名稱或金鑰容器發生衝突，就會發生此警告。</span><span class="sxs-lookup"><span data-stu-id="be1b4-104">This warning occurs if the assembly attributes <xref:System.Reflection.AssemblyKeyFileAttribute> or <xref:System.Reflection.AssemblyKeyNameAttribute> found in source conflict with the [/keyfile](../compiler-options/keyfile-compiler-option.md) or [/keycontainer](../compiler-options/keycontainer-compiler-option.md) command line option or key file name or key container specified in the Project Properties.</span></span>

  }
