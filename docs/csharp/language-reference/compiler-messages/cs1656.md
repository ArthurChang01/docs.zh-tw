 <span data-ttu-id="124c5-104">在唯讀內容中指派值給變數時，就會發生此錯誤。</span><span class="sxs-lookup"><span data-stu-id="124c5-104">This error occurs when an assignment to variable occurs in a read-only context.</span></span> <span data-ttu-id="124c5-105">唯讀內容包括 [foreach](../keywords/foreach-in.md) 反覆運算變數、[using](../keywords/using-statement.md) 變數和 [fixed](../keywords/fixed-statement.md) 變數。</span><span class="sxs-lookup"><span data-stu-id="124c5-105">Read-only contexts include [foreach](../keywords/foreach-in.md) iteration variables, [using](../keywords/using-statement.md) variables, and [fixed](../keywords/fixed-statement.md) variables.</span></span> <span data-ttu-id="124c5-106">若要解決這個錯誤，請避免在 `using` 區塊、`foreach` 陳述式和 `fixed` 陳述式中指派值給陳述式變數。</span><span class="sxs-lookup"><span data-stu-id="124c5-106">To resolve this error, avoid assignments to a statement variable in `using` blocks, `foreach` statements, and `fixed` statements.</span></span>  
  
## <a name="example"></a><span data-ttu-id="124c5-107">範例</span><span class="sxs-lookup"><span data-stu-id="124c5-107">Example</span></span>  
 <span data-ttu-id="124c5-108">下列範例會產生錯誤 CS1656，因為它嘗試取代 `foreach` 迴圈內集合的所有元素。</span><span class="sxs-lookup"><span data-stu-id="124c5-108">The following example generates error CS1656 because it tries to replace complete elements of a collection inside a `foreach` loop.</span></span> <span data-ttu-id="124c5-109">解決這個錯誤的一個方法是將 `foreach` 迴圈變更為 [for](../keywords/for.md) 迴圈。</span><span class="sxs-lookup"><span data-stu-id="124c5-109">One way to work around the error is to change the `foreach` loop to a [for](../keywords/for.md) loop.</span></span> <span data-ttu-id="124c5-110">另一個在此並未說明的方法是修改現有元素的成員；這可能適用於類別，但不適用於結構。</span><span class="sxs-lookup"><span data-stu-id="124c5-110">Another way, not shown here, is to modify the members of the existing element; this is possible with classes, but not with structs.</span></span>  
  
```csharp  
using System;  
using System.Collections;  
using System.Collections.Generic;  
using System.Text;  
  
namespace CS1656_2  
{  
  
    class Book  
    {  
        public string Title;  
        public string Author;  
        public double Price;  
        public Book(string t, string a, double p)  
        {  
            Title=t;  
            Author=a;  
            Price=p;  
  
        }  
    }  
  
    class Program  
    {  
        private List<Book> list;  
        static void Main(string[] args)  
        {  
            Program prog = new Program();  
            prog.list = new List<Book>();  
            prog.list.Add(new Book ("The C# Programming Language",  
                                    "Hejlsberg, Wiltamuth, Golde",  
                                     29.95));  
            prog.list.Add(new Book ("The C++ Programming Language",  
                                    "Stroustrup",  
                                     29.95));  
            prog.list.Add(new Book ("The C Programming Language",  
                                    "Kernighan, Ritchie",  
                                    29.95));  
            foreach(Book b in prog.list)  
            {  
                // Cannot modify an entire element in a foreach loop
                // even with reference types.  
                // Use a for or while loop instead  
                if (b.Title == "The C Programming Language")  
                    // Cannot assign to 'b' because it is a 'foreach
                    // iteration variable'  
                    b = new Book("Programming Windows, 5th Ed.", "Petzold", 29.95); //CS1656  
            }  
  
            //With a for loop you can modify elements  
            //for(int x = 0; x < prog.list.Count; x++)  
            //{  
            //    if(prog.list[x].Title== "The C Programming Language")  
            //        prog.list[x] = new Book("Programming Windows, 5th Ed.", "Petzold", 29.95);  
            //}  
            //foreach(Book b in prog.list)  
            //    Console.WriteLine(b.Title);  
  
        }  
    }  
}  
```  
  
## <a name="example"></a><span data-ttu-id="124c5-111">範例</span><span class="sxs-lookup"><span data-stu-id="124c5-111">Example</span></span>  
 <span data-ttu-id="124c5-112">下列範例示範如何在 `foreach` 迴圈以外的其他內容中產生 CS1656：</span><span class="sxs-lookup"><span data-stu-id="124c5-112">The following sample demonstrates how CS1656 can be generated in other contexts besides a `foreach` loop:</span></span>  
  
```csharp  
// CS1656.cs  
// compile with: /unsafe  
using System;  
  
class C : IDisposable  
{  
    public void Dispose() { }  
}  
  
class CMain  
{  
    unsafe public static void Main()  
    {  
        using (C c = new C())  
        {  
            // Cannot assign to 'c' because it is a 'using variable'  
            c = new C(); // CS1656  
        }  
  
        int[] ary = new int[] { 1, 2, 3, 4 };  
        fixed (int* p = ary)  
        {  
            // Cannot assign to 'p' because it is a 'fixed variable'  
            p = null; // CS1656  
        }  
    }  
}  
```
