## <a name="example"></a><span data-ttu-id="faea4-112">範例</span><span class="sxs-lookup"><span data-stu-id="faea4-112">Example</span></span>  
 <span data-ttu-id="faea4-113">下列範例會產生 CS0106。</span><span class="sxs-lookup"><span data-stu-id="faea4-113">The following sample generates CS0106.</span></span>  
  
```  
// CS0106.cs  
namespace MyNamespace  
{  
   interface I  
   {  
      void m();  
      static public void f();   // CS0106  
   }  
  
   public class MyClass  
   {  
      public void I.m() {}   // CS0106  
      public static void Main() {}  
   }  
}  
```
