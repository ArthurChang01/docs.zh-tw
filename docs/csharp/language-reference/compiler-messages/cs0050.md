## <a name="example"></a><span data-ttu-id="98106-106">範例</span><span class="sxs-lookup"><span data-stu-id="98106-106">Example</span></span>  
 <span data-ttu-id="98106-107">下列範例會產生 CS0050，因為沒有為 `MyClass` 提供存取範圍修飾詞，因此其存取範圍預設為 `private`。</span><span class="sxs-lookup"><span data-stu-id="98106-107">The following sample generates CS0050 because no accessibility modifier is supplied for `MyClass` and its accessibility therefore defaults to `private`.</span></span>

```csharp  
// try the following line instead  
// public class MyClass   
{  
}

{  
    public static MyClass MyMethod()   // CS0050  
    {  
        return new MyClass();  
    }  
  
    public static void Main() { }  
}  
```
