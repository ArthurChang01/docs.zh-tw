## <a name="example"></a><span data-ttu-id="adfa5-106">範例</span><span class="sxs-lookup"><span data-stu-id="adfa5-106">Example</span></span>  
 <span data-ttu-id="adfa5-107">下列範例會產生 CS0050，因為沒有提供 `MyClass` 所需的存取範圍修飾詞，因此其存取範圍預設為 `private`。</span><span class="sxs-lookup"><span data-stu-id="adfa5-107">The following sample generates CS0050 because no accessiblity modifier is supplied for `MyClass` and its accessibility therefore defaults to `private`.</span></span>

```csharp  
class MyClass //accessibility defaults to private  
// try the following line instead  
// public class MyClass   
{  
}  
  
public class MyClass2  
{  
    public static MyClass MyMethod()   // CS0050  
    {  
        return new MyClass();  
    }  
  
    public static void Main() { }  
}  
```
