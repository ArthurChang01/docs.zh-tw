# <a name="compiler-error-cs1942"></a><span data-ttu-id="0de0c-102">編譯器錯誤 CS1942</span><span class="sxs-lookup"><span data-stu-id="0de0c-102">Compiler Error CS1942</span></span>

<span data-ttu-id="0de0c-103">'clause' 子句中的運算式類型不正確。</span><span class="sxs-lookup"><span data-stu-id="0de0c-103">The type of the expression in the 'clause' clause is incorrect.</span></span> <span data-ttu-id="0de0c-104">呼叫 'method' 時發生型別推斷失敗。</span><span class="sxs-lookup"><span data-stu-id="0de0c-104">Type inference failed in the call to 'method'.</span></span>  
  
<span data-ttu-id="0de0c-105">為範圍變數指定了不正確的明確類型時，通常會產生此錯誤。</span><span class="sxs-lookup"><span data-stu-id="0de0c-105">This error is typically generated when the range variable has been given an incorrect explicit type.</span></span>  

## <a name="to-correct-this-error"></a><span data-ttu-id="0de0c-106">更正這個錯誤</span><span class="sxs-lookup"><span data-stu-id="0de0c-106">To correct this error</span></span>

1. <span data-ttu-id="0de0c-107">如果明確指定範圍變數的類型，請確定該類型與所逐一查看之集合中項目的類型相同，或是可從中隱含轉換的類型。</span><span class="sxs-lookup"><span data-stu-id="0de0c-107">If the range variable is explicitly typed, make sure that the type is either the same as, or implicitly convertible from, the type of the elements in the collection it iterates.</span></span> <span data-ttu-id="0de0c-108">如果範圍變數的前面加上 `var` 關鍵字，請移除 `var`。</span><span class="sxs-lookup"><span data-stu-id="0de0c-108">If the range variable is preceded with the `var` keyword, remove `var`.</span></span>  
  
## <a name="example"></a><span data-ttu-id="0de0c-109">範例</span><span class="sxs-lookup"><span data-stu-id="0de0c-109">Example</span></span>

<span data-ttu-id="0de0c-110">下列程式碼會產生 CS1942：</span><span class="sxs-lookup"><span data-stu-id="0de0c-110">The following code generates CS1942:</span></span>  

```csharp
// cs1942.cs  
class Program  
    {  
        static void Main(string[] args)  
        {  
            var x = from var i in Enumerable.Range(1, 100) // CS1949  
                    select i; //CS1942  
        }  
    }  
```

<span data-ttu-id="0de0c-111">CS1942 與 CS1949 相關，因為搭配使用 `var` 與範圍變數會因 `var` 不是類型而導致基礎 `Cast<T>` 作業失敗。</span><span class="sxs-lookup"><span data-stu-id="0de0c-111">CS1942 is related to CS1949 because the use of `var` with a range variable causes the underlying `Cast<T>` operation to fail because `var` is not a type.</span></span>
