# <a name="compiler-error-cs0702"></a>編譯器錯誤 CS0702
條件約束不能是特殊類別 'identifier'  
  
 下列類型不能作為條件約束：`System.Object,``System.Array`、`System.Delegate`、`System.Enum` 或 `System.ValueType`。  
  
## <a name="example"></a>範例  
 下列範例會產生 CS0702：  
  
```csharp  
// CS0702.cs  
class C<T> where T : System.Array  // CS0702  
{  
}  
```
