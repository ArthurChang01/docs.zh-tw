# <a name="compiler-error-cs0702"></a>編譯器錯誤 CS0702
條件約束不能是特殊類別 'identifier'  
  
 下列型別不能作為條件約束：<xref:System.Object>、<xref:System.Array> 或 <xref:System.ValueType>。  
  
## <a name="example"></a>範例  
 下列範例會產生 CS0702：  
  
```csharp  
// CS0702.cs  
class C<T> where T : System.Array  // CS0702  
{  
}  
```
