# <a name="compiler-error-cs1674"></a><span data-ttu-id="4b0da-102">編譯器錯誤 CS1674</span><span class="sxs-lookup"><span data-stu-id="4b0da-102">Compiler Error CS1674</span></span>
<span data-ttu-id="4b0da-103">'T'：在 using 陳述式中所用的類型，必須可以隱含轉換成 'System.IDisposable'</span><span class="sxs-lookup"><span data-stu-id="4b0da-103">'T': type used in a using statement must be implicitly convertible to 'System.IDisposable'</span></span>  
  
 <span data-ttu-id="4b0da-104">[using 陳述式](../../../csharp/language-reference/keywords/using-statement.md)是用來確定 `using` 區塊結尾物件的處置，因此這類陳述式中只能使用可處置的類型。</span><span class="sxs-lookup"><span data-stu-id="4b0da-104">The [using Statement](../../../csharp/language-reference/keywords/using-statement.md) is intended to be used to ensure the disposal of an object at the end of the `using` block, thus, only types which are disposable may be used in such a statement.</span></span> <span data-ttu-id="4b0da-105">例如，實值型別不是可處置的，而未限制為類別的型別參數也可能不假設為可處置的。</span><span class="sxs-lookup"><span data-stu-id="4b0da-105">For example, value types are not disposable, and type parameters which are not constrained to be classes may not be assumed to be disposable.</span></span>  
  
## <a name="example"></a><span data-ttu-id="4b0da-106">範例</span><span class="sxs-lookup"><span data-stu-id="4b0da-106">Example</span></span>

      int a = 0;

## <a name="example"></a><span data-ttu-id="4b0da-108">範例</span><span class="sxs-lookup"><span data-stu-id="4b0da-108">Example</span></span>  
 <span data-ttu-id="4b0da-109">下列範例會產生 CS1674。</span><span class="sxs-lookup"><span data-stu-id="4b0da-109">The following sample generates CS1674.</span></span>  
  
```csharp  
// CS1674_b.cs  
using System;  
class C {  
   public void Test() {  
      using (C c = new C()) {}   // CS1674  
   }  
}  
  
// OK  
class D : IDisposable {  
   void IDisposable.Dispose() {}  
   public void Dispose() {}  
  
   public static void Main() {  
      using (D d = new D()) {}  
   }  
}  
```
