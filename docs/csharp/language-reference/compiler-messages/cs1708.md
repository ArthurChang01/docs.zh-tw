# <a name="compiler-error-cs1708"></a><span data-ttu-id="bc303-102">編譯器錯誤 CS1708</span><span class="sxs-lookup"><span data-stu-id="bc303-102">Compiler Error CS1708</span></span>
<span data-ttu-id="bc303-103">固定大小緩衝區只能透過區域變數或欄位存取</span><span class="sxs-lookup"><span data-stu-id="bc303-103">Fixed size buffers can only be accessed through locals or fields</span></span>  
  
 <span data-ttu-id="bc303-104">C# 2.0 的新功能是可以在 `struct` 內定義內嵌陣列。</span><span class="sxs-lookup"><span data-stu-id="bc303-104">A new feature in C# 2.0 is the ability to define an in-line array inside of a `struct`.</span></span> <span data-ttu-id="bc303-105">這類陣列只能透過區域變數或欄位進行存取，而且可能未參考為運算式左側上的中繼值。</span><span class="sxs-lookup"><span data-stu-id="bc303-105">Such arrays can only be accessed via local variables or fields, and may not be referenced as intermediate values on the left-hand side of an expression.</span></span> <span data-ttu-id="bc303-106">此外，無法透過為 `static` 或 `readonly` 的欄位存取陣列。</span><span class="sxs-lookup"><span data-stu-id="bc303-106">Also, the arrays cannot be accessed by fields that are `static` or `readonly`.</span></span>  
  
 <span data-ttu-id="bc303-107">若要解決這個錯誤，請定義陣列變數，並將內嵌陣列指派給變數。</span><span class="sxs-lookup"><span data-stu-id="bc303-107">To resolve this error, define an array variable, and assign the in-line array to the variable.</span></span> <span data-ttu-id="bc303-108">或者，從表示內嵌陣列的欄位移除 `static` 或 `readonly` 修飾詞。</span><span class="sxs-lookup"><span data-stu-id="bc303-108">Or, remove the `static` or `readonly` modifier from the field representing the in-line array.</span></span>  
  
## <a name="example"></a><span data-ttu-id="bc303-109">範例</span><span class="sxs-lookup"><span data-stu-id="bc303-109">Example</span></span>  
 <span data-ttu-id="bc303-110">下列範例會產生 CS1708。</span><span class="sxs-lookup"><span data-stu-id="bc303-110">The following sample generates CS1708.</span></span>  
  
```csharp  
// CS1708.cs  
// compile with: /unsafe  
using System;  
  
unsafe public struct S  
{  
    public fixed char name[10];  
}  
  
public unsafe class C  
{  
    public S UnsafeMethod()  
    {  
        S myS = new S();  
        return myS;  
    }  
  
    static void Main()  
    {  
        C myC = new C();  
        myC.UnsafeMethod().name[3] = 'a';  // CS1708  
        // Uncomment the following 2 lines to resolve:  
        // S myS = myC.UnsafeMethod();  
        // myS.name[3] = 'a';  
  
        // The field cannot be static.  
        C._s1.name[3] = 'a';  // CS1708  
  
        // The field cannot be readonly.  
        myC._s2.name[3] = 'a';  // CS1708  
    }  
  
    static readonly S _s1;  
    public readonly S _s2;  
}  
```
