# <a name="compiler-warning-level-2-cs0108"></a><span data-ttu-id="d4617-102">編譯器警告 (層級 2) CS0108</span><span class="sxs-lookup"><span data-stu-id="d4617-102">Compiler Warning (level 2) CS0108</span></span>

<span data-ttu-id="d4617-103">'member1' 隱藏了繼承的成員 'member2'。</span><span class="sxs-lookup"><span data-stu-id="d4617-103">'member1' hides inherited member 'member2'.</span></span> <span data-ttu-id="d4617-104">如果是刻意要隱藏，請使用 new 關鍵字。</span><span class="sxs-lookup"><span data-stu-id="d4617-104">Use the new keyword if hiding was intended.</span></span>  
  
 <span data-ttu-id="d4617-105">所宣告的變數名稱與基底類別中的變數相同。</span><span class="sxs-lookup"><span data-stu-id="d4617-105">A variable was declared with the same name as a variable in a base class.</span></span> <span data-ttu-id="d4617-106">不過，未使用 [new](../../../csharp/language-reference/keywords/new.md) 關鍵字。</span><span class="sxs-lookup"><span data-stu-id="d4617-106">However, the [new](../../../csharp/language-reference/keywords/new.md) keyword was not used.</span></span> <span data-ttu-id="d4617-107">此警告通知您應該使用 **new**；宣告變數的方式就像已將 **new** 用於宣告一樣。</span><span class="sxs-lookup"><span data-stu-id="d4617-107">This warning informs you that you should use **new**; the variable is declared as if **new** had been used in the declaration.</span></span>  
  
 <span data-ttu-id="d4617-108">下列範例會產生 CS0108：</span><span class="sxs-lookup"><span data-stu-id="d4617-108">The following sample generates CS0108:</span></span>  

```csharp
// CS0108.cs  
// compile with: /W:2  
using System;  
  
namespace x  
{  
   public class clx  
   {  
      public int i = 1;  
   }  
  
   public class cly : clx  
   {  
      public static int i = 2;   // CS0108, use the new keyword  
      // Use the following line instead:  
      // public static new int i = 2;  
  
      public static void Main()  
      {  
         Console.WriteLine(i);  
      }  
   }  
}  
```
