# <a name="compiler-error-cs0233"></a>編譯器錯誤 CS0233
'identifier' 沒有預先定義的大小，因此只能在 unsafe 內容中使用 sizeof
  
 若沒有 [unsafe](../keywords/unsafe.md) 內容，則 [sizeof](../../../csharp/language-reference/operators/sizeof.md) 運算子便只能針對大小在編譯時間為常數的型別使用。 若您收到這項錯誤，請使用 unsafe 內容。
  
下列範例會產生 CS0233：
  
```csharp  
// CS0233.cs  
using System;  
using System.Runtime.InteropServices;  
  
[StructLayout(LayoutKind.Sequential)]  
public struct S  
{  
    public int a;  
}  
  
public class MyClass  
{  
    public static void Main()  
    {  
        S myS = new S();  
        Console.WriteLine(sizeof(S));   // CS0233  
        // Try the following instead:  
        // unsafe
        // {
        //     Console.WriteLine(sizeof(S));
        // }
   }  
}  
```
