# <a name="compiler-error-cs1029"></a>編譯器錯誤 CS1029
\#error: 'text'  
  
 顯示以 [#warning](../../../csharp/language-reference/preprocessor-directives/preprocessor-error.md) 指示詞定義的錯誤文字。  
  
 下列範例示範如何建立使用者定義的錯誤：  
  
```csharp  
// CS1029.cs  
class Sample  
{  
   static void Main()  
   {  
      #error Let's give an error here   // CS1029  
   }  
}  
```
