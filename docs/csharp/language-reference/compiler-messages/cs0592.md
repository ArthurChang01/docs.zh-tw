<span data-ttu-id="0a9be-103">屬性 'attribute' 在此宣告類型上無效。</span><span class="sxs-lookup"><span data-stu-id="0a9be-103">Attribute 'attribute' is not valid on this declaration type.</span></span> <span data-ttu-id="0a9be-104">它只在 'type' 宣告上有效。</span><span class="sxs-lookup"><span data-stu-id="0a9be-104">It is valid on 'type' declarations only.</span></span>  
  
 <span data-ttu-id="0a9be-105">當您定義屬性時，您會藉由指定 `AttributeTargets` 值來定義該屬性可套用的建構。</span><span class="sxs-lookup"><span data-stu-id="0a9be-105">When you define an attribute, you define what constructs it can be applied to by specifying an `AttributeTargets` value.</span></span> <span data-ttu-id="0a9be-106">在下列範例中，`MyAttribute` 屬性只能套用至介面，因為 `AttributeUsage` 屬性指定了 `AttributeTargets.Interface`。</span><span class="sxs-lookup"><span data-stu-id="0a9be-106">In the following example, the `MyAttribute` attribute can be applied to interfaces only, because the `AttributeUsage` attribute specifies `AttributeTargets.Interface`.</span></span> <span data-ttu-id="0a9be-107">由於該屬性套用至類別 (類別 `A`)，因此會產生此錯誤。</span><span class="sxs-lookup"><span data-stu-id="0a9be-107">The error is generated because the attribute is applied to a class (class `A`).</span></span>  
  
## <a name="example"></a><span data-ttu-id="0a9be-108">範例</span><span class="sxs-lookup"><span data-stu-id="0a9be-108">Example</span></span>

 <span data-ttu-id="0a9be-109">下列範例會產生 CS0592：</span><span class="sxs-lookup"><span data-stu-id="0a9be-109">The following sample generates CS0592:</span></span>  

```csharp
// CS0592.cs  
using System;  
  
[AttributeUsage(AttributeTargets.Interface)]  
public class MyAttribute : Attribute
{  
}  
  
[MyAttribute]  
// Generates CS0592 because MyAttribute is not valid for a class.
public class A
{  
    public static void Main()  
    {  
    }  
}  
```
