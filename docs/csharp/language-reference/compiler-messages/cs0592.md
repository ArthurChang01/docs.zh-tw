<span data-ttu-id="25932-103">屬性 'attribute' 在此宣告類型上無效。</span><span class="sxs-lookup"><span data-stu-id="25932-103">Attribute 'attribute' is not valid on this declaration type.</span></span> <span data-ttu-id="25932-104">它只在 'type' 宣告上有效。</span><span class="sxs-lookup"><span data-stu-id="25932-104">It is valid on 'type' declarations only.</span></span>

## <a name="example"></a><span data-ttu-id="25932-108">範例</span><span class="sxs-lookup"><span data-stu-id="25932-108">Example</span></span>

 <span data-ttu-id="25932-109">下列範例會產生 CS0592：</span><span class="sxs-lookup"><span data-stu-id="25932-109">The following sample generates CS0592:</span></span>  

```csharp
// CS0592.cs  
using System;  
  
[AttributeUsage(AttributeTargets.Interface)]  
public class MyAttribute : Attribute
{  
}  
  
[MyAttribute]  
// Generates CS0592 because MyAttribute is not valid for a class.
public class A
{  
    public static void Main()  
    {  
    }  
}  
```
