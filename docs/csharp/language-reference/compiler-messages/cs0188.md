# <a name="compiler-error-cs0188"></a><span data-ttu-id="a7073-102">編譯器錯誤 CS0188</span><span class="sxs-lookup"><span data-stu-id="a7073-102">Compiler Error CS0188</span></span>
<span data-ttu-id="a7073-103">指定其所有欄位前，無法使用 'this' 物件</span><span class="sxs-lookup"><span data-stu-id="a7073-103">The 'this' object cannot be used before all of its fields are assigned to</span></span>  
  
 <span data-ttu-id="a7073-104">建構函式必須先指派 `struct` 中的所有欄位，才能在 `struct` 中呼叫方法。</span><span class="sxs-lookup"><span data-stu-id="a7073-104">All fields in a `struct` have to be assigned by a constructor before the constructor can call a method in the `struct`.</span></span>  
  
 <span data-ttu-id="a7073-105">如果嘗試在結構建構函式中初始化屬性時出現這個錯誤，解決方法是變更建構函式參數以指定支援欄位而非屬性本身。</span><span class="sxs-lookup"><span data-stu-id="a7073-105">If you see this error when trying to initialize a property in a struct constructor, the solution is to change the constructor parameter to specify the backing field instead of the property itself.</span></span> <span data-ttu-id="a7073-106">結構中應避免使用自動實作屬性，因為自動實作屬性沒有支援欄位，因此無法透過建構函式初始化。</span><span class="sxs-lookup"><span data-stu-id="a7073-106">Auto-implemented properties should be avoided in structs because they have no backing field and therefore cannot be initialized in any way from the constructor.</span></span>  
  
 <span data-ttu-id="a7073-107">如需詳細資訊，請參閱[使用結構](../../../csharp/programming-guide/classes-and-structs/using-structs.md)。</span><span class="sxs-lookup"><span data-stu-id="a7073-107">For more information, see [Using Structs](../../../csharp/programming-guide/classes-and-structs/using-structs.md).</span></span>  
  
## <a name="example"></a><span data-ttu-id="a7073-108">範例</span><span class="sxs-lookup"><span data-stu-id="a7073-108">Example</span></span>  
 <span data-ttu-id="a7073-109">下列範例會產生 CS0188：</span><span class="sxs-lookup"><span data-stu-id="a7073-109">The following sample generates CS0188:</span></span>  
  
```csharp  
// CS0188.cs  
// compile with: /t:library  
namespace MyNamespace  
{  
    class MyClass  
    {  
        struct S  
        {  
            public int a;  
  
            void MyMethod()  
            {  
            }  
  
            S(int i)  
            {  
                // a = i;  
                MyMethod();  // CS0188  
            }  
        }  
        public static void Main()  
        { }  
  
    }  
}  
```
