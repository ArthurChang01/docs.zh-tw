<span data-ttu-id="5ecfa-103">建立物件時不能使用不安全的類型 'type name'。</span><span class="sxs-lookup"><span data-stu-id="5ecfa-103">Unsafe type 'type name' cannot be used in object creation.</span></span>

## <a name="example"></a><span data-ttu-id="5ecfa-110">範例</span><span class="sxs-lookup"><span data-stu-id="5ecfa-110">Example</span></span>

 <span data-ttu-id="5ecfa-111">下列範例會產生 CS1919，因為指標類型是不安全的︰</span><span class="sxs-lookup"><span data-stu-id="5ecfa-111">The following example generates CS1919 because a pointer type is unsafe:</span></span>

```csharp
unsafe public class C  
{  
    public static int Main()  
    {  
        var col1 = new int* { }; // CS1919

    }  
}  
```
