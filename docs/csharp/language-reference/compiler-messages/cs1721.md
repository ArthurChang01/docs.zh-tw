# <a name="compiler-error-cs1721"></a><span data-ttu-id="513a1-102">編譯器錯誤 CS1721</span><span class="sxs-lookup"><span data-stu-id="513a1-102">Compiler Error CS1721</span></span>

<span data-ttu-id="513a1-104">此錯誤訊息的最常見原因是嘗試使用多重繼承。</span><span class="sxs-lookup"><span data-stu-id="513a1-104">The most common cause of this error message is attempting to use multiple inheritance.</span></span> <span data-ttu-id="513a1-105">C# 中的類別只能直接繼承自一個類別。</span><span class="sxs-lookup"><span data-stu-id="513a1-105">A class in C# may only inherit directly from one class.</span></span> <span data-ttu-id="513a1-106">不過，一個類別可以實作任意數目的介面。</span><span class="sxs-lookup"><span data-stu-id="513a1-106">However, a class can implement any number of interfaces.</span></span>

## <a name="example"></a><span data-ttu-id="513a1-107">範例</span><span class="sxs-lookup"><span data-stu-id="513a1-107">Example</span></span>

<span data-ttu-id="513a1-108">下面的示例顯示了生成 CS1721 的一種方法：</span><span class="sxs-lookup"><span data-stu-id="513a1-108">The following example shows one way in which CS1721 is generated:</span></span>

```csharp

```

## <a name="to-correct-this-error"></a><span data-ttu-id="513a1-109">更正這個錯誤</span><span class="sxs-lookup"><span data-stu-id="513a1-109">To correct this error</span></span>

<span data-ttu-id="513a1-110">以下是更正此錯誤的不同方法：</span><span class="sxs-lookup"><span data-stu-id="513a1-110">The following are different ways to correct this error:</span></span>

    ```

    public interface B {}
    public class MyClass : A, B {}
    ```
