# <a name="compiler-error-cs0038"></a><span data-ttu-id="3837f-102">編譯器錯誤 CS0038</span><span class="sxs-lookup"><span data-stu-id="3837f-102">Compiler Error CS0038</span></span>
<span data-ttu-id="3837f-103">無法存取外部類型 'type1' 的非靜態成員 (透過巢狀型別 'type2')</span><span class="sxs-lookup"><span data-stu-id="3837f-103">Cannot access a nonstatic member of outer type 'type1' via nested type 'type2'</span></span>  
  
 <span data-ttu-id="3837f-104">類別中的欄位不會自動提供給巢狀類別。</span><span class="sxs-lookup"><span data-stu-id="3837f-104">A field in a class is not automatically available to a nested class.</span></span> <span data-ttu-id="3837f-105">若要提供給巢狀類別，此欄位必須是[靜態](../keywords/static.md)。</span><span class="sxs-lookup"><span data-stu-id="3837f-105">To be available to a nested class, the field must be [static](../keywords/static.md).</span></span> <span data-ttu-id="3837f-106">否則，您必須建立外部類別的執行個體。</span><span class="sxs-lookup"><span data-stu-id="3837f-106">Otherwise, you must create an instance of the outer class.</span></span> <span data-ttu-id="3837f-107">如需詳細資訊，請參閱[巢狀型別](../../programming-guide/classes-and-structs/nested-types.md)。</span><span class="sxs-lookup"><span data-stu-id="3837f-107">For more information, see [Nested Types](../../programming-guide/classes-and-structs/nested-types.md).</span></span>  
  
 <span data-ttu-id="3837f-108">下列範例會產生 CS0038：</span><span class="sxs-lookup"><span data-stu-id="3837f-108">The following sample generates CS0038:</span></span>  
  
```csharp  
// CS0038.cs  
class OuterClass  
{  
   public int count;  
   // try the following line instead  
   // public static int count;  
  
   class InnerClass  
   {  
      void func()  
      {  
         // or, create an instance  
         // OuterClass class_inst = new OuterClass();  
         // int count2 = class_inst.count;  
         int count2 = count;   // CS0038  
      }  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
