<span data-ttu-id="22433-103">'variable' 屬於類型 'type'。</span><span class="sxs-lookup"><span data-stu-id="22433-103">'variable' is of type 'type'.</span></span> <span data-ttu-id="22433-104">屬於非字串之參考型別的 const 欄位，只能以 null 初始化。</span><span class="sxs-lookup"><span data-stu-id="22433-104">A const field of a reference type other than string can only be initialized with null.</span></span>  
  
 <span data-ttu-id="22433-105">常數運算式是可在編譯時間完整評估的運算式。</span><span class="sxs-lookup"><span data-stu-id="22433-105">A constant-expression is an expression that can be fully evaluated at compile-time.</span></span> <span data-ttu-id="22433-106">因為套用新運算子是建立參考型別之非 null 值的唯一方法，且常數運算式中不允許使用新的運算子，所以字串以外之參考型別的常數，唯一可能的值就是 null。</span><span class="sxs-lookup"><span data-stu-id="22433-106">Because the only way to create a non-null value of a reference-type is to apply the new operator, and because the new operator is not permitted in a constant-expression, the only possible value for constants of reference-types other than string is null.</span></span>
