# <a name="nologo-c-compiler-options"></a>/nologo (C# 編譯器選項)
**/nologo** 選項會隱藏編譯器啟動時的登入橫幅，以及編譯期間的資訊訊息。  
  
## <a name="syntax"></a>語法  
  
```console  
/nologo  
```  
  
## <a name="remarks"></a>備註  
 這個選項不適用於開發環境；只有在從命令列編譯時才可用。  
  
 Visual Studio 不提供這個編譯器選項，您亦無法以程式設計方式變更。
