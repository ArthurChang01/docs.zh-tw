# <a name="-debug-c-compiler-options"></a><span data-ttu-id="a9613-102">-debug (C# 編譯器選項)</span><span class="sxs-lookup"><span data-stu-id="a9613-102">-debug (C# Compiler Options)</span></span>
<span data-ttu-id="a9613-103">**-debug** 選項可讓編譯器產生偵錯資訊，並將它放在一或多個輸出檔案中。</span><span class="sxs-lookup"><span data-stu-id="a9613-103">The **-debug** option causes the compiler to generate debugging information and place it in the output file or files.</span></span>  
  
## <a name="syntax"></a><span data-ttu-id="a9613-104">語法</span><span class="sxs-lookup"><span data-stu-id="a9613-104">Syntax</span></span>  
  
```console  
-debug[+ | -]  
-debug:{full | pdbonly}  
```  
  
## <a name="arguments"></a><span data-ttu-id="a9613-105">引數</span><span class="sxs-lookup"><span data-stu-id="a9613-105">Arguments</span></span>  
 `+` <span data-ttu-id="a9613-106">&#124;</span><span class="sxs-lookup"><span data-stu-id="a9613-106">&#124;</span></span> `-`  
 <span data-ttu-id="a9613-107">指定 `+` 或 **-debug** 會讓編譯器產生偵錯資訊，並將其放在程式資料庫 (.pdb 檔案) 中。</span><span class="sxs-lookup"><span data-stu-id="a9613-107">Specifying `+`, or just **-debug**, causes the compiler to generate debugging information and place it in a program database (.pdb file).</span></span> <span data-ttu-id="a9613-108">指定 `-` (當您未指定 **-debug** 時，它就會生效) 將不會建立任何偵錯資訊。</span><span class="sxs-lookup"><span data-stu-id="a9613-108">Specifying `-`, which is in effect if you do not specify **-debug**, causes no debug information to be created.</span></span>  
  
 `full` <span data-ttu-id="a9613-109">&#124;</span><span class="sxs-lookup"><span data-stu-id="a9613-109">&#124;</span></span> `pdbonly`  
 <span data-ttu-id="a9613-110">指定編譯器所產生的偵錯資訊類型。</span><span class="sxs-lookup"><span data-stu-id="a9613-110">Specifies the type of debugging information generated by the compiler.</span></span> <span data-ttu-id="a9613-111">完整引數 (當您未指定 **-debug:pdbonly** 時，它就會生效) 允許將偵錯工具附加至執行中的程式。</span><span class="sxs-lookup"><span data-stu-id="a9613-111">The full argument, which is in effect if you do not specify **-debug:pdbonly**, enables attaching a debugger to the running program.</span></span> <span data-ttu-id="a9613-112">指定 pdbonly 讓原始程式碼在偵錯工具中啟動程式時進行偵錯，但只有在將執行中的程式附加到偵錯工具時，才會顯示組譯工具。</span><span class="sxs-lookup"><span data-stu-id="a9613-112">Specifying pdbonly allows source code debugging when the program is started in the debugger but will only display assembler when the running program is attached to the debugger.</span></span>
