# <a name="c-language-specification"></a>C# 語言規格
C# 語言規格是 C# 語法和用法的限定來源。 此規格包含語言各方面的詳細資訊，包括 Visual C# 文件未涵蓋的許多重點。

您可以從 [Microsoft 下載中心](http://www.microsoft.com/download/details.aspx?id=7029)下載此規格的 5.0 版。 如果您已安裝 Visual Studio 2015，也可以在電腦的 Program Files (x86)/Microsoft Visual Studio 14.0/VC#/Specifications/1033 資料夾中找到此規格。 如果您安裝的是其他版本的 Visual Studio，或安裝的 Visual Studio 不是英文版，請視實際情況變更路徑。

目前尚未提供 6.0 版本的規格，但 [dotnet/csharplang GitHub 存放庫](https://github.com/dotnet/csharplang/blob/master/spec/README.md)中有非正式的草稿。
