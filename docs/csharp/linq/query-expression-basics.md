## <a name="what-is-a-query-expression"></a><span data-ttu-id="8cdf9-136">什麼是查詢運算式？</span><span class="sxs-lookup"><span data-stu-id="8cdf9-136">What is a query expression?</span></span>  

 <span data-ttu-id="8cdf9-137">「查詢運算式」**是以查詢語法表示的查詢。</span><span class="sxs-lookup"><span data-stu-id="8cdf9-137">A *query expression* is a query expressed in query syntax.</span></span> <span data-ttu-id="8cdf9-138">查詢運算式是第一類語言建構。</span><span class="sxs-lookup"><span data-stu-id="8cdf9-138">A query expression is a first-class language construct.</span></span> <span data-ttu-id="8cdf9-139">它就像任何其他運算式一樣，可以用於 C# 運算式有效的任何內容。</span><span class="sxs-lookup"><span data-stu-id="8cdf9-139">It is just like any other expression and can be used in any context in which a C# expression is valid.</span></span> <span data-ttu-id="8cdf9-140">查詢運算式包含以 SQL 或 XQuery 類似的宣告式語法所撰寫的一組子句。</span><span class="sxs-lookup"><span data-stu-id="8cdf9-140">A query expression consists of a set of clauses written in a declarative syntax similar to SQL or XQuery.</span></span> <span data-ttu-id="8cdf9-141">每個子句接著會包含一個或多個 C# 運算式，而且這些運算式本身可能是查詢運算式或包含查詢運算式。</span><span class="sxs-lookup"><span data-stu-id="8cdf9-141">Each clause in turn contains one or more C# expressions, and these expressions may themselves be either a query expression or contain a query expression.</span></span>  
  
 <span data-ttu-id="8cdf9-142">查詢運算式的開頭必須是 [from](../language-reference/keywords/from-clause.md) 子句，結尾則必須是 [select](../language-reference/keywords/select-clause.md) 或 [group](../language-reference/keywords/group-clause.md) 子句。</span><span class="sxs-lookup"><span data-stu-id="8cdf9-142">A query expression must begin with a [from](../language-reference/keywords/from-clause.md) clause and must end with a [select](../language-reference/keywords/select-clause.md) or [group](../language-reference/keywords/group-clause.md) clause.</span></span> <span data-ttu-id="8cdf9-143">在第一個 `from` 子句與最後一個 `select` 或 `group` 子句之間，它可以包含下列其中一個或多個選擇性子句︰[where](../language-reference/keywords/where-clause.md)、[orderby](../language-reference/keywords/orderby-clause.md)、[join](../language-reference/keywords/join-clause.md)、[let](../language-reference/keywords/let-clause.md)，甚至是額外的 [from](../language-reference/keywords/from-clause.md) 子句。</span><span class="sxs-lookup"><span data-stu-id="8cdf9-143">Between the first `from` clause and the last `select` or `group` clause, it can contain one or more of these optional clauses: [where](../language-reference/keywords/where-clause.md), [orderby](../language-reference/keywords/orderby-clause.md), [join](../language-reference/keywords/join-clause.md), [let](../language-reference/keywords/let-clause.md) and even additional [from](../language-reference/keywords/from-clause.md) clauses.</span></span> <span data-ttu-id="8cdf9-144">您也可以使用 [into](../language-reference/keywords/into.md) 關鍵字，讓 `join` 或 `group` 子句的結果作為相同查詢運算式中其他查詢子句的來源。</span><span class="sxs-lookup"><span data-stu-id="8cdf9-144">You can also use the [into](../language-reference/keywords/into.md) keyword to enable the result of a `join` or `group` clause to serve as the source for additional query clauses in the same query expression.</span></span>
