# <a name="compiler-error-cs0695"></a><span data-ttu-id="48512-102">編譯器錯誤 CS0695</span><span class="sxs-lookup"><span data-stu-id="48512-102">Compiler Error CS0695</span></span>
<span data-ttu-id="48512-103">'generic type' 不能同時實作 'generic interface' 和 'generic interface'，因為它們可能針對某些類型參數的替代進行整合</span><span class="sxs-lookup"><span data-stu-id="48512-103">'generic type' cannot implement both 'generic interface' and 'generic interface' because they may unify for some type parameter substitutions</span></span>  
  
 <span data-ttu-id="48512-104">如果泛型類別實作相同泛型介面的多個參數化，而且有將兩個介面設為相同的類型參數替換，則會發生這個錯誤。</span><span class="sxs-lookup"><span data-stu-id="48512-104">This error occurs when a generic class implements more than one parameterization of the same generic interface, and there exists a type parameter substitution which would make the two interfaces identical.</span></span> <span data-ttu-id="48512-105">若要避免這個錯誤，請僅實作其中一個介面，或變更類型參數來避免衝突。</span><span class="sxs-lookup"><span data-stu-id="48512-105">To avoid this error, implement only one of the interfaces, or change the type parameters to avoid the conflict.</span></span>  
  
 <span data-ttu-id="48512-106">下列範例會產生 CS0695：</span><span class="sxs-lookup"><span data-stu-id="48512-106">The following sample generates CS0695:</span></span>  
  
```csharp  
// CS0695.cs  
// compile with: /target:library  
  
interface I<T>  
{  
}  
  
class G<T1, T2> : I<T1>, I<T2>  // CS0695  
{  
}  
```
