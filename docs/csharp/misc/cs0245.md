# <a name="compiler-error-cs0245"></a>編譯器錯誤 CS0245
不能直接呼叫解構函式和 object.Finalize。 請考慮呼叫 IDisposable.Dispose (如果有)。  
  
 如需詳細資訊，請參閱 <<c0> [ 進行記憶體回收的程式設計基本](../../standard/garbage-collection/index.md)並[解構函式](../../csharp/programming-guide/classes-and-structs/destructors.md)。  
  
 下列範例會產生 CS0245：  
  
```csharp  
// CS0245.cs  
using System;  
using System.Collections;  
  
class MyClass // : IDisposable  
{  
   /*  
   public void Dispose()  
   {  
      // cleanup code goes here  
   }  
   */  
  
   void m()  
   {  
      this.Finalize();   // CS0245  
      // this.Dispose();  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
