# <a name="compiler-error-cs1526"></a><span data-ttu-id="cbfdd-102">編譯器錯誤 CS1526</span><span class="sxs-lookup"><span data-stu-id="cbfdd-102">Compiler Error CS1526</span></span>

 <span data-ttu-id="cbfdd-104">未正確指定用來動態配置物件記憶體的 [new](../../csharp/language-reference/keywords/new.md) 運算子。</span><span class="sxs-lookup"><span data-stu-id="cbfdd-104">The [new](../../csharp/language-reference/keywords/new.md) operator, used to dynamically allocate memory for an object, was not specified correctly.</span></span>  
  
## <a name="example"></a><span data-ttu-id="cbfdd-105">範例</span><span class="sxs-lookup"><span data-stu-id="cbfdd-105">Example</span></span>  
 <span data-ttu-id="cbfdd-106">下列範例會示範如何使用 `new` ，為陣列和物件配置空間。</span><span class="sxs-lookup"><span data-stu-id="cbfdd-106">The following sample shows how to use `new` to allocate space for an array and an object.</span></span>  
  
```  
// CS1526.cs  
public class y  
{  
   public static int i = 0;  
   public int myi = 0;  
}  
  
public class z  
{  
   public static void Main()  
   {  
      y py = new y;   // CS1526  
      y[] aoys = new y[10];   // Array of Ys  
  
      for (int i = 0; i < aoys.Length; i++)  
         aoys[i] = new y();   // an object of type y  
   }  
}  
```
