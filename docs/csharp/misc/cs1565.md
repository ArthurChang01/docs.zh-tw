# <a name="compiler-error-cs1565"></a>編譯器錯誤 CS1565
指定的選項衝突: Win32 資源檔; Win32 圖示  
  
 在相同編譯中同時指定 [/win32res](../../csharp/language-reference/compiler-options/win32res-compiler-option.md) 編譯器選項和 [/win32ico](../../csharp/language-reference/compiler-options/win32icon-compiler-option.md) 編譯器選項是無效的。
