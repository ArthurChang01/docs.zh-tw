<span data-ttu-id="23633-103">固定大小緩衝區欄位在欄位名稱後面必須有陣列大小規範</span><span class="sxs-lookup"><span data-stu-id="23633-103">A fixed size buffer field must have the array size specifier after the field name</span></span>  
  
 <span data-ttu-id="23633-104">不同於標準陣列，固定大小的緩衝區必須在宣告點指定常數大小。</span><span class="sxs-lookup"><span data-stu-id="23633-104">Unlike regular arrays, fixed size buffers require a constant size to be specified at the declaration point.</span></span> <span data-ttu-id="23633-105">若要解決這個錯誤，請加入一個正整數常值或常數正整數，並在識別項之後放置方括弧。</span><span class="sxs-lookup"><span data-stu-id="23633-105">To resolve this error, add a positive integer literal or a constant positive integer and put the square brackets after the identifier.</span></span>  
  
 <span data-ttu-id="23633-106">下列範例會產生 CS1641：</span><span class="sxs-lookup"><span data-stu-id="23633-106">The following sample generates CS1641:</span></span>  
  
```  
// CS1641.cs  
// compile with: /unsafe /target:library  
unsafe struct S {  
   fixed int [] a;  // CS1641  
  
   // OK  
   fixed int b [10];  
   const int c = 10;  
   fixed int d [c];  
}  
```
