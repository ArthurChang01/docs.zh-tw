# <a name="compiler-error-cs1555"></a>編譯器錯誤 CS1555
找不到為 Main 方法所指定的 'class'  
  
 已將類別指定給[-main](../language-reference/compiler-options/main-compiler-option.md)編譯器選項，但在原始程式碼中找不到類別名稱。
