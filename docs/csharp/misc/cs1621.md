# <a name="compiler-error-cs1621"></a><span data-ttu-id="8a068-102">編譯器錯誤 CS1621</span><span class="sxs-lookup"><span data-stu-id="8a068-102">Compiler Error CS1621</span></span>
<span data-ttu-id="8a068-103">在匿名方法或 Lambda 運算式內不可使用 yield 陳述式</span><span class="sxs-lookup"><span data-stu-id="8a068-103">The yield statement cannot be used inside an anonymous method or lambda expression</span></span>  
  
 <span data-ttu-id="8a068-104">yield 陳述式不能在迭代器的匿名方法區塊中。</span><span class="sxs-lookup"><span data-stu-id="8a068-104">The yield statement cannot be in an anonymous method block in an iterator.</span></span>  
  
## <a name="example"></a><span data-ttu-id="8a068-105">範例</span><span class="sxs-lookup"><span data-stu-id="8a068-105">Example</span></span>  
 <span data-ttu-id="8a068-106">下列範例會產生 CS1621：</span><span class="sxs-lookup"><span data-stu-id="8a068-106">The following sample generates CS1621:</span></span>  
  
```csharp  
// CS1621.cs  
  
using System.Collections;  
  
delegate object MyDelegate();  
  
class C : IEnumerable  
{  
    public IEnumerator GetEnumerator()  
    {  
        MyDelegate d = delegate  
        {  
            yield return this; // CS1621  
            return this;  
        };  
        d();  
        // Try this instead:  
        // MyDelegate d = delegate { return this; };  
        // yield return d();  
    }  
  
    public static void Main()  
    {  
    }  
}  
```
