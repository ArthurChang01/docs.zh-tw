# <a name="compiler-error-cs1621"></a><span data-ttu-id="77183-102">編譯器錯誤 CS1621</span><span class="sxs-lookup"><span data-stu-id="77183-102">Compiler Error CS1621</span></span>
<span data-ttu-id="77183-103">在匿名方法或 Lambda 運算式內不可使用 yield 陳述式</span><span class="sxs-lookup"><span data-stu-id="77183-103">The yield statement cannot be used inside an anonymous method or lambda expression</span></span>  
  
<span data-ttu-id="77183-104">您不能在[匿名方法](../language-reference/operators/delegate-operator.md)或[lambda 運算式](../programming-guide/statements-expressions-operators/lambda-expressions.md)中使用[yield](../language-reference/keywords/yield.md)語句。</span><span class="sxs-lookup"><span data-stu-id="77183-104">You cannot use the [yield](../language-reference/keywords/yield.md) statement in an [anonymous method](../language-reference/operators/delegate-operator.md) or a [lambda expression](../programming-guide/statements-expressions-operators/lambda-expressions.md).</span></span>
  
<span data-ttu-id="77183-105">下列範例會產生 CS1621：</span><span class="sxs-lookup"><span data-stu-id="77183-105">The following sample generates CS1621:</span></span>
  
```csharp  
// CS1621.cs  
  
using System.Collections;  
  
delegate object MyDelegate();  
  
class C : IEnumerable  
{  
    public IEnumerator GetEnumerator()  
    {  
        MyDelegate d = delegate  
        {  
            yield return this; // CS1621  
            return this;  
        };  
        d();  
        // Try this instead:  
        // MyDelegate d = delegate { return this; };  
        // yield return d();  
    }  
  
    public static void Main()  
    {  
    }  
}  
```
