 <span data-ttu-id="33557-104">[abstract](../../csharp/language-reference/keywords/abstract.md) 方法不能包含其實作。</span><span class="sxs-lookup"><span data-stu-id="33557-104">An [abstract](../../csharp/language-reference/keywords/abstract.md) method cannot contain its implementation.</span></span>  
  
 <span data-ttu-id="33557-105">下列範例會產生 CS0500：</span><span class="sxs-lookup"><span data-stu-id="33557-105">The following sample generates CS0500:</span></span>  
  
```csharp  
// CS0500.cs  
namespace x  
{  
   abstract public class clx  
   {  
      abstract public void f(){}   // CS0500  
      // try the following line instead  
      // abstract public void f();  
   }  
  
   public class cly  
   {  
      public static int Main()  
      {  
         return 0;  
      }  
   }  
}  
```
