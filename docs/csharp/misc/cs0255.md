<span data-ttu-id="ca91f-103">在 catch 或 finally 區塊中不可使用 stackalloc</span><span class="sxs-lookup"><span data-stu-id="ca91f-103">stackalloc may not be used in a catch or finally block</span></span>  
  
 <span data-ttu-id="ca91f-104">在 [catch](../../csharp/language-reference/keywords/stackalloc.md) 或 [finally](../../csharp/language-reference/keywords/try-catch.md) 區塊中不可使用 [stackalloc](../../csharp/language-reference/keywords/try-catch-finally.md) 關鍵字。</span><span class="sxs-lookup"><span data-stu-id="ca91f-104">The [stackalloc](../../csharp/language-reference/keywords/stackalloc.md) keyword may not be used in a [catch](../../csharp/language-reference/keywords/try-catch.md) or [finally](../../csharp/language-reference/keywords/try-catch-finally.md) block.</span></span> <span data-ttu-id="ca91f-105">如需詳細資訊，請參閱[例外狀況和例外狀況處理](../../csharp/programming-guide/exceptions/index.md)。</span><span class="sxs-lookup"><span data-stu-id="ca91f-105">For more information, see [Exceptions and Exception Handling](../../csharp/programming-guide/exceptions/index.md).</span></span>  
  
 <span data-ttu-id="ca91f-106">下列範例會產生 CS0255：</span><span class="sxs-lookup"><span data-stu-id="ca91f-106">The following sample generates CS0255:</span></span>  
  
```csharp  
// CS0255.cs  
// compile with: /unsafe  
using System;  
  
public class TestTryFinally  
{  
   public static unsafe void Test()  
   {  
      int i = 123;  
      string s = "Some string";  
      object o = s;  
  
      try  
      {  
         // Conversion is not valid; o contains a string not an int  
         i = (int) o;  
      }  
  
      finally  
      {  
         Console.Write("i = {0}", i);  
         int* fib = stackalloc int[100];   // CS0255  
      }  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
