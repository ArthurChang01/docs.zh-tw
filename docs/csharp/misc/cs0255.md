# <a name="compiler-error-cs0255"></a><span data-ttu-id="00a7f-102">編譯器錯誤 CS0255</span><span class="sxs-lookup"><span data-stu-id="00a7f-102">Compiler Error CS0255</span></span>
<span data-ttu-id="00a7f-103">在 catch 或 finally 區塊中不可使用 stackalloc</span><span class="sxs-lookup"><span data-stu-id="00a7f-103">stackalloc may not be used in a catch or finally block</span></span>  
  
 <span data-ttu-id="00a7f-104">[Stackalloc](../../csharp/language-reference/keywords/stackalloc.md)中不能使用關鍵字[攔截](../../csharp/language-reference/keywords/try-catch.md)或[最後](../../csharp/language-reference/keywords/try-catch-finally.md)區塊。</span><span class="sxs-lookup"><span data-stu-id="00a7f-104">The [stackalloc](../../csharp/language-reference/keywords/stackalloc.md) keyword may not be used in a [catch](../../csharp/language-reference/keywords/try-catch.md) or [finally](../../csharp/language-reference/keywords/try-catch-finally.md) block.</span></span> <span data-ttu-id="00a7f-105">如需詳細資訊，請參閱[例外狀況和例外狀況處理](../../csharp/programming-guide/exceptions/index.md)。</span><span class="sxs-lookup"><span data-stu-id="00a7f-105">For more information, see [Exceptions and Exception Handling](../../csharp/programming-guide/exceptions/index.md).</span></span>  
  
 <span data-ttu-id="00a7f-106">下列範例會產生 CS0255：</span><span class="sxs-lookup"><span data-stu-id="00a7f-106">The following sample generates CS0255:</span></span>  
  
```  
// CS0255.cs  
// compile with: /unsafe  
using System;  
  
public class TestTryFinally  
{  
   public static unsafe void Test()  
   {  
      int i = 123;  
      string s = "Some string";  
      object o = s;  
  
      try  
      {  
         // Conversion is not valid; o contains a string not an int  
         i = (int) o;  
      }  
  
      finally  
      {  
         Console.Write("i = {0}", i);  
         int* fib = stackalloc int[100];   // CS0255  
      }  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
