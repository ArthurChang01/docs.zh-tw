<span data-ttu-id="c0bac-104">您無法使用[stackalloc 運算子](../language-reference/operators/stackalloc.md)中[攔截](../language-reference/keywords/try-catch.md)或是[最後](../language-reference/keywords/try-catch-finally.md)區塊。</span><span class="sxs-lookup"><span data-stu-id="c0bac-104">You cannot use the [stackalloc operator](../language-reference/operators/stackalloc.md) in a [catch](../language-reference/keywords/try-catch.md) or [finally](../language-reference/keywords/try-catch-finally.md) block.</span></span> <span data-ttu-id="c0bac-105">如需詳細資訊，請參閱 <<c0> [ 例外狀況和例外狀況處理](../programming-guide/exceptions/index.md)。</span><span class="sxs-lookup"><span data-stu-id="c0bac-105">For more information, see [Exceptions and Exception Handling](../programming-guide/exceptions/index.md).</span></span>  
  
<span data-ttu-id="c0bac-106">下列範例會產生 CS0255：</span><span class="sxs-lookup"><span data-stu-id="c0bac-106">The following sample generates CS0255:</span></span>  
  
```csharp  
// CS0255.cs  
// compile with: /unsafe  
using System;  
  
public class TestTryFinally  
{  
   public static unsafe void Test()  
   {  
      int i = 123;  
      string s = "Some string";  
      object o = s;  
  
      try  
      {  
         // Conversion is not valid; o contains a string not an int  
         i = (int) o;  
      }  
      finally  
      {  
         Console.Write("i = {0}", i);  
         int* fib = stackalloc int[100];   // CS0255  
      }  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
