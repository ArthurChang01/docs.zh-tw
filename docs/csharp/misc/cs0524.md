# <a name="compiler-error-cs0524"></a>編譯器錯誤 CS0524
'type': 介面不能宣告類型  
  
 [interface](../../csharp/language-reference/keywords/interface.md) 不能包含使用者定義的類型；它應該只包含方法和屬性。  
  
## <a name="example"></a>範例  
 下列範例會產生 CS0524：  
  
```csharp  
// CS0524.cs  
public interface Clx  
{  
    public class Cly   // CS0524, delete user-defined type  
    {  
    }  
}  
```
