# <a name="compiler-error-cs1588"></a>編譯器錯誤 CS1588
無法決定 Common Language Runtime 目錄 -- '原因'  
  
 編譯器無法擷取 .NET Framework Common Language Runtime 安裝所在的目錄。 這個錯誤表示未正確安裝 Common Language Runtime。
