# <a name="compiler-warning-level-4-cs0402"></a>編譯器警告 (層級 4) CS0402
'identifier' : 進入點不能是泛型，也不能是泛型類型  
  
 在泛型類型中找到進入點。 若要移除這項警告，請實作非泛型類別或結構中的 Main。  
  
```csharp  
// CS0402.cs  
// compile with: /W:4  
class C<T>  
{  
   public static void Main()  // CS0402  
   {  
  
   }  
}  
  
class CMain  
{  
   public static void Main() {}  
}  
```
