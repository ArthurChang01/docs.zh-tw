# <a name="compiler-error-cs0588"></a>編譯器錯誤 CS0588
編譯器內部錯誤: 階段 'LEX'  
  
 請嘗試判斷編譯器失敗的原因是否為無法剖析未預期的語法。 如果情況並非如此，再次嘗試[告訴我們](https://docs.microsoft.com/visualstudio/ide/talk-to-us)。
