<span data-ttu-id="a4bf3-103">成員 'name' 具有參數修飾詞 'this'，此參數修飾詞不在第一個參數上。</span><span class="sxs-lookup"><span data-stu-id="a4bf3-103">Method 'name' has a parameter modifier 'this' which is not on the first parameter.</span></span>  
  
 <span data-ttu-id="a4bf3-104">`this` 修飾詞只允許出現在方法的第一個參數，它會通知編譯器該方法是一個擴充方法。</span><span class="sxs-lookup"><span data-stu-id="a4bf3-104">The `this` modifier is allowed only on the first parameter of a method, which indicates to the compiler that the method is an extension method.</span></span>
