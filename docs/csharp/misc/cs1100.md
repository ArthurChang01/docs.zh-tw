# <a name="compiler-error-cs1100"></a><span data-ttu-id="717f6-102">編譯器錯誤 CS1100</span><span class="sxs-lookup"><span data-stu-id="717f6-102">Compiler Error CS1100</span></span>
<span data-ttu-id="717f6-103">成員 'name' 具有參數修飾詞 'this'，此參數修飾詞不在第一個參數上。</span><span class="sxs-lookup"><span data-stu-id="717f6-103">Method 'name' has a parameter modifier 'this' which is not on the first parameter.</span></span>  
  
 <span data-ttu-id="717f6-104">`this` 修飾詞只允許出現在方法的第一個參數，它會通知編譯器該方法是一個擴充方法。</span><span class="sxs-lookup"><span data-stu-id="717f6-104">The `this` modifier is allowed only on the first parameter of a method, which indicates to the compiler that the method is an extension method.</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="717f6-105">更正這個錯誤</span><span class="sxs-lookup"><span data-stu-id="717f6-105">To correct this error</span></span>  
  
1. <span data-ttu-id="717f6-106">請從所有方法的第一個參數以外的所有參數移除 `this` 修飾詞。</span><span class="sxs-lookup"><span data-stu-id="717f6-106">Remove the `this` modifier from all except the first parameter of the method.</span></span>

 <span data-ttu-id="717f6-108">下列程式碼會產生 CS1100，因為 `this` 參數正在修改第二個參數：</span><span class="sxs-lookup"><span data-stu-id="717f6-108">The following code generates CS1100 because a `this` parameter is modifying the second parameter:</span></span>

    }
