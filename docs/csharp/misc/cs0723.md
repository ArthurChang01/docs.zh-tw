# <a name="compiler-error-cs0723"></a>編譯器錯誤 CS0723
無法宣告靜態類型 'type' 的變數  
  
 無法建立靜態類型的執行個體。  
  
 下列範例會產生 CS0723：  
  
```csharp  
// CS0723.cs  
public static class SC  
{  
}  
  
public class CMain  
{  
   public static void Main()  
   {  
      SC sc = null;  // CS0723  
   }  
}  
```
