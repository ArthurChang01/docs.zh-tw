## <a name="example"></a><span data-ttu-id="d1419-107">範例</span><span class="sxs-lookup"><span data-stu-id="d1419-107">Example</span></span>

```  
public interface I  
{  
    [CLSCompliant(false)]  
    int M();   // CS3010

{  
    public int M()  
    {  
        return 1;  
    }

    {  
    }  
}  
```
