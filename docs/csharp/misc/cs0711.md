 <span data-ttu-id="510ac-104">靜態類別無法具現化，因此不需要建構函式或解構函式。</span><span class="sxs-lookup"><span data-stu-id="510ac-104">A static class cannot be instantiated, hence it has no need for constructors or destructors.</span></span> <span data-ttu-id="510ac-105">若要避免這個錯誤，請移除靜態類別中的所有解構函式，或如果您真的想要建構並破壞執行個體，請將類別變成非靜態。</span><span class="sxs-lookup"><span data-stu-id="510ac-105">To avoid this error, remove any destructors from static classes, or, if you really want to construct and destroy instances, make the class non-static.</span></span>
