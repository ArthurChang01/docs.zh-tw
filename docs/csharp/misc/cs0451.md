# <a name="compiler-error-cs0451"></a><span data-ttu-id="831a8-102">編譯器錯誤 CS0451</span><span class="sxs-lookup"><span data-stu-id="831a8-102">Compiler Error CS0451</span></span>
<span data-ttu-id="831a8-103">new()' 條件約束不能和 'struct' 條件約束一起使用</span><span class="sxs-lookup"><span data-stu-id="831a8-103">The 'new()' constraint cannot be used with the 'struct' constraint</span></span>  
  
 <span data-ttu-id="831a8-104">指定泛型類型的條件約束時， `new()` 條件約束只能與類別類型條件約束、介面類型條件約束、參考類型條件約束和類型參數條件約束一起使用，而不是與實值類型條件約束一起使用。</span><span class="sxs-lookup"><span data-stu-id="831a8-104">When specifying constraints on the type of a generic, the `new()` constraint may only be used with class type constraints, interface type constraints, reference type constraints, and type parameter constraints, but not with value type constraints.</span></span>  
  
## <a name="example"></a><span data-ttu-id="831a8-105">範例</span><span class="sxs-lookup"><span data-stu-id="831a8-105">Example</span></span>  
 <span data-ttu-id="831a8-106">下列範例會產生 CS0451：</span><span class="sxs-lookup"><span data-stu-id="831a8-106">The following sample generates CS0451.</span></span>  
  
```csharp  
// CS0451.cs  
using System;  
public class C4
{  
   public void F4<T>() where T : struct, new() {}   // CS0451  
}  
  
// OK  
public class C5  
{  
   public void F5<T>() where T : struct {}  
}  
  
public class C6  
{  
   public void F6<T>() where T : new() {}  
}  
```
