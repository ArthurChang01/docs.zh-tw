# <a name="compiler-error-cs1103"></a>編譯器錯誤 CS1103
擴充方法的第一個參數不可為類型 'type'。  
  
 擴充方法的第一個參數不可為指標類型。  
  
## <a name="example"></a>範例  
 下列範例會產生 CS1103：  
  
```csharp  
// cs1103.cs  
public static class Extensions  
{  
    public unsafe static char* Test(this char* charP) { return charP; } // CS1103  
}   
```
