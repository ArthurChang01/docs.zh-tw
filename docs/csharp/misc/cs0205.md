 <span data-ttu-id="333c4-106">下列範例會產生 CS0205：</span><span class="sxs-lookup"><span data-stu-id="333c4-106">The following sample generates CS0205:</span></span>  
  
```csharp  
// CS0205.cs  
abstract public class MyClass

   abstract public void M();  
}

{  
   public override void M()  
   {  
      base.M();   // CS0205, delete this line  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
