 <span data-ttu-id="9c3e3-104">您無法呼叫 [abstract](../../csharp/language-reference/keywords/abstract.md) 方法，因為它沒有方法主體。</span><span class="sxs-lookup"><span data-stu-id="9c3e3-104">You cannot call an [abstract](../../csharp/language-reference/keywords/abstract.md) method because it does not have a method body.</span></span> <span data-ttu-id="9c3e3-105">如需詳細資訊，請參閱[抽象和密封類別以及類別成員](../../csharp/programming-guide/classes-and-structs/abstract-and-sealed-classes-and-class-members.md)。</span><span class="sxs-lookup"><span data-stu-id="9c3e3-105">For more information, see [Abstract and Sealed Classes and Class Members](../../csharp/programming-guide/classes-and-structs/abstract-and-sealed-classes-and-class-members.md).</span></span>  
  
 <span data-ttu-id="9c3e3-106">下列範例會產生 CS0205：</span><span class="sxs-lookup"><span data-stu-id="9c3e3-106">The following sample generates CS0205:</span></span>  
  
```csharp  
// CS0205.cs  
abstract public class MyClass  
{  
   abstract public void M();  
}  
  
public class MyClass2 : MyClass  
{  
   public override void M()  
   {  
      base.M();   // CS0205, delete this line  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
