# <a name="compiler-error-cs1513"></a>編譯器錯誤 CS1513
必須是 }  
  
 編譯器必須有右大括弧 (`}`)，但找不到。  
  
 下列範例會產生 CS1513：  
  
```  
// CS1513  
namespace y   // CS1513, no close curly brace  
{  
   class x  
   {  
      public static void Main()  
      {  
      }  
   }  
```
