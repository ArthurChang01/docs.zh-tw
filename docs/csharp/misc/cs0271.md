<span data-ttu-id="d07c7-103">這個內容中不能使用屬性或索引子 'property/indexer'，因為無法存取 get 存取子</span><span class="sxs-lookup"><span data-stu-id="d07c7-103">The property or indexer 'property/indexer' cannot be used in this context because the get accessor is inaccessible</span></span>  
  
 <span data-ttu-id="d07c7-104">當您嘗試存取無法存取的 `get` 存取子時，就會發生這個錯誤。</span><span class="sxs-lookup"><span data-stu-id="d07c7-104">This error occurs when you try to access an inaccessible `get` accessor.</span></span> <span data-ttu-id="d07c7-105">若要解決此錯誤，請增加存取子的存取範圍，或變更呼叫的位置。</span><span class="sxs-lookup"><span data-stu-id="d07c7-105">To resolve this error, increase the accessibility of the accessor, or change the calling location.</span></span> <span data-ttu-id="d07c7-106">如需詳細資訊，請參閱[存取子的存取範圍](../../csharp/programming-guide/classes-and-structs/restricting-accessor-accessibility.md)和[屬性](../../csharp/programming-guide/classes-and-structs/properties.md)。</span><span class="sxs-lookup"><span data-stu-id="d07c7-106">For more information, see [Accessor Accessibility](../../csharp/programming-guide/classes-and-structs/restricting-accessor-accessibility.md) and [Properties](../../csharp/programming-guide/classes-and-structs/properties.md).</span></span>  
  
 <span data-ttu-id="d07c7-107">下列範例會產生 CS0271：</span><span class="sxs-lookup"><span data-stu-id="d07c7-107">The following example generates CS0271:</span></span>  
  
```  
// CS0271.cs  
public class MyClass  
{  
   public int Property  
   {  
      private get { return 0; }  
      set { }  
   }  
  
   public int Property2  
   {  
      get { return 0; }  
      set { }  
   }  
}  
  
public class Test  
{  
   public static void Main(string[] args)   
   {  
      MyClass c = new MyClass();  
      int a = c.Property;   // CS0271  
      int b = c.Property2;   // OK  
   }  
}  
```
