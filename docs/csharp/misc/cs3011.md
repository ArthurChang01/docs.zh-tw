## <a name="example"></a><span data-ttu-id="239a3-107">範例</span><span class="sxs-lookup"><span data-stu-id="239a3-107">Example</span></span>  
 <span data-ttu-id="239a3-108">下列範例會產生 CS3011：</span><span class="sxs-lookup"><span data-stu-id="239a3-108">The following example generates CS3011:</span></span>  
  
```csharp  
// CS3011.cs  
  
using System;  
  
[assembly:CLSCompliant(true)]  
public abstract class I  
{  
    [CLSCompliant(false)]  
    public abstract int M();   // CS3011  
  
    // OK  
    [CLSCompliant(false)]  
    public void M2()  
    {  
    }  
}  
  
public class C : I  
{  
    public override int M()  
    {  
        return 1;  
    }  
  
    public static void Main()  
    {  
    }  
}  
```
