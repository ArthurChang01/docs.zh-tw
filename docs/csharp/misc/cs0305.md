## <a name="example"></a><span data-ttu-id="98417-106">範例</span><span class="sxs-lookup"><span data-stu-id="98417-106">Example</span></span>

```csharp  
// CS0305.cs

public class MyClass<T> {}  
  
class MyClass  
{  
   public static void Main()

      MyList<MyClass, MyClass> list1 = new MyList<MyClass>();   // CS0305  
      MyList<MyClass> list2 = new MyList<MyClass>();   // OK  
   }  
}  
```
