# <a name="compiler-error-cs0192"></a><span data-ttu-id="ae7b9-102">編譯器錯誤 CS0192</span><span class="sxs-lookup"><span data-stu-id="ae7b9-102">Compiler Error CS0192</span></span>
<span data-ttu-id="ae7b9-103">不能傳遞靜態唯讀欄位 'name' 的欄位給 ref 或 out (除非在靜態建構函式中)</span><span class="sxs-lookup"><span data-stu-id="ae7b9-103">Fields of static readonly field 'name' cannot be passed ref or out (except in a static constructor)</span></span>  
  
 <span data-ttu-id="ae7b9-104">使用標示的欄位 （變數） [readonly](../../csharp/language-reference/keywords/readonly.md)關鍵字不能傳遞到[ref](../../csharp/language-reference/keywords/ref.md)或[出](../../csharp/language-reference/keywords/out-parameter-modifier.md)除了建構函式內的參數。</span><span class="sxs-lookup"><span data-stu-id="ae7b9-104">A field (variable) marked with the [readonly](../../csharp/language-reference/keywords/readonly.md) keyword cannot be passed either to a [ref](../../csharp/language-reference/keywords/ref.md) or [out](../../csharp/language-reference/keywords/out-parameter-modifier.md) parameter except inside a constructor.</span></span> <span data-ttu-id="ae7b9-105">如需詳細資訊，請參閱[欄位](../../csharp/programming-guide/classes-and-structs/fields.md)。</span><span class="sxs-lookup"><span data-stu-id="ae7b9-105">For more information, see [Fields](../../csharp/programming-guide/classes-and-structs/fields.md).</span></span>  
  
 <span data-ttu-id="ae7b9-106">如果 `readonly` 欄位是 [static](../../csharp/language-reference/keywords/static.md) 而建構函式未標示為 `static`，則也會產生 CS0192。</span><span class="sxs-lookup"><span data-stu-id="ae7b9-106">CS0192 also results if the `readonly` field is [static](../../csharp/language-reference/keywords/static.md) and the constructor is not marked `static`.</span></span>  
  
## <a name="example"></a><span data-ttu-id="ae7b9-107">範例</span><span class="sxs-lookup"><span data-stu-id="ae7b9-107">Example</span></span>  
 <span data-ttu-id="ae7b9-108">下列範例會產生 CS0192。</span><span class="sxs-lookup"><span data-stu-id="ae7b9-108">The following sample generates CS0192.</span></span>  
  
```csharp 
// CS0192.cs  
class MyClass  
{  
    public readonly int TestInt = 6;  
    static void TestMethod(ref int testInt)  
    {  
        testInt = 0;  
    }  
  
    MyClass()  
    {  
        TestMethod(ref TestInt);   // OK  
    }  
  
    public void PassReadOnlyRef()  
    {  
        TestMethod(ref TestInt);   // CS0192  
    }  
  
    public static void Main()  
    {  
    }  
}  
```
