## <a name="example"></a><span data-ttu-id="4550b-108">範例</span><span class="sxs-lookup"><span data-stu-id="4550b-108">Example</span></span>  
 <span data-ttu-id="4550b-109">下列範例會產生 CS0131。</span><span class="sxs-lookup"><span data-stu-id="4550b-109">The following sample generates CS0131.</span></span>  
  
```csharp  
// CS0131.cs  
public class MyClass  
{  
    public int i = 0;  
    public void MyMethod()  
    {  
        i++ = 1;   // CS0131  
        // try the following line instead  
        // i = 1;  
    }  
    public static void Main() { }  
}  
```  
  
## <a name="example"></a><span data-ttu-id="4550b-110">範例</span><span class="sxs-lookup"><span data-stu-id="4550b-110">Example</span></span>

{  
    public static int Main()  
    {
