# <a name="compiler-error-cs0410"></a>編譯器錯誤 CS0410
沒有任何 'method' 的多載具有正確的參數和傳回類型  
  
 如果您嘗試具現化的委派具有包含錯誤參數類型的函式，則會發生這個錯誤。 委派的參數類型必須符合您指派給委派的函式。  
  
## <a name="example"></a>範例  
 下列範例會產生 CS0410：  
  
```csharp  
// CS0410.cs  
// compile with: /langversion:ISO-1  
  
class Test  
{  
    delegate void D(double d );  
    static void F(int i) { }  
  
    static void Main()  
    {  
        D d = new D(F);  // CS0410  
    }  
}  
```
