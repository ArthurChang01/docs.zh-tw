# <a name="compiler-error-cs0170"></a><span data-ttu-id="71953-102">編譯器錯誤 CS0170</span><span class="sxs-lookup"><span data-stu-id="71953-102">Compiler Error CS0170</span></span>
<span data-ttu-id="71953-103">使用可能未指派的欄位 'field'</span><span class="sxs-lookup"><span data-stu-id="71953-103">Use of possibly unassigned field 'field'</span></span>  
  
 <span data-ttu-id="71953-104">在結構中使用了欄位，但未先加以初始化。</span><span class="sxs-lookup"><span data-stu-id="71953-104">A field in a structure was used without first being initialized.</span></span> <span data-ttu-id="71953-105">若要解決這個問題，請先判斷哪個欄位未初始化，然後加以初始化，再嘗試存取。</span><span class="sxs-lookup"><span data-stu-id="71953-105">To solve this problem, first determine which field was uninitialized and then initialize it before you try to access it.</span></span> <span data-ttu-id="71953-106">如需初始化結構的詳細資訊, 請參閱[結構](../programming-guide/classes-and-structs/structs.md)和[使用結構](../programming-guide/classes-and-structs/using-structs.md)。</span><span class="sxs-lookup"><span data-stu-id="71953-106">For more information about initializing structs, see [Structs](../programming-guide/classes-and-structs/structs.md) and [Using Structs](../programming-guide/classes-and-structs/using-structs.md).</span></span>  
  
 <span data-ttu-id="71953-107">下列範例會產生 CS0170：</span><span class="sxs-lookup"><span data-stu-id="71953-107">The following sample generates CS0170:</span></span>  
  
```csharp  
// CS0170.cs  
public struct error  
{  
   public int i;  
}  
  
public class MyClass  
{  
   public static void Main()  
   {  
      error e;  
      // uncomment the next line to resolve this error  
      // e.i = 0;  
      System.Console.WriteLine( e.i );   // CS0170 because   
                                         //e.i was never assigned  
   }  
}  
```
