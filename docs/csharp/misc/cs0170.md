<span data-ttu-id="33f87-103">使用可能未指派的欄位 'field'</span><span class="sxs-lookup"><span data-stu-id="33f87-103">Use of possibly unassigned field 'field'</span></span>  
  
 <span data-ttu-id="33f87-104">在結構中使用了欄位，但未先加以初始化。</span><span class="sxs-lookup"><span data-stu-id="33f87-104">A field in a structure was used without first being initialized.</span></span> <span data-ttu-id="33f87-105">若要解決這個問題，請先判斷哪個欄位未初始化，然後加以初始化，再嘗試存取。</span><span class="sxs-lookup"><span data-stu-id="33f87-105">To solve this problem, first determine which field was uninitialized and then initialize it before you try to access it.</span></span> <span data-ttu-id="33f87-106">如需初始化結構的詳細資訊，請參閱[結構](../../csharp/programming-guide/classes-and-structs/structs.md)和[使用結構](../../csharp/programming-guide/classes-and-structs/using-structs.md)。</span><span class="sxs-lookup"><span data-stu-id="33f87-106">For more information about initializing structs, see [Structs](../../csharp/programming-guide/classes-and-structs/structs.md) and [Using Structs](../../csharp/programming-guide/classes-and-structs/using-structs.md).</span></span>  
  
 <span data-ttu-id="33f87-107">下列範例會產生 CS0170：</span><span class="sxs-lookup"><span data-stu-id="33f87-107">The following sample generates CS0170:</span></span>  
  
```  
// CS0170.cs  
public struct error  
{  
   public int i;  
}  
  
public class MyClass  
{  
   public static void Main()  
   {  
      error e;  
      // uncomment the next line to resolve this error  
      // e.i = 0;  
      System.Console.WriteLine( e.i );   // CS0170 because   
                                         //e.i was never assigned  
   }  
}  
```
