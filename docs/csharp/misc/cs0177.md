 <span data-ttu-id="98e02-104">未在方法主體中指派值給以 [out](../../csharp/language-reference/keywords/out.md) 關鍵字標記的參數。</span><span class="sxs-lookup"><span data-stu-id="98e02-104">A parameter marked with the [out](../../csharp/language-reference/keywords/out.md) keyword was not assigned a value in the method body.</span></span> <span data-ttu-id="98e02-105">如需詳細資訊，請參閱[傳遞參數](../../csharp/programming-guide/classes-and-structs/passing-parameters.md)</span><span class="sxs-lookup"><span data-stu-id="98e02-105">For more information, see [Passing Parameters](../../csharp/programming-guide/classes-and-structs/passing-parameters.md)</span></span>

```  
       int x = -1;  
       Foo(out x);  
   }  
}  
```
