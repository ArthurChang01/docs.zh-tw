 <span data-ttu-id="477a6-104">不允許轉換為衍生類別值的使用者定義轉換；您不需要這類運算子。</span><span class="sxs-lookup"><span data-stu-id="477a6-104">User-defined conversions to values of a derived class are not allowed; you do not need such an operator.</span></span>  
  
 <span data-ttu-id="477a6-105">如需使用者定義轉換的詳細資訊，請參閱＜C# 語言規格＞中的第 6 章。</span><span class="sxs-lookup"><span data-stu-id="477a6-105">See chapter 6 in the C# language specification for more information on user-defined conversions.</span></span>  
  
 <span data-ttu-id="477a6-106">下列範例會產生 CS0554：</span><span class="sxs-lookup"><span data-stu-id="477a6-106">The following sample generates CS0554:</span></span>  
  
```csharp  
// CS0554.cs  
namespace x  
{  
   public class ii  
   {  
      // delete the conversion routine to resolve CS0554  
      public static implicit operator ii(a aa) // CS0554  
      {  
         return new ii();  
      }  
   }  
  
   public class a : ii  
   {  
      public static void Main()  
      {  
      }  
   }  
}  
```
