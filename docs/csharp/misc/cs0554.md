 <span data-ttu-id="d27f0-105">如需使用者定義轉換的詳細資訊，請參閱＜C# 語言規格＞中的第 6 章。</span><span class="sxs-lookup"><span data-stu-id="d27f0-105">See chapter 6 in the C# language specification for more information on user-defined conversions.</span></span>

```csharp  
// CS0554.cs  
namespace x  
{  
   public class ii  
   {  
      // delete the conversion routine to resolve CS0554  
      public static implicit operator ii(a aa) // CS0554  
      {  
         return new ii();  
      }  
   }  
  
   public class a : ii  
   {  
      public static void Main()  
      {  
      }  
   }  
}  
```
