## <a name="example"></a><span data-ttu-id="9cf17-111">範例</span><span class="sxs-lookup"><span data-stu-id="9cf17-111">Example</span></span>  
 <span data-ttu-id="9cf17-112">下列程式碼會產生 CS0312：</span><span class="sxs-lookup"><span data-stu-id="9cf17-112">The following code generates CS0312:</span></span>  
  
```  
// cs0312.cs  
class Program  
{  
    static void MTyVar<T, U>() where T : U { }  
  
    static int Main()  
    {  
        MTyVar<int?, int>(); // CS0312  
        return 1;  
    }  
}  
```
