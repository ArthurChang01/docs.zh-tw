## <a name="example"></a><span data-ttu-id="45d8b-111">範例</span><span class="sxs-lookup"><span data-stu-id="45d8b-111">Example</span></span>

<span data-ttu-id="45d8b-112">下列程式碼會產生 CS0312：</span><span class="sxs-lookup"><span data-stu-id="45d8b-112">The following code generates CS0312:</span></span>

    static void MTyVar<T, U>() where T : U { }

        MTyVar<int?, int>(); // CS0312  
        return 1;  
    }
