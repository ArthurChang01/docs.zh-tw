# <a name="compiler-error-cs0468"></a>編譯器錯誤 CS0468

類型 'type1' 和類型 'type2' 之間模稜兩可

 當所要編譯的組件中有兩種類型的完整限定名稱相同時，就會產生這個錯誤。 如果這兩者都在加入的模組中，或者其中一個在加入的模組中，而另一個在來源中，則可能會發生這種情況。

> [!NOTE]
> 此編譯器錯誤不再在 Roslyn 中使用。
