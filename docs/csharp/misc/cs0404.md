# <a name="compiler-error-cs0404"></a>編譯器錯誤 CS0404
未預期的 '<' : 屬性不可為泛型  
  
 屬性中不允許泛型類型參數。 請移除類型參數和角括弧。  
  
 下列範例會產生 CS0404：  
  
```csharp  
// CS0404.cs  
[MyAttrib<int>]  // CS0404  
class C  
{  
   public static void Main()  
   {  
  
   }  
}  
```
