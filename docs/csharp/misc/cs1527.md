# <a name="compiler-error-cs1527"></a><span data-ttu-id="e57b7-102">編譯器錯誤 CS1527</span><span class="sxs-lookup"><span data-stu-id="e57b7-102">Compiler Error CS1527</span></span>

<span data-ttu-id="e57b7-103">在命名空間中定義的元素無法明確宣告為 private、protected、protected internal 或 private protected。</span><span class="sxs-lookup"><span data-stu-id="e57b7-103">Elements defined in a namespace cannot be explicitly declared as private, protected, protected internal or private protected.</span></span>
  
 <span data-ttu-id="e57b7-104">命名空間中的類型宣告可以有 [公用](../../csharp/language-reference/keywords/public.md) 或 [內部](../../csharp/language-reference/keywords/internal.md) 存取權。</span><span class="sxs-lookup"><span data-stu-id="e57b7-104">Type declarations in a namespace can have either [public](../../csharp/language-reference/keywords/public.md) or [internal](../../csharp/language-reference/keywords/internal.md) access.</span></span> <span data-ttu-id="e57b7-105">如果未指定任何可存取性，則 **內部** 是預設值。</span><span class="sxs-lookup"><span data-stu-id="e57b7-105">If no accessibility is specified, **internal** is the default.</span></span>  
  
 <span data-ttu-id="e57b7-106">下列範例會產生 CS1527：</span><span class="sxs-lookup"><span data-stu-id="e57b7-106">The following sample generates CS1527:</span></span>  
  
```csharp  
// CS1527.cs  
namespace Sample  
{  
   private class C1 {}             // CS1527  
   protected class C2 {}           // CS1527  
   protected internal class C3 {}  // CS1527  
   private protected class C4 {}   // CS1527
}  
```  
  
 <span data-ttu-id="e57b7-107">下列範例會產生 CS1527，因為未在程式碼中明確宣告命名空間時，所有類型宣告都是以隱含方式位於全域命名空間內。</span><span class="sxs-lookup"><span data-stu-id="e57b7-107">The following example generates CS1527 because when no namespace is explicitly declared in your program code, all type declarations are located implicitly within the global namespace.</span></span>  
  
```csharp  
//cs1527_2.cs  
using System;  
  
protected class C4 {}  
private struct S1 {}  
```
