 <span data-ttu-id="44b14-104">類別參考的格式，就像建立類別的物件一樣。</span><span class="sxs-lookup"><span data-stu-id="44b14-104">A reference to a class was formed as if an object to the class was being created.</span></span> <span data-ttu-id="44b14-105">例如，嘗試將變數傳遞至建構函式。</span><span class="sxs-lookup"><span data-stu-id="44b14-105">For example, there was an attempt to pass a variable to a constructor.</span></span> <span data-ttu-id="44b14-106">使用 [new](../language-reference/operators/new-operator.md) 運算子建立類別的物件。</span><span class="sxs-lookup"><span data-stu-id="44b14-106">Use the [new](../language-reference/operators/new-operator.md) operator to create an object of a class.</span></span>  
  
 <span data-ttu-id="44b14-107">下列範例會產生 CS1528：</span><span class="sxs-lookup"><span data-stu-id="44b14-107">The following sample generates CS1528:</span></span>  
  
```csharp  
// CS1528.cs  
using System;  
  
public class B  
{  
   public B(int i)  
   {  
      _i = i;  
   }  
  
   public void PrintB()  
   {  
      Console.WriteLine(_i);  
   }  
  
   private int _i;  
}  
  
public class mine  
{  
   public static void Main()  
   {  
      B b(3);   // CS1528, reference is not an object  
      // try one of the following  
      // B b;  
      // or  
      // B bb = new B(3);  
      // bb.PrintB();  
   }  
}  
```
