# <a name="compiler-error-cs1629"></a>編譯器錯誤 CS1629
Unsafe 程式碼不可出現在迭代器中  
  
 C# 語言規格不容許迭代器中有 Unsafe 程式碼。  
  
 下列範例會產生 CS1629：  
  
```  
// CS1629.cs  
// compile with: /unsafe    
using System.Collections.Generic;  
class C   
{  
   IEnumerator<int> IteratorMeth() {  
      int i;  
      unsafe  // CS1629  
      {  
         int *p = &i;  
         yield return *p;  
      }  
   }  
}  
```
