# <a name="compiler-error-cs1629"></a><span data-ttu-id="57408-102">編譯器錯誤 CS1629</span><span class="sxs-lookup"><span data-stu-id="57408-102">Compiler Error CS1629</span></span>
<span data-ttu-id="57408-103">Unsafe 程式碼不可出現在迭代器中</span><span class="sxs-lookup"><span data-stu-id="57408-103">Unsafe code may not appear in iterators</span></span>  
  
 <span data-ttu-id="57408-104">C# 語言規格不容許迭代器中有 Unsafe 程式碼。</span><span class="sxs-lookup"><span data-stu-id="57408-104">The C# language specification does not allow unsafe code in iterators.</span></span>  
  
 <span data-ttu-id="57408-105">下列範例會產生 CS1629：</span><span class="sxs-lookup"><span data-stu-id="57408-105">The following sample generates CS1629:</span></span>  
  
```csharp  
// CS1629.cs  
// compile with: /unsafe    
using System.Collections.Generic;  
class C   
{  
   IEnumerator<int> IteratorMeth() {  
      int i;  
      unsafe  // CS1629  
      {  
         int *p = &i;  
         yield return *p;  
      }  
   }  
}  
```
