 <span data-ttu-id="ba952-104">C# 語言規格不容許迭代器中有 Unsafe 程式碼。</span><span class="sxs-lookup"><span data-stu-id="ba952-104">The C# language specification does not allow unsafe code in iterators.</span></span>  
  
 <span data-ttu-id="ba952-105">下列範例會產生 CS1629：</span><span class="sxs-lookup"><span data-stu-id="ba952-105">The following sample generates CS1629:</span></span>  
  
```csharp  
// CS1629.cs  
// compile with: /unsafe    
using System.Collections.Generic;  
class C   
{  
   IEnumerator<int> IteratorMeth() {  
      int i;  
      unsafe  // CS1629  
      {  
         int *p = &i;  
         yield return *p;  
      }  
   }  
}  
```
