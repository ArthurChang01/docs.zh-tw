## <a name="example"></a><span data-ttu-id="10e77-108">範例</span><span class="sxs-lookup"><span data-stu-id="10e77-108">Example</span></span>  
 <span data-ttu-id="10e77-109">下列範例會產生 CS0688。</span><span class="sxs-lookup"><span data-stu-id="10e77-109">The following sample generates CS0688.</span></span> <span data-ttu-id="10e77-110">若要在不修改基底類別的情況下解決警告，請從覆寫方法中移除安全性屬性。</span><span class="sxs-lookup"><span data-stu-id="10e77-110">To resolve the warning without modifying the base class, remove the security attribute from the overriding method.</span></span> <span data-ttu-id="10e77-111">這不會解決安全性問題。</span><span class="sxs-lookup"><span data-stu-id="10e77-111">This will not solve the security problem.</span></span>  
  
```  
// CS0688.cs  
// compile with: /W:1  
using System;  
using System.Security.Permissions;  
  
class Base   
{  
    //Uncomment the following line to close the security hole  
    //[FileIOPermission(SecurityAction.LinkDemand, All=@"C:\\")]  
    public virtual void DoScaryFileStuff()  
    {  
    }  
}  
  
class Derived: Base  
{  
    [FileIOPermission(SecurityAction.LinkDemand, All=@"C:\\")] // CS0688  
    public override void DoScaryFileStuff()  
    {  
    }  
    static void Main()  
    {  
    }  
}  
```
