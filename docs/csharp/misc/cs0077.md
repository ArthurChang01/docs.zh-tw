# <a name="compiler-error-cs0077"></a><span data-ttu-id="c1d9e-102">編譯器錯誤 CS0077</span><span class="sxs-lookup"><span data-stu-id="c1d9e-102">Compiler Error CS0077</span></span>
<span data-ttu-id="c1d9e-103">as 運算子必須和參考類型或可為 Null 的類型一起使用 ('int' 是不可為 Null 的實值類型)。</span><span class="sxs-lookup"><span data-stu-id="c1d9e-103">The as operator must be used with a reference type or nullable type ('int' is a non-nullable value type).</span></span>  
  
 <span data-ttu-id="c1d9e-104">傳遞了 [實值類型](../language-reference/operators/type-testing-and-cast.md#as-operator) 給 [as](../language-reference/keywords/value-types.md)運算子。</span><span class="sxs-lookup"><span data-stu-id="c1d9e-104">The [as](../language-reference/operators/type-testing-and-cast.md#as-operator) operator was passed a [value type](../language-reference/keywords/value-types.md).</span></span> <span data-ttu-id="c1d9e-105">因為 `as` 可以傳回 [null](../language-reference/keywords/null.md)，所以只能傳遞 [參考類型](../language-reference/keywords/reference-types.md) 或可為 Null 的類型給它。</span><span class="sxs-lookup"><span data-stu-id="c1d9e-105">Because `as` can return [null](../language-reference/keywords/null.md), it can only be passed [reference types](../language-reference/keywords/reference-types.md) or nullable type.</span></span> <span data-ttu-id="c1d9e-106">如需可為 null 類型的詳細資訊, 請參閱[可為 null 的類型](../programming-guide/nullable-types/index.md)。</span><span class="sxs-lookup"><span data-stu-id="c1d9e-106">For more information about nullable types, see [Nullable Types](../programming-guide/nullable-types/index.md).</span></span>  
  
 <span data-ttu-id="c1d9e-107">下列範例會產生 CS0077：</span><span class="sxs-lookup"><span data-stu-id="c1d9e-107">The following sample generates CS0077:</span></span>  
  
```csharp  
// CS0077.cs  
using System;  
  
class C  
{  
}  
  
struct S  
{  
}  
  
class M  
{  
   public static void Main()  
   {  
      object o1, o2;  
      C c;  
      S s;  
  
      o1 = new C();  
      o2 = new S();  
  
      s = o2 as S;  // CS0077, S is not a reference type.  
      // try the following line instead  
      // c = o1 as C;  
   }  
}  
```
