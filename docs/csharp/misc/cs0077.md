# <a name="compiler-error-cs0077"></a>編譯器錯誤 CS0077
as 運算子必須和參考類型或可為 Null 的類型一起使用 ('int' 是不可為 Null 的實值類型)。  
  
 傳遞了 [實值類型](../../csharp/language-reference/operators/type-testing-and-conversion-operators.md#as-operator) 給 [as](../../csharp/language-reference/keywords/value-types.md)運算子。 因為 `as` 可以傳回 [null](../../csharp/language-reference/keywords/null.md)，所以只能傳遞 [參考類型](../../csharp/language-reference/keywords/reference-types.md) 或可為 Null 的類型給它。 如需可為 null 類型的詳細資訊，請參閱[可為 Null 的型別](../../csharp/programming-guide/nullable-types/index.md)。  
  
 下列範例會產生 CS0077：  
  
```csharp  
// CS0077.cs  
using System;  
  
class C  
{  
}  
  
struct S  
{  
}  
  
class M  
{  
   public static void Main()  
   {  
      object o1, o2;  
      C c;  
      S s;  
  
      o1 = new C();  
      o2 = new S();  
  
      s = o2 as S;  // CS0077, S is not a reference type.  
      // try the following line instead  
      // c = o1 as C;  
   }  
}  
```
