# <a name="compiler-error-cs0405"></a>編譯器錯誤 CS0405
類型參數 'type parameter' 有重複的條件約束 'constraint'  
  
 泛型宣告上的兩個條件約束完全相同。 若要清除錯誤，請移除重複項目。  
  
 下列範例會產生 CS0405：  
  
```csharp  
// CS0405.cs  
interface I  
{  
}  
  
class C<T> where T : I, I  // CS0405.cs  
{  
}  
```
