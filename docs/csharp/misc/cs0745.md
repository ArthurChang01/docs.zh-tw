 <span data-ttu-id="ccfda-104">`group` 子句的模式是後面接著選擇性 `group...by` 的 `into`(如下列範例所示)：</span><span class="sxs-lookup"><span data-stu-id="ccfda-104">The pattern for the `group` clause is `group...by` followed by an optional `into`, as shown in the following example:</span></span>

## <a name="example"></a><span data-ttu-id="ccfda-108">範例</span><span class="sxs-lookup"><span data-stu-id="ccfda-108">Example</span></span>  
 <span data-ttu-id="ccfda-109">下列程式碼會產生 CS0745：</span><span class="sxs-lookup"><span data-stu-id="ccfda-109">The following code generates CS0745:</span></span>  
  
```csharp  
// cs0745.cs  
using System;  
using System.Linq;  
  
public class C  
{  
    public static int Main()  
    {  
        string[] names = { "Bob", "Bill", "Jonetta", "Mary" };  
  
        var query = from name in names  
                    group name name[0]; // CS0745  
  
        return 1;  
    }  
}  
```
