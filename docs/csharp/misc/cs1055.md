# <a name="compiler-error-cs1055"></a><span data-ttu-id="28378-102">編譯器錯誤 CS1055</span><span class="sxs-lookup"><span data-stu-id="28378-102">Compiler Error CS1055</span></span>
<span data-ttu-id="28378-103">必須是 add 或 remove 存取子</span><span class="sxs-lookup"><span data-stu-id="28378-103">An add or remove accessor expected</span></span>  
  
 <span data-ttu-id="28378-104">如果您的 [event](../../csharp/language-reference/keywords/event.md) 未宣告為欄位，則必須同時定義 **add** 和 **remove** 存取子函式。</span><span class="sxs-lookup"><span data-stu-id="28378-104">If your [event](../../csharp/language-reference/keywords/event.md) is not declared as a field, it must define both **add** and **remove** accessor functions.</span></span>  
  
 <span data-ttu-id="28378-105">下列範例會產生 CS1055：</span><span class="sxs-lookup"><span data-stu-id="28378-105">The following sample generates CS1055:</span></span>  
  
```  
// CS1055.cs  
delegate void del();  
class Test  
{  
   public event del MyEvent  
   {  
      int i;   // CS1055  
      // uncomment accessors and delete previous line to resolve  
      // add  
      // {  
      //    MyEvent += value;  
      // }  
      // remove  
      // {  
      //    MyEvent -= value;  
      // }  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
