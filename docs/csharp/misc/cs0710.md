# <a name="compiler-error-cs0710"></a>編譯器錯誤 CS0710
靜態類別不能有執行個體建構函式  
  
 靜態類別無法具現化，因此不需要建構函式。 若要避免這個錯誤，請移除靜態類別所有的建構函式，或如果您真的很想要建構執行個體，請將類別變成非靜態。  
  
 下例會產生 CS0710：  
  
```csharp  
// CS0710.cs  
public static class C  
{  
   public C()  // CS0710  
   {  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
