# <a name="compiler-error-cs0690"></a>編譯器錯誤 CS0690
輸入檔 'file' 有無效的中繼資料。  
  
 您可以開啟中繼資料檔案，但因為當地語系化的一些問題，中繼資料已損毀。 這和錯誤 [CS0009](../../csharp/misc/cs0009.md)相類似，但您可以開啟中繼資料檔。  
  
 PE (處理序可執行檔) 是可以執行的檔案。
