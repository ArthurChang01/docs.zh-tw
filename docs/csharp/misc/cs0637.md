# <a name="compiler-error-cs0637"></a><span data-ttu-id="f8e41-102">編譯器錯誤 CS0637</span><span class="sxs-lookup"><span data-stu-id="f8e41-102">Compiler Error CS0637</span></span>

<span data-ttu-id="f8e41-103">Static 或 const 欄位不允許有 FieldOffset 屬性。</span><span class="sxs-lookup"><span data-stu-id="f8e41-103">The FieldOffset attribute is not allowed on static or const fields.</span></span>
  
 <span data-ttu-id="f8e41-104">[FieldOffset](xref:System.Runtime.InteropServices.FieldOffsetAttribute) 屬性不能用於標上 [static](../../csharp/language-reference/keywords/static.md) 或 [const](../../csharp/language-reference/keywords/const.md)的欄位。</span><span class="sxs-lookup"><span data-stu-id="f8e41-104">The [FieldOffset](xref:System.Runtime.InteropServices.FieldOffsetAttribute) attribute cannot be used on fields marked with [static](../../csharp/language-reference/keywords/static.md) or [const](../../csharp/language-reference/keywords/const.md).</span></span>  
  
 <span data-ttu-id="f8e41-105">下列範例會產生 CS0637：</span><span class="sxs-lookup"><span data-stu-id="f8e41-105">The following sample generates CS0637:</span></span>  
  
```csharp  
// CS0637.cs  
using System;  
using System.Runtime.InteropServices;  
  
[StructLayout(LayoutKind.Explicit)]  
public class MainClass  
{  
   [FieldOffset(3)]   // CS0637  
   public static int i;  
   public static void Main ()  
   {  
   }  
}  
```
