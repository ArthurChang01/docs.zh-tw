# <a name="compiler-error-cs0667"></a>編譯器錯誤 CS0667
'invalid feature' 功能已被取代。 請改用 'valid feature'。  
  
 現在已取代您嘗試使用的功能。 請更新您的程式碼，改用有效的功能。
