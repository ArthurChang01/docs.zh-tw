<span data-ttu-id="56d71-103">'constructor'：靜態建構函式不能有明確的 'this' 或 'base' 建構函式呼叫</span><span class="sxs-lookup"><span data-stu-id="56d71-103">'constructor' : static constructor cannot have an explicit 'this' or 'base' constructor call</span></span>  
  
 <span data-ttu-id="56d71-104">不允許在靜態建構函式中呼叫 `this` ，因為在建立任何類別執行個體之前會自動呼叫靜態建構函式。</span><span class="sxs-lookup"><span data-stu-id="56d71-104">Calling `this` in the static constructor is not allowed because the static constructor is called automatically before creating any instance of the class.</span></span> <span data-ttu-id="56d71-105">此外，靜態建構函式不會被繼承，也不能直接呼叫。</span><span class="sxs-lookup"><span data-stu-id="56d71-105">Also, static constructors are not inherited, and cannot be called directly.</span></span>  
  
 <span data-ttu-id="56d71-106">如需詳細資訊, 請參閱[此](../language-reference/keywords/this.md)和[基底](../language-reference/keywords/base.md)。</span><span class="sxs-lookup"><span data-stu-id="56d71-106">For more information, see [this](../language-reference/keywords/this.md) and [base](../language-reference/keywords/base.md).</span></span>  
  
## <a name="example"></a><span data-ttu-id="56d71-107">範例</span><span class="sxs-lookup"><span data-stu-id="56d71-107">Example</span></span>  
 <span data-ttu-id="56d71-108">下例會產生 CS0514：</span><span class="sxs-lookup"><span data-stu-id="56d71-108">The following example generates CS0514:</span></span>  
  
```csharp  
// CS0514.cs

{  
    static A() : base(0) // CS0514  
    {  
    }  
  
    public A(object o)  
    {  
    }  
}  
  
class B  
{  
    static B() : this(null) // CS0514  
    {  
    }  
  
    public B(object o)  
    {  
    }  
}  
```
