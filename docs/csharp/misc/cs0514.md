 <span data-ttu-id="686f3-104">不允許在靜態建構函式中呼叫 `this` ，因為在建立任何類別執行個體之前會自動呼叫靜態建構函式。</span><span class="sxs-lookup"><span data-stu-id="686f3-104">Calling `this` in the static constructor is not allowed because the static constructor is called automatically before creating any instance of the class.</span></span> <span data-ttu-id="686f3-105">此外，靜態建構函式不會被繼承，也不能直接呼叫。</span><span class="sxs-lookup"><span data-stu-id="686f3-105">Also, static constructors are not inherited, and cannot be called directly.</span></span>

## <a name="example"></a><span data-ttu-id="686f3-107">範例</span><span class="sxs-lookup"><span data-stu-id="686f3-107">Example</span></span>  
 <span data-ttu-id="686f3-108">下例會產生 CS0514：</span><span class="sxs-lookup"><span data-stu-id="686f3-108">The following example generates CS0514:</span></span>  
  
```csharp

    static A() : base(0) // CS0514  
    {  
    }  
  
    public A(object o)

    }

{  
    static B() : this(null) // CS0514  
    {  
    }  
  
    public B(object o)  
    {  
    }  
}  
```
