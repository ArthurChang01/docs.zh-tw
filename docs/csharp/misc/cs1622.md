# <a name="compiler-error-cs1622"></a><span data-ttu-id="256ee-102">編譯器錯誤 CS1622</span><span class="sxs-lookup"><span data-stu-id="256ee-102">Compiler Error CS1622</span></span>
<span data-ttu-id="256ee-103">無法從迭代器傳回值。</span><span class="sxs-lookup"><span data-stu-id="256ee-103">Cannot return a value from an iterator.</span></span> <span data-ttu-id="256ee-104">請使用 yield return 陳述式傳回值，或使用 yield break 結束反覆運算。</span><span class="sxs-lookup"><span data-stu-id="256ee-104">Use the yield return statement to return a value, or yield break to end the iteration.</span></span>  
  
 <span data-ttu-id="256ee-105">迭代器是一個特殊函式，可透過 yield 陳述式 (而非 return 陳述式) 來傳回值。</span><span class="sxs-lookup"><span data-stu-id="256ee-105">An iterator is a special function that returns a value via the yield statement rather than the return statement.</span></span> <span data-ttu-id="256ee-106">如需詳細資訊，請參閱 **iterator**。</span><span class="sxs-lookup"><span data-stu-id="256ee-106">For more information, see **iterators**.</span></span>  
  
 <span data-ttu-id="256ee-107">下列範例會產生 CS1622：</span><span class="sxs-lookup"><span data-stu-id="256ee-107">The following sample generates CS1622:</span></span>  
  
```csharp  
// CS1622.cs  
// compile with: /target:library  
using System.Collections;  
  
class C : IEnumerable  
{  
   public IEnumerator GetEnumerator()  
   {  
      return (IEnumerator) this;  // CS1622  
      yield return this;   // OK  
   }  
}  
```
