# <a name="compiler-warning-level-2-cs1668"></a>編譯器警告 (層級 2) CS1668
'path string' 中指定了無效的搜尋路徑 --  'system error message'  
  
 在命令列提供給[-lib](../language-reference/compiler-options/lib-compiler-option.md)的路徑無效，或 lib 環境變數中的路徑無效。 請檢查所用的路徑，確認其存在並可存取。 單引號中的錯誤訊息是作業系統傳回的錯誤。
