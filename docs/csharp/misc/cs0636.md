 <span data-ttu-id="3f655-104">您必須在結構宣告上使用 **StructLayout(LayoutKind.Explicit)** 屬性 (如果它包含使用 **FieldOffset** 屬性標記的任何成員)。</span><span class="sxs-lookup"><span data-stu-id="3f655-104">You must use the **StructLayout(LayoutKind.Explicit)** attribute on the struct declaration, if it contains any members marked with the **FieldOffset** attribute.</span></span> <span data-ttu-id="3f655-105">如需詳細資訊，請參閱 [FieldOffset](xref:System.Runtime.InteropServices.FieldOffsetAttribute)。</span><span class="sxs-lookup"><span data-stu-id="3f655-105">For more information, see [FieldOffset](xref:System.Runtime.InteropServices.FieldOffsetAttribute).</span></span>

{
