## <a name="example"></a><span data-ttu-id="7803d-106">範例</span><span class="sxs-lookup"><span data-stu-id="7803d-106">Example</span></span>  
 <span data-ttu-id="7803d-107">下列範例會產生 CS0471。</span><span class="sxs-lookup"><span data-stu-id="7803d-107">The following example generates CS0471.</span></span>  
  
```  
// CS0471.cs  
// compile with: /t:library  
class Test  
{  
    public void F(bool x, bool y) {}  
    public void F1()  
    {  
        int a = 1, b = 2, c = 3;  
        F(a<b, c>(3));    // CS0471  
        // To resolve, try the following instead:  
        // F((a<b), c>(3));  
    }  
}  
```
