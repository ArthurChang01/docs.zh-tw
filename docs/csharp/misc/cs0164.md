# <a name="compiler-warning-level-2-cs0164"></a>編譯器警告 (層級 2) CS0164
未參考此標籤  
  
 已宣告標籤，但從未使用。  
  
 下列範例會產生 CS0164：  
  
```csharp  
// CS0164.cs  
// compile with: /W:2  
public class a  
{  
   public int i = 0;  
  
   public static void Main()  
   {  
      int i = 0;   // CS0164  
      l1: i++;  
      // the following lines resolve this error  
      // if(i < 10)  
      //    goto l1;  
   }  
}  
```
