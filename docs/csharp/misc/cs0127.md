 <span data-ttu-id="eaa7a-104">傳回類型為 [void](../../csharp/language-reference/keywords/void.md) 的方法無法傳回值。</span><span class="sxs-lookup"><span data-stu-id="eaa7a-104">A method with a [void](../../csharp/language-reference/keywords/void.md) return type cannot return a value.</span></span> <span data-ttu-id="eaa7a-105">如需詳細資訊，請參閱[方法](../../csharp/programming-guide/classes-and-structs/methods.md)。</span><span class="sxs-lookup"><span data-stu-id="eaa7a-105">For more information, see [Methods](../../csharp/programming-guide/classes-and-structs/methods.md).</span></span>  
  
 <span data-ttu-id="eaa7a-106">下列範例會產生 CS0127：</span><span class="sxs-lookup"><span data-stu-id="eaa7a-106">The following sample generates CS0127:</span></span>  
  
```csharp  
// CS0127.cs  
namespace MyNamespace  
{  
   public class MyClass  
   {  
      public int hiddenMember2  
      {  
         get  
         {  
            return 0;  
         }  
         set   // CS0127, set has an implicit void return type  
         {  
            return 0;   // remove return statement to resolve this CS0127  
         }  
      }  
  
      public static void Main()  
      {  
      }  
   }  
}  
```
