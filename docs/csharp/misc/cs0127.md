# <a name="compiler-error-cs0127"></a>編譯器錯誤 CS0127
因為 'function' 傳回 void，return 關鍵字後面不可接物件運算式  
  
 傳回類型為 [void](../language-reference/keywords/void.md) 的方法無法傳回值。 如需詳細資訊，請參閱[方法](../programming-guide/classes-and-structs/methods.md)。  
  
 下列範例會產生 CS0127：  
  
```csharp  
// CS0127.cs  
namespace MyNamespace  
{  
   public class MyClass  
   {  
      public int hiddenMember2  
      {  
         get  
         {  
            return 0;  
         }  
         set   // CS0127, set has an implicit void return type  
         {  
            return 0;   // remove return statement to resolve this CS0127  
         }  
      }  
  
      public static void Main()  
      {  
      }  
   }  
}  
```
