<span data-ttu-id="9f9ef-103">因為 'function' 傳回 void，return 關鍵字後面不可接物件運算式</span><span class="sxs-lookup"><span data-stu-id="9f9ef-103">Since 'function' returns void, a return keyword must not be followed by an object expression</span></span>  
  
 <span data-ttu-id="9f9ef-104">傳回類型為 [void](../language-reference/keywords/void.md) 的方法無法傳回值。</span><span class="sxs-lookup"><span data-stu-id="9f9ef-104">A method with a [void](../language-reference/keywords/void.md) return type cannot return a value.</span></span> <span data-ttu-id="9f9ef-105">如需詳細資訊，請參閱[方法](../programming-guide/classes-and-structs/methods.md)。</span><span class="sxs-lookup"><span data-stu-id="9f9ef-105">For more information, see [Methods](../programming-guide/classes-and-structs/methods.md).</span></span>  
  
 <span data-ttu-id="9f9ef-106">下列範例會產生 CS0127：</span><span class="sxs-lookup"><span data-stu-id="9f9ef-106">The following sample generates CS0127:</span></span>  
  
```csharp  
// CS0127.cs  
namespace MyNamespace  
{  
   public class MyClass  
   {  
      public int hiddenMember2  
      {  
         get  
         {  
            return 0;  
         }  
         set   // CS0127, set has an implicit void return type  
         {  
            return 0;   // remove return statement to resolve this CS0127  
         }  
      }  
  
      public static void Main()  
      {  
      }  
   }  
}  
```
