 <span data-ttu-id="c758c-104">無法完成指派；請使用後置字元來更正指令。</span><span class="sxs-lookup"><span data-stu-id="c758c-104">An assignment could not be completed; use a suffix to correct the instruction.</span></span> <span data-ttu-id="c758c-105">每個類型的文件可識別類型的對應後置字元。</span><span class="sxs-lookup"><span data-stu-id="c758c-105">The documentation for each type identifies the corresponding suffix for the type.</span></span> <span data-ttu-id="c758c-106">如需有關轉換的詳細資訊，請參閱 <<c0> [ 轉型和類型轉換](../../csharp/programming-guide/types/casting-and-type-conversions.md)。</span><span class="sxs-lookup"><span data-stu-id="c758c-106">For more information on conversions, see [Casting and Type Conversions](../../csharp/programming-guide/types/casting-and-type-conversions.md).</span></span>  
  
 <span data-ttu-id="c758c-107">下列範例會產生 CS0664：</span><span class="sxs-lookup"><span data-stu-id="c758c-107">The following sample generates CS0664:</span></span>  
  
```csharp  
// CS0664.cs  
class Example  
{  
    static void Main()  
    {  
        decimal d1 = 1.0;   // CS0664, because 1.0 is interpreted  
                            // as a double.  
  
        // Try the following line instead.  
        decimal d2 = 1.0M;  // The M tells the compiler that 1.0 is a  
                            // decimal.  
        Console.WriteLine(d2);  
    }  
}  
```
