# <a name="compiler-error-cs1631"></a><span data-ttu-id="fe823-102">編譯器錯誤 CS1631</span><span class="sxs-lookup"><span data-stu-id="fe823-102">Compiler Error CS1631</span></span>
<span data-ttu-id="fe823-103">無法在 catch 子句主體中使用 yield 產生值</span><span class="sxs-lookup"><span data-stu-id="fe823-103">Cannot yield a value in the body of a catch clause</span></span>  
  
 <span data-ttu-id="fe823-104">catch 子句主體內不允許 yield 陳述式。</span><span class="sxs-lookup"><span data-stu-id="fe823-104">The yield statement is not allowed from within the body of a catch clause.</span></span> <span data-ttu-id="fe823-105">若要避免這個錯誤，請將 yield 陳述式移出 catch 子句主體外部。</span><span class="sxs-lookup"><span data-stu-id="fe823-105">To avoid this error, move the yield statement outside the body of the catch clause.</span></span>  
  
 <span data-ttu-id="fe823-106">下列範例會產生 CS1631：</span><span class="sxs-lookup"><span data-stu-id="fe823-106">The following sample generates CS1631:</span></span>  
  
```csharp  
// CS1631.cs  
using System;  
using System.Collections;  
  
public class C : IEnumerable  
{  
   public IEnumerator GetEnumerator()   
   {  
      try  
      {  
      }  
      catch(Exception e)  
      {  
        yield return this;  // CS1631  
      }  
   }    
  
   public static void Main()   
   {  
   }  
}  
```
