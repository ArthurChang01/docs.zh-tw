# <a name="compiler-error-cs0506"></a><span data-ttu-id="f6436-102">編譯器錯誤 CS0506</span><span class="sxs-lookup"><span data-stu-id="f6436-102">Compiler Error CS0506</span></span>
<span data-ttu-id="f6436-103">'function1': 無法覆寫繼承的成員 'function2'，因為其未標記為 "virtual"、"abstract" 或 "override"</span><span class="sxs-lookup"><span data-stu-id="f6436-103">'function1' : cannot override inherited member 'function2' because it is not marked "virtual", "abstract", or "override"</span></span>  
  
 <span data-ttu-id="f6436-104">已覆寫的方法未明確地標記為 **virtual**, **abstract**或 `override`。</span><span class="sxs-lookup"><span data-stu-id="f6436-104">A method was overridden that was not explicitly marked as **virtual**, **abstract**, or `override`.</span></span>  
  
 <span data-ttu-id="f6436-105">下列範例會產生 CS0506：</span><span class="sxs-lookup"><span data-stu-id="f6436-105">The following sample generates CS0506:</span></span>  
  
```csharp  
// CS0506.cs  
namespace MyNameSpace  
{  
   abstract public class ClassX  
   {  
      public int i = 0;  
  
      public int f()  
      {  
         return 0;  
      }  
      // Try the following definition for f() instead:  
      // abstract public int f();  
   }  
  
   public class ClassY : ClassX  
   {  
      public override int f()   // CS0506  
      {  
         return 0;  
      }  
  
      public static int Main()  
      {  
         return 0;  
      }  
   }  
}  
```
