## <a name="example"></a><span data-ttu-id="3dc15-106">範例</span><span class="sxs-lookup"><span data-stu-id="3dc15-106">Example</span></span>  
 <span data-ttu-id="3dc15-107">下列範例會產生 CS0551。</span><span class="sxs-lookup"><span data-stu-id="3dc15-107">The following sample generates CS0551.</span></span>  
  
```csharp  
// CS0551.cs  
// compile with: /target:library  
interface ii  
{  
   int i  
   {  
      get;  
      set;  
   }  
}  
  
public class a : ii  
{  
   int ii.i { set {} }   // CS0551  
  
   // OK  
   int ii.i      
   {  
      set {}  
      get { return 0; }  
   }  
}  
```
