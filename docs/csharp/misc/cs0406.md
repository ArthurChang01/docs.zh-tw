## <a name="example"></a><span data-ttu-id="7f20e-106">範例</span><span class="sxs-lookup"><span data-stu-id="7f20e-106">Example</span></span>  
 <span data-ttu-id="7f20e-107">下列範例會產生 CS0406。</span><span class="sxs-lookup"><span data-stu-id="7f20e-107">The following sample generates CS0406.</span></span>  
  
```csharp  
// CS0406.cs  
// compile with: /target:library  
interface I {}  
class C {}  
class D<T> where T : I, C {}   // CS0406  
class D2<T> where T : C, I {}   // OK  
```
