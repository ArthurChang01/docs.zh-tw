## <a name="example"></a><span data-ttu-id="51d4e-106">範例</span><span class="sxs-lookup"><span data-stu-id="51d4e-106">Example</span></span>  
 <span data-ttu-id="51d4e-107">下列範例會產生 CS0406。</span><span class="sxs-lookup"><span data-stu-id="51d4e-107">The following sample generates CS0406.</span></span>

class C {}  
class D<T> where T : I, C {}   // CS0406
