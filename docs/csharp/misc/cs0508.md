## <a name="example"></a><span data-ttu-id="ec6e7-106">範例</span><span class="sxs-lookup"><span data-stu-id="ec6e7-106">Example</span></span>  
 <span data-ttu-id="ec6e7-107">下列範例會產生 CS0508。</span><span class="sxs-lookup"><span data-stu-id="ec6e7-107">The following sample generates CS0508.</span></span>  
  
```csharp  
// CS0508.cs  
// compile with: /target:library  
abstract public class Clx  
{  
   public int i = 0;  
   // Return type is int.  
   abstract public int F();  
}  
  
public class Cly : Clx  
{  
   public override double F()  
   {  
      return 0.0;   // CS0508  
   }  
}  
```
