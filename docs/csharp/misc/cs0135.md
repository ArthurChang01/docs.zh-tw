 <span data-ttu-id="1339c-104">編譯器不允許隱藏名稱，這通常會導致您的程式碼出現邏輯錯誤。</span><span class="sxs-lookup"><span data-stu-id="1339c-104">The compiler does not allow hiding names, which commonly leads to logic errors in your code.</span></span>  
  
## <a name="example"></a><span data-ttu-id="1339c-105">範例</span><span class="sxs-lookup"><span data-stu-id="1339c-105">Example</span></span>  
 <span data-ttu-id="1339c-106">下列範例會產生 CS0135：</span><span class="sxs-lookup"><span data-stu-id="1339c-106">The following sample generates CS0135:</span></span>  
  
```csharp
// CS0135.cs  
public class MyClass2  
{  
   public static int i = 0;  
  
   public static void Main()  
   {  
      {  
         int i = 4;  
         i++;  
      }  
      i = 0;   // CS0135  
   }  
}  
```
