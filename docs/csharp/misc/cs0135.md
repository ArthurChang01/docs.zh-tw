## <a name="example"></a><span data-ttu-id="39aa2-105">範例</span><span class="sxs-lookup"><span data-stu-id="39aa2-105">Example</span></span>  
 <span data-ttu-id="39aa2-106">下列範例會產生 CS0135：</span><span class="sxs-lookup"><span data-stu-id="39aa2-106">The following sample generates CS0135:</span></span>  
  
```csharp
// CS0135.cs  
public class MyClass2  
{  
   public static int i = 0;  
  
   public static void Main()  
   {  
      {  
         int i = 4;  
         i++;  
      }  
      i = 0;   // CS0135  
   }  
}  
```
