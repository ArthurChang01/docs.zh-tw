# <a name="compiler-warning-level-1-cs0612"></a><span data-ttu-id="6057c-102">編譯器警告 (層級 1) CS0612</span><span class="sxs-lookup"><span data-stu-id="6057c-102">Compiler Warning (level 1) CS0612</span></span>
<span data-ttu-id="6057c-103">'member' 已過時</span><span class="sxs-lookup"><span data-stu-id="6057c-103">'member' is obsolete</span></span>  
  
 <span data-ttu-id="6057c-104">類別設計工具以 [過時](../programming-guide/concepts/attributes/common-attributes.md#Obsolete) 屬性標示成員。</span><span class="sxs-lookup"><span data-stu-id="6057c-104">The class designer marked a member with the [Obsolete](../programming-guide/concepts/attributes/common-attributes.md#Obsolete) attribute.</span></span> <span data-ttu-id="6057c-105">這表示類別未來的版本可能不支援該成員。</span><span class="sxs-lookup"><span data-stu-id="6057c-105">This means that the member might not be supported in a future version of the class.</span></span>  
  
 <span data-ttu-id="6057c-106">下例示範存取過時的成員如何產生 CS0612：</span><span class="sxs-lookup"><span data-stu-id="6057c-106">The following sample shows how accessing an obsolete member generates CS0612:</span></span>  
  
```csharp  
// CS0612.cs  
// compile with: /W:1  
using System;  
  
class MyClass  
{  
   [Obsolete]  
   public static void ObsoleteMethod()  
   {  
   }  
  
   [Obsolete]  
   public static int ObsoleteField;  
}  
  
class MainClass  
{  
   static public void Main()  
   {  
      MyClass.ObsoleteMethod();    // CS0612 here: method is deprecated  
      MyClass.ObsoleteField = 0;   // CS0612 here: field is deprecated  
   }  
}  
```
