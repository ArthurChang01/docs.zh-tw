# <a name="compiler-error-cs0522"></a><span data-ttu-id="328ad-102">編譯器錯誤 CS0522</span><span class="sxs-lookup"><span data-stu-id="328ad-102">Compiler Error CS0522</span></span>
<span data-ttu-id="328ad-103">'constructor' : 結構無法呼叫基底類別建構函式</span><span class="sxs-lookup"><span data-stu-id="328ad-103">'constructor' : structs cannot call base class constructors</span></span>  
  
 <span data-ttu-id="328ad-104">[struct](../../csharp/language-reference/keywords/struct.md) 無法呼叫基底類別建構函式；請移除基底類別建構函式的呼叫。</span><span class="sxs-lookup"><span data-stu-id="328ad-104">A [struct](../../csharp/language-reference/keywords/struct.md) cannot call a base class constructor; remove the call to the base class constructor.</span></span>  
  
 <span data-ttu-id="328ad-105">下列範例會產生 CS0522：</span><span class="sxs-lookup"><span data-stu-id="328ad-105">The following sample generates CS0522:</span></span>  
  
```csharp  
// CS0522.cs  
public class clx  
{  
   public clx(int i)  
   {  
   }  
  
   public static void Main()  
   {  
   }  
}  
  
public struct cly  
{  
   public cly(int i):base(0)   // CS0522  
   // try the following line instead  
   // public cly(int i)  
   {  
   }  
}  
```
