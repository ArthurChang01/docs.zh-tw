# <a name="compiler-error-cs1517"></a>編譯器錯誤 CS1517
前置處理器運算式無效  
  
 編譯器發現無效的前置處理器運算式。  
  
 如需詳細資訊，請參閱 [前置處理器指示詞](../../csharp/language-reference/preprocessor-directives/index.md)。  
  
 下列範例顯示一些有效和無效的前置處理器運算式：  
  
```csharp  
// CS1517.cs  
#if symbol      // OK  
#endif  
#if !symbol     // OK  
#endif  
#if (symbol)    // OK  
#endif  
#if true        // OK  
#endif  
#if false       // OK  
#endif  
#if 1           // CS1517  
#endif  
#if ~symbol     // CS1517  
#endif  
#if *           // CS1517  
#endif  
  
class x  
{  
   public static void Main()  
   {  
   }  
}  
```
