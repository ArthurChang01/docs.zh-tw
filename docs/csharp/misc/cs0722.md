 <span data-ttu-id="84f57-104">由於無法建立靜態類型的執行個體，因此將靜態類型當做傳回類型沒有任何意義。</span><span class="sxs-lookup"><span data-stu-id="84f57-104">A static type as a return type is not meaningful since instances of static types cannot be created.</span></span>  
  
 <span data-ttu-id="84f57-105">下列範例會產生 CS0722：</span><span class="sxs-lookup"><span data-stu-id="84f57-105">The following sample generates CS0722:</span></span>  
  
```csharp  
// CS0722.cs

{  
}

{  
   public SC F()  // CS0722

      return null;  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
