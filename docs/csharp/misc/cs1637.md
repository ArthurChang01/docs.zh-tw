# <a name="compiler-error-cs1637"></a>編譯器錯誤 CS1637
迭代器不能有 Unsafe 參數或 yield 類型  
  
 請檢查迭代器的引數清單和任何 yield 陳述式的類型，確認未使用任何不安全的類型。  
  
## <a name="example"></a>範例  
 下列範例會產生 CS1637：  
  
```csharp  
// CS1637.cs  
// compile with: /unsafe  
using System.Collections;  
  
public unsafe class C  
{  
    public IEnumerator Iterator1(int* p)  // CS1637  
    {  
        yield return null;  
    }  
}  
```
