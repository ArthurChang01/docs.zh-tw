# 編譯器錯誤 CS0621
'member' : 虛擬或抽象成員不可為私用  
  
 不允許私用**虛擬**或**抽象**成員。  
  
 下列範例會產生 CS0621：  
  
```  
// CS0621.cs abstract class MyClass { private virtual void DoNothing1()   // CS0621 { } private abstract void DoNothing2();   // CS0621 public static void Main() { } }  
```
