<span data-ttu-id="c7215-103">Conditional 屬性在 'function' 上無效，因為它是建構函式、解構函式、運算子或明確介面實作</span><span class="sxs-lookup"><span data-stu-id="c7215-103">The Conditional attribute is not valid on 'function' because it is a constructor, destructor, operator, or explicit interface implementation</span></span>

 <span data-ttu-id="c7215-105">例如，您無法在明確介面定義上使用某些屬性。</span><span class="sxs-lookup"><span data-stu-id="c7215-105">For example, you cannot use some attributes on an explicit interface definition.</span></span> <span data-ttu-id="c7215-106">下列範例會產生 CS0577：</span><span class="sxs-lookup"><span data-stu-id="c7215-106">The following sample generates CS0577:</span></span>  
  
```csharp  
// CS0577.cs  
// compile with: /target:library  
interface I  
{  
   void m();  
}  
  
public class MyClass : I  
{  
   [System.Diagnostics.Conditional("a")]   // CS0577  
   void I.m() {}  
}  
```
