# <a name="compiler-error-cs0021"></a><span data-ttu-id="1eeeb-102">編譯器錯誤 CS0021</span><span class="sxs-lookup"><span data-stu-id="1eeeb-102">Compiler Error CS0021</span></span>

<span data-ttu-id="1eeeb-103">無法套用有 [] 的索引至類型 'type' 的運算式</span><span class="sxs-lookup"><span data-stu-id="1eeeb-103">Cannot apply indexing with [] to an expression of type 'type'</span></span>  
  
<span data-ttu-id="1eeeb-104">嘗試透過不支援 [Indexers](../programming-guide/indexers/index.md)的資料類型上的索引子來存取值。</span><span class="sxs-lookup"><span data-stu-id="1eeeb-104">An attempt was made to access a value through an indexer on a data type that does not support [Indexers](../programming-guide/indexers/index.md).</span></span>  
  
<span data-ttu-id="1eeeb-105">嘗試在 C++ 組件中使用索引子時，可能會得到 CS0021。</span><span class="sxs-lookup"><span data-stu-id="1eeeb-105">You may get CS0021 when trying to use an indexer in a C++ assembly.</span></span> <span data-ttu-id="1eeeb-106">在此情況下，請為 C++ 類別加上 `DefaultMember` 屬性，讓 C# 編譯器知道哪一個索引子是預設值。</span><span class="sxs-lookup"><span data-stu-id="1eeeb-106">In this case, decorate the C++ class with the `DefaultMember` attribute so the C# compiler knows which indexer is the default.</span></span> <span data-ttu-id="1eeeb-107">下列範例會產生 CS0021：</span><span class="sxs-lookup"><span data-stu-id="1eeeb-107">The following sample generates CS0021.</span></span>  
  
## <a name="example"></a><span data-ttu-id="1eeeb-108">範例</span><span class="sxs-lookup"><span data-stu-id="1eeeb-108">Example</span></span>  

<span data-ttu-id="1eeeb-109">下列C++範例會將編譯為 .dll 檔案。</span><span class="sxs-lookup"><span data-stu-id="1eeeb-109">The following C++ example compiles to a .dll file.</span></span> <span data-ttu-id="1eeeb-110">請注意， [DefaultMember](xref:System.Reflection.DefaultMemberAttribute)屬性會加上批註，以便產生錯誤。</span><span class="sxs-lookup"><span data-stu-id="1eeeb-110">Note that the the [DefaultMember](xref:System.Reflection.DefaultMemberAttribute) attribute is commented out in order to generate the error.</span></span>  
  
```cpp  
// CPP0021.cpp  
// compile with: /clr /LD  
using namespace System::Reflection;  
// Uncomment the following line to resolve  
//[DefaultMember("myItem")]  
public ref class MyClassMC  
{  
        public:  
        property int myItem[int]  
        {  
            int get(int i){  return 5; }  
            void set(int i, int value) {}  
        }  
};  
```  

<span data-ttu-id="1eeeb-111">下列C#範例會呼叫 .dll 檔案。</span><span class="sxs-lookup"><span data-stu-id="1eeeb-111">The following C# example calls the .dll file.</span></span> <span data-ttu-id="1eeeb-112">它會嘗試透過索引子存取類別，但因為沒有任何成員宣告為預設索引子，所以會產生錯誤。</span><span class="sxs-lookup"><span data-stu-id="1eeeb-112">It attempts to access the class via an indexer, but because no member was declared as the default indexer, the error is generated.</span></span> <span data-ttu-id="1eeeb-113">您可以取消批註 `[DefaultMember("myItem")]` 行來解決錯誤。</span><span class="sxs-lookup"><span data-stu-id="1eeeb-113">You can address the error by uncommenting the line `[DefaultMember("myItem")]`.</span></span>
  
```csharp  
// CS0021.cs  
// compile with: /reference:CPP0021.dll  
public class MyClass  
{  
    public static void Main()  
    {  
        MyClassMC myMC = new MyClassMC();  
        int j = myMC[1]; // CS0021  
    }  
}  
```
