# <a name="compiler-error-cs1559"></a>編譯器錯誤 CS1559
'object' 是一個匯入物件，無法使用其 Main 方法  
  
 向 [/main](../language-reference/compiler-options/main-compiler-option.md) 編譯器選項指定的類別無效；類別不能當成 [Main](../programming-guide/main-and-command-args/index.md) 方法的位置使用。
