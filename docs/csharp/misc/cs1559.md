# <a name="compiler-error-cs1559"></a>編譯器錯誤 CS1559
'object' 是一個匯入物件，無法使用其 Main 方法  
  
 無效的類別指定要[/main](../../csharp/language-reference/compiler-options/main-compiler-option.md)編譯器選項的類別不能用做為位置[Main](../../csharp/programming-guide/main-and-command-args/index.md)方法。
