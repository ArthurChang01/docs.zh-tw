# <a name="compiler-warning-level-1-cs0602"></a>編譯器警告 (層級 1) CS0602
'old_feature' 功能已被取代。 請改用 'new_feature'  
  
 仍然支援在程式碼中使用的語言功能 (*old_feature*)，但該支援可能會在未來版本中移除。 您應該改用建議的語法 (*new_feature*)。
