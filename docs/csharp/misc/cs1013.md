# <a name="compiler-error-cs1013"></a>編譯器錯誤 CS1013
無效的數字  
  
 編譯器偵測到格式不正確的數字。 如需整數類型的詳細資訊，請參閱 [Integral Types Table](../../csharp/language-reference/keywords/integral-types-table.md)。  
  
## <a name="example"></a>範例  
 下列範例會產生 CS1013：  
  
```csharp  
// CS1013.cs  
class Sample  
{  
    static void Main()  
    {  
        int i = 0x;   // CS1013  
    }  
}  
```
