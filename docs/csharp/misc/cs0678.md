# <a name="compiler-error-cs0678"></a><span data-ttu-id="76032-102">編譯器錯誤 CS0678</span><span class="sxs-lookup"><span data-stu-id="76032-102">Compiler Error CS0678</span></span>

 <span data-ttu-id="76032-104">同時使用 [volatile](../language-reference/keywords/volatile.md) 和 [readonly](../language-reference/keywords/readonly.md) 關鍵字是互斥的。</span><span class="sxs-lookup"><span data-stu-id="76032-104">Use of the [volatile](../language-reference/keywords/volatile.md) and [readonly](../language-reference/keywords/readonly.md) keywords is mutually exclusive.</span></span>  
  
 <span data-ttu-id="76032-105">下列範例會產生 CS0678：</span><span class="sxs-lookup"><span data-stu-id="76032-105">The following sample generates CS0678:</span></span>  
  
```csharp  
// CS0678.cs

{  
   private readonly volatile int i;   // CS0678  
   // try the following line instead  
   // private volatile int i;

   {  
   }  
}  
```
