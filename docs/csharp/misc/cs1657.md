# <a name="compiler-error-cs1657"></a><span data-ttu-id="15b58-102">編譯器錯誤 CS1657</span><span class="sxs-lookup"><span data-stu-id="15b58-102">Compiler Error CS1657</span></span>
<span data-ttu-id="15b58-103">無法將 'parameter' 當做 ref 或 out 引數傳遞，因為 'reason'</span><span class="sxs-lookup"><span data-stu-id="15b58-103">Cannot pass 'parameter' as a ref or out argument because 'reason''</span></span>  
  
 <span data-ttu-id="15b58-104">在變數為唯讀的內容中，該變數當做 [ref](../../csharp/language-reference/keywords/ref.md) 或 [out](../../csharp/language-reference/keywords/out-parameter-modifier.md) 引數傳遞時，會發生這個錯誤。</span><span class="sxs-lookup"><span data-stu-id="15b58-104">This error occurs when a variable is passed as a [ref](../../csharp/language-reference/keywords/ref.md) or [out](../../csharp/language-reference/keywords/out-parameter-modifier.md) argument in a context in which that variable is readonly.</span></span> <span data-ttu-id="15b58-105">唯讀內容包括 [foreach](../../csharp/language-reference/keywords/foreach-in.md) 反覆運算變數、 [using](../../csharp/language-reference/keywords/using-statement.md) 變數和 `fixed` 變數。</span><span class="sxs-lookup"><span data-stu-id="15b58-105">Readonly contexts include [foreach](../../csharp/language-reference/keywords/foreach-in.md) iteration variables, [using](../../csharp/language-reference/keywords/using-statement.md) variables, and `fixed` variables.</span></span> <span data-ttu-id="15b58-106">若要解決這個錯誤，請不要在 `foreach`區塊、 `using` 陳述式和 `fixed` 陳述式中呼叫使用 `ref` 、 `out` 或 `using` 變數作為 `foreach` 或 `fixed` 參數的函式。</span><span class="sxs-lookup"><span data-stu-id="15b58-106">To resolve this error, do not call functions that take the `foreach`, `using` or `fixed` variable as a `ref` or `out` parameter in `using` blocks, `foreach` statements, and `fixed` statements.</span></span>  
  
## <a name="example"></a><span data-ttu-id="15b58-107">範例</span><span class="sxs-lookup"><span data-stu-id="15b58-107">Example</span></span>  
 <span data-ttu-id="15b58-108">下列範例會產生 CS1657：</span><span class="sxs-lookup"><span data-stu-id="15b58-108">The following sample generates CS1657:</span></span>  
  
```csharp  
// CS1657.cs  
using System;  
class C : IDisposable  
{  
    public int i;  
    public void Dispose() {}  
}  
  
class CMain  
{  
    static void f(ref C c)  
    {  
    }  
    static void Main()  
    {  
        using (C c = new C())  
        {  
            f(ref c);  // CS1657  
        }  
    }  
}  
```
