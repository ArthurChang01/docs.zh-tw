## <a name="example"></a><span data-ttu-id="b6873-108">範例</span><span class="sxs-lookup"><span data-stu-id="b6873-108">Example</span></span>  
 <span data-ttu-id="b6873-109">下列範例會產生 CS0403。</span><span class="sxs-lookup"><span data-stu-id="b6873-109">The following sample generates CS0403.</span></span>  
  
```csharp  
// CS0403.cs  
// compile with: /target:library  
class C<T>  
{  
   public void f()  
   {  
      T t = null;  // CS0403  
      T t2 = default(T);   // OK  
    }  
}  
  
class D<T> where T : class   
{  
   public void f()  
   {  
      T t = null;  // OK  
    }  
}  
```
