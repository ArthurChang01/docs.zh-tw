# <a name="compiler-error-cs1613"></a><span data-ttu-id="90f0e-102">編譯器錯誤 CS1613</span><span class="sxs-lookup"><span data-stu-id="90f0e-102">Compiler Error CS1613</span></span>
<span data-ttu-id="90f0e-103">找不到介面 'interface' 的 Managed coclass 包裝函式類別 'class' (是否遺漏了組件參考？)</span><span class="sxs-lookup"><span data-stu-id="90f0e-103">The managed coclass wrapper class 'class' for interface 'interface' cannot be found (are you missing an assembly reference?)</span></span>  
  
 <span data-ttu-id="90f0e-104">嘗試透過介面具現化 COM 物件。</span><span class="sxs-lookup"><span data-stu-id="90f0e-104">An attempt was made to instantiate a COM object from an interface.</span></span> <span data-ttu-id="90f0e-105">這個介面具有 **ComImport** 和 `CoClass` 屬性，但是編譯器找不到指定給 `CoClass` 屬性的類型。</span><span class="sxs-lookup"><span data-stu-id="90f0e-105">The interface has the **ComImport** and `CoClass` attributes, but the compiler cannot find the type given for the `CoClass` attribute.</span></span>  
  
 <span data-ttu-id="90f0e-106">若要解決這個錯誤，您可以嘗試下列其中一種方式：</span><span class="sxs-lookup"><span data-stu-id="90f0e-106">To resolve this error, you can try one of the following:</span></span>  
  
-   <span data-ttu-id="90f0e-107">加入具有 coclass 之組件的參考 (介面和 coclass 大部分的時間都應該在相同的組件中)。</span><span class="sxs-lookup"><span data-stu-id="90f0e-107">Add a reference to the assembly that has the coclass (most of the time the interface and coclass should be in the same assembly).</span></span> <span data-ttu-id="90f0e-108">請參閱[/參考](../../csharp/language-reference/compiler-options/reference-compiler-option.md)或[加入參考對話方塊](http://msdn.microsoft.com/library/2feb0fe2-0805-4cc9-8cba-b0315849dfb7)資訊。</span><span class="sxs-lookup"><span data-stu-id="90f0e-108">See [/reference](../../csharp/language-reference/compiler-options/reference-compiler-option.md) or [Add Reference Dialog Box](http://msdn.microsoft.com/library/2feb0fe2-0805-4cc9-8cba-b0315849dfb7) for information.</span></span>  
  
-   <span data-ttu-id="90f0e-109">修正介面上的 `CoClass` 屬性。</span><span class="sxs-lookup"><span data-stu-id="90f0e-109">Fix the `CoClass` attribute on the interface.</span></span>  
  
 <span data-ttu-id="90f0e-110">下列範例示範 **CoClassAttribute**的正確用法：</span><span class="sxs-lookup"><span data-stu-id="90f0e-110">The following sample demonstrates correct usage of **CoClassAttribute**:</span></span>  
  
```csharp  
// CS1613.cs  
using System;  
using System.Runtime.InteropServices;  
  
[Guid("1FFD7840-E82D-4268-875C-80A160C23296")]  
[ComImport()]  
[CoClass(typeof(A))]  
public interface IA{}  
public class A : IA {}  
  
public class AA  
{  
   public static void Main()  
   {  
      IA i;  
      i = new IA(); // This is equivalent to new A().  
                    // because of the CoClass attribute on IA  
   }  
}  
```
