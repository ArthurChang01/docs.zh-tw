# <a name="compiler-error-cs1918"></a><span data-ttu-id="56a9f-102">編譯器錯誤 CS1918</span><span class="sxs-lookup"><span data-stu-id="56a9f-102">Compiler Error CS1918</span></span>
<span data-ttu-id="56a9f-103">類型 'type' 且屬性為 'name' 的成員，無法以物件初始設定式進行指派，因為其為實值類型。</span><span class="sxs-lookup"><span data-stu-id="56a9f-103">Members of property 'name' of type 'type' cannot be assigned with an object initializer because it is of a value type.</span></span>  
  
 <span data-ttu-id="56a9f-104">如果您嘗試使用物件初始設定式所初始化的結構類型屬性，本身就是所初始化類別的屬性，會發生這個錯誤。</span><span class="sxs-lookup"><span data-stu-id="56a9f-104">This error occurs when you try to use an object initializer to initialize the properties of a struct type that is itself a property of the class that is being initialized.</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="56a9f-105">更正這個錯誤</span><span class="sxs-lookup"><span data-stu-id="56a9f-105">To correct this error</span></span>  
  
1. <span data-ttu-id="56a9f-106">如果您必須完整初始化物件初始設定式中屬性的欄位，請將結構變更為類別類型。</span><span class="sxs-lookup"><span data-stu-id="56a9f-106">If you must fully initialize the fields of the property in the object initializer, change the struct to a class type.</span></span> <span data-ttu-id="56a9f-107">否則，請在使用物件初始設定式建立物件之後，於不同的方法呼叫中初始化結構成員。</span><span class="sxs-lookup"><span data-stu-id="56a9f-107">Otherwise, initialize the struct members in a separate method call after you create the object by using the object initializer.</span></span>  
  
## <a name="example"></a><span data-ttu-id="56a9f-108">範例</span><span class="sxs-lookup"><span data-stu-id="56a9f-108">Example</span></span>  
 <span data-ttu-id="56a9f-109">下列範例會產生 CS1918：</span><span class="sxs-lookup"><span data-stu-id="56a9f-109">The following example generates CS1918:</span></span>  
  
```csharp  
// cs1918.cs  
public struct MyStruct  
{  
    public int i;  
  
}  
public class Test  
{  
    private MyStruct str = new MyStruct();  
    public MyStruct Str  
    {  
        get  
        {  
            return str;  
        }  
    }  
    public static int Main()  
    {  
        Test t = new Test { Str = { i = 1 } }; // CS1918  
        return 0;  
    }  
}  
```
