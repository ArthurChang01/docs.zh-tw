# <a name="compiler-error-cs0438"></a><span data-ttu-id="5fbf4-102">編譯器錯誤 CS0438</span><span class="sxs-lookup"><span data-stu-id="5fbf4-102">Compiler Error CS0438</span></span>
<span data-ttu-id="5fbf4-103">'module_1' 中的類型 'type' 與 'module_2' 中的命名空間 'namespace' 衝突。</span><span class="sxs-lookup"><span data-stu-id="5fbf4-103">The type 'type' in 'module_1' conflicts with the namespace 'namespace' in 'module_2'.</span></span>  
  
 <span data-ttu-id="5fbf4-104">當原始程式檔中的一個類型與另一個原始程式檔中的一個命名空間衝突時，就會發生這個錯誤。</span><span class="sxs-lookup"><span data-stu-id="5fbf4-104">This error occurs when a type in a source file conflicts with a namespace in another source file.</span></span> <span data-ttu-id="5fbf4-105">當其中一個或兩者來自加入的模組時，通常會發生這種情況。</span><span class="sxs-lookup"><span data-stu-id="5fbf4-105">This typically happens when one or both come from an added module.</span></span> <span data-ttu-id="5fbf4-106">若要解決，請重新命名造成衝突的類型或命名空間。</span><span class="sxs-lookup"><span data-stu-id="5fbf4-106">To resolve, rename the type or the namespace that caused the conflict.</span></span>  
  
 <span data-ttu-id="5fbf4-107">下列範例會產生 CS0438：</span><span class="sxs-lookup"><span data-stu-id="5fbf4-107">The following example generates CS0438:</span></span>  
  
 <span data-ttu-id="5fbf4-108">先編譯這個檔案：</span><span class="sxs-lookup"><span data-stu-id="5fbf4-108">Compile this file first:</span></span>  
  
```csharp  
// CS0438_1.cs  
// compile with: /target:module

{  
   public class A { }  
}  
```  
  
 <span data-ttu-id="5fbf4-109">然後，編譯這個檔案：</span><span class="sxs-lookup"><span data-stu-id="5fbf4-109">Then compile this file:</span></span>  
  
```csharp  
// CS0438_2.cs  
// compile with: /target:module  
namespace Util   
{  
   public class A { }  
}  
```  
  
 <span data-ttu-id="5fbf4-110">再編譯這個檔案：</span><span class="sxs-lookup"><span data-stu-id="5fbf4-110">And then compile this file:</span></span>  
  
```csharp  
// CS0438_3.cs  
// compile with: /addmodule:CS0438_1.netmodule /addmodule:CS0438_2.netmodule  
using System;  
public class Test  
{  
   public static void Main() {  
      Console.WriteLine(typeof(Util.A));   // CS0438  
   }  
}  
```
