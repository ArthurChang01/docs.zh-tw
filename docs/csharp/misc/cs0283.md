## <a name="example"></a><span data-ttu-id="86c84-106">範例</span><span class="sxs-lookup"><span data-stu-id="86c84-106">Example</span></span>  
 <span data-ttu-id="86c84-107">下列範例會產生 CS0283。</span><span class="sxs-lookup"><span data-stu-id="86c84-107">The following example generates CS0283.</span></span>  
  
```csharp  
// CS0283.cs

{  
}  
class MyClass   
{  
    // To resolve the error but retain the "const-ness",  
    // change const to readonly.  
    const MyTest test = new MyTest();   // CS0283  
  
    public static int Main() {  
        return 1;  
    }  
}  
```
