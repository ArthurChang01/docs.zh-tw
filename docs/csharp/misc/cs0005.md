# <a name="compiler-error-cs0005"></a>編譯器錯誤 CS0005
編譯器選項 'compiler_option' 後面必須使用引數  
  
 有一些編譯器選項需要參數。 如果未依編譯器選項傳遞需要的引數，就會產生 CS0005。  
  
 如需詳細資訊，請參閱 [C# 編譯器選項](../language-reference/compiler-options/index.md)。
