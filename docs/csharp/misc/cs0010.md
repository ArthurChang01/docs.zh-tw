# <a name="compiler-error-cs0010"></a>編譯器錯誤 CS0010
發生未預期的嚴重錯誤 -- 'error'。  
  
 當發生全然無法預料的狀況致使編譯停止時，會產生這個錯誤。  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
1.  請重新編譯另一個專案。 如果您收到相同的錯誤，請嘗試重新安裝 Visual Studio。 如果您收到這個錯誤，請將錯誤報告傳送給 Microsoft。
