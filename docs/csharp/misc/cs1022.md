# <a name="compiler-error-cs1022"></a>編譯器錯誤 CS1022
必須是類型或命名空間定義，或檔案結尾  
  
 原始程式碼檔沒有成對的大括弧。  
  
 下例會產生 CS1022：  
  
```csharp  
// CS1022.cs  
namespace x  
{  
}  
}   // CS1022  
```
