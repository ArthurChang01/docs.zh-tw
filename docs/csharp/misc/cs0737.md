 <span data-ttu-id="6ff58-105">實作介面成員的方法必須具有公用存取範圍。</span><span class="sxs-lookup"><span data-stu-id="6ff58-105">A method that implements an interface member must have public accessibility.</span></span> <span data-ttu-id="6ff58-106">所有介面成員都是 `public`。</span><span class="sxs-lookup"><span data-stu-id="6ff58-106">All interface members are `public`.</span></span>

## <a name="example"></a><span data-ttu-id="6ff58-109">範例</span><span class="sxs-lookup"><span data-stu-id="6ff58-109">Example</span></span>  
 <span data-ttu-id="6ff58-110">下列程式碼會產生 CS0737：</span><span class="sxs-lookup"><span data-stu-id="6ff58-110">The following code generates CS0737:</span></span>  
  
```csharp  
// cs0737.cs  
interface ITest  
{  
    // Default access of private with no modifier.  
    int Return42();  
    // Try the following line instead.  
    // public int Return42();  
}  
  
struct Struct1 : ITest // CS0737  
{  
    int Return42() { return (42); }  
}  
  
public class Test  
{  
    public static int Main(string[] args)  
    {  
        Struct1 s1 = new Struct1();  
  
        return (1);  
    }  
  
}  
```
