# <a name="compiler-error-cs1563"></a>編譯器錯誤 CS1563
輸出 'output file' 沒有任何原始程式檔  
  
 已指定[-out](../language-reference/compiler-options/out-compiler-option.md)編譯器選項，但未遵循任何原始程式碼檔。 您可以不使用 **/out**，或接著 **/out**指定原始程式碼檔。
