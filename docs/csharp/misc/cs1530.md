# <a name="compiler-error-cs1530"></a>編譯器錯誤 CS1530
在命名空間中定義的項目不允許關鍵字 'new'  
  
 不需要在 [命名空間](../../csharp/language-reference/keywords/new-modifier.md) 中的任何建構上指定 [new](../../csharp/language-reference/keywords/namespace.md)關鍵字。  
  
 下列範例會產生 CS1530：  
  
```csharp  
// CS1530.cs  
namespace a  
{  
   new class i   // CS1530  
   {  
   }  
  
   // try the following instead  
   class ii  
   {  
      public static void Main()  
      {  
      }  
   }  
}  
```
