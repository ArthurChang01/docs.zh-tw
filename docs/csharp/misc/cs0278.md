<span data-ttu-id="016b3-103">'type' 未實作 'pattern name' 模式。</span><span class="sxs-lookup"><span data-stu-id="016b3-103">'type' does not implement the 'pattern name' pattern.</span></span> <span data-ttu-id="016b3-104">'method name' 與 'method name' 模稜兩可。</span><span class="sxs-lookup"><span data-stu-id="016b3-104">'method name' is ambiguous with 'method name'.</span></span>

## <a name="example"></a><span data-ttu-id="016b3-110">範例</span><span class="sxs-lookup"><span data-stu-id="016b3-110">Example</span></span>  
 <span data-ttu-id="016b3-111">下列範例會產生 CS0278：</span><span class="sxs-lookup"><span data-stu-id="016b3-111">The following sample generates CS0278.</span></span>
