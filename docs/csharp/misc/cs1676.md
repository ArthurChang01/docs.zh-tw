# <a name="compiler-error-cs1676"></a><span data-ttu-id="a2523-102">編譯器錯誤 CS1676</span><span class="sxs-lookup"><span data-stu-id="a2523-102">Compiler Error CS1676</span></span>
<span data-ttu-id="a2523-103">參數 'number' 必須以 'keyword' 關鍵字宣告</span><span class="sxs-lookup"><span data-stu-id="a2523-103">Parameter 'number' must be declared with the 'keyword' keyword</span></span>  
  
 <span data-ttu-id="a2523-104">當匿名方法中的參數類型修飾詞和轉換方法的目標委派宣告中所使用的修飾詞不同時，就會發生這個錯誤。</span><span class="sxs-lookup"><span data-stu-id="a2523-104">This error occurs when the parameter type modifier in an anonymous method is different from that used in the declaration of the delegate you are casting the method to.</span></span>  
  
 <span data-ttu-id="a2523-105">下列範例會產生 CS1676：</span><span class="sxs-lookup"><span data-stu-id="a2523-105">The following sample generates CS1676:</span></span>  
  
```csharp  
// CS1676.cs  
delegate void E(ref int i);  
class Errors   
{  
   static void Main()  
   {  
      E e = delegate(out int i) { };   // CS1676  
      // To resolve, use the following line instead:  
      // E e = delegate(ref int i) { };  
   }  
}  
```
