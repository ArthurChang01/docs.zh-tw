# <a name="compiler-warning-level-2-cs0252"></a><span data-ttu-id="b0498-102">編譯器警告 (層級 2) CS0252</span><span class="sxs-lookup"><span data-stu-id="b0498-102">Compiler Warning (level 2) CS0252</span></span>
<span data-ttu-id="b0498-103">可能誤用了參考比較; 若您是要做數值比較，請將左邊轉換為類型 'type'</span><span class="sxs-lookup"><span data-stu-id="b0498-103">Possible unintended reference comparison; to get a value comparison, cast the left hand side to type 'type'</span></span>  
  
 <span data-ttu-id="b0498-104">編譯器正在進行參考比較。</span><span class="sxs-lookup"><span data-stu-id="b0498-104">The compiler is doing a reference comparison.</span></span> <span data-ttu-id="b0498-105">如果您想要比較字串值，請將運算式的左邊轉換為 `type`。</span><span class="sxs-lookup"><span data-stu-id="b0498-105">If you want to compare the value of strings, cast the left side of the expression to `type`.</span></span>  
  
 <span data-ttu-id="b0498-106">下列範例會產生 CS0252：</span><span class="sxs-lookup"><span data-stu-id="b0498-106">The following sample generates CS0252:</span></span>  
  
```csharp  
// CS0252.cs  
// compile with: /W:2  
using System;  
  
class MyClass  
{  
   public static void Main()  
   {  
      string s = "11";  
      object o = s + s;  
  
      bool b = o == s;   // CS0252  
      // try the following line instead  
      // bool b = (string)o == s;  
   }  
}  
```
