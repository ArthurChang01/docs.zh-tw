# <a name="compiler-error-cs0172"></a><span data-ttu-id="5fac6-102">編譯器錯誤 CS0172</span><span class="sxs-lookup"><span data-stu-id="5fac6-102">Compiler Error CS0172</span></span>
<span data-ttu-id="5fac6-103">無法判斷條件運算式的類型，因為 'type1' 和 'type2' 會互相隱含轉換。</span><span class="sxs-lookup"><span data-stu-id="5fac6-103">Type of conditional expression cannot be determined because 'type1' and 'type2' implicitly convert to one another</span></span>  
  
 <span data-ttu-id="5fac6-104">在條件陳述式中，您必須可以轉換 `:` 運算子任一側的類型。</span><span class="sxs-lookup"><span data-stu-id="5fac6-104">In a conditional statement, you must be able to convert the types on either side of the `:` operator.</span></span> <span data-ttu-id="5fac6-105">此外，不能有相互轉換常式；您只需要一個轉換。</span><span class="sxs-lookup"><span data-stu-id="5fac6-105">Also, there cannot be mutual conversion routines; you only need one conversion.</span></span> <span data-ttu-id="5fac6-106">如需詳細資訊，請參閱[轉換運算子](../../csharp/programming-guide/statements-expressions-operators/conversion-operators.md)。</span><span class="sxs-lookup"><span data-stu-id="5fac6-106">For more information, see [Conversion Operators](../../csharp/programming-guide/statements-expressions-operators/conversion-operators.md).</span></span>  
  
 <span data-ttu-id="5fac6-107">下列範例會產生 CS0172：</span><span class="sxs-lookup"><span data-stu-id="5fac6-107">The following sample generates CS0172:</span></span>  
  
```csharp  
// CS0172.cs  
public class Square  
{  
   public class Circle  
   {  
      public static implicit operator Circle(Square aa)  
      {  
         return null;  
      }  
  
      public static implicit operator Square(Circle aa)  
      // using explicit resolves this error  
      // public static explicit operator Square(Circle aa)  
      {  
         return null;  
      }  
   }  
  
   public static void Main()  
   {  
      Circle aa = new Circle();  
      Square ii = new Square();  
      object o = (1 == 1) ? aa : ii;   // CS0172  
      // the following cast would resolve this error  
      // (1 == 1) ? aa : (Circle)ii;  
   }  
}  
```
