<span data-ttu-id="53902-103">無法在匿名方法、Lambda 運算式或查詢運算式內使用 ref 或 out 參數 'parameter'</span><span class="sxs-lookup"><span data-stu-id="53902-103">Cannot use ref or out parameter 'parameter' inside an anonymous method, lambda expression, or query expression</span></span>  
  
 <span data-ttu-id="53902-104">如果您在匿名方法區塊內使用 ref 或 out 參數，就會發生此錯誤。</span><span class="sxs-lookup"><span data-stu-id="53902-104">This error occurs if you use a ref or out parameter inside an anonymous method block.</span></span> <span data-ttu-id="53902-105">要避免這個錯誤，請使用區域變數或其他建構。</span><span class="sxs-lookup"><span data-stu-id="53902-105">To avoid this error, use a local variable or some other construct.</span></span>  
  
 <span data-ttu-id="53902-106">下列範例會產生 C1628：</span><span class="sxs-lookup"><span data-stu-id="53902-106">The following sample generates CS1628:</span></span>  
  
```  
// CS1628.cs  
  
delegate int MyDelegate();  
  
class C  
{  
  public static void F(ref int i)  
  {  
      MyDelegate d = delegate { return i; };  // CS1628  
      // Try this instead:  
      // int tmp = i;  
      // MyDelegate d = delegate { return tmp; };  
  }  
  
  public static void Main()  
  {  
  
  }  
}  
```
