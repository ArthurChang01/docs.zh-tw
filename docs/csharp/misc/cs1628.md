<span data-ttu-id="16cda-103">無法使用 ref 或 out 參數 'parameter' 在匿名方法、 lambda 運算式或查詢運算式</span><span class="sxs-lookup"><span data-stu-id="16cda-103">Cannot use in ref or out parameter 'parameter' inside an anonymous method, lambda expression, or query expression</span></span>  
  
 <span data-ttu-id="16cda-104">如果您使用，就會發生這個錯誤`in`， `ref`，或`out`匿名方法區塊內的參數。</span><span class="sxs-lookup"><span data-stu-id="16cda-104">This error occurs if you use an `in`, `ref`, or `out` parameter inside an anonymous method block.</span></span> <span data-ttu-id="16cda-105">要避免這個錯誤，請使用區域變數或其他建構。</span><span class="sxs-lookup"><span data-stu-id="16cda-105">To avoid this error, use a local variable or some other construct.</span></span>  
  
 <span data-ttu-id="16cda-106">下列範例會產生 C1628：</span><span class="sxs-lookup"><span data-stu-id="16cda-106">The following sample generates CS1628:</span></span>  
  
```csharp  
// CS1628.cs

class C  
{  
  public static void F(ref int i)  
  {  
      MyDelegate d = delegate { return i; };  // CS1628  
      // Try this instead:  
      // int tmp = i;  
      // MyDelegate d = delegate { return tmp; };  
  }  
  
  public static void Main()  
  {  
  
  }  
}  
```
