# <a name="compiler-error-cs0587"></a>編譯器錯誤 CS0587
編譯器內部錯誤：階段 'stage'  
  
 請嘗試判斷編譯器失敗的原因是否為無法剖析未預期的語法。 如果這不是這樣，請嘗試[告訴我們](/visualstudio/ide/talk-to-us)。
