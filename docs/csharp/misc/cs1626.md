# <a name="compiler-error-cs1626"></a><span data-ttu-id="7c602-102">編譯器錯誤 CS1626</span><span class="sxs-lookup"><span data-stu-id="7c602-102">Compiler Error CS1626</span></span>
<span data-ttu-id="7c602-103">在具有 catch 子句的 try 區塊主體中不可使用 yield 產生值</span><span class="sxs-lookup"><span data-stu-id="7c602-103">Cannot yield a value in the body of a try block with a catch clause</span></span>  
  
 <span data-ttu-id="7c602-104">如果有與 try 區塊相關聯的 catch 子句，則 try 區塊中不允許 yield 陳述式。</span><span class="sxs-lookup"><span data-stu-id="7c602-104">A yield statement is not allowed in a try block if there is a catch clause associated with the try block.</span></span> <span data-ttu-id="7c602-105">若要避免這個錯誤，請將 yield 陳述式移出 catch 子句。</span><span class="sxs-lookup"><span data-stu-id="7c602-105">To avoid this error, move the yield statement out of the catch clause.</span></span>  
  
 <span data-ttu-id="7c602-106">下列範例會產生 CS1626：</span><span class="sxs-lookup"><span data-stu-id="7c602-106">The following sample generates CS1626:</span></span>  
  
```  
// CS1626.cs  
using System.Collections;  
  
class C : IEnumerable  
{  
   public IEnumerator GetEnumerator()  
   {  
      try  
      {  
         yield return this;  // CS1626  
      }  
      catch  
      {  
  
      }  
   }  
}  
  
public class CMain  
{  
   public static void Main() { }  
}  
```
