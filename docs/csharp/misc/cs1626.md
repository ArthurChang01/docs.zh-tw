# <a name="compiler-error-cs1626"></a><span data-ttu-id="e5fb9-102">編譯器錯誤 CS1626</span><span class="sxs-lookup"><span data-stu-id="e5fb9-102">Compiler Error CS1626</span></span>
<span data-ttu-id="e5fb9-103">在具有 catch 子句的 try 區塊主體中不可使用 yield 產生值</span><span class="sxs-lookup"><span data-stu-id="e5fb9-103">Cannot yield a value in the body of a try block with a catch clause</span></span>  
  
 <span data-ttu-id="e5fb9-104">如果有與 try 區塊相關聯的 catch 子句，則 try 區塊中不允許 yield 陳述式。</span><span class="sxs-lookup"><span data-stu-id="e5fb9-104">A yield statement is not allowed in a try block if there is a catch clause associated with the try block.</span></span> <span data-ttu-id="e5fb9-105">為了避免此錯誤，請將屈服語句移出 try/catch/finally 塊，或刪除 catch 塊。</span><span class="sxs-lookup"><span data-stu-id="e5fb9-105">To avoid this error, either move the yield statement out of the try/catch/finally block, or remove the catch block.</span></span>
  
 <span data-ttu-id="e5fb9-106">下列範例會產生 CS1626：</span><span class="sxs-lookup"><span data-stu-id="e5fb9-106">The following sample generates CS1626:</span></span>  
  
```csharp  
// CS1626.cs  
using System.Collections;  
  
class C : IEnumerable  
{  
   public IEnumerator GetEnumerator()  
   {  
      try  
      {  
         yield return this;  // CS1626  
      }  
      catch  
      {  
  
      }  
      finally
      {

      }
   }  
}  
  
public class CMain  
{  
   public static void Main() { }  
}  
```
