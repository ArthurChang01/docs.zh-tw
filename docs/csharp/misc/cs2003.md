# <a name="compiler-error-cs2003"></a>編譯器錯誤 CS2003
回應檔 'file' 已經加入多次  
  
 回應檔已傳遞多次給編譯器。 每個輸出檔只能傳遞一次回應檔給編譯器。  
  
 如需回應檔的詳細資訊，請參閱 [@ (指定回應檔)](../../csharp/language-reference/compiler-options/response-file-compiler-option.md)。
