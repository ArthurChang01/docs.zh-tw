# <a name="compiler-error-cs0199"></a><span data-ttu-id="e1d3f-102">編譯器錯誤 CS0199</span><span class="sxs-lookup"><span data-stu-id="e1d3f-102">Compiler Error CS0199</span></span>
<span data-ttu-id="e1d3f-103">不能傳遞靜態唯讀欄位 'name' 的欄位給 ref 或 out (除非在靜態建構函式中)</span><span class="sxs-lookup"><span data-stu-id="e1d3f-103">Fields of static readonly field 'name' cannot be passed ref or out (except in a static constructor)</span></span>  
  
 <span data-ttu-id="e1d3f-104">A [readonly](../../csharp/language-reference/keywords/readonly.md)變數必須具有相同[靜態](../../csharp/language-reference/keywords/static.md)與您要將它傳遞為建構函式的使用方式[ref](../../csharp/language-reference/keywords/ref.md)或[出](../../csharp/language-reference/keywords/out-parameter-modifier.md)參數。</span><span class="sxs-lookup"><span data-stu-id="e1d3f-104">A [readonly](../../csharp/language-reference/keywords/readonly.md) variable must have the same [static](../../csharp/language-reference/keywords/static.md) usage as the constructor in which you want to pass it as a [ref](../../csharp/language-reference/keywords/ref.md) or [out](../../csharp/language-reference/keywords/out-parameter-modifier.md) parameter.</span></span> <span data-ttu-id="e1d3f-105">如需詳細資訊，請參閱[傳遞參數](../../csharp/programming-guide/classes-and-structs/passing-parameters.md)。</span><span class="sxs-lookup"><span data-stu-id="e1d3f-105">For more information, see [Passing Parameters](../../csharp/programming-guide/classes-and-structs/passing-parameters.md).</span></span>  
  
## <a name="example"></a><span data-ttu-id="e1d3f-106">範例</span><span class="sxs-lookup"><span data-stu-id="e1d3f-106">Example</span></span>  
 <span data-ttu-id="e1d3f-107">下列範例會產生 CS0199：</span><span class="sxs-lookup"><span data-stu-id="e1d3f-107">The following sample generates CS0199:</span></span>  
  
```csharp  
// CS0199.cs  
class MyClass  
{  
    public static readonly int TestInt = 6;  
  
    static void TestMethod(ref int testInt)  
    {  
        testInt = 0;  
    }  
  
    MyClass()  
    {  
        TestMethod(ref TestInt);   // CS0199, TestInt is static  
    }  
  
    public static void Main()  
    {  
    }  
}  
```
