<span data-ttu-id="1a026-103">不能傳遞靜態唯讀欄位 'name' 的欄位給 ref 或 out (除非在靜態建構函式中)</span><span class="sxs-lookup"><span data-stu-id="1a026-103">Fields of static readonly field 'name' cannot be passed ref or out (except in a static constructor)</span></span>  
  
 <span data-ttu-id="1a026-104">[readonly](../../csharp/language-reference/keywords/readonly.md) 變數的 [static](../../csharp/language-reference/keywords/static.md) 使用情形必須與建構函式相同，而在此建構函式中，您想要將它傳遞為 [ref](../../csharp/language-reference/keywords/ref.md) 或 [out](../../csharp/language-reference/keywords/out.md) 參數。</span><span class="sxs-lookup"><span data-stu-id="1a026-104">A [readonly](../../csharp/language-reference/keywords/readonly.md) variable must have the same [static](../../csharp/language-reference/keywords/static.md) usage as the constructor in which you want to pass it as a [ref](../../csharp/language-reference/keywords/ref.md) or [out](../../csharp/language-reference/keywords/out.md) parameter.</span></span> <span data-ttu-id="1a026-105">如需詳細資訊，請參閱[傳遞參數](../../csharp/programming-guide/classes-and-structs/passing-parameters.md)。</span><span class="sxs-lookup"><span data-stu-id="1a026-105">For more information, see [Passing Parameters](../../csharp/programming-guide/classes-and-structs/passing-parameters.md).</span></span>  
  
## <a name="example"></a><span data-ttu-id="1a026-106">範例</span><span class="sxs-lookup"><span data-stu-id="1a026-106">Example</span></span>  
 <span data-ttu-id="1a026-107">下列範例會產生 CS0199：</span><span class="sxs-lookup"><span data-stu-id="1a026-107">The following sample generates CS0199:</span></span>  
  
```  
// CS0199.cs  
class MyClass  
{  
    public static readonly int TestInt = 6;  
  
    static void TestMethod(ref int testInt)  
    {  
        testInt = 0;  
    }  
  
    MyClass()  
    {  
        TestMethod(ref TestInt);   // CS0199, TestInt is static  
    }  
  
    public static void Main()  
    {  
    }  
}  
```
