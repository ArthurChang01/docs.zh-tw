# <a name="compiler-error-cs0199"></a>編譯器錯誤 CS0199
不能傳遞靜態唯讀欄位 'name' 的欄位給 ref 或 out (除非在靜態建構函式中)  
  
 [readonly](../../csharp/language-reference/keywords/readonly.md) 變數的 [static](../../csharp/language-reference/keywords/static.md) 使用情形必須與建構函式相同，而在此建構函式中，您想要將它傳遞為 [ref](../../csharp/language-reference/keywords/ref.md) 或 [out](../../csharp/language-reference/keywords/out-parameter-modifier.md) 參數。 如需詳細資訊，請參閱[傳遞參數](../../csharp/programming-guide/classes-and-structs/passing-parameters.md)。  
  
## <a name="example"></a>範例  
 下列範例會產生 CS0199：  
  
```csharp  
// CS0199.cs  
class MyClass  
{  
    public static readonly int TestInt = 6;  
  
    static void TestMethod(ref int testInt)  
    {  
        testInt = 0;  
    }  
  
    MyClass()  
    {  
        TestMethod(ref TestInt);   // CS0199, TestInt is static  
    }  
  
    public static void Main()  
    {  
    }  
}  
```
