# <a name="compiler-error-cs0685"></a><span data-ttu-id="2c77f-102">編譯器錯誤 CS0685</span><span class="sxs-lookup"><span data-stu-id="2c77f-102">Compiler Error CS0685</span></span>
<span data-ttu-id="2c77f-103">Conditional 成員 'member' 不能有 out 參數</span><span class="sxs-lookup"><span data-stu-id="2c77f-103">Conditional member 'member' cannot have an out parameter</span></span>  
  
 <span data-ttu-id="2c77f-104">在方法上使用 <xref:System.Diagnostics.ConditionalAttribute> 屬性時，該方法可能沒有 out 參數。</span><span class="sxs-lookup"><span data-stu-id="2c77f-104">When using the <xref:System.Diagnostics.ConditionalAttribute> attribute on a method, that method may not have an out parameter.</span></span> <span data-ttu-id="2c77f-105">這是因為在方法呼叫編譯為 Nothing 的情況下，不會定義用於 out 參數的變數值。</span><span class="sxs-lookup"><span data-stu-id="2c77f-105">This is because the value of the variable used for the out parameter would not be defined in the case that the method call is compiled to nothing.</span></span> <span data-ttu-id="2c77f-106">若要避免這個錯誤，請從條件式方法宣告中移除 out 參數，或不要使用條件式屬性。</span><span class="sxs-lookup"><span data-stu-id="2c77f-106">To avoid this error, remove the out parameter from the conditional method declaration, or don't use the Conditional Attribute.</span></span>  
  
## <a name="example"></a><span data-ttu-id="2c77f-107">範例</span><span class="sxs-lookup"><span data-stu-id="2c77f-107">Example</span></span>  
 <span data-ttu-id="2c77f-108">下列範例會產生 CS0685：</span><span class="sxs-lookup"><span data-stu-id="2c77f-108">The following sample generates CS0685:</span></span>  
  
```csharp  
// CS0685.cs  
using System.Diagnostics;  
  
class C  
{  
    [Conditional("DEBUG")]  
    void trace(out int i)  // CS0685  
    {  
        i = 1;  
    }  
}  
```
