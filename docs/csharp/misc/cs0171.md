# <a name="compiler-error-cs0171"></a><span data-ttu-id="17e63-102">編譯器錯誤 CS0171</span><span class="sxs-lookup"><span data-stu-id="17e63-102">Compiler Error CS0171</span></span>
<span data-ttu-id="17e63-103">程式控制權回到呼叫端前，必須完整指派自動實作的屬性 'name' 的支援欄位。</span><span class="sxs-lookup"><span data-stu-id="17e63-103">Backing field for automatically implemented property 'name' must be fully assigned before control is returned to the caller.</span></span> <span data-ttu-id="17e63-104">請考慮從「函式初始化運算式」呼叫無參數的函式。</span><span class="sxs-lookup"><span data-stu-id="17e63-104">Consider calling the parameterless constructor from a constructor initializer.</span></span>  
  
 <span data-ttu-id="17e63-105">[結構](../language-reference/keywords/struct.md) 中的建構函式必須初始化該結構中的所有欄位。</span><span class="sxs-lookup"><span data-stu-id="17e63-105">A constructor in a [struct](../language-reference/keywords/struct.md) must initialize all fields in the struct.</span></span> <span data-ttu-id="17e63-106">如需詳細資訊，請參閱[建構函式](../programming-guide/classes-and-structs/constructors.md)。</span><span class="sxs-lookup"><span data-stu-id="17e63-106">For more information, see [Constructors](../programming-guide/classes-and-structs/constructors.md).</span></span>  
  
 <span data-ttu-id="17e63-107">下列範例會產生 CS0171：</span><span class="sxs-lookup"><span data-stu-id="17e63-107">The following sample generates CS0171:</span></span>  
  
```csharp  
// CS0171.cs  
struct MyStruct  
{  
   MyStruct(int initField)   // CS0171  
   {  
      // i = initField;      // uncomment this line to resolve this error  
   }  
   public int i;  
}  
  
class MyClass  
{  
   public static void Main()  
   {  
      MyStruct aStruct = new MyStruct();  
   }  
}  
```
