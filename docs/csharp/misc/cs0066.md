 <span data-ttu-id="a7d5f-104">event 關鍵字需要 [委派](../../csharp/language-reference/keywords/delegate.md) 類型。</span><span class="sxs-lookup"><span data-stu-id="a7d5f-104">The event keyword requires a [delegate](../../csharp/language-reference/keywords/delegate.md) type.</span></span> <span data-ttu-id="a7d5f-105">如需詳細資訊，請參閱 <<c0> [ 事件](../../csharp/programming-guide/events/index.md)並[委派](../../csharp/programming-guide/delegates/index.md)。</span><span class="sxs-lookup"><span data-stu-id="a7d5f-105">For more information, see [Events](../../csharp/programming-guide/events/index.md) and [Delegates](../../csharp/programming-guide/delegates/index.md).</span></span>  
  
 <span data-ttu-id="a7d5f-106">下列範例會產生 CS0066：</span><span class="sxs-lookup"><span data-stu-id="a7d5f-106">The following sample generates CS0066:</span></span>  
  
```csharp  
// CS0066.cs  
using System;  
  
public class EventHandler  
{  
}  
  
// to fix the error, remove the event declaration and the  
// EventHandler class declaration, and uncomment the following line  
// public delegate void EventHandler();  
  
public class a  
{  
   public event EventHandler Click;   // CS0066  
  
   private void TestMethod()  
   {  
      if (Click != null)  
         Click();  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
