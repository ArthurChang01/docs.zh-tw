# <a name="compiler-error-cs0066"></a><span data-ttu-id="d65ee-102">編譯器錯誤 CS0066</span><span class="sxs-lookup"><span data-stu-id="d65ee-102">Compiler Error CS0066</span></span>
<span data-ttu-id="d65ee-103">'event': 事件必須為委派類型</span><span class="sxs-lookup"><span data-stu-id="d65ee-103">'event': event must be of a delegate type</span></span>  
  
 <span data-ttu-id="d65ee-104">event 關鍵字需要 [委派](../language-reference/builtin-types/reference-types.md) 類型。</span><span class="sxs-lookup"><span data-stu-id="d65ee-104">The event keyword requires a [delegate](../language-reference/builtin-types/reference-types.md) type.</span></span> <span data-ttu-id="d65ee-105">如需詳細資訊，請參閱[事件](../programming-guide/events/index.md)和[委派](../programming-guide/delegates/index.md)。</span><span class="sxs-lookup"><span data-stu-id="d65ee-105">For more information, see [Events](../programming-guide/events/index.md) and [Delegates](../programming-guide/delegates/index.md).</span></span>  
  
 <span data-ttu-id="d65ee-106">下列範例會產生 CS0066：</span><span class="sxs-lookup"><span data-stu-id="d65ee-106">The following sample generates CS0066:</span></span>  
  
```csharp  
// CS0066.cs  
using System;  
  
public class EventHandler  
{  
}  
  
// to fix the error, remove the event declaration and the  
// EventHandler class declaration, and uncomment the following line  
// public delegate void EventHandler();  
  
public class a  
{  
   public event EventHandler Click;   // CS0066  
  
   private void TestMethod()  
   {  
      if (Click != null)  
         Click();  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
