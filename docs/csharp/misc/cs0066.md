# <a name="compiler-error-cs0066"></a>編譯器錯誤 CS0066
'event': 事件必須為委派類型  
  
 event 關鍵字需要 [委派](../../csharp/language-reference/keywords/delegate.md) 類型。 如需詳細資訊，請參閱 <<c0> [ 事件](../../csharp/programming-guide/events/index.md)並[委派](../../csharp/programming-guide/delegates/index.md)。  
  
 下列範例會產生 CS0066：  
  
```csharp  
// CS0066.cs  
using System;  
  
public class EventHandler  
{  
}  
  
// to fix the error, remove the event declaration and the  
// EventHandler class declaration, and uncomment the following line  
// public delegate void EventHandler();  
  
public class a  
{  
   public event EventHandler Click;   // CS0066  
  
   private void TestMethod()  
   {  
      if (Click != null)  
         Click();  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
