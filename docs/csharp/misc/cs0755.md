# <a name="compiler-error-cs0755"></a><span data-ttu-id="edc30-102">編譯器錯誤 CS0755</span><span class="sxs-lookup"><span data-stu-id="edc30-102">Compiler Error CS0755</span></span>
<span data-ttu-id="edc30-103">兩個部分方法宣告必須都是擴充方法，或者都不是擴充方法。</span><span class="sxs-lookup"><span data-stu-id="edc30-103">Both partial method declarations must be extension methods or neither may be an extension method.</span></span>  
  
 <span data-ttu-id="edc30-104">部分方法是由定義宣告 (特徵標記) 和選擇性的實作宣告 (主體) 所組成。</span><span class="sxs-lookup"><span data-stu-id="edc30-104">A partial method consists of a defining declaration (signature) and an optional implementing declaration (body).</span></span> <span data-ttu-id="edc30-105">如果定義宣告是擴充方法，則實作宣告，如果有定義，也必須是擴充方法。</span><span class="sxs-lookup"><span data-stu-id="edc30-105">If the defining declaration is an extension method, the implementing declaration, if one is defined, must also be an extension method.</span></span> <span data-ttu-id="edc30-106">如果定義方法不是擴充方法，實作也不得是擴充方法。</span><span class="sxs-lookup"><span data-stu-id="edc30-106">And if the defining method is not an extension method, the implementing must not be one either.</span></span>

## <a name="example"></a><span data-ttu-id="edc30-109">範例</span><span class="sxs-lookup"><span data-stu-id="edc30-109">Example</span></span>  
 <span data-ttu-id="edc30-110">下列範例會產生 CS0755：</span><span class="sxs-lookup"><span data-stu-id="edc30-110">The following example generates CS0755:</span></span>  
  
```csharp  
// cs0755.cs  
    public static partial class Ext  
    {  
        static partial void Part(this C c); //Extension method  
  
        // Typically the implementing declaration is in a separate file.  
        static partial void Part(C c) //CS0755  
        {  
        }  
    }  
  
    public partial class C  
    {  
        public static int Main()  
        {  
            return 1;  
        }  
    }  
```
