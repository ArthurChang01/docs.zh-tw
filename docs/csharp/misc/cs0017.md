<span data-ttu-id="3f721-103">程式 'output file name' 有一個以上的已定義進入點。</span><span class="sxs-lookup"><span data-stu-id="3f721-103">Program 'output file name' has more than one entry point defined.</span></span> <span data-ttu-id="3f721-104">請以 /main 編譯以指定包含進入點的類型。</span><span class="sxs-lookup"><span data-stu-id="3f721-104">Compile with /main to specify the type that contains the entry point.</span></span>  
  
 <span data-ttu-id="3f721-105">程式只能有一個 [Main](../programming-guide/main-and-command-args/index.md) 方法。</span><span class="sxs-lookup"><span data-stu-id="3f721-105">A program can only have one [Main](../programming-guide/main-and-command-args/index.md) method.</span></span>  
  
 <span data-ttu-id="3f721-106">若要解決這個錯誤，您可以刪除程式碼中的所有 Main 方法（其中一個除外），也可以使用[-Main](../language-reference/compiler-options/main-compiler-option.md)編譯器選項來指定您想要使用的主要方法。</span><span class="sxs-lookup"><span data-stu-id="3f721-106">To resolve this error, you can either delete all Main methods in your code, except one, or you can use the [-main](../language-reference/compiler-options/main-compiler-option.md) compiler option to specify which Main method you want to use.</span></span>  
  
 <span data-ttu-id="3f721-107">下列範例會產生 CS0017：</span><span class="sxs-lookup"><span data-stu-id="3f721-107">The following sample generates CS0017:</span></span>  
  
```csharp

{  
   static public void Main()  
   {  
   }  
}  
  
public class cly  
{  
   public static void Main()   // CS0017, delete one Main or use /main  
   {  
   }  
}  
```
