<span data-ttu-id="72584-103">程式 'output file name' 有一個以上的已定義進入點。</span><span class="sxs-lookup"><span data-stu-id="72584-103">Program 'output file name' has more than one entry point defined.</span></span> <span data-ttu-id="72584-104">請以 /main 編譯以指定包含進入點的類型。</span><span class="sxs-lookup"><span data-stu-id="72584-104">Compile with /main to specify the type that contains the entry point.</span></span>  
  
 <span data-ttu-id="72584-105">程式只能有一個[Main](../../csharp/programming-guide/main-and-command-args/index.md)方法。</span><span class="sxs-lookup"><span data-stu-id="72584-105">A program can only have one [Main](../../csharp/programming-guide/main-and-command-args/index.md) method.</span></span>  
  
 <span data-ttu-id="72584-106">若要解決此錯誤，您可以刪除所有的 Main 方法中程式碼，只保留一個，或您可以使用[/main](../../csharp/language-reference/compiler-options/main-compiler-option.md)編譯器選項以指定您想要使用哪一個 Main 方法。</span><span class="sxs-lookup"><span data-stu-id="72584-106">To resolve this error, you can either delete all Main methods in your code, except one, or you can use the [/main](../../csharp/language-reference/compiler-options/main-compiler-option.md) compiler option to specify which Main method you want to use.</span></span>  
  
 <span data-ttu-id="72584-107">下列範例會產生 CS0017：</span><span class="sxs-lookup"><span data-stu-id="72584-107">The following sample generates CS0017:</span></span>  
  
```  
// CS0017.cs  
// compile with: /target:exe  
public class clx  
{  
   static public void Main()  
   {  
   }  
}  
  
public class cly  
{  
   public static void Main()   // CS0017, delete one Main or use /main  
   {  
   }  
}  
```
