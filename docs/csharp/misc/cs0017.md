<span data-ttu-id="e1ca7-103">程式 'output file name' 有一個以上的已定義進入點。</span><span class="sxs-lookup"><span data-stu-id="e1ca7-103">Program 'output file name' has more than one entry point defined.</span></span> <span data-ttu-id="e1ca7-104">請以 /main 編譯以指定包含進入點的類型。</span><span class="sxs-lookup"><span data-stu-id="e1ca7-104">Compile with /main to specify the type that contains the entry point.</span></span>  
  
 <span data-ttu-id="e1ca7-105">程式只能有一個 [Main](../programming-guide/main-and-command-args/index.md) 方法。</span><span class="sxs-lookup"><span data-stu-id="e1ca7-105">A program can only have one [Main](../programming-guide/main-and-command-args/index.md) method.</span></span>  
  
 <span data-ttu-id="e1ca7-106">若要解決這個錯誤，您可以刪除程式碼中的所有 Main 方法，只保留一個，或使用 [/main](../language-reference/compiler-options/main-compiler-option.md) 編譯器選項，指定要使用的 Main 方法。</span><span class="sxs-lookup"><span data-stu-id="e1ca7-106">To resolve this error, you can either delete all Main methods in your code, except one, or you can use the [/main](../language-reference/compiler-options/main-compiler-option.md) compiler option to specify which Main method you want to use.</span></span>  
  
 <span data-ttu-id="e1ca7-107">下列範例會產生 CS0017：</span><span class="sxs-lookup"><span data-stu-id="e1ca7-107">The following sample generates CS0017:</span></span>  
  
```csharp  
// CS0017.cs  
// compile with: /target:exe

{  
   static public void Main()  
   {  
   }  
}  
  
public class cly  
{  
   public static void Main()   // CS0017, delete one Main or use /main  
   {  
   }  
}  
```
