 <span data-ttu-id="611fd-106">若要解決這個錯誤，您可以刪除程式碼中的所有 Main 方法，只保留一個，或使用 [/main](../../csharp/language-reference/compiler-options/main-compiler-option.md) 編譯器選項，指定要使用的 Main 方法。</span><span class="sxs-lookup"><span data-stu-id="611fd-106">To resolve this error, you can either delete all Main methods in your code, except one, or you can use the [/main](../../csharp/language-reference/compiler-options/main-compiler-option.md) compiler option to specify which Main method you want to use.</span></span>  
  
 <span data-ttu-id="611fd-107">下列範例會產生 CS0017：</span><span class="sxs-lookup"><span data-stu-id="611fd-107">The following sample generates CS0017:</span></span>

```  
{  
   static public void Main()  
   {  
   }  
}

{  
   public static void Main()   // CS0017, delete one Main or use /main  
   {  
   }  
}  
```
