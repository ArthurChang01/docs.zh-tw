# <a name="compiler-error-cs0107"></a>編譯器錯誤 CS0107
有一個以上的保護修飾詞  
  
 在類別成員上只允許一個存取修飾詞 ([public](../language-reference/keywords/public.md)、 [private](../language-reference/keywords/private.md)、 [protected](../language-reference/keywords/protected.md)或 [internal](../language-reference/keywords/internal.md))，但不包括 **internal protected**。  
  
 下列範例會產生 CS0107：  
  
```csharp  
// CS0107.cs  
public class C  
{  
   private internal void f()   // CS0107, delete private or internal  
   {  
   }  
  
   public static int Main()  
   {  
      return 1;  
   }  
}  
```
