# <a name="compiler-error-cs0042"></a>編譯器錯誤 CS0042
建立偵錯資訊檔 'file' 時發生未預期的錯誤 — 'reason'  
  
 編譯器無法建立偵錯資訊； *reason* 包含錯誤狀況的其他資訊。
