# <a name="compiler-warning-level-3-cs0282"></a><span data-ttu-id="b4115-102">編譯器警告 (層級 3) CS0282</span><span class="sxs-lookup"><span data-stu-id="b4115-102">Compiler Warning (level 3) CS0282</span></span>
