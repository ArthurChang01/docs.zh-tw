# 編譯器錯誤 CS0174
'base' 參考需要基底類別  
  
 只有在 .NET Framework Common Language Runtime 編譯 `System.Object` 類別 \(這是唯一沒有基底類別的類別\) 的原始程式碼時，才會發生這個錯誤。
