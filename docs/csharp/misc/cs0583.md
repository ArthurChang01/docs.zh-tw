# <a name="compiler-error-cs0583"></a>編譯器錯誤 CS0583
編譯器內部錯誤。 編譯器中發生內部錯誤。 若要解決這個問題，請嘗試簡化或變更下方所列位置附近的程式。 清單頂端的位置較接近內部錯誤發生的點。 您可以使用 /errorreport 選項，將這類錯誤回報給 Microsoft。  
  
 請使用 '/bugreport' 選項建立錯誤報告檔案 ，並將報告提交給您指定的問題報告連絡人。  
  
 使用 [/bugreport](../../csharp/language-reference/compiler-options/bugreport-compiler-option.md) 編譯器選項，來擷取問題。
