# <a name="compiler-error-cs0065"></a><span data-ttu-id="38eca-102">編譯器錯誤 CS0065</span><span class="sxs-lookup"><span data-stu-id="38eca-102">Compiler Error CS0065</span></span>

<span data-ttu-id="38eca-103">'event': event 屬性必須同時有 add 和 remove 存取子</span><span class="sxs-lookup"><span data-stu-id="38eca-103">'event' : event property must have both add and remove accessors</span></span>

<span data-ttu-id="38eca-104">不是欄位的事件必須有這兩種存取方法。</span><span class="sxs-lookup"><span data-stu-id="38eca-104">An event that is not a field must have both access methods.</span></span>

<span data-ttu-id="38eca-105">下列範例會產生 CS0065：</span><span class="sxs-lookup"><span data-stu-id="38eca-105">The following sample generates CS0065:</span></span>

```csharp
// CS0065.cs
using System;
public delegate void EventHandler(object sender, int e);
public class MyClass
{
   public event EventHandler Click   // CS0065,
   {
      // to fix, uncomment the add and remove definitions
      /*
      add
      {
         Click += value;
      }
      remove
      {
         Click -= value;
      }
      */
   }

   public static void Main()
   {
   }
}
```
