# <a name="compiler-error-cs0065"></a><span data-ttu-id="f228c-102">編譯器錯誤 CS0065</span><span class="sxs-lookup"><span data-stu-id="f228c-102">Compiler Error CS0065</span></span>
<span data-ttu-id="f228c-103">'event': event 屬性必須同時有 add 和 remove 存取子</span><span class="sxs-lookup"><span data-stu-id="f228c-103">'event' : event property must have both add and remove accessors</span></span>  
  
 <span data-ttu-id="f228c-104">不是欄位的事件必須有這兩種存取方法。</span><span class="sxs-lookup"><span data-stu-id="f228c-104">An event that is not a field must have both access methods.</span></span>  
  
 <span data-ttu-id="f228c-105">下列範例會產生 CS0065：</span><span class="sxs-lookup"><span data-stu-id="f228c-105">The following sample generates CS0065:</span></span>  
  
```csharp  
// CS0065.cs  
using System;  
public delegate void Eventhandler(object sender, int e);  
public class MyClass  
{  
   public event EventHandler Click   // CS0065,  
   {  
      // to fix, uncomment the add and remove definitions  
      /*  
      add  
      {  
         Click += value;  
      }  
      remove  
      {  
         Click -= value;  
      }  
      */  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
