# <a name="compiler-error-cs1922"></a><span data-ttu-id="2d7f5-102">編譯器錯誤 CS1922</span><span class="sxs-lookup"><span data-stu-id="2d7f5-102">Compiler Error CS1922</span></span>

<span data-ttu-id="2d7f5-103">集合初始設定式需要其類型 'type' 實作 System.Collections.IEnumerable。</span><span class="sxs-lookup"><span data-stu-id="2d7f5-103">Collection initializer requires its type 'type' to implement System.Collections.IEnumerable.</span></span>

<span data-ttu-id="2d7f5-104">若要搭配使用集合初始設定式與類型，類型必須實作 `IEnumerable`。</span><span class="sxs-lookup"><span data-stu-id="2d7f5-104">In order to use a collection initializer with a type, the type must implement `IEnumerable`.</span></span> <span data-ttu-id="2d7f5-105">如果您在想要使用物件初始設定式時不小心使用到集合初始設定式語法，則會發生這個錯誤。</span><span class="sxs-lookup"><span data-stu-id="2d7f5-105">This error can occur if you accidentally use collection initializer syntax when you meant to use an object initializer.</span></span>

## <a name="to-correct-this-error"></a><span data-ttu-id="2d7f5-106">更正這個錯誤</span><span class="sxs-lookup"><span data-stu-id="2d7f5-106">To correct this error</span></span>

- <span data-ttu-id="2d7f5-107">如果類型不代表集合，請使用物件初始設定式語法，而非集合初始設定式語法。</span><span class="sxs-lookup"><span data-stu-id="2d7f5-107">If the type does not represent a collection, use object initializer syntax instead of collection initializer syntax.</span></span>

- <span data-ttu-id="2d7f5-108">如果類型代表集合，請先進行修改來實作 `IEnumerable` ，才能使用集合初始設定式來初始化該類型的物件。</span><span class="sxs-lookup"><span data-stu-id="2d7f5-108">If the type does represent a collection, modify it to implement `IEnumerable` before you can use collection initializers to initialize objects of that type.</span></span>

- <span data-ttu-id="2d7f5-109">如果類型代表集合，而且您無法存取原始碼，則只要初始化其項目即可，方法是使用這些項目的類別建構函式或其他初始化方法。</span><span class="sxs-lookup"><span data-stu-id="2d7f5-109">If the type represents a collection and you do not have access to the source code, just initialize its elements by using their class constructors or other initialization methods.</span></span>

## <a name="example"></a><span data-ttu-id="2d7f5-110">範例</span><span class="sxs-lookup"><span data-stu-id="2d7f5-110">Example</span></span>

<span data-ttu-id="2d7f5-111">下列程式碼會產生 CS1922：</span><span class="sxs-lookup"><span data-stu-id="2d7f5-111">The following code produces CS1922:</span></span>

```csharp
// cs1922.cs
public class Test
{
    public static void Main()
    {
        // Collection initializer.
        var tc = new TestClass  {1,"hello"} ; // CS1922

        // Object initializer.
        var tc2 = new TestClass { memberA = 1, memberB = "hello" }; // OK
    }
}

public class TestClass
{
    public int memberA { get; set; }
    public string memberB { get; set; }
}
```
