 <span data-ttu-id="264be-104">當您宣告 [屬性](../../csharp/programming-guide/classes-and-structs/using-properties.md)時，您也必須宣告其存取子方法。</span><span class="sxs-lookup"><span data-stu-id="264be-104">When you declare a [property](../../csharp/programming-guide/classes-and-structs/using-properties.md), you must also declare its accessor methods.</span></span> <span data-ttu-id="264be-105">不過，屬性不能有多個 `get` 存取子方法或多個 `set` 存取子方法。</span><span class="sxs-lookup"><span data-stu-id="264be-105">However, a property cannot have more than one `get` accessor method or more than one `set` accessor method.</span></span>  
  
## <a name="example"></a><span data-ttu-id="264be-106">範例</span><span class="sxs-lookup"><span data-stu-id="264be-106">Example</span></span>

```  
        {  
            return 0;  
        }

            return 0;  
        }  
    }  
  
    public static void Main() {}  
}  
```
