# <a name="compiler-error-cs1007"></a><span data-ttu-id="74825-102">編譯器錯誤 CS1007</span><span class="sxs-lookup"><span data-stu-id="74825-102">Compiler Error CS1007</span></span>
<span data-ttu-id="74825-103">屬性存取子已定義</span><span class="sxs-lookup"><span data-stu-id="74825-103">Property accessor already defined</span></span>  
  
 <span data-ttu-id="74825-104">當您宣告 [屬性](../../csharp/programming-guide/classes-and-structs/using-properties.md)時，您也必須宣告其存取子方法。</span><span class="sxs-lookup"><span data-stu-id="74825-104">When you declare a [property](../../csharp/programming-guide/classes-and-structs/using-properties.md), you must also declare its accessor methods.</span></span> <span data-ttu-id="74825-105">不過，屬性不能有多個 `get` 存取子方法或多個 `set` 存取子方法。</span><span class="sxs-lookup"><span data-stu-id="74825-105">However, a property cannot have more than one `get` accessor method or more than one `set` accessor method.</span></span>  
  
## <a name="example"></a><span data-ttu-id="74825-106">範例</span><span class="sxs-lookup"><span data-stu-id="74825-106">Example</span></span>  
 <span data-ttu-id="74825-107">下列範例會產生 CS1007：</span><span class="sxs-lookup"><span data-stu-id="74825-107">The following sample generates CS1007:</span></span>  
  
```csharp  
// CS1007.cs  
public class clx  
{  
    public int MyProperty  
    {  
        get  
        {  
            return 0;  
        }  
        get   // CS1007, this is the second get method  
        {  
            return 0;  
        }  
    }  
  
    public static void Main() {}  
}  
```
