# <a name="compiler-error-cs0721"></a>編譯器錯誤 CS0721
'type'：靜態類型不能當做參數使用  
  
 靜態類型作為參數不具意義。 由於無法建立靜態類型的任何執行個體，因此無法將任何執行個體作為參數傳遞。  
  
 下列範例會產生 CS0721：  
  
```csharp  
// CS0721.cs  
public static class SC  
{  
}  
  
public class CMain  
{  
   public void F(SC sc)  // CS0721  
   {  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
