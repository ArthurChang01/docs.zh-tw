 <span data-ttu-id="26bd0-104">**事件** 定義中的 **加入** 或 [移除](../../csharp/language-reference/keywords/event.md) 關鍵字必須有主體。</span><span class="sxs-lookup"><span data-stu-id="26bd0-104">An **add** or **remove** keyword in an [event](../../csharp/language-reference/keywords/event.md) definition must have a body.</span></span> <span data-ttu-id="26bd0-105">如需詳細資訊，請參閱[事件](../../csharp/programming-guide/events/index.md)。</span><span class="sxs-lookup"><span data-stu-id="26bd0-105">For more information, see [Events](../../csharp/programming-guide/events/index.md).</span></span>  
  
 <span data-ttu-id="26bd0-106">下列範例會產生 CS0073：</span><span class="sxs-lookup"><span data-stu-id="26bd0-106">The following sample generates CS0073:</span></span>  
  
```  
// CS0073.cs  
delegate void del();  
  
class Test  
{  
   public event del MyEvent  
   {  
      add;   // CS0073  
      // try the following lines instead  
      // add  
      // {  
      //    MyEvent += value;  
      // }  
      remove  
      {  
         MyEvent -= value;  
      }  
  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
