# <a name="compiler-error-cs0412"></a><span data-ttu-id="8d39b-102">編譯器錯誤 CS0412</span><span class="sxs-lookup"><span data-stu-id="8d39b-102">Compiler Error CS0412</span></span>
<span data-ttu-id="8d39b-103">'generic': 參數或區域變數不能與方法類型參數的名稱相同</span><span class="sxs-lookup"><span data-stu-id="8d39b-103">'generic': a parameter or local variable cannot have the same name as a method type parameter</span></span>  
  
 <span data-ttu-id="8d39b-104">泛型方法的類型參數和方法或方法參數之一的區域變數之間有名稱衝突。</span><span class="sxs-lookup"><span data-stu-id="8d39b-104">There is a name conflict between the type parameter of a generic method and a local variable in the method or one of the method's parameters.</span></span> <span data-ttu-id="8d39b-105">若要避免這個錯誤，請重新命名任何衝突的參數或區域變數。</span><span class="sxs-lookup"><span data-stu-id="8d39b-105">To avoid this error, rename any conflicting parameters or local variables.</span></span>  
  
## <a name="example"></a><span data-ttu-id="8d39b-106">範例</span><span class="sxs-lookup"><span data-stu-id="8d39b-106">Example</span></span>  
 <span data-ttu-id="8d39b-107">下列範例會產生 CS0412：</span><span class="sxs-lookup"><span data-stu-id="8d39b-107">The following sample generates CS0412:</span></span>  
  
```csharp  
// CS0412.cs  
using System;  
  
class C  
{  
    // Parameter name is the same as method type parameter name  
    public void G<T>(int T)  // CS0412  
    {  
    }  
    public void F<T>()  
    {  
        // Method local variable name is the same as method type  
        // parameter name  
        double T = 0.0;  // CS0412  
        Console.WriteLine(T);  
    }  
  
    public static void Main()  
    {  
    }  
}  
```
