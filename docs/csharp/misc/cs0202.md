# <a name="compiler-error-cs0202"></a><span data-ttu-id="65de8-102">編譯器錯誤 CS0202</span><span class="sxs-lookup"><span data-stu-id="65de8-102">Compiler Error CS0202</span></span>
<span data-ttu-id="65de8-103">foreach 要求 'type.GetEnumerator()' 的傳回類型 'type' 必須要有適合的公用 MoveNext 方法以及公用 Current 屬性</span><span class="sxs-lookup"><span data-stu-id="65de8-103">foreach requires that the return type 'type' of 'type.GetEnumerator()' must have a suitable public MoveNext method and public Current property</span></span>  
  
 <span data-ttu-id="65de8-104"><xref:System.Collections.IEnumerable.GetEnumerator%2A> 函式，用來啟用 foreach 陳述式的使用，無法傳回指標或陣列；它必須傳回可作為列舉值之類別的執行個體。</span><span class="sxs-lookup"><span data-stu-id="65de8-104">A <xref:System.Collections.IEnumerable.GetEnumerator%2A> function, used to enable the use of foreach statements, cannot return a pointer or array; it must return an instance of a class that is able to act as an enumerator.</span></span> <span data-ttu-id="65de8-105">作為列舉值的適當需求包括公用 Current 屬性和公用 MoveNext 方法。</span><span class="sxs-lookup"><span data-stu-id="65de8-105">The proper requirements to serve as an enumerator include a public Current property and a public MoveNext method.</span></span>

 <span data-ttu-id="65de8-108">下列範例會產生 CS0202：</span><span class="sxs-lookup"><span data-stu-id="65de8-108">The following sample generates CS0202:</span></span>  
  
```  
// CS0202.cs  
  
public class C1  
{  
   public int Current  
   {  
      get  
      {  
         return 0;  
      }  
   }  
  
   public bool MoveNext ()  
   {  
      return false;  
   }  
  
   public static implicit operator C1 (int c1)  
   {  
      return 0;  
   }  
}  
  
public class C2  
{  
   public int Current  
   {  
      get  
      {  
         return 0;  
      }  
   }  
  
   public bool MoveNext ()  
   {  
      return false;  
   }  
  
   public C1[] GetEnumerator ()  
   // try the following line instead  
   // public C1 GetEnumerator ()  
   {  
      return null;  
   }  
}  
  
public class MainClass  
{  
   public static void Main ()  
   {  
      C2 c2 = new C2();  
  
      foreach (C1 x in c2)   // CS0202  
      {  
         System.Console.WriteLine(x.Current);  
      }  
   }  
}  
```
