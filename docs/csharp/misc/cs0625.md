# <a name="compiler-error-cs0625"></a><span data-ttu-id="ebbf9-102">編譯器錯誤 CS0625</span><span class="sxs-lookup"><span data-stu-id="ebbf9-102">Compiler Error CS0625</span></span>
<span data-ttu-id="ebbf9-103">'field': 標記為 structlayout （layoutkind.explicit) 的執行個體欄位類型必須要有 FieldOffset 屬性。</span><span class="sxs-lookup"><span data-stu-id="ebbf9-103">'field': instance field types marked with StructLayout(LayoutKind.Explicit) must have a FieldOffset attribute.</span></span>
  
<span data-ttu-id="ebbf9-104">使用明確 **StructLayout** 屬性標記結構時，結構中的所有欄位都必須有 [FieldOffset](xref:System.Runtime.InteropServices.FieldOffsetAttribute) 屬性。</span><span class="sxs-lookup"><span data-stu-id="ebbf9-104">When a struct is marked with an explicit **StructLayout** attribute, all fields in the struct must have the [FieldOffset](xref:System.Runtime.InteropServices.FieldOffsetAttribute) attribute.</span></span> <span data-ttu-id="ebbf9-105">如需詳細資訊，請參閱 [StructLayoutAttribute 類別](xref:System.Runtime.InteropServices.StructLayoutAttribute)。</span><span class="sxs-lookup"><span data-stu-id="ebbf9-105">For more information, see [StructLayoutAttribute Class](xref:System.Runtime.InteropServices.StructLayoutAttribute).</span></span> 

<span data-ttu-id="ebbf9-106">下列範例會產生 CS0625：</span><span class="sxs-lookup"><span data-stu-id="ebbf9-106">The following sample generates CS0625:</span></span>  
  
```csharp  
// CS0625.cs  
// compile with: /target:library  
using System;  
using System.Runtime.InteropServices;  
  
[StructLayout(LayoutKind.Explicit)]  
struct A  
{  
   public int i;   // CS0625 not static; an instance field  
}  
  
// OK  
[StructLayout(LayoutKind.Explicit)]  
struct B  
{  
   [FieldOffset(5)]  
   public int i;  
}  
```
