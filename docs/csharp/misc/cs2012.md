# <a name="compiler-error-cs2012"></a>編譯器錯誤 CS2012
無法開啟 'file' 進行寫入  
  
 使用 [/bugreport:](../language-reference/compiler-options/bugreport-compiler-option.md)`file` 編譯器選項時，無法開啟檔案進行寫入。 請確定您已指定有效的檔案名稱，而且檔案不是唯讀的。
