## <a name="example"></a><span data-ttu-id="75e96-108">範例</span><span class="sxs-lookup"><span data-stu-id="75e96-108">Example</span></span>  
 <span data-ttu-id="75e96-109">下列程式碼會產生 CS1928：</span><span class="sxs-lookup"><span data-stu-id="75e96-109">The following code generates CS1928:</span></span>

```  
    static void Main()  
    {  
        Test t = new Test();  
        t.M("hi"); // CS1928

    }  
}  
```
