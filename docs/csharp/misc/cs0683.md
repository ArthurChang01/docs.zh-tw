# <a name="compiler-error-cs0683"></a><span data-ttu-id="ce080-102">編譯器錯誤 CS0683</span><span class="sxs-lookup"><span data-stu-id="ce080-102">Compiler Error CS0683</span></span>
<span data-ttu-id="ce080-103">'explicitmethod' 明確方法實作無法實作 'method'，因為它是存取子。</span><span class="sxs-lookup"><span data-stu-id="ce080-103">"'explicitmethod' explicit method implementation cannot implement 'method' because it is an accessor</span></span>  
  
 <span data-ttu-id="ce080-104">下列範例會產生 CS0683：</span><span class="sxs-lookup"><span data-stu-id="ce080-104">The following sample generates CS0683:</span></span>  
  
```csharp  
// CS0683.cs  
interface IExample  
{  
   int Test { get; }  
}  
  
class CExample : IExample  
{  
   int IExample.get_Test() { return 0; } // CS0683  
   int IExample.Test { get{ return 0; } } // correct  
}  
```
