# 編譯器錯誤 CS0517
'class' 沒有基底類別且無法呼叫基底建構函式  
  
 只有在 .NET Framework Common Language Runtime 編譯物件類別 \(這是唯一沒有基底類別的類別\) 的原始程式碼時，才會發生 CS0517。
