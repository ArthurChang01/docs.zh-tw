# <a name="compiler-error-cs0761"></a><span data-ttu-id="f33fc-102">編譯器錯誤 CS0761</span><span class="sxs-lookup"><span data-stu-id="f33fc-102">Compiler Error CS0761</span></span>
<span data-ttu-id="f33fc-103">' Method\<T > ' 的部分方法宣告有不一致的類型參數條件約束。</span><span class="sxs-lookup"><span data-stu-id="f33fc-103">Partial method declarations of 'method\<T>' have inconsistent type parameter constraints.</span></span>  
  
 <span data-ttu-id="f33fc-104">如果部分方法具有實作，泛型類型條件約束必須與方法簽章上所定義的條件約束完全相同。</span><span class="sxs-lookup"><span data-stu-id="f33fc-104">If a partial method has an implementation, the generic type constraint must be identical to the constraint defined on the method signature.</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="f33fc-105">更正這個錯誤</span><span class="sxs-lookup"><span data-stu-id="f33fc-105">To correct this error</span></span>

## <a name="example"></a><span data-ttu-id="f33fc-107">範例</span><span class="sxs-lookup"><span data-stu-id="f33fc-107">Example</span></span>

    }
