# <a name="compiler-warning-level-2-cs0251"></a><span data-ttu-id="f247b-102">編譯器警告 (層級 2) CS0251</span><span class="sxs-lookup"><span data-stu-id="f247b-102">Compiler Warning (level 2) CS0251</span></span>
<span data-ttu-id="f247b-103">正在以負值索引檢索陣列 (陣列索引一律從 0 開始)</span><span class="sxs-lookup"><span data-stu-id="f247b-103">Indexing an array with a negative index (array indices always start at zero)</span></span>  
  
 <span data-ttu-id="f247b-104">請不要使用負值索引來檢索陣列。</span><span class="sxs-lookup"><span data-stu-id="f247b-104">Do not use a negative number to index into an array.</span></span>  
  
 <span data-ttu-id="f247b-105">下列範例會產生 CS0251：</span><span class="sxs-lookup"><span data-stu-id="f247b-105">The following sample generates CS0251:</span></span>  
  
```csharp  
// CS0251.cs  
// compile with: /W:2  
class MyClass  
{  
   public static void Main()  
   {  
      int[] myarray = new int[] {1,2,3};     
      try  
      {  
         myarray[-1]++;   // CS0251  
         // try the following line instead  
         // myarray[1]++;  
      }  
      catch (System.IndexOutOfRangeException e)  
      {  
         System.Console.WriteLine("{0}", e);  
      }  
   }  
}  
```
