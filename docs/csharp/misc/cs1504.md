# <a name="compiler-error-cs1504"></a>編譯器錯誤 CS1504
原始程式檔 'file' 無法開啟 ('reason')  
  
 編譯器無法開啟或讀取原始程式檔。 檔案可能被另一個應用程式鎖定，或者可能有其他作業系統問題。 訊息包含作業系統的原因，解釋編譯器為何無法開啟或讀取檔案。
