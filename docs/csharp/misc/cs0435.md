   public class A { }

```  
  
 <span data-ttu-id="76a7a-109">然後，編譯這個檔案：</span><span class="sxs-lookup"><span data-stu-id="76a7a-109">Then, compile this file:</span></span>

{  
   public class A { }  
}

   {  
      Console.WriteLine(typeof(Util.A)); // CS0435  
   }

```
