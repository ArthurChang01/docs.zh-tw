# <a name="compiler-warning-level-2-cs0435"></a><span data-ttu-id="760c7-102">編譯器警告 (層級 2) CS0435</span><span class="sxs-lookup"><span data-stu-id="760c7-102">Compiler Warning (level 2) CS0435</span></span>
<span data-ttu-id="760c7-103">'assembly' 中的命名空間 'namespace' 與 'assembly' 中匯入的類型 'type' 相衝突。</span><span class="sxs-lookup"><span data-stu-id="760c7-103">The namespace 'namespace' in 'assembly' conflicts with the imported type 'type' in 'assembly'.</span></span> <span data-ttu-id="760c7-104">使用"程式集"中定義的命名空間。</span><span class="sxs-lookup"><span data-stu-id="760c7-104">Using the namespace defined in 'assembly'.</span></span>  
  
 <span data-ttu-id="760c7-105">原始程式檔 (file_2) 中的命名空間與 file_1 中匯入的類型衝突時，會發出這個警告。</span><span class="sxs-lookup"><span data-stu-id="760c7-105">This warning is issued when a namespace in a source file (file_2) conflicts with an imported type in file_1.</span></span> <span data-ttu-id="760c7-106">編譯器會使用原始程式檔中的類型。</span><span class="sxs-lookup"><span data-stu-id="760c7-106">The compiler uses the one in the source file.</span></span>  
  
 <span data-ttu-id="760c7-107">下列範例會產生 CS0435：</span><span class="sxs-lookup"><span data-stu-id="760c7-107">The following example generates CS0435:</span></span>  
  
 <span data-ttu-id="760c7-108">先編譯這個檔案：</span><span class="sxs-lookup"><span data-stu-id="760c7-108">Compile this file first:</span></span>  
  
```csharp  
// CS0435_1.cs  
// compile with: /t:library  
public class Util
{  
   public class A { }  
}  
```  
  
 <span data-ttu-id="760c7-109">然後，編譯這個檔案：</span><span class="sxs-lookup"><span data-stu-id="760c7-109">Then, compile this file:</span></span>  
  
```csharp  
// CS0435_2.cs  
// compile with: /r:CS0435_1.dll  
  
using System;  
  
namespace Util
{  
   public class A { }  
}  
  
public class Test
{  
   public static void Main()
   {  
      Console.WriteLine(typeof(Util.A)); // CS0435  
   }  
}  
```
