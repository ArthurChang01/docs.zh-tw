# 編譯器錯誤 CS1630
無效的 \/errorreport 選項 'option'; 必須是 prompt、send、queue 或 none  
  
 命令列選項 [\/errorreport](../../csharp/language-reference/compiler-options/errorreport-compiler-option.md) 後面必須接著 **prompt**、**send**、**queue** 或 **none**，並指定發生編譯器內部錯誤時所要採用的動作。
