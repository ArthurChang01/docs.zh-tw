# <a name="compiler-warning-level-1-cs2014"></a>編譯器警告 (層級 1) CS2014
編譯器選項 'old option' 已過時，請改用 'new option'  
  
 編譯器選項的格式已被取代。 如需詳細資訊，請參閱 [C# 編譯器選項](../../csharp/language-reference/compiler-options/index.md) 。
