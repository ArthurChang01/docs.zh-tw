# <a name="compiler-error-cs0008"></a>編譯器錯誤 CS0008
從檔案 'file' 讀取中繼資料時發生未預期的錯誤 — 'description'  
  
 已順利開啟 DLL 來擷取中繼資料但損毀，因此無法從中讀取資料。 如需詳細資訊，請參閱 [/reference (C# 編譯器選項)](../language-reference/compiler-options/reference-compiler-option.md)。
