# <a name="compiler-error-cs0250"></a>編譯器錯誤 CS0250

不要直接呼叫您的基底類別 Finalize 方法。 會從您的解構函式自動呼叫它。  
  
 程式無法嘗試強制清除基底類別資源。  
  
 請參閱[完成項](../../csharp/programming-guide/classes-and-structs/destructors.md)如需詳細資訊。  
  
 下列範例會產生 CS0250：

```csharp
// CS0250.cs  
class B  
{  
}  
  
class C : B  
{  
   ~C()  
   {  
      base.Finalize();   // CS0250  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
