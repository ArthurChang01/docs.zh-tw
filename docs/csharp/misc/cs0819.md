# <a name="compiler-error-cs0819"></a>編譯器錯誤 CS0819
隱含類型區域變數不能有多個宣告子。  
  
 明確類型宣告中允許多個宣告子，但隱含類型變數則不允許。  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
1. 在個別行上，宣告並指派每個隱含類型區域變數的值。  
  
## <a name="example"></a>範例  
 下列程式碼會產生 CS0819：  
  
```csharp  
// cs0819.cs  
class A  
{  
    public static int Main()  
    {  
        var a = 3, b = 2; // CS0819  
        return -1;  
    }  
}  
```
