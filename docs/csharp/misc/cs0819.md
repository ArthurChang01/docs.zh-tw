# <a name="compiler-error-cs0819"></a><span data-ttu-id="20a55-102">編譯器錯誤 CS0819</span><span class="sxs-lookup"><span data-stu-id="20a55-102">Compiler Error CS0819</span></span>

 <span data-ttu-id="20a55-104">明確類型宣告中允許多個宣告子，但隱含類型變數則不允許。</span><span class="sxs-lookup"><span data-stu-id="20a55-104">Multiple declarators are allowed in explicit type declarations, but not with implicitly typed variables.</span></span>

## <a name="to-correct-this-error"></a><span data-ttu-id="20a55-105">更正這個錯誤</span><span class="sxs-lookup"><span data-stu-id="20a55-105">To correct this error</span></span>

<span data-ttu-id="20a55-106">有兩個選項：</span><span class="sxs-lookup"><span data-stu-id="20a55-106">There are two options:</span></span>

1. <span data-ttu-id="20a55-107">如果變數屬於相同類型，請使用明確宣告。</span><span class="sxs-lookup"><span data-stu-id="20a55-107">If the variables are of the same type, use explicit declarations.</span></span>

## <a name="example-1"></a><span data-ttu-id="20a55-109">範例 1</span><span class="sxs-lookup"><span data-stu-id="20a55-109">Example 1</span></span>

<span data-ttu-id="20a55-110">下列程式碼會產生 CS0819：</span><span class="sxs-lookup"><span data-stu-id="20a55-110">The following code generates CS0819:</span></span>

```csharp
// cs0819.cs
class Program
{
    public static void Main()
    {
        var a = 3, b = 2; // CS0819

        // First correction option.
        //int a = 3, b = 2;

        // Second correction option.
        //var a = 3;
        //var b = 2;
    }
}
```

        //    using (var font2 = new Font("Arial", 10.0f)
        //    {
        //    }
        //}


    }
