# <a name="compiler-error-cs0081"></a><span data-ttu-id="1005b-102">編譯器錯誤 CS0081</span><span class="sxs-lookup"><span data-stu-id="1005b-102">Compiler Error CS0081</span></span>
<span data-ttu-id="1005b-103">類型參數宣告必須是識別項，而非類型。</span><span class="sxs-lookup"><span data-stu-id="1005b-103">Type parameter declaration must be an identifier not a type</span></span>  
  
 <span data-ttu-id="1005b-104">當您宣告泛型方法或類型時，請指定類型參數作為識別項 (例如 "T" 或 "inputType")。</span><span class="sxs-lookup"><span data-stu-id="1005b-104">When you declare a generic method or type, specify the type parameter as an identifier, for example "T" or "inputType".</span></span> <span data-ttu-id="1005b-105">用戶端程式碼呼叫方法時，會提供類型，以取代方法或類別主體中的所有識別項。</span><span class="sxs-lookup"><span data-stu-id="1005b-105">When client code calls the method, it supplies the type, which replaces each occurrence of the identifier in the method or class body.</span></span> <span data-ttu-id="1005b-106">如需詳細資訊，請參閱[泛型型別參數](../../csharp/programming-guide/generics/generic-type-parameters.md)。</span><span class="sxs-lookup"><span data-stu-id="1005b-106">For more information, see [Generic Type Parameters](../../csharp/programming-guide/generics/generic-type-parameters.md).</span></span>  
  
```  
// CS0081.cs  
class MyClass  
{  
   public void F<int>() {}   // CS0081  
   public void F<T>(T input) {}   // OK  
  
   public static void Main()  
   {  
      MyClass a = new MyClass();  
      a.F<int>(2);  
      a.F<double>(.05);  
   }  
}  
```
