## <a name="example"></a><span data-ttu-id="84880-105">範例</span><span class="sxs-lookup"><span data-stu-id="84880-105">Example</span></span>  
 <span data-ttu-id="84880-106">下列範例會產生 CS1686。</span><span class="sxs-lookup"><span data-stu-id="84880-106">The following sample generates CS1686.</span></span>  
  
```  
// CS1686.cs  
// compile with: /unsafe /target:library  
class MyClass  
{  
   public unsafe delegate int * MyDelegate();  
  
   public unsafe int * Test()  
   {  
      int j = 0;  
      MyDelegate d = delegate { return &j; };   // CS1686  
      return &j;   // OK  
   }  
}  
```
