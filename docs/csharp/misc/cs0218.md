# <a name="compiler-error-cs0218"></a><span data-ttu-id="fd140-102">編譯器錯誤 CS0218</span><span class="sxs-lookup"><span data-stu-id="fd140-102">Compiler Error CS0218</span></span>
<span data-ttu-id="fd140-103">類型 ('type') 必須包含運算子 true 和運算子 false 的宣告</span><span class="sxs-lookup"><span data-stu-id="fd140-103">The type ('type') must contain declarations of operator true and operator false</span></span>  
  
 <span data-ttu-id="fd140-104">如果您定義了一個使用者定義類型的運算子，然後嘗試使用該運算子作為最少運算運算子，則使用者定義運算子必須已定義運算子 true 和運算子 false。</span><span class="sxs-lookup"><span data-stu-id="fd140-104">If you define an operator for a user-defined type, and then try to use the operator as a short-circuit operator, the user-defined operator must have operator true and operator false defined.</span></span> <span data-ttu-id="fd140-105">如需詳細資訊最少運算運算子，請參閱[（& s) （& s) 運算子](../../csharp/language-reference/operators/conditional-and-operator.md)和[&#124; &#124;運算子](../../csharp/language-reference/operators/conditional-or-operator.md)。</span><span class="sxs-lookup"><span data-stu-id="fd140-105">For more information on short-circuit operators, see [&& Operator](../../csharp/language-reference/operators/conditional-and-operator.md) and [&#124;&#124; Operator](../../csharp/language-reference/operators/conditional-or-operator.md).</span></span>  
  
 <span data-ttu-id="fd140-106">下列範例會產生 CS0218：</span><span class="sxs-lookup"><span data-stu-id="fd140-106">The following sample generates CS0218:</span></span>  
  
```  
// CS0218.cs  
using System;  
public class MyClass  
{  
   // uncomment these operator declarations to resolve this CS0218  
   /*  
   public static bool operator true (MyClass f)  
   {  
      return false;  
   }  
  
   public static bool operator false (MyClass f)  
   {  
      return false;  
   }  
   */  
  
   public static implicit operator int(MyClass x)  
   {  
      return 0;  
   }  
  
   public static MyClass operator & (MyClass f1, MyClass f2)  
   {  
      return new MyClass();  
   }  
  
   public static void Main()  
   {  
      MyClass f = new MyClass();  
      int i = f && f;   // CS0218, requires operators true and false  
   }  
}  
```
