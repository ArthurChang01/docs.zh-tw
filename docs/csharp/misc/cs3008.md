# <a name="compiler-warning-level-1-cs3008"></a>編譯器警告 (層級 1) CS3008
只有大小寫不同的識別項 'identifier' 不符合 CLS 標準  
  
 A[公用](../../csharp/language-reference/keywords/public.md)，[保護](../../csharp/language-reference/keywords/protected.md)，或`protected internal`識別項會中斷與 Common Language Specification (CLS) 相容性，如果它是以底線字元開頭 (\_)。 如需 CLS 合規性的詳細資訊，請參閱[撰寫符合 CLS 標準的程式碼](http://msdn.microsoft.com/en-us/4c705105-69a2-4e5e-b24e-0633bc32c7f3)和[語言獨立性以及與語言無關的元件](../../../docs/standard/language-independence-and-language-independent-components.md)。  
  
## <a name="example"></a>範例  
 下列範例會產生 CS3008：  
  
```  
// CS3008.cs  
  
using System;  
  
[assembly:CLSCompliant(true)]  
public class a  
{  
    public static int _a = 0;  // CS3008  
    // OK, private  
    // private static int _a1 = 0;  
  
    public static void Main()  
    {  
    }  
}  
```
