<span data-ttu-id="17e74-103">'member' 無法同時為 extern 並宣告為主體</span><span class="sxs-lookup"><span data-stu-id="17e74-103">'member' cannot be extern and declare a body</span></span>  
  
 <span data-ttu-id="17e74-104">類別成員標記為 [extern](../../csharp/language-reference/keywords/extern.md)時，表示成員的定義位於另一個檔案中。</span><span class="sxs-lookup"><span data-stu-id="17e74-104">When a class member is marked [extern](../../csharp/language-reference/keywords/extern.md), it means that the member's definition is located in another file.</span></span> <span data-ttu-id="17e74-105">因此，標記為 **extern** 的類別成員不能定義於類別中。</span><span class="sxs-lookup"><span data-stu-id="17e74-105">Therefore, a class member marked as **extern** cannot be defined in the class.</span></span> <span data-ttu-id="17e74-106">請移除 `extern` 關鍵字，或刪除定義。</span><span class="sxs-lookup"><span data-stu-id="17e74-106">Either remove the `extern` keyword or delete the definition.</span></span> <span data-ttu-id="17e74-107">如需詳細資訊，請參閱[方法](../../csharp/programming-guide/classes-and-structs/methods.md)。</span><span class="sxs-lookup"><span data-stu-id="17e74-107">For more information, see [Methods](../../csharp/programming-guide/classes-and-structs/methods.md).</span></span>  
  
 <span data-ttu-id="17e74-108">下列範例會產生 CS0179：</span><span class="sxs-lookup"><span data-stu-id="17e74-108">The following sample generates CS0179:</span></span>  
  
```csharp

   public extern int ExternMethod(int aa)   // CS0179

      return 0;  
   }  
   // try the following line instead  
   // public extern int ExternMethod(int aa);  
  
   public static void Main()  
   {  
   }  
}  
```
