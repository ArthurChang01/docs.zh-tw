# <a name="compiler-warning-level-4-cs0422"></a>編譯器警告 (層級 4) CS0422

已不再支援 /incremental 選項

 C# 2.0 中不支援增量編譯 **（/incr**或 **/增量**）。

> [!NOTE]
> 此編譯器警告不再在 Roslyn 中使用。
