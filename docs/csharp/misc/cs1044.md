 <span data-ttu-id="8e212-104">編譯器發現無效的陳述式。</span><span class="sxs-lookup"><span data-stu-id="8e212-104">The compiler found an invalid statement.</span></span>  
  
 <span data-ttu-id="8e212-105">下列範例會產生 CS1044：</span><span class="sxs-lookup"><span data-stu-id="8e212-105">The following sample generates CS1044:</span></span>  
  
```csharp  
// CS1044.cs  
using System;  
  
public class MyClass : IDisposable  
{  
   public void Dispose()  
   {  
      Console.WriteLine("Res1.Dispose()");  
   }  
  
   public static void Main()  
   {  
      using (MyClass mc1 = new MyClass(),  
             MyClass mc2 = new MyClass())   // CS1044, remove an instantiation  
      {  
      }  
   }  
}  
```
