# <a name="compiler-error-cs0426"></a><span data-ttu-id="65b20-102">編譯器錯誤 CS0426</span><span class="sxs-lookup"><span data-stu-id="65b20-102">Compiler Error CS0426</span></span>
<span data-ttu-id="65b20-103">類型 'type' 中沒有類型名稱 'identifier'</span><span class="sxs-lookup"><span data-stu-id="65b20-103">The type name 'identifier' does not exist in the type 'type'</span></span>  
  
 <span data-ttu-id="65b20-104">當您參考的類型巢狀於另一個類型內，但此類型不存在時，就會發生這個錯誤。</span><span class="sxs-lookup"><span data-stu-id="65b20-104">This error occurs when you reference a type nested within another type, but no such nested type exists.</span></span> <span data-ttu-id="65b20-105">如果您輸入的巢狀類型名稱不正確，就可能會發生這種情況。</span><span class="sxs-lookup"><span data-stu-id="65b20-105">This can occur if you mistype the name of the nested type.</span></span> <span data-ttu-id="65b20-106">請檢查使用的名稱拼字，並確認封入類型具有預期的成員。</span><span class="sxs-lookup"><span data-stu-id="65b20-106">Check the spelling of the names used, and verify that the enclosing type has the expected member.</span></span>  
  
 <span data-ttu-id="65b20-107">下列範例會產生 CS0426，因為類別 C 沒有巢狀類型 A：</span><span class="sxs-lookup"><span data-stu-id="65b20-107">The following sample generates CS0426 because class C has no nested type A:</span></span>  
  
```csharp  
// CS0426.cs  
  
class C  
{  
    // No nested types are declared.     
}  
  
class D  
{  
   public static void Main()  
   {  
       C c = new C();  
       // Attempt to reference a nested type A:  
       C.A a; // CS0426 because no such type C.A  
   }  
}  
```
