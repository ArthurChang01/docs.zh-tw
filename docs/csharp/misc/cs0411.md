<span data-ttu-id="4923c-103">方法 'method' 的類型引數不能從使用方式推斷。</span><span class="sxs-lookup"><span data-stu-id="4923c-103">The type arguments for method 'method' cannot be inferred from the usage.</span></span> <span data-ttu-id="4923c-104">請嘗試明確指定類型引數。</span><span class="sxs-lookup"><span data-stu-id="4923c-104">Try specifying the type arguments explicitly.</span></span>  
  
 <span data-ttu-id="4923c-105">如果您在未明確提供類型引數的情況下呼叫泛型方法，而且編譯器無法推斷想要的類型引數，則會發生這個錯誤。</span><span class="sxs-lookup"><span data-stu-id="4923c-105">This error occurs if you call a generic method without explicitly providing the type arguments and the compiler cannot infer which type arguments are intended.</span></span> <span data-ttu-id="4923c-106">若要避免這個錯誤，請在角括弧中加入想要的類型引數。</span><span class="sxs-lookup"><span data-stu-id="4923c-106">To avoid this error, add the intended type arguments in angle brackets.</span></span>  
  
## <a name="example"></a><span data-ttu-id="4923c-107">範例</span><span class="sxs-lookup"><span data-stu-id="4923c-107">Example</span></span>  
 <span data-ttu-id="4923c-108">下列範例會產生 CS0411：</span><span class="sxs-lookup"><span data-stu-id="4923c-108">The following sample generates CS0411:</span></span>  
  
```csharp  
// CS0411.cs  
class C  
{  
    void G<T>()  
    {  
    }  
  
    public static void Main()  
    {  
        G();  // CS0411  
        // Try this instead:  
        // G<int>();  
    }  
}  
```
