# <a name="compiler-error-cs0411"></a><span data-ttu-id="6068b-102">編譯器錯誤 CS0411</span><span class="sxs-lookup"><span data-stu-id="6068b-102">Compiler Error CS0411</span></span>
<span data-ttu-id="6068b-103">方法 'method' 的類型引數不能從使用方式推斷。</span><span class="sxs-lookup"><span data-stu-id="6068b-103">The type arguments for method 'method' cannot be inferred from the usage.</span></span> <span data-ttu-id="6068b-104">請嘗試明確指定類型引數。</span><span class="sxs-lookup"><span data-stu-id="6068b-104">Try specifying the type arguments explicitly.</span></span>  
  
 <span data-ttu-id="6068b-105">如果您在未明確提供類型引數的情況下呼叫泛型方法，而且編譯器無法推斷想要的類型引數，則會發生這個錯誤。</span><span class="sxs-lookup"><span data-stu-id="6068b-105">This error occurs if you call a generic method without explicitly providing the type arguments and the compiler cannot infer which type arguments are intended.</span></span> <span data-ttu-id="6068b-106">若要避免這個錯誤，請在角括弧中加入想要的類型引數。</span><span class="sxs-lookup"><span data-stu-id="6068b-106">To avoid this error, add the intended type arguments in angle brackets.</span></span>  
  
## <a name="example"></a><span data-ttu-id="6068b-107">範例</span><span class="sxs-lookup"><span data-stu-id="6068b-107">Example</span></span>  
 <span data-ttu-id="6068b-108">下列範例會產生 CS0411：</span><span class="sxs-lookup"><span data-stu-id="6068b-108">The following sample generates CS0411:</span></span>  
  
```csharp  
// CS0411.cs  
class C  
{  
    void G<T>()  
    {  
    }  
  
    public static void Main()  
    {  
        G();  // CS0411  
        // Try this instead:  
        // G<int>();  
    }  
}  
```  
  
## <a name="example"></a><span data-ttu-id="6068b-109">範例</span><span class="sxs-lookup"><span data-stu-id="6068b-109">Example</span></span>  
 <span data-ttu-id="6068b-110">其他可能的錯誤情況包括參數是沒有任何類型資訊的 `null`時：</span><span class="sxs-lookup"><span data-stu-id="6068b-110">Other possible error cases include when the parameter is `null`, which has no type information:</span></span>  
  
```csharp  
// CS0411b.cs  
class C  
{  
    public void F<T>(T t) where T : C   
    {  
    }  
  
    public static void Main()  
    {  
        C c = new C();  
        c.F(null);  // CS0411  
    }

```
