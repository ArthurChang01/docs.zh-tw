# <a name="compiler-error-cs1109"></a>編譯器錯誤 CS1109
擴充方法必須定義在最上層靜態類別上，'name' 為巢狀類別。  
  
 擴充方法不能定義在巢狀類別中。  
  
## <a name="example"></a>範例  
 下列範例會產生 CS1109，因為類別 `Extension` 巢狀於類別 `Out`內：  
  
```csharp  
// cs1109.cs  
public class Test  
{  
}  
static class Out  
{  
    static class Extension  
    {  
        static void ExtMethod(this Test c) // CS1109  
        {  
        }  
    }  
}  
```
