# <a name="compiler-error-cs1958"></a><span data-ttu-id="ee1e1-102">編譯器錯誤 CS1958</span><span class="sxs-lookup"><span data-stu-id="ee1e1-102">Compiler Error CS1958</span></span>
<span data-ttu-id="ee1e1-103">物件與集合初始設定式運算式不可套用到委派建立運算式。</span><span class="sxs-lookup"><span data-stu-id="ee1e1-103">Object and collection initializer expressions may not be applied to a delegate creation expression,</span></span>  
  
 <span data-ttu-id="ee1e1-104">不像是類別或結構，委派沒有成員，因此沒有可供物件初始設定式初始化的項目。</span><span class="sxs-lookup"><span data-stu-id="ee1e1-104">A delegate has no members like a class or struct has, and so there is nothing for an object initializer to initialize.</span></span> <span data-ttu-id="ee1e1-105">如果您遇到這個錯誤，可能是因為委派建立運算式之後有大括號。</span><span class="sxs-lookup"><span data-stu-id="ee1e1-105">If you encounter this error, it is probably because there are braces after the delegate creation expression.</span></span> <span data-ttu-id="ee1e1-106">只要移除大括號，這個錯誤就會消失。</span><span class="sxs-lookup"><span data-stu-id="ee1e1-106">Just remove the braces and this error will disappear.</span></span>

## <a name="example"></a><span data-ttu-id="ee1e1-109">範例</span><span class="sxs-lookup"><span data-stu-id="ee1e1-109">Example</span></span>  
 <span data-ttu-id="ee1e1-110">下列程式碼會產生 CS1958：</span><span class="sxs-lookup"><span data-stu-id="ee1e1-110">The following code produces CS1958:</span></span>  
  
```csharp  
// cs1958.cs  
public class MemberInitializerTest  
{     
    delegate void D<T>();  
    public static void GenericMethod<T>() { }  
    public static void Run()  
    {  
        D<int> genD = new D<int>(GenericMethod<int>) { }; // CS1958  
       // Try the following line instead  
      // D<int> genD = new D<int>(GenericMethod<int>);  
    }  
}  
```
