## <a name="example"></a><span data-ttu-id="919f9-108">範例</span><span class="sxs-lookup"><span data-stu-id="919f9-108">Example</span></span>

<span data-ttu-id="919f9-109">以下代碼生成 CS1955，因為代碼嘗試使用[調用運算式`()`](../language-reference/operators/member-access-operators.md#invocation-expression-)調用欄位和屬性。</span><span class="sxs-lookup"><span data-stu-id="919f9-109">The following code generates CS1955 because the code is trying to invoke a field and a property by using the [invocation expression `()`](../language-reference/operators/member-access-operators.md#invocation-expression-).</span></span> <span data-ttu-id="919f9-110">不能調用欄位或屬性。</span><span class="sxs-lookup"><span data-stu-id="919f9-110">You cannot call a field or a property.</span></span> <span data-ttu-id="919f9-111">使用[成員訪問運算式`.`](../language-reference/operators/member-access-operators.md#member-access-expression-)訪問它存儲的值。</span><span class="sxs-lookup"><span data-stu-id="919f9-111">Use the [member access expression `.`](../language-reference/operators/member-access-operators.md#member-access-expression-) to access the value it stores.</span></span>
  
```csharp  
// cs1955.cs  
class A  
{  
    public int x = 0;  
    public int X  
    {  
        get { return x; }  
        set { x = value; }  
    }  
}  
  
class Test  
{  
    static int Main()  
    {  
        A a = new A();  
        a.x(); // CS1955  
        a.X(); // CS1955  
        // Try this line instead:  
        // int num = a.x;  
    }  
}  
```
