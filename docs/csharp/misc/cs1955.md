## <a name="example"></a><span data-ttu-id="61687-108">範例</span><span class="sxs-lookup"><span data-stu-id="61687-108">Example</span></span>

<span data-ttu-id="61687-109">下列程式碼會產生 CS1955，因為程式碼嘗試使用叫用欄位和屬性[叫用運算子`()` ](../language-reference/operators/member-access-operators.md#invocation-operator-)。</span><span class="sxs-lookup"><span data-stu-id="61687-109">The following code generates CS1955 because the code is trying to invoke a field and a property by using the [invocation operator `()`](../language-reference/operators/member-access-operators.md#invocation-operator-).</span></span> <span data-ttu-id="61687-110">您不能呼叫欄位或屬性。</span><span class="sxs-lookup"><span data-stu-id="61687-110">You cannot call a field or a property.</span></span> <span data-ttu-id="61687-111">使用[成員存取運算子`.`](../language-reference/operators/member-access-operators.md#member-access-operator-)存取它所儲存的值。</span><span class="sxs-lookup"><span data-stu-id="61687-111">Use the [member access operator `.`](../language-reference/operators/member-access-operators.md#member-access-operator-) to access the value it stores.</span></span>
  
```csharp  
// cs1955.cs  
class A  
{  
    public int x = 0;  
    public int X  
    {  
        get { return x; }  
        set { x = value; }  
    }  
}  
  
class Test  
{  
    static int Main()  
    {  
        A a = new A();  
        a.x(); // CS1955  
        a.X(); // CS1955  
        // Try this line instead:  
        // int num = a.x;  
    }  
}  
```
