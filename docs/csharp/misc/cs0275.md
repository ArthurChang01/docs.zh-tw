# <a name="compiler-error-cs0275"></a>編譯器錯誤 CS0275
'accessor': 存取範圍修飾詞不可使用於介面的存取子上  
  
 當您對屬性或介面中索引子的任何一個存取子使用存取修飾詞，就會發生此錯誤。 若要解決，請移除存取修飾詞。  
  
## <a name="example"></a>範例  
 下列範例會產生 CS0275：  
  
```csharp  
// CS0275.cs  
public interface MyInterface  
{  
    int Property  
    {  
        get;  
        internal set;   // CS0275  
    }  
}  
```
