# <a name="compiler-error-cs0692"></a>編譯器錯誤 CS0692
類型參數 'identifier' 重複  
  
 在類型參數清單中，相同的名稱只能使用一次。 重新命名或移除重複的類型參數。  
  
## <a name="example"></a>範例  
 下列範例會產生 CS0692：  
  
```csharp  
// CS0692.cs  
// compile with: /target:library  
class C <T, A, T>   // CS0692  
{  
}  
  
class D <T, T>   // CS0692  
{  
}  
```
