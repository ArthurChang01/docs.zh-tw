# <a name="compiler-error-cs0215"></a>編譯器錯誤 CS0215
運算子 True 或 False 的傳回類型必須為 bool  
  
 使用者定義[，則為 true 和 false](../language-reference/operators/true-false-operators.md)運算子必須具有傳回型別[bool](../language-reference/keywords/bool.md)。 如需詳細資訊，請參閱[多載運算子](../programming-guide/statements-expressions-operators/overloadable-operators.md)和 [operator](../language-reference/keywords/operator.md) 關鍵字文章。  
  
 下列範例會產生 CS0215：  
  
```csharp  
// CS0215.cs  
class MyClass  
{  
   public static int operator true (MyClass MyInt)   // CS0215  
   // try the following line instead  
   // public static bool operator true (MyClass MyInt)  
   {  
      return true;  
   }  
  
   public static int operator false (MyClass MyInt)   // CS0215  
   // try the following line instead  
   // public static bool operator false (MyClass MyInt)  
   {  
      return true;  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
