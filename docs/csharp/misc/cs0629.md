# <a name="compiler-error-cs0629"></a><span data-ttu-id="8205f-102">編譯器錯誤 CS0629</span><span class="sxs-lookup"><span data-stu-id="8205f-102">Compiler Error CS0629</span></span>
<span data-ttu-id="8205f-103">Conditional 成員 'member' 無法在類型 'Type Name' 中實作介面成員 'base class member'</span><span class="sxs-lookup"><span data-stu-id="8205f-103">Conditional member 'member' cannot implement interface member 'base class member' in type 'Type Name'</span></span>  
  
 <span data-ttu-id="8205f-104">無法在介面實作上使用 [條件式](../../csharp/programming-guide/concepts/attributes/common-attributes.md#Conditional) 屬性。</span><span class="sxs-lookup"><span data-stu-id="8205f-104">The [Conditional](../../csharp/programming-guide/concepts/attributes/common-attributes.md#Conditional) attribute cannot be used on the implementation of an interface.</span></span>  
  
 <span data-ttu-id="8205f-105">下列範例會產生 CS0629：</span><span class="sxs-lookup"><span data-stu-id="8205f-105">The following sample generates CS0629:</span></span>  
  
```csharp  
// CS0629.cs  
interface MyInterface  
{  
   void MyMethod();  
}  
  
public class MyClass : MyInterface  
{  
   [System.Diagnostics.Conditional("debug")]  
   public void MyMethod()    // CS0629, remove the Conditional attribute  
   {  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
