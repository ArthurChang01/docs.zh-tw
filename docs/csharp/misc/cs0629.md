# <a name="compiler-error-cs0629"></a><span data-ttu-id="a5f0f-102">編譯器錯誤 CS0629</span><span class="sxs-lookup"><span data-stu-id="a5f0f-102">Compiler Error CS0629</span></span>
<span data-ttu-id="a5f0f-103">Conditional 成員 'member' 無法在類型 'Type Name' 中實作介面成員 'base class member'</span><span class="sxs-lookup"><span data-stu-id="a5f0f-103">Conditional member 'member' cannot implement interface member 'base class member' in type 'Type Name'</span></span>  
  
 <span data-ttu-id="a5f0f-104">[條件](http://msdn.microsoft.com/library/e1c4913b-74d0-421a-8a6d-c14b3f0e68fb)屬性不能在介面實作。</span><span class="sxs-lookup"><span data-stu-id="a5f0f-104">The [Conditional](http://msdn.microsoft.com/library/e1c4913b-74d0-421a-8a6d-c14b3f0e68fb) attribute cannot be used on the implementation of an interface.</span></span>  
  
 <span data-ttu-id="a5f0f-105">下列範例會產生 CS0629：</span><span class="sxs-lookup"><span data-stu-id="a5f0f-105">The following sample generates CS0629:</span></span>  
  
```csharp  
// CS0629.cs  
interface MyInterface  
{  
   void MyMethod();  
}  
  
public class MyClass : MyInterface  
{  
   [System.Diagnostics.Conditional("debug")]  
   public void MyMethod()    // CS0629, remove the Conditional attribute  
   {  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
