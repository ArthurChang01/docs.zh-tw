# <a name="compiler-error-cs0244"></a>編譯器錯誤 CS0244
'is' 或 'as' 在指標類型上都無效  
  
 [Is](../../csharp/language-reference/operators/type-testing-and-cast.md#is-operator)和[as](../../csharp/language-reference/operators/type-testing-and-cast.md#as-operator)運算子不能用於指標類型。 如需詳細資訊，請參閱[不安全的程式碼和指標](../../csharp/programming-guide/unsafe-code-pointers/index.md)。  
  
 下列範例會產生 CS0244：  
  
```csharp  
// CS0244.cs  
// compile with: /unsafe  
  
class UnsafeTest  
{  
   unsafe static void SquarePtrParam (int* p)  
   {  
      bool b = p is object;   // CS0244 p is pointer  
   }  
  
   unsafe public static void Main()  
   {  
      int i = 5;  
      SquarePtrParam (&i);  
   }  
}  
```
