<span data-ttu-id="bedcb-103">不能搭配使用別名 'identifier' 與 '::'，因為別名會參考類型。</span><span class="sxs-lookup"><span data-stu-id="bedcb-103">Cannot use alias 'identifier' with '::' since the alias references a type.</span></span> <span data-ttu-id="bedcb-104">請改用 '.'。</span><span class="sxs-lookup"><span data-stu-id="bedcb-104">Use '.' instead.</span></span>  
  
 <span data-ttu-id="bedcb-105">您已搭配使用 "::" 與參考類型的別名。</span><span class="sxs-lookup"><span data-stu-id="bedcb-105">You used "::" with an alias that references a type.</span></span> <span data-ttu-id="bedcb-106">若要解決這個錯誤，請使用 "." 運算子。</span><span class="sxs-lookup"><span data-stu-id="bedcb-106">To resolve this error, use the "." operator.</span></span>  
  
 <span data-ttu-id="bedcb-107">下列範例會產生 CS0431：</span><span class="sxs-lookup"><span data-stu-id="bedcb-107">The following sample generates CS0431:</span></span>  
  
```csharp  
// CS0431.cs  
using A = Outer;  
  
public class Outer   
{  
   public class Inner   
   {  
      public static void Meth() {}  
   }  
}  
  
public class MyClass  
{  
   public static void Main()  
   {  
      A::Inner.Meth();   // CS0431  
      A.Inner.Meth();   // OK  
   }  
}  
```
