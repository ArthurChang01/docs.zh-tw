<span data-ttu-id="5c0da-103">最好不要定義名為 'global' 的別名，因為 'global::' 一定會去參考全域命名空間，而不會去參考別名。</span><span class="sxs-lookup"><span data-stu-id="5c0da-103">Defining an alias named 'global' is ill-advised since 'global::' always references the global namespace and not an alias</span></span>

## <a name="example"></a><span data-ttu-id="5c0da-105">範例</span><span class="sxs-lookup"><span data-stu-id="5c0da-105">Example</span></span>  
 <span data-ttu-id="5c0da-106">下列範例會產生 CS0440：</span><span class="sxs-lookup"><span data-stu-id="5c0da-106">The following example generates CS0440:</span></span>  
  
```  
// CS0440.cs  
// Compile with: /W:1  
  
using global = MyClass;   // CS0440  
class MyClass  
{  
    static void Main()  
    {  
        // Note how global refers to the global namespace  
        // even though it is redefined above.  
        global::System.Console.WriteLine();  
    }  
}  
```
