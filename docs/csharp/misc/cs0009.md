# 編譯器錯誤 CS0009
無法開啟中繼資料檔 'file' — 'description'  
  
 使用 [\/reference](../../csharp/language-reference/compiler-options/reference-compiler-option.md) 編譯器選項指定的檔案未包含有效的中繼資料。
