# <a name="compiler-error-cs0585"></a>編譯器錯誤 CS0585

編譯器內部錯誤：階段 'stage'

 請嘗試判斷編譯器失敗的原因是否為無法剖析未預期的語法。 如果不是這樣，請嘗試[與我們交談](/visualstudio/ide/feedback-options)。

> [!NOTE]
> 此編譯器錯誤不再在 Roslyn 中使用。
