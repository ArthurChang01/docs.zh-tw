## <a name="example"></a><span data-ttu-id="8671f-108">範例</span><span class="sxs-lookup"><span data-stu-id="8671f-108">Example</span></span>  
 <span data-ttu-id="8671f-109">下列範例會產生 CS3021：</span><span class="sxs-lookup"><span data-stu-id="8671f-109">The following example generates CS3021:</span></span>  
  
```csharp  
// CS3021.cs  
using System;  
// Uncomment the following line to declare the assembly CLS Compliant,  
// and avoid the warning without removing the attribute on the class.  
//[assembly: CLSCompliant(true)]  
  
// Remove the next line to avoid the warning.  
[CLSCompliant(false)]               // CS3021  
public class C  
{  
    public static void Main()  
    {  
    }  
}  
```
