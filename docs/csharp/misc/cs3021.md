 <span data-ttu-id="8cc78-104">如果會發生這個警告`[CLSCompliant(false)]`沒有組件層級 CLSCompliant 屬性設為 true 之組件中的類別上會出現 (也就是行`[assembly: CLSCompliant(true)]`)。</span><span class="sxs-lookup"><span data-stu-id="8cc78-104">This warning occurs if `[CLSCompliant(false)]` appears on a class in an assembly which does not have an assembly-level CLSCompliant attribute set to true (that is, the line `[assembly: CLSCompliant(true)]`).</span></span> <span data-ttu-id="8cc78-105">因為組件未將它自己宣告為符合 CLS 規範，所以組件內的任何項目都不需要將它自己宣告為不符合規範，因為它是假設為不符合規範。</span><span class="sxs-lookup"><span data-stu-id="8cc78-105">Since the assembly is not declaring itself CLS compliant, there is no need for anything within the assembly to declare itself non-compliant, since it is assumed to be non-compliant.</span></span> <span data-ttu-id="8cc78-106">如需 cls 符合性的詳細資訊，請參閱[撰寫符合 CLS 標準的程式碼](http://msdn.microsoft.com/library/4c705105-69a2-4e5e-b24e-0633bc32c7f3)。</span><span class="sxs-lookup"><span data-stu-id="8cc78-106">For more information on CLS Compliance, see [Writing CLS-Compliant Code](http://msdn.microsoft.com/library/4c705105-69a2-4e5e-b24e-0633bc32c7f3).</span></span>

## <a name="example"></a><span data-ttu-id="8cc78-108">範例</span><span class="sxs-lookup"><span data-stu-id="8cc78-108">Example</span></span>  
 <span data-ttu-id="8cc78-109">下列範例會產生 CS3021：</span><span class="sxs-lookup"><span data-stu-id="8cc78-109">The following example generates CS3021:</span></span>  
  
```csharp  
// CS3021.cs  
using System;  
// Uncomment the following line to declare the assembly CLS Compliant,  
// and avoid the warning without removing the attribute on the class.  
//[assembly: CLSCompliant(true)]  
  
// Remove the next line to avoid the warning.  
[CLSCompliant(false)]               // CS3021  
public class C  
{  
    public static void Main()  
    {  
    }  
}  
```
