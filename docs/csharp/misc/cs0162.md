# <a name="compiler-warning-level-2-cs0162"></a><span data-ttu-id="64901-102">編譯器警告 (層級 2) CS0162</span><span class="sxs-lookup"><span data-stu-id="64901-102">Compiler Warning (level 2) CS0162</span></span>
<span data-ttu-id="64901-103">偵測到不會執行到的程式碼</span><span class="sxs-lookup"><span data-stu-id="64901-103">Unreachable code detected</span></span>  
  
 <span data-ttu-id="64901-104">編譯器偵測到從未執行的程式碼。</span><span class="sxs-lookup"><span data-stu-id="64901-104">The compiler detected code that will never be executed.</span></span>  
  
 <span data-ttu-id="64901-105">下列範例會產生 CS0162：</span><span class="sxs-lookup"><span data-stu-id="64901-105">The following sample generates CS0162:</span></span>  
  
```csharp  
// CS0162.cs  
// compile with: /W:2  
public class A  
{  
   public static void Main()  
   {  
      goto lab1;  
      {  
         // The following statements cannot be reached:  
         int i = 9;   // CS0162   
         i++;  
      }  
      lab1:  
      {  
      }  
   }  
}  
```
