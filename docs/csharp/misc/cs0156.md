# <a name="compiler-error-cs0156"></a><span data-ttu-id="9af1e-102">編譯器錯誤 CS0156</span><span class="sxs-lookup"><span data-stu-id="9af1e-102">Compiler Error CS0156</span></span>
<span data-ttu-id="9af1e-103">最內層 catch 子句中巢狀 finally 子句不允許沒有引數的 throw 陳述式</span><span class="sxs-lookup"><span data-stu-id="9af1e-103">A throw statement with no arguments is not allowed in a finally clause that is nested inside the nearest enclosing catch clause</span></span>  
  
 <span data-ttu-id="9af1e-104">A[擲回](../../csharp/language-reference/keywords/throw.md)不含任何參數的陳述式只能出現在**攔截**不接受任何參數的子句。</span><span class="sxs-lookup"><span data-stu-id="9af1e-104">A [throw](../../csharp/language-reference/keywords/throw.md) statement with no parameters can only appear in a **catch** clause that takes no parameters.</span></span>  
  
 <span data-ttu-id="9af1e-105">如需詳細資訊，請參閱[例外狀況處理陳述式](../../csharp/language-reference/keywords/exception-handling-statements.md)和[例外狀況和例外狀況處理](../../csharp/programming-guide/exceptions/index.md)。</span><span class="sxs-lookup"><span data-stu-id="9af1e-105">For more information, see [Exception Handling Statements](../../csharp/language-reference/keywords/exception-handling-statements.md) and [Exceptions and Exception Handling](../../csharp/programming-guide/exceptions/index.md).</span></span>  
  
 <span data-ttu-id="9af1e-106">下列範例會產生 CS0156：</span><span class="sxs-lookup"><span data-stu-id="9af1e-106">The following sample generates CS0156:</span></span>  
  
```  
// CS0156.cs  
using System;  
  
namespace MyNamespace  
{  
   public class MyClass2 : Exception  
   {  
   }  
  
   public class MyClass  
   {  
      public static void Main()  
      {  
         try  
         {  
            throw;   // CS0156  
         }  
  
         catch(MyClass2)  
         {  
            throw;   // this throw is valid  
         }  
      }  
   }  
}  
```
