# <a name="compiler-error-cs0156"></a>編譯器錯誤 CS0156
最內層 catch 子句中巢狀 finally 子句不允許沒有引數的 throw 陳述式  
  
 沒有任何參數的 [throw](../../csharp/language-reference/keywords/throw.md) 陳述式只會出現在不接受任何參數的 **catch** 子句中。  
  
 如需詳細資訊，請參閱 < [Exception Handling Statements](../../csharp/language-reference/keywords/exception-handling-statements.md)並[例外狀況和例外狀況處理](../../csharp/programming-guide/exceptions/index.md)。  
  
 下列範例會產生 CS0156：  
  
```csharp  
// CS0156.cs  
using System;  
  
namespace MyNamespace  
{  
   public class MyClass2 : Exception  
   {  
   }  
  
   public class MyClass  
   {  
      public static void Main()  
      {  
         try  
         {  
            throw;   // CS0156  
         }  
  
         catch(MyClass2)  
         {  
            throw;   // this throw is valid  
         }  
      }  
   }  
}  
```
