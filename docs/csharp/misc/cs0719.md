 <span data-ttu-id="1df0f-104">靜態類型的陣列毫無意義，因為陣列項目是執行個體，但它不可能建立靜態類型的執行個體。</span><span class="sxs-lookup"><span data-stu-id="1df0f-104">An array of static type does not make sense since array elements are instances, but it is not possible to create instances of static types.</span></span>  
  
 <span data-ttu-id="1df0f-105">下列範例會產生 CS0719：</span><span class="sxs-lookup"><span data-stu-id="1df0f-105">The following sample generates CS0719:</span></span>  
  
```csharp  
// CS0719.cs

{  
   public static void F()  
   {  
   }  
}

{  
   public static void Main()  
   {  
      SC[] sca = new SC[10];  // CS0719  
   }  
}  
```
