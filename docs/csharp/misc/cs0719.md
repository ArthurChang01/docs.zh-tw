 <span data-ttu-id="428fb-104">靜態類型的陣列毫無意義，因為陣列項目是執行個體，但它不可能建立靜態類型的執行個體。</span><span class="sxs-lookup"><span data-stu-id="428fb-104">An array of static type does not make sense since array elements are instances, but it is not possible to create instances of static types.</span></span>

```  
// CS0719.cs

{  
   public static void F()  
   {  
   }  
}  
  
public class CMain  
{  
   public static void Main()  
   {  
      SC[] sca = new SC[10];  // CS0719  
   }  
}  
```
