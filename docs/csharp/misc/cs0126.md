# <a name="compiler-error-cs0126"></a>編譯器錯誤 CS0126
需要可轉換為 'type' 類型的物件  
  
 有 return 陳述式存在，但此陳述式未傳回預期類型的值。 如需詳細資訊，請參閱 [屬性](../../csharp/programming-guide/classes-and-structs/properties.md)。  
  
 下列範例會產生 CS0126：  
  
```csharp  
// CS0126.cs  
public class a  
{  
   public int i  
   {  
      set  
      {  
      }  
      get  
      {  
         return;   // CS0126, specify a value to return  
      }  
   }  
}  
```
