# <a name="compiler-error-cs0448"></a><span data-ttu-id="1bee8-102">編譯器錯誤 CS0448</span><span class="sxs-lookup"><span data-stu-id="1bee8-102">Compiler Error CS0448</span></span>
<span data-ttu-id="1bee8-103">++ 或 -- 運算子的傳回類型必須是包含類型或衍生自包含類型</span><span class="sxs-lookup"><span data-stu-id="1bee8-103">The return type for ++ or -- operator must be the containing type or derived from the containing type</span></span>  
  
 <span data-ttu-id="1bee8-104">當您覆寫 `++` 或 `--` 運算子時，這些運算子必須傳回與包含類型相同的類型，或傳回衍生自包含類型的類型。</span><span class="sxs-lookup"><span data-stu-id="1bee8-104">When you override the `++` or `--` operators, they must return the same type as the containing type, or return a type that is derived from the containing type.</span></span>  
  
## <a name="example"></a><span data-ttu-id="1bee8-105">範例</span><span class="sxs-lookup"><span data-stu-id="1bee8-105">Example</span></span>  
 <span data-ttu-id="1bee8-106">下列範例會產生 CS0448。</span><span class="sxs-lookup"><span data-stu-id="1bee8-106">The following sample generates CS0448.</span></span>  
  
```  
// CS0448.cs  
class C5  
{  
   public static int operator ++(C5 c) { return null; }   // CS0448  
   public static C5 operator --(C5 c) { return null; }   // OK  
   public static void Main() {}  
}  
```  
  
## <a name="example"></a><span data-ttu-id="1bee8-107">範例</span><span class="sxs-lookup"><span data-stu-id="1bee8-107">Example</span></span>  
 <span data-ttu-id="1bee8-108">下列範例會產生 CS0448。</span><span class="sxs-lookup"><span data-stu-id="1bee8-108">The following sample generates CS0448.</span></span>  
  
```  
// CS0448_b.cs  
public struct S  
{  
   public static S? operator ++(S s) { return new S(); }   // CS0448  
   public static S? operator --(S s) { return new S(); }   // CS0448  
}  
  
public struct T  
{  
// OK  
   public static T operator --(T t) { return new T(); }  
   public static T operator ++(T t) { return new T(); }  
  
   public static T? operator --(T? t) { return new T(); }  
   public static T? operator ++(T? t) { return new T(); }  
  
   public static void Main() {}  
}  
```
