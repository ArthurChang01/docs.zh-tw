# <a name="compiler-warning-level-1-cs3004"></a>編譯器警告 (層級 1) CS3004
混合和分解的 Unicode 字元不符合 CLS 規範  
  
 為了符合 Common Language Specification (CLS) 規範，只能在 [public](../../csharp/language-reference/keywords/public.md)、 [protected](../../csharp/language-reference/keywords/protected.md)、 or `protected internal` 識別項中使用組成的 UNICODE 字元。 如需 CLS 合規性的詳細資訊，請參閱[撰寫符合 CLS 標準的程式碼](http://msdn.microsoft.com/library/4c705105-69a2-4e5e-b24e-0633bc32c7f3)和[語言獨立性以及與語言無關的元件](../../../docs/standard/language-independence-and-language-independent-components.md)。
