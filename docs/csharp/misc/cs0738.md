# <a name="compiler-error-cs0738"></a><span data-ttu-id="02929-102">編譯器錯誤 CS0738</span><span class="sxs-lookup"><span data-stu-id="02929-102">Compiler Error CS0738</span></span>
<span data-ttu-id="02929-103">'type name' 未實作介面成員 'member name'。</span><span class="sxs-lookup"><span data-stu-id="02929-103">'type name' does not implement interface member 'member name'.</span></span> <span data-ttu-id="02929-104">'method name' 無法實作 'interface member'，因為它沒有符合的傳回類型 'type name'。</span><span class="sxs-lookup"><span data-stu-id="02929-104">'method name' cannot implement 'interface member' because it does not have the matching return type of ' type name'.</span></span>  
  
 <span data-ttu-id="02929-105">類別中實作方法的傳回值必須符合其所實作之介面成員的傳回值。</span><span class="sxs-lookup"><span data-stu-id="02929-105">The return value of an implementing method in a class must match the return value of the interface member that it implements.</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="02929-106">更正這個錯誤</span><span class="sxs-lookup"><span data-stu-id="02929-106">To correct this error</span></span>

## <a name="example"></a><span data-ttu-id="02929-108">範例</span><span class="sxs-lookup"><span data-stu-id="02929-108">Example</span></span>  
 <span data-ttu-id="02929-109">因為類別方法傳回 `void` ，而且同名的介面成員傳回 `int`，所以下列程式碼會產生 CS0738：</span><span class="sxs-lookup"><span data-stu-id="02929-109">The following code generates CS0738 because the class method returns `void` and the interface member of the same name returns `int`:</span></span>

```csharp  
interface ITest  
{  
    int TestMethod();  
}  
public class Test: ITest  
{  
    public void TestMethod() { } // CS0738  
    // Try the following line instead.  
    // public int TestMethod();  
}  
```
