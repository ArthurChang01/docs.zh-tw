## <a name="example"></a><span data-ttu-id="05e36-108">範例</span><span class="sxs-lookup"><span data-stu-id="05e36-108">Example</span></span>  
 <span data-ttu-id="05e36-109">因為類別方法傳回 `void` ，而且同名的介面成員傳回 `int`，所以下列程式碼會產生 CS0738：</span><span class="sxs-lookup"><span data-stu-id="05e36-109">The following code generates CS0738 because the class method returns `void` and the interface member of the same name returns `int`:</span></span>

```  
{  
    int TestMethod();  
}

    // Try the following line instead.  
    // public int TestMethod();

```
