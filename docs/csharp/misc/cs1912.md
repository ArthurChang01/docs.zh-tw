# <a name="compiler-error-cs1912"></a><span data-ttu-id="004ea-102">編譯器錯誤 CS1912</span><span class="sxs-lookup"><span data-stu-id="004ea-102">Compiler Error CS1912</span></span>
<span data-ttu-id="004ea-103">成員 'name' 的初始設定重複。</span><span class="sxs-lookup"><span data-stu-id="004ea-103">Duplicate initialization of member 'name'.</span></span>  
  
 <span data-ttu-id="004ea-104">物件初始設定式一次只能初始化一個成員。</span><span class="sxs-lookup"><span data-stu-id="004ea-104">An object initializer can initialize each member only one time.</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="004ea-105">更正這個錯誤</span><span class="sxs-lookup"><span data-stu-id="004ea-105">To correct this error</span></span>  
  
1. <span data-ttu-id="004ea-106">移除物件初始設定式中的第二個成員初始化。</span><span class="sxs-lookup"><span data-stu-id="004ea-106">Remove the second initialization of the member in the object initializer.</span></span>  
  
## <a name="example"></a><span data-ttu-id="004ea-107">範例</span><span class="sxs-lookup"><span data-stu-id="004ea-107">Example</span></span>  
 <span data-ttu-id="004ea-108">下列程式碼會產生 CS1912，因為 `memberA` 已重複初始化：</span><span class="sxs-lookup"><span data-stu-id="004ea-108">The following code generates CS1912 because `memberA` is initialized two times:</span></span>  
  
```csharp  
// cs1912.cs  
using System.Linq;  
  
public class TestClass  
{  
    public int memberA { get; set; }  
    public int memberB { get; set; }  
}  
  
public class Test  
{  
    static void Main()  
    {  
        TestClass tc = new TestClass() { memberA = 5, memberA = 6, memberB = 2}; // CS1912  
    }  
}  
```
