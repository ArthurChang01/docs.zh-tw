# <a name="compiler-error-cs0756"></a><span data-ttu-id="29186-102">編譯器錯誤 CS0756</span><span class="sxs-lookup"><span data-stu-id="29186-102">Compiler Error CS0756</span></span>

 <span data-ttu-id="29186-104">部分方法的定義宣告是指定方法簽章的部分，而不是實作 (方法主體)。</span><span class="sxs-lookup"><span data-stu-id="29186-104">The defining declaration of a partial method is the part that specifies the method signature, but not the implementation (method body).</span></span> <span data-ttu-id="29186-105">部分方法針對每個唯一的簽章只能有一個定義宣告。</span><span class="sxs-lookup"><span data-stu-id="29186-105">A partial method must have exactly one defining declaration for each unique signature.</span></span> <span data-ttu-id="29186-106">部分方法的每個多載版本必須有自己的定義宣告。</span><span class="sxs-lookup"><span data-stu-id="29186-106">Each overloaded version of a partial method must have its own defining declaration.</span></span>
