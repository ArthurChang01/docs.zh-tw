# <a name="compiler-error-cs2007"></a>編譯器錯誤 CS2007
無法辨認的命令列選項: 'option'  
  
 編譯器所傳遞的字串不是 [編譯器選項](../../csharp/language-reference/compiler-options/index.md)，即使它的開頭為正斜線 (/) 也是一樣。  
  
 下列範例會產生 CS2007：  
  
```csharp  
// CS2007.cs  
// compile with: /recur  
// CS2007 expected  
class x  
{  
   public static void Main() {}  
}  
```
