# <a name="compiler-error-cs0754"></a><span data-ttu-id="e796f-102">編譯器錯誤 CS0754</span><span class="sxs-lookup"><span data-stu-id="e796f-102">Compiler Error CS0754</span></span>

## <a name="example"></a><span data-ttu-id="e796f-107">範例</span><span class="sxs-lookup"><span data-stu-id="e796f-107">Example</span></span>  
 <span data-ttu-id="e796f-108">下列程式碼會產生 CS0754：</span><span class="sxs-lookup"><span data-stu-id="e796f-108">The following code generates CS0754:</span></span>  
  
```csharp  
// cs0754.cs  
using System;  
  
    public interface IF  
    {  
        void Part();  
    }  
  
    public partial class C : IF  
    {  
        partial void IF.Part(); //CS0754  
        public static int Main()  
        {  
            return 1;  
        }  
    }  
```
