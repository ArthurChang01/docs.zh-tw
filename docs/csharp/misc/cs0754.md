# <a name="compiler-error-cs0754"></a>編譯器錯誤 CS0754
部分方法不可明確地實作介面方法。  
  
 部分方法不可以宣告為介面中所定義方法的明確實作。  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
1. 請從方法宣告中移除明確介面限定性條件。  
  
## <a name="example"></a>範例  
 下列程式碼會產生 CS0754：  
  
```csharp  
// cs0754.cs  
using System;  
  
    public interface IF  
    {  
        void Part();  
    }  
  
    public partial class C : IF  
    {  
        partial void IF.Part(); //CS0754  
        public static int Main()  
        {  
            return 1;  
        }  
    }  
```
