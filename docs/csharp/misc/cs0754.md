## <a name="example"></a><span data-ttu-id="dca43-107">範例</span><span class="sxs-lookup"><span data-stu-id="dca43-107">Example</span></span>  
 <span data-ttu-id="dca43-108">下列程式碼會產生 CS0754：</span><span class="sxs-lookup"><span data-stu-id="dca43-108">The following code generates CS0754:</span></span>  
  
```csharp  
// cs0754.cs  
using System;  
  
    public interface IF  
    {  
        void Part();  
    }  
  
    public partial class C : IF  
    {  
        partial void IF.Part(); //CS0754  
        public static int Main()  
        {  
            return 1;  
        }  
    }  
```
