# <a name="compiler-error-cs0014"></a>編譯器錯誤 CS0014
找不到所需的檔案 'file'  
  
 編譯器需要檔案，但檔案不在系統上。 請確認路徑是否正確。 如果檔案是[!INCLUDE[vsprvs](../../csharp/includes/vsprvs_md.md)]系統檔案，則您可能需要修復您的安裝或其他移除並重新安裝[!INCLUDE[vsprvs](../../csharp/includes/vsprvs_md.md)]完全。
