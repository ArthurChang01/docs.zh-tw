# <a name="compiler-error-cs0014"></a>編譯器錯誤 CS0014
找不到所需的檔案 'file'  
  
 編譯器需要檔案，但檔案不在系統上。 請確認路徑是否正確。 如果檔案是 Visual Studio 系統檔案，您可能需要修復您安裝或移除，並完全重新安裝 Visual Studio。
