# <a name="compiler-error-cs1525"></a><span data-ttu-id="96ff8-102">編譯器錯誤 CS1525</span><span class="sxs-lookup"><span data-stu-id="96ff8-102">Compiler Error CS1525</span></span>
<span data-ttu-id="96ff8-103">無效的運算式詞彙 'character'</span><span class="sxs-lookup"><span data-stu-id="96ff8-103">Invalid expression term 'character'</span></span>  
  
 <span data-ttu-id="96ff8-104">編譯器偵測到運算式中有無效的字元。</span><span class="sxs-lookup"><span data-stu-id="96ff8-104">The compiler detected an invalid character in an expression.</span></span>  
  
 <span data-ttu-id="96ff8-105">下列範例會產生 CS1525：</span><span class="sxs-lookup"><span data-stu-id="96ff8-105">The following sample generates CS1525:</span></span>  
  
```  
// CS1525.cs  
class x  
{  
   public static void Main()  
   {  
      int i = 0;  
      i = i +   // OK - identifier  
      'c' +     // OK - character  
      (5) +     // OK - parenthesis  
      [ +       // CS1525, operator not a valid expression element  
      throw +   // CS1525, keyword not allowed in expression  
      void;     // CS1525, void not allowed in expression  
   }  
}  
```  
  
 <span data-ttu-id="96ff8-106">空白標籤也會產生 CS1525，如下列範例所示：</span><span class="sxs-lookup"><span data-stu-id="96ff8-106">An empty label can also generate CS1525, as in the following sample:</span></span>  
  
```  
// CS1525b.cs  
using System;  
public class MyClass  
{  
   public static void Main()  
   {  
      goto FoundIt;  
      FoundIt:      // CS1525  
      // Uncomment the following line to resolve:  
      // System.Console.Write("Hello");  
   }  
}  
```
