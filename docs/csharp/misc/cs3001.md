# <a name="compiler-warning-level-1-cs3001"></a>編譯器警告 (層級 1) CS3001
引數類型 'type' 不符合 CLS 標準  
  
 [public](../language-reference/keywords/public.md)、 [protected](../language-reference/keywords/protected.md)、 or `protected internal` 方法必須接受其類型符合 Common Language Specification (CLS) 標準的參數。 如需 CLS 符合性的詳細資訊, 請參閱[語言獨立性和與語言無關的元件](../../standard/language-independence.md)。  
  
## <a name="example"></a>範例  
 下列範例會產生 CS3001：  
  
```csharp  
// CS3001.cs  
  
[assembly:System.CLSCompliant(true)]  
public class a  
{  
    public void bad(ushort i)   // CS3001  
    {  
    }  
  
    private void OK(ushort i)   // OK, private method  
    {  
    }  
  
    public static void Main()  
    {  
    }  
}  
```
