# <a name="compiler-warning-level-1-cs1030"></a>編譯器警告 (層級 1) CS1030
\#警告: 'text'  
  
 會顯示以 [#warning](../../csharp/language-reference/preprocessor-directives/preprocessor-warning.md) 指示詞定義的警告文字。  
  
 下列範例示範如何建立使用者定義的警告：  
  
```csharp  
// CS1030.cs  
class Sample  
{  
   static void Main()  
   {  
      #warning Let's give a warning here   // CS1030  
   }  
}  
```
