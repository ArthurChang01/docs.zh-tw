# <a name="compiler-error-cs1670"></a>編譯器錯誤 CS1670
params 在此內容中無效  
  
 許多 C# 功能與變數引數清單不相容，且不允許`params`關鍵字，包括下列：  
  
-   匿名方法的參數清單  
  
-   多載運算子  
  
## <a name="example"></a>範例  
 下列範例會產生 CS1670：  
  
```  
// CS1670.cs  
public class C  
{  
    public bool operator +(params int[] paramsList)  // CS1670  
    {  
        return false;  
    }  
  
    static void Main()  
    {  
    }  
}  
```
