# <a name="compiler-error-cs0020"></a>編譯器錯誤 CS0020
除以常數零  
  
 運算式在除法運算的分母中使用常值 (非變數) 零。 除數為零未定義，因此無效。  
  
 下列範例會產生 CS0020：  
  
```  
// CS0020.cs  
namespace x  
{  
   public class b  
   {  
      public static void Main()  
      {  
         1 / 0;   // CS0020  
      }  
   }  
}  
```
