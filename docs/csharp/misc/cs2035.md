# <a name="compiler-error-cs2035"></a>編譯器錯誤 CS2035
命令列語法錯誤： 遺漏 ':\<編號 >' 的 'compiler_option' 選項  
  
 有一些編譯器選項需要值。  
  
## <a name="example"></a>範例  
 下列範例會產生 CS2035。  
  
```csharp  
// CS2035.cs  
// compile with: /baseaddress  
// CS2035 expected  
```
