# <a name="compiler-error-cs0689"></a>編譯器錯誤 CS0689
無法由 'identifier' 衍生，因為它是類型參數  
  
 類型參數不能指定泛型類別的基底類別或介面。 而是衍生自特定的類別或介面，或特定的泛型類別，或是包含為成員的未知類型。  
  
 下列範例會產生 CS0689：  
  
```csharp 
// CS0689.cs  
class A<T> : T   // CS0689  
{  
}  
```
