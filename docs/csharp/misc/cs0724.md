## <a name="example"></a><span data-ttu-id="b503b-105">範例</span><span class="sxs-lookup"><span data-stu-id="b503b-105">Example</span></span>  
 <span data-ttu-id="b503b-106">下列範例會產生 CS0724。</span><span class="sxs-lookup"><span data-stu-id="b503b-106">The following example generates CS0724.</span></span>  
  
```csharp  
// CS0724.cs  
using System;  
  
class X  
{  
    static void Test()  
    {  
        try  
        {  
            throw new Exception();  
        }  
        catch  
        {  
            try  
            {  
            }  
            finally  
            {  
                throw; // CS0724  
            }  
        }  
    }  
  
    static void Main()  
    {  
    }  
}  
```
