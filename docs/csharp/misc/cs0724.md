# <a name="compiler-error-cs0724"></a>編譯器錯誤 CS0724
不需要 CLSCompliant 屬性，因為組件並沒有 CLSCompliant 屬性  
  
 下列範例會產生 CS0724，因為 `throw` 陳述式位於 `finally` 子句區塊內。  
  
## <a name="example"></a>範例  
 下列範例會產生 CS0724。  
  
```csharp  
// CS0724.cs  
using System;  
  
class X  
{  
    static void Test()  
    {  
        try  
        {  
            throw new Exception();  
        }  
        catch  
        {  
            try  
            {  
            }  
            finally  
            {  
                throw; // CS0724  
            }  
        }  
    }  
  
    static void Main()  
    {  
    }  
}  
```
