# <a name="compiler-error-cs0758"></a><span data-ttu-id="c1f9e-102">編譯器錯誤 CS0758</span><span class="sxs-lookup"><span data-stu-id="c1f9e-102">Compiler Error CS0758</span></span>
<span data-ttu-id="c1f9e-103">兩個部分方法宣告都必須使用 params 參數，或兩者都不使用 params 參數</span><span class="sxs-lookup"><span data-stu-id="c1f9e-103">Both partial method declarations must use a params parameter or neither may use a params parameter</span></span>  
  
 <span data-ttu-id="c1f9e-104">如果部分方法有一個部分指定 `params` 參數，則另一個部分也必須指定相同參數。</span><span class="sxs-lookup"><span data-stu-id="c1f9e-104">If one part of a partial method specifies a `params` parameter, the other part must specify one also.</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="c1f9e-105">更正這個錯誤</span><span class="sxs-lookup"><span data-stu-id="c1f9e-105">To correct this error</span></span>

## <a name="example"></a><span data-ttu-id="c1f9e-107">範例</span><span class="sxs-lookup"><span data-stu-id="c1f9e-107">Example</span></span>  
 <span data-ttu-id="c1f9e-108">下列程式碼會產生 CS0758：</span><span class="sxs-lookup"><span data-stu-id="c1f9e-108">The following code generates CS0758:</span></span>

    public partial class C

        partial void Part(int i, params char[] array);

        }  
  
    }
