# <a name="compiler-error-cs0075"></a><span data-ttu-id="1cf0a-102">編譯器錯誤 CS0075</span><span class="sxs-lookup"><span data-stu-id="1cf0a-102">Compiler Error CS0075</span></span>
<span data-ttu-id="1cf0a-103">要做轉型的負值必須置於括號之內</span><span class="sxs-lookup"><span data-stu-id="1cf0a-103">To cast a negative value, you must enclose the value in parentheses</span></span>  
  
 <span data-ttu-id="1cf0a-104">如果您使用會識別預先定義之類型的關鍵字來將類型轉型，則不需要括弧。</span><span class="sxs-lookup"><span data-stu-id="1cf0a-104">If you are casting using a keyword that identifies a predefined type, then you do not need parentheses.</span></span> <span data-ttu-id="1cf0a-105">否則，您必須加上括弧，因為 (x) –y 不會視為轉型運算式。</span><span class="sxs-lookup"><span data-stu-id="1cf0a-105">Otherwise, you must put the parentheses because (x) –y will not be considered a cast expression.</span></span> <span data-ttu-id="1cf0a-106">取自 C# 規格的第 7.6.6 節：</span><span class="sxs-lookup"><span data-stu-id="1cf0a-106">From the C# Specification, Section 7.6.6:</span></span>  
  
 <span data-ttu-id="1cf0a-107">*從去除混淆規則，如果 x 和 y 是識別項，(x) y、 (得出，和 (也一樣是轉型運算式，但 (x)-y 不是，即使 x 識別類型。不過，如果 x 是識別預先定義的類型 （例如 int) 關鍵字，然後四種形式全都是轉型運算式 （因為這類關鍵字本身不可能會是一個運算式）。*</span><span class="sxs-lookup"><span data-stu-id="1cf0a-107">*From the disambiguation rule it follows that, if x and y are identifiers, (x)y, (x)(y), and (x)(-y) are cast-expressions, but (x)-y is not, even if x identifies a type. However, if x is a keyword that identifies a predefined type (such as int), then all four forms are cast-expressions (because such a keyword could not possibly be an expression by itself).*</span></span>  
  
 <span data-ttu-id="1cf0a-108">下列程式碼會產生 CS0075：</span><span class="sxs-lookup"><span data-stu-id="1cf0a-108">The following code generates CS0075:</span></span>  
  
```  
// CS0075  
namespace MyNamespace  
{  
    enum MyEnum { }  
    public class MyClass  
    {  
        public static void Main()  
        {  
            // To fix the error, place the negative  
            // values below in parentheses  
            int i = (System.Int32) - 4; //CS0075  
            MyEnum e = (MyEnum) - 1;    //CS0075  
            System.Console.WriteLine(i); //to avoid warning  
            System.Console.WriteLine(e); //to avoid warning  
        }  
    }  
}  
```
