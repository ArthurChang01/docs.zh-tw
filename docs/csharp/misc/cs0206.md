## <a name="example"></a><span data-ttu-id="bcfe4-106">範例</span><span class="sxs-lookup"><span data-stu-id="bcfe4-106">Example</span></span>  
 <span data-ttu-id="bcfe4-107">下列範例會產生 CS0206：</span><span class="sxs-lookup"><span data-stu-id="bcfe4-107">The following sample generates CS0206:</span></span>  
  
```csharp  
// CS0206.cs  
public class MyClass  
{  
    public static int P  
    {  
        get  
        {  
            return 0;  
        }  
        set  
        {  
        }  
    }  
  
    public static void MyMeth(ref int i)  
    // public static void MyMeth(int i)  
    {  
    }  
  
    public static void Main()  
    {  
        MyMeth(ref P);   // CS0206  
        // try the following line instead  
        // MyMeth(P);   // CS0206  
    }  
}  
```
