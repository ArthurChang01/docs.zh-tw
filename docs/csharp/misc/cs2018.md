# 編譯器錯誤 CS2018
找不到訊息檔案 'cscmsgs.dll'  
  
 找不到包含編譯器錯誤和警告訊息的 .dll 檔案。 這個檔案必須位於與其他編譯器支援檔相同的目錄中。
