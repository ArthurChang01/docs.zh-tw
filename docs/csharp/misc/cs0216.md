# <a name="compiler-error-cs0216"></a><span data-ttu-id="22da3-102">編譯器錯誤 CS0216</span><span class="sxs-lookup"><span data-stu-id="22da3-102">Compiler Error CS0216</span></span>
<span data-ttu-id="22da3-103">運算子 'operator' 的配對運算子 'missing_operator' 也需要定義</span><span class="sxs-lookup"><span data-stu-id="22da3-103">The operator 'operator' requires a matching operator 'missing_operator' to also be defined</span></span>  
  
 <span data-ttu-id="22da3-104">使用者定義的 [true](../../csharp/language-reference/keywords/true.md) 運算子需要使用者定義的 [false](../../csharp/language-reference/keywords/false.md) 運算子，反之亦然。</span><span class="sxs-lookup"><span data-stu-id="22da3-104">A user-defined [true](../../csharp/language-reference/keywords/true.md) operator requires a user-defined [false](../../csharp/language-reference/keywords/false.md) operator, and vice versa.</span></span> <span data-ttu-id="22da3-105">如需詳細資訊，請參閱[運算子](../../csharp/programming-guide/statements-expressions-operators/operators.md)。</span><span class="sxs-lookup"><span data-stu-id="22da3-105">For more information, see [Operators](../../csharp/programming-guide/statements-expressions-operators/operators.md).</span></span>  
  
 <span data-ttu-id="22da3-106">下列範例會產生 CS0216：</span><span class="sxs-lookup"><span data-stu-id="22da3-106">The following sample generates CS0216:</span></span>  
  
```csharp  
// CS0216.cs  
class MyClass  
{  
   public static bool operator true (MyClass MyInt)   // CS0216  
   {  
      return true;  
   }  
  
   // to resolve, uncomment the following operator definition  
   /*  
   public static bool operator false (MyClass MyInt)  
   {  
      return true;  
   }  
   */  
  
   public static void Main()  
   {  
   }  
}  
```
