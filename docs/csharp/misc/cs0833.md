# <a name="compiler-error-cs0833"></a>編譯器錯誤 CS0833
匿名類型不可具有多個同名的屬性。  
  
 匿名類型就像任何類型，不能有兩個同名的屬性。  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
1.  請為類型中的每個屬性指定唯一名稱。  
  
## <a name="example"></a>範例  
 下列範例會產生 CS0833：  
  
```csharp  
// cs0833.cs  
using System;  
  
public class C  
{  
    public static int Main()  
    {  
        var c = new { p1 = 1, p1 = 2 }; // CS0833  
        return 1;  
    }  
}  
```
