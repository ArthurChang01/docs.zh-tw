# <a name="compiler-error-cs1057"></a>編譯器錯誤 CS1057
'member': 靜態類別不能包含 protected 的成員  
  
 在靜態類別內宣告 Protected 成員，即會產生這個錯誤。  
  
## <a name="example"></a>範例  
 下列範例會產生 CS1057。  
  
```csharp  
// CS1057.cs  
using System;  
  
static class Class1  
{  
    protected static int x;   // CS1057  
    public static void Main()  
    {  
    }  
}  
```
