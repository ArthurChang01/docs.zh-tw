 <span data-ttu-id="6c516-104">[In](../language-reference/keywords/in-parameter-modifier.md)、 [ref](../language-reference/keywords/ref.md)或[out](../language-reference/keywords/out-parameter-modifier.md)的關鍵字不能與[params](../language-reference/keywords/params.md)關鍵字搭配使用。</span><span class="sxs-lookup"><span data-stu-id="6c516-104">The keywords [in](../language-reference/keywords/in-parameter-modifier.md), [ref](../language-reference/keywords/ref.md) or [out](../language-reference/keywords/out-parameter-modifier.md) cannot be used with the [params](../language-reference/keywords/params.md) keyword.</span></span>  
  
 <span data-ttu-id="6c516-105">下列範例會產生 CS1611：</span><span class="sxs-lookup"><span data-stu-id="6c516-105">The following sample generates CS1611:</span></span>  
  
```csharp  
// CS1611.cs  
public class MyClass  
{  
   public static void Test(params ref int[] a)   // CS1611, remove ref  
   {  
   }  
  
   public static void Main()  
   {  
      Test(1);  
   }  
}  
```
