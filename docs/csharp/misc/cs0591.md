# 編譯器錯誤 CS0591
對 'attribute' 屬性的引數值無效  
  
 將無效的引數或兩個互斥的引數傳遞至屬性。  
  
## 範例  
 下列範例會產生 CS0591：  
  
```  
// CS0591.cs using System; [AttributeUsage(0)]   // CS0591 class I: Attribute { } public class a { public static void Main() { } }  
```
