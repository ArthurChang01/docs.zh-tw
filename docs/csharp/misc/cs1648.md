# <a name="compiler-error-cs1648"></a><span data-ttu-id="84324-102">編譯器錯誤 CS1648</span><span class="sxs-lookup"><span data-stu-id="84324-102">Compiler Error CS1648</span></span>
<span data-ttu-id="84324-103">唯讀欄位 'identifier' 的成員無法修改 (除非是在建構函式或變數初始設定式中)</span><span class="sxs-lookup"><span data-stu-id="84324-103">Members of readonly field 'identifier' cannot be modified (except in a constructor or a variable initializer)</span></span>  
  
 <span data-ttu-id="84324-104">當您嘗試修改不允許修改的唯讀欄位成員時，就會發生這個錯誤。</span><span class="sxs-lookup"><span data-stu-id="84324-104">This error occurs when you attempt to modify a member of a field which is readonly where it is not allowed to be modified.</span></span> <span data-ttu-id="84324-105">若要解決這個錯誤，請將唯讀欄位的指派限制為建構函式或變數初始設定式，或移除欄位宣告的唯讀關鍵字。</span><span class="sxs-lookup"><span data-stu-id="84324-105">To resolve this error, limit assignments to readonly fields to the constructor or variable initializer, or remove the readonly keyword from the declaration of the field.</span></span>  
  
 <span data-ttu-id="84324-106">下列範例會產生 CS1648：</span><span class="sxs-lookup"><span data-stu-id="84324-106">The following sample generates CS1648:</span></span>  
  
```csharp  
// CS1648.cs  
public struct Inner  
  {  
    public int i;  
  }  
  
class Outer  
{    
  public readonly Inner inner = new Inner();  
}  
  
class D  
{  
   static void Main()  
   {  
      Outer outer = new Outer();  
      outer.inner.i = 1;  // CS1648  
   }  
}  
```
