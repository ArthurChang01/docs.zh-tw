## <a name="example"></a><span data-ttu-id="e3618-106">範例</span><span class="sxs-lookup"><span data-stu-id="e3618-106">Example</span></span>  
 <span data-ttu-id="e3618-107">下列範例會產生 CS0136：</span><span class="sxs-lookup"><span data-stu-id="e3618-107">The following sample generates CS0136:</span></span>  
  
```csharp
// CS0136.cs  
namespace MyNamespace  
{  
   public class MyClass  
   {  
      public static void Main()  
      {  
         int i = 0;  
         {  
            char i = 'a';   // CS0136, hides int i  
         }  
         i++;  
      }  
   }  
}  
```
