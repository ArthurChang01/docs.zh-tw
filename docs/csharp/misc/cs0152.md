 <span data-ttu-id="be171-106">下列範例會產生 CS0152：</span><span class="sxs-lookup"><span data-stu-id="be171-106">The following sample generates CS0152:</span></span>  
  
```  
// CS0152.cs  
namespace MyNamespace  
{  
   public class MyClass  
   {  
      public static void Main()  
      {  
         int i = 0;  
  
         switch (i)  
         {  
            case 1:  
               i++;  
               return;  
  
            case 1:   // CS0152, two case 1 statements  
               i++;  
               return;  
         }  
      }  
   }  
}  
```
