# <a name="compiler-error-cs0757"></a><span data-ttu-id="4c286-102">編譯器錯誤 CS0757</span><span class="sxs-lookup"><span data-stu-id="4c286-102">Compiler Error CS0757</span></span>
<span data-ttu-id="4c286-103">部分方法不能有多重實作的宣告。</span><span class="sxs-lookup"><span data-stu-id="4c286-103">A partial method may not have multiple implementing declarations.</span></span>  
  
 <span data-ttu-id="4c286-104">部分方法是由一個定義宣告 (簽章) 以及一個或零個實作宣告 (主體) 所組成。</span><span class="sxs-lookup"><span data-stu-id="4c286-104">A partial method consists of exactly one defining declaration (signature) and one or zero implementing declarations (body).</span></span> <span data-ttu-id="4c286-105">同一個定義宣告不能有多個實作宣告。</span><span class="sxs-lookup"><span data-stu-id="4c286-105">Multiple implementing declarations for the same identical defining declarations are not allowed.</span></span> <span data-ttu-id="4c286-106">部分方法可以多載，而且每個多載的版本可以有一個或零個實作宣告。</span><span class="sxs-lookup"><span data-stu-id="4c286-106">Partial methods may be overloaded, and each overloaded version may have one or zero implementing declarations.</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="4c286-107">更正這個錯誤</span><span class="sxs-lookup"><span data-stu-id="4c286-107">To correct this error</span></span>

## <a name="example"></a><span data-ttu-id="4c286-109">範例</span><span class="sxs-lookup"><span data-stu-id="4c286-109">Example</span></span>  
 <span data-ttu-id="4c286-110">下列範例會產生 CS0757：</span><span class="sxs-lookup"><span data-stu-id="4c286-110">The following example generates CS0757:</span></span>  
  
```csharp  
// cs0757.cs  
using System;  
  
    public partial class C  
    {  
        // Defining declaration.  
        partial void Part();  
  
        // Implementing declaration.  
        partial void Part()  
        {  
            //...Do something.  
        }  
  
        // Second implementing declaration.  
        partial void Part() // CS0757  
        {  
            //...Do something.  
        }  
  
        public static int Main()  
        {  
            return 1;  
        }  
    }  
```
