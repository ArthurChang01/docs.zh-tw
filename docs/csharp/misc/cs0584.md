# 編譯器錯誤 CS0584
編譯器內部錯誤：階段 'stage' 符號 'symbol'  
  
 請嘗試判斷編譯器失敗的原因是否為無法剖析未預期的語法。 如果不是這種狀況，請嘗試[告訴我們](/visual-studio/ide/talk-to-us)。
