# <a name="compiler-error-cs0415"></a><span data-ttu-id="8ee6b-102">編譯器錯誤 CS0415</span><span class="sxs-lookup"><span data-stu-id="8ee6b-102">Compiler Error CS0415</span></span>
<span data-ttu-id="8ee6b-103">'IndexerName' 屬性只在非明確介面成員宣告的索引子上有效</span><span class="sxs-lookup"><span data-stu-id="8ee6b-103">The 'IndexerName' attribute is valid only on an indexer that is not an explicit interface member declaration</span></span>  
  
 <span data-ttu-id="8ee6b-104">如果您在明確介面實作的索引子上使用 IndexerName 屬性，就會發生這個錯誤。</span><span class="sxs-lookup"><span data-stu-id="8ee6b-104">This error occurs if you use an IndexerName attribute on an indexer that was an explicit implementation of an interface.</span></span> <span data-ttu-id="8ee6b-105">請盡可能從索引子的宣告中移除介面名稱，以避免發生這個錯誤。</span><span class="sxs-lookup"><span data-stu-id="8ee6b-105">This error may be avoided by removing the interface name from the declaration of the indexer, if possible.</span></span> <span data-ttu-id="8ee6b-106">如需詳細資訊，請參閱 [IndexerNameAttribute 類別](xref:System.Runtime.CompilerServices.IndexerNameAttribute)。</span><span class="sxs-lookup"><span data-stu-id="8ee6b-106">For more information, see the [IndexerNameAttribute Class](xref:System.Runtime.CompilerServices.IndexerNameAttribute).</span></span>  
  
 <span data-ttu-id="8ee6b-107">下列範例會產生 CS0415：</span><span class="sxs-lookup"><span data-stu-id="8ee6b-107">The following sample generates CS0415:</span></span>  
  
```csharp
// CS0415.cs  
using System;  
using System.Runtime.CompilerServices;  
  
public interface IA  
{  
    int this[int index]  
    {  
        get;  
        set;  
    }  
}  
  
public class A : IA  
{  
    [IndexerName("Item")]  // CS0415  
    int IA.this[int index]  
    // Try this line instead:  
    // public int this[int index]  
    {  
        get { return 0; }  
        set { }  
    }  
  
    static void Main()  
    {  
    }  
}  
```
