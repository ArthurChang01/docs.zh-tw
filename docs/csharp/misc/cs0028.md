# <a name="compiler-warning-level-4-cs0028"></a>編譯器警告 (層級 4) CS0028
'function declaration' 的進入點簽章錯誤  
  
 `Main` 的方法宣告無效：它是使用無效的簽章進行宣告。 `Main` 必須宣告為靜態，而且必須傳回 [int](../language-reference/builtin-types/integral-numeric-types.md) 或 [void](../language-reference/builtin-types/void.md)。 如需詳細資訊，請參閱[Main （）和命令列引數](../programming-guide/main-and-command-args/index.md)。  
  
 下列範例會產生 CS0028：  
  
```csharp  
// CS0028.cs  
// compile with: /W:4 /warnaserror  
public class a  
{  
    public static double Main(int i)   // CS0028  
    // Try the following line instead:  
    // public static void Main()  
    {  
    }  
}  
```
