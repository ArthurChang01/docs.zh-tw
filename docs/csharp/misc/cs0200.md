## <a name="examples"></a><span data-ttu-id="bf6ac-107">範例</span><span class="sxs-lookup"><span data-stu-id="bf6ac-107">Examples</span></span>
<span data-ttu-id="bf6ac-108">下列範例會產生 CS0200：</span><span class="sxs-lookup"><span data-stu-id="bf6ac-108">The following sample generates CS0200:</span></span>

```csharp
// CS0200.cs
public class Example
{
    private int _mi;
    int I
    {
        get
        {
            return _mi;
        }
        // uncomment the set accessor and declaration for _mi
        /*
        set
        {
            _mi = value;
        }
        */
    }

    public static void Main()
    {  
        Example example = new Example();
        example.I = 9;   // CS0200
    }
}  
```

<span data-ttu-id="bf6ac-109">以下範例使用[自動實現的屬性](../programming-guide/classes-and-structs/auto-implemented-properties.md),[物件初始化器](../programming-guide/classes-and-structs/object-and-collection-initializers.md),並且仍然產生 CS0200:</span><span class="sxs-lookup"><span data-stu-id="bf6ac-109">The following sample uses [auto-implemented properties](../programming-guide/classes-and-structs/auto-implemented-properties.md), [object initializers](../programming-guide/classes-and-structs/object-and-collection-initializers.md), and still generates CS0200:</span></span>

```csharp
// CS0200.cs
public class Example
{
    int I
    {
        get;
        // uncomment the set accessor and declaration
        //set;
    }

    public static void Main()
    {  
        var example = new Example
        {
            I = 9   // CS0200
        };
    }
}
```

<span data-ttu-id="bf6ac-110">分配給屬性或索引器"屬性"是唯讀的,可以透過添加集存取器或分配給物件構造函數中的屬性來實現。</span><span class="sxs-lookup"><span data-stu-id="bf6ac-110">Assignment to a property or indexer 'property' that is read only, can be achieved through adding a set accessor or by assigning to the property in the object's constructor.</span></span>

```csharp

{
    int I { get; }

    public Example()
    {
        I = -7;
    }
}
```
