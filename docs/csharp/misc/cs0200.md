# <a name="compiler-error-cs0200"></a>編譯器錯誤 CS0200
屬性或索引器屬性「不能分配給 -- 它是唯讀的

嘗試將值分配給[屬性](../programming-guide/classes-and-structs/using-properties.md),但該屬性沒有集訪問器,或者賦值不在構造函數之外。 請加入 set 存取子來解決此錯誤。 有關詳細資訊,請參閱[如何聲明和使用讀取寫入屬性](../programming-guide/classes-and-structs/how-to-declare-and-use-read-write-properties.md)。

## <a name="examples"></a>範例
下列範例會產生 CS0200：

```csharp
// CS0200.cs
public class Example
{
    private int _mi;
    int I
    {
        get
        {
            return _mi;
        }
        // uncomment the set accessor and declaration for _mi
        /*
        set
        {
            _mi = value;
        }
        */
    }

    public static void Main()
    {  
        Example example = new Example();
        example.I = 9;   // CS0200
    }
}  
```

以下範例使用[自動實現的屬性](../programming-guide/classes-and-structs/auto-implemented-properties.md),[物件初始化器](../programming-guide/classes-and-structs/object-and-collection-initializers.md),並且仍然產生 CS0200:

```csharp
// CS0200.cs
public class Example
{
    int I
    {
        get;
        // uncomment the set accessor and declaration
        //set;
    }

    public static void Main()
    {  
        var example = new Example
        {
            I = 9   // CS0200
        };
    }
}
```

分配給屬性或索引器"屬性"是唯讀的,可以透過添加集存取器或分配給物件構造函數中的屬性來實現。

```csharp
public class Example
{
    int I { get; }

    public Example()
    {
        I = -7;
    }
}
```
