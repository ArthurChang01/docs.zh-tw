<span data-ttu-id="337ae-104">已嘗試將值指派給[屬性](../programming-guide/classes-and-structs/using-properties.md)，但屬性沒有 set 存取子，或指派不在此函式的外部。</span><span class="sxs-lookup"><span data-stu-id="337ae-104">An attempt was made to assign a value to a [property](../programming-guide/classes-and-structs/using-properties.md), but the property does not have a set accessor or the assignment was outside of the constructor.</span></span> <span data-ttu-id="337ae-105">請加入 set 存取子來解決此錯誤。</span><span class="sxs-lookup"><span data-stu-id="337ae-105">Resolve the error by adding a set accessor.</span></span> <span data-ttu-id="337ae-106">如需詳細資訊，請參閱[如何宣告及使用讀寫屬性](../programming-guide/classes-and-structs/how-to-declare-and-use-read-write-properties.md)。</span><span class="sxs-lookup"><span data-stu-id="337ae-106">For more information, see [How to declare and use read-write properties](../programming-guide/classes-and-structs/how-to-declare-and-use-read-write-properties.md).</span></span>

## <a name="examples"></a><span data-ttu-id="337ae-107">範例</span><span class="sxs-lookup"><span data-stu-id="337ae-107">Examples</span></span>

<span data-ttu-id="337ae-108">下列範例會產生 CS0200：</span><span class="sxs-lookup"><span data-stu-id="337ae-108">The following sample generates CS0200:</span></span>

```csharp
// CS0200.cs
public class Example
{
    private int _mi;
    int I
    {
        get
        {
            return _mi;
        }
        // uncomment the set accessor and declaration for _mi
        /*
        set
        {
            _mi = value;
        }
        */
    }

    public static void Main()
    {  
        Example example = new Example();
        example.I = 9;   // CS0200
    }
}  
```

<span data-ttu-id="337ae-109">下列範例會使用[自動執行的屬性](../programming-guide/classes-and-structs/auto-implemented-properties.md)和[物件初始化運算式](../programming-guide/classes-and-structs/object-and-collection-initializers.md)，而且仍然會產生 CS0200：</span><span class="sxs-lookup"><span data-stu-id="337ae-109">The following sample uses [auto-implemented properties](../programming-guide/classes-and-structs/auto-implemented-properties.md) and [object initializers](../programming-guide/classes-and-structs/object-and-collection-initializers.md) and still generates CS0200:</span></span>

```csharp
// CS0200.cs
public class Example
{
    int I
    {
        get;
        // uncomment the set accessor and declaration
        //set;
    }

    public static void Main()
    {  
        var example = new Example
        {
            I = 9   // CS0200
        };
    }
}
```

<span data-ttu-id="337ae-110">若要指派給唯讀的屬性或索引子 ' property '，請加入 set 存取子，或在物件的函式中指派值。</span><span class="sxs-lookup"><span data-stu-id="337ae-110">To assign to a property or indexer 'property' that's read-only, add a set accessor or assign the value in the object's constructor.</span></span>

```csharp
public class Example
{
    int I { get; }

    public Example()
    {
        I = -7;
    }
}
```
