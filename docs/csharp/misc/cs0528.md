# <a name="compiler-error-cs0528"></a>編譯器錯誤 CS0528
'interface' 已經列於介面清單  
  
 介面繼承清單包含重複項目。 一個 [介面](../../csharp/language-reference/keywords/interface.md) 只能在繼承清單中指定一次。  
  
 下列範例會產生 CS0528：  
  
```csharp  
// CS0528.cs  
namespace x  
{  
   public interface a  
   {  
   }  
  
   public class b : a, a   // CS0528  
   {  
      public void Main()  
      {  
      }  
   }  
}  
```
