# <a name="compiler-warning-level-1-cs1692"></a>編譯器警告 (層級 1) CS1692

無效的數字

一些前置處理器指示詞，例如 `#pragma` 和 `#line`使用數字作為參數。 這些數字之一無效，因為它太大、格式錯誤、包含不合法的字元等等。 若要更正這個錯誤，請更正數字。

## <a name="example"></a>範例

下列範例會產生 CS1692：

```csharp
// CS1692.cs

#pragma warning disable a  // CS1692
// Try this instead:
// #pragma warning disable 1691

class A
{
    static void Main()
    {
    }
}
```
