# <a name="compiler-error-cs0525"></a>編譯器錯誤 CS0525
介面不能包含欄位  
  
 [介面](../../csharp/language-reference/keywords/interface.md) 可以包含方法和屬性，但不能包含欄位。  
  
 下列範例會產生 CS0525：  
  
```  
// CS0525.cs  
namespace x  
{  
   public interface clx  
   {  
      public int i;   // CS0525  
   }  
}  
```
