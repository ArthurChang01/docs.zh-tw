# <a name="compiler-error-cs0076"></a>編譯器錯誤 CS0076
列舉程式名稱 'value__' 為保留字而且無法使用  
  
 [列舉](../../csharp/language-reference/keywords/enum.md) 中的項目不能有稱為 **value__** 的識別項。
