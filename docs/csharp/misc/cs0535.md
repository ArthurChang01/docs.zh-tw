 <span data-ttu-id="9ff60-104">[類別](../../csharp/language-reference/keywords/class.md) 衍生自 [介面](../../csharp/language-reference/keywords/interface.md)，但是類別未實作一個或多個介面的成員。</span><span class="sxs-lookup"><span data-stu-id="9ff60-104">A [class](../../csharp/language-reference/keywords/class.md) derived from an [interface](../../csharp/language-reference/keywords/interface.md), but the class did not implement one or more of the interface's members.</span></span> <span data-ttu-id="9ff60-105">類別必須實作從中衍生或宣告為 `abstract`之介面的所有成員。</span><span class="sxs-lookup"><span data-stu-id="9ff60-105">A class must implement all members of interfaces from which it derives or else be declared `abstract`.</span></span>  
  
## <a name="example"></a><span data-ttu-id="9ff60-106">範例</span><span class="sxs-lookup"><span data-stu-id="9ff60-106">Example</span></span>  
 <span data-ttu-id="9ff60-107">下列範例會產生 CS0535。</span><span class="sxs-lookup"><span data-stu-id="9ff60-107">The following sample generates CS0535.</span></span>  
  
```csharp  
// CS0535.cs  
public interface A  
{  
   void F();  
}  
  
public class B : A {}   // CS0535 A::F is not implemented  
  
// OK  
public class C : A {  
   public void F() {}  
   public static void Main() {}  
}  
```
