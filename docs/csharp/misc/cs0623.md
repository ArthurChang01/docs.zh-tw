# <a name="compiler-error-cs0623"></a>編譯器錯誤 CS0623
陣列初始設定式只可用於變數或欄位初始設定式中。 請嘗試改用新的運算式。  
  
 嘗試在內容中使用陣列初始設定式來初始化陣列，這是不允許的作業。  
  
## <a name="example"></a>範例  
 下列範例會產生 CS0623，因為編譯器會解譯{4}為外部陣列初始設定式內的內嵌的陣列初始設定式：  
  
```csharp  
//cs0632.cs  
using System;  
  
class X  
{  
    public int[] x = { 2, 3, {4}}; //CS0623  
}  
```
