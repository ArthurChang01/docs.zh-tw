# <a name="compiler-error-cs0113"></a><span data-ttu-id="6a669-102">編譯器錯誤 CS0113</span><span class="sxs-lookup"><span data-stu-id="6a669-102">Compiler Error CS0113</span></span>
<span data-ttu-id="6a669-103">標記為 override 的成員 'function' 不能標記為 new 或 virtual</span><span class="sxs-lookup"><span data-stu-id="6a669-103">A member 'function' marked as override cannot be marked as new or virtual</span></span>  
  
 <span data-ttu-id="6a669-104">使用 [new](../../csharp/language-reference/keywords/new-modifier.md) 和 [override](../../csharp/language-reference/keywords/override.md) 關鍵字來標記方法是互斥的。</span><span class="sxs-lookup"><span data-stu-id="6a669-104">It is mutually exclusive to mark a method with the [new](../../csharp/language-reference/keywords/new-modifier.md) and [override](../../csharp/language-reference/keywords/override.md) keywords.</span></span>  
  
 <span data-ttu-id="6a669-105">下列範例會產生 CS0113：</span><span class="sxs-lookup"><span data-stu-id="6a669-105">The following sample generates CS0113:</span></span>  
  
```csharp  
// CS0113.cs  
namespace MyNamespace  
{  
   abstract public class MyClass  
   {  
      public abstract void Foo();  
   }  
  
   public class MyClass2 : MyClass  
   {  
      override new public void Foo()   // CS0113, remove new keyword  
      {  
      }  
  
      public static int Main()  
      {  
         return 0;  
      }  
   }  
}  
```
