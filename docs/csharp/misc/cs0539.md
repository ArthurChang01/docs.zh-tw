 <span data-ttu-id="a5ba2-106">下列範例會產生 CS0539：</span><span class="sxs-lookup"><span data-stu-id="a5ba2-106">The following sample generates CS0539:</span></span>

```csharp  
      void m();

      {  
         return 0;  
      }  
   }  
}  
```
