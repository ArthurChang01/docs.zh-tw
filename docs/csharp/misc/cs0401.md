## <a name="example"></a><span data-ttu-id="f4a62-105">範例</span><span class="sxs-lookup"><span data-stu-id="f4a62-105">Example</span></span>  
 <span data-ttu-id="f4a62-106">下列範例會產生 CS0401。</span><span class="sxs-lookup"><span data-stu-id="f4a62-106">The following sample generates CS0401.</span></span>  
  
```csharp

using System;  
 class C<T> where T : new(), IDisposable {}  // CS0401

   static void F<U>() where U : new(), IDisposable{}   // CS0401  
}  
```
