## <a name="example"></a><span data-ttu-id="2cfcb-105">範例</span><span class="sxs-lookup"><span data-stu-id="2cfcb-105">Example</span></span>

```csharp

 class C<T> where T : new(), IDisposable {}  // CS0401

```
