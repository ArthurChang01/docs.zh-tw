# <a name="compiler-error-cs0502"></a><span data-ttu-id="288da-102">編譯器錯誤 CS0502</span><span class="sxs-lookup"><span data-stu-id="288da-102">Compiler Error CS0502</span></span>
<span data-ttu-id="288da-103">'member' 不能同時為 abstract 和 sealed</span><span class="sxs-lookup"><span data-stu-id="288da-103">'member' cannot be both abstract and sealed</span></span>  
  
 <span data-ttu-id="288da-104">類別成員不能同時為 [abstract](../../csharp/language-reference/keywords/abstract.md) 和 [sealed](../../csharp/language-reference/keywords/sealed.md)。</span><span class="sxs-lookup"><span data-stu-id="288da-104">A class member cannot be both [abstract](../../csharp/language-reference/keywords/abstract.md) and [sealed](../../csharp/language-reference/keywords/sealed.md).</span></span>  
  
 <span data-ttu-id="288da-105">下列範例會產生 CS0502：</span><span class="sxs-lookup"><span data-stu-id="288da-105">The following sample generates CS0502:</span></span>  
  
```csharp  
// CS0502.cs  
public class B  
{  
   abstract public void F();  
}  
  
public class C : B  
{  
   abstract sealed override public void F()   // CS0502  
   {  
   }  
}  
  
public class CMain  
{  
    public static void Main()  
    { }  
}  
```
