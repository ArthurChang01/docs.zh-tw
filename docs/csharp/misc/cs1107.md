## <a name="example"></a><span data-ttu-id="8a145-105">範例</span><span class="sxs-lookup"><span data-stu-id="8a145-105">Example</span></span>  
 <span data-ttu-id="8a145-106">下列範例會產生 CS1107：</span><span class="sxs-lookup"><span data-stu-id="8a145-106">The following example generates CS1107:</span></span>  
  
```  
// cs1107.cs  
public static class Test  
{  
    // Extension methods.  
        public static void TestMethod(this this t) {} // CS1107  
  
    // Regular Instance Method  
        public void TestMethod(ref ref int i) {} // CS1107  
  
}  
```
