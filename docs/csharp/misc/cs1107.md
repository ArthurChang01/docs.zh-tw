 <span data-ttu-id="9d435-104">參數修改`this`器（如 、、`ref``in`和`out`在參數定義中出現多個次）是一個錯誤。</span><span class="sxs-lookup"><span data-stu-id="9d435-104">It is an error for parameter modifiers such as `this`, `ref`, `in`, and `out` to appear more than one time in a parameter definition.</span></span>  
  
## <a name="example"></a><span data-ttu-id="9d435-105">範例</span><span class="sxs-lookup"><span data-stu-id="9d435-105">Example</span></span>  
 <span data-ttu-id="9d435-106">下列範例會產生 CS1107：</span><span class="sxs-lookup"><span data-stu-id="9d435-106">The following example generates CS1107:</span></span>  
  
```csharp  
// cs1107.cs  
public static class Test
{
    // Extension methods.  
    public static void TestMethod(this this t) { } // CS1107  

}

public class TestTwo
{
    // Regular Instance Method  
    public void TestMethod(ref ref int i) { } // CS1107  

    // Regular Instance Method  
    public void TestMethod(in in double d) { } // CS1107  
}
```
