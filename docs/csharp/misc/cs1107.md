 <span data-ttu-id="27c84-104">它是參數修飾詞的錯誤，例如`this`， `ref`， `in`，和`out`到參數定義中出現超過一次。</span><span class="sxs-lookup"><span data-stu-id="27c84-104">It is an error for parameter modifiers such as `this`, `ref`, `in`, and `out` to appear more than one time in a parameter definition.</span></span>  
  
## <a name="example"></a><span data-ttu-id="27c84-105">範例</span><span class="sxs-lookup"><span data-stu-id="27c84-105">Example</span></span>  
 <span data-ttu-id="27c84-106">下列範例會產生 CS1107：</span><span class="sxs-lookup"><span data-stu-id="27c84-106">The following example generates CS1107:</span></span>  
  
```csharp  
// cs1107.cs  
public static class Test
{
    // Extension methods.  
    public static void TestMethod(this this t) { } // CS1107  

}

public class TestTwo
{ 
    // Regular Instance Method  
    public void TestMethod(ref ref int i) { } // CS1107  

    // Regular Instance Method  
    public void TestMethod(in in double d) { } // CS1107  
}
```
