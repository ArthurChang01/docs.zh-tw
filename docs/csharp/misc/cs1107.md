# <a name="compiler-error-cs1107"></a><span data-ttu-id="0bcc7-102">編譯器錯誤 CS1107</span><span class="sxs-lookup"><span data-stu-id="0bcc7-102">Compiler Error CS1107</span></span>
<span data-ttu-id="0bcc7-103">一個參數只能有一個 'modifier name' 修飾詞</span><span class="sxs-lookup"><span data-stu-id="0bcc7-103">A parameter can only have one 'modifier name' modifier.</span></span>  
  
 <span data-ttu-id="0bcc7-104">它是參數修飾詞的錯誤，例如`this`， `ref`， `in`，和`out`到參數定義中出現超過一次。</span><span class="sxs-lookup"><span data-stu-id="0bcc7-104">It is an error for parameter modifiers such as `this`, `ref`, `in`, and `out` to appear more than one time in a parameter definition.</span></span>  
  
## <a name="example"></a><span data-ttu-id="0bcc7-105">範例</span><span class="sxs-lookup"><span data-stu-id="0bcc7-105">Example</span></span>  
 <span data-ttu-id="0bcc7-106">下列範例會產生 CS1107：</span><span class="sxs-lookup"><span data-stu-id="0bcc7-106">The following example generates CS1107:</span></span>  
  
```csharp  
// cs1107.cs  
public static class Test
{
    // Extension methods.  
    public static void TestMethod(this this t) { } // CS1107  

}

public class TestTwo
{ 
    // Regular Instance Method  
    public void TestMethod(ref ref int i) { } // CS1107  

    // Regular Instance Method  
    public void TestMethod(in in double d) { } // CS1107  
}
```
