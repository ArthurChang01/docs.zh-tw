# <a name="compiler-error-cs1507"></a>編譯器錯誤 CS1507
建立模組時無法連結資源檔案 'file'  
  
 [-linkresource](../language-reference/compiler-options/linkresource-compiler-option.md)已用於具有[/target： module](../language-reference/compiler-options/target-module-compiler-option.md)的相同編譯中，這是不允許的。 例如，下列選項會產生 CS1507：  
  
```console  
csc /linkresource:rf.resource /target:module in.cs  
```  
  
 不過，允許內嵌資源（[/resource](../language-reference/compiler-options/resource-compiler-option.md)）。
