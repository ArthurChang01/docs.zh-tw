# <a name="compiler-error-cs1594"></a><span data-ttu-id="07cb5-102">編譯器錯誤 CS1594</span><span class="sxs-lookup"><span data-stu-id="07cb5-102">Compiler Error CS1594</span></span>
<span data-ttu-id="07cb5-103">委派 'delegate' 有一些無效的引數</span><span class="sxs-lookup"><span data-stu-id="07cb5-103">Delegate 'delegate' has some invalid arguments</span></span>  
  
 <span data-ttu-id="07cb5-104">傳遞至 [委派](../language-reference/builtin-types/reference-types.md) 引動過程的引數類型和委派宣告中的參數類型不一致。</span><span class="sxs-lookup"><span data-stu-id="07cb5-104">The type of an argument passed to a [delegate](../language-reference/builtin-types/reference-types.md) invocation does not agree with the type of the parameter in the delegate declaration.</span></span>  
  
 <span data-ttu-id="07cb5-105">下列範例會產生 CS1594：</span><span class="sxs-lookup"><span data-stu-id="07cb5-105">The following sample generates CS1594:</span></span>  
  
```csharp  
// CS1594.cs  
using System;  
delegate string func(int i);   // declare delegate  
  
class a  
{  
   public static void Main()  
   {  
      func dt = new func(z);  
      x(dt);  
   }  
  
   public static string z(int j)  
   {  
      Console.WriteLine(j);  
      return j.ToString();  
   }  
  
   public static void x(func hello)  
   {  
      hello("8");   // CS1594  
      // try the following line instead  
      // hello(8);  
   }  
}  
```
