# <a name="compiler-warning-level-4-cs0078"></a><span data-ttu-id="cc405-102">編譯器警告 (層級 4) CS0078</span><span class="sxs-lookup"><span data-stu-id="cc405-102">Compiler Warning (level 4) CS0078</span></span>
<span data-ttu-id="cc405-103">字尾 'l' 很容易與數字 '1' 混淆 -- 請使用 'L' 以避免困擾</span><span class="sxs-lookup"><span data-stu-id="cc405-103">The 'l' suffix is easily confused with the digit '1' -- use 'L' for clarity</span></span>  
  
 <span data-ttu-id="cc405-104">當編譯器偵測到 long 轉換使用小寫 l 而不是大寫 L 時，就會發出警告。</span><span class="sxs-lookup"><span data-stu-id="cc405-104">The compiler warns when it detects a cast to long using a lowercase l instead of an uppercase L.</span></span>  
  
 <span data-ttu-id="cc405-105">下列範例會產生 CS0078：</span><span class="sxs-lookup"><span data-stu-id="cc405-105">The following sample generates CS0078:</span></span>  
  
```  
// CS0078.cs  
// compile with: /W:4  
class test {  
   public static void TestL(long i)  
   {  
   }  
  
   public static void TestL(int i)  
   {  
   }  
  
   public static void Main()  
   {  
      TestL(25l);   // CS0078  
      // try the following line instead  
      // TestL(25L);  
   }  
}  
```
