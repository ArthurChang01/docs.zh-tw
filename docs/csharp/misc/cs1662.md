<span data-ttu-id="873a8-103">無法將匿名方法區塊轉換成委派類型 'delegate type'，因為區塊中的某些傳回類型無法隱含地轉換成委派傳回類型。</span><span class="sxs-lookup"><span data-stu-id="873a8-103">Cannot convert anonymous method block to delegate type 'delegate type' because some of the return types in the block are not implicitly convertible to the delegate return type</span></span>  
  
 <span data-ttu-id="873a8-104">如果匿名方法區塊的 return 陳述式有類型無法隱含地轉換為委派的傳回類型，就會發生此錯誤。</span><span class="sxs-lookup"><span data-stu-id="873a8-104">This error occurs if the anonymous method block's return statement had a type that was not implicitly convertible to the return type of the delegate.</span></span>  
  
 <span data-ttu-id="873a8-105">下列範例會產生 CS1662：</span><span class="sxs-lookup"><span data-stu-id="873a8-105">The following sample generates CS1662:</span></span>  
  
```csharp  
// CS1662.cs  
  
delegate int MyDelegate(int i);  
  
class C  
{  
  
  public static void Main()  
  {  
     MyDelegate d = delegate(int i) { return 1.0; };  // CS1662  
     // Try this instead:  
     // MyDelegate d = dekegate(int i) { return (int)1.0; };  
  }  
}  
```
