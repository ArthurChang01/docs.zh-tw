# <a name="compiler-error-cs1623"></a><span data-ttu-id="d0043-102">編譯器錯誤 CS1623</span><span class="sxs-lookup"><span data-stu-id="d0043-102">Compiler Error CS1623</span></span>
<span data-ttu-id="d0043-103">迭代器不能有 ref 或 out 參數</span><span class="sxs-lookup"><span data-stu-id="d0043-103">Iterators cannot have in ref or out parameters</span></span>  
  
 <span data-ttu-id="d0043-104">如果迭代器方法接受，就會發生這個錯誤`in`， `ref`，或`out`參數。</span><span class="sxs-lookup"><span data-stu-id="d0043-104">This error occurs if an iterator method takes an `in`, `ref`, or `out` parameter.</span></span> <span data-ttu-id="d0043-105">若要避免這個錯誤，請移除`in`， `ref`，或`out`從方法簽章的關鍵字。</span><span class="sxs-lookup"><span data-stu-id="d0043-105">To avoid this error, remove the `in`, `ref`, or `out` keyword from the method signature.</span></span>  
  
## <a name="example"></a><span data-ttu-id="d0043-106">範例</span><span class="sxs-lookup"><span data-stu-id="d0043-106">Example</span></span>  
 <span data-ttu-id="d0043-107">下列範例會產生 CS1623：</span><span class="sxs-lookup"><span data-stu-id="d0043-107">The following sample generates CS1623:</span></span>  
  
```csharp  
// CS1623.cs  
using System.Collections;

class C : IEnumerable
{
    public IEnumerator GetEnumerator()
    {
        yield return 0;
    }

    // To resolve the error, remove in  
    public IEnumerator GetEnumerator(in short i)  // CS1623  
    {
        yield return i;
    }
    // To resolve the error, remove ref  
    public IEnumerator GetEnumerator(ref int i)  // CS1623  
    {
        yield return i;
    }

    // To resolve the error, remove out  
    public IEnumerator GetEnumerator(out float f)  // CS1623  
    {
        f = 0.0F;
        yield return f;
    }
}
```
