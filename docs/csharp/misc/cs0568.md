# <a name="compiler-error-cs0568"></a><span data-ttu-id="1b2eb-102">編譯器錯誤 CS0568</span><span class="sxs-lookup"><span data-stu-id="1b2eb-102">Compiler Error CS0568</span></span>
<span data-ttu-id="1b2eb-103">結構無法包含明確無參數的建構函式</span><span class="sxs-lookup"><span data-stu-id="1b2eb-103">Structs cannot contain explicit parameterless constructors</span></span>  
  
 <span data-ttu-id="1b2eb-104">每個[結構](../language-reference/builtin-types/struct.md)都已有無參數的函式，可將物件初始化為零。</span><span class="sxs-lookup"><span data-stu-id="1b2eb-104">Each [struct](../language-reference/builtin-types/struct.md) already has a parameterless constructor that initializes the object to zero.</span></span> <span data-ttu-id="1b2eb-105">因此，您可以為結構建立的建構函式必須採用一或多個參數。</span><span class="sxs-lookup"><span data-stu-id="1b2eb-105">Therefore, the constructors that you can create for a struct must take one or more parameters.</span></span>  
  
 <span data-ttu-id="1b2eb-106">下列範例會產生 CS0568：</span><span class="sxs-lookup"><span data-stu-id="1b2eb-106">The following sample generates CS0568:</span></span>  
  
```csharp  
// CS0568.cs  
public struct ClassY  
{  
   public int field1;  
   public ClassY(){}   // CS0568, cannot have no param constructor  
   // Try following instead:  
   // public ClassY(int i)  
   // {  
   //    field1 = i;  
   // }  
}  
  
public class ClassX  
{  
   public static void Main()  
   {  
   }  
}  
```
