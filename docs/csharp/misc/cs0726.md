# <a name="compiler-error-cs0726"></a>編譯器錯誤 CS0726
'format specifier' 不是有效的格式規範  
  
 這個錯誤發生在偵錯工具中。 當您在其中一個偵錯工具視窗中輸入變數名稱時，可以接著依序輸入逗號和格式規範。 例如， `myInt, h` 和 `myString,nq`。 此錯誤發生於編譯器無法辨識[C# 中的格式規範](/visualstudio/debugger/format-specifiers-in-csharp)。
