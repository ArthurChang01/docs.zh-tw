# <a name="compiler-error-cs0752"></a><span data-ttu-id="7b6b1-102">編譯器錯誤 CS0752</span><span class="sxs-lookup"><span data-stu-id="7b6b1-102">Compiler Error CS0752</span></span>
<span data-ttu-id="7b6b1-103">部分方法不可有 out 參數</span><span class="sxs-lookup"><span data-stu-id="7b6b1-103">A partial method cannot have out parameters</span></span>  
  
 <span data-ttu-id="7b6b1-104">部分方法不可有 [out](../../csharp/language-reference/keywords/out-parameter-modifier.md) 參數。</span><span class="sxs-lookup"><span data-stu-id="7b6b1-104">A partial method cannot have an [out](../../csharp/language-reference/keywords/out-parameter-modifier.md) parameter.</span></span> <span data-ttu-id="7b6b1-105">不允許 out 參數，因為如果部分方法遭到編譯器移除，則無法保證曾經指派 out 參數。</span><span class="sxs-lookup"><span data-stu-id="7b6b1-105">Out parameters are not allowed because if the partial method is removed by the compiler then there is no guarantee that the out parameter is ever assigned.</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="7b6b1-106">更正這個錯誤</span><span class="sxs-lookup"><span data-stu-id="7b6b1-106">To correct this error</span></span>  
  
1. <span data-ttu-id="7b6b1-107">從參數中移除 out 修飾詞並改用方法的傳回值，或者從方法宣告中移除部分修飾詞。</span><span class="sxs-lookup"><span data-stu-id="7b6b1-107">Remove the out modifier from the parameter and use the return value of the method instead, or else remove the partial modifier from the method declaration.</span></span>  
  
## <a name="example"></a><span data-ttu-id="7b6b1-108">範例</span><span class="sxs-lookup"><span data-stu-id="7b6b1-108">Example</span></span>  
 <span data-ttu-id="7b6b1-109">下列程式碼會產生 CS0752：</span><span class="sxs-lookup"><span data-stu-id="7b6b1-109">The following code generates CS0752:</span></span>  
  
```csharp  
// cs0752.cs  
public partial class C  
{  
    partial void Part(out int num); // CS0752  
    // try the following line instead  
    // partial void Part(int num);

    {  
        return 1;  
    }  
}  
```
