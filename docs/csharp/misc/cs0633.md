## <a name="example"></a><span data-ttu-id="fe194-106">範例</span><span class="sxs-lookup"><span data-stu-id="fe194-106">Example</span></span>  
 <span data-ttu-id="fe194-107">這個範例使用 <xref:System.Diagnostics.ConditionalAttribute>來說明 CS0633。</span><span class="sxs-lookup"><span data-stu-id="fe194-107">This example illustrates CS0633 using the <xref:System.Diagnostics.ConditionalAttribute>.</span></span> <span data-ttu-id="fe194-108">下列範例會產生 CS0633。</span><span class="sxs-lookup"><span data-stu-id="fe194-108">The following sample generates CS0633.</span></span>  
  
```  
// CS0633a.cs  
#define DEBUG  
using System.Diagnostics;  
public class Test  
{  
   [Conditional("DEB+UG")]   // CS0633  
   // try the following line instead  
   // [Conditional("DEBUG")]  
   public static void Main() { }  
}  
```  
  
## <a name="example"></a><span data-ttu-id="fe194-109">範例</span><span class="sxs-lookup"><span data-stu-id="fe194-109">Example</span></span>  
 <span data-ttu-id="fe194-110">這個範例使用 <xref:System.Runtime.CompilerServices.IndexerNameAttribute>來說明 CS0633。</span><span class="sxs-lookup"><span data-stu-id="fe194-110">This example illustrates CS0633 using the <xref:System.Runtime.CompilerServices.IndexerNameAttribute>.</span></span>  
  
```  
// CS0633b.cs  
// compile with: /target:module  
#define DEBUG  
using System.Runtime.CompilerServices;  
public class Test  
{  
   [IndexerName("Invalid+Identifier")]   // CS0633  
   // try the following line instead  
   // [IndexerName("DEBUG")]  
   public int this[int i]   
   {   
      get { return i; }   
   }  
}  
```
