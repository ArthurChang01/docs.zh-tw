# <a name="compiler-error-cs0175"></a><span data-ttu-id="debf8-102">編譯器錯誤 CS0175</span><span class="sxs-lookup"><span data-stu-id="debf8-102">Compiler Error CS0175</span></span>
<span data-ttu-id="debf8-103">在此內容中使用關鍵字 'base' 無效</span><span class="sxs-lookup"><span data-stu-id="debf8-103">Use of keyword 'base' is not valid in this context</span></span>  
  
 <span data-ttu-id="debf8-104">[base](../../csharp/language-reference/keywords/base.md) 關鍵字必須用來指定基底類別的特定成員。</span><span class="sxs-lookup"><span data-stu-id="debf8-104">The [base](../../csharp/language-reference/keywords/base.md) keyword must be used to specify a particular member of the base class.</span></span> <span data-ttu-id="debf8-105">如需詳細資訊，請參閱[建構函式](../../csharp/programming-guide/classes-and-structs/constructors.md)。</span><span class="sxs-lookup"><span data-stu-id="debf8-105">For more information, see [Constructors](../../csharp/programming-guide/classes-and-structs/constructors.md).</span></span>  
  
 <span data-ttu-id="debf8-106">下列範例會產生 CS0175：</span><span class="sxs-lookup"><span data-stu-id="debf8-106">The following sample generates CS0175:</span></span>  
  
```  
// CS0175.cs  
using System;  
class BaseClass  
{  
    public int TestInt = 0;  
}  
  
class MyClass : BaseClass  
{  
    public static void Main()  
    {  
        MyClass aClass = new MyClass();  
        aClass.BaseTest();  
    }  
  
    public void BaseTest()  
    {  
        Console.WriteLine(base); // CS0175  
        // Try the following line instead:  
        // Console.WriteLine(base.TestInt);  
        base = 9;   // CS0175  
  
        // Try the following line instead:  
        // base.TestInt = 9;  
    }  
}  
```
