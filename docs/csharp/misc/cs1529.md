## <a name="example"></a><span data-ttu-id="8fb41-105">範例</span><span class="sxs-lookup"><span data-stu-id="8fb41-105">Example</span></span>  
 <span data-ttu-id="8fb41-106">下列範例會產生 CS1529：</span><span class="sxs-lookup"><span data-stu-id="8fb41-106">The following sample generates CS1529:</span></span>  
  
```csharp  
// CS1529.cs  
namespace X  
{  
    namespace Subspace  
    {  
        using Microsoft;  
  
        class SomeClass  
        {  
        };  
  
        using Microsoft;      // CS1529, place before class definition  
    }  
  
    using System.Reflection;  // CS1529, place before namespace 'Subspace'  
}  
  
using System;                 // CS1529, place at the beginning of the file  
```
