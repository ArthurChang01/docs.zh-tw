# <a name="compiler-error-cs0041"></a>編譯器錯誤 CS0041
偵錯資訊的 'type' 完整名稱太長。 不使用 '/debug' 選項進行編譯。  
  
 使用 [/debug](../language-reference/compiler-options/debug-compiler-option.md) 編譯器選項時，會發生此錯誤。 如果您遇到這個錯誤，請嘗試刪除 bin 目錄中的 PDB 檔案，並重新編譯。 如果您仍然遇到這個錯誤, 可能必須修復或重新安裝 Visual Studio。
