# <a name="compiler-error-cs0557"></a><span data-ttu-id="229fd-102">編譯器錯誤 CS0557</span><span class="sxs-lookup"><span data-stu-id="229fd-102">Compiler Error CS0557</span></span>
<span data-ttu-id="229fd-103">類型 'class' 中重複的使用者定義轉換</span><span class="sxs-lookup"><span data-stu-id="229fd-103">Duplicate user-defined conversion in type 'class'</span></span>  
  
 <span data-ttu-id="229fd-104">類別中不允許重複轉換常式。</span><span class="sxs-lookup"><span data-stu-id="229fd-104">Duplicate conversion routines are not allowed in a class.</span></span>  
  
 <span data-ttu-id="229fd-105">下列範例會產生 CS0557：</span><span class="sxs-lookup"><span data-stu-id="229fd-105">The following example generates CS0557:</span></span>  
  
```csharp  
// CS0557.cs  
namespace x  
{  
    public class ii  
    {  
        public class iii  
        {  
        public static implicit operator int(iii aa)  
        {  
            return 0;  
        }  
  
    // CS0557, delete duplicate  
        public static explicit operator int(iii aa)  
        {  
            return 0;  
        }  
        }  
  
        public static void Main()  
        {  
        }  
    }  
}  
```
