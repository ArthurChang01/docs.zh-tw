# <a name="compiler-warning-level-1-cs2023"></a>編譯器警告 (層級 1) CS2023
因為在回應檔中已指定 /noconfig 選項，所以將會忽略該選項。  
  
 在回應檔中指定了[-noconfig](../language-reference/compiler-options/noconfig-compiler-option.md)編譯器選項，這是不允許的。  
  
 [-Nowarn](../language-reference/compiler-options/nowarn-compiler-option.md)選項無法隱藏這個警告。
