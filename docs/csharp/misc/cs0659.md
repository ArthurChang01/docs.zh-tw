# <a name="compiler-warning-level-3-cs0659"></a><span data-ttu-id="fbaa3-102">編譯器警告 (層級 3) CS0659</span><span class="sxs-lookup"><span data-stu-id="fbaa3-102">Compiler Warning (level 3) CS0659</span></span>

<span data-ttu-id="fbaa3-103">'class' 覆寫了 Object.Equals(object o)，但並沒有覆寫 Object.GetHashCode()</span><span class="sxs-lookup"><span data-stu-id="fbaa3-103">'class' overrides Object.Equals(object o) but does not override Object.GetHashCode()</span></span>  

<span data-ttu-id="fbaa3-104">編譯器偵測到的覆寫<xref:System.Object.Equals%2A?displayProperty=nameWithType>方法，但不是覆寫的<xref:System.Object.GetHashCode%2A?displayProperty=nameWithType>方法。</span><span class="sxs-lookup"><span data-stu-id="fbaa3-104">The compiler detected an override of the <xref:System.Object.Equals%2A?displayProperty=nameWithType> method but no override of the <xref:System.Object.GetHashCode%2A?displayProperty=nameWithType> method.</span></span> <span data-ttu-id="fbaa3-105">覆寫<xref:System.Object.Equals%2A>暗示您也需要覆寫<xref:System.Object.GetHashCode%2A>。</span><span class="sxs-lookup"><span data-stu-id="fbaa3-105">An override of <xref:System.Object.Equals%2A> implies that you also want to override <xref:System.Object.GetHashCode%2A>.</span></span>  

<span data-ttu-id="fbaa3-106">下列程式碼會產生 CS0659:</span><span class="sxs-lookup"><span data-stu-id="fbaa3-106">The following code generates CS0659:</span></span>  

```csharp
// CS0659.cs  
// compile with: /W:3 /target:library  
class Test
{  
   public override bool Equals(object o) { return true; }   // CS0659  
}  
  
// OK  
class Test2  
{  
   public override bool Equals(object o) { return true; }  
   public override int GetHashCode() { return 0; }  
}  
```
