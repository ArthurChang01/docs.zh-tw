# <a name="compiler-error-cs0538"></a><span data-ttu-id="7e6f6-102">編譯器錯誤 CS0538</span><span class="sxs-lookup"><span data-stu-id="7e6f6-102">Compiler Error CS0538</span></span>
<span data-ttu-id="7e6f6-103">在明確介面宣告中的 'name' 不是介面</span><span class="sxs-lookup"><span data-stu-id="7e6f6-103">'name' in explicit interface declaration is not an interface</span></span>  
  
 <span data-ttu-id="7e6f6-104">嘗試明確宣告 [介面](../../csharp/language-reference/keywords/interface.md)，但未指定介面。</span><span class="sxs-lookup"><span data-stu-id="7e6f6-104">An attempt was made to explicitly declare an [interface](../../csharp/language-reference/keywords/interface.md), but an interface was not specified.</span></span>  
  
 <span data-ttu-id="7e6f6-105">下列範例會產生 CS0538：</span><span class="sxs-lookup"><span data-stu-id="7e6f6-105">The following sample generates CS0538:</span></span>  
  
```csharp  
// CS0538.cs  
interface MyIFace  
{  
   void F();  
}  
  
public class MyClass  
{  
   public void G()  
   {  
   }  
}  
  
class C: MyIFace  
{  
   void MyIFace.F()  
   {  
   }  
  
   void MyClass.G()   // CS0538, MyClass not an interface  
   {  
   }  
}  
```
