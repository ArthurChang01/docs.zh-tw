# <a name="compiler-error-cs0538"></a>編譯器錯誤 CS0538
在明確介面宣告中的 'name' 不是介面  
  
 嘗試明確宣告 [介面](../language-reference/keywords/interface.md)，但未指定介面。  
  
 下列範例會產生 CS0538：  
  
```csharp  
// CS0538.cs  
interface MyIFace  
{  
   void F();  
}  
  
public class MyClass  
{  
   public void G()  
   {  
   }  
}  
  
class C: MyIFace  
{  
   void MyIFace.F()  
   {  
   }  
  
   void MyClass.G()   // CS0538, MyClass not an interface  
   {  
   }  
}  
```
