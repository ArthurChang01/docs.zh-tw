# <a name="compiler-error-cs1577"></a>編譯器錯誤 CS1577
組件產生失敗 -- 原因  
  
 C# 編譯器成功編譯原始碼之後，編譯的組件產生部分失敗。 請參閱 ALink 公用程式的錯誤說明文件 ([Al.exe 工具錯誤和警告](http://msdn.microsoft.com/library/7f125d49-0a03-47a6-9ba9-d61a679a7d4b)) 如需詳細資訊。
