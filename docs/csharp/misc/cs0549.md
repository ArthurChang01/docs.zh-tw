# <a name="compiler-error-cs0549"></a><span data-ttu-id="b80e2-102">編譯器錯誤 CS0549</span><span class="sxs-lookup"><span data-stu-id="b80e2-102">Compiler Error CS0549</span></span>
<span data-ttu-id="b80e2-103">'function' 是密封類別 'class' 的新虛擬成員</span><span class="sxs-lookup"><span data-stu-id="b80e2-103">'function' is a new virtual member in sealed class 'class'</span></span>  
  
 <span data-ttu-id="b80e2-104">[密封](../language-reference/keywords/sealed.md)[類別](../language-reference/keywords/class.md)不能當做基類使用。</span><span class="sxs-lookup"><span data-stu-id="b80e2-104">A [sealed](../language-reference/keywords/sealed.md)[class](../language-reference/keywords/class.md) cannot be used as a base class.</span></span>  <span data-ttu-id="b80e2-105">因此，虛擬方法在密封類別中毫無作用。</span><span class="sxs-lookup"><span data-stu-id="b80e2-105">Therefore, it is useless to have a virtual method in sealed class.</span></span>  
  
 <span data-ttu-id="b80e2-106">下列範例會產生 CS0549：</span><span class="sxs-lookup"><span data-stu-id="b80e2-106">The following sample generates CS0549:</span></span>  
  
```csharp

sealed public class MyClass  
{  
   virtual public void TestMethod() {}   // CS0549

}  
```
