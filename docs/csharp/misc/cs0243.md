# <a name="compiler-error-cs0243"></a><span data-ttu-id="99491-102">編譯器錯誤 CS0243</span><span class="sxs-lookup"><span data-stu-id="99491-102">Compiler error CS0243</span></span>

<span data-ttu-id="99491-103">Conditional 屬性不能用在 'method' 上，因為它是一個覆寫方法</span><span class="sxs-lookup"><span data-stu-id="99491-103">The Conditional attribute is not valid on 'method' because it is an override method</span></span>  
  
<span data-ttu-id="99491-104">在<xref:System.Diagnostics.ConditionalAttribute>使用[重寫](../language-reference/keywords/override.md)關鍵字標記的方法上不允許該屬性。</span><span class="sxs-lookup"><span data-stu-id="99491-104">The <xref:System.Diagnostics.ConditionalAttribute> attribute is not allowed on a method that is marked with the [override](../language-reference/keywords/override.md) keyword.</span></span> <span data-ttu-id="99491-105">有關詳細資訊,請參閱[知道何時使用覆寫和新建關鍵字](../programming-guide/classes-and-structs/knowing-when-to-use-override-and-new-keywords.md)。</span><span class="sxs-lookup"><span data-stu-id="99491-105">For more information, see [Knowing When to Use Override and New Keywords](../programming-guide/classes-and-structs/knowing-when-to-use-override-and-new-keywords.md).</span></span>  
  
<span data-ttu-id="99491-106">編譯器從不綁定以重寫方法。</span><span class="sxs-lookup"><span data-stu-id="99491-106">The compiler never binds to override methods.</span></span> <span data-ttu-id="99491-107">它僅綁定到基方法,並且公共語言運行時根據需要調用重寫。</span><span class="sxs-lookup"><span data-stu-id="99491-107">It only binds to the base method, and the common language runtime calls the override, as appropriate.</span></span>  
  
<span data-ttu-id="99491-108">以下代碼產生 CS0243:</span><span class="sxs-lookup"><span data-stu-id="99491-108">The following code generates CS0243:</span></span>  

```csharp
// CS0243.cs  
// compile with: /target:library  
public class MyClass  
{  
   public virtual void M() {}  
}  
  
public class MyClass2 : MyClass  
{  
   [System.Diagnostics.ConditionalAttribute("MySymbol")]   // CS0243  
   // remove Conditional attribute or remove override keyword  
   public override void M() {}  
}  
```
