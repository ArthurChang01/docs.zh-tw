<span data-ttu-id="a1f8a-103">Conditional 屬性不能用在 'method' 上，因為它是一個覆寫方法</span><span class="sxs-lookup"><span data-stu-id="a1f8a-103">The Conditional attribute is not valid on 'method' because it is an override method</span></span>  
  
 <span data-ttu-id="a1f8a-104">在以 [override](http://msdn.microsoft.com/en-us/e1c4913b-74d0-421a-8a6d-c14b3f0e68fb) 關鍵字標記的方法上不允許 [Conditional](../../csharp/language-reference/keywords/override.md) 屬性。</span><span class="sxs-lookup"><span data-stu-id="a1f8a-104">The [Conditional](http://msdn.microsoft.com/en-us/e1c4913b-74d0-421a-8a6d-c14b3f0e68fb) attribute is not allowed on a method that is marked with the [override](../../csharp/language-reference/keywords/override.md) keyword.</span></span> <span data-ttu-id="a1f8a-105">如需詳細資訊，請參閱[知道何時使用覆寫和 New 關鍵字](../../csharp/programming-guide/classes-and-structs/knowing-when-to-use-override-and-new-keywords.md)。</span><span class="sxs-lookup"><span data-stu-id="a1f8a-105">For more information, see [Knowing When to Use Override and New Keywords](../../csharp/programming-guide/classes-and-structs/knowing-when-to-use-override-and-new-keywords.md).</span></span>  
  
 <span data-ttu-id="a1f8a-106">編譯器絕不會繫結至覆寫方法；它只能繫結至基底方法，而 Common Language Runtime 會適當地呼叫覆寫。</span><span class="sxs-lookup"><span data-stu-id="a1f8a-106">The compiler never binds to override methods; it only binds to the base method and the common language runtime calls the override, as appropriate.</span></span>  
  
 <span data-ttu-id="a1f8a-107">下列範例會產生 CS0243：</span><span class="sxs-lookup"><span data-stu-id="a1f8a-107">The following sample generates CS0243:</span></span>  
  
```  
// CS0243.cs  
// compile with: /target:library  
public class MyClass  
{  
   public virtual void M() {}  
}  
  
public class MyClass2 : MyClass  
{  
   [System.Diagnostics.ConditionalAttribute("MySymbol")]   // CS0243  
   // remove Conditional attribute or remove override keyword  
   public override void M() {}  
}  
```
