# <a name="compiler-error-cs0139"></a>編譯器錯誤 CS0139
沒有可中斷或繼續的封閉式迴圈  
  
 在迴圈外部遇到中斷或繼續陳述式。  
  
 如需詳細資訊，請參閱 [跳躍陳述式](../../csharp/language-reference/keywords/jump-statements.md)。  
  
 下列範例會產生 CS0139 兩次：  
  
```csharp  
// CS0139.cs  
namespace x  
{  
   public class a  
   {  
      public static void Main()  
      {  
         continue;  // CS0139  
         break;     // CS0139  
      }  
   }  
}  
```
