## <a name="to-correct-this-error"></a><span data-ttu-id="fb8b6-108">更正這個錯誤</span><span class="sxs-lookup"><span data-stu-id="fb8b6-108">To correct this error</span></span>  
  
1.  <span data-ttu-id="fb8b6-109">使用下面的程式碼作為範例，其中一種解決方案是指定一般 `ImplStruct` 作為 `TestMethod`呼叫中的第一個類型引數。</span><span class="sxs-lookup"><span data-stu-id="fb8b6-109">Using the code that follows as an example, one solution is to specify an ordinary `ImplStruct` as the first type argument in the call to `TestMethod`.</span></span> <span data-ttu-id="fb8b6-110">然後，修改 `TestMethod` ，以在其傳回陳述式中建立可為 Null 的 `Implstruct` 版本：</span><span class="sxs-lookup"><span data-stu-id="fb8b6-110">Then modify `TestMethod` to create a nullable version of `Implstruct` in its return statement:</span></span>  
  
    ```csharp  
    return new Nullable<T>(t);  
    ```  
  
## <a name="example"></a><span data-ttu-id="fb8b6-111">範例</span><span class="sxs-lookup"><span data-stu-id="fb8b6-111">Example</span></span>  
 <span data-ttu-id="fb8b6-112">下列程式碼會產生 CS0313：</span><span class="sxs-lookup"><span data-stu-id="fb8b6-112">The following code generates CS0313:</span></span>  
  
```csharp  
// cs0313.cs  
public interface BaseInterface { }  
public struct ImplStruct : BaseInterface { }  
  
public class TestClass  
{  
    public T? TestMethod<T, U>(T t) where T : struct, U  
    {  
        return t;  
    }  
}  
  
public class NullableTest  
{  
    public static void Run()  
    {  
  
        TestClass tc = new TestClass();  
        tc.TestMethod<ImplStruct?, BaseInterface>(new ImplStruct?()); // CS0313  
    }  
    public static void Main()  
    { }  
}  
```
