# <a name="compiler-error-cs0586"></a>編譯器錯誤 CS0586
編譯器內部錯誤：階段 'stage'  
  
 請嘗試判斷編譯器失敗的原因是否為無法剖析未預期的語法。 如果不的話，請嘗試[告訴我們](https://docs.microsoft.com/visualstudio/ide/talk-to-us)。
