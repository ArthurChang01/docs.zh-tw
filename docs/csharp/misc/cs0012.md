# <a name="compiler-error-cs0012"></a><span data-ttu-id="2d5ba-102">編譯器錯誤 CS0012</span><span class="sxs-lookup"><span data-stu-id="2d5ba-102">Compiler Error CS0012</span></span>
<span data-ttu-id="2d5ba-103">類型 'type' 定義在未被參考的組件中。</span><span class="sxs-lookup"><span data-stu-id="2d5ba-103">The type 'type' is defined in an assembly that is not referenced.</span></span> <span data-ttu-id="2d5ba-104">您必須加入對組件 'assembly' 的參考。</span><span class="sxs-lookup"><span data-stu-id="2d5ba-104">You must add a reference to assembly 'assembly'.</span></span>  
  
 <span data-ttu-id="2d5ba-105">找不到參考類型的定義。</span><span class="sxs-lookup"><span data-stu-id="2d5ba-105">The definition for a referenced type was not found.</span></span> <span data-ttu-id="2d5ba-106">如果必要的 DLL 檔案未包含在編譯中，就會發生此狀況。</span><span class="sxs-lookup"><span data-stu-id="2d5ba-106">This could occur if a required .DLL file is not included in the compilation.</span></span> <span data-ttu-id="2d5ba-107">如需詳細資訊, 請參閱[加入參考對話方塊](/visualstudio/ide/how-to-add-or-remove-references-by-using-the-reference-manager)和[/ReferenceC# (編譯器選項)](../language-reference/compiler-options/reference-compiler-option.md)。</span><span class="sxs-lookup"><span data-stu-id="2d5ba-107">For more information, see [Add Reference Dialog Box](/visualstudio/ide/how-to-add-or-remove-references-by-using-the-reference-manager) and [/reference (C# Compiler Options)](../language-reference/compiler-options/reference-compiler-option.md).</span></span>  
  
 <span data-ttu-id="2d5ba-108">下列編譯序列將會導致 CS0012：</span><span class="sxs-lookup"><span data-stu-id="2d5ba-108">The following sequence of compilations will result in CS0012:</span></span>  
  
```csharp  
// cs0012a.cs  
// compile with: /target:library  
public class A {}  
```  
  
 <span data-ttu-id="2d5ba-109">然後：</span><span class="sxs-lookup"><span data-stu-id="2d5ba-109">Then:</span></span>  
  
```csharp  
// cs0012b.cs  
// compile with: /target:library /reference:cs0012a.dll  
public class B  
{  
   public static A f()  
   {  
      return new A();  
   }  
}  
```  
  
 <span data-ttu-id="2d5ba-110">然後：</span><span class="sxs-lookup"><span data-stu-id="2d5ba-110">Then:</span></span>  
  
```csharp  
// cs0012c.cs  
// compile with: /reference:cs0012b.dll  
class C  
{  
   public static void Main()  
   {  
      object o = B.f();   // CS0012  
   }  
}  
```  
  
 <span data-ttu-id="2d5ba-111">您可以使用 `/reference:cs0012b.dll;cs0012a.dll`編譯來解決這個 CS0012，或在 Visual Studio 中使用 [Add Reference Dialog Box](/visualstudio/ide/how-to-add-or-remove-references-by-using-the-reference-manager) 來加入 cs0012b.dll 和 cs0012a.dll 的參考。</span><span class="sxs-lookup"><span data-stu-id="2d5ba-111">You could resolve this CS0012 by compiling with `/reference:cs0012b.dll;cs0012a.dll`, or in Visual Studio by using the [Add Reference Dialog Box](/visualstudio/ide/how-to-add-or-remove-references-by-using-the-reference-manager) to add a reference to cs0012a.dll in addition to cs0012b.dll.</span></span>
