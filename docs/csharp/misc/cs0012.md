 <span data-ttu-id="91bd5-105">找不到參考類型的定義。</span><span class="sxs-lookup"><span data-stu-id="91bd5-105">The definition for a referenced type was not found.</span></span> <span data-ttu-id="91bd5-106">如果必要的 DLL 檔案未包含在編譯中，就會發生此狀況。</span><span class="sxs-lookup"><span data-stu-id="91bd5-106">This could occur if a required .DLL file is not included in the compilation.</span></span> <span data-ttu-id="91bd5-107">如需詳細資訊，請參閱[加入參考對話方塊](http://msdn.microsoft.com/library/2feb0fe2-0805-4cc9-8cba-b0315849dfb7)和[/reference （C# 編譯器選項）](../../csharp/language-reference/compiler-options/reference-compiler-option.md)。</span><span class="sxs-lookup"><span data-stu-id="91bd5-107">For more information, see [Add Reference Dialog Box](http://msdn.microsoft.com/library/2feb0fe2-0805-4cc9-8cba-b0315849dfb7) and [/reference (C# Compiler Options)](../../csharp/language-reference/compiler-options/reference-compiler-option.md).</span></span>  
  
 <span data-ttu-id="91bd5-108">下列編譯序列將會導致 CS0012：</span><span class="sxs-lookup"><span data-stu-id="91bd5-108">The following sequence of compilations will result in CS0012:</span></span>  
  
```

```  
  
 <span data-ttu-id="91bd5-109">然後：</span><span class="sxs-lookup"><span data-stu-id="91bd5-109">Then:</span></span>  
  
```

public class B  
{  
   public static A f()  
   {  
      return new A();  
   }  
}  
```  
  
 <span data-ttu-id="91bd5-110">然後：</span><span class="sxs-lookup"><span data-stu-id="91bd5-110">Then:</span></span>  
  
```

{  
   public static void Main()  
   {  
      object o = B.f();   // CS0012  
   }  
}  
```
