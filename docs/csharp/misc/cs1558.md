# <a name="compiler-error-cs1558"></a><span data-ttu-id="15fc8-102">編譯器錯誤 CS1558</span><span class="sxs-lookup"><span data-stu-id="15fc8-102">Compiler Error CS1558</span></span>
<span data-ttu-id="15fc8-103">'class' 沒有合適的靜態 Main 方法</span><span class="sxs-lookup"><span data-stu-id="15fc8-103">'class' does not have a suitable static Main method</span></span>  
  
 <span data-ttu-id="15fc8-104">[/Main](../../csharp/language-reference/compiler-options/main-compiler-option.md) 編譯器選項指定了在其中尋找 **Main** 方法的類別。</span><span class="sxs-lookup"><span data-stu-id="15fc8-104">The [/main](../../csharp/language-reference/compiler-options/main-compiler-option.md) compiler option specified a class in which to look for a **Main** method.</span></span> <span data-ttu-id="15fc8-105">不過， [Main](../../csharp/programming-guide/main-and-command-args/index.md) 方法未正確定義。</span><span class="sxs-lookup"><span data-stu-id="15fc8-105">However, the [Main](../../csharp/programming-guide/main-and-command-args/index.md) method was not defined correctly.</span></span>  
  
 <span data-ttu-id="15fc8-106">下列範例會產生 CS1558，因為傳回類型無效。</span><span class="sxs-lookup"><span data-stu-id="15fc8-106">The following example generates CS1558 because of invalid return type.</span></span>  
  
```  
// CS1558.cs  
// compile with: /main:MyNamespace.MyClass  
  
namespace MyNamespace  
{  
   public class MyClass  
   {  
      public static float Main()   
      {  
         return 0.0; // CS1558 because the return type is a float.  
      }  
   }  
}  
```
