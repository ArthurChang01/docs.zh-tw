# <a name="compiler-warning-level-3-cs0169"></a>編譯器警告 (層級 3) CS0169
絕不會使用私用欄位 'class member'  
  
 已宣告私用變數但從未參考。 在您宣告類別的私用成員但未使用時，通常會產生這個警告。  
  
 下列範例會產生 CS0169：  
  
```csharp  
// compile with: /W:3  
using System;  
public class ClassX  
{  
   int i;   // CS0169, i is not used anywhere  
  
   public static void Main()  
   {  
   }  
}  
```
