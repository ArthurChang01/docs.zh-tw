# <a name="compiler-error-cs5001"></a><span data-ttu-id="8e7b5-102">編譯器錯誤 CS5001</span><span class="sxs-lookup"><span data-stu-id="8e7b5-102">Compiler Error CS5001</span></span>
<span data-ttu-id="8e7b5-103">程式 'program' 未包含適合進入點的靜態 'Main' 方法</span><span class="sxs-lookup"><span data-stu-id="8e7b5-103">Program 'program' does not contain a static 'Main' method suitable for an entry point</span></span>  
  
 <span data-ttu-id="8e7b5-104">此錯誤發生時沒有靜態[Main](../../csharp/programming-guide/main-and-command-args/index.md)產生可執行檔的程式碼中找不到具有正確的簽章的方法。</span><span class="sxs-lookup"><span data-stu-id="8e7b5-104">This error occurs when no static [Main](../../csharp/programming-guide/main-and-command-args/index.md) method with a correct signature is found in the code that produces an executable file.</span></span> <span data-ttu-id="8e7b5-105">如果所定義進入點函式 `Main`的大小寫 (例如小寫 `main`) 錯誤，也會發生這個錯誤。</span><span class="sxs-lookup"><span data-stu-id="8e7b5-105">This error also occurs if the entry point function, `Main`, is defined with the wrong case, such as lower-case `main`.</span></span>  
  
-   <span data-ttu-id="8e7b5-106">`Main`必須宣告為靜態，且必須傳回[void](../../csharp/language-reference/keywords/void.md)或[int](../../csharp/language-reference/keywords/int.md)，而且必須是沒有參數，否則型別的一個參數`string[]`。</span><span class="sxs-lookup"><span data-stu-id="8e7b5-106">`Main` must be declared as static and it must return [void](../../csharp/language-reference/keywords/void.md) or [int](../../csharp/language-reference/keywords/int.md), and it must have either no parameters or else one parameter of type `string[]`.</span></span>
