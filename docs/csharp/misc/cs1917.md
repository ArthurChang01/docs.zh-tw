## <a name="example"></a><span data-ttu-id="7caab-108">範例</span><span class="sxs-lookup"><span data-stu-id="7caab-108">Example</span></span>

```  
// cs1917.cs  
class CS1917  
{  
    public struct TestStruct  
    {  
        public int i;  
    }  
    public class C  
    {  
        public readonly TestStruct str = new TestStruct();  
        public static int Main()  
        {  
            C c = new C { str = { i = 1 } }; // CS1917  
            return 0;  
        }  
    }  
}  
```
