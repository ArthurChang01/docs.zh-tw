# <a name="compiler-error-cs0160"></a><span data-ttu-id="686ae-102">編譯器錯誤 CS0160</span><span class="sxs-lookup"><span data-stu-id="686ae-102">Compiler Error CS0160</span></span>
<span data-ttu-id="686ae-103">之前的 catch 子句已取得所有屬於此類型或超級類型 ('type') 的例外狀況</span><span class="sxs-lookup"><span data-stu-id="686ae-103">A previous catch clause already catches all exceptions of this or of a super type ('type')</span></span>  
  
<span data-ttu-id="686ae-104">一系列的 [catch](../language-reference/keywords/try-catch.md) 陳述式必須依照衍生的遞減順序。</span><span class="sxs-lookup"><span data-stu-id="686ae-104">A series of [catch](../language-reference/keywords/try-catch.md) statements needs to be in decreasing order of derivation.</span></span> <span data-ttu-id="686ae-105">例如，最具衍生性的物件必須最先出現。</span><span class="sxs-lookup"><span data-stu-id="686ae-105">For example, the most derived objects must appear first.</span></span>
  
 <span data-ttu-id="686ae-106">如需詳細資訊, 請參閱[例外狀況和例外狀況處理](../programming-guide/exceptions/index.md)。</span><span class="sxs-lookup"><span data-stu-id="686ae-106">For more information, see [Exceptions and Exception Handling](../programming-guide/exceptions/index.md).</span></span>  
  
 <span data-ttu-id="686ae-107">下列範例會產生 CS0160：</span><span class="sxs-lookup"><span data-stu-id="686ae-107">The following sample generates CS0160:</span></span>  
  
```csharp  
// CS0160.cs  
public class MyClass2 : System.Exception {}  
public class MyClass  
{  
   public static void Main()  
   {  
      try {}  
  
      catch(System.Exception) {}   // Second-most derived; should be second catch  
      catch(MyClass2) {}   // CS0160  Most derived; should be first catch  
   }  
}  
```
