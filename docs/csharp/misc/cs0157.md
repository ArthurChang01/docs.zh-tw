 <span data-ttu-id="c8c67-106">下列範例會產生 CS0157：</span><span class="sxs-lookup"><span data-stu-id="c8c67-106">The following sample generates CS0157:</span></span>

```csharp  
{  
   public class MyClass2 : Exception  
   {  
   }

   {  
      public static void Main()  
      {  
         try  
         {  
         }  
  
         finally  
         {  
            return;   // CS0157, cannot leave finally clause  
         }  
      }  
   }  
}  
```
