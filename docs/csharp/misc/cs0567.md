# <a name="compiler-error-cs0567"></a>編譯器錯誤 CS0567
介面無法包含運算子  
  
 [介面](../language-reference/keywords/interface.md) 定義不允許運算子。  
  
 下列範例會產生 CS0567：  
  
```csharp  
// CS0567.cs  
interface IA  
{  
   int operator +(int aa, int bb);   // CS0567  
}  
  
class Sample  
{  
   public static void Main()
   {  
   }  
}  
```
