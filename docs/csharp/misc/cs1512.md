 <span data-ttu-id="fd067-104">[base](../../csharp/language-reference/keywords/base.md) 關鍵字已使用在方法、屬性或建構函式之外。</span><span class="sxs-lookup"><span data-stu-id="fd067-104">The [base](../../csharp/language-reference/keywords/base.md) keyword was used outside of a method, property, or constructor.</span></span>

```

{  
    private String xx = base.ToString();   // CS1512  
    // Try putting this initialization in the constructor instead:  
    // public CMyClass()  
    // {  
    //    xx = base.ToString();  
    // }  
  
    public static void Main()  
    {  
        CMyClass z = new CMyClass();  
    }  
}  
```
