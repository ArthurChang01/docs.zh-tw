 <span data-ttu-id="3d0a3-105">下列範例會產生 CS1512：</span><span class="sxs-lookup"><span data-stu-id="3d0a3-105">The following example generates CS1512:</span></span>  
  
```csharp

{  
    private String xx = base.ToString();   // CS1512  
    // Try putting this initialization in the constructor instead:

    // {  
    //    xx = base.ToString();  
    // }  
  
    public static void Main()  
    {  
        CMyClass z = new CMyClass();  
    }  
}  
```
