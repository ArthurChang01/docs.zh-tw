 <span data-ttu-id="33698-104">[base](../language-reference/keywords/base.md) 關鍵字已使用在方法、屬性或建構函式之外。</span><span class="sxs-lookup"><span data-stu-id="33698-104">The [base](../language-reference/keywords/base.md) keyword was used outside of a method, property, or constructor.</span></span>  
  
 <span data-ttu-id="33698-105">下列範例會產生 CS1512：</span><span class="sxs-lookup"><span data-stu-id="33698-105">The following example generates CS1512:</span></span>  
  
```csharp  
// CS1512.cs

class Base {}  
  
class CMyClass : Base  
{  
    private String xx = base.ToString();   // CS1512  
    // Try putting this initialization in the constructor instead:  
    // public CMyClass()  
    // {  
    //    xx = base.ToString();  
    // }  
  
    public static void Main()  
    {  
        CMyClass z = new CMyClass();  
    }  
}  
```
