## <a name="example"></a><span data-ttu-id="66d31-108">範例</span><span class="sxs-lookup"><span data-stu-id="66d31-108">Example</span></span>  
 <span data-ttu-id="66d31-109">下列範例會產生 CS3018。</span><span class="sxs-lookup"><span data-stu-id="66d31-109">The following sample generates CS3018.</span></span>  
  
```  
// CS3018.cs  
// compile with: /target:library  
using System;  
  
[assembly: CLSCompliant(true)]  
[CLSCompliant(false)]  
public class Outer  
{  
   [CLSCompliant(true)]   // CS3018  
   public class Nested {}  
  
   // OK  
   public class Nested2 {}  
  
   [CLSCompliant(false)]  
   public class Nested3 {}  
}  
```
