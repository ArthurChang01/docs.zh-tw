# <a name="compiler-error-cs1627"></a><span data-ttu-id="9bdba-102">編譯器錯誤 CS1627</span><span class="sxs-lookup"><span data-stu-id="9bdba-102">Compiler Error CS1627</span></span>
<span data-ttu-id="9bdba-103">yield return 之後應接著運算式</span><span class="sxs-lookup"><span data-stu-id="9bdba-103">Expression expected after yield return</span></span>  
  
 <span data-ttu-id="9bdba-104">如果使用 `yield` 但沒有運算式，就會發生這個錯誤。</span><span class="sxs-lookup"><span data-stu-id="9bdba-104">This error occurs if `yield` is used without an expression.</span></span> <span data-ttu-id="9bdba-105">若要避免這個錯誤，請在陳述式中插入適當的運算式。</span><span class="sxs-lookup"><span data-stu-id="9bdba-105">To avoid this error, insert the appropriate expression in the statement.</span></span>  
  
 <span data-ttu-id="9bdba-106">下列範例會產生 CS1627：</span><span class="sxs-lookup"><span data-stu-id="9bdba-106">The following sample generates CS1627:</span></span>  
  
```csharp  
// CS1627.cs  
using System.Collections;  
  
class C : IEnumerable  
{  
   public IEnumerator GetEnumerator()  
   {  
      yield return;   // CS1627  
      // To resolve, add the following line:  
      // yield return 0;  
   }  
}  
  
public class CMain  
{  
   public static void Main() { }  
}  
```
