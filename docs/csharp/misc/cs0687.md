<span data-ttu-id="f3f71-103">命名空間別名限定詞 '::' 一定會解析為類型或命名空間，所以不能用在這裡。</span><span class="sxs-lookup"><span data-stu-id="f3f71-103">The namespace alias qualifier '::' always resolves to a type or namespace so is illegal here.</span></span> <span data-ttu-id="f3f71-104">請考慮用 '.' 替代。</span><span class="sxs-lookup"><span data-stu-id="f3f71-104">Consider using '.' instead.</span></span>

## <a name="example"></a><span data-ttu-id="f3f71-108">範例</span><span class="sxs-lookup"><span data-stu-id="f3f71-108">Example</span></span>  
 <span data-ttu-id="f3f71-109">下列範例會產生 CS0687：</span><span class="sxs-lookup"><span data-stu-id="f3f71-109">The following sample generates CS0687:</span></span>  
  
```csharp  
// CS0687.cs  
  
using M = Test;  
using System;  
  
public class Test   
{  
    public static int x = 77;  
  
    public static void Main()   
    {  
        Console.WriteLine(M::x); // CS0687  
        // To resolve use the following line instead:  
        // Console.WriteLine(M.x);  
    }  
}  
```
