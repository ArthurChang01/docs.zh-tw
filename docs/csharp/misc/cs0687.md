## <a name="example"></a><span data-ttu-id="209d6-108">範例</span><span class="sxs-lookup"><span data-stu-id="209d6-108">Example</span></span>  
 <span data-ttu-id="209d6-109">下列範例會產生 CS0687：</span><span class="sxs-lookup"><span data-stu-id="209d6-109">The following sample generates CS0687:</span></span>  
  
```  
// CS0687.cs  
  
using M = Test;  
using System;  
  
public class Test   
{  
    public static int x = 77;  
  
    public static void Main()   
    {  
        Console.WriteLine(M::x); // CS0687  
        // To resolve use the following line instead:  
        // Console.WriteLine(M.x);  
    }  
}  
```
