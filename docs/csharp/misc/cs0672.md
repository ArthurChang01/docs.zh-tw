# <a name="compiler-warning-level-1-cs0672"></a><span data-ttu-id="4d3ee-102">編譯器警告 (層級 1) CS0672</span><span class="sxs-lookup"><span data-stu-id="4d3ee-102">Compiler Warning (level 1) CS0672</span></span>
<span data-ttu-id="4d3ee-103">成員 'member1' 會覆寫過時成員 'member2'。</span><span class="sxs-lookup"><span data-stu-id="4d3ee-103">Member 'member1' overrides obsolete member 'member2.</span></span> <span data-ttu-id="4d3ee-104">將 Obsolete 屬性加入 'member1'</span><span class="sxs-lookup"><span data-stu-id="4d3ee-104">Add the Obsolete attribute to 'member1'</span></span>  
  
 <span data-ttu-id="4d3ee-105">編譯器發現方法的 `override` 標記為 `obsolete`。</span><span class="sxs-lookup"><span data-stu-id="4d3ee-105">The compiler found an `override` to a method marked as `obsolete`.</span></span> <span data-ttu-id="4d3ee-106">不過，覆寫方法本身未標記為過時。</span><span class="sxs-lookup"><span data-stu-id="4d3ee-106">However, the overriding method was not itself marked as obsolete.</span></span> <span data-ttu-id="4d3ee-107">覆寫方法仍會產生 [CS0612](../../csharp/misc/cs0612.md)(呼叫時)。</span><span class="sxs-lookup"><span data-stu-id="4d3ee-107">The overriding method will still generate [CS0612](../../csharp/misc/cs0612.md), if called.</span></span>  
  
 <span data-ttu-id="4d3ee-108">請檢閱方法宣告，並明確表示方法 (和其所有覆寫) 是否應該標記為 `obsolete`。</span><span class="sxs-lookup"><span data-stu-id="4d3ee-108">Review your method declarations and explicitly indicate whether a method (and all of its overrides) should be marked `obsolete`.</span></span>  
  
 <span data-ttu-id="4d3ee-109">下列範例會產生 CS0672：</span><span class="sxs-lookup"><span data-stu-id="4d3ee-109">The following sample generates CS0672:</span></span>  
  
```csharp  
// CS0672.cs  
// compile with: /W:1  
class MyClass  
{  
   [System.Obsolete]  
   public virtual void ObsoleteMethod()  
   {  
   }  
}  
  
class MyClass2 : MyClass  
{  
   public override void ObsoleteMethod()   // CS0672  
   {  
   }  
}  
  
class MainClass  
{  
   static public void Main()  
   {  
   }  
}  
```
