## <a name="example"></a><span data-ttu-id="8c12c-106">範例</span><span class="sxs-lookup"><span data-stu-id="8c12c-106">Example</span></span>  
 <span data-ttu-id="8c12c-107">下列範例會產生 CS0704：</span><span class="sxs-lookup"><span data-stu-id="8c12c-107">The following sample generates CS0704:</span></span>  
  
```csharp  
// CS0704.cs  
class B  
{  
    public class I { }  
}  
  
class C<T> where T : B  
{  
    T.I f;  // CS0704 – member lookup on type parameter  
    // Try this instead:  
    // B.I f;  
}  
  
class CMain  
{  
    public static void Main() {}  
}  
```
