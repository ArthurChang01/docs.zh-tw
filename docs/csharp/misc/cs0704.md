# <a name="compiler-error-cs0704"></a>編譯器錯誤 CS0704
無法在 'type' 中進行成員查詢，因為它是類型參數  
  
 無法透過類型參數指定內部類型。 請嘗試明確地使用所需的類型。  
  
## <a name="example"></a>範例  
 下列範例會產生 CS0704：  
  
```csharp  
// CS0704.cs  
class B  
{  
    public class I { }  
}  
  
class C<T> where T : B  
{  
    T.I f;  // CS0704 – member lookup on type parameter  
    // Try this instead:  
    // B.I f;  
}  
  
class CMain  
{  
    public static void Main() {}  
}  
```
