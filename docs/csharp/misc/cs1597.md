# <a name="compiler-error-cs1597"></a>編譯器錯誤 CS1597
方法或存取子區塊後的分號無效  
  
 不需要 (或不允許) 用分號結束方法或存取子區塊。  
  
 下例會產生 CS1597：  
  
```csharp  
// CS1597.cs  
class TestClass  
{  
   public static void Main()  
   {  
   };   // CS1597, remove semicolon  
}  
```
