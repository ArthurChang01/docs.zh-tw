# <a name="compiler-error-cs0025"></a>編譯器錯誤 CS0025
找不到標準程式庫檔 'file'  
  
 找不到編譯器所需的一或多個檔案。 請檢查路徑正確，而且檔案存在。  
  
 如果檔案是 Visual Studio 系統檔案，您可能需要修復您的 Visual Studio 安裝，或完全重新安裝。
