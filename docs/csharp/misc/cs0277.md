 <span data-ttu-id="ef7a9-105">當您嘗試實作介面的屬性，但類別中屬性存取子的實作不是公用時，就會發生這個錯誤。</span><span class="sxs-lookup"><span data-stu-id="ef7a9-105">This error occurs when you try to implement a property of an interface, but the implementation of the property accessor in the class is not public.</span></span> <span data-ttu-id="ef7a9-106">實作介面成員的方法必須具有公用存取範圍。</span><span class="sxs-lookup"><span data-stu-id="ef7a9-106">Methods that implement interface members need to have public accessibility.</span></span> <span data-ttu-id="ef7a9-107">若要解決，請移除屬性存取子上的存取修飾詞。</span><span class="sxs-lookup"><span data-stu-id="ef7a9-107">To resolve, remove the access modifier on the property accessor.</span></span>  
  
## <a name="example"></a><span data-ttu-id="ef7a9-108">範例</span><span class="sxs-lookup"><span data-stu-id="ef7a9-108">Example</span></span>  
 <span data-ttu-id="ef7a9-109">下列範例會產生 CS0277：</span><span class="sxs-lookup"><span data-stu-id="ef7a9-109">The following example generates CS0277:</span></span>  
  
```csharp  
// CS0277.cs  
public interface MyInterface  
{  
    int Property  
    {  
        get;  
        set;  
    }  
}  
  
public class MyClass : MyInterface   // CS0277  
{  
    public int Property  
    {  
        get { return 0; }  
        // Try this instead:  
        //set { }  
        protected set { }  
    }  
}  
```
