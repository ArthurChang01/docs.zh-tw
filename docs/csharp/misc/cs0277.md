## <a name="example"></a><span data-ttu-id="20261-108">範例</span><span class="sxs-lookup"><span data-stu-id="20261-108">Example</span></span>  
 <span data-ttu-id="20261-109">下列範例會產生 CS0277：</span><span class="sxs-lookup"><span data-stu-id="20261-109">The following example generates CS0277:</span></span>  
  
```csharp

    int Property  
    {  
        get;  
        set;  
    }  
}

{  
    public int Property  
    {  
        get { return 0; }  
        // Try this instead:  
        //set { }  
        protected set { }  
    }  
}  
```
