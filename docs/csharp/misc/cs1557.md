# <a name="compiler-error-cs1557"></a>編譯器錯誤 CS1557
Main 方法無法使用 'class'，因為兩者在不同的輸出檔中  
  
 在多重輸出檔編譯中，針對一個輸出檔指定了 [/main](../language-reference/compiler-options/main-compiler-option.md) 編譯器選項。 不過，在原始程式碼中找不到此類別可進行 /main 碼編譯; 在編譯中之其他輸出檔之一的原始程式碼檔案中找到它。
