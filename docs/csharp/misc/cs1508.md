# <a name="compiler-error-cs1508"></a>編譯器錯誤 CS1508
在這個組件中已經使用過資源識別項 'identifier'  
  
 在編譯中，相同的識別碼（***identifier***）已傳遞給兩個或多個[/resource](../language-reference/compiler-options/resource-compiler-option.md)或[-linkresource](../language-reference/compiler-options/linkresource-compiler-option.md)編譯器選項。  
  
 例如，下列選項會產生 CS1508：  
  
```console  
/resource:anyfile.bmp,DuplicatIdent /linkresource:a.bmp,DuplicatIdent  
```
