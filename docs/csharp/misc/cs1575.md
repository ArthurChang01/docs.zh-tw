# <a name="compiler-error-cs1575"></a><span data-ttu-id="f3136-102">編譯器錯誤 CS1575</span><span class="sxs-lookup"><span data-stu-id="f3136-102">Compiler Error CS1575</span></span>
<span data-ttu-id="f3136-103">stackalloc 運算式在類型之後需要有 []</span><span class="sxs-lookup"><span data-stu-id="f3136-103">A stackalloc expression requires [] after type</span></span>  
  
 <span data-ttu-id="f3136-104">要求的配置大小與 [stackalloc](../../csharp/language-reference/keywords/stackalloc.md)，必須在方括號中指定。</span><span class="sxs-lookup"><span data-stu-id="f3136-104">The size of the requested allocation, with [stackalloc](../../csharp/language-reference/keywords/stackalloc.md), must be specified in square brackets.</span></span>  
  
 <span data-ttu-id="f3136-105">下例會產生 CS1575：</span><span class="sxs-lookup"><span data-stu-id="f3136-105">The following sample generates CS1575:</span></span>  
  
```csharp  
// CS1575.cs  
// compile with: /unsafe  
public class MyClass  
{  
   unsafe public static void Main()  
   {  
      int *p = stackalloc int (30);   // CS1575  
      // try the following line instead  
      // int *p = stackalloc int [30];  
   }  
}  
```
