# <a name="compiler-error-cs0821"></a>編譯器錯誤 CS0821
隱含類型區域變數不可為 fixed  
  
 在 `fixed` 內容中不支援隱含類型區域變數和匿名類型。  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
1.  請從變數移除 `fixed` 修飾詞，或為變數指定明確類型。  
  
## <a name="example"></a>範例  
 下列程式碼會產生 CS0821：  
  
```csharp  
class A  
{  
    static int x;  
  
    public static int Main()  
    {  
        unsafe  
        {  
            fixed (var p = &x) { }  
        }  
        return -1;  
    }  
}  
```
