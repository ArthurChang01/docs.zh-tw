# <a name="compiler-warning-level-1-cs1635"></a><span data-ttu-id="e9d7b-102">編譯器警告 (層級 1) CS1635</span><span class="sxs-lookup"><span data-stu-id="e9d7b-102">Compiler Warning (level 1) CS1635</span></span>
<span data-ttu-id="e9d7b-103">無法還原警告 'warning code'，因為它已全域停用</span><span class="sxs-lookup"><span data-stu-id="e9d7b-103">Cannot restore warning 'warning code' because it was disabled globally</span></span>  
  
 <span data-ttu-id="e9d7b-104">如果您使用 **/nowarn** 命令列選項或專案設定來停用整個編譯單元的警告，但使用 `#pragma warning restore` 來嘗試還原該警告，則會發生這個警告。</span><span class="sxs-lookup"><span data-stu-id="e9d7b-104">This warning occurs if you use the **/nowarn** command line option or project setting to disable a warning for the entire compilation unit, but you use `#pragma warning restore` to attempt to restore that warning.</span></span> <span data-ttu-id="e9d7b-105">若要解決這個錯誤，請移除 /nowarn 命令列選項或專案設定，或移除透過命令列或專案設定所停用之任何警告的 `#pragma warning restore` 。</span><span class="sxs-lookup"><span data-stu-id="e9d7b-105">To resolve this error, remove the /nowarn command line option or project setting, or remove the `#pragma warning restore` for any warnings you are disabling via the command line or project settings.</span></span> <span data-ttu-id="e9d7b-106">如需詳細資訊，請參閱 [#pragma 警告](../../csharp/language-reference/preprocessor-directives/preprocessor-pragma-warning.md) 主題。</span><span class="sxs-lookup"><span data-stu-id="e9d7b-106">For more information, see the [#pragma warning](../../csharp/language-reference/preprocessor-directives/preprocessor-pragma-warning.md) topic.</span></span>  
  
 <span data-ttu-id="e9d7b-107">下列範例會產生 CS1635：</span><span class="sxs-lookup"><span data-stu-id="e9d7b-107">The following sample generates CS1635:</span></span>  
  
```  
// CS1635.cs  
// compile with: /w:1 /nowarn:162  
  
enum MyEnum {one=1,two=2,three=3};  
  
class MyClass  
{  
    public static void Main()  
    {  
#pragma warning disable 162  
  
    if (MyEnum.three == MyEnum.two)  
        System.Console.WriteLine("Duplicate");  
  
#pragma warning restore 162  
    }  
}  
```
