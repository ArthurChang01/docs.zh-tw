# <a name="compiler-error-cs1675"></a>編譯器錯誤 CS1675
列舉不能有類型參數  
  
 若要解決這個錯誤，請從 `enum` 宣告中移除類型參數。  
  
## <a name="example"></a>範例  
 下列範例會產生 CS1675：  
  
```csharp  
// CS1675.cs  
enum E<T>  // CS1675  
{  
}  
  
class CMain  
{  
    public static void Main()  
    {  
    }  
}  
```
