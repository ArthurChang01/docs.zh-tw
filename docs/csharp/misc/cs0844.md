## <a name="example"></a><span data-ttu-id="0feb1-111">範例</span><span class="sxs-lookup"><span data-stu-id="0feb1-111">Example</span></span>

```csharp  
            num = 5; // CS0844
```

```csharp  
    {  
        int num;  
        public void TestMethod()  
        {  
            this.num = 5; // Error fixed.
            int num = 6;
        }  
        public static int Main()  
        {  
            return 1;  
        }  
    }  
```
