# <a name="compiler-error-cs0709"></a>編譯器錯誤 CS0709
'derived class': 不能衍生自靜態類別 '基底類別'  
  
 靜態類別無法具現化或從中衍生。 換句話說，靜態類別不能是任何其他類別的基底類別。  
  
## <a name="example"></a>範例  
 下列範例會產生 CS0709。  
  
```csharp  
// CS0709.cs  
// compile with: /target:library  
public static class Base {}  
public class Derived : Base {}   // CS0709  
```
