# <a name="compiler-error-cs0226"></a>編譯器錯誤 CS0226
__arglist 運算式只可出現於呼叫或 new 運算式中。  
  
 不支援的關鍵字 `__arglist` 只可出現於方法呼叫或 new 運算式中。  
  
## <a name="example"></a>範例  
 下列程式碼會產生 CS0226：  
  
```csharp  
// cs0226.cs  
using System;  
  
public class C  
    {  
    public static int Main ()  
        {  
        __arglist(1,"This is a string"); // CS0226  
        return 0;  
        }  
    }  
```
