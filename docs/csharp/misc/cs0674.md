# <a name="compiler-error-cs0674"></a><span data-ttu-id="cf262-102">編譯器錯誤 CS0674</span><span class="sxs-lookup"><span data-stu-id="cf262-102">Compiler Error CS0674</span></span>
<span data-ttu-id="cf262-103">請勿使用 'System.ParamArrayAttribute'。</span><span class="sxs-lookup"><span data-stu-id="cf262-103">Do not use 'System.ParamArrayAttribute'.</span></span> <span data-ttu-id="cf262-104">請改用 'params' 關鍵字。</span><span class="sxs-lookup"><span data-stu-id="cf262-104">Use the 'params' keyword instead.</span></span>  
  
 <span data-ttu-id="cf262-105">C# 編譯器不允許使用 <xref:System.ParamArrayAttribute?displayProperty=nameWithType>；請改用 [params](../language-reference/keywords/params.md) 。</span><span class="sxs-lookup"><span data-stu-id="cf262-105">The C# compiler does not allow for the use of <xref:System.ParamArrayAttribute?displayProperty=nameWithType>; use [params](../language-reference/keywords/params.md) instead.</span></span>  
  
 <span data-ttu-id="cf262-106">下列範例會產生 CS0674：</span><span class="sxs-lookup"><span data-stu-id="cf262-106">The following sample generates CS0674:</span></span>  
  
```csharp  
// CS0674.cs  
using System;  
public class MyClass   
{  
  
   public static void UseParams([ParamArray] int[] list)   // CS0674  
   // try the following line instead  
   // public static void UseParams(params int[] list)   
   {  
      for ( int i = 0 ; i < list.Length ; i++ )  
         Console.WriteLine(list[i]);  
      Console.WriteLine();  
   }  
  
   public static void Main()   
   {  
      UseParams(1, 2, 3);  
   }  
}  
```
