 <span data-ttu-id="0ed26-104">在 [類別](../../csharp/language-reference/keywords/interface.md) 或 [結構](../../csharp/language-reference/keywords/class.md) 之外發現明確的 [介面](../../csharp/language-reference/keywords/struct.md)宣告。</span><span class="sxs-lookup"><span data-stu-id="0ed26-104">An explicit [interface](../../csharp/language-reference/keywords/interface.md) declaration was found outside a [class](../../csharp/language-reference/keywords/class.md) or [struct](../../csharp/language-reference/keywords/struct.md).</span></span>  
  
 <span data-ttu-id="0ed26-105">下列範例會產生 CS0541：</span><span class="sxs-lookup"><span data-stu-id="0ed26-105">The following sample generates CS0541:</span></span>  
  
```csharp  
// CS0541.cs  
namespace x  
{  
   interface IFace  
   {  
      void F();  
   }  
  
   interface IFace2 : IFace  
   {  
      void IFace.F();   // CS0541  
   }  
}  
```
