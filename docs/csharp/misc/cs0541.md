 <span data-ttu-id="65c70-104">在 [類別](../language-reference/keywords/interface.md) 或 [結構](../language-reference/keywords/class.md) 之外發現明確的 [介面](../language-reference/builtin-types/struct.md)宣告。</span><span class="sxs-lookup"><span data-stu-id="65c70-104">An explicit [interface](../language-reference/keywords/interface.md) declaration was found outside a [class](../language-reference/keywords/class.md) or [struct](../language-reference/builtin-types/struct.md).</span></span>  
  
 <span data-ttu-id="65c70-105">下列範例會產生 CS0541：</span><span class="sxs-lookup"><span data-stu-id="65c70-105">The following sample generates CS0541:</span></span>  
  
```csharp

{  
   interface IFace  
   {  
      void F();

      void IFace.F();   // CS0541  
   }  
}  
```
