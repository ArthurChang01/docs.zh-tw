# <a name="compiler-warning-level-1-cs2029"></a>編譯器警告 (層級 1) CS2029
無效的 '/define' 值; 'identifier' 不是有效的識別項  
  
 如果 [/define](../../csharp/language-reference/compiler-options/define-compiler-option.md) 選項中所使用的值具有一些無效的字元，就會發生這個警告。  
  
 [/nowarn](../../csharp/language-reference/compiler-options/nowarn-compiler-option.md) 選項無法隱藏這個警告。
