# <a name="compiler-error-cs1056"></a>編譯器錯誤 CS1056
不應該有字元 'character'  
  
 C# 編譯器遇到未預期的字元，而且無法識別目前正在處理的語彙基元。 例如，如果編譯器在處理識別項期間遇到歐元字元，將無法分類識別項，因為歐元字元只在字串中有效，而且編譯器知道它不是處理字串。
