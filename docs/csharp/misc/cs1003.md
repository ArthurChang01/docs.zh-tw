# <a name="compiler-error-cs1003"></a>編譯器錯誤 CS1003
語法錯誤，應為 'char'  
  
 編譯器會因為數種錯誤狀況的任何一種而產生這個錯誤。 請檢閱您的程式碼，找出語法錯誤。  
  
 下列範例會產生 CS1003：  
  
```csharp  
// CS1003.cs  
public class b  
{  
   public static void Main()  
   {  
      int[] a;  
      a[);   // CS1003  
   }  
}  
```
