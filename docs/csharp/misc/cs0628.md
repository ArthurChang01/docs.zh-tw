# <a name="compiler-warning-level-4-cs0628"></a><span data-ttu-id="a083c-102">編譯器警告 (層級 4) CS0628</span><span class="sxs-lookup"><span data-stu-id="a083c-102">Compiler Warning (level 4) CS0628</span></span>

 <span data-ttu-id="a083c-104">[sealed](../../csharp/language-reference/keywords/sealed.md) 類別不能引入 [protected](../../csharp/language-reference/keywords/protected.md) 成員，因為沒有其他類別可以繼承自 `sealed` 類別，並且使用 `protected` 成員。</span><span class="sxs-lookup"><span data-stu-id="a083c-104">A [sealed](../../csharp/language-reference/keywords/sealed.md) class cannot introduce a [protected](../../csharp/language-reference/keywords/protected.md) member because no other class will be able to inherit from the `sealed` class and use the `protected` member.</span></span>

```  
{  
   protected int i;   // CS0628  
}

{  
   public static void Main()  
   {  
   }  
}  
```
