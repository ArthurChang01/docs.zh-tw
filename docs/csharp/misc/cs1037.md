# <a name="compiler-error-cs1037"></a>編譯器錯誤 CS1037
必須是可多載的運算子  
  
 使用 [/doc](../../csharp/language-reference/compiler-options/doc-compiler-option.md)指定註解時，編譯器會遇到無效的連結。
