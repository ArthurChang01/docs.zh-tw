# <a name="compiler-error-cs0117"></a>編譯器錯誤 CS0117
'type' 不包含 'identifier' 的定義  
  
-   在某些情況下，當參考到資料類型沒有的成員時，就會發生這個錯誤。  
  
## <a name="example"></a>範例  
 下列範例會產生 CS0117：  
  
```csharp  
// CS0117.cs  
public class BaseClass { }  
  
    public class base021 : BaseClass  
    {  
        public void TestInt()  
        {  
            int i = base.someMember; //CS0117  
        }  
        public static int Main()  
        {  
            return 1;  
        }  
    }  
```
