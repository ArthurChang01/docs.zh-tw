# <a name="compiler-error-cs1583"></a>編譯器錯誤 CS1583
'file' 不是有效的 Win32 資源檔  
  
 當您指定的檔案名具有[-win32res](../language-reference/compiler-options/win32res-compiler-option.md)編譯器選項，但它不是有效或格式正確的 Win32 資源檔時，就會發生此錯誤。 在 Visual Studio 中，檔案名稱是在專案設計工具的 [應用程式] 窗格中指定。
