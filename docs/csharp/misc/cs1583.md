# <a name="compiler-error-cs1583"></a>編譯器錯誤 CS1583
'file' 不是有效的 Win32 資源檔  
  
 當您指定的檔案名稱時，會發生此錯誤[/win32res](../../csharp/language-reference/compiler-options/win32res-compiler-option.md)不是無效或格式正確的 Win32 資源檔的編譯器選項。 在 Visual Studio 中，檔案名稱是在專案設計工具的 [應用程式] 窗格中指定。
