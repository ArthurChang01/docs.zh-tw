# <a name="compiler-error-cs1566"></a>編譯器錯誤 CS1566
讀取資源檔 'file' 時發生錯誤 — 'reason'  
  
 編譯器在傳遞至[-resource](../language-reference/compiler-options/resource-compiler-option.md)編譯器選項的檔案名時發生問題。
