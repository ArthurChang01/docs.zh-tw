# <a name="compiler-error-cs0736"></a><span data-ttu-id="fa599-102">編譯器錯誤 CS0736</span><span class="sxs-lookup"><span data-stu-id="fa599-102">Compiler Error CS0736</span></span>
<span data-ttu-id="fa599-103">'type name' 未實作介面成員 'member name'。</span><span class="sxs-lookup"><span data-stu-id="fa599-103">'type name' does not implement interface member 'member name'.</span></span> <span data-ttu-id="fa599-104">'method name' 無法實作介面成員，因為其為靜態。</span><span class="sxs-lookup"><span data-stu-id="fa599-104">'method name' cannot implement an interface member because it is static.</span></span>

## <a name="example"></a><span data-ttu-id="fa599-110">範例</span><span class="sxs-lookup"><span data-stu-id="fa599-110">Example</span></span>  
 <span data-ttu-id="fa599-111">下列程式碼會產生 CS0736，因為 `Program.testMethod` 宣告為靜態：</span><span class="sxs-lookup"><span data-stu-id="fa599-111">The following code generates CS0736 because `Program.testMethod` is declared as static:</span></span>  
  
```csharp  
// cs0736.cs  
namespace CS0736  
{     
  
    interface ITest  
    {  
        int testMethod(int x);  
    }  
  
    class Program : ITest // CS0736  
    {  
        public static int testMethod(int x) { return 0; }  
        // Try the following line instead.  
        // public int testMethod(int x) { return 0; }  
        public static void Main() { }  
    }      
}  
```
