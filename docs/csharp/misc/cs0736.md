# <a name="compiler-error-cs0736"></a><span data-ttu-id="80974-102">編譯器錯誤 CS0736</span><span class="sxs-lookup"><span data-stu-id="80974-102">Compiler Error CS0736</span></span>
<span data-ttu-id="80974-103">'type name' 未實作介面成員 'member name'。</span><span class="sxs-lookup"><span data-stu-id="80974-103">'type name' does not implement interface member 'member name'.</span></span> <span data-ttu-id="80974-104">'method name' 無法實作介面成員，因為其為靜態。</span><span class="sxs-lookup"><span data-stu-id="80974-104">'method name' cannot implement an interface member because it is static.</span></span>  
  
 <span data-ttu-id="80974-105">靜態方法隱含或明確地宣告為介面成員的實作時，會產生這個錯誤。</span><span class="sxs-lookup"><span data-stu-id="80974-105">This error is generated when a static method is either implicitly or explicitly declared as an implementation of an interface member.</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="80974-106">更正這個錯誤</span><span class="sxs-lookup"><span data-stu-id="80974-106">To correct this error</span></span>  
  
- <span data-ttu-id="80974-107">從方法宣告中移除 [靜態](../language-reference/keywords/static.md) 修飾詞。</span><span class="sxs-lookup"><span data-stu-id="80974-107">Remove the [static](../language-reference/keywords/static.md) modifier from the method declaration.</span></span>  
  
- <span data-ttu-id="80974-108">變更介面方法的名稱。</span><span class="sxs-lookup"><span data-stu-id="80974-108">Change the name of the interface method.</span></span>  
  
- <span data-ttu-id="80974-109">重新定義包含類型，讓它不是繼承自介面。</span><span class="sxs-lookup"><span data-stu-id="80974-109">Redefine the containing type so that it does not inherit from the interface.</span></span>  
  
## <a name="example"></a><span data-ttu-id="80974-110">範例</span><span class="sxs-lookup"><span data-stu-id="80974-110">Example</span></span>  
 <span data-ttu-id="80974-111">下列程式碼會產生 CS0736，因為 `Program.testMethod` 宣告為靜態：</span><span class="sxs-lookup"><span data-stu-id="80974-111">The following code generates CS0736 because `Program.testMethod` is declared as static:</span></span>  
  
```csharp  
// cs0736.cs  
namespace CS0736  
{
  
    interface ITest  
    {  
        int testMethod(int x);  
    }  
  
    class Program : ITest // CS0736  
    {  
        public static int testMethod(int x) { return 0; }  
        // Try the following line instead.  
        // public int testMethod(int x) { return 0; }  
        public static void Main() { }  
    }
}  
```
