# <a name="compiler-warning-level-2-cs0279"></a><span data-ttu-id="2ea61-102">編譯器警告 (層級 2) CS0279</span><span class="sxs-lookup"><span data-stu-id="2ea61-102">Compiler Warning (level 2) CS0279</span></span>
<span data-ttu-id="2ea61-103">'類型名稱' 未實作 '模式名稱' 模式，</span><span class="sxs-lookup"><span data-stu-id="2ea61-103">'type name' does not implement the 'pattern name' pattern.</span></span> <span data-ttu-id="2ea61-104">因為 '方法名稱' 為靜態或並非公用。</span><span class="sxs-lookup"><span data-stu-id="2ea61-104">'method name' is either static or not public.</span></span>  
  
 <span data-ttu-id="2ea61-105">C# 中有幾個依賴已定義模式的陳述式，例如 `foreach` 和 `using`。</span><span class="sxs-lookup"><span data-stu-id="2ea61-105">There are several statements in C# that rely on defined patterns, such as `foreach` and `using`.</span></span> <span data-ttu-id="2ea61-106">例如， `foreach` 依賴實作可列舉模式的集合類別。</span><span class="sxs-lookup"><span data-stu-id="2ea61-106">For example, `foreach` relies on the collection class implementing the enumerable pattern.</span></span> <span data-ttu-id="2ea61-107">當編譯器由於方法已宣告為 `static` 或不是 `public`而無法進行比對時，就會發生這個錯誤</span><span class="sxs-lookup"><span data-stu-id="2ea61-107">This error occurs when the compiler is unable to make the match due to a method being declared `static` or not `public`.</span></span> <span data-ttu-id="2ea61-108">模式中的方法必須是類別的執行個體，而且必須是公用。</span><span class="sxs-lookup"><span data-stu-id="2ea61-108">Methods in patterns are required to be instances of classes, and to be public.</span></span>  
  
## <a name="example"></a><span data-ttu-id="2ea61-109">範例</span><span class="sxs-lookup"><span data-stu-id="2ea61-109">Example</span></span>  
 <span data-ttu-id="2ea61-110">下列範例會產生 CS0279：</span><span class="sxs-lookup"><span data-stu-id="2ea61-110">The following example generates CS0279:</span></span>  
  
```csharp  
// CS0279.cs  
  
using System;  
using System.Collections;  
  
public class myTest : IEnumerable  
{  
    IEnumerator IEnumerable.GetEnumerator()  
    {  
        yield return 0;  
    }  
  
    internal IEnumerator GetEnumerator()  
    {  
        yield return 0;  
    }  
  
    public static void Main()  
    {  
        foreach (int i in new myTest()) {}  // CS0279  
    }  
}  
```
