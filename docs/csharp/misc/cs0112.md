# <a name="compiler-error-cs0112"></a><span data-ttu-id="2d272-102">編譯器錯誤 CS0112</span><span class="sxs-lookup"><span data-stu-id="2d272-102">Compiler Error CS0112</span></span>
<span data-ttu-id="2d272-103">靜態成員 'function' 不能標記為 override、virtual 或 abstract</span><span class="sxs-lookup"><span data-stu-id="2d272-103">A static member 'function' cannot be marked as override, virtual or abstract</span></span>  
  
 <span data-ttu-id="2d272-104">如果方法宣告使用 `override`、 **virtual**或 **abstract** 關鍵字，就不能同時使用 **static** 關鍵字。</span><span class="sxs-lookup"><span data-stu-id="2d272-104">Any method declaration that uses the `override`, **virtual**, or **abstract** keyword cannot also use the **static** keyword.</span></span>  
  
 <span data-ttu-id="2d272-105">如需詳細資訊，請參閱[方法](../programming-guide/classes-and-structs/methods.md)。</span><span class="sxs-lookup"><span data-stu-id="2d272-105">For more information, see [Methods](../programming-guide/classes-and-structs/methods.md).</span></span>  
  
 <span data-ttu-id="2d272-106">下列範例會產生 CS0112：</span><span class="sxs-lookup"><span data-stu-id="2d272-106">The following sample generates CS0112:</span></span>  
  
```csharp  
// CS0112.cs  
namespace MyNamespace  
{  
   abstract public class MyClass  
   {  
      public abstract void Foo();  
   }  
   public class MyClass2 : MyClass  
   {  
      override public static void Foo()   // CS0112, remove static keyword  
      {  
      }  
      public static int Main()  
      {  
         return 0;  
      }  
   }  
}  
```
