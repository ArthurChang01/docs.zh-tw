# <a name="compiler-error-cs0104"></a><span data-ttu-id="8e792-102">編譯器錯誤 CS0104</span><span class="sxs-lookup"><span data-stu-id="8e792-102">Compiler Error CS0104</span></span>
<span data-ttu-id="8e792-103">'reference' 是 'identifier' 和 'identifier' 之間模稜兩可的參考</span><span class="sxs-lookup"><span data-stu-id="8e792-103">'reference' is an ambiguous reference between 'identifier' and 'identifier'</span></span>  
  
 <span data-ttu-id="8e792-104">您的程式包含兩個命名空間的 [using](../language-reference/keywords/using.md) 指示詞，且您的程式碼參考在兩個命名空間中都出現的名稱。</span><span class="sxs-lookup"><span data-stu-id="8e792-104">Your program contains [using](../language-reference/keywords/using.md) directives for two namespaces and your code references a name that appears in both namespaces.</span></span>  
  
 <span data-ttu-id="8e792-105">下列範例會產生 CS0104：</span><span class="sxs-lookup"><span data-stu-id="8e792-105">The following sample generates CS0104:</span></span>  
  
```csharp  
// CS0104.cs  
using x;  
using y;  
  
namespace x  
{  
   public class Test  
   {  
   }  
}  
  
namespace y  
{  
   public class Test  
   {  
   }  
}  
  
public class a  
{  
   public static void Main()  
   {  
      Test test = new Test();   // CS0104, is Test in x or y namespace?  
      // try the following line instead  
      // y.Test test = new y.Test();  
   }  
}  
```
