# 編譯器錯誤 CS0148
委派 'delegate' 沒有有效的建構函式  
  
 您已匯入和使用以其他編譯器所建立的受管理程式 \(使用 .NET Framework Common Language Runtime\)。 該編譯器允許格式不正確的 [委派](../../csharp/language-reference/keywords/delegate.md)建構函式。 如需詳細資訊，請參閱[委派](../../csharp/programming-guide/delegates/index.md)。
