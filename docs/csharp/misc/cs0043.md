# <a name="compiler-error-cs0043"></a>編譯器錯誤 CS0043

PDB 檔 'file' 有不正確或過時的格式。 請將其刪除或重新建置。

 刪除這個編譯的 .pdb 檔案，並重新編譯。

> [!NOTE]
> Roslyn 中已不再使用此編譯器錯誤。
