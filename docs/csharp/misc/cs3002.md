# <a name="compiler-warning-level-1-cs3002"></a>編譯器警告 (層級 1) CS3002
'method' 的傳回類型不符合 CLS 標準  
  
 [公用](../../csharp/language-reference/keywords/public.md)、[受保護](../../csharp/language-reference/keywords/protected.md) 或 `protected internal` 方法必須傳回其類型符合 Common Language Specification (CLS) 規範的值。 如需 CLS 合規性的詳細資訊，請參閱[語言獨立性以及與語言無關的元件](../../standard/language-independence.md)。
  
## <a name="example"></a>範例  
 下列範例會產生 CS3002：  
  
```csharp  
// CS3002.cs  
  
[assembly:System.CLSCompliant(true)]  
public class a  
{  
    public ushort bad()   // CS3002, public method  
    {  
        ushort a;  
        a = ushort.MaxValue;  
        return a;  
    }  
  
    private ushort OK()   // OK, private method  
    {  
        ushort a;  
        a = ushort.MaxValue;  
        return a;  
    }  
  
    public static void Main()  
    {  
    }  
}  
```
