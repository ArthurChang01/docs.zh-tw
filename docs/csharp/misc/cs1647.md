# <a name="compiler-error-cs1647"></a>編譯器錯誤 CS1647
'code' 附近的運算式太長或太複雜，造成編譯困難  
  
 處理您的程式碼時，編譯器中發生堆疊溢位。 若要解決這個錯誤，請簡化您的程式碼。 如果您的程式碼是有效的取得協助[開發人員社群](https://developercommunity.visualstudio.com/spaces/61/index.html)站台或[StackOverflow](https://stackoverflow.com/)。
