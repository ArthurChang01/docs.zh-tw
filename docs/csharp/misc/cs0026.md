 <span data-ttu-id="b997b-104">[this](../../csharp/language-reference/keywords/this.md) 關鍵字參考的物件是某個類型的執行個體。</span><span class="sxs-lookup"><span data-stu-id="b997b-104">The [this](../../csharp/language-reference/keywords/this.md) keyword refers to an object, which is an instance of a type.</span></span> <span data-ttu-id="b997b-105">由於靜態方法與任何包含類別執行個體皆無關，因此 "this" 關鍵字是無意義且不被允許的。</span><span class="sxs-lookup"><span data-stu-id="b997b-105">Since static methods are independent of any instance of the containing class, the "this" keyword is meaningless and is therefore not allowed.</span></span> <span data-ttu-id="b997b-106">如需詳細資訊，請參閱[靜態類別和靜態類別成員](../../csharp/programming-guide/classes-and-structs/static-classes-and-static-class-members.md)和[物件](../../csharp/programming-guide/classes-and-structs/objects.md)。</span><span class="sxs-lookup"><span data-stu-id="b997b-106">For more information, see [Static Classes and Static Class Members](../../csharp/programming-guide/classes-and-structs/static-classes-and-static-class-members.md) and [Objects](../../csharp/programming-guide/classes-and-structs/objects.md).</span></span>  
  
## <a name="example"></a><span data-ttu-id="b997b-107">範例</span><span class="sxs-lookup"><span data-stu-id="b997b-107">Example</span></span>  
 <span data-ttu-id="b997b-108">下列範例會產生 CS0026：</span><span class="sxs-lookup"><span data-stu-id="b997b-108">The following example generates CS0026:</span></span>  
  
```csharp  
// CS0026.cs  
public class A  
{  
   public static int i = 0;  
  
   public static void Main()  
   {  
// CS0026  
      this.i = this.i + 1;     
      // Try the following line instead:  
      // i = i + 1;  
   }  
}  
```
