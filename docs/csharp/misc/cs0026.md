## <a name="example"></a><span data-ttu-id="15e7b-107">範例</span><span class="sxs-lookup"><span data-stu-id="15e7b-107">Example</span></span>  
 <span data-ttu-id="15e7b-108">下列範例會產生 CS0026：</span><span class="sxs-lookup"><span data-stu-id="15e7b-108">The following example generates CS0026:</span></span>  
  
```csharp  
// CS0026.cs  
public class A  
{  
   public static int i = 0;  
  
   public static void Main()  
   {

      this.i = this.i + 1;
      // Try the following line instead:  
      // i = i + 1;  
   }  
}  
```
