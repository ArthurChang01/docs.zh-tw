<span data-ttu-id="4df2d-103">類型 'type name' 必須是參考類型，才能在泛型類型或方法 'identifier of generic' 中作為參數 'parameter name' 使用</span><span class="sxs-lookup"><span data-stu-id="4df2d-103">The type 'type name' must be a reference type in order to use it as parameter 'parameter name' in the generic type or method 'identifier of generic'</span></span>
