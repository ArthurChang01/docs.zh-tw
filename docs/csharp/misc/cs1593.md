# <a name="compiler-error-cs1593"></a><span data-ttu-id="a0b4f-102">編譯器錯誤 CS1593</span><span class="sxs-lookup"><span data-stu-id="a0b4f-102">Compiler Error CS1593</span></span>
<span data-ttu-id="a0b4f-103">委派 'del' 不採用 'number' 引數</span><span class="sxs-lookup"><span data-stu-id="a0b4f-103">Delegate 'del' does not take 'number' arguments</span></span>  
  
 <span data-ttu-id="a0b4f-104">傳遞至 [委派](../language-reference/builtin-types/reference-types.md) 引動過程的引數數目和委派宣告的參數數目不一致。</span><span class="sxs-lookup"><span data-stu-id="a0b4f-104">The number of arguments passed to a [delegate](../language-reference/builtin-types/reference-types.md) invocation does not agree with the number of parameters in the delegate declaration.</span></span>  
  
 <span data-ttu-id="a0b4f-105">下例會產生 CS1593：</span><span class="sxs-lookup"><span data-stu-id="a0b4f-105">The following sample generates CS1593:</span></span>  
  
```csharp  
// CS1593.cs  
using System;  
delegate string func(int i);   // declare delegate  
  
class a  
{  
   public static void Main()  
   {  
      func dt = new func(z);  
      x(dt);  
   }  
  
   public static string z(int j)  
   {  
      Console.WriteLine(j);  
      return j.ToString();  
   }  
  
   public static void x(func hello)  
   {  
      hello(8, 9);   // CS1593  
      // try the following line instead  
      // hello(8);  
   }  
}  
```
