# <a name="compiler-error-cs1585"></a>編譯器錯誤 CS1585
成員修飾詞 'keyword' 必須在成員類型和名稱之前  
  
 方法簽章中的存取規範未出現在正確的位置。  
  
 下列範例會產生 CS1585：  
  
```csharp  
// CS1585.cs  
public class Class1  
{  
   public void static Main(string[] args)   // CS1585  
   // try the following line instead  
   // public static void Main(string[] args)  
   {  
   }  
}  
```
