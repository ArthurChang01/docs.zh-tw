# <a name="compiler-error-cs1618"></a><span data-ttu-id="9a371-102">編譯器錯誤 CS1618</span><span class="sxs-lookup"><span data-stu-id="9a371-102">Compiler Error CS1618</span></span>
<span data-ttu-id="9a371-103">無法以 'method' 建立委派，因為它有 Conditional 屬性</span><span class="sxs-lookup"><span data-stu-id="9a371-103">Cannot create delegate with 'method' because it has a Conditional attribute</span></span>  
  
 <span data-ttu-id="9a371-104">因為有些組建沒有這個方法，所以您無法使用條件式方法來建立委派。</span><span class="sxs-lookup"><span data-stu-id="9a371-104">You cannot create a delegate with a conditional method because the method might not exist in some builds.</span></span>  
  
 <span data-ttu-id="9a371-105">下列範例會產生 CS1618：</span><span class="sxs-lookup"><span data-stu-id="9a371-105">The following sample generates CS1618:</span></span>  
  
```csharp  
// CS1618.cs  
using System;  
using System.Diagnostics;  
  
delegate void del();  
  
class MakeAnError {  
   public static void Main() {  
      del d = new del(ConditionalMethod);   // CS1618  
      // Invalid because on builds where DEBUG is not set,   
      // there will be no "ConditionalMethod".  
   }  
   // To fix the error, remove the next line:  
   [Conditional("DEBUG")]  
   public static void ConditionalMethod()   
   {  
      Console.WriteLine("Do something only in debug");  
   }  
}  
```
