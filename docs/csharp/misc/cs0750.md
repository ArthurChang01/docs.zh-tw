# <a name="compiler-error-cs0750"></a><span data-ttu-id="51032-102">編譯器錯誤 CS0750</span><span class="sxs-lookup"><span data-stu-id="51032-102">Compiler Error CS0750</span></span>
<span data-ttu-id="51032-103">部分方法不可有存取修飾詞或 virtual、abstract、override、new、sealed 或 extern 修飾詞。</span><span class="sxs-lookup"><span data-stu-id="51032-103">A partial method cannot have access modifiers or the virtual, abstract, override, new, sealed, or extern modifiers.</span></span>  
  
 <span data-ttu-id="51032-104">由於其特殊行為，部分方法可以接受的修飾詞有限。</span><span class="sxs-lookup"><span data-stu-id="51032-104">Because of their special behavior, partial methods are subject to restrictions as to the modifiers they can accept.</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="51032-105">更正這個錯誤</span><span class="sxs-lookup"><span data-stu-id="51032-105">To correct this error</span></span>  
  
1. <span data-ttu-id="51032-106">從方法宣告中移除未授權的修飾詞。</span><span class="sxs-lookup"><span data-stu-id="51032-106">Remove the unauthorized modifier from the method declaration.</span></span>  
  
## <a name="example"></a><span data-ttu-id="51032-107">範例</span><span class="sxs-lookup"><span data-stu-id="51032-107">Example</span></span>  
 <span data-ttu-id="51032-108">下列範例會產生 CS0750：</span><span class="sxs-lookup"><span data-stu-id="51032-108">The following example generates CS0750:</span></span>  
  
```csharp  
// cs0750.cs  
using System;  
  
public class Base  
{  
    protected virtual void PartG()  
    {  
    }  
  
    protected void PartH()  
    {  
    }  
    protected virtual void PartI()  
    {  
    }  
}  
  
public partial class C:Base  
{  
    // All these partial method declarations  
    // will generate CS0750.  
    public partial void PartA();  
    private partial void PartB();  
    protected partial void PartC();  
    internal partial void PartD();  
    virtual partial void PartE();  
    abstract partial void PartF();  
    override partial void PartG();  
    new partial void PartH();  
    sealed override partial void PartI();  
    extern partial void PartJ();  
  
    public static int Main()  
    {  
        return 1;  
    }  
}  
```
