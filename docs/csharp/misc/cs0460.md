# <a name="compiler-error-cs0460"></a><span data-ttu-id="7c6f6-102">編譯器錯誤 CS0460</span><span class="sxs-lookup"><span data-stu-id="7c6f6-102">Compiler Error CS0460</span></span>
<span data-ttu-id="7c6f6-103">覆寫及明確介面實作方法的條件約束，繼承自基底方法，所以無法直接指定。</span><span class="sxs-lookup"><span data-stu-id="7c6f6-103">Constraints for override and explicit interface implementation methods are inherited from the base method, so they cannot be specified directly</span></span>  
  
 <span data-ttu-id="7c6f6-104">當屬於衍生類別的泛型方法覆寫基底類別中的方法時，您不能指定受覆寫的方法上的條件約束。</span><span class="sxs-lookup"><span data-stu-id="7c6f6-104">When a generic method that is part of a derived class overrides a method in the base class, you may not specify constraints on the overridden method.</span></span> <span data-ttu-id="7c6f6-105">衍生類別中的覆寫方法會從基底類別中的方法繼承其條件約束。</span><span class="sxs-lookup"><span data-stu-id="7c6f6-105">The override method in the derived class inherits its constraints from the method in the base class.</span></span>  
  
## <a name="example"></a><span data-ttu-id="7c6f6-106">範例</span><span class="sxs-lookup"><span data-stu-id="7c6f6-106">Example</span></span>  
 <span data-ttu-id="7c6f6-107">下列範例會產生 CS0460：</span><span class="sxs-lookup"><span data-stu-id="7c6f6-107">The following sample generates CS0460.</span></span>  
  
```  
// CS0460.cs  
// compile with: /target:library  
class BaseClass   
{  
   BaseClass() { }  
}  
  
interface I  
{  
   void F1<T>() where T : BaseClass;  
   void F2<T>() where T : struct;  
   void F3<T>() where T : BaseClass;  
}  
  
class ExpImpl : I  
{  
   void I.F1<T>() where T : BaseClass {}   // CS0460  
   void I.F2<T>() where T : class {}  // CS0460  
}  
```
