<span data-ttu-id="d88de-103">參數 'number' 不可以 'keyword' 關鍵字宣告</span><span class="sxs-lookup"><span data-stu-id="d88de-103">Parameter 'number' should not be declared with the 'keyword' keyword</span></span>  
  
 <span data-ttu-id="d88de-104">當匿名方法中的參數類型修飾詞不符合轉換方法的目標委派宣告中所使用的修飾詞時，就會發生這個錯誤。</span><span class="sxs-lookup"><span data-stu-id="d88de-104">This error occurs when the parameter type modifier in an anonymous method does not match that used in the declaration of the delegate, to which you are casting the method.</span></span>  
  
## <a name="example"></a><span data-ttu-id="d88de-105">範例</span><span class="sxs-lookup"><span data-stu-id="d88de-105">Example</span></span>  
 <span data-ttu-id="d88de-106">下列範例會產生 CS1677：</span><span class="sxs-lookup"><span data-stu-id="d88de-106">The following sample generates CS1677:</span></span>  
  
```csharp  
// CS1677.cs  
delegate void D(int i);  
class Errors  
{  
    static void Main()   
    {  
        D d = delegate(out int i) { };   // CS1677  
        // To resolve, use the following line instead:  
        // D d = delegate(int i) { };  
  
        D d = delegate(ref int j){}; // CS1677  
        // To resolve, use the following line instead:  
        // D d = delegate(int j){};  
    }  
}  
```
