# <a name="compiler-error-cs0662"></a><span data-ttu-id="bfe97-102">編譯器錯誤 CS0662</span><span class="sxs-lookup"><span data-stu-id="bfe97-102">Compiler Error CS0662</span></span>

<span data-ttu-id="bfe97-103">'method' 不能在 ref 參數上只指定 Out 屬性。</span><span class="sxs-lookup"><span data-stu-id="bfe97-103">'method' cannot specify only Out attribute on a ref parameter.</span></span> <span data-ttu-id="bfe97-104">請同時使用 In 與 Out 屬性，或都不使用。</span><span class="sxs-lookup"><span data-stu-id="bfe97-104">Use both In and Out attributes, or neither.</span></span>  
  
 <span data-ttu-id="bfe97-105">介面方法所包含的參數，使用了僅指定 [Out](../language-reference/keywords/ref.md) 屬性的 [ref](xref:System.Runtime.InteropServices.OutAttribute) 。</span><span class="sxs-lookup"><span data-stu-id="bfe97-105">An interface method has a parameter that uses [ref](../language-reference/keywords/ref.md) with just the [Out](xref:System.Runtime.InteropServices.OutAttribute) attribute.</span></span> <span data-ttu-id="bfe97-106">使用 `ref` Out **屬性的** 參數也必須使用 [In](xref:System.Runtime.InteropServices.InAttribute) 屬性。</span><span class="sxs-lookup"><span data-stu-id="bfe97-106">A `ref` parameter that uses the **Out** attribute must also use the [In](xref:System.Runtime.InteropServices.InAttribute) attribute.</span></span>  
  
 <span data-ttu-id="bfe97-107">下列範例會產生 CS0662：</span><span class="sxs-lookup"><span data-stu-id="bfe97-107">The following sample generates CS0662:</span></span>  
  
```csharp
// CS0662.cs  
using System.Runtime.InteropServices;  
  
interface I  
{  
   void method([Out] ref int i);   // CS0662  
   // try one of the following lines instead  
   // void method(ref int i);  
   // void method([Out, In]ref int i);  
}  
  
class test  
{  
   public static void Main()  
   {  
   }  
}  
```
