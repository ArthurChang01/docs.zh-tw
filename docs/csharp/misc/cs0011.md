## <a name="example"></a><span data-ttu-id="6192e-107">範例</span><span class="sxs-lookup"><span data-stu-id="6192e-107">Example</span></span>

```csharp  
{  
   public class B { }  
}  
```  
  
## <a name="example"></a><span data-ttu-id="6192e-108">範例</span><span class="sxs-lookup"><span data-stu-id="6192e-108">Example</span></span>  
 <span data-ttu-id="6192e-109">第二個檔案會建立可定義類別 `C` 的 DLL，這個類別衍生自已在前一個範例中建立的類別 `B` 。</span><span class="sxs-lookup"><span data-stu-id="6192e-109">The second file creates a DLL that defines a class `C` that is derived from the class `B` that was created in the previous example.</span></span>

```csharp  
// compile with: /target:library /reference:CS0011_1.dll  
// post-build command: del /f CS0011_1.dll

```  
  
## <a name="example"></a><span data-ttu-id="6192e-110">範例</span><span class="sxs-lookup"><span data-stu-id="6192e-110">Example</span></span>

