## <a name="example"></a><span data-ttu-id="90151-107">範例</span><span class="sxs-lookup"><span data-stu-id="90151-107">Example</span></span>

```  
}  
```  
  
## <a name="example"></a><span data-ttu-id="90151-108">範例</span><span class="sxs-lookup"><span data-stu-id="90151-108">Example</span></span>  
 <span data-ttu-id="90151-109">第二個檔案會建立可定義類別 `C` 的 DLL，這個類別衍生自已在前一個範例中建立的類別 `B` 。</span><span class="sxs-lookup"><span data-stu-id="90151-109">The second file creates a DLL that defines a class `C` that is derived from the class `B` that was created in the previous example.</span></span>
