# <a name="compiler-error-cs0110"></a><span data-ttu-id="0fd2e-102">編譯器錯誤 CS0110</span><span class="sxs-lookup"><span data-stu-id="0fd2e-102">Compiler Error CS0110</span></span>
<span data-ttu-id="0fd2e-103">'const declaration' 常數值的評估牽涉循環定義</span><span class="sxs-lookup"><span data-stu-id="0fd2e-103">The evaluation of the constant value for 'const declaration' involves a circular definition</span></span>  
  
 <span data-ttu-id="0fd2e-104">[const](../../csharp/language-reference/keywords/const.md) 變數 (`a`) 的宣告不能參考也參考 (`b`) 的另一個 const 變數 (`a`)。</span><span class="sxs-lookup"><span data-stu-id="0fd2e-104">The declaration of a [const](../../csharp/language-reference/keywords/const.md) variable (`a`) cannot reference another const variable (`b`) that also references (`a`).</span></span>  
  
 <span data-ttu-id="0fd2e-105">下列範例會產生 CS0110：</span><span class="sxs-lookup"><span data-stu-id="0fd2e-105">The following sample generates CS0110:</span></span>  
  
```  
// CS0110.cs  
namespace MyNamespace  
{  
   public class A  
   {  
      public static void Main()  
      {  
      }  
   }  
  
   public class B : A  
   {  
      public const int i = c + 1;   // CS0110, c already references i  
      public const int c = i + 1;  
      // the following line would be OK  
      // public const int c = 10;  
   }  
}  
```
