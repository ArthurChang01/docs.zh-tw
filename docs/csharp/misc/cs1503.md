# <a name="compiler-error-cs1503"></a>編譯器錯誤 CS1503
引數 'number' 無法從 TypeA 轉換為 TypeB  
  
 方法中有一個引數類型不符合具現化類別時所傳遞的類型。 此錯誤通常會與 CS1502 一起出現。 請參閱 [CS1502](../language-reference/compiler-messages/cs1502.md) 以了解如何解決此錯誤的討論。
