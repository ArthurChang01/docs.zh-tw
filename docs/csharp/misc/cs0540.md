# <a name="compiler-error-cs0540"></a><span data-ttu-id="99e3c-102">編譯器錯誤 CS0540</span><span class="sxs-lookup"><span data-stu-id="99e3c-102">Compiler Error CS0540</span></span>
<span data-ttu-id="99e3c-103">'介面成員': 包含類型未實作介面 'interface'</span><span class="sxs-lookup"><span data-stu-id="99e3c-103">'interface member' : containing type does not implement interface 'interface'</span></span>  
  
 <span data-ttu-id="99e3c-104">您嘗試在 [類別](../language-reference/keywords/class.md) 中實作非衍生自 [介面](../language-reference/keywords/interface.md)的介面成員。</span><span class="sxs-lookup"><span data-stu-id="99e3c-104">You attempted to implement an interface member in a [class](../language-reference/keywords/class.md) that does not derive from the [interface](../language-reference/keywords/interface.md).</span></span> <span data-ttu-id="99e3c-105">您應該刪除介面成員的實作，或將介面加入類別的基底類別清單中。</span><span class="sxs-lookup"><span data-stu-id="99e3c-105">You should either delete the implementation of the interface member or add the interface to the base-class list of the class.</span></span>  
  
## <a name="example"></a><span data-ttu-id="99e3c-106">範例</span><span class="sxs-lookup"><span data-stu-id="99e3c-106">Example</span></span>  
 <span data-ttu-id="99e3c-107">下列範例會產生 CS0540。</span><span class="sxs-lookup"><span data-stu-id="99e3c-107">The following sample generates CS0540.</span></span>  
  
```csharp  
// CS0540.cs  
interface I  
{  
   void m();  
}  
  
public class Clx  
{  
   void I.m() {}   // CS0540  
}  
  
// OK  
public class Cly : I  
{  
   void I.m() {}  
   public static void Main() {}  
}  
```  
  
## <a name="example"></a><span data-ttu-id="99e3c-108">範例</span><span class="sxs-lookup"><span data-stu-id="99e3c-108">Example</span></span>  
 <span data-ttu-id="99e3c-109">下列範例會產生 CS0540。</span><span class="sxs-lookup"><span data-stu-id="99e3c-109">The following sample generates CS0540.</span></span>

   void IDisposable.Dispose() {}   // CS0540

   void IDisposable.Dispose() {}
