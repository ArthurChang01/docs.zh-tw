# <a name="compiler-error-cs0003"></a>編譯器錯誤 CS0003
記憶體不足  
  
 編譯器所配置的虛擬記憶體不足，無法完成編譯。 關閉所有不必要的應用程式，然後重新編譯。  
  
 您也可能想要增加分頁檔大小，並確定您具有可用的磁碟空間。  
  
 此錯誤也可能會出現的結果具有不相符的版本為[!INCLUDE[winsdklong](../../csharp/language-reference/compiler-options/includes/winsdklong_md.md)]和 C# 編譯器或一個或多個損毀的檔案支援 C# 編譯器; 重新安裝 Visual Studio。
