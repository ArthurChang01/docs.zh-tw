# <a name="compiler-error-cs0003"></a>編譯器錯誤 CS0003
記憶體不足  
  
 編譯器所配置的虛擬記憶體不足，無法完成編譯。 關閉所有不必要的應用程式，然後重新編譯。  
  
 您也可能想要增加分頁檔大小，並確定您具有可用的磁碟空間。  
  
 下列項目的版本不符時，也可能會顯示這個錯誤： [!INCLUDE[winsdklong](~/includes/winsdklong-md.md)] 和 C# 編譯器，或是支援 C# 編譯器的一個或多個損毀檔案；請重新安裝 Visual Studio。
