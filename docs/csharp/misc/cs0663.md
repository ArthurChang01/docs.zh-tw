# <a name="compiler-error-cs0663"></a><span data-ttu-id="c14a7-102">編譯器錯誤 CS0663</span><span class="sxs-lookup"><span data-stu-id="c14a7-102">Compiler Error CS0663</span></span>

 <span data-ttu-id="c14a7-104">不允許只有在參數上使用不同 [ref](../../csharp/language-reference/keywords/ref.md) 和 [out](../../csharp/language-reference/keywords/out.md) 的方法。</span><span class="sxs-lookup"><span data-stu-id="c14a7-104">Methods that differ only on their use of [ref](../../csharp/language-reference/keywords/ref.md) and [out](../../csharp/language-reference/keywords/out.md) on a parameter are not allowed.</span></span>  
  
 <span data-ttu-id="c14a7-105">下列範例會產生 CS0663：</span><span class="sxs-lookup"><span data-stu-id="c14a7-105">The following sample generates CS0663:</span></span>  
  
```  
// CS0663.cs  
class TestClass  
{  
   public static void Main()  
   {  
   }  
  
   public void Test(ref int i)  
   {  
   }  
  
   public void Test(out int i)   // CS0663  
   {  
   }  
}  
```
