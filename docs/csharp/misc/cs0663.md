# <a name="compiler-error-cs0663"></a><span data-ttu-id="17296-102">編譯器錯誤 CS0663</span><span class="sxs-lookup"><span data-stu-id="17296-102">Compiler Error CS0663</span></span>
<span data-ttu-id="17296-103">不可定義只有 ref 和 out 不同的多載方法。</span><span class="sxs-lookup"><span data-stu-id="17296-103">Cannot define overloaded methods that differ only on ref and out.</span></span>  
  
 <span data-ttu-id="17296-104">只有在其使用不同的方法[中](../../csharp/language-reference/keywords/in-parameter-modifier.md)， [ref](../../csharp/language-reference/keywords/ref.md)並[出](../../csharp/language-reference/keywords/out-parameter-modifier.md)參數上不允許。</span><span class="sxs-lookup"><span data-stu-id="17296-104">Methods that differ only on their use of [in](../../csharp/language-reference/keywords/in-parameter-modifier.md), [ref](../../csharp/language-reference/keywords/ref.md) and [out](../../csharp/language-reference/keywords/out-parameter-modifier.md) on a parameter are not allowed.</span></span>  
  
 <span data-ttu-id="17296-105">下列範例會產生 CS0663：</span><span class="sxs-lookup"><span data-stu-id="17296-105">The following sample generates CS0663:</span></span>  
  
```csharp  
// CS0663.cs  
class TestClass  
{  
   public static void Main()  
   {  
   }  
  
   public void Test(ref int i)  
   {  
   }  
  
   public void Test(out int i)   // CS0663  
   {  
   }  
}  
```
