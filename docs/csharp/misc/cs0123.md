 <span data-ttu-id="f92d1-108">下列範例會產生 CS0123。</span><span class="sxs-lookup"><span data-stu-id="f92d1-108">The following sample generates CS0123.</span></span>  
  
```  
// CS0123.cs  
delegate void D();  
delegate void D2(int i);  
  
public class C  
{  
   public static void f(int i) {}  
  
   public static void Main()  
   {  
      D d = new D(f);   // CS0123  
      D2 d2 = new D2(f);   // OK  
   }  
}  
```
