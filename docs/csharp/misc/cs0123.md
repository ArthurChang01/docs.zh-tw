<span data-ttu-id="42388-103">'method' 沒有任何多載符合委派 'delegate'</span><span class="sxs-lookup"><span data-stu-id="42388-103">No overload for 'method' matches delegate 'delegate'</span></span>

 <span data-ttu-id="42388-108">下列範例會產生 CS0123。</span><span class="sxs-lookup"><span data-stu-id="42388-108">The following sample generates CS0123.</span></span>  
  
```csharp

delegate void D();  
delegate void D2(int i);  
  
public class C  
{  
   public static void f(int i) {}  
  
   public static void Main()  
   {  
      D d = new D(f);   // CS0123  
      D2 d2 = new D2(f);   // OK  
   }  
}  
```
