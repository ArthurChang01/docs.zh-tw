# <a name="compiler-error-cs1004"></a>編譯器錯誤 CS1004
重複的 'modifier' 修飾詞  
  
 偵測到重複的修飾詞，例如 **access** 修飾詞。  
  
 下列範例會產生 CS1004：  
  
```csharp  
// CS1004.cs  
public class clx  
{  
   public public static void Main()   // CS1004, two public keywords  
   {  
   }  
}  
```
