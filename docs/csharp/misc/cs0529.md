<span data-ttu-id="cbecd-103">繼承的介面 'interface1' 造成 'interface2' 介面階層架構中出現循環</span><span class="sxs-lookup"><span data-stu-id="cbecd-103">Inherited interface 'interface1' causes a cycle in the interface hierarchy of 'interface2'</span></span>  
  
 <span data-ttu-id="cbecd-104">[介面](../../csharp/language-reference/keywords/interface.md) 的繼承清單包含本身的直接或間接參考。</span><span class="sxs-lookup"><span data-stu-id="cbecd-104">The inheritance list for an [interface](../../csharp/language-reference/keywords/interface.md) includes a direct or indirect reference to itself.</span></span> <span data-ttu-id="cbecd-105">介面無法繼承自本身。</span><span class="sxs-lookup"><span data-stu-id="cbecd-105">An interface cannot inherit from itself.</span></span>  
  
 <span data-ttu-id="cbecd-106">下列範例會產生 CS0529：</span><span class="sxs-lookup"><span data-stu-id="cbecd-106">The following sample generates CS0529:</span></span>  
  
```csharp  
// CS0529.cs  
namespace x  
{  
   public interface a  
   {  
   }  
  
   public interface b : a, c  
   {  
   }  
  
   public interface c : b   // CS0529, b inherits from c  
   {  
   }  
}  
```
