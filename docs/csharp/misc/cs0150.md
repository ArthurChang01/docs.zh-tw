# <a name="compiler-error-cs0150"></a><span data-ttu-id="699e5-102">編譯器錯誤 CS0150</span><span class="sxs-lookup"><span data-stu-id="699e5-102">Compiler Error CS0150</span></span>
<span data-ttu-id="699e5-103">必須是常數值</span><span class="sxs-lookup"><span data-stu-id="699e5-103">A constant value is expected</span></span>  
  
 <span data-ttu-id="699e5-104">找到必須有常數的變數。</span><span class="sxs-lookup"><span data-stu-id="699e5-104">A variable was found where a constant was expected.</span></span> <span data-ttu-id="699e5-105">如需詳細資訊，請參閱 [switch](../../csharp/language-reference/keywords/switch.md)。</span><span class="sxs-lookup"><span data-stu-id="699e5-105">For more information, see [switch](../../csharp/language-reference/keywords/switch.md).</span></span>  
  
 <span data-ttu-id="699e5-106">下列範例會產生 CS0150：</span><span class="sxs-lookup"><span data-stu-id="699e5-106">The following sample generates CS0150:</span></span>  
  
```csharp  
// CS0150.cs  
namespace MyNamespace  
{  
   public class MyClass  
   {  
      public static void Main()  
      {  
         int i = 0;  
         int j = 0;  
  
         switch(i)  
         {  
            case j:   // CS0150, j is a variable int, not a constant int  
            // try the following line instead  
            // case 1:  
         }  
      }  
   }  
}  
```  
  
 <span data-ttu-id="699e5-107">以變數值指定並以陣列初始設定式初始化陣列大小時，也會產生這個錯誤。</span><span class="sxs-lookup"><span data-stu-id="699e5-107">This error is also produced when an array size is specified with a variable value and initialized with an array initializer.</span></span> <span data-ttu-id="699e5-108">若要移除此錯誤，請以一或多個個別陳述式初始化陣列。</span><span class="sxs-lookup"><span data-stu-id="699e5-108">To remove the error, initialize the array in a separate statement or statements.</span></span>  
  
```csharp  
// CS0150.cs  
    namespace MyNamespace  
    {  
        public class MyClass  
        {  
            public static void Main()  
            {  
                int size = 2;  
                double[] nums = new double[size] { 46.9, 89.4 }; //CS0150  
                // Try the following lines instead  
                // double[] nums = new double[size];  
                // nums[0] = 46.9;   
                // nums[1] = 89.4;  
            }  
        }  
  
    }  
```
