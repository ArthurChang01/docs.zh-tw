# <a name="compiler-error-cs0569"></a>編譯器錯誤 CS0569
'method2'：因為此語言不支援 'method1'，所以無法覆寫  
  
 當您從以另一種語言撰寫的基底類別衍生，以及當編譯器不支援您嘗試覆寫的方法時，就會發生此錯誤。
