# <a name="compiler-error-cs0442"></a>編譯器錯誤 CS0442
'Property': 抽象屬性不能有私用存取子  
  
 當您使用存取修飾詞 "private" 修改抽象存取子時，會發生這個錯誤。 若要解決，請使用不同的存取修飾詞，或將屬性設為非抽象。  
  
## <a name="example"></a>範例  
 下列範例會產生 CS0442：  
  
```csharp  
// CS0442.cs  
public abstract class MyClass   
{  
    public abstract int AbstractProperty   
    {  
        get;  
        private set;   // CS0442  
        // Try this instead:  
        // set;  
    }  
}  
```
