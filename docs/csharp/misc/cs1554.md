 <span data-ttu-id="b214f-104">使用者定義的 [運算子](../../csharp/language-reference/keywords/operator.md) 傳回類型必須出現在關鍵字運算子之前。</span><span class="sxs-lookup"><span data-stu-id="b214f-104">The return type for a user-defined [operator](../../csharp/language-reference/keywords/operator.md) must appear before the keyword operator.</span></span>
