# <a name="compiler-error-cs1554"></a><span data-ttu-id="26ed2-102">編譯器錯誤 CS1554</span><span class="sxs-lookup"><span data-stu-id="26ed2-102">Compiler Error CS1554</span></span>
<span data-ttu-id="26ed2-103">宣告無效;使用 '\<類型 > 運算子 op (...' 改為</span><span class="sxs-lookup"><span data-stu-id="26ed2-103">Declaration is not valid; use '\<type> operator op (...' instead</span></span>  
  
 <span data-ttu-id="26ed2-104">使用者定義的 [運算子](../../csharp/language-reference/keywords/operator.md) 傳回類型必須出現在關鍵字運算子之前。</span><span class="sxs-lookup"><span data-stu-id="26ed2-104">The return type for a user-defined [operator](../../csharp/language-reference/keywords/operator.md) must appear before the keyword operator.</span></span>  
  
 <span data-ttu-id="26ed2-105">下列範例會產生 CS1554：</span><span class="sxs-lookup"><span data-stu-id="26ed2-105">The following sample generates CS1554:</span></span>  
  
```csharp  
// CS1554.cs  
class MyClass  
{  
   public static operator ++ MyClass (MyClass f)    // CS1554  
   // try the following line instead  
   // public static MyClass operator ++ (MyClass f)  
   {  
      return new MyClass ();  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
