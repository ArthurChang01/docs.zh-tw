# <a name="compiler-error-cs1554"></a><span data-ttu-id="af364-102">編譯器錯誤 CS1554</span><span class="sxs-lookup"><span data-stu-id="af364-102">Compiler Error CS1554</span></span>

<span data-ttu-id="af364-104">多載[運算子](../language-reference/operators/operator-overloading.md)的傳回類型必須出現在`operator`關鍵字之前。</span><span class="sxs-lookup"><span data-stu-id="af364-104">The return type of an [overloaded operator](../language-reference/operators/operator-overloading.md) must appear before the `operator` keyword.</span></span>

```csharp  
// CS1554.cs

{  
   public static operator ++ MyClass (MyClass f)    // CS1554  
   // try the following line instead  
   // public static MyClass operator ++ (MyClass f)  
   {  
      return new MyClass ();  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
