# <a name="compiler-error-cs0544"></a><span data-ttu-id="dd84f-102">編譯器錯誤 CS0544</span><span class="sxs-lookup"><span data-stu-id="dd84f-102">Compiler Error CS0544</span></span>

 <span data-ttu-id="dd84f-104">已嘗試將非屬性資料類型覆寫為 [屬性](../programming-guide/classes-and-structs/properties.md)，但不允許這樣做。</span><span class="sxs-lookup"><span data-stu-id="dd84f-104">An attempt was made to override a nonproperty data type as a [property](../programming-guide/classes-and-structs/properties.md), which is not allowed.</span></span>

```csharp  
// CS0544.cs  
// compile with: /target:library  
public class a  
{  
   public int i;  
}  
  
public class b : a  
{  
   public override int i {   // CS0544  
   // try the following line instead  
   // public new int i {  
      get  
      {  
         return 0;  
      }  
   }  
}  
```
