# <a name="compiler-error-cs0544"></a><span data-ttu-id="beb84-102">編譯器錯誤 CS0544</span><span class="sxs-lookup"><span data-stu-id="beb84-102">Compiler Error CS0544</span></span>
<span data-ttu-id="beb84-103">'property override': 無法覆寫，因為 'non-property' 不是屬性</span><span class="sxs-lookup"><span data-stu-id="beb84-103">'property override': cannot override because 'non-property' is not a property</span></span>  
  
 <span data-ttu-id="beb84-104">已嘗試將非屬性資料類型覆寫為 [屬性](../../csharp/programming-guide/classes-and-structs/properties.md)，但不允許這樣做。</span><span class="sxs-lookup"><span data-stu-id="beb84-104">An attempt was made to override a nonproperty data type as a [property](../../csharp/programming-guide/classes-and-structs/properties.md), which is not allowed.</span></span>  
  
 <span data-ttu-id="beb84-105">下列範例會產生 CS0544：</span><span class="sxs-lookup"><span data-stu-id="beb84-105">The following sample generates CS0544:</span></span>  
  
```csharp  
// CS0544.cs

public class a  
{  
   public int i;  
}  
  
public class b : a  
{  
   public override int i {   // CS0544  
   // try the following line instead  
   // public new int i {  
      get  
      {  
         return 0;  
      }  
   }  
}  
```
