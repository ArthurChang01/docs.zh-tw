# <a name="compiler-error-cs0720"></a>編譯器錯誤 CS0720
'static class': 不能在靜態類別中宣告索引子  
  
 索引子在靜態類別中沒有意義，因為它們只能用於執行個體，不可能建立靜態類型的執行個體。  
  
## <a name="example"></a>範例  
 下列範例會產生 CS0720：  
  
```csharp  
// CS0720.cs  
  
public static class Test  
{  
    public int this[int index]  // CS0720  
    {  
        get { return 1; }  
        set {}  
    }  
  
    static void Main() {}  
}  
```
