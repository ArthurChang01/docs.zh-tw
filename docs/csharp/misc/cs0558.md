# <a name="compiler-error-cs0558"></a><span data-ttu-id="ca726-102">編譯器錯誤 CS0558</span><span class="sxs-lookup"><span data-stu-id="ca726-102">Compiler Error CS0558</span></span>

 <span data-ttu-id="ca726-104">使用者定義的運算子都必須指定 **static** 和 **public** 存取 [修飾詞](../language-reference/keywords/modifiers.md) 。</span><span class="sxs-lookup"><span data-stu-id="ca726-104">Both the **static** and **public** access [modifiers](../language-reference/keywords/modifiers.md) must be specified on user-defined operators.</span></span>

```csharp  
{  
   public class ii  
   {  
      public class iii  
      {  
         static implicit operator int(iii aa)   // CS0558, add public  
         {  
            return 0;  
         }  
      }  
  
      public static void Main()  
      {  
      }  
   }  
}  
```
