 <span data-ttu-id="989bf-104">使用者定義的運算子都必須指定 **static** 和 **public** 存取 [修飾詞](../../csharp/language-reference/keywords/modifiers.md) 。</span><span class="sxs-lookup"><span data-stu-id="989bf-104">Both the **static** and **public** access [modifiers](../../csharp/language-reference/keywords/modifiers.md) must be specified on user-defined operators.</span></span>

      }
