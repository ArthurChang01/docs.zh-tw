# <a name="compiler-error-cs1678"></a><span data-ttu-id="12c87-102">編譯器錯誤 CS1678</span><span class="sxs-lookup"><span data-stu-id="12c87-102">Compiler Error CS1678</span></span>
<span data-ttu-id="12c87-103">參數 'number' 宣告為類型 'type1'，但應該是 'type2'</span><span class="sxs-lookup"><span data-stu-id="12c87-103">Parameter 'number' is declared as type 'type1' but should be 'type2'</span></span>  
  
 <span data-ttu-id="12c87-104">當匿名方法中的參數類型和轉換方法的委派宣告不同時，就會發生這個錯誤。</span><span class="sxs-lookup"><span data-stu-id="12c87-104">This error occurs when the parameter type in an anonymous method is different from the declaration of the delegate you are casting the method to.</span></span>  
  
 <span data-ttu-id="12c87-105">下列範例會產生 CS1678：</span><span class="sxs-lookup"><span data-stu-id="12c87-105">The following sample generates CS1678:</span></span>  
  
```  
// CS1678  
delegate void D(int i);  
class Errors   
{  
   static void Main()   
   {  
      D d = delegate(string s) { };   // CS1678  
      // To resolve, use the following line instead:  
      // D d = delegate(int s) { };  
   }  
}  
```
