<span data-ttu-id="50227-105">C#（發音為"請參閱夏普"）是一種現代、物件導向且型別安全的程式設計語言。</span><span class="sxs-lookup"><span data-stu-id="50227-105">C# (pronounced "See Sharp") is a modern, object-oriented, and type-safe programming language.</span></span> <span data-ttu-id="50227-106">C# 源自於是 C 系列語言，使用 C、C++、Java 和 JavaScript 的程式設計人員會立即感到熟悉。</span><span class="sxs-lookup"><span data-stu-id="50227-106">C# has its roots in the C family of languages and will be immediately familiar to C, C++, Java, and JavaScript programmers.</span></span>

<span data-ttu-id="50227-107">本教程概述了 C# 8 及更早版本中語言的主要組成部分。</span><span class="sxs-lookup"><span data-stu-id="50227-107">This tour provides an overview of the major components of the language in C# 8 and earlier.</span></span> <span data-ttu-id="50227-108">如果要通過互動式示例流覽語言，請嘗試[C#](../tutorials/intro-to-csharp/index.md)教程的介紹。</span><span class="sxs-lookup"><span data-stu-id="50227-108">If you want to explore the language through interactive examples, try the [introduction to C#](../tutorials/intro-to-csharp/index.md) tutorials.</span></span>

<span data-ttu-id="50227-109">C# 是物件導向的語言，但 C# 更進一步支援「元件導向」\*\*\*\*\*\* 程式設計。</span><span class="sxs-lookup"><span data-stu-id="50227-109">C# is an object-oriented language, but C# further includes support for ***component-oriented*** programming.</span></span> <span data-ttu-id="50227-110">現代軟體設計逐漸依賴功能性上獨立與屬於自我描述套件的軟體元件。</span><span class="sxs-lookup"><span data-stu-id="50227-110">Contemporary software design increasingly relies on software components in the form of self-contained and self-describing packages of functionality.</span></span> <span data-ttu-id="50227-111">這些元件的關鍵是它們提供具有屬性、方法和事件的程式設計模型。</span><span class="sxs-lookup"><span data-stu-id="50227-111">Key to such components is that they present a programming model with properties, methods, and events.</span></span> <span data-ttu-id="50227-112">它們具有提供有關元件的聲明性資訊的屬性。</span><span class="sxs-lookup"><span data-stu-id="50227-112">They have attributes that provide declarative information about the component.</span></span> <span data-ttu-id="50227-113">它們包含自己的文檔。</span><span class="sxs-lookup"><span data-stu-id="50227-113">They incorporate their own documentation.</span></span> <span data-ttu-id="50227-114">C# 提供語言構造，直接支援這些概念，使 C# 成為創建和使用軟體元件的自然語言。</span><span class="sxs-lookup"><span data-stu-id="50227-114">C# provides language constructs to support directly these concepts, making C# a natural language in which to create and use software components.</span></span>

<span data-ttu-id="50227-115">幾個 C# 功能有助於構建堅固耐用的應用程式。</span><span class="sxs-lookup"><span data-stu-id="50227-115">Several C# features aid in the construction of robust and durable applications.</span></span> <span data-ttu-id="50227-116">***垃圾回收***會自動回收被無法訪問的未使用物件佔用的記憶體。</span><span class="sxs-lookup"><span data-stu-id="50227-116">***Garbage collection*** automatically reclaims memory occupied by unreachable unused objects.</span></span> <span data-ttu-id="50227-117">***異常處理***為錯誤偵測和恢復提供了結構化且可擴展的方法。</span><span class="sxs-lookup"><span data-stu-id="50227-117">***Exception handling*** provides a structured and extensible approach to error detection and recovery.</span></span> <span data-ttu-id="50227-118">語言***的型別安全***設計使得無法從未初始化變數讀取、無法讀取超出其邊界的索引陣列或執行未選中的類型強制轉換。</span><span class="sxs-lookup"><span data-stu-id="50227-118">The ***type-safe*** design of the language makes it impossible to read from uninitialized variables, to index arrays beyond their bounds, or to perform unchecked type casts.</span></span>

<span data-ttu-id="50227-119">C# 有***統一的型別系統***。</span><span class="sxs-lookup"><span data-stu-id="50227-119">C# has a ***unified type system***.</span></span> <span data-ttu-id="50227-120">所有的 C# 型別 (包括 `int` 和 `double` 等基本型別) 都繼承自單一的 `object` 根型別。</span><span class="sxs-lookup"><span data-stu-id="50227-120">All C# types, including primitive types such as `int` and `double`, inherit from a single root `object` type.</span></span> <span data-ttu-id="50227-121">因此，所有型別都擁有相同的一組常見作業，且任何型別的值都能以相同方式儲存、傳送及操作。</span><span class="sxs-lookup"><span data-stu-id="50227-121">Thus, all types share a set of common operations, and values of any type can be stored, transported, and operated upon in a consistent manner.</span></span> <span data-ttu-id="50227-122">此外，C# 支援使用者定義的參考型別和數值型別，因此能動態配置物件，以及內嵌儲存輕量結構。</span><span class="sxs-lookup"><span data-stu-id="50227-122">Furthermore, C# supports both user-defined reference types and value types, allowing dynamic allocation of objects as well as in-line storage of lightweight structures.</span></span>

<span data-ttu-id="50227-123">為了確保 C# 程式和庫能夠隨著時間的推移以相容的方式發展，在 C# 設計中，已非常強調***版本控制***。</span><span class="sxs-lookup"><span data-stu-id="50227-123">To ensure that C# programs and libraries can evolve over time in a compatible manner, much emphasis has been placed on ***versioning*** in C#'s design.</span></span> <span data-ttu-id="50227-124">許多程式設計語言很少注意這個問題。</span><span class="sxs-lookup"><span data-stu-id="50227-124">Many programming languages pay little attention to this issue.</span></span> <span data-ttu-id="50227-125">因此，引入依賴庫的較新版本時，用這些其他語言編寫的程式中斷的頻率超過必要。</span><span class="sxs-lookup"><span data-stu-id="50227-125">As a result, programs written in those other languages break more often than necessary when newer versions of dependent libraries are introduced.</span></span> <span data-ttu-id="50227-126">C#設計中受版本控制注意事項直接影響的方面包括單獨`virtual`和`override`修改器、方法重載解析規則以及對顯式介面成員聲明的支援。</span><span class="sxs-lookup"><span data-stu-id="50227-126">Aspects of C#'s design that were directly influenced by versioning considerations include the separate `virtual` and `override` modifiers, the rules for method overload resolution, and support for explicit interface member declarations.</span></span>

<span data-ttu-id="50227-127">在較新版本中，C# 採用了其他程式設計範例。</span><span class="sxs-lookup"><span data-stu-id="50227-127">In more recent versions, C# has embraced other programming paradigms.</span></span> <span data-ttu-id="50227-128">C# 包含支援函數程式設計技術（如 lambda 運算式）的功能。</span><span class="sxs-lookup"><span data-stu-id="50227-128">C# has included features that support functional programming techniques like lambda expressions.</span></span> <span data-ttu-id="50227-129">其他新功能支援分離資料和演算法，如模式匹配。</span><span class="sxs-lookup"><span data-stu-id="50227-129">Other new features support separating data and algorithms, like pattern matching.</span></span>
