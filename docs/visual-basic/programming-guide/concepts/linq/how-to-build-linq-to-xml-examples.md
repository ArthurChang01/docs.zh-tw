# <a name="how-to-build-linq-to-xml-examples---visual-basic"></a>如何：建立 LINQ to XML 範例-Visual Basic

此文件中的各種片段與範例使用各種命名空間中的類別和型別。 編譯 Visual Basic 程式碼時，您必須提供適當的 `Imports` 陳述式。

## <a name="example"></a>範例

下列程式碼包含 Visual Basic 範例建置與執行所需的 `Imports` 陳述式。 並非每個範例都需要使用所有 `Imports` 陳述式。
  
```vb
Imports System.Diagnostics
Imports System.Collections
Imports System.Collections.Generic
Imports System.Collections.Specialized
Imports System.Text
Imports System.Linq
Imports System.Xml
Imports System.Xml.Linq
Imports System.Xml.Schema
Imports System.Xml.XPath
Imports System.Xml.Xsl
Imports System.IO
Imports System.Threading
Imports System.Reflection
Imports System.IO.Packaging
```
