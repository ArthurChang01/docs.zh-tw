# <a name="language-integrated-axes-in-visual-basic-linq-to-xml"></a>Visual Basic 中已整合語言的軸心方法 (LINQ to XML)
本節描述直接內建功能[!INCLUDE[vbprvb](../../../../csharp/programming-guide/concepts/linq/includes/vbprvb_md.md)]以方便存取 XML 語言。 LINQ to XML 文件中的許多範例都使用這些整合式 [!INCLUDE[vbprvb](../../../../csharp/programming-guide/concepts/linq/includes/vbprvb_md.md)] 座標軸。  
  
## <a name="in-this-section"></a>本章節內容  
  
|主題|說明|  
|-----------|-----------------|  
|[XML 子代軸屬性](../../../../visual-basic/language-reference/xml-axis/xml-child-axis-property.md)|提供存取的下列其中一項目子系︰<xref:System.Xml.Linq.XElement>物件，<xref:System.Xml.Linq.XDocument>物件、 集合<xref:System.Xml.Linq.XElement>物件或一系列<xref:System.Xml.Linq.XDocument>物件。</xref:System.Xml.Linq.XDocument> </xref:System.Xml.Linq.XElement> </xref:System.Xml.Linq.XDocument> </xref:System.Xml.Linq.XElement> 這個座標軸等同於<xref:System.Xml.Linq.XContainer.Elements%2A>軸。</xref:System.Xml.Linq.XContainer.Elements%2A>|  
|[XML 子系軸屬性](../../../../visual-basic/language-reference/xml-axis/xml-descendant-axis-property.md)|可讓您存取下列的下階︰<xref:System.Xml.Linq.XElement>物件，<xref:System.Xml.Linq.XDocument>物件或一系列<xref:System.Xml.Linq.XElement>物件。</xref:System.Xml.Linq.XElement> </xref:System.Xml.Linq.XDocument> </xref:System.Xml.Linq.XElement> 這個座標軸等同於<xref:System.Xml.Linq.XContainer.Descendants%2A>軸。</xref:System.Xml.Linq.XContainer.Descendants%2A>|  
|[XML 屬性 (Attribute) 軸屬性 (Property)](../../../../visual-basic/language-reference/xml-axis/xml-attribute-axis-property.md)|提供的屬性存取<xref:System.Xml.Linq.XElement>物件。</xref:System.Xml.Linq.XElement> 這個座標軸大約會等於<xref:System.Xml.Linq.XElement.Attribute%2A>軸。</xref:System.Xml.Linq.XElement.Attribute%2A> 這個座標軸與<xref:System.Xml.Linq.XElement.Attribute%2A>它不會傳回屬性的值中軸<xref:System.Xml.Linq.XAttribute>物件。</xref:System.Xml.Linq.XAttribute> </xref:System.Xml.Linq.XElement.Attribute%2A>|  
|[擴充索引子屬性](../../../../visual-basic/language-reference/xml-axis/extension-indexer-property.md)|提供集合中個別項目的存取。|
