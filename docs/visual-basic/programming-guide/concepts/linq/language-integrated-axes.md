# <a name="language-integrated-axes-in-visual-basic-linq-to-xml"></a>Visual Basic 中已整合語言的軸心方法 (LINQ to XML)
本章節描述直接內建的 Visual Basic 語言，以讓您輕鬆存取 XML 的功能。 許多 LINQ to XML 文件的範例使用這些整合式的 Visual Basic 軸。  
  
## <a name="in-this-section"></a>本節內容  
  
|主題|描述|  
|-----------|-----------------|  
|[XML 子代軸屬性](../../../../visual-basic/language-reference/xml-axis/xml-child-axis-property.md)|提供下列任一項目之子系的存取：<xref:System.Xml.Linq.XElement> 物件、<xref:System.Xml.Linq.XDocument> 物件、<xref:System.Xml.Linq.XElement> 物件的集合，或是 <xref:System.Xml.Linq.XDocument> 物件的集合。 這個座標軸等同於 <xref:System.Xml.Linq.XContainer.Elements%2A> 座標軸。|  
|[XML 子系軸屬性](../../../../visual-basic/language-reference/xml-axis/xml-descendant-axis-property.md)|提供下列任一項目之子系的存取：<xref:System.Xml.Linq.XElement> 物件、<xref:System.Xml.Linq.XDocument> 物件、<xref:System.Xml.Linq.XElement> 物件的集合。 這個座標軸等同於 <xref:System.Xml.Linq.XContainer.Descendants%2A> 座標軸。|  
|[XML 屬性 (Attribute) 軸屬性 (Property)](../../../../visual-basic/language-reference/xml-axis/xml-attribute-axis-property.md)|提供 <xref:System.Xml.Linq.XElement> 物件之屬性的存取。 這個座標軸大致等同於 <xref:System.Xml.Linq.XElement.Attribute%2A> 座標軸。 這個座標軸與 <xref:System.Xml.Linq.XElement.Attribute%2A> 座標軸不同之處在於，前者會傳回屬性的值，而非 <xref:System.Xml.Linq.XAttribute> 物件的值。|  
|[擴充索引子屬性](../../../../visual-basic/language-reference/xml-axis/extension-indexer-property.md)|提供集合中個別項目的存取。|
