# <a name="how-to-retrieve-a-collection-of-attributes-linq-to-xml-visual-basic"></a>如何︰ 擷取屬性 (LINQ to XML) 的集合 (Visual Basic)
本主題將介紹<xref:System.Xml.Linq.XElement.Attributes%2A>方法。</xref:System.Xml.Linq.XElement.Attributes%2A> 這個方法會擷取項目的屬性。  
  
## <a name="example"></a>範例  
 下列範例顯示如何逐一查看項目之屬性的集合。  
  
```vb  
Dim val = _  
    <Value ID="1243" Type="int" ConvertableTo="double">100</Value>  
Dim listOfAttributes As IEnumerable(Of XAttribute) = _  
    From att In val.Attributes() _  
    Select att  
For Each att As XAttribute In listOfAttributes  
    Console.WriteLine(att)  
Next  
```  
  
 此程式碼會產生下列輸出：  
  
```  
ID="1243"  
Type="int"  
ConvertableTo="double"  
```
