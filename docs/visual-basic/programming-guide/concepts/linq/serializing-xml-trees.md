# <a name="serializing-xml-trees-visual-basic"></a>序列化 XML 樹狀結構 (Visual Basic)
序列化 XML 樹狀結構表示從 XML 樹狀結構產生 XML。 您可以序列化到檔案、<xref:System.IO.TextWriter> 類別的具體實作，或 <xref:System.Xml.XmlWriter> 的具體實作。  
  
 您可以控制各方面的序列化。 例如，您可以控制是否要縮排序列化的 XML，以及是否要撰寫 XML 宣告。
