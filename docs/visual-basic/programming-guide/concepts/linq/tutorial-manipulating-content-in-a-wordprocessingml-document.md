# <a name="tutorial-manipulating-content-in-a-wordprocessingml-document-visual-basic"></a>教學課程：操作 WordprocessingML 檔中的內容（Visual Basic）
本教學課程顯示如何應用功能性轉換方法與 LINQ to XML 來管理 XML 文件。 Visual Basic 範例會查詢並管理 Microsoft Word 所儲存之 Office Open XML WordprocessingML 檔中的資訊。  
  
 如需詳細資訊，請參閱[Eric 白的 Blog](http://www.ericwhite.com)。
