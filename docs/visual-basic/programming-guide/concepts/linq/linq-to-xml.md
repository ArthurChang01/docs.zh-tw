# <a name="linq-to-xml-visual-basic"></a>LINQ to XML (Visual Basic)
LINQ to XML 提供記憶體中 XML 程式設計介面，會利用.net language-integrated Query (LINQ) 架構。 LINQ to XML 會使用最新的.NET Framework 語言功能，而且相當於更新、 重新設計文件物件模型 (DOM) XML 程式設計介面。  
  
 LINQ 技術系列物件 (LINQ to Objects)、 關聯式資料庫 (LINQ to SQL) 和 XML (LINQ to XML) 提供一致的查詢體驗。
