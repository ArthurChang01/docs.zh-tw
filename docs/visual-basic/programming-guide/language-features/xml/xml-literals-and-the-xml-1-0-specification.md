# <a name="xml-literals-and-the-xml-10-specification-visual-basic"></a>XML 常值和 XML 1.0 規格 (Visual Basic)
在 Visual Basic XML 常值的語法支援大部分的可延伸標記語言 (XML) 1.0 規格。 如需有關 XML 1.0 規格的詳細資訊，請參閱[可延伸標記語言 (XML) 1.0](http://go.microsoft.com/fwlink/?LinkId=73927) W3C 網站上。  
  
## <a name="what-visual-basic-does-not-support"></a>Visual Basic 不支援是什麼  
  
-   XML 常值不能包含文件類型定義 (DTD)。  
  
-   XML 文件常值的開頭必須為 XML 文件宣告。  
  
-   XML 常值不能包含超過 65535 個字元，在一行上。  
  
-   XML 命名空間前置詞、 項目名稱和屬性名稱不能包含超過 1024 個字元。  
  
## <a name="extra-features-that-visual-basic-supports"></a>Visual Basic 支援的額外功能  
  
-   文件和項目常值中允許的內嵌的運算式語法不是有效的 XML。
