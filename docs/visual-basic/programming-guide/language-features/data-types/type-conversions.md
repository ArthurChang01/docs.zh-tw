# <a name="type-conversions-in-visual-basic"></a>Visual Basic 中的類型轉換
將值從一種資料類型變更為其他類型的程序稱為*轉換*。 轉換為*擴展*或*縮小*，取決於相關型別的資料容量。 它們也是*隱含*或*明確*，取決於在原始程式碼中的語法。
