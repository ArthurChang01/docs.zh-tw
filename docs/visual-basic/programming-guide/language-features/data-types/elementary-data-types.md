# <a name="elementary-data-types-visual-basic"></a>基礎資料類型 (Visual Basic)
[!INCLUDE[vbprvb](../../../../csharp/programming-guide/concepts/linq/includes/vbprvb_md.md)]提供一組預先定義的資料型別，可用於許多程式設計項目。 本章節描述這些類型，以及如何使用它們。  
  
> [!NOTE]
>  在 Visual Basic 中的每個基本資料型別支援的結構或類別中<xref:System>命名空間。</xref:System> 編譯器會使用每個資料型別關鍵字做為別名的基礎結構或類別。 例如，使用保留的字宣告變數`Byte`等同於宣告使用完整的結構名稱<xref:System.Byte?displayProperty=fullName>.</xref:System.Byte?displayProperty=fullName>
