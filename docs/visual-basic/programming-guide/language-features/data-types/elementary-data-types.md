# <a name="elementary-data-types-visual-basic"></a>基礎資料類型 (Visual Basic)
Visual Basic 會提供一組預先定義的資料類型, 您可以將其用於許多程式設計項目。 本節說明這些類型, 以及如何使用它們。  
  
> [!NOTE]
> Visual Basic 中的每個基本資料類型都受到結構或<xref:System>命名空間中的類別支援。 編譯器會使用每個資料類型關鍵字做為基礎結構或類別的別名。 例如, 使用保留字`Byte`來宣告變數, 就和使用完整結構名稱<xref:System.Byte?displayProperty=nameWithType>進行宣告一樣。
