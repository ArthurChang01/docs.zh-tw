# <a name="structures-visual-basic"></a>結構 (Visual Basic)
A*結構*是使用者定義型別 (UDT) 的先前版本所支援之一般化[!INCLUDE[vbprvb](../../../../csharp/programming-guide/concepts/linq/includes/vbprvb_md.md)]。 除了欄位之外，結構還公開屬性、 方法和事件。 結構可以實作一個或多個介面，而且您可以宣告每個欄位個別的存取層級。  
  
 您可以結合以建立結構的不同類型的資料項目。 結構會將一或多個*元素*和與結構本身。 當您宣告結構時，它會變成*複合資料型別*，而且您可以宣告該型別的變數。  
  
 當您想要的單一變數來保存相關的資訊片段，結構會很有用。 比方說，您可能要將員工的名稱、 電話分機號碼和薪資放在一起。 您可以使用數個變數，這項資訊，或您可以使用單一員工變數而定義的結構。 當您有許多員工，且因此變數的多個執行個體時，結構的好處就顯而易見。
