## <a name="beginning-of-lifetime"></a><span data-ttu-id="5944e-116">存留期的開頭</span><span class="sxs-lookup"><span data-stu-id="5944e-116">Beginning of Lifetime</span></span>

 <span data-ttu-id="5944e-122">在程式內的區塊內宣告的變數（例如 `For` 迴圈）會在程式的進入時初始化。</span><span class="sxs-lookup"><span data-stu-id="5944e-122">Variables declared within a block inside a procedure (such as a `For` loop) are initialized on entry to the procedure.</span></span> <span data-ttu-id="5944e-123">無論您的程式碼是否執行區塊，這些初始化都會生效。</span><span class="sxs-lookup"><span data-stu-id="5944e-123">These initializations take effect whether or not your code ever executes the block.</span></span>
