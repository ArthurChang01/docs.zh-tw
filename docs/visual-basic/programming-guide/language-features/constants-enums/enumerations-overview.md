# <a name="enumerations-overview-visual-basic"></a>列舉類型的概觀 (Visual Basic)
列舉型別提供便利的方式來使用相關常數組和名稱相關聯的常數值。 例如，您可以宣告列舉來當作一組與當週的日次建立關聯的整數常數，接著在程式碼中使用日次的名稱而不是它們的整數值。
