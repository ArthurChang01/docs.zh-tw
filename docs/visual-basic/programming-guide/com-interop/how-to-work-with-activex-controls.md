# <a name="how-to-work-with-activex-controls-visual-basic"></a>如何：使用 ActiveX 控制項 (Visual Basic)
ActiveX 控制項是 COM 元件或您可以插入 Web 網頁或其他應用程式來重複使用封裝的功能，其他人程式化的物件。 您可以使用 Visual Basic 6.0 和較早版本所開發的 ActiveX 控制項加入至功能**工具箱**的[!INCLUDE[vsprvs](../../../csharp/includes/vsprvs_md.md)]。  
  
### <a name="to-add-activex-controls-to-the-toolbox"></a>若要將 ActiveX 控制項加入至工具箱  
  
1.  在**工具**] 功能表上，按一下 [**選擇工具箱項目**。  
  
     **[選擇工具箱**] 對話方塊隨即出現。  
  
2.  按一下 [ **COM 元件**] 索引標籤。  
  
3.  選取您要使用，然後按一下 ActiveX 控制項旁的核取方塊**確定**。  
  
     新的控制項出現在其他工具**工具箱**。  
  
    > [!NOTE]
    >  您可以使用 「 Aximp 」 公用程式來手動建立 ActiveX 控制項的 interop 組件。 如需詳細資訊，請參閱[Aximp.exe （Windows Form ActiveX 控制項匯入工具）](http://msdn.microsoft.com/library/482c0d83-7144-4497-b626-87d2351b78d0)。
