# <a name="anonymous-type-initializers-are-not-valid-in-debug-windows"></a>偵錯視窗中的匿名類型初始設定式無效
在偵錯工作階段期間，已於 [監看式]  或 [即時運算]  視窗中輸入匿名類型初始設定式。 偵錯視窗中的匿名類型初始設定式無法使用。  
  
 **錯誤 ID:** BC36577  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 在程式碼編輯器中，測試匿名類型初始設定式。
