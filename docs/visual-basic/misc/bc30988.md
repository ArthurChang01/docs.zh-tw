<span data-ttu-id="17f5b-103">當您宣告類別的執行個體時， `New` 關鍵字後面必須接著類型名稱或 `With`。</span><span class="sxs-lookup"><span data-stu-id="17f5b-103">When you declare an instance of a class, the `New` keyword must be followed by a type name or by `With`.</span></span> <span data-ttu-id="17f5b-104">例如，下列陳述式每個都宣告 `client` 為 `Customer` 類別的執行個體。</span><span class="sxs-lookup"><span data-stu-id="17f5b-104">For example, the following statements each declare `client` to be an instance of the `Customer` class.</span></span> <span data-ttu-id="17f5b-105">`Customer` 後面是類型名稱 `New`。</span><span class="sxs-lookup"><span data-stu-id="17f5b-105">The type name `Customer` follows `New`.</span></span>  
  
```

```  
  
 <span data-ttu-id="17f5b-106">開頭為 [!INCLUDE[vb_orcas_long](~/includes/vb-orcas-long-md.md)]，您可以宣告物件是匿名類型的執行個體，這樣就不用指定資料類型。</span><span class="sxs-lookup"><span data-stu-id="17f5b-106">Beginning with [!INCLUDE[vb_orcas_long](~/includes/vb-orcas-long-md.md)], you can declare an object to be an instance of an anonymous type, in which case you do not specify a data type.</span></span> <span data-ttu-id="17f5b-107">在匿名類型宣告中，關鍵字 `With` 跟隨在 `New`的後面。</span><span class="sxs-lookup"><span data-stu-id="17f5b-107">In anonymous type declarations, the keyword `With` follows `New`.</span></span>
