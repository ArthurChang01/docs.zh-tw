# <a name="option-optionname-requires-argument"></a>選項\<選項名稱 > 需要\<引數 >

 **錯誤 ID:** BC2006  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 請提供遺漏的引數。
