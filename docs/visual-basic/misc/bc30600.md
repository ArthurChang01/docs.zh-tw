# <a name="withevents-variable-does-not-raise-any-instance-events-that-are-accessible-to-containername"></a>' WithEvents ' 變數不會引發 '\<容器 > ' 可存取的任何實例事件
您所參考的物件未引發可存取的事件。  
  
 **錯誤識別碼:** BC30600  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
1. 修改事件的存取修飾詞，或選擇可在目前內容中存取的事件。
