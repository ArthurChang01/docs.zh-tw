# <a name="cannot-convert-argument-ltargumentnamegt-of-type-lttype1gt-to-type-lttype2gt"></a>無法轉換引數 '&lt;argumentname&gt;'的類型'&lt;type1&gt;' 為 type'&lt;type2&gt;'
轉換作業嘗試將引數轉換為不相容的類型。  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
1.  確定您嘗試轉換引數的目標類型有效。
