# <a name="cannot-convert-argument-argumentname-of-type-type1-to-type-type2"></a>無法轉換引數 '\<引數名稱 >' 的類型'\<type1 >' 為類型 '\<type2 >'
轉換作業嘗試將引數轉換為不相容的類型。  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
1.  確定您嘗試轉換引數的目標類型有效。
