# <a name="local-variable-variablename-is-already-declared-in-the-current-block"></a>區域變數 '\<變數名稱 >' 已在目前區塊中宣告
同一個區塊中宣告了兩個具有相同名稱的變數 (可能是程序或控制區塊，例如 `If...Then`)。  
  
 **錯誤 ID:** BC30288  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 將其中一個宣告移至不同的區塊，或變更其中一個變數的名稱。
