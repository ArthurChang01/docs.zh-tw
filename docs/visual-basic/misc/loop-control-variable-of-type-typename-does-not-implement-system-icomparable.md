# 類型 &#39;&lt;類型名稱&gt;&#39; 的迴圈控制變數無法實作 &#39;System.IComparable&#39; 介面
變數所屬的類型未實作 `System.IComparable` 介面。`System.IComparable` 必須由實值類型或類別實作。  
  
### 更正這個錯誤  
  
1.  確定變數是實值類型。
