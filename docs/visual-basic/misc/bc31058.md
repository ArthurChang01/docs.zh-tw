# <a name="conflicts-with-name1-which-is-implicitly-created-for-name2-in-type-name3"></a>與衝突 '\<name1 > '，其會隱含地建立的'\<name2 >' 中\<類型 >'\<name3 >'
類型成員名稱與另一個隱含建立的成員名稱相衝突。  
  
 **錯誤 ID:** BC31058  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 請重新命名明確宣告的成員，以移除命名衝突。
