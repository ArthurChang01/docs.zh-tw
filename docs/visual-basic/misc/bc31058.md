# <a name="conflicts-with-name1-which-is-implicitly-created-for-name2-in-type-name3"></a>與 '\<name1 > ' 衝突, 其已針對類型 > '\< \<name3 > ' 中\<的 ' name2 > ' 隱含建立
類型成員名稱與另一個隱含建立的成員名稱相衝突。  
  
 **錯誤識別碼:** BC31058  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 請重新命名明確宣告的成員，以移除命名衝突。
