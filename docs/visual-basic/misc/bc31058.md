# <a name="conflicts-with-ltname1gt-which-is-implicitly-created-for-ltname2gt-in-lttypegt-ltname3gt"></a>與衝突 '&lt;name1&gt;'，其會隱含地建立的'&lt;name2&gt;' 中&lt;型別&gt;'&lt;name3&gt;'
類型成員名稱與另一個隱含建立的成員名稱相衝突。  
  
 **錯誤 ID:** BC31058  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請重新命名明確宣告的成員，以移除命名衝突。
