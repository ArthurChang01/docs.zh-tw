# <a name="argument-not-specified-for-parameter-parametername"></a>未指定參數的引數 '\<參數名稱 >'
尚未提供必要參數的引數。  
  
 **錯誤 ID:** BC36569  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 請為指定的參數提供引數。
