# 無法同時在變數及其類型上指定陣列修飾詞
陣列修飾詞存在於變數和其類型上，這表示陣列的陣列。  
  
 **錯誤 ID︰**BC31087  
  
### 更正這個錯誤  
  
-   請從類型規範中移除陣列修飾詞。
