# <a name="type-parameter-lttypeparameternamegt-can-only-have-one-constraint-that-is-a-class"></a>類型參數 '&lt;型別參數名稱&gt;' 只能有一個是類別的條件約束
條件約束清單包含多個類別。  
  
 類型參數的條件約束清單可以接受任意數目的介面，但只能有一個類別。 型別引數提供的型別參數必須繼承自該類別，，和 Visual Basic 不支援多重繼承。  
  
 **錯誤 ID:** BC32047  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   選取一個類別，且條件約束清單中只包含該類別。  
  
-   您也許可以定義其他的類型參數，以滿足這份條件約束清單中不能包含的一或多個類別。
