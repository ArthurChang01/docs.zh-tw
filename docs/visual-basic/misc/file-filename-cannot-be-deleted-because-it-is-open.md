# <a name="file-filename-cannot-be-deleted-because-it-is-open"></a>檔案 '\<檔案名稱 >' 無法刪除，因為它已開啟
嘗試刪除已開啟的檔案。  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 請先關閉檔案，再進行刪除。
