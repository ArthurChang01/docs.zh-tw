# <a name="file-39ltfilenamegt39-cannot-be-deleted-because-it-is-open"></a>檔案 &#39;&lt;filename&gt;&#39; 已開啟，所以無法刪除
嘗試刪除已開啟的檔案。  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請先關閉檔案，再進行刪除。
