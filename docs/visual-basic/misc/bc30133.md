# <a name="goto-statements-are-not-valid-in-the-immediate-window"></a>即時運算視窗中的 'GoTo' 陳述式無效
`GoTo` 陳述式會執行分支，但在偵錯內容中則不允許執行。  
  
 **錯誤 ID:** BC30133  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請不要在 [即時運算] `GoTo`**視窗中發出** 陳述式。
