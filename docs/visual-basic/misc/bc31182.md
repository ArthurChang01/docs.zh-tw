# <a name="attribute-value-is-not-valid-expecting-value1-or-value2"></a>屬性值無效，必須是 'value1' 或 'value2'
XML 文件常值中指定了無效的屬性值。 XML 文件常值的有效屬性值是以 XML 1.0 規格為基礎。  
  
 **錯誤 ID:** BC31182  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 根據 XML 1.0 規格，將屬性值變更為有效的值。
