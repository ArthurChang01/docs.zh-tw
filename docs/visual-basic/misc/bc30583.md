# <a name="39ltmethodnamegt39-cannot-be-implemented-more-than-once"></a>&#39;&lt;methodname&gt; &#39;不能超過一次地實作
您嘗試實作已經實作的方法。  
  
 **錯誤 ID︰** BC30583  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
1.  請檢查方法名稱以確定拼字正確。  
  
2.  如果方法多餘，請移除額外的宣告。
