# <a name="classes-can-inherit-only-from-other-classes"></a>類別只能繼承自其他類別
`Inherits` 陳述式指定類別以外的類型作為其基底類型。  
  
 **錯誤 ID︰** BC30258  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   將基底類型變更為已定義的類別。
