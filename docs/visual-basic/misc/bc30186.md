# 即時運算視窗中的 &#39;Using&#39; 陳述式無效
`Using` 陳述式只能用於原始程式碼中。  
  
 **錯誤 ID：**BC30186  
  
### 更正這個錯誤  
  
-   從 \[即時運算\] 視窗移除 `Using` 陳述式。
