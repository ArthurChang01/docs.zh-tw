# <a name="39using39-statements-are-not-valid-in-the-immediate-window"></a>'Using' 陳述式不是有效的即時運算視窗中
`Using`陳述式僅能在原始程式碼中。  
  
 **錯誤 ID：** BC30186  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   從 [即時運算] `Using`**視窗移除** 陳述式。
