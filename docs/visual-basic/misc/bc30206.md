# <a name="option-must-be-followed-by-compare-explicit-infer-or-strict"></a>'Option' 必須在 'Compare'、'Explicit'、'Infer' 或 'Strict' 之前
您只能透過 `Option` 陳述式設定的選項為 `Compare`、 `Explicit`、 `Infer`和 `Strict`。  
  
 **錯誤 ID:** BC30206  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 請查看選項關鍵字 (`Compare`、 `Explicit`、 `Infer`或 `Strict`) 是否拼字錯誤。  
  
- 查看 `Option` 陳述式是否不完整。
