# <a name="39option39-must-be-followed-by-39compare39-39explicit39-39infer39-or-39strict39"></a>&#39;選項&#39;後面必須接著&#39;比較&#39;， &#39;Explicit&#39;， &#39;Infer&#39;，或&#39;Strict&#39;
您只能透過 `Option` 陳述式設定的選項為 `Compare`、 `Explicit`、 `Infer`和 `Strict`。  
  
 **錯誤識別碼：** BC30206  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請查看選項關鍵字 (`Compare`、 `Explicit`、 `Infer`或 `Strict`) 是否拼字錯誤。  
  
-   查看 `Option` 陳述式是否不完整。
