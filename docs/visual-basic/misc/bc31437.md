# <a name="elementname-cannot-be-declared-partial-because-partial-methods-must-be-subs"></a>' @no__t 0elementname > ' 不可以宣告為 ' Partial '，因為部分方法必須是 Sub
只有 `Sub` 程序可以宣告為部分方法。 例如，因為 `partialMethod` 是函式，所以下列程式碼會造成這個錯誤：  
  
```vb  
' Partial Private Function partialMethod(ByVal n As Integer) As Integer  
' End Function  
```  
  
 **錯誤識別碼：** BC31437  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 將您要宣告為部分方法的項目轉換成 `Sub`。  
  
- 這種情況下請勿使用部分方法。
