# <a name="inherits-not-valid-in-modules"></a>'Inherits' 在模組中無效
模組內發生 `Inherits` 陳述式。 模組不可繼承自類別。  
  
 **錯誤 ID:** BC30230  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請移除 `Inherits` 陳述式，或將模組重新輸入為類別。
