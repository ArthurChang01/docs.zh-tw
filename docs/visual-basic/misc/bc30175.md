# <a name="lttype1gt-lttypenamegt-and-lttype2gt-lttypenamegt-declared-in-ltnamegt-conflict-in-ltnamespacegt-ltnamespacenamegt"></a>&lt;type1&gt; '&lt;typename&gt;' 並&lt;type2&gt; '&lt;typename&gt;'，在中宣告'&lt;名稱&gt;' 中, 產生衝突&lt;命名空間&gt; '&lt;namespacename&gt;'
兩個宣告的項目 (例如類別和模組) 在相同的命名空間中同名。  
  
 **錯誤 ID:** BC30175  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   將其中一個項目的宣告移至不同的命名空間。  
  
-   變更其中一個項目的名稱。
