# <a name="lttype1gt-39lttypenamegt39-and-lttype2gt-39lttypenamegt39-declared-in-39ltnamegt39-conflict-in-ltnamespacegt-39ltnamespacenamegt39"></a>&lt;type1&gt; &#39;&lt;typename&gt;&#39; 和&lt;type2&gt; &#39;&lt;typename&gt;&#39; 宣告中 （& s) #39;&lt;名稱&gt;&#39; 中的衝突&lt;命名空間&gt;&#39;&lt;namespacename&gt;&#39;
兩個宣告的項目 (例如類別和模組) 在相同的命名空間中同名。  
  
 **錯誤 ID︰** BC30175  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   將其中一個項目的宣告移至不同的命名空間。  
  
-   變更其中一個項目的名稱。
