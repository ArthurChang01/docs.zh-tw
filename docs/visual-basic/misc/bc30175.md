# <a name="type1-typename-and-type2-typename-declared-in-name-conflict-in-namespace-namespacename"></a>\<type1 > '\<類型名稱 >' 並\<type2 >'\<類型名稱 > '，在中宣告'\<名稱 >' 中, 產生衝突\<命名空間 > '\<命名空間名稱 >'
兩個宣告的項目 (例如類別和模組) 在相同的命名空間中同名。  
  
 **錯誤 ID:** BC30175  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 將其中一個項目的宣告移至不同的命名空間。  
  
- 變更其中一個項目的名稱。
