# <a name="unable-to-create-temp-file-in-path-39ltfilenamegt39-lterror-messagegt"></a>無法在路徑中建立暫存檔&#39; &lt;filename&gt;&#39;:&lt;錯誤訊息&gt;
Visual Basic 編譯器呼叫組件連結器 (Al.exe，也稱為 Alink)，以產生資訊清單的組件。 連結器在建立檔案或將記憶體內資源寫入其中時回報了錯誤。 這可能是設定問題。  
  
 **錯誤 ID︰** BC30138  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
1.  請檢查引用的錯誤訊息，請參閱主題[Al.exe （組件連結器）](../../framework/tools/al-exe-assembly-linker.md)以取得進一步說明和建議。  
  
2.  如果錯誤持續發生，請收集情況的相關資訊，並通知 Microsoft 產品支援服務。
