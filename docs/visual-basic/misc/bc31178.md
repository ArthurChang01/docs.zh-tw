# <a name="expected-closing-3939-for-xml-entity"></a>必須是右 &#39;&#39;XML 實體
XML 實體參考 (例如 `>`) 遺漏結尾分號 (;)。  
  
 **錯誤 ID︰** BC31178  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   在 XML 實體參考結尾加入分號。
