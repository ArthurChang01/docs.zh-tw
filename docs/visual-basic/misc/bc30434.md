# <a name="events-in-a-module-cannot-be-declared-39ltspecifiergt39"></a>在模組中的事件不可以宣告&#39;&lt;規範&gt;&#39;
您已經在 `Module` 陳述式內使用對事件無效的規範。 模組絕不會執行個體化、不支援繼承，且不能實作介面。  
  
 **錯誤 ID：** BC30434  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請移除規範。
