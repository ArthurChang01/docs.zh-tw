# <a name="field-39ltfieldnamegt39-is-of-an-unsupported-type"></a>欄位 '&lt;fieldname&gt;' 屬於不受支援的型別
欄位宣告為型別無法由[!INCLUDE[vsprvs](../../csharp/includes/vsprvs_md.md)]。  
  
 **錯誤 ID︰** BC30656  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
1.  將欄位定義為支援的類型。  
  
2.  開發可描述欄位中資料的新類型。
