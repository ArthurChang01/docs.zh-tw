# <a name="field-39ltfieldnamegt39-is-of-an-unsupported-type"></a>欄位&#39; &lt;fieldname&gt; &#39;是不支援的類型
欄位宣告為無法由 Visual Studio 的型別。  
  
 **錯誤 ID︰** BC30656  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
1.  將欄位定義為支援的類型。  
  
2.  開發可描述欄位中資料的新類型。
