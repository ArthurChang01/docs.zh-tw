# <a name="39byval39-and-39byref39-cannot-be-combined"></a>&#39;ByVal&#39;和&#39;ByRef&#39;無法合併
`ByVal` 與 `ByRef` 關鍵字互斥。  
  
 **錯誤 ID︰** BC30641  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請指定 `ByVal` 或 `ByRef`，但兩者不能同時指定。
