# 無法結合 &#39;ByVal&#39; 和 &#39;ByRef&#39;
`ByVal` 與 `ByRef` 關鍵字互斥。  
  
 **錯誤 ID︰**BC30641  
  
### 更正這個錯誤  
  
-   請指定 `ByVal` 或 `ByRef`，但兩者不能同時指定。
