# <a name="field-or-property-ltmembernamegt-is-not-found"></a>欄位或屬性 '&lt;membername&gt;' 找不到
屬性參考了未定義的欄位或屬性。  
  
 **錯誤 ID:** BC30661  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
1.  請檢查欄位或屬性的拼字。  
  
2.  請確認您的專案可以存取正確的屬性類別，並視需要將它加入專案參考中。
