# <a name="-expected"></a><span data-ttu-id="20469-102">必須是 '{'</span><span class="sxs-lookup"><span data-stu-id="20469-102">'{' expected</span></span>
<span data-ttu-id="20469-103">若要使用物件初始設定式來宣告具名或匿名類型的執行個體，您必須以大括弧 ({ 和 }) 括住欄位或屬性清單和其初始值。</span><span class="sxs-lookup"><span data-stu-id="20469-103">To declare an instance of a named or anonymous type by using an object initializer, you must enclose the list of fields or properties and their initial values in braces ({ and }).</span></span>  
  
```  
Dim client As New Customer() With {.Name = "Microsoft", .City = "Seattle"}  
Dim emp = New Employee() With {.Name = "Rob Young", .ID = 55555}  
Dim anon = New With {.ID = 123456}  
```
