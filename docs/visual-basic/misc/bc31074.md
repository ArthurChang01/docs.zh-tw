# <a name="delegate-constructor-for-type-typename-necessary-for-the-implementation-of-delegates-cannot-be-found"></a>委派建構函式的型別 '\<類型名稱 >'，找不到委派實作所需
包含類型的組件可能已損毀或無效。  
  
 **錯誤 ID:** BC31074  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
1. 請確認包含這種類型的組件有效。  
  
2. 再次編譯程式，看看錯誤是否重複發生。  
  
3. 如果錯誤持續發生，請儲存您的工作，然後重新啟動 Visual Studio。
