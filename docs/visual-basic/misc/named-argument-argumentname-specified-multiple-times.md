# <a name="named-argument-ltargumentnamegt-specified-multiple-times"></a>具名引數 '&lt;引數名稱&gt;' 指定了多次
您已多次指定具名引數。  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請移除所有規格，只留下其中一個規格。
