# <a name="named-argument-argumentname-specified-multiple-times"></a>具名引數 '\<引數名稱 >' 已指定多次
您已多次指定具名引數。  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 請移除所有規格，只留下其中一個規格。
