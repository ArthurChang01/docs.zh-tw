# <a name="no-accessible-overloaded-39ltmethodnamegt39-can-be-called-with-these-arguments-without-a-narrowing-conversion"></a>沒有可存取多載 &#39;&lt;methodname&gt;&#39; 可以呼叫且不用縮小轉換這些引數
已呼叫多載方法，但沒有方法可在未縮小轉換的情況下，符合所提供的引數清單。  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
1.  請指定 `Option Strict Off`。  
  
2.  變更引數，以符合多載方法的其中一個特徵標記。
