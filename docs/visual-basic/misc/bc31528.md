# <a name="construct-makes-an-indirect-reference-to-assembly-assemblyname-which-contains-typename"></a>建構會間接參考組件 '\<組件名稱 > '，其中包含'\<類型名稱 >'
建構會間接參考組件 '\<組件名稱 >'，其中包含\<類型名稱 >。 將檔案參考加入\<檔名 > 至您的專案。  
  
 運算式使用類別、結構、介面、列舉或委派這類類型，但您的組件沒有定義該類型之組件的專案參考。  
  
 **錯誤 ID:** BC31528  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 在專案屬性中，加入包含組件之檔案的參考，而這個組件定義所使用的類型。
