# <a name="construct-makes-an-indirect-reference-to-assembly-39ltassemblynamegt39-which-contains-39lttypenamegt39"></a>建構會間接參考組件&#39; &lt;assemblyname&gt;&#39;，其中包含&#39;&lt;類型名稱&gt;&#39;
建構會間接參考組件 '\<assemblyname >'，其中包含\<類型名稱 >。 將檔案參考加入\<檔名 > 至您的專案。  
  
 運算式使用類別、結構、介面、列舉或委派這類類型，但您的組件沒有定義該類型之組件的專案參考。  
  
 **錯誤 ID︰** BC31528  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   在專案屬性中，加入包含組件之檔案的參考，而這個組件定義所使用的類型。
