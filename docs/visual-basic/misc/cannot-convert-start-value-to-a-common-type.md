# <a name="cannot-convert-start-value-of-type-lttype1gt-limit-value-of-lttype2gt-and-step-value-of-lttype3gt-to-a-common-type"></a>無法轉換類型的起始值 '&lt;type1&gt;'，限制的值'&lt;type2&gt;'和間距的值'&lt;類型 3>&gt;' 為一般類型
指定的 `For...Next` 控制項值具有不同的類型，因此無效，因為它們無法轉換成一般類型。  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
1.  變更一個或多個值的類型，以全部轉換成一般類型。
