# <a name="no-mouse-is-present"></a>滑鼠不存在
已呼叫 `My.Computer.Mouse` 物件的其中一個屬性，但是電腦未安裝滑鼠或滑鼠連接埠。  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   在 `Try...Catch` 物件的屬性呼叫周圍加入 `My.Computer.Mouse` 區塊。  
  
     — 或 —  
  
-   在電腦上安裝滑鼠。
