# <a name="type-characters-cannot-be-used-in-range-variable-declarations"></a>範圍變數宣告中不能使用類型字元
LINQ 查詢中的範圍變數宣告名稱結尾已加入類型字元。  
  
 **錯誤 ID︰** BC36601  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
1.  從範圍變數宣告中移除類型字元。
