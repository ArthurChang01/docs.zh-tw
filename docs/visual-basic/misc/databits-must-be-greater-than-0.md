# <a name="databits-must-be-greater-than-0"></a>DataBits 必須大於 0
為 `DataBits` 方法指定的 `My.Computer.Ports.OpenSerialPort` 引數必須大於零。  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   將 `DataBits` 引數的值變更為正數。
