# <a name="property-accessors-cannot-be-declared-39ltaccessmodifiergt39-in-a-39notoverridable39-property"></a><span data-ttu-id="4d2d5-102">屬性存取子不可以宣告 &#39;&lt;存取修飾詞&gt;&#39; 在 &#39;NotOverridable &#39;屬性</span><span class="sxs-lookup"><span data-stu-id="4d2d5-102">Property accessors cannot be declared &#39;&lt;accessmodifier&gt;&#39; in a &#39;NotOverridable&#39; property</span></span>
<span data-ttu-id="4d2d5-103">A [Get 陳述式](../../visual-basic/language-reference/statements/get-statement.md)或[Set 陳述式](../../visual-basic/language-reference/statements/set-statement.md)中`NotOverridable`屬性包含`Private`關鍵字。</span><span class="sxs-lookup"><span data-stu-id="4d2d5-103">A [Get Statement](../../visual-basic/language-reference/statements/get-statement.md) or [Set Statement](../../visual-basic/language-reference/statements/set-statement.md) in a `NotOverridable` property includes the `Private` keyword.</span></span>  
  
 <span data-ttu-id="4d2d5-104">下列幾行推論說明無法在 `NotOverridable` 中搭配使用 `Private` 和 [Private](../../visual-basic/language-reference/statements/property-statement.md)的原因：</span><span class="sxs-lookup"><span data-stu-id="4d2d5-104">The following line of reasoning explains why `NotOverridable` and `Private` cannot be combined in a [Property Statement](../../visual-basic/language-reference/statements/property-statement.md):</span></span>  
  
1.  <span data-ttu-id="4d2d5-105">未覆寫基底類別屬性或程序的屬性或程序具有預設值 [NotOverridable](../../visual-basic/language-reference/modifiers/notoverridable.md)。</span><span class="sxs-lookup"><span data-stu-id="4d2d5-105">A property or procedure that does not override a base class property or procedure has a default setting of [NotOverridable](../../visual-basic/language-reference/modifiers/notoverridable.md).</span></span>  
  
2.  <span data-ttu-id="4d2d5-106">但是，覆寫基底類別屬性或程序之衍生類別中的屬性或程序具有預設值 [Overridable](../../visual-basic/language-reference/modifiers/overridable.md)。</span><span class="sxs-lookup"><span data-stu-id="4d2d5-106">However, a property or procedure in a derived class that overrides a base class property or procedure has a default setting of [Overridable](../../visual-basic/language-reference/modifiers/overridable.md).</span></span> <span data-ttu-id="4d2d5-107">若要終止覆寫的階層架構，您可以將它宣告為 `NotOverridable`。</span><span class="sxs-lookup"><span data-stu-id="4d2d5-107">To terminate the hierarchy of overriding, you can declare it `NotOverridable`.</span></span> <span data-ttu-id="4d2d5-108">這是您可以使用 `NotOverridable`的唯一內容。</span><span class="sxs-lookup"><span data-stu-id="4d2d5-108">This is the only context in which you can use `NotOverridable`.</span></span> <span data-ttu-id="4d2d5-109">也就是說，您只能搭配 `NotOverridable` 使用 [NotOverridable](../../visual-basic/language-reference/modifiers/overrides.md)。</span><span class="sxs-lookup"><span data-stu-id="4d2d5-109">That is, you can use `NotOverridable` only in combination with [Overrides](../../visual-basic/language-reference/modifiers/overrides.md).</span></span>  
  
3.  <span data-ttu-id="4d2d5-110">如果基底類別屬性或程序宣告為 [Private](../../visual-basic/language-reference/modifiers/private.md)，衍生的類別就不能覆寫該屬性或程序，因為無法存取它。</span><span class="sxs-lookup"><span data-stu-id="4d2d5-110">If a base class property or procedure is declared [Private](../../visual-basic/language-reference/modifiers/private.md), a derived class cannot override that property or procedure because it cannot access it.</span></span> <span data-ttu-id="4d2d5-111">因此，您無法搭配 `Private` 使用 `Overridable`。</span><span class="sxs-lookup"><span data-stu-id="4d2d5-111">Because of this, you cannot use `Private` in combination with `Overridable`.</span></span>  
  
4.  <span data-ttu-id="4d2d5-112">若要覆寫屬性或程序，覆寫屬性或程序不僅必須具有完全相同的簽章，也必須具有相同的存取層級。</span><span class="sxs-lookup"><span data-stu-id="4d2d5-112">To override a property or procedure, the overriding property or procedure must have not only the identical signature but also the same access level.</span></span> <span data-ttu-id="4d2d5-113">這表示覆寫屬性或程序無法指定 `Private`，因為可覆寫的屬性或程序無法指定 `Private`。</span><span class="sxs-lookup"><span data-stu-id="4d2d5-113">This means that an overriding property or procedure cannot specify `Private`, because an overridable property or procedure cannot specify `Private`.</span></span>  
  
5.  <span data-ttu-id="4d2d5-114">由於您只能在覆寫屬性或程序上指定 `NotOverridable` ，因此無法搭配 `Private`使用。</span><span class="sxs-lookup"><span data-stu-id="4d2d5-114">Because you can specify `NotOverridable` only on an overriding property or procedure, you cannot combine it with `Private`.</span></span>  
  
 <span data-ttu-id="4d2d5-115">同理可證，覆寫屬性的個別屬性程序 (`Get` 和 `Set`) 不可為 `Private`。</span><span class="sxs-lookup"><span data-stu-id="4d2d5-115">By the same reasoning, the individual property procedures (`Get` and `Set`) of an overriding property cannot be `Private`.</span></span>
