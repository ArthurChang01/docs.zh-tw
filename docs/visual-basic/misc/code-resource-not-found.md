# <a name="code-resource-not-found"></a>找不到程式碼資源
已呼叫程式碼資源中的程序，但找不到該程式碼資源。  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請檢查資源可用且已正確地參考。
