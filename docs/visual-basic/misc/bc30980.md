<span data-ttu-id="2160d-103">如果變數用於在宣告中建立其初始值，則編譯器無法推斷變數的資料類型。</span><span class="sxs-lookup"><span data-stu-id="2160d-103">The compiler cannot infer the data type of a variable if the variable is used in establishing its initial value in the declaration.</span></span>

    Dim d = someFunction(d)  
    ```  
  
- <span data-ttu-id="2160d-106">`For` 迴圈</span><span class="sxs-lookup"><span data-stu-id="2160d-106">`For` loop</span></span>  
  
    ```  
    ' Does not compile with Option Infer on.

    Next  
    ```  
  
- <span data-ttu-id="2160d-107">`For Each` 迴圈</span><span class="sxs-lookup"><span data-stu-id="2160d-107">`For Each` loop</span></span>

    ' Does not compile with Option Infer on.

    Next  
    ```
