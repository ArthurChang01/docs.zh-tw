# <a name="type-of-variablename-cannot-be-inferred-from-an-expression-containing-variablename"></a>無法從包含 ' \<variablename > ' 的運算式推斷 ' @no__t 0variablename > ' 的類型
如果變數用於在宣告中建立其初始值，則編譯器無法推斷變數的資料類型。  
  
 例如，如果 `Option Infer` 設定為 `On`，則下列範例不會編譯：  
  
- 宣告  
  
    ```vb  
    ' Does not compile with Option Infer on.  
    Dim m = m  
    Dim d = someFunction(d)  
    ```  
  
- `For` 迴圈  
  
    ```vb  
    ' Does not compile with Option Infer on.  
    For j = 1 To j  
    Next  
    ```  
  
- `For Each` 迴圈  
  
    ```vb  
    ' Does not compile with Option Infer on.  
    For Each customer In customer.Orders  
    Next  
    ```  
  
 **錯誤識別碼：** BC30980  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 如果兩個變數是要用來參考不同的值，請變更您所宣告變數的名稱。  
  
- 在指派的右側，使用常值，而非初始值中的變數名稱。  
  
- 使用 `As` 子句來指定所宣告變數的類型：
