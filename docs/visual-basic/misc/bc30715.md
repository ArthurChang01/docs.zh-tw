# <a name="39catch39-statements-are-not-valid-in-the-immediate-window"></a>'Catch' 陳述式不是有效的即時運算視窗中
`Catch`陳述式僅能在原始程式碼中。  
  
 **錯誤 ID︰** BC30715  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請從偵錯程式碼中移除 `Catch` 陳述式。
