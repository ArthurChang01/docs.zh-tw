# <a name="lttype1gt-39lttypenamegt39-and-lttype2gt-39lttypenamegt39-conflict-in-ltnamespacegt-39ltnamespacenamegt39"></a>&lt;type1&gt; &#39; &lt;typename&gt; &#39;和&lt;type2&gt; &#39; &lt;typename&gt; &#39;衝突中&lt;命名空間&gt; &#39;&lt;命名空間名稱&gt;&#39;
兩個宣告的項目 (例如類別和模組) 在相同的命名空間中同名。  
  
 **錯誤 ID：** BC30179  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   將其中一個項目的宣告移至不同的命名空間。  
  
-   變更其中一個項目的名稱。
