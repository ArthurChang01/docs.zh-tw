# <a name="lttype1gt-lttypenamegt-and-lttype2gt-lttypenamegt-conflict-in-ltnamespacegt-ltnamespacenamegt"></a>&lt;type1&gt; '&lt;typename&gt;' 並&lt;type2&gt; '&lt;typename&gt;' 中產生衝突&lt;命名空間&gt;'&lt;命名空間名稱&gt;'
兩個宣告的項目 (例如類別和模組) 在相同的命名空間中同名。  
  
 **錯誤 ID:** BC30179  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   將其中一個項目的宣告移至不同的命名空間。  
  
-   變更其中一個項目的名稱。
