# <a name="overload-resolution-failed-because-no-accessible-39ltmethodgt39-accepts-this-number-of-arguments"></a>多載解析失敗，因為沒有可存取 &#39;&lt;方法&gt;&#39; 接受此數目的引數
您已呼叫多載的方法，但編譯器找不到具有相符引數清單的方法。  
  
 **錯誤 ID︰** BC30516  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請檢查您提供的引數清單。
