# <a name="argument-value-ltpathnamegt-contains-characters-that-are-not-valid-in-a-path-name"></a>引數的值 '&lt;pathname&gt;' 包含不是有效的路徑名稱的字元
路徑名稱包含一個或多個無效字元。  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請從路徑名稱中移除該字元。
