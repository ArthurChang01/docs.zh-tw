# <a name="argument-value-39ltpathnamegt39-contains-characters-that-are-not-valid-in-a-path-name"></a>引數的值 &#39;&lt;pathname&gt;&#39; 包含在路徑名稱中無效的字元
路徑名稱包含一個或多個無效字元。  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請從路徑名稱中移除該字元。
