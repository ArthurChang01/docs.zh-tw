# <a name="unable-to-open-response-file-39ltfilenamegt39"></a>無法開啟回應檔 &#39;&lt;filename&gt;&#39;
無法使用您嘗試存取的檔案。  
  
 **錯誤 ID︰** BC2011  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請確定您的網路連線仍在作用中。  
  
-   請檢查您對檔案的權限。
