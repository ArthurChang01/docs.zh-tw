# <a name="ltmembername1gt-and-ltmembername2gt-cannot-overload-each-other-because-they-differ-only-by-the-types-of-optional-parameters"></a>'&lt;1&gt&gt;'和'&lt;成員名稱 2>&gt;' 無法互相多載，因為它們的差異只在選擇性參數類型不同
多載成員的一或多個必要參數必須具有不同的資料類型。  
  
 **錯誤 ID:** BC30696  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請至少將一個必要的參數定義為不同的資料類型。
