# <a name="39inherits39-statement-must-precede-all-declarations-in-a-class"></a>&#39;繼承&#39;陳述式必須在所有宣告都之前類別中
`Inherits` 陳述式必須位在類別定義的開頭。  
  
 **錯誤識別碼：** BC30683  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   將 `Inherits` 陳述式移至類別的頂端。
