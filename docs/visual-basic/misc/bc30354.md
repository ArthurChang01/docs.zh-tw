# <a name="interface-can-inherit-only-from-another-interface"></a>介面只可以繼承自另一個介面
您嘗試繼承自不是介面的項目。  
  
 **錯誤 ID:** BC30354  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   確定您嘗試繼承的一或多個基底介面確實是介面。
