# <a name="xml-literals-and-xml-axis-properties-are-not-available"></a>無法使用 XML 常值和 XML 軸屬性
無法使用 XML 常值和 XML 軸屬性。 請加入 System.Xml、System.Xml.Linq 和 System.Core 的參考。  
  
 要編譯的程式碼包括 XML 常值或 XML 軸屬性，但未參考編譯 XML 常值或 XML 軸屬性所需的組件。  
  
 **錯誤 ID:** BC31190  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   加入 System.Xml.dll、System.Xml.Linq.dll 和 System.Core.dll 組件的參考。
