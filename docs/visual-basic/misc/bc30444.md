# <a name="39set39-is-already-declared"></a>&#39;設定 &#39;已宣告
`Set` 陳述式在程式碼區塊中出現一次以上。 `Set` 陳述式會宣告用來指派屬性值的 `Set` 屬性程序。  
  
 **錯誤 ID︰** BC30444  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請移除多餘的 `Set` 陳述式。
