# <a name="information-for-the-type-of-typename-has-not-been-loaded-into-the-runtime"></a>'\<typename > ' 類型的資訊尚未載入至執行時間
參考了執行階段尚未載入的類型。  
  
 **錯誤 ID︰** BC30750  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
1. 請重組您的程式碼，以在目前範圍內定義類型。  
  
2. 請確認已定義成員，而且成員名稱拼字正確。  
  
3. 嘗試存取模組中所宣告的其中一個成員。 在某些情況下，偵錯環境找不到成員，因為未載入宣告它們的模組。
