# <a name="unable-to-find-entry-point-name-in-dll-filename-error"></a>找不到進入點 '\<名稱 >' dll'\<檔案名稱 >':\<錯誤 >
在指定的 DLL 中找不到程序名稱。  
  
 **錯誤 ID:** BC31014  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
1. 確認程序名稱拼字正確，而且大小寫符合 DLL 中函式的大小寫。
