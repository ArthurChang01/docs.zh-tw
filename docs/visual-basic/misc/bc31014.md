# <a name="unable-to-find-entry-point-ltnamegt-in-dll-ltfilenamegt-lterrorgt"></a>找不到進入點 '&lt;名稱&gt;'在 DLL'&lt;filename&gt;':&lt;錯誤&gt;
在指定的 DLL 中找不到程序名稱。  
  
 **錯誤 ID:** BC31014  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
1.  確認程序名稱拼字正確，而且大小寫符合 DLL 中函式的大小寫。
