# <a name="enums-must-be-declared-as-an-integral-type"></a>列舉必須宣告為整數類型
您可以在列舉中使用的唯一有效類型是 `SByte`、 `Byte`、 `Short`、 `UShort`、 `Integer`、 `UInteger`、 `Long`和 `ULong`。 不可以使用任何其他資料類型。  
  
 **錯誤 ID：** BC30650  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   指定的資料類型為`SByte`， `Byte`， `Short`， `UShort`， `Integer`， `UInteger`， `Long`，或`ULong`。
