# <a name="39custom39-modifier-can-only-be-used-immediately-before-an-39event39-declaration"></a>'Custom' 修飾詞只用於之前 'Event' 宣告
在自訂事件宣告中， `Custom` 關鍵字必須在 `Event` 關鍵字前面。  
  
 **錯誤 ID︰** BC31140  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請將 `Custom` 關鍵字移到 `Event` 關鍵字前面。  
  
     -或-  
  
-   請移除 `Custom` 關鍵字。
