# <a name="no-accessible-overloaded-39ltmethodnamegt39-can-be-called-with-these-arguments-without-a-widening-conversion-ltlistgt"></a>沒有可存取多載 &#39;&lt;methodname&gt;&#39; 可以擴展轉換這些引數呼叫：&lt;清單&gt;
已呼叫多載方法，但沒有任何方法可在未擴展轉換的情況下，符合所提供的引數清單。  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請指定 `Option Strict Off`。  
  
-   變更引數，以符合多載方法的其中一個特徵標記。
