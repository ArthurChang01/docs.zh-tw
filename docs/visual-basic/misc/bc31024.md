# <a name="unable-to-create-a-net-runtime-interface-error"></a>無法建立.NET Runtime 介面：\<錯誤 >
建立介面時發生編譯器內部錯誤，或介面上的方法呼叫失敗。  
  
 **錯誤 ID:** BC31024  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
1.  儲存工作並重新啟動 Visual Studio。  
  
2.  如果錯誤重複發生，請重新安裝 Visual Studio。  
  
3.  如果錯誤在重新安裝之後持續發生，請通知 Microsoft 產品支援服務。
