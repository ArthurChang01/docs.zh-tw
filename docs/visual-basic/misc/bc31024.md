# <a name="unable-to-create-a-net-runtime-interface-lterrorgt"></a>無法建立.NET Runtime 介面：&lt;錯誤&gt;
建立介面時發生編譯器內部錯誤，或介面上的方法呼叫失敗。  
  
 **錯誤 ID︰** BC31024  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
1.  儲存您的工作，並重新啟動 [!INCLUDE[vsprvs](~/includes/vsprvs-md.md)]。  
  
2.  如果問題重複發生，請重新安裝 [!INCLUDE[vsprvs](~/includes/vsprvs-md.md)]。  
  
3.  如果錯誤在重新安裝之後持續發生，請通知 Microsoft 產品支援服務。
