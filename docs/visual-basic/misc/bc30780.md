# <a name="continue-statements-are-not-valid-in-the-immediate-window"></a>即時運算視窗中的 'Continue' 陳述式無效
'Continue' 陳述式只能用於原始程式碼中。  
  
 **錯誤 ID:** BC30780  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   移除偵錯的程式碼中的 'Continue' 陳述式。
