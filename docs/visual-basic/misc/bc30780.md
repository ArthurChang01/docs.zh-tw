# 即時運算視窗中的 &#39;Continue&#39; 陳述式無效
`Continue` 陳述式只能用於原始程式碼中。  
  
 **錯誤 ID︰**BC30780  
  
### 更正這個錯誤  
  
-   從偵錯程式碼中移除 `Continue` 陳述式。
