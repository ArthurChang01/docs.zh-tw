# <a name="39continue39-statements-are-not-valid-in-the-immediate-window"></a>'Continue' 陳述式不是有效的即時運算視窗中
`Continue`陳述式僅能在原始程式碼中。  
  
 **錯誤 ID︰** BC30780  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   從偵錯程式碼中移除 `Continue` 陳述式。
