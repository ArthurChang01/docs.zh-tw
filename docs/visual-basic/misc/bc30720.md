# <a name="on-error-statements-are-not-valid-in-the-immediate-window"></a>[即時運算] 視窗中的 'On Error' 陳述式無效
`On Error` 陳述式只能用於原始程式碼中。  
  
 **錯誤 ID︰** BC30720  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 請從偵錯程式碼中移除 `On Error` 陳述式。
