# [即時運算] 視窗中的 &#39;On Error&#39; 陳述式無效
`On Error` 陳述式只能用於原始程式碼中。  
  
 **錯誤 ID︰**BC30720  
  
### 更正這個錯誤  
  
-   請從偵錯程式碼中移除 `On Error` 陳述式。
