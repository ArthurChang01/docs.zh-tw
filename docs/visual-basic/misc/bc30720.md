# <a name="39on-error39-statements-are-not-valid-in-the-immediate-window"></a>' Error' 陳述式不是有效的即時運算視窗中
`On Error`陳述式僅能在原始程式碼中。  
  
 **錯誤 ID︰** BC30720  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請從偵錯程式碼中移除 `On Error` 陳述式。
