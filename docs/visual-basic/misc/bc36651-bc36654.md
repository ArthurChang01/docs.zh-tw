<span data-ttu-id="07360-103">在方法中的型別參數的資料型別 '\<方法名稱 >' 無法從這些引數推斷，因為可能會有一個以上的類型。</span><span class="sxs-lookup"><span data-stu-id="07360-103">Data type(s) of the type parameter(s) in method '\<methodname>' cannot be inferred from these arguments because more than one type is possible.</span></span> <span data-ttu-id="07360-104">明確指定資料類型或許可以更正這個錯誤。</span><span class="sxs-lookup"><span data-stu-id="07360-104">Specifying the data type(s) explicitly might correct this error.</span></span>
