# <a name="data-types-of-the-type-parameters-in-method-methodname-cannot-be-inferred-from-these-arguments-because-more-than-one-type-is-possible"></a><span data-ttu-id="3d24e-102">無法從這些引數推斷方法 '\<方法名稱 > ' 中類型參數的資料類型，因為可能會有一個以上的類型</span><span class="sxs-lookup"><span data-stu-id="3d24e-102">Data type(s) of the type parameter(s) in method '\<methodname>' cannot be inferred from these arguments because more than one type is possible</span></span>

<span data-ttu-id="3d24e-103">無法從這些引數推斷方法 '\<方法名稱 > ' 中類型參數的資料類型，因為可能會有一個以上的類型。</span><span class="sxs-lookup"><span data-stu-id="3d24e-103">Data type(s) of the type parameter(s) in method '\<methodname>' cannot be inferred from these arguments because more than one type is possible.</span></span> <span data-ttu-id="3d24e-104">明確指定資料類型或許可以更正這個錯誤。</span><span class="sxs-lookup"><span data-stu-id="3d24e-104">Specifying the data type(s) explicitly might correct this error.</span></span>

<span data-ttu-id="3d24e-105">您嘗試使用類型推斷來判斷泛型程序呼叫中的類型參數的類型。</span><span class="sxs-lookup"><span data-stu-id="3d24e-105">An attempt has been made to use type inference to determine the type or types of the type parameter or parameters in a call to a generic procedure.</span></span> <span data-ttu-id="3d24e-106">編譯器發現一或多個類型參數的多個可能資料類型，並報告這個錯誤。</span><span class="sxs-lookup"><span data-stu-id="3d24e-106">The compiler finds more than one possible data type for one or more of the type parameters, and it reports this error.</span></span>

> [!NOTE]
> <span data-ttu-id="3d24e-107">當無法指定引數時 (例如，針對查詢運算式中的查詢運算子)，會顯示不含第二句的錯誤訊息。</span><span class="sxs-lookup"><span data-stu-id="3d24e-107">When specifying arguments is not an option (for example, for query operators in query expressions), the error message appears without the second sentence.</span></span>

<span data-ttu-id="3d24e-108">下列程式碼示範這個錯誤。</span><span class="sxs-lookup"><span data-stu-id="3d24e-108">The following code demonstrates the error.</span></span>

```vb
Option Strict Off
Module Module1
    Sub Main()
        '' Not valid.
        'targetMethod(1, "2")
    End Sub

    Sub targetMethod(Of T)(ByVal p1 As T, ByVal p2 As T)
    End Sub

End Module
```

<span data-ttu-id="3d24e-109">**錯誤識別碼：** BC36654 （在 LINQ 查詢中）和 BC36651 （在查詢之外）</span><span class="sxs-lookup"><span data-stu-id="3d24e-109">**Error ID:** BC36654 (within LINQ queries) and BC36651 (outside queries)</span></span>
