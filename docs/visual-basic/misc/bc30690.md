# <a name="structure-39ltstructurenamegt39-cannot-be-indexed-because-it-has-no-default-property"></a>結構&#39; &lt;structurename&gt; &#39;無法索引，因為它沒有預設屬性
嘗試使用結構沒有預設屬性的預設屬性語法。  
  
 **錯誤 ID︰** BC30690  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請使用標準屬性語法來存取結構的屬性。
