# <a name="array-initializer-has-too-few-dimensions"></a>陣列初始設定式擁有太少維度
您尚未在初始設定式中提供陣列的足夠維度。  
  
 **錯誤 ID︰** BC30565  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請檢查您的陣列初始設定式，以判斷需要多少維度。  
  
-   請提供遺漏的引數。
