# <a name="non-intrinsic-type-names-are-not-allowed-in-conditional-compilation-expressions"></a>條件式編譯運算式中不允許非內建的類型名稱
條件式編譯運算式參考非內建的類型，但只允許內建類型。  
  
 如需有關內建函式的型別中[!INCLUDE[vbprvb](../../csharp/programming-guide/concepts/linq/includes/vbprvb_md.md)]，請參閱[資料類型摘要](../../visual-basic/language-reference/keywords/data-types-summary.md)。  
  
 **錯誤識別碼：** BC31426  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   以內建類型取代非內建類型。
