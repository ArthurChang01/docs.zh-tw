# <a name="39withevents39-variable-does-not-raise-any-events"></a>&#39;WithEvents&#39;變數沒有引發任何事件
使用 `WithEvents` 關鍵字所宣告的物件變數未參照可引發事件的類別執行個體 。  
  
 **錯誤 ID︰** BC30591  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
1.  請確定物件變數參照可引發事件的類別執行個體。
