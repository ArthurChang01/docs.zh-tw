<span data-ttu-id="8644a-103">在匿名類型屬性的類型建立之前，它不能用以建立另一個屬性的類型。</span><span class="sxs-lookup"><span data-stu-id="8644a-103">Until the type of an anonymous type property is established, it cannot be used to establish the type of another property.</span></span> <span data-ttu-id="8644a-104">例如， `.IDName = .LastName` 在下列宣告中是無效的，因為 `.LastName` 尚未初始化。</span><span class="sxs-lookup"><span data-stu-id="8644a-104">For example, in the following declaration `.IDName = .LastName` is not valid because `.LastName` has not yet been initialized.</span></span>  
  
```  
' Not valid.   
' Dim anon1 = New With {Key .IDName = .LastName, Key .LastName = "Jones"}   
```
