# <a name="referenced-array-element-has-a-value-of-nothing"></a>所參考陣列項目的值為 'Nothing'
尚未設定所參考的陣列項目，或先前設定為 `Nothing`。  
  
 **錯誤 ID:** BC30744  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請確認已正確地設定陣列中的值。
