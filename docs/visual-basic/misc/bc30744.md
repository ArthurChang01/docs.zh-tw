# <a name="referenced-array-element-has-a-value-of-39nothing39"></a>參考的陣列元素的值為&#39;做任何動作&#39;
尚未設定所參考的陣列項目，或先前設定為 `Nothing`。  
  
 **錯誤 ID︰** BC30744  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請確認已正確地設定陣列中的值。
