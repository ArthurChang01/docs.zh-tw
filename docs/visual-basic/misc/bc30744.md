# <a name="referenced-array-element-has-a-value-of-39nothing39"></a>所參考的陣列項目具有的值 &#39;執行任何動作 &#39;
尚未設定所參考的陣列項目，或先前設定為 `Nothing`。  
  
 **錯誤 ID︰** BC30744  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請確認已正確地設定陣列中的值。
