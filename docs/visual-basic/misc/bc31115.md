# <a name="39addhandler39-declaration-must-end-with-a-matching-39end-addhandler39"></a>&#39;AddHandler &#39;宣告的結尾必須符合 &#39;End AddHandler &#39;
`AddHandler` 宣告的結尾必須是 `End AddHandler` 陳述式。  
  
 **錯誤 ID︰** BC31115  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請檢查 `AddHandler` 宣告的結尾是否為 `End AddHandler` 陳述式。
