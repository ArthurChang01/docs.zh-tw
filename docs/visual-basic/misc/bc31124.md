# <a name="39end-addhandler39-must-be-preceded-by-a-matching-39addhandler39-declaration"></a>&#39;End AddHandler &#39;之前必須搭配相對應的 &#39;AddHandler &#39;宣告
`End AddHandler` 陳述式沒有相對應的 `AddHandler` 陳述式。 `End AddHandler` 之前必須搭配相對應的 `AddHandler` 陳述式。  
  
 **錯誤 ID︰** BC31124  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請檢查前面的 `AddHandler` 陳述式有效而且拼寫正確。
