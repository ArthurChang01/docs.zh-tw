# <a name="argument-39ltargumentnamegt39-cannot-be-converted-to-type-39date39"></a>引數 &#39;&lt;引數名稱&gt;&#39; 無法轉換為類型 &#39; 日期 &#39;
轉換嘗試將變數變更為類型 `Date`，但未將此值辨識為有效的日期。  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請尋找有意義的轉換。
