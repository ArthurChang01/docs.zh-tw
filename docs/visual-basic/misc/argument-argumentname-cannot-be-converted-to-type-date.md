# <a name="argument-argumentname-cannot-be-converted-to-type-date"></a>引數 '\<引數名稱 >' 無法轉換成類型 'Date'
轉換嘗試將變數變更為類型 `Date`，但未將此值辨識為有效的日期。  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請尋找有意義的轉換。
