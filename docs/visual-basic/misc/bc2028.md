# <a name="option-39ltoptiongt39-cannot-be-followed-by-3939-or-39-39-ignored"></a>選項&#39;&lt;選項&gt;&#39;後面不能&#39;+&#39;或&#39;-&#39;; 已忽略
編譯器選項已指定值 `+` 或 `-`不支援這些值。  
  
 如需編譯器選項和其所支援值的詳細資訊，請參閱 [Visual Basic Compiler Options Listed Alphabetically](../../visual-basic/reference/command-line-compiler/compiler-options-listed-alphabetically.md)。  
  
 **錯誤 ID︰** BC2028  
  
## <a name="to-correct-this-error-and-use-the-option"></a>更正這個錯誤並使用選項  
  
-   指定沒有任何值的編譯器選項。  
  
## <a name="to-correct-this-error-and-not-use-the-option"></a>更正這個錯誤但不使用選項  
  
-   移除編譯器選項。
