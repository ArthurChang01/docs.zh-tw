# <a name="only-one-of-39public39-39private39-39protected39-39friend39-or-39protected-friend39-can-be-specified"></a>只有其中一個&#39;公用&#39;，&#39;私人&#39;，&#39;保護&#39;， &#39;Friend&#39;，或&#39;Protected Friend&#39;可以指定
項目宣告中發生協助工具衝突。 只能指定一個協助工具。  
  
 **錯誤 ID：** BC30176  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請選擇一個協助工具，並移除宣告中的其他關鍵字。
