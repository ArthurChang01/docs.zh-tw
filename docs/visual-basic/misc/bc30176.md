# <a name="only-one-of-public-private-protected-friend-or-protected-friend-can-be-specified"></a>只能指定 'Public'、'Private'、'Protected'、'Friend' 或 'Protected Friend' 其中之一
項目宣告中發生協助工具衝突。 只能指定一個協助工具。  
  
 **錯誤 ID:** BC30176  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 請選擇一個協助工具，並移除宣告中的其他關鍵字。
