# 只能指定 &#39;Public&#39;、&#39;Private&#39;、&#39;Protected&#39;、&#39;Friend&#39; 或 &#39;Protected Friend&#39; 其中之一
項目宣告中發生協助工具衝突。 只能指定一個協助工具。  
  
 **錯誤 ID：**BC30176  
  
### 更正這個錯誤  
  
-   請選擇一個協助工具，並移除宣告中的其他關鍵字。
