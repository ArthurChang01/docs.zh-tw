# <a name="label-labelname-is-already-defined-in-the-current-methodmultiline-lambda"></a>標籤 '\<標籤名稱 >' 已經定義在目前的方法/多行 lambda 中
標籤名稱會在程序中重複。 程序內的所有名稱必須都是唯一的。  
  
 **錯誤 ID:** BC30094  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
1.  請重新命名重複的標籤。
