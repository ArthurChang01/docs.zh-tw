# <a name="the-maximum-number-of-errors-has-been-exceeded"></a>已經超過錯誤最大數目
已達到 100 個錯誤的編譯器上限。  
  
 **錯誤 ID︰** BC2020  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請透過偵錯和依次處理每個錯誤，來減少所產生的錯誤數目。
