# <a name="lttype1gt-39lttypename1gt39-conflicts-with-a-member-implicitly-declared-for-event-39lteventnamegt39-in-lttype2gt-39lttypename2gt39"></a>&lt;type1&gt; &#39;&lt;typename1&gt;&#39; 隱含宣告的事件 &#39; 的成員互相衝突&lt;eventname&gt;&#39; 在&lt;type2&gt; &#39;&lt;2>&gt;&#39;
類型成員名稱與為事件隱含建立的成員名稱相衝突。 事件會隱含建立數個隱含變數。 例如，宣告 `Event X` 會隱含宣告名稱 `XEventHandler`、 `XEvent`、 `add_X`和 `remove_X`。  
  
 **錯誤 ID︰** BC31061  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請重新命名明確宣告的成員，以移除命名衝突。
