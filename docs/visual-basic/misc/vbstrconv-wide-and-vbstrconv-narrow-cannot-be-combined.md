# 無法合併 VbStrConv.Wide 和 VbStrConv.Narrow
您的應用程式嘗試合併 `VbStrConv` 列舉成員 `Wide` 和 `Narrow`，但它們互斥。  
  
### 更正這個錯誤  
  
1.  請移除 `VbStrConv.Wide` 或 `VbStrConv.Narrow`。
