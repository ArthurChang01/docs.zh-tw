# <a name="39for39-loop-control-variable-already-in-use-by-an-enclosing-39for39-loop"></a>&#39;如&#39;迴圈控制變數已在使用之封入&#39;如&#39;迴圈
巢狀 `For` 迴圈必須全部使用唯一的控制變數。  
  
 **錯誤 ID︰** BC30975  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   變更其中一個控制變數名稱。
