# <a name="reference-required-to-module-ltmodulenamegt-containing-the-implemented-interface-ltinterfacenamegt"></a>需要模組參考 '&lt;modulename&gt;'包含實作的介面'&lt;interfacename&gt;'
需要模組參考 '\<modulename >' 包含實作的介面'\<介面名稱 >'。 請在專案中加入一個參考。  
  
 此介面是在專案中未直接參考的模組中所定義。 Visual Basic 編譯器需要參考，以避免發生模稜兩可的情況的介面定義一個以上的模組中。  
  
 **錯誤 ID:** BC30010  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請在您的專案參考中包含未參考之模組的名稱。
