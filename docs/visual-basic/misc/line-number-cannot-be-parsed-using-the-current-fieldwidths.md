# <a name="line-number-cannot-be-parsed-using-the-current-fieldwidths"></a>行\<數字 > 無法使用目前的 FieldWidths 剖析
無法剖析指定的行，因為其欄位的寬度不是指定的寬度。  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 調整 `FieldWidths` 以便正確地剖析此行，或插入例外狀況處理程式碼以便處理此行。
