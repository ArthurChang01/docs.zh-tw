# <a name="conversion-operators-cannot-convert-from-a-type-to-its-base-type"></a>轉換運算子無法從類型轉換為其基底類型
轉換運算子是以參數類型所衍生自的傳回類型進行宣告。  
  
 在編譯時期，Visual Basic 會考慮存在從任何參考類型到其繼承階層架構中，也就是任何類型從它衍生或從中衍生的任何類型的預先定義的轉換。 這類轉換可能會在執行階段失敗，但編譯器無法預測執行階段結果，因此它可以編譯任何這類轉換。  
  
 因為編譯器認為已定義這項轉換，所以不允許您重新定義它。  
  
 **錯誤 ID︰** BC33026  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請完整移除這個運算子定義。 它已預先定義。
