# <a name="xml-literals-and-xml-axis-properties-are-not-available-during-this-debugging-session-because-they-are-not-used-in-this-assembly"></a>這個偵錯工作階段中無法使用 XML 常值和 XML 軸屬性，因為它們不用在這個組件中
XML 常值或 XML 軸屬性已被參照在**監看式**或**即時運算** 視窗中的 xml 中的偵錯工作階段[!INCLUDE[vbprvb](../../csharp/programming-guide/concepts/linq/includes/vbprvb_md.md)]無法使用的功能。 這是 未使用中的 XML 組件的情況[!INCLUDE[vbprvb](../../csharp/programming-guide/concepts/linq/includes/vbprvb_md.md)]功能或已發行的組建。  
  
 **錯誤 ID︰** BC31196  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請使用組件的偵錯組建來啟動新的偵錯工作階段。
