# <a name="39exit39-statements-are-not-valid-in-the-immediate-window"></a>'結束' 陳述式不是有效的即時運算視窗中
`Exit`陳述式僅能在原始程式碼中。  
  
 **錯誤 ID︰** BC30713  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請從偵錯程式碼中移除 `Exit` 陳述式。
