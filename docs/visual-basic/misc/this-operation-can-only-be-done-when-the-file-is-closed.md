# <a name="this-operation-can-only-be-done-when-the-file-is-closed"></a>只有在 File 關閉時，才能執行這項作業
開啟檔案時，嘗試執行的作業只能在關閉檔案時執行。  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請關閉檔案，然後再次嘗試這項作業。
