# <a name="39before39-and-39after39-arguments-cannot-be-combined"></a>&#39;之前 &#39;和 &#39;之後 &#39;引數不能合併
引數清單同時包含互斥的 `Before` 和 `After` 引數。  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
1.  移除 `Before` 或 `After` 引數。
