# <a name="before-and-after-arguments-cannot-be-combined"></a>無法合併 'Before' 和 'After' 引數
引數清單同時包含互斥的 `Before` 和 `After` 引數。  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
1. 移除 `Before` 或 `After` 引數。
