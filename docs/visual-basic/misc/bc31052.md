# <a name="modules-cannot-be-declared-specifier"></a>模組不可以宣告為 '\<規範 >'
已使用無效的存取修飾詞宣告模組。  
  
 **錯誤 ID:** BC31052  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請將存取修飾詞取代為有效的存取修飾詞 (例如 `Public` 或 `Friend`)。
