# <a name="modules-cannot-be-declared-39ltspecifiergt39"></a>模組不可以宣告 &#39;&lt;規範&gt;&#39;
已使用無效的存取修飾詞宣告模組。  
  
 **錯誤 ID︰** BC31052  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請將存取修飾詞取代為有效的存取修飾詞 (例如 `Public` 或 `Friend`)。
