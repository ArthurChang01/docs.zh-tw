# <a name="format-not-valid-in-resource-file"></a>資源檔中的格式無效
您的專案中的資源檔不符合 Windows 資源檔的標準格式，或是有不相符的資源檔和 Visual Basic 專案。  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
1.  請重新建立資源來源和資源檔，並遵循 Windows 軟體開發套件中所述的語法。  
  
2.  請從專案移除無效的資源檔。
