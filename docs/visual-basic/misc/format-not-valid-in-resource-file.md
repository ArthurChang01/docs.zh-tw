# <a name="format-not-valid-in-resource-file"></a>資源檔中的格式無效
您的專案中的資源檔不符合標準的 Windows 資源檔格式，或有不相符的資源檔和[!INCLUDE[vbprvb](../../csharp/programming-guide/concepts/linq/includes/vbprvb_md.md)]專案。  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
1.  請重新建立資源來源和資源檔，並遵循 Windows 軟體開發套件中所述的語法。  
  
2.  請從專案移除無效的資源檔。
