# <a name="new-cannot-be-used-on-class-classname-because-it-contains-a-mustoverride-member-that-has-not-been-overridden"></a>'New' 不能在類別\<類別名稱 >' 因為它包含一個尚未被覆寫的 'MustOverride' 成員
您已嘗試在包含尚未覆寫之 `New` 成員的類別上使用 `MustOverride` 。  
  
 **錯誤 ID:** BC30376  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 請移除 `New` 陳述式。
