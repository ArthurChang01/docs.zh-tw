# <a name="39new39-cannot-be-used-on-class-39ltclassnamegt39-because-it-contains-a-39mustoverride39-member-that-has-not-been-overridden"></a>&#39;新 &#39;不能在類別 &#39;&lt;classname&gt;&#39;，因為它包含 &#39;MustOverride &#39;尚未被覆寫的成員
您已嘗試在包含尚未覆寫之 `New` 成員的類別上使用 `MustOverride` 。  
  
 **錯誤 ID︰** BC30376  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請移除 `New` 陳述式。
