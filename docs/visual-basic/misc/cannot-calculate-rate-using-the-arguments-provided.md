# <a name="cannot-calculate-rate-using-the-arguments-provided"></a>無法使用提供的引數來計算比率
比率的計算不包含必要的引數。  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
1.  請檢查運算式中的引數拼法。 拼錯的變數名稱可能會隱含地建立初始化為零的數值變數。  
  
2.  請檢查運算式中先前的變數作業，特別是那些作為引數從其他程序傳遞至程序的變數。
