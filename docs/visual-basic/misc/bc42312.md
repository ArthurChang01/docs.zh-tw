# <a name="xml-comment-block-cannot-be-associated-with-any-language-element-that-supports-the-application-of-xml-documentation-comments"></a>XML 註解區塊不能與任何支援 XML 文件註解之應用程式的語言項目相關聯
XML 註解區塊不能與任何有效的語言項目相關聯。 將會忽略 XML 註解。 如果註解區塊中的 XML 標記沒有相符的語言項目，也會發生這個錯誤。  
  
 **錯誤 ID:** BC42312  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 尋找並移除多餘的 XML 標記，或提供它所描述的語言項目。
