# <a name="type-typename-is-imported-from-different-versions-of-assembly-assemblyname"></a>類型 '\<類型名稱 >' 從不同版本的組件匯入'\<組件名稱 >'
類型 '\<類型名稱 >' 從不同版本的組件匯入'\<組件名稱 >'。 相同組件的不同版本不可用於相同專案中。  
  
 類型已從相同組件的不同版本匯入，因而產生模稜兩可。  
  
 **錯誤 ID:** BC31525  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請移除相同組件之多個版本的參考。
