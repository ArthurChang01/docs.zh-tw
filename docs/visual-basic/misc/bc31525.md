# <a name="type-39lttypenamegt39-is-imported-from-different-versions-of-assembly-39ltassemblynamegt39"></a>類型 '&lt;typename&gt;'匯入從不同版本的組件'&lt;assemblyname&gt;'
類型 '\<typename >' 從不同版本的組件匯入'\<assemblyname >'。 相同組件的不同版本不可用於相同專案中。  
  
 類型已從相同組件的不同版本匯入，因而產生模稜兩可。  
  
 **錯誤 ID︰** BC31525  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請移除相同組件之多個版本的參考。
