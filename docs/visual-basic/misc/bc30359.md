# <a name="default-can-be-applied-to-only-one-property-name-in-a-type"></a>可以套用至中只有一個屬性名稱的 'Default'\<類型 >
您嘗試將 `Default` 套用至類別、介面或結構內的多個屬性名稱。  
  
 **錯誤 ID:** BC30359  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 請移除全部 `Default` 宣告，只留下其中一個宣告。
