# <a name="enum-enumname-must-contain-at-least-one-member"></a>列舉 '\<列舉名稱 >' 必須包含至少一個成員
發生空的列舉。 在 `Enum` 陳述式與 `End Enum` 陳述式之間至少必須宣告一個成員。  
  
 **錯誤 ID:** BC30280  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
1.  請將成員宣告加入列舉。
