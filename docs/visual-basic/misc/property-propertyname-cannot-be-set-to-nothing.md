# <a name="property-ltpropertynamegt-cannot-be-set-to-nothing"></a>屬性&lt;propertyname&gt;不設定為 Nothing
這個屬性需要 `Nothing`以外的值。 這可能是不正確計算值的結果。  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請提供有效的值。
