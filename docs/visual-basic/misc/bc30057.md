# <a name="too-many-arguments-to-39ltprocedurenamegt39"></a>引數太多 &#39;&lt;程序名稱&gt;&#39;
程序呼叫中所提供的引數數目超過為程序所定義的引數數目。  
  
 **錯誤 ID︰** BC30057  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請參考程序的定義，並從呼叫移除多餘的引數。
