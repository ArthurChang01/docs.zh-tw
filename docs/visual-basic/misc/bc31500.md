# <a name="39shared39-attribute-property-39ltpropertyfieldgt39-cannot-be-the-target-of-an-assignment"></a>&#39;共用&#39;屬性&#39; &lt;propertyfield&gt; &#39;不可以是指派的目標
已嘗試將值指派給屬性 (attribute) 中的 `ReadOnly` 或 `Shared` 屬性 (property)。  
  
 **錯誤 ID︰** BC31500  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
1.  請移除屬性指派陳述式。  
  
2.  如果使用您所開發的屬性，請從屬性 (attribute) 的屬性 (property) 中移除 `ReadOnly` 或 `Shared` 修飾詞。
