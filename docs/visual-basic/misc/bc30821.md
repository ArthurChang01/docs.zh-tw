# <a name="39ltrsetstmtgt39-is-not-declared"></a>&#39;&lt;rsetstmt&gt;&#39; 不宣告
'\<rsetstmt >' 未宣告。 不再支援 RSet 陳述式。請使用 Microsoft.VisualBasic.RSet 代替。  
  
 內建的幾個函式[!INCLUDE[vbprvb](~/includes/vbprvb-md.md)]在舊版中都已移至<xref:Microsoft.VisualBasic?displayProperty=nameWithType>命名空間。 因此其功能可更廣泛地供所有程式設計語言使用。  
  
 **錯誤 ID︰** BC30821  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   改為在 `RSet` 命名空間中使用 `Microsoft.VisualBasic` 函式。
