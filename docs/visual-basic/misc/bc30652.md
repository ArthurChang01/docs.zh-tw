# <a name="reference-required-to-assembly-ltassemblynamegt-containing-the-type-ltmembernamegt"></a>需要組件參考 '&lt;assemblyname&gt;'包含類型'&lt;membername&gt;'
需要組件參考 '\<組件名稱 >' 包含類型'\<成員名稱 >'。 請在專案中加入一個參考。  
  
 專案所含的類型取決於未包含在專案參考之組件的另一個類型。  
  
 **錯誤 ID:** BC30652  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請在專案中加入相依類型的專案參考。
