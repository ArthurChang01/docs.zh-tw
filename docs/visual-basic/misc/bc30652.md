# <a name="reference-required-to-assembly-39ltassemblynamegt39-containing-the-type-39ltmembernamegt39"></a>需要組件參考&#39; &lt;assemblyname&gt; &#39;包含型別&#39;&lt;成員名稱&gt;&#39;
需要組件參考 '\<assemblyname >' 包含型別'\<成員名稱 >'。 請在專案中加入一個參考。  
  
 專案所含的類型取決於未包含在專案參考之組件的另一個類型。  
  
 **錯誤 ID：** BC30652  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請在專案中加入相依類型的專案參考。
