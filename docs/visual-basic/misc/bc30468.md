# <a name="type-declaration-characters-are-not-valid-in-this-context"></a>類型宣告字元在此內容中無效
您嘗試使用在這個內容中無效的字元，例如 $。  
  
 **錯誤 ID:** BC30468  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
1.  移除字元。
