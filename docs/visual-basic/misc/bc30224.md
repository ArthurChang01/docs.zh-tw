# <a name="39is39-expected"></a>&#39;是&#39;預期
出現的 `TypeOf` 運算子沒有 `Is` 子句。  
  
 **錯誤 ID︰** BC30224  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   在 `Is` 運算子後面加入 `TypeOf` 子句 (例如 `TypeOf MyControl Is ComboBox`)。
