# 必須是 &#39;Is&#39;
出現的 `TypeOf` 運算子沒有 `Is` 子句。  
  
 **錯誤 ID︰**BC30224  
  
### 更正這個錯誤  
  
-   在 `TypeOf` 運算子後面加入 `Is` 子句 \(例如 `TypeOf MyControl Is ComboBox`\)。
