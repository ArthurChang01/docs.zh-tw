# <a name="lttypenamegt-39ltmembernamegt39-must-be-declared-39shadows39-because-another-member-with-this-name-is-declared-39shadows39"></a>&lt;typename&gt; &#39;&lt;membername&gt;&#39; 必須宣告 &#39;Shadows &#39;因為此名稱的另一個成員宣告為 &#39;Shadows &#39;
這個屬性或程序的名稱與另一個以 `Shadows` 修飾詞標記的屬性或程序相同。  
  
 **錯誤 ID︰** BC30695  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請使用 `Shadows` 修飾詞。
