# <a name="lttypenamegt-ltmembernamegt-must-be-declared-shadows-because-another-member-with-this-name-is-declared-shadows"></a>&lt;typename&gt; '&lt;membername&gt;' 必須宣告為 'Shadows' 因為使用此名稱的另一個成員宣告為 'Shadows'
這個屬性或程序的名稱與另一個以 `Shadows` 修飾詞標記的屬性或程序相同。  
  
 **錯誤 ID:** BC30695  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請使用 `Shadows` 修飾詞。
