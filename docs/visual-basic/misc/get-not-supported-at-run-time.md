# <a name="get-not-supported-at-run-time"></a>在執行階段不支援 Get
您嘗試在執行階段讀取只有設計階段才能存取的屬性。  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
1.  請檢查屬性，並判斷在哪些情況下可以設定屬性。  
  
2.  移除屬性的參考。
