# <a name="unable-to-sign-assembly-lterror-messagegt"></a>無法簽署組件：&lt;錯誤訊息&gt;
Visual Basic 編譯器呼叫組件連結器 (Al.exe，也稱為 Alink)，以產生資訊清單的組件。 連結器在建立組件後進行簽署時回報了錯誤。 這可能是因為金鑰無效。  
  
 **錯誤 ID:** BC30146  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
1.  檢查引用的錯誤訊息，並參考主題[Al.exe （組件連結器）](../../framework/tools/al-exe-assembly-linker.md)取得進一步說明和建議。  
  
2.  如果錯誤持續發生，請收集情況的相關資訊，並通知 Microsoft 產品支援服務。
