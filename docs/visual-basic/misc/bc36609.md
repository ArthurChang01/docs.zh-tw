# <a name="query-expressions-are-not-valid-in-debug-windows"></a>查詢運算式在偵錯視窗中無效
在偵錯工作階段期間，已於 [監看式]  或 [即時運算]  視窗中輸入 LINQ 查詢。 在偵錯視窗中，沒有可用的查詢。  
  
 **錯誤 ID:** BC36609
