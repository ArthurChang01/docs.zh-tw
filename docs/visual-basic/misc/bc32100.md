# <a name="39as39-comma-or-3939-expected"></a>&#39;做為 （& s) #39; 逗號或 &#39;) （& s) #39;必須是
提供的類型參數缺少 `As`、逗號 (`,`) 或右括弧 ( `)` )。  
  
 **錯誤 ID︰** BC32100  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請提供遺漏 `As`、逗號或右括弧。
