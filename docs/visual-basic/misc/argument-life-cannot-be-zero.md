# <a name="argument-life-cannot-be-zero"></a>引數 'Life' 不可以為零
`Life`的引數無效，該引數必須是指定資產使用年限長度的 `Double` 。  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請檢查運算式中的引數拼法。 拼錯的變數名稱可能會隱含地建立初始化為零的數值變數。  
  
-   請檢查運算式中先前的變數作業，特別是那些作為引數從其他程序傳遞至程序的變數。
