# <a name="39namespace39-statements-can-occur-only-at-file-or-namespace-level"></a>&#39;命名空間&#39;陳述式只可以發生在檔案或命名空間層級
`Namespace` 陳述式必須出現在 `Option` 陳述式、 `Imports` 陳述式和全域屬性後面，但在原始程式檔中的所有其他宣告前面。  
  
 **錯誤 ID︰** BC30618  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   將 `Namespace` 陳述式移至命名空間宣告或原始程式檔的上方。
