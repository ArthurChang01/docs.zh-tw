# <a name="collection-index-must-be-in-the-range-1-to-the-size-of-the-collection"></a>集合索引必須在 1 到集合大小的範圍間
提供的索引值超出 1 到集合大小的有效範圍。  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
1.  將提供的值變更為有效值。
