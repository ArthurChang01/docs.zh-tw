# <a name="cannot-rename-with-different-drive"></a>無法以不同的磁碟機重新命名
`Rename` 陳述式無法在檔案路徑中指定磁碟機代號。  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請從檔案路徑移除磁碟機代號。
