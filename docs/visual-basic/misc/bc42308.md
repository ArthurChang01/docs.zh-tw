# <a name="xml-comment-parameter-must-have-a-name-attribute"></a>XML 註解參數必須要有 'name' 屬性
XML 註解參數必須要有 'name' 屬性。 將會忽略 XML 註解。  
  
 XML 標記遺漏必要的 `name` 屬性。  
  
 **錯誤 ID:** BC42308  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 為標記提供必要的 `name` 屬性。
