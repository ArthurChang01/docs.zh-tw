# &#39;System.Runtime.InteropServices.DllImportAttribute&#39; 無法套用至 &#39;Declare&#39;
`DllImportAttribute` 屬性已套用至 `Declare` 函式。 這個屬性只能與空白 `Function` 或 `Sub` 搭配使用。  
  
 **錯誤 ID︰**BC31523  
  
### 更正這個錯誤  
  
1.  請從 `Declare` 陳述式中移除 `DllImportAttribute` 屬性。
