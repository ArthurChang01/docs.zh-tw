# <a name="methods-declared-overrides-cannot-be-declared-overridable-because-they-are-implicitly-overridable"></a>宣告為 'Overrides' 的方法無法宣告為 'Overridable'，因為它們可以被隱含地覆寫
與大多數的方法不同，標記為 `Overrides` 修飾詞的方法預設是可覆寫的。  
  
 **錯誤 ID:** BC30730  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   省略標記為 `Overridable` 修飾詞之方法中的 `Overrides` 修飾詞。
