# <a name="3939-expected"></a>&#39;)&#39;預期
清單，例如程序定義程序呼叫中的引數清單，未以右括弧 (`)`) 字元終止。  
  
 **錯誤 ID︰** BC30198  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請將右括弧字元加入清單的結尾。
