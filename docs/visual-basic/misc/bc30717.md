# <a name="39try39-statements-are-not-valid-in-the-immediate-window"></a>'Try' 陳述式不是有效的即時運算視窗中
`Try`陳述式僅能在原始程式碼中。  
  
 **錯誤 ID︰** BC30717  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請從偵錯程式碼中移除 `Try` 陳述式。
