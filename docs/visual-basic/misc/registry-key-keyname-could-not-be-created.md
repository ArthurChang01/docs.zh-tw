# <a name="registry-key-keyname-could-not-be-created"></a>登錄機碼 '\<keyname >' 無法建立
無法建立指定的登錄機碼，原因可能是安全性權限不足。  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 請確定您有足夠的安全性權限可以存取機碼。
