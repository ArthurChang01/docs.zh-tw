# 指數無效
您嘗試使用具有浮點常值的無效指數。 浮點常值類型與保存浮點常值所需的大小不一致時，通常會造成這個錯誤。  
  
 **錯誤 ID：**BC30495  
  
### 更正這個錯誤  
  
1.  請檢查浮點常值類型，並將它變更為有效的類型。
