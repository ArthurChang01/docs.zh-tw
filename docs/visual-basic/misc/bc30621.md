# <a name="39end-structure39-must-be-preceded-by-a-matching-39structure39"></a>&#39;End Structure&#39;之前必須搭配相對應的&#39;結構&#39;
`End Structure` 陳述式只能用在以 `Structure` 陳述式開頭的結構區塊結尾處。  
  
 **錯誤識別碼：** BC30621  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請確定目前的結構區塊是從 `Structure` 陳述式開始。
