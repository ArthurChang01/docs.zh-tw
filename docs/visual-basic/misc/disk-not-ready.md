# <a name="disk-not-ready"></a>磁碟未就緒
指定的磁碟機中沒有磁片或是磁碟的門沒有關上。  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
1.  檢查指定的磁碟機，並視需要插入磁片。  
  
2.  關好磁碟機的門。
