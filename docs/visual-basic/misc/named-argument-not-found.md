# <a name="named-argument-not-found-visual-basic"></a>找不到具名引數 (Visual Basic)
除非具名引數出現在程序定義中，否則不能用於程序引動過程。 您已指定具名引數，但程序並未定義為接受該名稱的引數。  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 檢查引數名稱的拼字。
