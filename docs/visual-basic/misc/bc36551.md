## <a name="example"></a><span data-ttu-id="a7feb-108">範例</span><span class="sxs-lookup"><span data-stu-id="a7feb-108">Example</span></span>

```  
Namespace StringUtility  
    <Extension()> _  
    Module StringExtensions  
        <Extension()> _  
        Public Sub Print (ByVal str As String)  
            Console.WriteLine(str)  
        End Sub  
    End Module  
End Namespace  
```
