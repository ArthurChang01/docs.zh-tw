# <a name="name-cannot-expose-the-underlying-delegate-type-delegatetype-of-the-event-it-is-implementing-outside-the-project-through-specifier-type"></a>'\<名稱 >' 無法公開 （expose) 的基礎委派類型'\<委派 >' 透過在專案外其實作之事件\<規範 > '\<類型 >'
這個程式碼會在公用類別外部公開私用類型。  
  
 **錯誤 ID:** BC30915  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 宣告 `Public`類型。  
  
     -或-  
  
- 使用另一種類型。
