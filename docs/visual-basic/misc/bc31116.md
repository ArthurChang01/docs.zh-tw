# <a name="39removehandler39-declaration-must-end-with-a-matching-39end-removehandler39"></a>&#39;RemoveHandler&#39;宣告必須搭配相對應的結束&#39;End RemoveHandler&#39;
`RemoveHandler` 宣告的結尾必須是 `End RemoveHandler` 陳述式。  
  
 **錯誤 ID︰** BC31116  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請檢查 `RemoveHandler` 宣告的結尾是否為 `End RemoveHandler` 陳述式。
