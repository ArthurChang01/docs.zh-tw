# <a name="xml-comment-parameter-39ltparametergt39-does-not-match-a-parameter-on-the-corresponding-ltkeywordgt-statement"></a>XML 註解參數&#39;&lt;參數&gt;&#39;在相對應的參數不相符&lt;關鍵字&gt;陳述式
XML 註解參數\<參數 > 在相對應的參數不相符\<關鍵字 > 陳述式。 將會忽略 XML 註解。  
  
 已針對方法的部分參數，但不是全部參數，指定方法的 XML 註解標記。  
  
 **錯誤識別碼：** BC42307  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請為方法中的所有參數提供 XML 註解標記。
