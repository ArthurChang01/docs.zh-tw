# <a name="imports-alias-ltnamegt-conflicts-with-ltname2gt-declared-in-the-root-namespace"></a>Imports 別名&lt;名稱&gt;'與衝突'&lt;name2&gt;' 中的根命名空間宣告
`Imports` 陳述式中所使用的別名名稱與根命名空間中所宣告的另一個名稱相衝突。  
  
 **錯誤 ID:** BC31403  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   為 `Imports` 陳述式所使用的別名，選擇唯一的名稱。
