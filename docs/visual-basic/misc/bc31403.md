# <a name="imports-alias-39ltnamegt39-conflicts-with-39ltname2gt39-declared-in-the-root-namespace"></a>Imports 別名 &#39;&lt;名稱&gt;&#39; 與衝突 &#39;&lt;name2&gt;&#39; 根命名空間中宣告
`Imports` 陳述式中所使用的別名名稱與根命名空間中所宣告的另一個名稱相衝突。  
  
 **錯誤 ID︰** BC31403  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   為 `Imports` 陳述式所使用的別名，選擇唯一的名稱。
