# <a name="project-level-conditional-compilation-constant-39ltconstantnamegt39-is-not-valid-lterrorgt"></a>專案層級的條件式編譯常數 &#39;&lt;常數名稱&gt;&#39; 不是有效：&lt;錯誤&gt;
專案指定了無效的條件式編譯選項。  
  
 **錯誤 ID：** BC31030  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
1.  請確認與這個專案搭配使用的條件式編譯常數。
