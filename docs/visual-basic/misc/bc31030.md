# <a name="project-level-conditional-compilation-constant-constantname-is-not-valid-error"></a>專案層級的條件式編譯常數 '\<常數名稱 >' 無效：\<錯誤 >
專案指定了無效的條件式編譯選項。  
  
 **錯誤 ID:** BC31030  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
1.  請確認與這個專案搭配使用的條件式編譯常數。
