# <a name="name-cannot-expose-type-type1-in-specifier-type2-typename-through-name"></a>'\<名稱 >' 無法公開 （expose) 的型別\<type1 > 中\<規範 > \<type2 >'\<類型名稱 >' 透過\<名稱 >
您已嘗試在公用類別 `MyClass`外部公開私用類型。  
  
 **錯誤 ID:** BC30508  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 宣告 `Public`類型。
