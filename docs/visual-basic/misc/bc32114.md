# <a name="type-parameter-with-a-structure-constraint-cannot-be-used-as-a-constraint"></a><span data-ttu-id="5b594-102">具有 'Structure' 條件約束的類型參數不可以當做條件約束使用</span><span class="sxs-lookup"><span data-stu-id="5b594-102">Type parameter with a 'Structure' constraint cannot be used as a constraint</span></span>
<span data-ttu-id="5b594-103">具有 `Structure` 條件約束的類型參數是用作另一個類型參數的條件約束。</span><span class="sxs-lookup"><span data-stu-id="5b594-103">A type parameter with a `Structure` constraint is used as the constraint for another type parameter.</span></span>  
  
 <span data-ttu-id="5b594-104">`Structure` 條件約束需要傳遞給其類型參數的類型引數都是實值類型。</span><span class="sxs-lookup"><span data-stu-id="5b594-104">The `Structure` constraint requires that the type argument passed to its type parameter be a value type.</span></span> <span data-ttu-id="5b594-105">不過，無法實作或繼承實值類型，因此使用它作為條件約束 (即需要另一種類型參數來實作它或繼承它) 並沒有意義。</span><span class="sxs-lookup"><span data-stu-id="5b594-105">However, a value type cannot be implemented or inherited, so it is not meaningful to use it as a constraint, which would require the other type parameter to implement it or inherit from it.</span></span>  
  
 <span data-ttu-id="5b594-106">這種狀況之唯一有意義的解釋是類型引數必須是完全相同的類型。</span><span class="sxs-lookup"><span data-stu-id="5b594-106">The only meaningful interpretation of this situation is that both type arguments must be of the exact same type.</span></span> <span data-ttu-id="5b594-107">如果是這種狀況，泛型類型只需要一個類型參數。</span><span class="sxs-lookup"><span data-stu-id="5b594-107">If that is the case, your generic type needs only one type parameter.</span></span>  
  
 <span data-ttu-id="5b594-108">下列陳述式可能會產生這個錯誤。</span><span class="sxs-lookup"><span data-stu-id="5b594-108">The following statement can generate this error.</span></span>  
  
 `Class c1(Of t1 As Structure, t2 As t1)`  
  
 <span data-ttu-id="5b594-109">傳遞給 `t2` 的類型無法實作或繼承傳遞給 `t1`的類型，因為傳遞給 `t1` 的類型必須是實值類型。</span><span class="sxs-lookup"><span data-stu-id="5b594-109">The type passed to `t2` cannot implement or inherit the type passed to `t1`, because the type passed to `t1` must be a value type.</span></span>  
  
 <span data-ttu-id="5b594-110">**錯誤 ID:** BC32114</span><span class="sxs-lookup"><span data-stu-id="5b594-110">**Error ID:** BC32114</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="5b594-111">更正這個錯誤</span><span class="sxs-lookup"><span data-stu-id="5b594-111">To correct this error</span></span>  
  
- <span data-ttu-id="5b594-112">從另一個類型參數的條件約束清單中，移除限制為 `Structure` 的類型參數。</span><span class="sxs-lookup"><span data-stu-id="5b594-112">Remove the type parameter constrained to `Structure` from the constraint list on the other type parameter.</span></span>  
  
- <span data-ttu-id="5b594-113">如果這兩個類型參數都需要相同的實值類型，請定義只有一個類型參數的泛型類型。</span><span class="sxs-lookup"><span data-stu-id="5b594-113">If both type parameters require the same value type, define the generic type with only one type parameter.</span></span>
