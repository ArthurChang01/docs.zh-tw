# 找不到檔案 &lt;檔案名稱&gt;
找不到指定的檔案。 這可能是因為提供了不正確的檔案路徑。  
  
 **錯誤 ID︰**BC2001  
  
### 更正這個錯誤  
  
-   請檢查檔案名稱和路徑是否正確。
