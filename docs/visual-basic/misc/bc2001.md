# <a name="file-filename-could-not-be-found"></a>檔案\<檔名 > 找不到
找不到指定的檔案。 這可能是因為提供了不正確的檔案路徑。  
  
 **錯誤 ID:** BC2001  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 請檢查檔案名稱和路徑是否正確。
