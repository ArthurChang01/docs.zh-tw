# <a name="named-argument-expected"></a>必須有具名引數
程序呼叫中的引數清單會先依名稱傳遞引數，再依位置傳遞引數。 所有位置引數都必須在程序呼叫中依名稱所傳遞之任何引數的前面。  
  
 **錯誤 ID︰** BC30241  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請重寫引數清單，將依位置所傳遞的引數放在依名稱所傳遞的引數前面。
