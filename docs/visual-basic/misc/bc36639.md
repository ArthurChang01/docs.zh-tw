# <a name="byref-parameter-ltparameternamegt-cannot-be-used-in-a-lambda-expression"></a><span data-ttu-id="0004c-102">'ByRef' 參數 '&lt;parametername&gt;' 不能用於 lambda 運算式</span><span class="sxs-lookup"><span data-stu-id="0004c-102">'ByRef' parameter '&lt;parametername&gt;' cannot be used in a lambda expression</span></span>
<span data-ttu-id="0004c-103">在 `Sub` 或函式內宣告的 Lambda 運算式不能使用 `ByRef` 或函式的任何 `Sub` 參數。</span><span class="sxs-lookup"><span data-stu-id="0004c-103">A lambda expression declared within a `Sub` or function cannot use any `ByRef` parameters of that `Sub` or function.</span></span> <span data-ttu-id="0004c-104">例如，下列程式碼會導致這個錯誤，因為 Lambda 運算式中使用了 `ByRef` 參數 `n` 。</span><span class="sxs-lookup"><span data-stu-id="0004c-104">For example, the following code will cause this error because the `ByRef` parameter `n` is used in the lambda expression.</span></span>  
  
```  
'' Not valid.   
'Sub ExampleSub(ByRef n As Integer)  
  
'    Dim lambda = Function(p As Integer) p + n  
  
'End Sub  
```  
  
 <span data-ttu-id="0004c-105">**錯誤 ID:** BC36639</span><span class="sxs-lookup"><span data-stu-id="0004c-105">**Error ID:** BC36639</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="0004c-106">更正這個錯誤</span><span class="sxs-lookup"><span data-stu-id="0004c-106">To correct this error</span></span>  
  
-   <span data-ttu-id="0004c-107">將 `ByRef` 參數指派給區域變數，然後在 Lambda 運算式中使用此區域變數，如下列程式碼所示：</span><span class="sxs-lookup"><span data-stu-id="0004c-107">Assign the `ByRef` parameter to a local variable, and use the local variable in the lambda expression, as shown in the following code:</span></span>  
  
    ```  
    Sub ExampleSub(ByRef n As Integer)  
  
        Dim temp = n  
        Dim lambda = Function(p As Integer) p + temp  
  
    End Sub  
    ```
