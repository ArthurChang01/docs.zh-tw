# <a name="method-ltmethodnamegt-has-no-parameter-named-ltparameternamegt"></a>方法 '&lt;methodname&gt;'具有名為無參數'&lt;parametername&gt;'
提供的具名引數不符合目標方法的任何引數。  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
1.  確定已正確提供具名引數。
