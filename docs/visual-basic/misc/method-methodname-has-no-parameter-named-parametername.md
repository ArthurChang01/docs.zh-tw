# <a name="method-methodname-has-no-parameter-named-parametername"></a>方法 '\<方法名稱 >' 沒有名稱為參數'\<參數名稱 >'
提供的具名引數不符合目標方法的任何引數。  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
1. 確定已正確提供具名引數。
