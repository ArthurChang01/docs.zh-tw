# <a name="division-by-zero-occurred-while-evaluating-this-expression"></a>評估此運算式時發生除數為零的情形
評估這個運算式時，您嘗試將數字除以零。  
  
 **錯誤 ID︰** BC30542  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
1.  檢查您的常數和變數值，判斷運算式的分母為何會出現零。
