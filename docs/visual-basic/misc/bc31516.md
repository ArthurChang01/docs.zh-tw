# <a name="type-39lttypenamegt39-cannot-be-used-as-an-attribute-because-it-is-not-declared-39public39"></a>型別&#39; &lt;typename&gt; &#39;無法用做為屬性中，因為它未宣告為&#39;公用&#39;
屬性必須是公用的。  
  
 **錯誤 ID︰** BC31516  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
1.  請將 `Public` 修飾詞加入屬性類別。
