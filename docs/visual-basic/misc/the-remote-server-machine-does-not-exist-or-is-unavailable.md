# 遠端伺服器電腦不存在或無法使用 (Visual Basic)
您嘗試連線到無法使用或不存在的遠端伺服器。 這個錯誤可能是由於您的網路連線中斷所造成。  
  
### 更正這個錯誤  
  
-   重新連線到網路，然後再試一次。
