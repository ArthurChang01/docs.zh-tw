# <a name="the-remote-server-machine-does-not-exist-or-is-unavailable-visual-basic"></a>遠端伺服器電腦不存在或無法使用 (Visual Basic)
您嘗試連線到無法使用或不存在的遠端伺服器。 這個錯誤可能是由於您的網路連線中斷所造成。  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   重新連線到網路，然後再試一次。
