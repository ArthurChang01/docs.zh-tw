# <a name="wrong-number-of-type-arguments-passed-to-method-39ltgenericprocedurenamegt39"></a>傳遞至方法的型別引數數目錯誤&#39;&lt;泛型程序名稱&gt;&#39;
呼叫泛型程序時所使用的類型引數數目，與所定義的類型參數數目不符。  
  
 **錯誤 ID︰** BC30951  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   針對定義給泛型程序的每個類型參數，各提供一個類型引數。  
  
     -或-  
  
-   完全不使用類型引數來呼叫泛型程序，並讓編譯器嘗試推斷類型引數。
