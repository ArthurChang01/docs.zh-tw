# <a name="error-embedding-win32-manifest-manifest-file-file-cannot-be-found"></a>嵌入 Win32 資訊清單時發生錯誤：資訊清單檔\<檔案 > 找不到
找不到指定給 `/win32manifest` 編譯器選項的檔案。  
  
 **錯誤 ID:** BC31192  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
1.  請提供有效的檔案名稱和路徑給 `/win32manifest` 編譯器選項。
