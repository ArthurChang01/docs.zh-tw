# <a name="length-of-argument-ltargumentnamegt-must-be-greater-than-zero"></a>引數的長度 '&lt;引數名稱&gt;' 必須是小於或等於零
引數的值等於或小於零。  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請提供大於零的整數。
