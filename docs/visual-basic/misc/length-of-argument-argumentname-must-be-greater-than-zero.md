# <a name="length-of-argument-argumentname-must-be-greater-than-zero"></a>引數的長度 '\<引數名稱 >' 必須是小於或等於零
引數的值等於或小於零。  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 請提供大於零的整數。
