# <a name="39on-error39-statements-are-not-valid-within-39using39-statements"></a>&#39;錯誤&#39;陳述式不在有效&#39;使用&#39;陳述式
`On Error` 陳述式會出現在 `Using` 陳述式內，但在該內容中無效。  
  
 **錯誤識別碼：** BC36013  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   在 `Try…Catch` 陳述式的位置使用結構化的錯誤處理，例如 `On Error` 區塊。
