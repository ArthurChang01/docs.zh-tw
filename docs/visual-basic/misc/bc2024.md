# <a name="the-default-assembly-reference-reference-has-already-been-automatically-added-ignored"></a>預設組件參考 '\<參考 >' 有已被自動新增; 已忽略
已嘗試加入預設組件參考，但組件已經加入，因此忽略該嘗試。  
  
 **錯誤 ID:** BC2024  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 請移除多餘的程式碼。
