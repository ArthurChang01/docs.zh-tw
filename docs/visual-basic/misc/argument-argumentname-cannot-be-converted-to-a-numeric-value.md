# <a name="argument-ltargumentnamegt-cannot-be-converted-to-a-numeric-value"></a>引數 '&lt;引數名稱&gt;' 無法轉換成數值
已嘗試將變數，例如字串，轉換為數值。  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
1.  請尋找有意義的轉換。
