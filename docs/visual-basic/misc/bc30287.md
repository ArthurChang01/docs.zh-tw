# <a name="-expected"></a>必須是 '.'
程式碼有 `Handles` 子句，其中包含一個驚嘆號 (`!`)。  
  
 **錯誤 ID:** BC30287  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
1.  如果 `Handles` 子句參考物件內的事件，請使用句號 (`.`) 來分隔物件與事件。
