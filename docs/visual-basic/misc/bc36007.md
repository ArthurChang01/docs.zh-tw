# <a name="39end-using39-must-be-preceded-by-a-matching-39using39"></a>&#39;結束使用&#39;之前必須搭配相對應的&#39;使用&#39;
`End Using` 陳述式的前面沒有配對的 `Using` 宣告。  
  
 **錯誤 ID︰** BC36007  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   若 `End Using` 陳述式為多餘，請予以移除。  
  
-   提供遺漏[Using 陳述式](../../visual-basic/language-reference/statements/using-statement.md)遺漏時。  
  
-   請將 `End Using` 陳述式移至程式碼中的適當位置。
