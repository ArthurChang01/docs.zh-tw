# <a name="attribute-constant-ltconstantnamegt-cannot-be-the-target-of-an-assignment"></a>屬性常數 '&lt;常數名稱&gt;' 不可以是指派的目標
已嘗試指派值在屬性中宣告的常數。  
  
 **錯誤 ID:** BC31510  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請移除錯誤的指派。  
  
-   如果您使用您開發的自訂屬性，請將屬性中的成員重新定義為欄位。
