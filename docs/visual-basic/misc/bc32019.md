# <a name="resume-or-goto-expected"></a>必須是 'Resume' 或 'GoTo'
`On Error` 陳述式未包含 `Resume` 子句或 `GoTo` 子句。 它必須使用其中一個子句或另一個子句。  
  
 **錯誤 ID:** BC32019  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 將 `Resume` 子句或 `GoTo` 子句加入 `On Error` 陳述式。
