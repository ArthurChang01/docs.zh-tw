# <a name="unused-local-constant-ltconstantgt"></a>未使用的區域常數: '&lt;常數&gt;'
已宣告您的程式碼中任何地方都未使用的區域常數。  
  
 **錯誤 ID:** BC42099  
  
### <a name="to-remove-this-warning"></a>移除這個警告  
  
1.  這個錯誤是警告，可加以忽略。 若要移除警告，請從您的程式碼中移除未使用的常數宣告。
