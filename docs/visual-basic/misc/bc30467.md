# <a name="ltname1gt-for-the-imports-ltname2gt-does-not-refer-to-a-namespace-class-structure-enum-or-module"></a>'&lt;name1&gt;'的匯入'&lt;name2&gt;' 未參考命名空間、 類別、 結構、 列舉或模組
您已嘗試在不是 `Imports` 、 `Namespace`、 `Class`、 `Structure`或 `Enum`的某個項目上使用 `Module`陳述式。 `Imports` 陳述式從參考的專案和組件中匯入命名空間名稱，或匯入與陳述式所在之模組相同的專案內所定義的命名空間名稱。  
  
 **錯誤 ID:** BC30467  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請檢查您嘗試匯入的實體，並確定它可以與 `Imports` 陳述式搭配使用。
