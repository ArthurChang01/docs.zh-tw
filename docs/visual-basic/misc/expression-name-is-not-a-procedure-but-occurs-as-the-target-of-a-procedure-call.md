# 運算式 &#39;&lt;名稱&gt;&#39; 不是程序，但當做程序呼叫的目標
您對不是程序的運算式執行了程序呼叫。  
  
### 更正這個錯誤  
  
1.  請確定未拼錯程序名稱。  
  
2.  提供 \<`name`\> 呼叫的程序。
