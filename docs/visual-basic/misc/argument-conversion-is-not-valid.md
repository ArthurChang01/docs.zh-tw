# <a name="argument-39conversion39-is-not-valid"></a>引數 &#39;轉換 &#39;不正確
`VbStrConv` 列舉的值無效，可能是因為它小於零。 列舉值必須大於或等於零。  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
1.  請確定值大於或等於零。
