# <a name="method-methodname-cannot-be-called-with-number-arguments"></a>方法 '\<方法名稱 >' 無法使用呼叫\<數字 > 引數
方法呼叫包含錯誤的引數數目。  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
1.  請計算您提供的引數數目。  
  
2.  比對這個數目與您想要呼叫之方法中的參數數目。  
  
3.  如果這兩個數目不一致，請調整引數數目。
