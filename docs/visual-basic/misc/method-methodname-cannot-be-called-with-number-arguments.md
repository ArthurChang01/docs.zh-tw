# <a name="method-ltmethodnamegt-cannot-be-called-with-ltnumbergt-arguments"></a>方法 '&lt;methodname&gt;' 無法使用呼叫&lt;數目&gt;引數
方法呼叫包含錯誤的引數數目。  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
1.  請計算您提供的引數數目。  
  
2.  比對這個數目與您想要呼叫之方法中的參數數目。  
  
3.  如果這兩個數目不一致，請調整引數數目。
