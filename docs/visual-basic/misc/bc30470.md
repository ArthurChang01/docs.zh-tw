# <a name="myclass-cannot-be-used-outside-of-a-class"></a>'MyClass' 無法在類別外使用
您已在類別結構外使用 `MyClass` 關鍵字。  
  
 **錯誤 ID:** BC30470  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請將 `MyClass` 宣告移至您的類別結構。
