# <a name="expected-closing-39gt39-for-xml-cdata-section"></a>預期的結尾 &#39;]]&gt;&#39; XML CDATA 區段
XML 註解常值遺漏必要的結尾 ']]>' 字元。  
  
 **錯誤 ID︰** BC31162  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請將必要的結尾 ']]>' 字元加入 XML 註解常值的結尾。
