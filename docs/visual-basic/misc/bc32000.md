# <a name="local-variable-variablename-cannot-be-referred-to-before-it-is-declared"></a>區域變數 '\<變數名稱 >' 無法參考宣告之前
程序內的變數使用於其宣告之前。  
  
 **錯誤 ID:** BC32000  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 將宣告移至變數的第一個參考之前。
