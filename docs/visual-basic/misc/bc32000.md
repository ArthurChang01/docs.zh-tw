# <a name="local-variable-39ltvariablenamegt39-cannot-be-referred-to-before-it-is-declared"></a>本機變數 &#39;&lt;variablename&gt;&#39; 無法參考宣告之前
程序內的變數使用於其宣告之前。  
  
 **錯誤識別碼：** BC32000  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   將宣告移至變數的第一個參考之前。
