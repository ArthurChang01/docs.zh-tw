# 在宣告區域變數 &#39;&lt;變數名稱&gt;&#39; 之前無法參考此區域變數。
程序內的變數使用於其宣告之前。  
  
 **錯誤識別碼：**BC32000  
  
### 更正這個錯誤  
  
-   將宣告移至變數的第一個參考之前。
