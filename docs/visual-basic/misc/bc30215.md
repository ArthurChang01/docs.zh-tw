# <a name="sub-or-function-expected"></a>必須有 'Sub' 或 'Function'
`Declare` 陳述式未指定 `Sub` 或 `Function`。 `Declare` 可識別外部程序，而且必須指定程序的類型。  
  
 **錯誤 ID:** BC30215  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 將 `Sub` 或 `Function` 關鍵字加入 `Declare` 陳述式。
