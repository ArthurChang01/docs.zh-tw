# <a name="39sub39-or-39function39-expected"></a>&#39;Sub &#39;或 &#39;函式 &#39;必須是
`Declare` 陳述式未指定 `Sub` 或 `Function`。 `Declare` 可識別外部程序，而且必須指定程序的類型。  
  
 **錯誤 ID︰** BC30215  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   將 `Sub` 或 `Function` 關鍵字加入 `Declare` 陳述式。
