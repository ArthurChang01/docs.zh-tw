# <a name="resume-statements-are-not-valid-in-the-immediate-window"></a>即時運算視窗中的 'Resume' 陳述式無效
Resume 陳述式只能用於原始程式碼中。  
  
 **錯誤 ID:** BC30714  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請從偵錯程式碼中移除 `Resume` 陳述式。
