# <a name="for-loop-control-variable-39ltvariablenamegt39-already-in-use-by-an-enclosing-for-loop"></a>For 迴圈控制變數 &#39;&lt;variablename&gt;&#39; 已被封閉式 For 迴圈所使用
巢狀 `For` 迴圈都必須使用唯一的控制變數。  
  
 **錯誤 ID︰** BC30069  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   變更其中一個控制變數名稱。
