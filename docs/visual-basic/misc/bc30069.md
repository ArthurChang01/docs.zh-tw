# <a name="for-loop-control-variable-ltvariablenamegt-already-in-use-by-an-enclosing-for-loop"></a>For 迴圈控制變數 '&lt;variablename&gt;' 已在使用中由封閉式 For 迴圈
巢狀 `For` 迴圈都必須使用唯一的控制變數。  
  
 **錯誤 ID:** BC30069  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   變更其中一個控制變數名稱。
