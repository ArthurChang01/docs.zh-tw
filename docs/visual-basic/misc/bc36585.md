# <a name="elementname-is-not-a-parameter-of-extension-method-methodname-defined-in-typename"></a>'\<項目名稱 >' 不是擴充方法的參數'\<方法名稱 >' 中定義 '\<類型名稱 >'
程序呼叫可提供擴充方法未定義的其他引數。  
  
 **錯誤 ID:** BC36585  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請從引數清單中移除多餘的引數。
