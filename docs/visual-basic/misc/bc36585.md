# <a name="ltelementnamegt-is-not-a-parameter-of-extension-method-ltmethodnamegt-defined-in-lttypenamegt"></a>'&lt;elementname&gt;'不是擴充方法的參數'&lt;methodname&gt;'中所定義的'&lt;typename&gt;'
程序呼叫可提供擴充方法未定義的其他引數。  
  
 **錯誤 ID:** BC36585  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請從引數清單中移除多餘的引數。
