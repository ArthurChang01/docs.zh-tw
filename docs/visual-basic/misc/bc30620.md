# &#39;Option Strict&#39; 之後只能是 &#39;On&#39; 或 &#39;Off&#39;
`Option Strict` 陳述式必須指定 `On` 或 `Off`。 所有其他值都無效。  
  
 **錯誤 ID：**BC30620  
  
### 更正這個錯誤  
  
-   為 `Option Strict` 陳述式指定 `On` 或 `Off`。
