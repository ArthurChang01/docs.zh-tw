# <a name="option-strict-can-be-followed-only-by-on-or-off"></a>'Option Strict' 之後只能是 'On' 或 'Off'
`Option Strict` 陳述式必須指定 `On` 或 `Off`。 所有其他值都無效。  
  
 **錯誤 ID:** BC30620  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   為 `On` 陳述式指定 `Off` 或 `Option Strict` 。
