# <a name="39option-strict39-can-be-followed-only-by-39on39-or-39off39"></a>&#39; 嚴格選項 &#39;後面只能接 &#39; 在 &#39;或 &#39; 關閉 &#39;
`Option Strict` 陳述式必須指定 `On` 或 `Off`。 所有其他值都無效。  
  
 **錯誤 ID：** BC30620  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   為 `On` 陳述式指定 `Off` 或 `Option Strict` 。
