# <a name="error-embedding-win32-manifest-manifest"></a>嵌入 Win32 資訊清單時發生錯誤： \<資訊清單 >
針對 `/win32manifest` 編譯器選項指定的檔案不可以內嵌。 這可能是資訊清單檔案的安全性權限不足的結果。  
  
 **錯誤識別碼：** BC31191  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
1. 請確定執行 Visual Basic 編譯器命令的身分識別可以存取 Win32 資訊清單檔案。
