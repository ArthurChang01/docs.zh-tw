# <a name="comment-statements-cannot-be-evaluated-not-in-toc"></a>無法評估註解陳述式 （不在目錄）
在偵錯工作階段期間，已於 [即時運算] 視窗中輸入註解陳述式。 無法評估 [即時運算] 視窗中的註解陳述式。  
  
 **錯誤 ID︰** BC30974
