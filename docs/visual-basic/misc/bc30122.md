# <a name="stop-statements-are-not-valid-in-the-immediate-window"></a>即時運算視窗中的 'Stop' 陳述式無效
`Stop` 和 `End` 陳述式在偵錯內容中暫停執行且不允許使用。  
  
 **錯誤 ID:** BC30122  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請不要在 [即時運算] `Stop``End` 視窗中發出 **或** 陳述式。
