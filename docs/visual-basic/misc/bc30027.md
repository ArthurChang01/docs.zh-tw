# <a name="end-function-expected"></a>必須有 'End Function'
`End` 陳述式必須符合它所終止的區塊。 未正確地終止 `Function` 程序定義。  
  
 **錯誤 ID:** BC30027  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 將 `End Function` 陳述式加入程序的結尾。
