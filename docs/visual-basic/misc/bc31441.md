# <a name="method-methodname1-must-be-declared-private-in-order-to-implement-partial-method-methodname2"></a><span data-ttu-id="70cf5-102">方法 '\<methodname1 >' 必須宣告為 'Private' 才能實作部分方法'\<methodname2 >'</span><span class="sxs-lookup"><span data-stu-id="70cf5-102">Method '\<methodname1>' must be declared 'Private' in order to implement partial method '\<methodname2>'</span></span>
<span data-ttu-id="70cf5-103">部分方法的實作必須宣告 `Private`。</span><span class="sxs-lookup"><span data-stu-id="70cf5-103">The implementation of a partial method must be declared `Private`.</span></span> <span data-ttu-id="70cf5-104">例如，下列程式碼會造成這個錯誤。</span><span class="sxs-lookup"><span data-stu-id="70cf5-104">For example, the following code causes this error.</span></span>  
  
```vb

    ' Declaration of the partial method.  
    Partial Private Sub valueChanged()  
    End Sub  
  
End Class  
```  
  
```vb  
Partial Class Product  
  
    ' Implementation of the partial method, with Private missing,   
    ' causes this error.   
    'Sub valueChanged()  
    '    MsgBox(Value was changed to " & Me.Quantity)  
    'End Sub  
  
End Class  
```
