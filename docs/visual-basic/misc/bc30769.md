# <a name="the-targeted-version-of-the-net-compact-framework-does-not-support-the-39end39-statement"></a>.NET Compact Framework 的目標的版本不支援 &#39;結束 &#39;陳述式
您使用的 .NET Compact Framework 版本不支援 `End` 陳述式。  
  
 **錯誤 ID︰** BC30769  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請移除陳述式。
