# <a name="39ltmethodnamegt39-cannot-override-a-method-that-has-been-shadowed"></a>&#39;&lt;methodname&gt; &#39;無法覆寫已遮蔽方法
方法已嘗試覆寫可遮蔽另一個成員的方法。  
  
 **錯誤 ID︰** BC31406  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請使用 `Shadows` 修飾詞，來覆寫已遮蔽的成員。
