# &#39;&lt;方法名稱&gt;&#39; 無法覆寫已遮蔽 (Shadows) 的方法
方法已嘗試覆寫可遮蔽另一個成員的方法。  
  
 **錯誤 ID︰**BC31406  
  
### 更正這個錯誤  
  
-   請使用 `Shadows` 修飾詞，來覆寫已遮蔽的成員。
