# <a name="39ltnamegt39-necessary-for-compiling-this-construct-cannot-be-found"></a>&#39;&lt;名稱&gt;&#39;編譯這個建構找不到所需
找不到編譯的必要項目。  
  
 **錯誤 ID：** BC30930  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
1.  提供必要的協助程式項目。  
  
2.  確定程式碼是有效的 Managed 程式碼。
