# <a name="specified-registry-key-does-not-exist"></a>指定的登錄機碼不存在
已指定的登錄機碼不存在。  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 檢查您所提供的登錄機碼，確定指定的路徑正確。  
  
- 先建立登錄機碼，再嘗試寫入其中。
