# 指定的登錄機碼不存在
已指定的登錄機碼不存在。  
  
### 更正這個錯誤  
  
-   檢查您所提供的登錄機碼，確定指定的路徑正確。  
  
-   先建立登錄機碼，再嘗試寫入其中。
