# <a name="too-many-arguments-to-extension-method-ltmethodnamegt-defined-in-lttypenamegt"></a>引數太多擴充方法 '&lt;methodname&gt;'中所定義的'&lt;typename&gt;'
擴充方法呼叫中所提供的引數數目超過為方法所定義的引數數目。  
  
 **錯誤 ID:** BC36582  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請參考擴充方法的定義，並從呼叫移除多餘的引數。
