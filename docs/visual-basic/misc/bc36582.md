# <a name="too-many-arguments-to-extension-method-39ltmethodnamegt39-defined-in-39lttypenamegt39"></a>擴充方法 &#39; 引數太多&lt;methodname&gt;&#39; 定義中 （& s) #39;&lt;typename&gt;&#39;
擴充方法呼叫中所提供的引數數目超過為方法所定義的引數數目。  
  
 **錯誤 ID︰** BC36582  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請參考擴充方法的定義，並從呼叫移除多餘的引數。
