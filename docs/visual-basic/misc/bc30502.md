# &#39;Shared&#39; 無法在屬性宣告上和 &#39;&lt;規範&gt;&#39; 結合
您嘗試結合 `Shared` 修飾詞與規範，這種結合在屬性宣告中無效。  
  
 **錯誤 ID︰**BC30502  
  
### 更正這個錯誤  
  
-   請移除規範。
