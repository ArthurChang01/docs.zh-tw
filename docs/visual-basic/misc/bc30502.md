# <a name="39shared39-cannot-be-combined-with-39ltspecifiergt39-on-a-property-declaration"></a>&#39;共用 &#39;無法結合 &#39;&lt;規範&gt;&#39; 在屬性宣告上
您嘗試結合 `Shared` 修飾詞與規範，這種結合在屬性宣告中無效。  
  
 **錯誤 ID︰** BC30502  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請移除規範。
