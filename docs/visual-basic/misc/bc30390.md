# <a name="ltqualifiedmethodgt-is-not-accessible-in-this-context-because-it-is-ltmodifiergt"></a>'&lt;完整方法&gt;'不在此內容中存取因為它是'&lt;修飾詞&gt;'
您嘗試存取因已套用修飾詞而無法在此內容中存取的方法。  
  
 **錯誤 ID:** BC30390  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請移除修飾詞。
