# <a name="expression-of-type-lttypename1gt-can-never-be-of-type-lttypename2gt"></a><span data-ttu-id="8433d-102">運算式的類型 '&lt;typename1&gt;'不能屬於類型'&lt;2&gt&gt;'</span><span class="sxs-lookup"><span data-stu-id="8433d-102">Expression of type '&lt;typename1&gt;' can never be of type '&lt;typename2&gt;'</span></span>
<span data-ttu-id="8433d-103">`TypeOf`...`Is` 運算式會測試指向其未包含之資料類型的物件參考變數。</span><span class="sxs-lookup"><span data-stu-id="8433d-103">A `TypeOf`...`Is` expression tests an object reference variable to a data type that it cannot hold.</span></span>  
  
 <span data-ttu-id="8433d-104">在某些情況下，編譯器可決定 `TypeOf`...`Is` 測試只會失敗；例如，如果兩個類別之間沒有任何繼承關聯性。</span><span class="sxs-lookup"><span data-stu-id="8433d-104">In some cases the compiler can determine that a `TypeOf`...`Is` test can only fail, for example if there is no inheritance relationship between two classes.</span></span>  
  
 <span data-ttu-id="8433d-105">下列程式碼可能會產生此錯誤。</span><span class="sxs-lookup"><span data-stu-id="8433d-105">The following code can generate this error.</span></span>  
  
 `Dim refVar as System.Windows.Forms.Form`  
  
 `If TypeOf refVar Is System.Array`  
  
 `End If`  
  
 <span data-ttu-id="8433d-106">由於 <xref:System.Windows.Forms.Form> 和 <xref:System.Array> 是完全不相關的類型，因此編譯器可決定讓 `TypeOf`...`Is` 運算式針對 `False` 的任何值傳回 `refVar`。</span><span class="sxs-lookup"><span data-stu-id="8433d-106">Because <xref:System.Windows.Forms.Form> and <xref:System.Array> are totally unrelated types, the compiler can determine that the `TypeOf`...`Is` expression returns `False` for any value of `refVar`.</span></span>  
  
 <span data-ttu-id="8433d-107">**錯誤 ID:** BC31430</span><span class="sxs-lookup"><span data-stu-id="8433d-107">**Error ID:** BC31430</span></span>
