[!INCLUDE[vbprvb](~/includes/vbprvb-md.md)]編譯器會無法復原從下列錯誤： <`error`>。  
  
 已停用某些 IDE 功能。  
  
 [!INCLUDE[vbprvb](~/includes/vbprvb-md.md)] 編譯器中發生內部錯誤。

1.  儲存您的工作，並重新啟動 [!INCLUDE[vsprvs](~/includes/vsprvs-md.md)]。  
  
2.  如果問題重複發生，請重新安裝 [!INCLUDE[vsprvs](~/includes/vsprvs-md.md)]。  
  
3.  如果錯誤在重新安裝之後持續發生，請通知 Microsoft 產品支援服務。
