# <a name="visual-basic-compiler-is-unable-to-recover-from-the-following-error-error"></a>Visual Basic 編譯器會無法從下列錯誤復原：\<錯誤 >
Visual Basic 編譯器會無法從下列錯誤復原： <`error`>。  
  
 已停用某些 IDE 功能。  
  
 在 Visual Basic 編譯器中發生內部錯誤。  
  
 **錯誤 ID:** BC31021  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
1. 儲存工作並重新啟動 Visual Studio。  
  
2. 如果錯誤重複發生，請重新安裝 Visual Studio。  
  
3. 如果錯誤在重新安裝之後持續發生，請通知 Microsoft 產品支援服務。
