# <a name="visual-basic-compiler-is-unable-to-recover-from-the-following-error-lterrorgt"></a>Visual Basic 編譯器會無法修復下列錯誤︰&lt;錯誤&gt;
[!INCLUDE[vbprvb](../../csharp/programming-guide/concepts/linq/includes/vbprvb_md.md)]編譯器會無法修復下列錯誤︰ `error`>。  
  
 已停用某些 IDE 功能。  
  
 發生內部錯誤發生在[!INCLUDE[vbprvb](../../csharp/programming-guide/concepts/linq/includes/vbprvb_md.md)]編譯器。  
  
 **錯誤 ID︰** BC31021  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
1.  儲存工作並重新啟動[!INCLUDE[vsprvs](../../csharp/includes/vsprvs_md.md)]。  
  
2.  如果問題重複發生，請重新安裝 [!INCLUDE[vsprvs](../../csharp/includes/vsprvs_md.md)]。  
  
3.  如果錯誤在重新安裝之後持續發生，請通知 Microsoft 產品支援服務。
