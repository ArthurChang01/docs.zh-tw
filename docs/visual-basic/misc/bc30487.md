# <a name="operator-ltoperatornamegt-is-not-defined-for-type-lttypenamegt"></a>運算子 '&lt;運算子&gt;'未定義類型'&lt;typename&gt;'
您嘗試在未定義它的類型 (例如`+`) 上使用一元運算子 ( `-`、 `Not`或 `String`)。  
  
 **錯誤 ID:** BC30487  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請將一元運算子取代為運算式的適當運算子。
