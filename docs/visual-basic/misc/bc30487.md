# <a name="operator-39ltoperatornamegt39-is-not-defined-for-type-39lttypenamegt39"></a>運算子&#39;&lt;運算子名稱&gt;&#39;類型未定義&#39;&lt;類型名稱&gt;&#39;
您嘗試在未定義它的類型 (例如`+`) 上使用一元運算子 ( `-`、 `Not`或 `String`)。  
  
 **錯誤 ID︰** BC30487  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請將一元運算子取代為運算式的適當運算子。
