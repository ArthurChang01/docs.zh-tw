# <a name="39error39-statements-are-not-valid-in-the-immediate-window"></a>&#39;錯誤&#39;陳述式不是有效的即時運算視窗中
`Error` 陳述式只能用於原始程式碼中。  
  
 **錯誤 ID︰** BC30724  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請從偵錯程式碼中移除 `Error` 陳述式。
