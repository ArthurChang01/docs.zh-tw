# <a name="shared-39sub-new39-cannot-be-declared-39ltspecifiergt39"></a>共用 &#39;新的子 &#39;不可以宣告為 &#39;&lt;規範&gt;&#39;
您已嘗試使用 `Shared Sub New` 宣告中的規範，這是無效的作業。  
  
 **錯誤 ID︰** BC30480  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請移除規範。
