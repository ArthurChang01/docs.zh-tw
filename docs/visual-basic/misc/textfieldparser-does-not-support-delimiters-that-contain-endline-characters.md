# <a name="textfieldparser-does-not-support-delimiters-that-contain-endline-characters"></a>TextFieldParser 不支援包含行尾字元的分隔符號
已提供包含行尾字元的分隔符號，但`TextFieldParser`不支援包含行尾字元的分隔符號。  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 請提供正確的分隔符號。
