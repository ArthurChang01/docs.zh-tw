# <a name="expected-closing--for-embedded-expression"></a>必須是右 ' %>' 的內嵌運算式
XML 常值中的內嵌運算式遺漏必要的結尾 `%>` 字元。  
  
 **錯誤 ID:** BC31159  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   將必要的結尾 `%>` 字元加入內嵌運算式的結尾。
