# <a name="default-values-cannot-be-supplied-for-parameters-that-are-not-declared-optional"></a>無法提供預設值給未宣告為 'Optional' 的參數。
呼叫程序時必須提供必要的引數，而且它們在宣告中不採用預設值。  
  
 **錯誤 ID:** BC32024  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   宣告引數為 `Optional` 或移除預設值。
