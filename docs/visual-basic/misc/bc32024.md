# <a name="default-values-cannot-be-supplied-for-parameters-that-are-not-declared-39optional39"></a>預設值無法提供給未宣告的參數 &#39; 選擇性 &#39;
呼叫程序時必須提供必要的引數，而且它們在宣告中不採用預設值。  
  
 **錯誤識別碼：** BC32024  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   宣告引數為 `Optional` 或移除預設值。
