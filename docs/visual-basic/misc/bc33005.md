# <a name="end-operator-expected"></a>必須是 'End Operator'
`End` 陳述式必須符合它所終止的區塊。 未正確地終止 `Operator` 程序定義。  
  
 **錯誤 ID:** BC33005  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
1.  將 `End Operator` 陳述式加入程序的結尾。
