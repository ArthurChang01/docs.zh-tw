<span data-ttu-id="34590-103">在無效的位置遇到 `Implements` 陳述式。</span><span class="sxs-lookup"><span data-stu-id="34590-103">An `Implements` statement was encountered in an invalid location.</span></span>

## <a name="to-correct-this-error"></a><span data-ttu-id="34590-105">更正這個錯誤</span><span class="sxs-lookup"><span data-stu-id="34590-105">To correct this error</span></span>  
  
-   <span data-ttu-id="34590-106">請將 `Implements` 陳述式緊接著放在任何 `Inherits` 陳述式之後，但在任何宣告之前。</span><span class="sxs-lookup"><span data-stu-id="34590-106">Place `Implements` statements immediately following any `Inherits` statements, but before any declarations.</span></span> <span data-ttu-id="34590-107">例如: </span><span class="sxs-lookup"><span data-stu-id="34590-107">For example:</span></span>

    ```  
       End Sub  
    End Class  
    ```
