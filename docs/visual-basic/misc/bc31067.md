# <a name="method-in-a-structure-cannot-be-declared-protected-or-protected-friend"></a>結構中的方法不可宣告為 'Protected' 或 'Protected Friend'
結構中的方法已宣告為 `Protected` 或 `Protected Friend`。  
  
 **錯誤 ID:** BC31067  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 將結構內的方法宣告為 `Public` 或 `Private`。
