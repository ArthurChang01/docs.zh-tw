# &#39;&lt;修飾詞&gt;&#39; 在方法宣告中無效
`Function` 或 `Sub` 陳述式包含無效的關鍵字。  
  
 **錯誤 ID︰**BC30242  
  
### 更正這個錯誤  
  
-   請從 `Function` 或 `Sub` 陳述式中移除無效的關鍵字。
