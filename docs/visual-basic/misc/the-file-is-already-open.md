# <a name="the-file-is-already-open"></a>檔案已經開啟
已嘗試開啟已開啟的檔案。  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   關閉檔案，然後再嘗試開啟檔案。  
  
     -或-  
  
-   從您的程式碼中移除多餘的 `FileOpen` 陳述式。
