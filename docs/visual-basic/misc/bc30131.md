# <a name="security-attribute-39ltattributenamegt39-cannot-be-applied-to-a-module"></a>安全性屬性&#39; &lt;attributename&gt; &#39;無法套用至模組
Common Language Runtime 不接受模組上的安全屬性。

**錯誤 ID︰** BC30131

## <a name="to-correct-this-error"></a>更正這個錯誤

- 請從宣告中移除屬性。
