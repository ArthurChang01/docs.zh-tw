# <a name="line-number-cannot-be-parsed-using-the-current-delimiters"></a>行\<數字 > 無法使用目前的分隔符號剖析
無法剖析指定的行，因為使用了未指定的分隔符號。  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 調整 `Delimiters` 以便正確地剖析此行，或插入例外狀況處理程式碼以便處理此行。
