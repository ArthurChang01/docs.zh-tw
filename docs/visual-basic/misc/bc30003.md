# <a name="39next39-expected"></a>&#39;下一步&#39;預期
`On Error Resume` 建構發生卻沒有結束 `Next`。  
  
 **錯誤識別碼：** BC30003  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   將 `Next` 關鍵字加入 `On Error` 陳述式的結尾。
