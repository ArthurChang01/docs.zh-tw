# <a name="next-expected"></a>必須是 'Next'
`On Error Resume` 建構發生卻沒有結束 `Next`。  
  
 **錯誤 ID:** BC30003  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   將 `Next` 關鍵字加入 `On Error` 陳述式的結尾。
