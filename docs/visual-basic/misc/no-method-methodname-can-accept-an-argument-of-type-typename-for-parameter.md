# <a name="no-method-39ltmethodnamegt39-can-accept-an-argument-of-type-39lttypenamegt39-for-parameter-39ltparameternamegt39"></a>沒有方法 &#39;&lt;methodname&gt;&#39; 可以接受引數的類型 &#39;&lt;typename&gt;&#39; 參數 &#39;&lt;參數名稱&gt;&#39;
方法呼叫中包含無效類型的引數。  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
1.  請檢查引數的類型。  
  
2.  比對引數類型與方法的參數類型，確定它們相符。 視需要變更類型。
