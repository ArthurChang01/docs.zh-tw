# <a name="no-method-methodname-can-accept-an-argument-of-type-typename-for-parameter-parametername"></a>沒有方法 '\<方法名稱 >' 可接受類型的引數'\<類型名稱 >' 參數 '\<參數名稱 >'
方法呼叫中包含無效類型的引數。  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
1. 請檢查引數的類型。  
  
2. 比對引數類型與方法的參數類型，確定它們相符。 視需要變更類型。
