# <a name="no-method-ltmethodnamegt-can-accept-an-argument-of-type-lttypenamegt-for-parameter-ltparameternamegt"></a>沒有方法 '&lt;methodname&gt;'可接受類型的引數'&lt;typename&gt;'的參數'&lt;parametername&gt;'
方法呼叫中包含無效類型的引數。  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
1.  請檢查引數的類型。  
  
2.  比對引數類型與方法的參數類型，確定它們相符。 視需要變更類型。
