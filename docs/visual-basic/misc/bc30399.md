# <a name="39mybase39-cannot-be-used-with-method-39ltmethodnamegt39-because-it-is-declared-39mustoverride39"></a>&#39;MyBase&#39;不能與方法&#39; &lt;methodname&gt; &#39;因為它宣告為&#39;MustOverride&#39;
您嘗試搭配使用 `MyBase` 與已宣告 `MustOverride`的方法。  
  
 **錯誤 ID︰** BC30399  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請移除 `MustOverride` 宣告。
