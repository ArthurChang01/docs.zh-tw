# <a name="mybase-cannot-be-used-with-method-methodname-because-it-is-declared-mustoverride"></a>' MyBase ' 不能與方法 '\<方法名稱 > ' 搭配使用, 因為它已宣告為 ' MustOverride '
您嘗試搭配使用 `MyBase` 與已宣告 `MustOverride`的方法。  
  
 **錯誤識別碼:** BC30399  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 請移除 `MustOverride` 宣告。
