# <a name="xml-comment-tag-include-must-have-a-attribute-attribute"></a>XML 註解標記 'include' 必須有 '\<屬性 >' 屬性
XML 註解標記 'include' 必須有\<屬性 > 屬性。 將會忽略 XML 註解。  
  
 傳遞給 `file` Include `path` 標記的 `<`或`>` 屬性遺漏或不完整。  
  
 **錯誤 ID:** BC42310  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 加入或完成必要的屬性，以\<包含 > 標記。
