# <a name="39notoverridable39-cannot-be-specified-on-methods-that-do-not-override-another-method"></a>&#39;NotOverridable&#39;不可指定於不覆寫其他方法的方法
根據預設，屬性和方法是 `NotOverridable` `NotOverridable` 修飾詞只能用於覆寫另一個屬性或方法的方法上。  
  
 **錯誤 ID︰** BC31088  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   從不覆寫另一個成員的屬性和方法中，移除 `NotOverridable` 修飾詞。
