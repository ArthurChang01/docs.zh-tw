# <a name="shared-sub-new-cannot-have-any-parameters"></a>共用的 'Sub New' 不可以有任何參數
您已在共用 `Sub New` 陳述式中加入一個或多個參數，這是無效的作業。  
  
 **錯誤 ID:** BC30479  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 請移除一個或多個參數。
