# <a name="39raiseevent39-statements-are-not-valid-in-the-immediate-window"></a>'RaiseEvent' 陳述式不是有效的即時運算視窗中
`RaiseEvent`陳述式不能在**即時運算**視窗。  
  
 **錯誤 ID︰** BC30739  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   在 [程式碼編輯器] `RaiseEvent`**中，透過程式碼測試**陳述式。
