# <a name="argument-argumentname-is-not-a-valid-value"></a>引數 '\<引數名稱 >' 不是有效的值
引數的值無效。  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
1.  請檢查引數的值。  
  
2.  如果透過計算來提供引數，請檢查計算所處理的值。
