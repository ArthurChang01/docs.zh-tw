# <a name="cannot-inherit-interface-39ltinterfacename1gt39-because-it-could-be-identical-to-interface-39ltinterfacename2gt39-from-which-the-interface-39ltinterfacename3gt39-inherits-for-some-type-arguments"></a><span data-ttu-id="35924-102">無法繼承介面 &#39;&lt;介面名稱 1>.<&gt;&#39;，因為它可能與相同介面 &#39;&lt;介面名稱 2>&gt;&#39; 從中介面 &#39;&lt;interfacename3&gt;&#39; 繼承某些型別引數</span><span class="sxs-lookup"><span data-stu-id="35924-102">Cannot inherit interface &#39;&lt;interfacename1&gt;&#39; because it could be identical to interface &#39;&lt;interfacename2&gt;&#39; from which the interface &#39;&lt;interfacename3&gt;&#39; inherits for some type arguments</span></span>
<span data-ttu-id="35924-103">泛型介面繼承自兩個以上的泛型介面，而且兩個繼承的類型引數之特定值可能發生衝突。</span><span class="sxs-lookup"><span data-stu-id="35924-103">A generic interface inherits from two or more generic interfaces, and two of the inheritances could conflict for certain values of type arguments.</span></span>  
  
 <span data-ttu-id="35924-104">下列陳述式可能會產生此錯誤。</span><span class="sxs-lookup"><span data-stu-id="35924-104">The following statements can generate this error.</span></span>  
  
```

End Interface  
Public Interface interfaceX(Of v)  
    Inherits interfaceA(Of v)  
End Interface  
Public Interface derivedInterface(Of t1, t2)  
    Inherits interfaceA(Of t1), interfaceX(Of t2)  
End Interface  
```  
  
 <span data-ttu-id="35924-105">如果建構或實作 `derivedInterface` 時針對 `t1` 和 `t2`提供相同的類型，其一定會繼承含相同類型引數之 `interfaceA` 的兩個版本。</span><span class="sxs-lookup"><span data-stu-id="35924-105">If `derivedInterface` is constructed or implemented supplying the same type to both `t1` and `t2`, it must inherit two versions of `interfaceA` with identical type arguments.</span></span> <span data-ttu-id="35924-106">這樣會造成無法確定存取哪一個版本的情況。</span><span class="sxs-lookup"><span data-stu-id="35924-106">Doing so would produce an ambiguity about which version to access.</span></span>
