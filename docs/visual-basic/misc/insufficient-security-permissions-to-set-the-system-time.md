# <a name="insufficient-security-permissions-to-set-the-system-time"></a>安全性權限不足，無法設定系統時間
安全性權限不足，無法設定系統時間。  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
1. 變更安全性權限。
