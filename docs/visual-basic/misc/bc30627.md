# <a name="option-statements-must-precede-any-declarations-or-imports-statements"></a>'Option' 陳述式必須在所有的宣告或 'Imports' 陳述式之前
`Option` 陳述式應該是您原始程式碼中的第一個非註解陳述式。  
  
 **錯誤 ID:** BC30627  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   將 `Option` 陳述式移至原始程式碼的上方，緊接在 `Imports` 和 `Namespace` 陳述式之前。
