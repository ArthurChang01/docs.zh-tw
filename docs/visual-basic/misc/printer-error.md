# 圖片錯誤
印表機發生錯誤，但未將任何其他資訊傳回給傳送檔案的電腦。  
  
### 更正這個錯誤  
  
-   請實際檢查印表機。 請確定電腦與印表機之間的所有連線都穩固。 大多數印表機都會顯示錯誤資訊 \(例如 \[離線\] 或 \[紙張用完\]\)。
