# <a name="cannot-inherit-interface-ltinterfacename1gt-because-the-interface-ltinterfacename2gt-from-which-it-inherits-could-be-identical-to-interface-ltinterfacename3gt-from-which-the-interface-ltinterfacename4gt-inherits-for-some-type-arguments"></a><span data-ttu-id="e3f06-102">無法繼承介面 '&lt;介面名稱 1>.<&gt;' 因為介面 '&lt;介面名稱 2&gt&gt;'所繼承它可能是與介面相同。'&lt;3>&gt;'來源介面 '&lt;interfacename4&gt;' 繼承某些型別引數</span><span class="sxs-lookup"><span data-stu-id="e3f06-102">Cannot inherit interface '&lt;interfacename1&gt;' because the interface '&lt;interfacename2&gt;' from which it inherits could be identical to interface '&lt;interfacename3&gt;' from which the interface '&lt;interfacename4&gt;' inherits for some type arguments</span></span>
<span data-ttu-id="e3f06-103">泛型介面繼承自兩個以上的泛型介面，而且兩個繼承的類型引數之特定值可能發生衝突。</span><span class="sxs-lookup"><span data-stu-id="e3f06-103">A generic interface inherits from two or more generic interfaces, and two of the inheritances could conflict for certain values of type arguments.</span></span>  
  
 <span data-ttu-id="e3f06-104">下列陳述式可能會產生此錯誤。</span><span class="sxs-lookup"><span data-stu-id="e3f06-104">The following statements can generate this error.</span></span>  
  
```

End Interface  
Public Interface interfaceX(Of v)  
    Inherits interfaceA(Of v)  
End Interface  
Public Interface interfaceY(Of w)  
    Inherits interfaceA(Of w)  
End Interface  
Public Interface derivedInterface(Of t1, t2)  
    Inherits interfaceX(Of t1), interfaceY(Of t2)  
End Interface  
```  
  
 <span data-ttu-id="e3f06-105">如果建構或實作 `derivedInterface` 時針對 `t1` 和 `t2`提供相同的類型，其一定會繼承含相同類型引數之 `interfaceA` 的兩個版本。</span><span class="sxs-lookup"><span data-stu-id="e3f06-105">If `derivedInterface` is constructed or implemented supplying the same type to both `t1` and `t2`, it must inherit two versions of `interfaceA` with identical type arguments.</span></span> <span data-ttu-id="e3f06-106">這樣會造成無法確定存取哪一個版本的情況。</span><span class="sxs-lookup"><span data-stu-id="e3f06-106">Doing so would produce an ambiguity about which version to access.</span></span>
