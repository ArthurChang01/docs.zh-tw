<span data-ttu-id="b8705-103">泛型介面繼承自兩個以上的泛型介面，而且兩個繼承的類型引數之特定值可能發生衝突。</span><span class="sxs-lookup"><span data-stu-id="b8705-103">A generic interface inherits from two or more generic interfaces, and two of the inheritances could conflict for certain values of type arguments.</span></span>
