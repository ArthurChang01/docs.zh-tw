# <a name="cannot-inherit-interface-interfacename1-because-the-interface-interfacename2-from-which-it-inherits-could-be-identical-to-interface-interfacename3-from-which-the-interface-interfacename4-inherits-for-some-type-arguments"></a><span data-ttu-id="a8b29-102">無法繼承介面 '\<介面名稱 1> > ', 因為它所\<繼承的介面 ' 介面名稱 2> > ' 可能\<與介面 '\<介面 3> > ' 的介面 '4> > ' 針對某些類型引數繼承</span><span class="sxs-lookup"><span data-stu-id="a8b29-102">Cannot inherit interface '\<interfacename1>' because the interface '\<interfacename2>' from which it inherits could be identical to interface '\<interfacename3>' from which the interface '\<interfacename4>' inherits for some type arguments</span></span>

<span data-ttu-id="a8b29-103">泛型介面繼承自兩個以上的泛型介面，而且兩個繼承的類型引數之特定值可能發生衝突。</span><span class="sxs-lookup"><span data-stu-id="a8b29-103">A generic interface inherits from two or more generic interfaces, and two of the inheritances could conflict for certain values of type arguments.</span></span>

 <span data-ttu-id="a8b29-104">下列陳述式可能會產生此錯誤。</span><span class="sxs-lookup"><span data-stu-id="a8b29-104">The following statements can generate this error.</span></span>

```vb
Public Interface interfaceA(Of u)
End Interface
Public Interface interfaceX(Of v)
    Inherits interfaceA(Of v)
End Interface
Public Interface interfaceY(Of w)
    Inherits interfaceA(Of w)
End Interface
Public Interface derivedInterface(Of t1, t2)
    Inherits interfaceX(Of t1), interfaceY(Of t2)
End Interface
```

<span data-ttu-id="a8b29-105">如果建構或實作 `derivedInterface` 時針對 `t1` 和 `t2`提供相同的類型，其一定會繼承含相同類型引數之 `interfaceA` 的兩個版本。</span><span class="sxs-lookup"><span data-stu-id="a8b29-105">If `derivedInterface` is constructed or implemented supplying the same type to both `t1` and `t2`, it must inherit two versions of `interfaceA` with identical type arguments.</span></span> <span data-ttu-id="a8b29-106">這樣會造成無法確定存取哪一個版本的情況。</span><span class="sxs-lookup"><span data-stu-id="a8b29-106">Doing so would produce an ambiguity about which version to access.</span></span>
