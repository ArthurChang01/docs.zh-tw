# <a name="operator-without-an-39as39-clause-type-of-object-assumed"></a>運算子不含 '，' 」 子句中。假設是物件的類型
運算子程序未指定 `As` 子句。  
  
 `As` 子句會識別要與程式設計項目相關聯的資料類型。 在[Operator 陳述式](../../visual-basic/language-reference/statements/operator-statement.md)，它會指定運算子程序會傳回呼叫程式碼的值的資料型別。 如果您未在 `As` 陳述式中包含 `Operator` 子句，傳回資料類型會預設為 `Object`。  
  
 根據預設，這個訊息是一個警告。 如需隱藏警告或將警告視為錯誤的相關資訊，請參閱 [Configuring Warnings in Visual Basic](https://docs.microsoft.com/visualstudio/ide/configuring-warnings-in-visual-basic)。  
  
 **錯誤 ID︰** BC41005  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請在 `As` 陳述式中包含 `Operator` 子句，以指定傳回資料類型。
