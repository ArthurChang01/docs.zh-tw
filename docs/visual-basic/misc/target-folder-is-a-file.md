# <a name="target-folder-is-a-file"></a>目標資料夾是一個檔案
嘗試執行資料夾相關動作時，您將指定檔案。  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請檢查所提供的路徑，確定您將指定資料夾，而非檔案。
