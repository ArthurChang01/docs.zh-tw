# <a name="xml-declaration-does-not-allow-attribute-attributename"></a>XML 宣告不允許屬性 'attributeName'
已指定 XML 文件常值的無效屬性。 XML 文件常值的有效屬性值是以 XML 1.0 規格為基礎。  
  
 **錯誤 ID:** BC31154  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請從 XML 文件常值中移除無效的屬性。
