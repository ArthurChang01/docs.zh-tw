# <a name="lteventnamegt-is-not-an-event-of-ltcontainernamegt"></a>'&lt;eventname&gt;'不是事件的'&lt;containername&gt;'
未針對物件宣告指定的事件。  
  
 **錯誤 ID:** BC30676  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
1.  檢查事件名稱的拼字。  
  
2.  確認您存取的物件正確。 使用完整的物件參考，或視需要使用 `Imports` 陳述式匯入適當的命名空間。
