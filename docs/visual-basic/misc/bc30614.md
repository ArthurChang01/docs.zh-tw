# <a name="mustoverride-method-ltmethodnamegt-cannot-be-called-with-myclass"></a>'MustOverride' 方法 '&lt;methodname&gt;' 無法以 'MyClass' 呼叫
`MyClass` 相當於 `Me`，但其上的所有方法引動過程都會當做所要叫用的方法為 `NotOverridable`。  
  
 **錯誤 ID:** BC30614  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   移除 `MustOverride` 修飾詞，或是在基底類別中宣告此方法，然後繼承並覆寫該類別。
