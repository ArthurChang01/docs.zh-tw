# <a name="project-level-conditional-compilation-constant-is-not-valid-lterrorgt"></a>專案層級的條件式編譯常數無效：&lt;錯誤&gt;
專案指定了無效的條件式編譯選項。  
  
 **錯誤 ID:** BC31031  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
1.  請確認與這個專案搭配使用的條件式編譯常數。
