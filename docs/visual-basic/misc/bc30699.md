# <a name="membername-is-not-declared-or-the-module-containing-it-is-not-loaded-in-the-debugging-session"></a>'\<成員名稱 >' 未宣告或包含它的模組未載入偵錯工作階段中
找不到指定的成員。  
  
 **錯誤 ID:** BC30699  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
1. 請確認已定義成員，而且拼寫正確。  
  
2. 存取模組中所宣告的其中一個成員。 在某些情況下，偵錯環境找不到成員，因為尚未載入宣告它們的模組。
