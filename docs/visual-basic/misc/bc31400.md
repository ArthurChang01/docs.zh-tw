# <a name="local-variables-within-methods-of-structures-cannot-be-declared-static"></a>結構方法中的區域變數無法宣告為 'Static'
`Static` 修飾詞不能與結構中的區域變數搭配使用。  
  
 **錯誤 ID:** BC31400  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
1.  請從區域變數中移除 `Static` 修飾詞。  
  
2.  將變數宣告為具有較廣範圍的靜態變數。
