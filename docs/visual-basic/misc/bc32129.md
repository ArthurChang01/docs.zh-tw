# <a name="enum-types-cannot-be-nullable"></a>列舉類型不可為 Null
用來宣告列舉的基礎類型不可為 Null。 例如，下列程式碼會造成此錯誤：  
  
```vb  
'' Not valid.  
' Enum exampleEnum As Integer?  
'     Member declarations.  
' End Enum  
```  
  
 **錯誤 ID:** BC32129  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請不要在 `Enum` 宣告中使用可為 Null 的基礎類型。
