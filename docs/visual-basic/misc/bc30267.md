# <a name="ltdeclaration1gt-cannot-override-ltdeclaration2gt-because-it-is-declared-notoverridable"></a>'&lt;declaration1&gt;'無法覆寫'&lt;declaration2&gt;' 因為其宣告為 'NotOverridable'
程序或屬性宣告嘗試覆寫相同名稱的繼承項目，但繼承項目指定為 `NotOverridable`。  
  
 **錯誤 ID:** BC30267  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   從繼承項目的宣告中移除 `NotOverridable` 關鍵字，或移除覆寫宣告。
