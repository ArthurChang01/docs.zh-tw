# <a name="declaration1-cannot-override-declaration2-because-it-is-declared-notoverridable"></a>'\<declaration1 >' 無法覆寫'\<declaration2 >' 因為其宣告為 'NotOverridable'
程序或屬性宣告嘗試覆寫相同名稱的繼承項目，但繼承項目指定為 `NotOverridable`。  
  
 **錯誤 ID:** BC30267  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 從繼承項目的宣告中移除 `NotOverridable` 關鍵字，或移除覆寫宣告。
