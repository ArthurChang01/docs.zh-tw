# 無法執行所要求的作業 (Visual Basic)
作業無法執行，因為它會使專案的目前狀態無效。 例如，您可能嘗試以程式設計方式修改正在執行的程式碼。  
  
### 更正這個錯誤  
  
-   請停止執行程式碼、進行必要的修改，然後重試作業。
