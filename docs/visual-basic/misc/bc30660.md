# <a name="attributes-cannot-be-applied-to-local-variables"></a>屬性不可以套用至區域變數
屬性不可以套用至區域變數。  
  
 **錯誤 ID：** BC30660  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
1.  在更高一層的範圍宣告這個變數，例如類別或模組層級。
