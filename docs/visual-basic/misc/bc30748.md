# <a name="cannot-convert-to-type"></a>無法轉換為 '\<類型 >'
嘗試使用無效的轉換。  
  
 **錯誤 ID:** BC30748  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 請選擇相容的轉換類型。
