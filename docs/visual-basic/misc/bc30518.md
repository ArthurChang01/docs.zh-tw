# <a name="overload-resolution-failed-because-no-accessible-method-can-be-called-with-these-argumentserror"></a>多載解析失敗，因為沒有可存取 '\<方法 >' 無法使用這些引數呼叫：\<錯誤 >
您已呼叫多載的方法，但編譯器找不到具有相符引數清單的方法。  
  
 **錯誤 ID:** BC30518  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 請檢查您提供的引數清單。
