# <a name="overload-resolution-failed-because-no-accessible-ltmethodgt-can-be-called-with-these-argumentslterrorgt"></a>多載解析失敗，因為沒有可存取 '&lt;方法&gt;' 可以使用這些引數呼叫：&lt;錯誤&gt;
您已呼叫多載的方法，但編譯器找不到具有相符引數清單的方法。  
  
 **錯誤 ID:** BC30518  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請檢查您提供的引數清單。
