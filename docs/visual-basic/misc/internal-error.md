# <a name="internal-error-visual-basic"></a>內部錯誤 (Visual Basic)
在 Visual Basic 中發生內部問題。  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 確定這個錯誤不是由 `Error` 陳述式或 `Raise` 方法所產生。 除非是，否則請連絡 Microsoft 產品支援服務，並回報出現訊息的情況。
