# <a name="catch-must-end-with-a-matching-end-try"></a>'Catch' 之後必須搭配相對應的 'End Try'
`Catch` 陳述式出現在您的程式碼中，但沒有對應的 `End Try` 陳述式。 `Catch` 陳述式必須以 `End Try` 陳述式為結尾。  
  
 **錯誤 ID:** BC30441  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
1.  請移除 `Catch` 陳述式。  
  
2.  加入 `End Try` 陳述式，以作為區塊的結尾。
