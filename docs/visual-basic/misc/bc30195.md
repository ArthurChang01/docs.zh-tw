# <a name="expected-one-of-39dim39-39const39-39public39-39private39-39protected39-39friend39-39shadows39-39readonly39-or-39shared39"></a>需要的是&#39;Dim&#39;， &#39;Const&#39;，&#39;公用&#39;，&#39;私人&#39;，&#39;保護&#39;， &#39;Friend&#39;，&#39;陰影&#39;， &#39;ReadOnly&#39;或&#39;共用&#39;
宣告陳述式遺漏宣告關鍵字。 其中一個可能的原因是屬性宣告呼叫方法。  
  
 **錯誤 ID︰** BC30195  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
1.  確認已在屬性宣告內宣告方法。  
  
2.  使用適當的宣告關鍵字作為陳述式的開頭。
