# <a name="types-declared-private-must-be-inside-another-type"></a>宣告為 'Private' 的類型必須在另一個類型內
`Private` 修飾詞用在不是另一種類型內的類型。  
  
 **錯誤 ID:** BC31089  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
1.  使用較不嚴格的存取修飾詞 (例如 `Friend`)。  
  
2.  在另一種類型內宣告類型。
