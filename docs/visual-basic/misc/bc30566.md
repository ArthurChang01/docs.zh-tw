# <a name="array-initializer-has-too-many-dimensions"></a>陣列初始設定式擁有太多維度
您在初始設定式中提供的陣列維度太多。  
  
 **錯誤 ID:** BC30566  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請檢查您的陣列初始設定式，以判斷需要多少維度。  
  
-   請移除額外的引數。
