# <a name="39sub-main39-is-declared-more-than-once-in-39ltnamespacegt39-ltmessagegt"></a>&#39;Sub Main &#39;宣告一次在超過 &#39;&lt;命名空間&gt;&#39;:&lt;訊息&gt;
`Sub Main` 只能在命名空間內宣告一次。  
  
 **錯誤 ID︰** BC30738  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   確定您的專案中只有一個 `Sub Main` 程序。
