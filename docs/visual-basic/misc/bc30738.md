# <a name="sub-main-is-declared-more-than-once-in-ltnamespacegt-ltmessagegt"></a>'Sub Main' 中多次宣告 '&lt;命名空間&gt;':&lt;訊息&gt;
`Sub Main` 只能在命名空間內宣告一次。  
  
 **錯誤 ID:** BC30738  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   確定您的專案中只有一個 `Sub Main` 程序。
