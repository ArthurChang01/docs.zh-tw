# <a name="39readonly39-and-39writeonly39-cannot-be-combined"></a>&#39;ReadOnly&#39;和&#39;WriteOnly&#39;無法合併
`ReadOnly` 與 `WriteOnly` 修飾詞互斥。  
  
 **錯誤 ID︰** BC30635  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請指定 `ReadOnly` 或 `WriteOnly`，但兩者不能同時指定。
