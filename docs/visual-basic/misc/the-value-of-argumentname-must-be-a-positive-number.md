# <a name="the-value-of-ltargumentnamegt-must-be-a-positive-number"></a>值&lt;引數名稱&gt;必須是正數
<xref:Microsoft.VisualBasic.Logging.FileLogTraceListener.ReserveDiskSpace%2A> 屬性的值必須大於或等於零。  
  
 <xref:Microsoft.VisualBasic.Logging.FileLogTraceListener.ReserveDiskSpace%2A> 屬性必須先指定以位元組為單位的可用磁碟空間量，訊息才能寫入記錄檔。  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   將 <xref:Microsoft.VisualBasic.Logging.FileLogTraceListener.ReserveDiskSpace%2A> 屬性設為正數。
