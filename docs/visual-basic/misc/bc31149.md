# <a name="duplicate-xml-attribute-attributename"></a>重複的 XML 屬性 'attributeName'
為 XML 元素常值指定的屬性與為 XML 元素常值預先定義的屬性相符。 XML 元素常值中的屬性名稱必須是唯一的。  
  
 **錯誤 ID:** BC31149  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   將屬性重新命名為對 XML 元素常值是唯一的名稱。
