# &#39;&lt;類型名稱&gt;&#39; 是不支援的類型
[!INCLUDE[vsprvs](../../csharp/includes/vsprvs-md.md)] 無法表示指定的類型。  
  
 **錯誤 ID：**BC30649  
  
### 更正這個錯誤  
  
-   使用支援的類型。  
  
-   開發新的類型。
