# <a name="lttypenamegt-is-an-unsupported-type"></a>'&lt;typename&gt;' 是不支援的類型
Visual studio 無法表示指定的型別。  
  
 **錯誤 ID:** BC30649  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   使用支援的類型。  
  
-   開發新的類型。
