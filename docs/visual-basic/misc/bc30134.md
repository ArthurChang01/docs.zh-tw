# <a name="labels-are-not-valid-in-the-immediate-window"></a>即時運算視窗中的標籤無效
線條標籤啟用分支，但偵錯內容不允許。  
  
 **錯誤 ID：** BC30134  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請勿在 [即時運算]  視窗中輸入行標籤。
