# <a name="option-strict-on-requires-each-lambda-expression-parameter-to-be-declared-with-an-as-clause-if-its-type-cannot-be-inferred"></a><span data-ttu-id="e7409-102">Option Strict On 要求，如果無法推斷每個 Lambda 運算式參數的類型，就必須使用 'As' 子句來宣告這些參數</span><span class="sxs-lookup"><span data-stu-id="e7409-102">Option Strict On requires each lambda expression parameter to be declared with an 'As' clause if its type cannot be inferred</span></span>
<span data-ttu-id="e7409-103">您已經在 Lambda 運算式中宣告參數，未使用 `As` 子句， `Option Strict` 為 On。</span><span class="sxs-lookup"><span data-stu-id="e7409-103">You have declared a parameter in a lambda expression without using an `As` clause, with `Option Strict` on.</span></span>  
  
```  
' Not valid when Option Strict is on.  
' Dim increment1 = Function (n) n + 1  
```  
  
 <span data-ttu-id="e7409-104">如果可以推斷 `n` 的類型，上一個宣告即有效。</span><span class="sxs-lookup"><span data-stu-id="e7409-104">The previous declaration is valid if the type of `n` can be inferred.</span></span> <span data-ttu-id="e7409-105">例如，如果您將前一個 Lambda 運算式指派給函式委派 `Del`：</span><span class="sxs-lookup"><span data-stu-id="e7409-105">For example, if you are assigning the previous lambda expression to a function delegate, `Del`:</span></span>  
  
```  
Delegate Function Del(ByVal p As Integer) As Integer  
```  
  
 <span data-ttu-id="e7409-106">現在就可以從參數 `n` 推斷 `p`的類型：</span><span class="sxs-lookup"><span data-stu-id="e7409-106">Now the type of `n` can be inferred from parameter `p`:</span></span>  
  
```  
Dim increment2 as Del = Function(n) n + 1  
```  
  
 <span data-ttu-id="e7409-107">**錯誤 ID:** BC36642</span><span class="sxs-lookup"><span data-stu-id="e7409-107">**Error ID:** BC36642</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="e7409-108">更正這個錯誤</span><span class="sxs-lookup"><span data-stu-id="e7409-108">To correct this error</span></span>  
  
- <span data-ttu-id="e7409-109">將 `As` 子句加入參數宣告中：</span><span class="sxs-lookup"><span data-stu-id="e7409-109">Add an `As` clause to the parameter declaration:</span></span>  
  
    ```  
    Dim increment3 = Function (n As Integer) n + 1  
    ```
