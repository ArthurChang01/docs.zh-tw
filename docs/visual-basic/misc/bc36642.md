```  
' Dim increment1 = Function (n) n + 1  
```  
  
 <span data-ttu-id="8f6c9-104">如果可以推斷 `n` 的類型，上一個宣告即有效。</span><span class="sxs-lookup"><span data-stu-id="8f6c9-104">The previous declaration is valid if the type of `n` can be inferred.</span></span> <span data-ttu-id="8f6c9-105">例如，如果您將前一個 Lambda 運算式指派給函式委派 `Del`：</span><span class="sxs-lookup"><span data-stu-id="8f6c9-105">For example, if you are assigning the previous lambda expression to a function delegate, `Del`:</span></span>
