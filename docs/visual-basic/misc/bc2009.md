# <a name="option-ltoptionnamegt-can-be-followed-only-by-3939-or-39-39"></a>選項&lt;選項名稱&gt;後面只能接 &#39; + （& s) #39; 或 （& s) #39;-（& s) #39;
選項之後的文字不是加號 (+) 或減號 (-)。  
  
 **錯誤 ID︰** BC2009  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   以加號 (+) 取代選項之後的文字，以開啟選項。  
  
     — 或 —  
  
-   以減號 (-) 取代選項之後的文字，以關閉選項。
