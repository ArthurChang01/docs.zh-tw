# <a name="option-ltoptionnamegt-can-be-followed-only-by--or--"></a>選項&lt;optionname&gt;可以之後只能是 '+' 或 '-'
選項之後的文字不是加號 (+) 或減號 (-)。  
  
 **錯誤 ID:** BC2009  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   以加號 (+) 取代選項之後的文字，以開啟選項。  
  
     — 或 —  
  
-   以減號 (-) 取代選項之後的文字，以關閉選項。
