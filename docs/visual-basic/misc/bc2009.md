# 選項 &lt;選項名稱&gt; 之後只能是 &#39;+&#39; 或 &#39;-&#39;
選項之後的文字不是加號 \(\+\) 或減號 \(\-\)。  
  
 **錯誤 ID︰**BC2009  
  
### 更正這個錯誤  
  
-   以加號 \(\+\) 取代選項之後的文字，以開啟選項。  
  
     — 或 —  
  
-   以減號 \(\-\) 取代選項之後的文字，以關閉選項。
