# <a name="39ltstatementnamegt39-statement-requires-an-array"></a>&#39;&lt;陳述式名稱&gt;&#39; 陳述式需要陣列
`Erase` 和 `ReDim` 陳述式需要陣列資料類型。 提供的類型不是陣列。  
  
 **錯誤 ID：** BC30049  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   將資料類型變更為陣列類型。
