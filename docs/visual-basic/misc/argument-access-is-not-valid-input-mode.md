# <a name="argument-access-is-not-valid-valid-values-for-input-mode-are-openaccessread-and-openaccessdefault"></a>引數 'Access' 無效。 Input 模式的有效值是 'OpenAccess.Read' 和 'OpenAccess.Default'
存取值在 `Input` 模式中無效。  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 將此值變更為 `OpenAccess.Read` 或 `OpenAccess.Default`。
