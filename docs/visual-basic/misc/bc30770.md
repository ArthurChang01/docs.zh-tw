# <a name="event-lteventnamegt-event-specified-by-the-defaultevent-attribute-is-not-a-publicly-accessible-event-for-this-class"></a>事件 '&lt;eventname&gt;' 'DefaultEvent' 屬性所指定的事件就不會公開存取的事件，此類別
<xref:System.ComponentModel.DefaultEventAttribute> 屬性必須在要套用屬性的類別中，指定可公開存取事件的名稱。  
  
 **錯誤 ID:** BC30770  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
1.  請確定類別會定義可公開存取的事件。  
  
2.  請確定類別中的事件名稱符合 <xref:System.ComponentModel.DefaultEventAttribute> 屬性指定的名稱。
