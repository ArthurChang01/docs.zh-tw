# <a name="39systemruntimeinteropservicesdllimportattribute39-cannot-be-applied-to-a-sub-function-or-operator-with-a-non-empty-body"></a>&#39;System.Runtime.InteropServices.DllImportAttribute &#39;無法套用至子程序、 函數或運算子，非空白主體
`DllImportAttribute` 屬性已套用至不是空的 `Sub`、 `Function`或 `Operator` 。  
  
 **錯誤 ID︰** BC31522  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請移除 `Sub`、 `Function`或 `Operator` 中的所有程式碼，以使用這個屬性。
