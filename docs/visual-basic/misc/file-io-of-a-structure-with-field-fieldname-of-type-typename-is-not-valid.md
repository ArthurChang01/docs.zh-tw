# <a name="file-io-of-a-structure-with-field-39ltfieldnamegt39-of-type-39lttypenamegt39-is-not-valid"></a>結構與欄位 &#39; 的檔案 I/O&lt;fieldname&gt;&#39; 類型 &#39;&lt;typename&gt;&#39; 不正確
您嘗試寫入檔案的結構包含不支援的類型。  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請檢查結構以決定不受支援類型的發生位置，然後將它變更為有效的類型。
