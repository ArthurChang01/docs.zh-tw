# <a name="file-io-of-a-structure-with-field-39ltfieldnamegt39-of-type-39lttypenamegt39-is-not-valid"></a>檔案 I/O 的結構與欄位&#39; &lt;fieldname&gt; &#39;型別的&#39; &lt;typename&gt; &#39;無效
您嘗試寫入檔案的結構包含不支援的類型。  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請檢查結構以決定不受支援類型的發生位置，然後將它變更為有效的類型。
