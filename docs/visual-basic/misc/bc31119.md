# <a name="39end-removehandler39-must-be-the-first-statement-on-a-line"></a>&#39;End RemoveHandler &#39;必須是該行的第一個陳述式
`End RemoveHandler` 陳述式接在冒號 (:) 陳述式分隔符號的後面。 `End RemoveHandler` 必須是其原始程式碼行上的唯一陳述式。  
  
 **錯誤 ID︰** BC31119  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請將多個陳述式分成不同的行。
