# <a name="39end-removehandler39-must-be-preceded-by-a-matching-39removehandler39-declaration"></a>&#39;End RemoveHandler&#39;之前必須搭配相對應的&#39;RemoveHandler&#39;宣告
`End RemoveHandler` 陳述式沒有相對應的 `RemoveHandler` 陳述式。 `End RemoveHandler` 之前必須搭配相對應的 `RemoveHandler` 陳述式。  
  
 **錯誤識別碼：** BC31125  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請檢查前面的 `RemoveHandler` 陳述式是否有效且拼寫正確。
