# <a name="cannot-calculate-number-of-periods-using-the-arguments-provided"></a>無法使用提供的引數來計算週期的數量
呼叫`NPer`函式不包含所有必要的引數。  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請確定函式呼叫中包含 `Rate`、 `Prnt` 和 `PV` 值。
