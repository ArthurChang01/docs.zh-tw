# <a name="39lttypenamegt39-is-an-enum-type-and-cannot-be-used-as-an-expression"></a>&#39;&lt;typename&gt; &#39;是列舉類型，無法當成運算式使用
需要運算式的地方就會出現類型名稱。 運算式必須包含變數、常數、常值、屬性和 `Function` 程序呼叫的組合。  
  
 **錯誤 ID︰** BC30107  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請移除類型名稱，然後使用有效的項目建構運算式。
