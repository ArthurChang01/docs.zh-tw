# <a name="expected-3939-at-start-of-an-embedded-expression"></a>必須是 &#39; %= &#39;內嵌運算式的開頭
內嵌運算式的起始識別項 `<%=`未正確編碼。 請注意，您無法在 XML 項目常值的名稱中使用百分比字元 (%)。  
  
 **錯誤 ID︰** BC31179  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   確定內嵌運算式的起始識別項未編碼為 `<%=`。
