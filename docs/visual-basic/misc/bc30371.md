# <a name="module-ltmodulenamegt-cannot-be-used-as-a-type"></a>模組 '&lt;modulename&gt;' 無法用做為類型
您已嘗試使用模組作為類型。 無法使用模組，但可以當成 `Get` 類型使用。  
  
 **錯誤 ID:** BC30371  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請將模組取代為可當成有效類型使用的某個項目。
