# <a name="file39s-open-mode-wasn39t-set-to-a-valid-value"></a>檔案 &#39; s 開啟模式不 &#39; 設為有效的值
針對檔案開啟模式提供的值不正確。 下表列出 <xref:Microsoft.VisualBasic.OpenMode> 列舉的有效值：  
  
|值|模式|  
|-----------|----------|  
|1|`OpenMode.Input`|  
|2|`OpenMode.Output`|  
|4|`OpenMode.Random`|  
|8|`OpenMode.Append`|  
|32|`OpenMode.Binary`|  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   檢查針對檔案開啟模式提供的值。
