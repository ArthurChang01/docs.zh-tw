# <a name="type-typename-is-not-defined-or-the-module-containing-it-is-not-loaded-in-the-debugging-session"></a>未定義類型 '\<typename > '，或未在調試會話中載入包含它的模組
參考的類型無法使用或未定義。  
  
 **錯誤 ID︰** BC30743  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
1. 驗證類型名稱的拼寫正確。  
  
2. 定義目前類別或模組內的類型。
