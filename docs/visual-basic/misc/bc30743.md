# <a name="type-typename-is-not-defined-or-the-module-containing-it-is-not-loaded-in-the-debugging-session"></a>類型 '\<類型名稱 >' 未定義或包含它的模組未載入偵錯工作階段中
參考的類型無法使用或未定義。  
  
 **錯誤 ID:** BC30743  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
1.  驗證類型名稱的拼寫正確。  
  
2.  定義目前類別或模組內的類型。
