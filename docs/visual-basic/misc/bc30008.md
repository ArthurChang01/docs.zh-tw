# <a name="reference-required-to-module-39ltmodulenamegt39-containing-the-base-class-39ltclassnamegt39"></a>需要模組參考&#39; &lt;modulename&gt; &#39;包含基底類別&#39;&lt;類別名稱&gt;&#39;
需要模組參考 '\<模組名稱 >' 包含基底類別\<類別名稱 >'。 請在專案中加入一個參考。  
  
 此類別是在專案中未直接參考的模組中所定義。 Visual Basic 編譯器需要參考，以避免發生模稜兩可的情況已定義多個模組中的類別。  
  
 **錯誤 ID︰** BC30008  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請在您的專案參考中包含未參考之模組的名稱。
