# <a name="reference-required-to-module-ltmodulenamegt-containing-the-base-class-ltclassnamegt"></a>需要模組參考 '&lt;modulename&gt;'包含的基底類別'&lt;classname&gt;'
需要模組參考 '\<modulename >' 包含基底類別\<類別名稱 >'。 請在專案中加入一個參考。  
  
 此類別是在專案中未直接參考的模組中所定義。 Visual Basic 編譯器需要參考，以避免模稜兩可，以防多個模組中定義的類別。  
  
 **錯誤 ID:** BC30008  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請在您的專案參考中包含未參考之模組的名稱。
