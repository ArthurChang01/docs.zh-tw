# <a name="expected-closing-39gt39-for-xml-tag"></a>必須是右&#39; &gt; &#39; XML 標記
XML 常值的 XML 標記遺漏必要的結尾 '>' 字元。  
  
 **錯誤 ID︰** BC31166  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   在 XML 常值的 XML 標記結尾加入結尾 '>' 字元。
