# <a name="public-member-membername-on-type-typename-not-found"></a>Public 成員 '\<成員名稱 >' 在類型'\<類型名稱 >' 找不到
找不到您嘗試存取的成員。 存取晚期繫結物件的非公用成員時，就會發生這個錯誤。  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
1. 如果成員存在，請確定它已宣告為 `Public`。
