# <a name="public-member-39ltmembernamegt39-on-type-39lttypenamegt39-not-found"></a>Public 成員&#39; &lt;membername&gt; &#39;類型上&#39; &lt;typename&gt; &#39;找不到
找不到您嘗試存取的成員。 存取晚期繫結物件的非公用成員時，就會發生這個錯誤。  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
1.  如果成員存在，請確定它已宣告為 `Public`。
