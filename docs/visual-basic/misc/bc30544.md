# <a name="method-cannot-contain-both-a-39try39-statement-and-an-39on-error39-or-39resume39-statement"></a>方法不可同時包含 &#39;再試一次 &#39;陳述式和 &#39; On Error &#39;或 &#39;繼續 &#39;陳述式
您已搭配 `Try` 或 `On Error` 使用 `Resume`陳述式。  
  
 **錯誤 ID︰** BC30544  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
1.  移除 `On Error` 或 `Resume`。
