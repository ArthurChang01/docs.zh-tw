# <a name="method-cannot-contain-both-a-try-statement-and-an-on-error-or-resume-statement"></a>方法不可以同時包含 'Try' 陳述式以及 'On Error' 或 'Resume' 陳述式
您已搭配 `Try` 或 `On Error` 使用 `Resume`陳述式。  
  
 **錯誤 ID:** BC30544  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
1.  移除 `On Error` 或 `Resume`。
