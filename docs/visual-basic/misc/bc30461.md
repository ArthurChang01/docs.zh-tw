# <a name="classes-cannot-be-declared-specifier"></a>類別不可以宣告為 '\<規範 >'
您已嘗試宣告具有無效規範的 `Class` 或 `Module` 。  
  
 **錯誤 ID:** BC30461  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 請移除規範。
