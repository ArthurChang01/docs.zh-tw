# <a name="no-accessible-overloaded-39ltmethodnamegt39-can-be-called-with-these-arguments-ltlistgt"></a>沒有可存取多載 &#39;&lt;methodname&gt;&#39; 可以使用這些引數呼叫：&lt;清單&gt;
已呼叫多載方法，但沒有任何方法可在未提供引數清單的情況下相符。  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   比對引數清單與您想要呼叫的多載。  
  
-   調整引數清單，讓簽章符合多載。
