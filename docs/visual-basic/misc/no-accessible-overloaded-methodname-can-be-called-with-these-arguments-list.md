# <a name="no-accessible-overloaded-methodname-can-be-called-with-these-arguments-list"></a>沒有可存取的多載 '\<方法名稱 >' 無法使用這些引數呼叫：\<清單 >
已呼叫多載方法，但沒有任何方法可在未提供引數清單的情況下相符。  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 比對引數清單與您想要呼叫的多載。  
  
- 調整引數清單，讓簽章符合多載。
