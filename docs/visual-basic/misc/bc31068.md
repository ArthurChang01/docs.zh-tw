# <a name="delegate-in-an-interface-cannot-be-declared-specifier"></a>介面中的委派不可以宣告 '\<規範 >'
指定的無效修飾詞具有在介面中宣告的委派。  
  
 **錯誤 ID:** BC31068  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
1.  移除修飾詞，例如 `Public`、 `Private`、 `Shared`、 `Friend`、 `Protected`、 `Protected Friend`和 `Overridable`。
