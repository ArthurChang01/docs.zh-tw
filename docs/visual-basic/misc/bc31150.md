# <a name="end-tag-tagname-expected"></a>結束標記\</tagName > 預期
XML 項目常值中的結束標記不符合對應的開始標記。  
  
 **錯誤 ID:** BC31150  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   修改結束 XML 標記，以符合 XML 項目常值的開始 XML 標記。
