# 必須有 &#39;Compare&#39;
`Option` 陳述式指定 `Text` 或 `Binary`，但不指定 `Compare`。  
  
 **錯誤識別碼：**BC30208  
  
### 更正這個錯誤  
  
-   將 `Compare` 關鍵字加入 `Option` 陳述式中，例如 `Option Compare Binary`。
