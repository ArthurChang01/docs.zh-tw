# &#39;&lt;成員&gt;&#39; 已在這個結構中宣告
某個列舉成員已在這個結構中多次宣告。  
  
 **錯誤 ID︰**BC31421  
  
### 更正這個錯誤  
  
-   移除多餘的宣告。
