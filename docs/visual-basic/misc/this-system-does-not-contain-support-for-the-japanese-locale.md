# 此系統沒有包含日本地區設定的支援
您嘗試使用 `VbStrConv.Japanese` 列舉成員，但您的系統不支援日本地區設定。  
  
### 更正這個錯誤  
  
1.  請升級電腦以支援日本地區設定，或在支援該地區設定的電腦上執行應用程式。
