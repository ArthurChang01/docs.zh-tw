# <a name="unable-to-parse-xml-error"></a>無法剖析 XML:\<錯誤 >
所使用的 XML 檔案發生錯誤。  
  
 **錯誤 ID:** BC31023  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請確認 XML 格式正確且有效，而且檔案未損毀。
