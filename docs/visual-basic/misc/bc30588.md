# <a name="omitted-argument-cannot-match-a-paramarray-parameter"></a>省略的引數無法符合 ParamArray 參數
您省略了比對 `ParamArray` 參數所需的引數。 `ParamArray`雖然可讓您將任意數目的引數傳遞至程序，但只能用於引數清單的最後一個引數。  
  
 **錯誤 ID:** BC30588  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   提供省略的引數。
