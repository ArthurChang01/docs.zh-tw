# <a name="39raiseevent39-declaration-must-end-with-a-matching-39end-raiseevent39"></a>&#39;RaiseEvent &#39;宣告的結尾必須符合 &#39;End RaiseEvent &#39;
`RaiseEvent` 宣告的結尾必須是 `End RaiseEvent` 陳述式。  
  
 **錯誤 ID︰** BC31117  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請檢查 `RaiseEvent` 宣告的結尾是否為 `End RaiseEvent` 陳述式。
