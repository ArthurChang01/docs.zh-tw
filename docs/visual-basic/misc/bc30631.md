# &#39;Get&#39; 陳述式之後必須搭配相對應的 &#39;End Get&#39;
`Get` 屬性程序的結尾必須是 `End Get` 陳述式。  
  
 **錯誤 ID︰**BC30631  
  
### 更正這個錯誤  
  
-   請確定 `Get` 屬性程序的結尾為 `End Get` 建構。
