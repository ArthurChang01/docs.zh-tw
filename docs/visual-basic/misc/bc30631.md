# <a name="39get39-statement-must-end-with-a-matching-39end-get39"></a>&#39;取得&#39;陳述式必須搭配相對應的結束&#39;End Get&#39;
`Get` 屬性程序的結尾必須是 `End Get` 陳述式。  
  
 **錯誤 ID︰** BC30631  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請確定 `Get` 屬性程序的結尾為 `End Get` 建構。
