# <a name="get-statement-must-end-with-a-matching-end-get"></a>'Get' 陳述式之後必須搭配相對應的 'End Get'
`Get` 屬性程序的結尾必須是 `End Get` 陳述式。  
  
 **錯誤 ID:** BC30631  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請確定 `Get` 屬性程序的結尾為 `End Get` 建構。
