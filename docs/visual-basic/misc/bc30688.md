# <a name="events-in-interfaces-cannot-be-declared-ltimplementsgt"></a>在介面中的事件不可宣告為 '&lt;實作&gt;'
在介面中宣告的事件不能實作其他介面的事件。  
  
 **錯誤 ID:** BC30688  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
1.  請移除 `Implements` 陳述式。  
  
2.  實作類別或結構中的事件。
