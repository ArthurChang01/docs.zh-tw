# <a name="ltmethod1gt-cannot-override-ltmethod2gt-because-it-expands-the-access-of-the-base-method"></a>'&lt;method1&gt;'無法覆寫'&lt;method2&gt;' 因為它會展開基底方法的存取權
程序指定 `Overrides` ，但宣告較要被覆寫之方法不嚴格的協助工具。 您無法展開協助工具，這表示您無法讓覆寫的方法較它所覆寫的方法更容易存取。 例如，如果基底類別方法是 `Protected`，您無法使用 `Public` 方法覆寫它。  
  
 **錯誤 ID:** BC32203  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請移除 `Overrides` 關鍵字或將存取範圍變更成至少和基底類別方法的協助工具相同嚴格度。
