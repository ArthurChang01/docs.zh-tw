# <a name="modulename-is-a-module-and-cannot-be-referenced-as-an-assembly"></a>'\<模組名稱 >' 是一個模組，而且無法當做組件參考
嘗試將模組當做組件來使用。  
  
 **錯誤 ID:** BC31076  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   變更組件的參考。
