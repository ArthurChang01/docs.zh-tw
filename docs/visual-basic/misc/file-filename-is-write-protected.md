# <a name="file-39ltfilenamegt39-is-write-protected"></a>檔案 &#39;&lt;filename&gt;&#39; 受到寫入
您嘗試寫入的檔案有防寫保護，因此無法予以變更。  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
1.  請將檔案的權限變更為讀取/寫入，或寫入至不同的檔案。
