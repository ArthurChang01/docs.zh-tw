# <a name="39exit39-must-be-followed-by-39sub39-39function39-39property39-39do39-39for39-39while39-39select39-or-39try39"></a>&#39;結束 &#39;後面必須接著 &#39;Sub &#39;、 &#39;函式 &#39;、 &#39;屬性 &#39;，&#39; 請勿 &#39;、 &#39;&#39;，&#39;而 &#39;，&#39;Select &#39;，或 &#39;再試一次 &#39;
`Exit` 陳述式包含無效的關鍵字。 `Exit` 必須指定區塊，以將其中的控制項轉移到區塊後面的陳述式，例如 `End While`。  
  
 **錯誤 ID︰** BC30240  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   在 `Exit`後面加入有效的關鍵字，或移除 `Exit` 陳述式。
