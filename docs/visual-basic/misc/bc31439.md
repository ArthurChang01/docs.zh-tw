# <a name="partial-method-ltmethodnamegt-cannot-use-the-implements-keyword"></a>部分方法 '&lt;methodname&gt;' 不能使用 'Implements' 關鍵字
部分方法定義無法實作介面或介面成員。  
  
 **錯誤 ID:** BC31439  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
1.  從部分方法定義中移除 `Implements` 陳述式。
