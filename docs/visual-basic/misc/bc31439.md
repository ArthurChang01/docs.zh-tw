# <a name="partial-method-39ltmethodnamegt39-cannot-use-the-39implements39-keyword"></a>部分方法&#39; &lt;methodname&gt; &#39;不能使用&#39;實作&#39;關鍵字
部分方法定義無法實作介面或介面成員。  
  
 **錯誤 ID︰** BC31439  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
1.  從部分方法定義中移除 `Implements` 陳述式。
