# <a name="39declare39-statements-in-a-structure-cannot-be-declared-39ltspecifiergt39"></a>&#39;宣告&#39;結構中的陳述式不可以宣告為&#39;&lt;規範&gt;&#39;
`Declare` 宣告的規範在 `Structure` 宣告內無效。  
  
 **錯誤 ID︰** BC30791  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請移除規範。
