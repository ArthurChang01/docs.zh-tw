# 結構中的 &#39;Declare&#39; 陳述式不可以宣告為 &#39;&lt;規範&gt;&#39;
`Declare` 宣告的規範在 `Structure` 宣告內無效。  
  
 **錯誤 ID︰**BC30791  
  
### 更正這個錯誤  
  
-   請移除規範。
