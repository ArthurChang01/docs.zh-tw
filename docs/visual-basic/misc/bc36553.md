<span data-ttu-id="ff54a-103">'Optional' 不能套用至擴充方法的第一個參數。</span><span class="sxs-lookup"><span data-stu-id="ff54a-103">'Optional' cannot be applied to the first parameter of an extension method.</span></span> <span data-ttu-id="ff54a-104">第一個參數會指定要擴充的類型。</span><span class="sxs-lookup"><span data-stu-id="ff54a-104">The first parameter specifies which type to extend.</span></span>  
  
 <span data-ttu-id="ff54a-105">擴充方法的第一個參數會指定方法可擴充的資料類型。</span><span class="sxs-lookup"><span data-stu-id="ff54a-105">The first parameter of an extension method specifies the data type that the method extends.</span></span> <span data-ttu-id="ff54a-106">執行方法時，第一個參數會繫結至叫用方法的資料類型執行個體。</span><span class="sxs-lookup"><span data-stu-id="ff54a-106">When the method is executed, the first parameter is bound to the instance of the data type that invokes the method.</span></span> <span data-ttu-id="ff54a-107">因此，第一個參數是必要項目，不得為選用項目。</span><span class="sxs-lookup"><span data-stu-id="ff54a-107">Therefore, the first parameter is required and cannot be optional.</span></span>  
  
 <span data-ttu-id="ff54a-108">只有第一個參數有此限制。</span><span class="sxs-lookup"><span data-stu-id="ff54a-108">The restriction applies only to the first parameter.</span></span> <span data-ttu-id="ff54a-109">其他參數可以是選擇性或必要，取決於任何其他方法中的相同規則。</span><span class="sxs-lookup"><span data-stu-id="ff54a-109">Other parameters can be optional or not, following the same rules as in any other method.</span></span> <span data-ttu-id="ff54a-110">如需詳細資訊，請參閱 [Parameter List](../../visual-basic/language-reference/statements/parameter-list.md)。</span><span class="sxs-lookup"><span data-stu-id="ff54a-110">For more information, see [Parameter List](../../visual-basic/language-reference/statements/parameter-list.md).</span></span>  
  
 <span data-ttu-id="ff54a-111">**錯誤 ID︰** BC36553</span><span class="sxs-lookup"><span data-stu-id="ff54a-111">**Error ID:** BC36553</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="ff54a-112">更正這個錯誤</span><span class="sxs-lookup"><span data-stu-id="ff54a-112">To correct this error</span></span>

## <a name="example"></a><span data-ttu-id="ff54a-115">範例</span><span class="sxs-lookup"><span data-stu-id="ff54a-115">Example</span></span>  
 <span data-ttu-id="ff54a-116">下列範例中的第一個參數是指定 `Print` 方法可擴充 `String` 資料類型的唯一指示，</span><span class="sxs-lookup"><span data-stu-id="ff54a-116">The first parameter in the following example is the only indication that the `Print` method extends the `String` data type.</span></span> <span data-ttu-id="ff54a-117">因此不可為選擇性參數。</span><span class="sxs-lookup"><span data-stu-id="ff54a-117">Therefore, it cannot be optional.</span></span>  
  
```  
<Extension()>  
Public Sub Print (ByVal str As String)  
    Console.WriteLine(str)  
End Sub  
```  
  
 <span data-ttu-id="ff54a-118">如下所示呼叫擴充方法時， `str` 方法中的參數會繫結至 `greeting`，也就是呼叫 `String` 的 `Print`執行個體。</span><span class="sxs-lookup"><span data-stu-id="ff54a-118">When the extension method is called as follows, parameter `str` in the method is bound to `greeting`, the instance of `String` that calls `Print`.</span></span> <span data-ttu-id="ff54a-119">編譯器會使用 `greeting` 作為擴充方法 `Print`的引數。</span><span class="sxs-lookup"><span data-stu-id="ff54a-119">The compiler uses `greeting` as the argument to extension method `Print`.</span></span>  
  
```  
Dim greeting As String = "Hello"  
greeting.Print()  
```
