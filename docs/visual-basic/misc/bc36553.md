## <a name="example"></a><span data-ttu-id="d4034-115">範例</span><span class="sxs-lookup"><span data-stu-id="d4034-115">Example</span></span>  
 <span data-ttu-id="d4034-116">下列範例中的第一個參數是指定 `Print` 方法可擴充 `String` 資料類型的唯一指示，</span><span class="sxs-lookup"><span data-stu-id="d4034-116">The first parameter in the following example is the only indication that the `Print` method extends the `String` data type.</span></span> <span data-ttu-id="d4034-117">因此不可為選擇性參數。</span><span class="sxs-lookup"><span data-stu-id="d4034-117">Therefore, it cannot be optional.</span></span>  
  
```

Public Sub Print (ByVal str As String)  
    Console.WriteLine(str)  
End Sub  
```  
  
 <span data-ttu-id="d4034-118">如下所示呼叫擴充方法時， `str` 方法中的參數會繫結至 `greeting`，也就是呼叫 `String` 的 `Print`執行個體。</span><span class="sxs-lookup"><span data-stu-id="d4034-118">When the extension method is called as follows, parameter `str` in the method is bound to `greeting`, the instance of `String` that calls `Print`.</span></span> <span data-ttu-id="d4034-119">編譯器會使用 `greeting` 作為擴充方法 `Print`的引數。</span><span class="sxs-lookup"><span data-stu-id="d4034-119">The compiler uses `greeting` as the argument to extension method `Print`.</span></span>  
  
```  
Dim greeting As String = "Hello"  
greeting.Print()  
```
