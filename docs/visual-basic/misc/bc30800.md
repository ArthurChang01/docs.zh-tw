# <a name="method-arguments-must-be-enclosed-in-parentheses"></a>方法引數必須放在括弧之中
管理程序呼叫的規則會在較新版本的 Visual Basic 更簡單。 所有引數必須放在括弧之中。  
  
 **錯誤 ID︰** BC30800  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請將引數清單用括弧括住。例如：  
  
    ```  
    MySub(ArrayIndex, NewValue)  
    ```
