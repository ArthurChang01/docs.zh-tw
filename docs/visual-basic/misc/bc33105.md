# <a name="if-operands-cannot-be-named-arguments"></a><span data-ttu-id="1f5a7-102">'If' 運算元不可以命名為引數。</span><span class="sxs-lookup"><span data-stu-id="1f5a7-102">'If' operands cannot be named arguments</span></span>
<span data-ttu-id="1f5a7-103">`If` 運算子的運算元使用具名引數無效。</span><span class="sxs-lookup"><span data-stu-id="1f5a7-103">Using named arguments in the operands of the `If` operator is not valid.</span></span> <span data-ttu-id="1f5a7-104">下列範例會產生這個錯誤：</span><span class="sxs-lookup"><span data-stu-id="1f5a7-104">The following example causes this error:</span></span>

```

```

