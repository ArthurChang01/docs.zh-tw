# <a name="if-operands-cannot-be-named-arguments"></a><span data-ttu-id="6d93e-102">'If' 運算元不可以命名為引數。</span><span class="sxs-lookup"><span data-stu-id="6d93e-102">'If' operands cannot be named arguments</span></span>
<span data-ttu-id="6d93e-103">`If` 運算子的運算元使用具名引數無效。</span><span class="sxs-lookup"><span data-stu-id="6d93e-103">Using named arguments in the operands of the `If` operator is not valid.</span></span> <span data-ttu-id="6d93e-104">下列範例會產生這個錯誤：</span><span class="sxs-lookup"><span data-stu-id="6d93e-104">The following example causes this error:</span></span>  
  
```vb  
Dim i As Integer  
Dim result As String  
' Not valid.  
' result = (If(i > 0, TruePart:="positive", FalsePart:="not positive")  
```  
  
 <span data-ttu-id="6d93e-105">這允許具名引數，不同於 `IIf` 函式，如下列程式碼所示：</span><span class="sxs-lookup"><span data-stu-id="6d93e-105">This differs from the `IIf` function, which does allow named arguments, as shown in the following code:</span></span>  
  
```vb  
' Valid.  
IIf(i > 0, TruePart:="positive", FalsePart:="not positive")  
```  
  
 <span data-ttu-id="6d93e-106">**錯誤識別碼：** BC33105</span><span class="sxs-lookup"><span data-stu-id="6d93e-106">**Error ID:** BC33105</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="6d93e-107">更正這個錯誤</span><span class="sxs-lookup"><span data-stu-id="6d93e-107">To correct this error</span></span>  
  
- <span data-ttu-id="6d93e-108">移除運算元的名稱指派，如下列程式碼所示。</span><span class="sxs-lookup"><span data-stu-id="6d93e-108">Remove the name assignments from the operands, as shown in the following code.</span></span>  
  
    ```vb  
    result = If(i > 0, "positive", "not positive")  
    ```
