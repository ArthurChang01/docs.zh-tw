# <a name="xml-comment-type-parameter-ltparametergt-does-not-match-a-type-parameter-on-the-corresponding-ltkeywordgt-statement"></a>XML 註解類型參數&lt;參數&gt;不符合對應的型別參數&lt;關鍵字&gt;陳述式
XML 註解參數\<參數 > 不符合對應的型別參數\<關鍵字 > 陳述式。 將會忽略 XML 註解。  
  
 已針對方法的部分類別參數，但不是全部類別參數，指定方法的 XML 註解標記。  
  
 **錯誤 ID︰** BC42317  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請為方法中的所有類型參數提供 XML 註解標記。
