# <a name="argument-39ltargumentnamegt39-must-be-greater-than-or-equal-to-zero"></a>引數&#39;&lt;引數名稱&gt;&#39;必須大於或等於零
引數無效，因為它小於零。  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
1.  請檢查引數的拼字。 拼錯的變數名稱可能會隱含地建立初始化為零的數值變數。  
  
2.  請檢查運算式中先前的變數作業，特別是那些作為引數從其他程序傳遞至程序的變數。
