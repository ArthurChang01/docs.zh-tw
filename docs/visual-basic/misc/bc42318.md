# <a name="xml-comment-type-parameter-must-have-a-39name39-attribute"></a>XML 註解類型參數必須有 &#39; 名稱 &#39;屬性
XML 註解類型參數必須要有 'name' 屬性。 將會忽略 XML 註解。  
  
 XML 註解標記所需的 `name` 屬性遺漏或不完整。  
  
 **錯誤 ID︰** BC42318  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   加入或完成傳遞給標記的必要 `name` 屬性。
