# <a name="constructors-cannot-implement-interface-methods"></a>建構函式無法實作介面方法
您嘗試使用 `New` 關鍵字來實作介面方法，這不是有效的做法。  
  
 **錯誤 ID：** BC30550  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   使用 `Implements` 關鍵字實作介面。
