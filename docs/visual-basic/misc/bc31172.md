# <a name="an-embedded-expression-may-not-be-used-here"></a>內嵌運算式不能用於此處
內嵌運算式已包含在 XML 常值中不支援的位置。 請注意，內嵌運算式不能用來指定 XML 命名空間值。  
  
 **錯誤 ID:** BC31172  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請從不支援的位置移除內嵌運算式。
