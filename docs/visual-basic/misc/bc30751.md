# <a name="unable-to-get-type-information-for-39lttypenamegt39"></a>無法取得類型資訊 &#39;&lt;typename&gt;&#39;
Common Language Runtime 尚未載入所參照的類型。  
  
 **錯誤 ID︰** BC30751  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請重組您的程式碼，以在目前範圍內定義類型。
