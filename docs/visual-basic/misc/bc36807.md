# <a name="xml-elements-cannot-be-selected-from-type-39type39"></a>無法從型別選取 XML 項目&#39;型別&#39;
已針對類型不是 <xref:System.Xml.Linq.XElement>、 <xref:System.Xml.Linq.XDocument>或 `IEnumerable(Of XElement)`的物件，參考 XML 子項目。 如需詳細資訊，請參閱 [XML Child Axis Property](../../visual-basic/language-reference/xml-axis/xml-child-axis-property.md)。  
  
```vb  
' Generates an error.  
Dim var = "sample text".<child>  
```  
  
 **錯誤 ID︰** BC36807  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請確定所參考屬性的物件強型別為<xref:System.Xml.Linq.XElement>， <xref:System.Xml.Linq.XDocument>，或`IEnumerable(Of XElement)`。 以下是一個範例：  
  
```vb  
Dim elem As XElement = <root>  
                         <child />  
                       </root>  
Dim var = elem.<child>  
```
