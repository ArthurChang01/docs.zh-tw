# <a name="cannot-find-net-framework-directory-error"></a>找不到.NET Framework 目錄：\<錯誤 >
找不到儲存 [!INCLUDE[dnprdnshort](~/includes/dnprdnshort-md.md)] 組件的目錄。  
  
 **錯誤 ID:** BC31508  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
1. 再次編譯程式，看看錯誤是否重複發生。  
  
2. 如果錯誤持續發生，請儲存您的工作，然後重新啟動 Visual Studio。  
  
3. 如果錯誤重複發生，請重新安裝 Visual Basic。  
  
4. 如果錯誤在重新安裝之後持續發生，請通知 Microsoft 產品支援服務。
