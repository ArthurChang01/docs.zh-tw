# <a name="method-or-data-member-not-found"></a>找不到方法或資料成員
找不到您已嘗試存取的方法或資料成員。  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請確定未拼錯方法或資料成員的名稱。
