# <a name="argument-39ltargumentnamegt39-cannot-be-a-multidimensional-array"></a>引數&#39;&lt;引數名稱&gt;&#39;不可以是多維度陣列
array 引數的順位大於 1，但它只能是一維陣列。  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請將引數順位變更為 1。
