# <a name="array-dimensions-cannot-have-a-negative-size"></a>陣列維度大小不可以是負數
一或多個指定的陣列界限是負數。 您只有在使用上限 -1 宣告空白陣列時，才能指定負數註標。 這類陣列沒有項目，但它不是 `Nothing`。  
  
 **錯誤 ID:** BC30611  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 請將負數陣列限制取代為正數。
