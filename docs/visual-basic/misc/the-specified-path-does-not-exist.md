# <a name="the-specified-path-does-not-exist"></a>指定路徑不存在
指定的路徑不存在，可能是因為結合字串時發生錯誤。  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請確認路徑是否正確。
