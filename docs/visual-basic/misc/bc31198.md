# <a name="xml-literal-cannot-appear-here-unless-it-is-enclosed-in-parentheses"></a><span data-ttu-id="adde5-102">XML 常值不可出現在這裡，除非加上括弧</span><span class="sxs-lookup"><span data-stu-id="adde5-102">XML literal cannot appear here unless it is enclosed in parentheses</span></span>
<span data-ttu-id="adde5-103">運算式中使用的 XML 常值宣告，其位置對 [!INCLUDE[vbprvb](~/includes/vbprvb-md.md)] 編譯器來說模稜兩可。</span><span class="sxs-lookup"><span data-stu-id="adde5-103">An XML literal declaration is used in an expression in a location that is ambiguous to the [!INCLUDE[vbprvb](~/includes/vbprvb-md.md)] compiler.</span></span> <span data-ttu-id="adde5-104">也就是說，[!INCLUDE[vbprvb](~/includes/vbprvb-md.md)]編譯器無法判斷是否小於-比字元 (<) 的目的要當做比較運算子還是 XML 常值的開頭。</span><span class="sxs-lookup"><span data-stu-id="adde5-104">That is, the [!INCLUDE[vbprvb](~/includes/vbprvb-md.md)] compiler cannot determine whether the less-than character (<) is intended as a comparative operator or the start of an XML literal.</span></span> <span data-ttu-id="adde5-105">下列程式碼提供一個範例：</span><span class="sxs-lookup"><span data-stu-id="adde5-105">The following code provides an example:</span></span>  

```vb  
' Generates an error.

                  Select element  
```  
  
 <span data-ttu-id="adde5-106">**錯誤 ID：** BC31198</span><span class="sxs-lookup"><span data-stu-id="adde5-106">**Error ID:** BC31198</span></span>
