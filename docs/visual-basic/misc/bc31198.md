# <a name="xml-literal-cannot-appear-here-unless-it-is-enclosed-in-parentheses"></a><span data-ttu-id="5d005-102">XML 常值不可出現在這裡，除非加上括弧</span><span class="sxs-lookup"><span data-stu-id="5d005-102">XML literal cannot appear here unless it is enclosed in parentheses</span></span>
<span data-ttu-id="5d005-103">XML 常值宣告會在模稜兩可，給 Visual Basic 編譯器的位置中的運算式。</span><span class="sxs-lookup"><span data-stu-id="5d005-103">An XML literal declaration is used in an expression in a location that is ambiguous to the Visual Basic compiler.</span></span> <span data-ttu-id="5d005-104">也就是 Visual Basic 編譯器無法判斷是否小於-非字元 (<) 的目的要當做比較運算子還是 XML 常值的開頭。</span><span class="sxs-lookup"><span data-stu-id="5d005-104">That is, the Visual Basic compiler cannot determine whether the less-than character (<) is intended as a comparative operator or the start of an XML literal.</span></span> <span data-ttu-id="5d005-105">下列程式碼提供一個範例：</span><span class="sxs-lookup"><span data-stu-id="5d005-105">The following code provides an example:</span></span>  

```vb  
' Generates an error.  
Dim queryResult = From element In elements _  
                  Where <sample>Value</sample> = "Value" _  
                  Select element  
```  
  
 <span data-ttu-id="5d005-106">**錯誤 ID：** BC31198</span><span class="sxs-lookup"><span data-stu-id="5d005-106">**Error ID:** BC31198</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="5d005-107">更正這個錯誤</span><span class="sxs-lookup"><span data-stu-id="5d005-107">To correct this error</span></span>

```vb  
                  Where (<sample> Value</sample>) = "Value" _  
                  Select element  
```  
  
-   <span data-ttu-id="5d005-109">修改程式碼以參考現有的 XML 常值，如下列範例所示：</span><span class="sxs-lookup"><span data-stu-id="5d005-109">Modify your code to refer to an existing XML literal, as shown in the following example:</span></span>  
  
```vb  
Dim queryResult = From element In elements _  
                  Where e.<sample>.Value = "Value" _  
                  Select element  
```
