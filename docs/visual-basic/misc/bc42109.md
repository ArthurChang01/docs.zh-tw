# <a name="variable-39ltvariablenamegt39-is-used-before-it-has-been-assigned-a-value-visual-basic-error"></a>變數 &#39;&lt;variablename&gt;&#39; 之前已在指派值 （Visual Basic 錯誤），已使用
為變數指派值之前，已使用該變數，導致 <xref:System.NullReferenceException>  
  
 **錯誤 ID︰** BC42109  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請確定已為變數指派值。
