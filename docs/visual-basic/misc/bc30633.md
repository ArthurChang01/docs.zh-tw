# <a name="39set39-statement-must-end-with-a-matching-39end-set39"></a>&#39;設定&#39;陳述式必須搭配相對應的結束&#39;集結尾&#39;
`Set` 屬性程序的結尾必須是 `End Set` 陳述式。  
  
 **錯誤 ID：** BC30633  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請確定 `Set` 屬性程序的結尾為 `End Set` 建構。
