# <a name="ltelementnamegt-is-not-a-method-parameter"></a>'&lt;elementname&gt;' 不是方法參數
程序呼叫可提供方法未定義的其他引數。  
  
 **錯誤 ID:** BC36568  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請從引數清單中移除多餘的引數。
