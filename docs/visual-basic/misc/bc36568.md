# <a name="39ltelementnamegt39-is-not-a-method-parameter"></a>&#39;&lt;elementname&gt; &#39;不是方法參數
程序呼叫可提供方法未定義的其他引數。  
  
 **錯誤 ID︰** BC36568  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請從引數清單中移除多餘的引數。
