# <a name="ltnamegt-is-declared-in-project-ltprojectname1gt-which-is-not-referenced-by-project-ltprojectname2gt"></a>'&lt;名稱&gt;'在專案中宣告'&lt;projectname1&gt;'，專案未參考的'&lt;專案名稱 2>&gt;'
程式設計項目定義於目前專案未參考的專案中。  
  
 **錯誤 ID:** BC32004  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   在目前專案中，加入包含所要項目之專案的參考。
