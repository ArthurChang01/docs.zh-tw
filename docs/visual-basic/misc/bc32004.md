# <a name="name-is-declared-in-project-projectname1-which-is-not-referenced-by-project-projectname2"></a>' \<name > ' 已在專案 ' \<projectname1 > ' 中宣告，但專案 ' \<projectname2 > ' 未參考
程式設計項目定義於目前專案未參考的專案中。  
  
 **錯誤 ID︰** BC32004  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 在目前專案中，加入包含所要項目之專案的參考。
