# <a name="39ltnamegt39-is-declared-in-project-39ltprojectname1gt39-which-is-not-referenced-by-project-39ltprojectname2gt39"></a>&#39;&lt;名稱&gt;&#39; 宣告專案 &#39;&lt;projectname1&gt;&#39;，這不參考專案 &#39;&lt;專案名稱 2>&gt;&#39;
程式設計項目定義於目前專案未參考的專案中。  
  
 **錯誤 ID︰** BC32004  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   在目前專案中，加入包含所要項目之專案的參考。
