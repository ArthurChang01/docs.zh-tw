# <a name="nullable-types-are-not-supported-because-the-current-version-of-mscorlibdll-does-not-define-the-type-nullableof-t"></a>不支援可為 Null 的類型，因為目前 'mscorlib.dll' 的版本未定義類型 'Nullable(of T)'
您使用的架構版本不支援可為 Null 的類型。  
  
 **錯誤 ID:** BC32130
