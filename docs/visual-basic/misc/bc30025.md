# <a name="property-missing-end-property"></a>屬性遺漏 'End Property'
`End` 陳述式必須符合它所終止的區塊。 未正確地終止 `Property` 定義。  
  
 **錯誤 ID:** BC30025  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   將 `End Property` 陳述式加到屬性定義結尾。
