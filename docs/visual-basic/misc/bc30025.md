# <a name="property-missing-39end-property39"></a>遺漏屬性 &#39;End 屬性 &#39;
`End` 陳述式必須符合它所終止的區塊。 未正確地終止 `Property` 定義。  
  
 **錯誤 ID︰** BC30025  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   將 `End Property` 陳述式加到屬性定義結尾。
