# 無法開啟資源檔 &#39;&lt;檔案名稱&gt;&#39;：&lt;錯誤&gt;
開啟指定的資源檔時發生錯誤。  
  
 **錯誤 ID︰**BC31509  
  
### 更正這個錯誤  
  
1.  確認指定的名稱和位置正確。  
  
2.  確認檔案未損毀。
