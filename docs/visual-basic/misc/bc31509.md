# <a name="unable-to-open-resource-file-filename-error"></a>無法開啟資源檔 '\<檔案名稱 >':\<錯誤 >
開啟指定的資源檔時發生錯誤。  
  
 **錯誤 ID:** BC31509  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
1. 確認指定的名稱和位置正確。  
  
2. 確認檔案未損毀。
