# <a name="39ltlsetstmtgt39-is-not-declared"></a>&#39;&lt;lsetstmt&gt; &#39;未宣告
'\<lsetstmt >' 未宣告。 不再支援  LSet 陳述式。請使用 Microsoft.VisualBasic.LSet 代替。  
  
 Visual basic 在舊版中內建的數個函數都已移至<xref:Microsoft.VisualBasic?displayProperty=nameWithType>命名空間。 因此其功能可更廣泛地供所有程式設計語言使用。  
  
 **錯誤 ID︰** BC30820  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   改為在 `LSet` 命名空間中使用 `Microsoft.VisualBasic` 函式。
