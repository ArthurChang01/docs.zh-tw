# <a name="39ltlsetstmtgt39-is-not-declared"></a>'&lt;lsetstmt&gt;' 未宣告
'\<lsetstmt >' 未宣告。 不再支援  LSet 陳述式。請使用 Microsoft.VisualBasic.LSet 代替。  
  
 已內建函式的幾個函式[!INCLUDE[vbprvb](../../csharp/programming-guide/concepts/linq/includes/vbprvb_md.md)]先前版本中都已移至<xref:Microsoft.VisualBasic?displayProperty=fullName>命名空間。</xref:Microsoft.VisualBasic?displayProperty=fullName> 因此其功能可更廣泛地供所有程式設計語言使用。  
  
 **錯誤 ID︰** BC30820  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   改為在 `LSet` 命名空間中使用 `Microsoft.VisualBasic` 函式。
