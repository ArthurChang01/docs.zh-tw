# <a name="registry-key-ltkeynamegt-could-not-be-found"></a>登錄機碼 '&lt;keyname&gt;' 找不到
找不到登錄機碼。  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
1.  請檢查機碼名稱的拼寫。  
  
2.  請確定您有足夠的安全性權限可以存取機碼。
