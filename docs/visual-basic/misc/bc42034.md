# <a name="file-name-already-declared-with-a-different-guid-and-checksum-value"></a>已使用不同的 GUID 與總和檢查碼值宣告的檔名
已嘗試宣告檔案，但已存在具有不同 GUID 與總和檢查碼值的檔案。  
  
 **錯誤 ID︰** BC42034  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   檢查檔案名稱和路徑，確定這兩者皆正確。  
  
-   刪除不需要的多餘檔案。
