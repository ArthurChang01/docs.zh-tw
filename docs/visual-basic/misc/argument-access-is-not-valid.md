# <a name="argument-39access39-is-not-valid-valid-values-for-output-mode-are-39openaccesswrite39-and-39openaccessdefault39"></a>引數&#39;存取&#39;不正確。 Output 模式的有效值為&#39;OpenAccess.Write&#39;和&#39;OpenAccess.Default&#39;
存取值在 Output 模式中無效。  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
1.  將此值變更為 `OpenAccess.Write` 或 `OpenAccess.Default`。
