# <a name="comma-or-3939-expected-procedure-parameter-list"></a>以逗號或 &#39;) &#39;預期 （程序參數清單）
參數清單包含未預期的項目。 請使用逗號 (`,`) 來分隔這類清單中的參數，並使用右括弧 (`)`) 來結束清單。  
  
 **錯誤 ID︰** BC30213  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請加入逗號並繼續類型參數清單，或者加入右括弧字元來終止清單。
