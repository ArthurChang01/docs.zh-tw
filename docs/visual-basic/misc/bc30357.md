# <a name="39inherits39-statements-must-precede-all-declarations-in-an-interface"></a>&#39; 繼承 &#39;陳述式必須在所有宣告都之前在介面中
`Inherits` 陳述式必須是類別定義中第一個非空白的非註解行。  
  
 **錯誤識別碼：** BC30357  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   將 `Inherits` 陳述式移動到所有宣告之前。
