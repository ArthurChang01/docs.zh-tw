# <a name="inherits-statements-must-precede-all-declarations-in-an-interface"></a>'Inherits' 陳述式必須位在介面中的所有宣告之前。
`Inherits` 陳述式必須是類別定義中第一個非空白的非註解行。  
  
 **錯誤 ID:** BC30357  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   將 `Inherits` 陳述式移動到所有宣告之前。
