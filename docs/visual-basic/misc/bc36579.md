# <a name="too-many-type-arguments"></a>類型引數太多
用來叫用泛型方法的類型引數數目超過類型參數數目。  
  
 當您叫用泛型方法時，您必須針對該方法所定義的每個類型參數提供一個類型引數。  
  
 **錯誤 ID:** BC36579  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   從您的類型引數清單中移除類型引數，讓您叫用之泛型方法的每個類型參數各有一個對應的類型引數。
