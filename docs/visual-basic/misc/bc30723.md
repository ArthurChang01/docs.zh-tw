# <a name="method-call-did-not-return-a-value"></a>方法呼叫未傳回值
所呼叫的方法未如預期傳回值。  
  
 **錯誤 ID︰** BC30723  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請確認所呼叫的方法是傳回值的函式。
