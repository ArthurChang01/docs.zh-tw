# <a name="the-targeted-version-of-the-net-compact-framework-does-not-support-using-the-ansi-auto-or-unicode-modifier"></a>.NET Compact Framework 的目標版本不支援使用 Ansi、Auto 或 Unicode 修飾詞
您使用的 .NET Compact Framework 版本不支援 `Ansi`、 `Auto`和 `Unicode` 修飾詞。  
  
 **錯誤 ID:** BC30763  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請移除修飾詞。
