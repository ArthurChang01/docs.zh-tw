# <a name="specifier1-and-specifier2-cannot-be-combined"></a>'\<specifier1 >' 和'\<specifier2 >' 無法結合
同時使用兩個不相容的規範 (例如 `Shadows` 和 `Overrides` )。  
  
 **錯誤 ID:** BC31408  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請移除其中一個或另一個規範。
