# <a name="39ltspecifier1gt39-and-39ltspecifier2gt39-cannot-be-combined"></a>&#39;&lt;specifier1&gt; &#39;和&#39; &lt;specifier2&gt; &#39;無法合併
同時使用兩個不相容的規範 (例如 `Shadows` 和 `Overrides` )。  
  
 **錯誤 ID︰** BC31408  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請移除其中一個或另一個規範。
