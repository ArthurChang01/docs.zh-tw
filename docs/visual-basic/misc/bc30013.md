<span data-ttu-id="e13ab-103">`#ElseIf`、 `#Else`和 `#End If` 都是條件式編譯指示詞。</span><span class="sxs-lookup"><span data-stu-id="e13ab-103">`#ElseIf`, `#Else`, and `#End If` are conditional compilation directives.</span></span> <span data-ttu-id="e13ab-104">`#ElseIf`、 `#Else`或 `#End If` 的前面沒有對應的 `#If` 指示詞。</span><span class="sxs-lookup"><span data-stu-id="e13ab-104">The `#ElseIf`, `#Else`, or `#End If` is not preceded by a corresponding `#If` directive.</span></span>  
  
 <span data-ttu-id="e13ab-105">**錯誤識別碼:** BC30013</span><span class="sxs-lookup"><span data-stu-id="e13ab-105">**Error ID:** BC30013</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="e13ab-106">更正這個錯誤</span><span class="sxs-lookup"><span data-stu-id="e13ab-106">To correct this error</span></span>  
  
1. <span data-ttu-id="e13ab-107">檢查是否未使用介入條件式編譯區塊將預期的 `#If` 與問題中的子句隔開，或不正確地放置 `#End If`。</span><span class="sxs-lookup"><span data-stu-id="e13ab-107">Check that the intended `#If` is not separated from the clause in question by an intervening conditional compilation block or an incorrectly placed `#End If`.</span></span>
