# <a name="implementsclause-cannot-implement-typename-because-typename-is-a-reserved-name"></a>'\<實作子句 >' 無法實作'\<類型名稱 >' 由於 '\<類型名稱 >' 是保留的名稱
`Implements` 陳述式指定了保留的名稱。  
  
 **錯誤 ID:** BC31415  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   重新命名無法實作的介面成員。
