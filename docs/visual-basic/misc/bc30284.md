# <a name="type1-typename-cannot-be-declared-overrides-because-it-does-not-override-a-type1-in-a-base-type2"></a>\<type1 > '\<類型名稱 >' 不可宣告為 'Overrides' 因為不覆寫\<type1 > 基底中\<type2 >
`Sub`、 `Function`或 `Property` 陳述式指定 `Overrides` ，但基底類別中沒有相同名稱的類型存在。  
  
 **錯誤 ID:** BC30284  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
1.  請檢查類型名稱拼寫正確。  
  
2.  請移除多餘的 `Overrides` 關鍵字。
