# <a name="lttype1gt-lttypenamegt-cannot-be-declared-overrides-because-it-does-not-override-a-lttype1gt-in-a-base-lttype2gt"></a>&lt;type1&gt; '&lt;typename&gt;' 不可宣告為 'Overrides' 因為不覆寫&lt;type1&gt;基底中&lt;type2&gt;
`Sub`、 `Function`或 `Property` 陳述式指定 `Overrides` ，但基底類別中沒有相同名稱的類型存在。  
  
 **錯誤 ID:** BC30284  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
1.  請檢查類型名稱拼寫正確。  
  
2.  請移除多餘的 `Overrides` 關鍵字。
