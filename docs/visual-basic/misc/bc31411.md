# <a name="lttypenamegt-must-be-declared-mustinherit-because-it-contains-methods-declared-mustoverride"></a>'&lt;typename&gt;' 必須宣告為 'MustInherit' 因為它包含宣告為 'MustOverride' 方法
包含 `MustOverride` 成員的類型必須宣告為 `MustInherit`。  
  
 **錯誤 ID:** BC31411  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   在類型中加入 `MustInherit` 修飾詞。
