# <a name="39lttypenamegt39-must-be-declared-39mustinherit39-because-it-contains-methods-declared-39mustoverride39"></a>&#39;&lt;typename&gt; &#39;必須宣告&#39;MustInherit&#39;因為它包含宣告的方法&#39;MustOverride&#39;
包含 `MustOverride` 成員的類型必須宣告為 `MustInherit`。  
  
 **錯誤識別碼：** BC31411  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   在類型中加入 `MustInherit` 修飾詞。
