# <a name="argument-39ltargumentnamegt39-must-be-in-the-range-of--32768-to-65535"></a>引數 &#39;&lt;引數名稱&gt;&#39; 必須在-32768 到 65535 的範圍內
引數的值超出允許的範圍 -32,768 到 65,535 之間。  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
1.  請確定引數的值落在有效範圍內。  
  
2.  如果透過計算來提供引數，請檢查計算所處理的值。
