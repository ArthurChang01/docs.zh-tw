# <a name="expected-beginning-39lt39-for-an-xml-tag"></a>預期開頭&#39; &lt; &#39; XML 標記
XML 常值的 XML 標記缺少必要的開頭 '<' 字元。  
  
 **錯誤 ID︰** BC31165  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請在 XML 常值的 XML 標記開頭加入 '<' 字元。
