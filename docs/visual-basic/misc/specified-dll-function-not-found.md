# <a name="specified-dll-function-not-found"></a>找不到所指定的 DLL 函式
在使用者程式庫參考中找到動態連結程式庫 (DLL)，但在 DLL 內找不到指定的 DLL 函式。  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
1.  在函式宣告中指定有效的序數。  
  
2.  確定 DLL 名稱和別名正確。
