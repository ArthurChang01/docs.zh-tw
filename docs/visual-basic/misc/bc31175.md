# <a name="xml-dtds-are-not-supported"></a>不支援 XML DTD
文件類型定義 (DTD) 已指定為 XML 常值。  
  
 **錯誤 ID:** BC31175  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   移除 DTD。 使用 XML 結構描述定義 (XSD) 來指定 XML 文件的結構描述。
