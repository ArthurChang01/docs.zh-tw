# <a name="methods-cannot-be-declared-39static39"></a>方法不可以宣告&#39;靜態&#39;
`Static` 修飾詞不能套用至整個程序。  
  
 **錯誤 ID︰** BC30810  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請使用 `Static` 修飾詞，將程序內的個別項目宣告為 static。
