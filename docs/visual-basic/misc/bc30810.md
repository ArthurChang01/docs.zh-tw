# 方法不可以宣告為 &#39;Static&#39;
`Static` 修飾詞不能套用至整個程序。  
  
 **錯誤 ID︰**BC30810  
  
### 更正這個錯誤  
  
-   請使用 `Static` 修飾詞，將程序內的個別項目宣告為 static。
