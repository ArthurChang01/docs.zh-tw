# <a name="argument-cannot-be-less-than-zero"></a>引數不可以小於零
已提供小於零的值，但引數不能小於零。  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請檢查如何計算發生問題的值，確定它是正確的。
