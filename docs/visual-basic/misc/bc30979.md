# <a name="39option-infer39-can-be-followed-only-by-39on39-or-39off39"></a>&#39;選項 Infer&#39;後面只能接&#39;上&#39;或&#39;關閉&#39;
`Option Infer` 陳述式必須指定 `On` 或 `Off`。 其他值無效。  
  
 **錯誤 ID︰** BC30979  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請指定 `Option Infer On` 或 `Option Infer Off`。
