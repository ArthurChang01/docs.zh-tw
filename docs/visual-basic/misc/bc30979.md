# <a name="option-infer-can-be-followed-only-by-on-or-off"></a>'Option Infer' 之後只能是 'On' 或 'Off'
`Option Infer` 陳述式必須指定 `On` 或 `Off`。 其他值無效。  
  
 **錯誤 ID:** BC30979  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 請指定 `Option Infer On` 或 `Option Infer Off`。
