# <a name="39ltnamespacegt39-is-not-a-valid-name-and-cannot-be-used-as-the-root-namespace-name"></a>&#39;&lt;命名空間&gt;&#39;不是有效的名稱，也不能做為根命名空間名稱
合格的命名空間名稱不是有效的項目名稱。 名稱只能包含字母字元、十進位數字和底線，而且必須以字母字元或底線 (`_`) 開頭。  
  
 **錯誤 ID︰** BC30113  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   如果命名空間是一組巢狀命名空間的一部分，請檢查是否有任何外部名稱包含無效的字元。
