# <a name="39ltpropertyname1gt39-cannot-override-39ltpropertyname2gt39-because-they-differ-by-39readonly39-or-39writeonly39"></a>&#39;&lt;propertyname1&gt; &#39;無法覆寫&#39; &lt;propertyname2&gt; &#39;因為它們的不同&#39;ReadOnly&#39;或&#39;WriteOnly&#39;
您嘗試以第二個屬性覆寫第一個屬性，且這兩個屬性的差異僅在於 `ReadOnly` 或 `WriteOnly`。  
  
 **錯誤 ID：** BC30362  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   確定屬性的區分方式不只是透過 `ReadOnly` 和 `WriteOnly`。
