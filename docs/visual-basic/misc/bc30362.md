# <a name="propertyname1-cannot-override-propertyname2-because-they-differ-by-readonly-or-writeonly"></a>'\<屬性名稱 1&gt >' 無法覆寫'\<propertyname2 >' 因為其 'ReadOnly' 或 'WriteOnly' 不同
您嘗試以第二個屬性覆寫第一個屬性，且這兩個屬性的差異僅在於 `ReadOnly` 或 `WriteOnly`。  
  
 **錯誤 ID:** BC30362  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   確定屬性的區分方式不只是透過 `ReadOnly` 和 `WriteOnly`。
