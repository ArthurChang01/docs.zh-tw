# <a name="members-in-a-module-cannot-be-declared-ltspecifiergt"></a>在模組中的成員不可以宣告 '&lt;規範&gt;'
您已經在 `Module` 陳述式內使用成員上無效的規範。 模組絕不會執行個體化、不支援繼承，且不能實作介面。  
  
 **錯誤 ID:** BC30436  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請移除規範。
