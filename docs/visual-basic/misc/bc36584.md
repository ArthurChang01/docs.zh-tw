# <a name="parameter-parametername-in-extension-method-methodname-defined-in-typename-already-has-a-matching-argument"></a>參數 '\<參數名稱 >' 中擴充方法'\<方法名稱 >' 中定義 '\<類型名稱 >' 已經有符合的引數
在相同的程序呼叫中，依名稱多次傳遞引數。  
  
 **錯誤 ID:** BC36584  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請確認是否拼錯其中一個引數名稱。  
  
-   請從引數清單中移除重複引數。
