# <a name="parameter-39ltparameternamegt39-in-extension-method-39ltmethodnamegt39-defined-in-39lttypenamegt39-already-has-a-matching-argument"></a>參數&#39; &lt;parametername&gt; &#39;中擴充方法&#39; &lt;methodname&gt; &#39;中定義&#39; &lt;typename&gt; &#39;已經具有相符引數
在相同的程序呼叫中，依名稱多次傳遞引數。  
  
 **錯誤 ID︰** BC36584  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請確認是否拼錯其中一個引數名稱。  
  
-   請從引數清單中移除重複引數。
