# <a name="parameter-ltparameternamegt-in-extension-method-ltmethodnamegt-defined-in-lttypenamegt-already-has-a-matching-argument"></a>參數 '&lt;parametername&gt;'中擴充方法'&lt;methodname&gt;'中所定義的'&lt;typename&gt;' 已有相符的引數
在相同的程序呼叫中，依名稱多次傳遞引數。  
  
 **錯誤 ID:** BC36584  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請確認是否拼錯其中一個引數名稱。  
  
-   請從引數清單中移除重複引數。
