# <a name="array-initializer-has-ltnumbergt-too-many-elements"></a>陣列初始設定式&lt;數目&gt;太多項目
您的陣列初始設定式包含太多項目。  
  
 **錯誤 ID︰** BC30568  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   使用 `ReDim` 變更陣列的大小。
