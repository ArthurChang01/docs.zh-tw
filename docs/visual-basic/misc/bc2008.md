# <a name="no-input-sources-specified"></a>沒有指定輸入來源
編譯時，未指定任何輸入來源。  
  
 **錯誤 ID︰** BC2008  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請確定您已包含原始程式檔。
