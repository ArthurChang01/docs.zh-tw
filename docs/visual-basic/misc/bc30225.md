# <a name="option-ltspecifiergt-statement-can-only-appear-once-per-file"></a>'選項&lt;規範&gt;' 陳述式可以只出現一次每個檔案
`Option Compare`、 `Option Explicit`和 `Option Strict` 陳述式可能在相同的原始程式檔中各使用一次。  
  
 **錯誤 ID:** BC30225  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請移除重複的 `Option` 陳述式。
