# <a name="the-custom-designed-version-of-systemruntimecompilerservicesextensionattribute-found-by-the-compiler-is-not-valid"></a><span data-ttu-id="9068c-102">這個由編譯器所發現的 'System.Runtime.CompilerServices.ExtensionAttribute' 自訂版本無效</span><span class="sxs-lookup"><span data-stu-id="9068c-102">The custom-designed version of 'System.Runtime.CompilerServices.ExtensionAttribute' found by the compiler is not valid</span></span>
<span data-ttu-id="9068c-103">這個由編譯器所發現的 'System.Runtime.CompilerServices.ExtensionAttribute' 自訂版本無效。</span><span class="sxs-lookup"><span data-stu-id="9068c-103">The custom-designed version of 'System.Runtime.CompilerServices.ExtensionAttribute' found by the compiler is not valid.</span></span> <span data-ttu-id="9068c-104">它的屬性使用方式旗標必須設定成允許組件、類別和方法。</span><span class="sxs-lookup"><span data-stu-id="9068c-104">Its attribute usage flags must be set to allow assemblies, classes, and methods.</span></span>  
  
 <span data-ttu-id="9068c-105">這個由編譯器所發現的 <xref:System.Runtime.CompilerServices.ExtensionAttribute?displayProperty=nameWithType> 自訂版本，未將其屬性使用方式旗標設定成允許將屬性套用至組件、方法和類別。</span><span class="sxs-lookup"><span data-stu-id="9068c-105">The custom-designed version of <xref:System.Runtime.CompilerServices.ExtensionAttribute?displayProperty=nameWithType> that the compiler found does not set its attribute usage flags to enable application of the attribute to assemblies, methods, and classes.</span></span> <span data-ttu-id="9068c-106">您必須至少套用至上述三個程式項目。</span><span class="sxs-lookup"><span data-stu-id="9068c-106">Application to at least those three program elements is required.</span></span>  
  
 <span data-ttu-id="9068c-107">**錯誤 ID:** BC36558</span><span class="sxs-lookup"><span data-stu-id="9068c-107">**Error ID:** BC36558</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="9068c-108">更正這個錯誤</span><span class="sxs-lookup"><span data-stu-id="9068c-108">To correct this error</span></span>  
  
-   <span data-ttu-id="9068c-109">變更屬性定義，以允許將屬性至少套用至組件、方法和類別，如下列範例所示。</span><span class="sxs-lookup"><span data-stu-id="9068c-109">Change the attribute definition to enable the attribute to apply at least to assemblies, methods, and classes, as shown in the following examples.</span></span>  
  
-   <span data-ttu-id="9068c-110">使用 <xref:System.Runtime.CompilerServices.ExtensionAttribute?displayProperty=nameWithType> ，而不是自訂版本。</span><span class="sxs-lookup"><span data-stu-id="9068c-110">Use <xref:System.Runtime.CompilerServices.ExtensionAttribute?displayProperty=nameWithType> instead of the custom-designed version.</span></span>  
  
## <a name="example"></a><span data-ttu-id="9068c-111">範例</span><span class="sxs-lookup"><span data-stu-id="9068c-111">Example</span></span>  
 <span data-ttu-id="9068c-112">下列範例使用 `AttributeUsage` 屬性指定新版 `ExtensionAttribute` 可以套用至哪些程式項目。</span><span class="sxs-lookup"><span data-stu-id="9068c-112">The following example uses the `AttributeUsage` attribute to specify which program elements the new version of `ExtensionAttribute` can apply to.</span></span> <span data-ttu-id="9068c-113">此範例指定 `AttributeTargets` 列舉的三個成員： `Assembly`、 `Class`和 `Method`。</span><span class="sxs-lookup"><span data-stu-id="9068c-113">The example specifies three members of the `AttributeTargets` enumeration: `Assembly`, `Class`, and `Method`.</span></span> <span data-ttu-id="9068c-114">省略其中任何一個項目都會造成這個錯誤。</span><span class="sxs-lookup"><span data-stu-id="9068c-114">The omission of any one of these elements will cause this error.</span></span>  
  
```  
Namespace System.Runtime.CompilerServices  
    <AttributeUsage(AttributeTargets.Assembly Or _  
        AttributeTargets.Class Or AttributeTargets.Method)>  
    Class ExtensionAttribute  
        Inherits System.Attribute  
        ' Definitions of methods, fields, and properties.  
    End Class  
End Namespace  
```  
  
 <span data-ttu-id="9068c-115">或者，您可以使用 `ExtensionAttribute` 的 `All` 成員，允許 `AttributeTargets`套用至所有程式項目。</span><span class="sxs-lookup"><span data-stu-id="9068c-115">Alternatively, you could allow `ExtensionAttribute` to apply to all program elements by using the `All` member of `AttributeTargets`.</span></span>  
  
```xml  
<AttributeUsage(AttributeTargets.All)>  
```  
  
 <span data-ttu-id="9068c-116">刪除 `AttributeUsage` 行 (如下列程式碼所示) 會產生相同的結果。</span><span class="sxs-lookup"><span data-stu-id="9068c-116">Deleting the `AttributeUsage` line, as shown in the following code, produces the same result.</span></span>  
  
```  
Namespace System.Runtime.CompilerServices  
    Class ExtensionAttribute  
        Inherits System.Attribute  
        ' Definitions of methods, fields, and properties.  
    End Class  
End Namespace  
```
