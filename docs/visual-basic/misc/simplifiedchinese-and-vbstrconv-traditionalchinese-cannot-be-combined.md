# 無法合併 SimplifiedChinese 和 VbStrConv.TraditionalChinese
您的應用程式嘗試合併 `VbStrConv` 列舉成員 `SimplifiedChinese` 和 `TraditionalChinese`，但它們互斥。  
  
### 更正這個錯誤  
  
-   請移除  `VbStrConv.SimplifiedChinese` 或 `VbStrConv.TraditionalChinese`。
