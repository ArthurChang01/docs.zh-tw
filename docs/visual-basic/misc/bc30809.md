# <a name="39wend39-statements-are-no-longer-supported-use-39end-while39-statements-instead"></a>&#39;Wend&#39;不再支援陳述式。使用&#39;End While&#39;陳述式改為
`End While` 建構會取代 `Wend` 關鍵字。  
  
 **錯誤 ID︰** BC30809  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   使用 `End While` 結束 `While` 區塊。
