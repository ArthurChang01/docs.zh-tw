# <a name="39systemstathreadattribute39-and-39systemmtathreadattribute39-cannot-both-be-applied-to-the-same-method"></a>&#39;System.STAThreadAttribute &#39;和 &#39;System.MTAThreadAttribute &#39;無法同時套用至相同的方法
`System.STAThreadAttribute` 和 `System.MTAThreadAttribute` 屬性互為獨佔模式。  
  
 **錯誤 ID︰** BC31512  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
1.  請套用 `System.MTAThreadAttribute` 或 `System.STAThreadAttribute`，但兩者不能同時套用。
