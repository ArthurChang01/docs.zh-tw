# <a name="39variant39-is-no-longer-a-supported-type-use-the-39object39-type-instead"></a>&#39;Variant&#39;已不再支援的型別。使用&#39;物件&#39;改為輸入
`Object` 資料類型取代 `Variant`。  
  
 **錯誤 ID︰** BC30804  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請將項目宣告為 `Object`。
