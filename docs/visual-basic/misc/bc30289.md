# <a name="statement-cannot-appear-within-a-method-body"></a>陳述式不可以出現在方法主體中
程序包含無效的陳述式 (例如另一個程序宣告)。 程序宣告不可為巢狀。  
  
 **錯誤 ID:** BC30289  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請移除程序宣告，或將它移至封入程序外部。
