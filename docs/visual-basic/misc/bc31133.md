# <a name="39addhandler39-and-39removehandler39-methods-must-have-exactly-one-parameter"></a>&#39;AddHandler &#39;和 &#39;RemoveHandler &#39;方法必須有一個參數
自訂事件宣告必須有 `AddHandler` 或 `RemoveHandler` 宣告，每個宣告會接受自訂事件的 `As` 子句所指定之委派類型的單一參數。  
  
 **錯誤 ID︰** BC31133  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   從參數清單中移除額外的參數，並將參數類型變更為與自訂事件的 `As` 子句所指定之委派類型相同的類型。
