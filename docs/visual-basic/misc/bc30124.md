# <a name="property-without-a-39readonly39-or-39writeonly39-specifier-must-provide-both-a-39get39-and-a-39set39"></a>沒有屬性 &#39;ReadOnly &#39;或 &#39;WriteOnly &#39;規範必須兩者都提供 &#39;Get &#39;和 &#39;設定 &#39;
如果屬性未宣告為 `ReadOnly` 或 `WriteOnly`，則必須提供用於讀取及寫入其值的程序。  
  
 **錯誤 ID︰** BC30124  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
1.  確定您在 `Get` 陳述式和 `Set` 陳述式之間同時包含 `Property` 程序和 `End Property` 程序。  
  
2.  請確認已正確地終止 `Property` 宣告內的其他程序。
