# <a name="required-attribute-39version39-missing-from-xml-declaration"></a>必要屬性 &#39; 版本 &#39;XML 宣告中遺漏
遺漏 XML 文件常值的 `version` 屬性。  
  
 **錯誤 ID︰** BC31153  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   將 `version` 屬性加入 XML 文件常值，如下列範例所示： `<?xml version="1.0"?>`
