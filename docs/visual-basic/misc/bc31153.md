# <a name="required-attribute-version-missing-from-xml-declaration"></a>XML 宣告遺漏必要的屬性 'version'
遺漏 XML 文件常值的 `version` 屬性。  
  
 **錯誤 ID:** BC31153  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   將 `version` 屬性加入 XML 文件常值，如下列範例所示： `<?xml version="1.0"?>`
