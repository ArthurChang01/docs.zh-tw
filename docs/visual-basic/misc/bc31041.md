# <a name="statement-is-not-valid-in-an-interface"></a>介面中的陳述式無效
在介面中遇到無效的陳述式。  
  
 **錯誤 ID︰** BC31041  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請確認介面中的陳述式定義項目，但未提供實作。
