# <a name="warning-treated-as-error-error"></a>將警告視為錯誤：\<錯誤 >
開啟 [警告視為錯誤]  選項執行組件編譯時，出現警告。  
  
 **錯誤 ID:** BC31072  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
1.  請更正這個警告。  
  
2.  關閉 [警告視為錯誤]  選項再編譯組件。
