# <a name="39type39-statements-are-no-longer-supported-use-39structure39-statements-instead"></a>&#39;型別&#39;不再支援陳述式。使用&#39;結構&#39;陳述式改為
您無法再使用 `Type` 關鍵字來定義使用者定義類型。  
  
 **錯誤 ID︰** BC30802  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   使用 `Structure` 陳述式來定義新的類型。
