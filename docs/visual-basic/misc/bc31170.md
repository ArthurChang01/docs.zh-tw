# <a name="character-39character39-0xlthexadecimalgt-is-not-allowed-in-an-xml-name"></a>字元&#39;字元&#39;(0 x&lt;十六進位&gt;) XML 名稱中不允許
XML 常值或 XML 軸屬性的名稱包含不支援的字元 。  
  
 **錯誤 ID：** BC31170  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   移除 XML 常值或 XML 軸屬性的名稱中不支援的字元。
