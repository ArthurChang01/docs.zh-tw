# <a name="character-character-0xhexadecimal-is-not-allowed-in-an-xml-name"></a>Character '' (0x\<十六進位 >) 的 XML 名稱中不允許
XML 常值或 XML 軸屬性的名稱包含不支援的字元 。  
  
 **錯誤 ID:** BC31170  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   移除 XML 常值或 XML 軸屬性的名稱中不支援的字元。
