# <a name="at-least-one-parameter-type-of-this-binary-operator-must-be-the-containing-type-39lttypenamegt39"></a>此二元運算子至少一個參數類型必須是包含類型 &#39;&lt;typename&gt;&#39;
二元運算子定義所指定兩個參數的類型，不是定義運算子之類別或結構的類型。  
  
 當您在類別或結構中定義運算子時，至少有一個參數必須是該類別或結構的類型。  
  
 **錯誤 ID︰** BC33021  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   將一個或兩個參數的類型變更為定義運算子之類別或結構的類型。
