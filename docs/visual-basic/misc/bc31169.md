# <a name="character-character-0xlthexadecimalgt-is-not-allowed-at-the-beginning-of-an-xml-name"></a>Character '' (0x&lt;十六進位&gt;) 不允許在 XML 名稱的開頭
XML 常值或 XML 軸屬性名稱開頭包含不支援的字元。  
  
 **錯誤 ID:** BC31169  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請移除 XML 常值或 XML 軸屬性名稱開頭中不支援的字元。
