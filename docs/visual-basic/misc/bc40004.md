# <a name="lttype1gt-39ltmembernamegt39-conflicts-with-lttype2gt-39ltmembernamegt39-on-the-base-class-lttype3gt-39ltclassnamegt39-and-should-be-declared-39shadows39"></a>&lt;type1&gt; &#39; &lt;membername&gt; &#39;與衝突&lt;type2&gt; &#39; &lt;membername&gt; &#39;基底類別上&lt;type3&gt; &#39; &lt;classname&gt; &#39;與應該宣告為&#39;陰影&#39;
宣告的程式設計項目與基底類別中所定義的項目同名。 在這種情況下，這個類別中的項目應該會遮蔽基底類別項目。  
  
 這個訊息是一個警告。 預設會假設為`Shadows` 。 如需隱藏警告或將警告視為錯誤的詳細資訊，請參閱[Visual Basic 中的 設定警告](/visualstudio/ide/configuring-warnings-in-visual-basic)。  
  
 **錯誤 ID︰** BC40004  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   將 `Shadows` 關鍵字加入宣告中，或變更所宣告項目的名稱。
