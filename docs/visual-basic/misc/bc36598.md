# <a name="instance-of-restricted-type-39lttypenamegt39-cannot-be-used-in-a-query-expression"></a>限制類型的執行個體&#39; &lt;typename&gt; &#39;不能在查詢運算式
受限制之類型的執行個體 (例如 <xref:System.ArgIterator>、 <xref:System.RuntimeArgumentHandle>和 <xref:System.TypedReference>) 不能用於 Lambda 運算式或查詢運算式。  
  
 **錯誤 ID︰** BC36598
