# <a name="parameter-already-declared-with-name-argumentname"></a>已使用名稱宣告的參數 '\<引數名稱 >'
程序宣告以相同名稱定義兩個引數。 在程序宣告中的所有引數名稱必須是唯一的。  
  
 **錯誤 ID:** BC30237  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   變更其中一個引數的名稱。
