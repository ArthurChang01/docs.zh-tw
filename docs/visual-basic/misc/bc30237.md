# <a name="parameter-already-declared-with-name-39ltargumentnamegt39"></a>參數已經宣告過名稱為 &#39;&lt;引數名稱&gt;&#39;
程序宣告以相同名稱定義兩個引數。 在程序宣告中的所有引數名稱必須是唯一的。  
  
 **錯誤 ID：** BC30237  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   變更其中一個引數的名稱。
