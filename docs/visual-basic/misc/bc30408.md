# <a name="method-39ltmethodnamegt39-does-not-have-the-same-signature-as-delegate-39ltdelegatenamegt39"></a>方法&#39; &lt;methodname&gt; &#39;沒有相同的簽章與委派&#39;&lt;委派名稱&gt;&#39;
您嘗試使用之方法和委派的簽章不符。 `Delegate` 陳述式定義委派類別的參數類型和傳回類型。 任何具有相符參數、類型和傳回類型的程序都可用來建立這個委派類型的執行個體。 每個委派類別會為傳遞的建構函式定義物件方法的規格。  
  
 **錯誤識別碼︰** BC30408  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   檢查簽章以找出不相符的情況，然後加以更正。
