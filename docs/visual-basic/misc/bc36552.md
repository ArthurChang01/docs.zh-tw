 <span data-ttu-id="01e62-105">不含參數的擴充方法無效，因為第一個參數指定方法所擴充的資料類型。</span><span class="sxs-lookup"><span data-stu-id="01e62-105">An extension method without parameters is not valid because the first parameter specifies which data type the method extends.</span></span> <span data-ttu-id="01e62-106">第一個參數會繫結至叫用方法的資料類型執行個體。</span><span class="sxs-lookup"><span data-stu-id="01e62-106">The first parameter is bound to the instance of the data type that invokes the method.</span></span>

## <a name="example"></a><span data-ttu-id="01e62-110">範例</span><span class="sxs-lookup"><span data-stu-id="01e62-110">Example</span></span>  
 <span data-ttu-id="01e62-111">下列範例中的第一個參數表示 `Print` 方法可擴充 `String` 資料類型。</span><span class="sxs-lookup"><span data-stu-id="01e62-111">The first parameter in the following example indicates that the `Print` method extends the `String` data type.</span></span>  
  
```  
<Extension()> _  
Public Sub Print (ByVal str As String)  
    Console.WriteLine(str)  
End Sub  
```  
  
 <span data-ttu-id="01e62-112">如下所示呼叫擴充方法時， `str` 方法中的參數會繫結至 `greeting`，也就是呼叫 `String` 的 `Print`執行個體。</span><span class="sxs-lookup"><span data-stu-id="01e62-112">When the extension method is called as follows, parameter `str` in the method is bound to `greeting`, the instance of `String` that calls `Print`.</span></span> <span data-ttu-id="01e62-113">編譯器將使用 `greeting` 作為擴充方法 `Print`的引數。</span><span class="sxs-lookup"><span data-stu-id="01e62-113">The compiler will use `greeting` as the argument to extension method `Print`.</span></span>  
  
```  
Dim greeting As String = "Hello"  
greeting.Print()  
```
