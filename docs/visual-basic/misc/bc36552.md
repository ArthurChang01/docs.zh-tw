## <a name="example"></a><span data-ttu-id="d0d84-110">範例</span><span class="sxs-lookup"><span data-stu-id="d0d84-110">Example</span></span>  
 <span data-ttu-id="d0d84-111">下列範例中的第一個參數表示 `Print` 方法可擴充 `String` 資料類型。</span><span class="sxs-lookup"><span data-stu-id="d0d84-111">The first parameter in the following example indicates that the `Print` method extends the `String` data type.</span></span>  
  
```  
<Extension()> _  
Public Sub Print (ByVal str As String)  
    Console.WriteLine(str)  
End Sub  
```  
  
 <span data-ttu-id="d0d84-112">如下所示呼叫擴充方法時， `str` 方法中的參數會繫結至 `greeting`，也就是呼叫 `String` 的 `Print`執行個體。</span><span class="sxs-lookup"><span data-stu-id="d0d84-112">When the extension method is called as follows, parameter `str` in the method is bound to `greeting`, the instance of `String` that calls `Print`.</span></span> <span data-ttu-id="d0d84-113">編譯器將使用 `greeting` 作為擴充方法 `Print`的引數。</span><span class="sxs-lookup"><span data-stu-id="d0d84-113">The compiler will use `greeting` as the argument to extension method `Print`.</span></span>

