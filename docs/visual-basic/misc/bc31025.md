# 無法開啟金鑰檔 &#39;&lt;檔案名稱&gt;&#39;：&lt;錯誤&gt;

 **錯誤 ID︰**BC31025  
  
### 更正這個錯誤  
  
-   請確認金鑰檔存在，而且未遭其他應用程式鎖定。
