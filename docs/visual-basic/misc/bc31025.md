# <a name="unable-to-open-key-file-ltfilenamegt-lterrorgt"></a>無法開啟金鑰檔 '&lt;檔名&gt;':&lt;錯誤&gt;
無法開啟針對這個專案所指定的金鑰檔。  
  
 **錯誤 ID:** BC31025  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請確認金鑰檔存在，而且未遭其他應用程式鎖定。
