# <a name="set-not-supported-read-only-property"></a>不支援 Set (唯讀屬性)
您嘗試設定或變更唯讀屬性。  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
1.  從您的程式碼中移除屬性的參考。  
  
2.  變更參考，使其僅在執行階段傳回屬性的值。
