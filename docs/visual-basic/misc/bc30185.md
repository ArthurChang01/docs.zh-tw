# <a name="39enum39-must-end-with-a-matching-39end-enum39"></a>&#39;列舉 &#39;必須以相符 &#39; 結尾End Enum &#39;
`Enum` 陳述式沒有相對應的 `End Enum` 陳述式。 必須使用 `End Enum` 陳述式來結束列舉。  
  
 **錯誤 ID︰** BC30185  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
1.  請確認已正確地格式化 `Enum` 成員。  
  
2.  將 `End Enum` 陳述式加入列舉的結尾。
