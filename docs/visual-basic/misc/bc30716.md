# <a name="39finally39-statements-are-not-valid-in-the-immediate-window"></a>最後' 陳述式不是有效的即時運算視窗中
`Finally`陳述式僅能在原始程式碼中。  
  
 **錯誤 ID︰** BC30716  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請從偵錯程式碼中移除 `Finally` 陳述式。
