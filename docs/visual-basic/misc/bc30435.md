# <a name="members-in-a-structure-cannot-be-declared-specifier"></a>在結構中的成員不可以宣告 '\<規範 >'
您已經在 `Structure` 陳述式內使用成員上無效的規範。  
  
 **錯誤 ID:** BC30435  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請移除規範。
