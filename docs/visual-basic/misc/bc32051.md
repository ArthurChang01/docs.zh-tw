# <a name="type-argument-inference-failed-for-type-parameter-typeparametername-of-genericproceduresignature"></a><span data-ttu-id="300f9-102">' @No__t-1genericproceduresignature > ' 的類型參數 ' \<typeparametername > ' 的類型引數推斷失敗</span><span class="sxs-lookup"><span data-stu-id="300f9-102">Type argument inference failed for type parameter '\<typeparametername>' of '\<genericproceduresignature>'</span></span>

<span data-ttu-id="300f9-103">**錯誤識別碼：** BC32051</span><span class="sxs-lookup"><span data-stu-id="300f9-103">**Error ID:** BC32051</span></span>

<span data-ttu-id="300f9-104">' @No__t-1genericproceduresignature > ' 的類型參數 ' \<typeparametername > ' 的類型引數推斷失敗。</span><span class="sxs-lookup"><span data-stu-id="300f9-104">Type argument inference failed for type parameter '\<typeparametername>' of '\<genericproceduresignature>'.</span></span> <span data-ttu-id="300f9-105">無法從傳遞給參數 ' @no__t 0parametername > ' 的引數推斷類型引數。</span><span class="sxs-lookup"><span data-stu-id="300f9-105">Type argument could not be inferred from the argument passed to parameter '\<parametername>'.</span></span>

 <span data-ttu-id="300f9-106">泛型程序是在未提供任何類型引數的情況下呼叫，而且編譯器無法推斷要傳遞給其中一個參數的類型。</span><span class="sxs-lookup"><span data-stu-id="300f9-106">A generic procedure is called without supplying any type arguments, and the compiler cannot infer the type to pass to one of the parameters.</span></span>

 <span data-ttu-id="300f9-107">通常，當您呼叫泛型程序時，會針對泛型程序所定義的每個類型參數提供類型引數。</span><span class="sxs-lookup"><span data-stu-id="300f9-107">Normally, when you call a generic procedure, you supply a type argument for each type parameter that the generic procedure defines.</span></span> <span data-ttu-id="300f9-108">如果您未提供任何類型引數，則編譯器會嘗試推斷要傳遞給類型參數的類型。</span><span class="sxs-lookup"><span data-stu-id="300f9-108">If you do not supply any type arguments, then the compiler attempts to infer the types to be passed to the type parameters.</span></span> <span data-ttu-id="300f9-109">如果呼叫的內容提供類型參數的衝突資料類型資訊，則類型推斷會失敗。</span><span class="sxs-lookup"><span data-stu-id="300f9-109">If the context of the call provides conflicting data type information for a type parameter, then the type inference fails.</span></span>

 <span data-ttu-id="300f9-110">下列程式碼可能會產生此錯誤：</span><span class="sxs-lookup"><span data-stu-id="300f9-110">The following code can generate this error:</span></span>

```vb
Public Sub DoSomething(Of T)(arg1 As T(), arg2 As T)
End Sub

Public Sub Main()
    DoSomething(6, 42)
End Sub
```  
  
 <span data-ttu-id="300f9-111">在上述範例中，編譯器會根據傳遞給 `Integer` 的值 42 來推斷 `t` 的類型 `arg2`。</span><span class="sxs-lookup"><span data-stu-id="300f9-111">In the preceding example, the compiler infers type `Integer` for `t` based on the value of 42 passed to `arg2`.</span></span> <span data-ttu-id="300f9-112">不過，該推斷需要 `arg1` 為類型 `Integer()`(即 `Integer`的陣列)，而且傳遞給 `arg1` 的值 6 不符合該類型。</span><span class="sxs-lookup"><span data-stu-id="300f9-112">However, that inference would require `arg1` to be of type `Integer()`, that is, an array of `Integer`, and the value 6 passed to `arg1` does not match that type.</span></span>

## <a name="to-correct-this-error"></a><span data-ttu-id="300f9-113">更正這個錯誤</span><span class="sxs-lookup"><span data-stu-id="300f9-113">To correct this error</span></span>

- <span data-ttu-id="300f9-114">請將類型引數提供給泛型程序，這樣編譯器就不需要推斷它們。</span><span class="sxs-lookup"><span data-stu-id="300f9-114">Supply type arguments to the generic procedure, so that the compiler does not have to infer them.</span></span>

- <span data-ttu-id="300f9-115">請提供類型符合類型引數之類型的一般引數。</span><span class="sxs-lookup"><span data-stu-id="300f9-115">Supply normal arguments with types that match those of the type arguments.</span></span>
