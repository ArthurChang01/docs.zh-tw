# <a name="39ltassemblynamegt39-is-an-assembly-and-cannot-be-referenced-as-a-module"></a>&#39;&lt;assemblyname&gt; &#39;是組件，無法當做模組來參考
嘗試將組件當成模組來使用。  
  
 **錯誤 ID︰** BC31077  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請變更模組的參考。
