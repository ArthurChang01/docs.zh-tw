# <a name="argument-basepath-must-be-a-path-to-a-folder"></a>引數 BasePath 必須是資料夾路徑
引數 `BasePath` 必須是由資料夾路徑所組成。 您可能不正確地剖析字串，並提供無法辨識為有效路徑的值。  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   檢查您提供給 `BasePath` 的值，確定它是有效的資料夾路徑。
