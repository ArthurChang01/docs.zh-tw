# <a name="39end-region39-must-be-preceded-by-a-matching-39region39"></a>' #End region' 前面必須有相符的 '#Region'
使用`#Region`您可以指定一段程式碼來展開或摺疊時使用的大綱功能[!INCLUDE[vsprvs](../../csharp/includes/vsprvs_md.md)]程式碼編輯器。 `#Region` 陳述式的開頭和結尾必須在相同的程式碼區塊中。  
  
 **錯誤 ID：** BC30680  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請在對應 `#Region`  `#End``Region` 。
