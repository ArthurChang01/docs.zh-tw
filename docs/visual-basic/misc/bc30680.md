# <a name="39end-region39-must-be-preceded-by-a-matching-39region39"></a>&#39; #End 區域 &#39;之前必須搭配相對應的 &#39; #Region &#39;
透過 `#Region` ，您可以在使用 [!INCLUDE[vsprvs](~/includes/vsprvs-md.md)] 程式碼編輯器的大綱功能時，指定要展開或摺疊程式碼區塊。 `#Region` 陳述式的開頭和結尾必須在相同的程式碼區塊中。  
  
 **錯誤 ID：** BC30680  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請在對應 `#Region`  `#End``Region` 。
