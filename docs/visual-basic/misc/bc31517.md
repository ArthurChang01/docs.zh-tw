# <a name="type-39lttypenamegt39-cannot-be-used-as-an-attribute-because-its-container-39ltcontainernamegt39-is-not-declared-39public39"></a>類型 &#39;&lt;typename&gt;&#39; 不能用做為屬性中，因為其容器 &#39;&lt;containername&gt;&#39; 未宣告為 &#39;公用 &#39;
未使用 `Public` 修飾詞來宣告在其中定義這個屬性的類別或模組。 未指定存取修飾詞的類別和模組預設會宣告為 `Friend` 。  
  
 **錯誤 ID：** BC31517  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
1.  將 `Public` 修飾詞加入在其中定義這個屬性的類別或模組。
