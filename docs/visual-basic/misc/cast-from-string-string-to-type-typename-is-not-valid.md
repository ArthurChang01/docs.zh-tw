# <a name="cast-from-string-quotltstringgtquot-to-type-lttypenamegt-is-not-valid"></a>從字串轉換&quot;&lt;字串&gt;&quot;輸入 '&lt;typename&gt;' 無效
嘗試將字串變數轉換成另一種類型不正確。  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
1.  請檢查轉換中的類型，並確定您嘗試轉換成有意義的類型。
