# <a name="events-declared-with-an-39as39-clause-must-have-a-delegate-type"></a>事件宣告為 &#39;做為 &#39;子句必須具有委派類型
指定的類型不是委派。  
  
 **錯誤 ID︰** BC31044  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
1.  請移除 `As` 子句。  
  
2.  使用 `As` 子句來指定委派。
