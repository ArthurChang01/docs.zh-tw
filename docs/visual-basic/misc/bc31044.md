# <a name="events-declared-with-an-as-clause-must-have-a-delegate-type"></a>使用 'As' 子句宣告的事件必須有委派類型
指定的類型不是委派。  
  
 **錯誤 ID:** BC31044  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
1. 請移除 `As` 子句。  
  
2. 使用 `As` 子句來指定委派。
