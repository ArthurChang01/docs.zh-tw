# <a name="late-bound-assignment-to-a-field-of-value-type-lttypenamegt-is-not-valid-when-ltnamegt-is-the-result-of-a-late-bound-expression"></a>晚期繫結指派至欄位的實值型別 '&lt;typename&gt;'時無效'&lt;名稱&gt;' 是晚期繫結運算式的結果
您嘗試使用無效的晚期繫結指派。  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請將指派設定為早期繫結。
