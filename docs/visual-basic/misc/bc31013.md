# <a name="unable-to-load-dll-ltfilenamegt-lterrorgt"></a>無法載入 DLL '&lt;檔名&gt;':&lt;錯誤&gt;
無法載入這個專案所參考的 DLL。  
  
 **錯誤 ID:** BC31013  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
1.  請確認檔案位於指定的位置，且未被其他應用程式鎖定。
