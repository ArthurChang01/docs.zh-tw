# <a name="case-statements-are-not-valid-in-the-immediate-window"></a>即時運算視窗中的 'Case' 陳述式無效
`Case` 陳述式只能用於原始程式碼中。  
  
 **錯誤 ID:** BC30719  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 請從 [即時運算] `Case`**視窗中移除** 陳述式。
