# <a name="39currency39-is-no-longer-a-supported-type-use-the-39decimal39-type-instead"></a>&#39;貨幣 &#39;已不再支援的型別。使用 &#39; 十進位 &#39;改為輸入
`Currency` 資料類型取代了 `Decimal` 資料類型。  
  
 **錯誤 ID︰** BC30803  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   使用 `Decimal` 資料類型取代 `Currency`。
