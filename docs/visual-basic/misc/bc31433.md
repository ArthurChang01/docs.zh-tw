# <a name="method-ltmethodnamegt-cannot-be-declared-partial-because-only-one-method-ltmethodnamegt-can-be-marked-partial"></a>方法 '&lt;methodname&gt;' 不可以宣告 'Partial'，因為只有一個方法 '&lt;methodname&gt;' 可以標記為 'Partial'
您不能有兩個部分方法宣告具有相同名稱和簽章。  
  
 **錯誤 ID:** BC31433  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
1.  請變更其中一個部分方法宣告的名稱或簽章。
