# <a name="end-statement-not-valid"></a>'End' 陳述式無效
`End` 關鍵字的用法不正確，或不適合用在目前的內容中。  
  
 **錯誤 ID:** BC30678  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
1.  僅使用 `End` 會導致程序遺失範圍。  
  
2.  搭配使用 `End` 與其他關鍵字來終止一段程式碼 (例如結構或屬性)。
