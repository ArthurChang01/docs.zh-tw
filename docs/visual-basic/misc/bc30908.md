<span data-ttu-id="92d41-103">類別或介面的定義包含[繼承陳述式](../../visual-basic/language-reference/statements/inherits-statement.md)指定巢狀的類型。</span><span class="sxs-lookup"><span data-stu-id="92d41-103">A class or interface definition includes an [Inherits Statement](../../visual-basic/language-reference/statements/inherits-statement.md) that specifies a type nested within it.</span></span>  
  
 <span data-ttu-id="92d41-104">繼承必須是線性，而非循環。</span><span class="sxs-lookup"><span data-stu-id="92d41-104">Inheritance must be linear, not circular.</span></span> <span data-ttu-id="92d41-105">類型無法繼承自繼承它的類型。</span><span class="sxs-lookup"><span data-stu-id="92d41-105">A type cannot inherit from a type that inherits from it.</span></span>  
  
 <span data-ttu-id="92d41-106">相關的限制是類型無法繼承自尚未定義的類型。</span><span class="sxs-lookup"><span data-stu-id="92d41-106">A related restriction is that a type cannot inherit from a type that is not yet defined.</span></span> <span data-ttu-id="92d41-107">繼承涉及重複使用基底類別成員的功能，因此需要定義這些成員。</span><span class="sxs-lookup"><span data-stu-id="92d41-107">Inheritance involves the ability to reuse members of the base class, which in turn requires that these members be defined.</span></span> <span data-ttu-id="92d41-108">因此，[!INCLUDE[vbprvb](../../csharp/programming-guide/concepts/linq/includes/vbprvb_md.md)]無法編譯程式碼，如以下範例。</span><span class="sxs-lookup"><span data-stu-id="92d41-108">Therefore, [!INCLUDE[vbprvb](../../csharp/programming-guide/concepts/linq/includes/vbprvb_md.md)] cannot compile code such as the following example.</span></span>  
  
```

    ' The following statement is INVALID because innerClass is not defined.  
    Inherits innerClass  
    Public Class innerClass  
        Public Sub doSomething()  
        End Sub  
    End Class  
End Class  
```
