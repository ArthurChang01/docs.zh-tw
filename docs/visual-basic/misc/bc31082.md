# <a name="39ltvariablenamegt39-is-not-a-local-variable-or-parameter-and-so-cannot-be-used-as-a-39catch39-variable"></a>&#39;&lt;variablename&gt; &#39;不是區域變數或參數，所以無法用來做為&#39;攔截&#39;變數
`Try...Catch...Finally` 陳述式中的變數必須在本機宣告，或為目前程序的參數。  
  
 **錯誤 ID︰** BC31082  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
1.  宣告 `Try...Catch...Finally` 陳述式的區域變數或參數。
