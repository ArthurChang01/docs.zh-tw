# <a name="end-statement-cannot-be-used-in-class-library-projects"></a>不可以在類別庫專案中使用 'End' 陳述式
用來建立 DLL 的類別庫專案不允許使用 `End` 關鍵字來停止執行程序中的程式碼。  
  
 **錯誤 ID:** BC30615  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 使用控制結構 (例如 `While` 和 `For` ) 來控制程式執行流程。
