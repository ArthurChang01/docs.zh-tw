# <a name="39end39-statement-cannot-be-used-in-class-library-projects"></a>&#39;結束 &#39;陳述式不能在類別庫專案
用來建立 DLL 的類別庫專案不允許使用 `End` 關鍵字來停止執行程序中的程式碼。  
  
 **錯誤 ID：** BC30615  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   使用控制結構 (例如 `While` 和 `For` ) 來控制程式執行流程。
