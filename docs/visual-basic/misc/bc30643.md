# 屬性 &#39;&lt;屬性名稱&gt;&#39; 屬於不支援的類型
參考組件中的屬性包含無法表示的類型。  
  
 **錯誤 ID：**BC30643  
  
### 更正這個錯誤  
  
-   指定可搭配您的應用程式使用的類型。
