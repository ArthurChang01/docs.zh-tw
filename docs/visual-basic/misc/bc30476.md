# <a name="withevents-variables-cannot-be-typed-as-arrays"></a>'WithEvents' 變數的類型不可以是陣列
已嘗試使用 `WithEvents` 宣告陣列。 您可以使用 `WithEvents`宣告許多個別變數，數目不限，但不能用這種方式宣告陣列。  
  
 **錯誤 ID:** BC30476  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
1.  個別宣告變數。
