# <a name="only-one-of-notoverridable-mustoverride-or-overridable-can-be-specified"></a>只可以指定 'NotOverridable'、'MustOverride' 或 'Overridable' 其中之一
項目宣告中發生覆寫規格衝突。 覆寫關鍵字互斥。  
  
 **錯誤 ID:** BC30177  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請選擇一個覆寫規範，並移除宣告中的其他關鍵字。
