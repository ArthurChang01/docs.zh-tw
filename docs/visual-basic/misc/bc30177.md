# <a name="only-one-of-39notoverridable39-39mustoverride39-or-39overridable39-can-be-specified"></a>只有其中一個&#39;NotOverridable&#39;， &#39;MustOverride&#39;，或&#39;Overridable&#39;可以指定
項目宣告中發生覆寫規格衝突。 覆寫關鍵字互斥。  
  
 **錯誤 ID︰** BC30177  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請選擇一個覆寫規範，並移除宣告中的其他關鍵字。
