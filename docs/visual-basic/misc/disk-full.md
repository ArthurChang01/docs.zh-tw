# 磁碟已滿 (Visual Basic)
磁碟上的空間不足，無法完成 `Print`、`Write` 或 `FileClose` 作業，或建立必要的檔案。  
  
### 更正這個錯誤  
  
-   將一些檔案移至另一個磁碟，或刪除一些檔案，以在磁碟上建立更多的空間。
