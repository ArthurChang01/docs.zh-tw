# &#39;Sub New&#39; 不可宣告為 &#39;&lt;修飾詞&gt;&#39;
您嘗試使用修飾詞宣告 `Sub New` \(例如 `Overrides`\)，但此做法無效。  
  
 **錯誤 ID：**BC30364  
  
### 更正這個錯誤  
  
-   請移除修飾詞。
