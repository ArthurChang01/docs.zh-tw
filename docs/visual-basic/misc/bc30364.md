# <a name="sub-new-cannot-be-declared-modifier"></a>'Sub New' 不可宣告為 '\<修飾詞 >'
您嘗試使用修飾詞宣告 `Sub New` (例如 `Overrides`)，但此做法無效。  
  
 **錯誤 ID:** BC30364  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 請移除修飾詞。
