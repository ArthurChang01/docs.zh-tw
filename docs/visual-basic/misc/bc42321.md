# <a name="badly-formed-xml-file-commentfile-cannot-be-included"></a>XML 格式不正確。 檔案 '\<註解檔案 >' 不能包含
XML 註解中其中一個 `<include>` 標記參考了不存在的檔案或檔案中的 XML 格式不正確。  
  
 **錯誤 ID:** BC42321  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
1.  請確認檔案存在。  
  
2.  請確認檔案中的 XML 有效。
