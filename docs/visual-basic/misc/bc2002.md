# <a name="source-file-filename-specified-multiple-times"></a>原始程式檔\<檔名 > 指定多次
已指定多次檔案。  
  
 **錯誤 ID:** BC2002  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請移除多餘的檔案規範。
