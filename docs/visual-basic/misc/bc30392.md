# <a name="catch-cannot-catch-type-lttypenamegt-because-it-is-not-systemexception-or-a-class-that-inherits-from-systemexception"></a>'Catch' 無法攔截類型 '&lt;typename&gt;' 因為它不是 'System.Exception' 或繼承自 'System.Exception' 的類別
`Catch` 只能攔截例外狀況，而您已嘗試攔截非衍生自例外狀況的項目。  
  
 **錯誤 ID:** BC30392  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
1.  移除 `Catch` 陳述式，或將 `Catch` 的目標變更為實際例外狀況。
