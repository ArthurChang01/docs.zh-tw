# <a name="label-ltlabelnamegt-is-not-defined"></a>標籤 '&lt;labelname&gt;' 未定義
在參考的範圍內，已參考但未定義行標籤或行號。 標籤必須位於包含參考的程序內。  
  
 **錯誤 ID:** BC30132  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   重新建構程式碼，讓行標籤和參考位於相同的程序中。
