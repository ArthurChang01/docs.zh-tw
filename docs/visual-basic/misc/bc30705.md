# <a name="referenced-object-39ltobjectnamegt39-has-a-value-of-39nothing39"></a>參考的物件 '&lt;objectname&gt;' 的值為 'Nothing'
參考的物件尚未設定，或已設定為 `Nothing`值。  
  
 **錯誤 ID︰** BC30705  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請確認物件名稱拼字正確，且已設定該物件。
