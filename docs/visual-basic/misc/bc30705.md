# <a name="referenced-object-39ltobjectnamegt39-has-a-value-of-39nothing39"></a>參考的物件&#39; &lt;objectname&gt; &#39;的值為&#39;做任何動作&#39;
參考的物件尚未設定，或已設定為 `Nothing`值。  
  
 **錯誤 ID︰** BC30705  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請確認物件名稱拼字正確，且已設定該物件。
