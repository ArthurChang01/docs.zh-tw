# <a name="cast-from-type-lttypename1gt-to-type-lttypename2gt-is-not-valid"></a>從類型轉換成 '&lt;typename1&gt;' 到類型'&lt;2&gt&gt;' 無效
嘗試將變數從一種類型轉換成另一種類型不正確。  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
1.  請檢查轉換中的類型，並確定您嘗試轉換成有意義的類型。
