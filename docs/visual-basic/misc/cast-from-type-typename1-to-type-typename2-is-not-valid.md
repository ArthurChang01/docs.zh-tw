# <a name="cast-from-type-39lttypename1gt39-to-type-39lttypename2gt39-is-not-valid"></a>從型別轉型&#39; &lt;typename1&gt; &#39;輸入&#39; &lt;2>&gt; &#39;無效
嘗試將變數從一種類型轉換成另一種類型不正確。  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
1.  請檢查轉換中的類型，並確定您嘗試轉換成有意義的類型。
