# <a name="interface-ltinterfacenamegt-can-be-implemented-only-once-by-this-type"></a>介面 '&lt;interfacename&gt;' 此類型可以一次實作
指定的介面已經透過這個類型進行實作，因此無法重新進行實作。  
  
 **錯誤 ID:** BC31033  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   只能使用這個類型實作這個介面一次。
