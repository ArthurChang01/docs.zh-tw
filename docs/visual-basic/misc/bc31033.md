# 介面 &#39;&lt;介面名稱&gt;&#39; 只能由此類型實作一次
指定的介面已經透過這個類型進行實作，因此無法重新進行實作。  
  
 **錯誤 ID︰**BC31033  
  
### 更正這個錯誤  
  
-   只能使用這個類型實作這個介面一次。
