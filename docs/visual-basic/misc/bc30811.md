# <a name="redim-statements-can-no-longer-be-used-to-declare-array-variables"></a>無法再使用 'ReDim' 陳述式來宣告陣列變數
`ReDim` 只能用來變更現有陣列的大小。  
  
 **錯誤 ID:** BC30811  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 當宣告陣列時指定其大小；例如：  
  
    ```  
    Dim X(20) As Integer  
    ```
