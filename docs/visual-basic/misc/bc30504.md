# <a name="property-propertyname-cannot-be-declared-default-because-it-overrides-a-property-on-the-base-type-typename-that-is-not-default"></a>屬性 '\<屬性名稱 >' 不能宣告 'Default'，因為它會覆寫基底上的屬性\<類型 >'\<類型名稱 >' 非預設
您已嘗試宣告屬性 `Default`，這不是有效的宣告，因為它會覆寫未在基底類別上宣告為 `Default` 的屬性。  
  
 **錯誤 ID:** BC30504  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請移除 `Default` 宣告。
