# <a name="expression-is-of-type-39lttypenamegt39-which-is-not-a-collection-type"></a>運算式屬於類型&#39; &lt;typename&gt;&#39;，這不是集合型別
`For Each` 陳述式中所指定的群組變數不是集合物件或陣列，而且其類型未實作 <xref:System.Collections.IEnumerable> 介面。 類型必須支援 Visual Basic 集合設計模式或實作<xref:System.Collections.IEnumerable>。  
  
 **錯誤 ID：** BC32023  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   群組變數宣告為類別類型為支援 Visual Basic 集合設計或實作<xref:System.Collections.IEnumerable>。
