# <a name="ltspecifiergt-is-not-valid-on-a-structure-declaration"></a>'&lt;規範&gt;' 在 Structure 宣告中無效
您使用了對 `Dim` 宣告無效的指定名稱，例如 `Override`或 `Structure` 。  
  
 **錯誤 ID:** BC30395  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請移除規範。
