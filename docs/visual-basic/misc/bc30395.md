# <a name="39ltspecifiergt39-is-not-valid-on-a-structure-declaration"></a>&#39;&lt;規範&gt;&#39; 在 Structure 宣告中無效
您使用了對 `Dim` 宣告無效的指定名稱，例如 `Override`或 `Structure` 。  
  
 **錯誤識別碼：** BC30395  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請移除規範。
