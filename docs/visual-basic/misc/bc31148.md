# <a name="xml-namespace-prefix-39prefix39-is-not-defined"></a>XML 命名空間前置詞 &#39; 前置詞 &#39;未定義
已針對 XML 常值或 XML 軸屬性，指定未定義的 XML 命名空間前置詞。  
  
 **錯誤 ID︰** BC31148  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   使用 `Imports` 陳述式，來定義 XML 命名空間前置詞。  
  
 -或-  
  
-   將 XML 常值或 XML 軸屬性的 XML 命名空間前置詞修改為先前定義的 XML 命名空間前置詞。
