# <a name="xml-namespace-prefix-prefix-is-not-defined"></a>未定義 XML 命名空間前置詞 'prefix'
已針對 XML 常值或 XML 軸屬性，指定未定義的 XML 命名空間前置詞。  
  
 **錯誤 ID:** BC31148  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 使用 `Imports` 陳述式，來定義 XML 命名空間前置詞。  
  
 -或-  
  
- 將 XML 常值或 XML 軸屬性的 XML 命名空間前置詞修改為先前定義的 XML 命名空間前置詞。
