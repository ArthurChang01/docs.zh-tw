# <a name="argument-matching-parameter-parametername-narrows-from-type1-to-type2"></a>對應參數的引數 '\<參數名稱 >' 會從'\<type1 >' 到 '\<type2 >'
您已呼叫多載的方法，但編譯器找不到可以呼叫且不用縮小轉換的方法。 縮小轉換會將值變更為資料類型，而可能無法精確保留一些可能的值。  
  
 **錯誤 ID:** BC30520  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 請指定 `Option Strict Off`。
