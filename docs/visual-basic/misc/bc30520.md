# <a name="argument-matching-parameter-39ltparameternamegt39-narrows-from-39lttype1gt39-to-39lttype2gt39"></a>引數符合參數&#39; &lt;parametername&gt; &#39;從縮小&#39; &lt;type1&gt; &#39;至&#39; &lt;type2&gt;&#39;
您已呼叫多載的方法，但編譯器找不到可以呼叫且不用縮小轉換的方法。 縮小轉換會將值變更為資料類型，而可能無法精確保留一些可能的值。  
  
 **錯誤 ID：** BC30520  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請指定 `Option Strict Off`。
