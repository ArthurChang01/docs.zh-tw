# <a name="referenced-membername-has-a-value-of-nothing"></a>參考 '\<成員名稱 >' 的值為 'Nothing'
所參考的項目尚未獲指派值，或已獲指派值 `Nothing`。  
  
 **錯誤 ID:** BC30759  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請確認項目名稱的拼寫正確且設定正確。
