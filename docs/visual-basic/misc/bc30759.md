# 參考的 &#39;&lt;成員名稱&gt;&#39; 具有 &#39;Nothing&#39; 的值
所參考的項目尚未獲指派值，或已獲指派值 `Nothing`。  
  
 **錯誤 ID︰**BC30759  
  
### 更正這個錯誤  
  
-   請確認項目名稱的拼寫正確且設定正確。
