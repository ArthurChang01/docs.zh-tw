# <a name="referenced-39ltmembernamegt39-has-a-value-of-39nothing39"></a>參考 &#39;&lt;membername&gt;&#39; 具有的值 &#39;執行任何動作 &#39;
所參考的項目尚未獲指派值，或已獲指派值 `Nothing`。  
  
 **錯誤 ID︰** BC30759  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請確認項目名稱的拼寫正確且設定正確。
