# <a name="definition-of-method-ltmethodgt-is-not-accessible-in-this-context"></a>方法定義&lt;方法&gt;不在此內容中存取
不存在的彙總函式已隨附於 `Into` 運算子。  
  
 **錯誤 ID︰** BC36594  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請檢查彙總函式名稱的拼字，並進行任何必要的更正。  
  
-   請將方法以自訂彙總函式 (擴充方法) 加入。 如需有關如何建立自訂的彙總函式的詳細資訊，請參閱[Aggregate 子句](../../visual-basic/language-reference/queries/aggregate-clause.md)。
