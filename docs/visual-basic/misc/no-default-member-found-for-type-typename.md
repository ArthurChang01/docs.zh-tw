# <a name="no-default-member-found-for-type-typename"></a>沒有預設成員找到類型 '\<類型名稱 >'
已嘗試晚期繫結索引 `Get` 或 `Set` 呼叫。  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
1.  請確定物件具有 `Get` 或 `Set`的預設屬性，視您要進行何者而定。
