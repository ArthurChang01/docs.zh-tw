# <a name="event-lteventnamegt-cannot-be-found"></a>事件 '&lt;eventname&gt;' 找不到
您嘗試處理編譯器找不到的事件。  
  
 **錯誤 ID:** BC30590  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請確定事件名稱的拼字正確。
