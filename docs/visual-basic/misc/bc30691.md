# <a name="lttypenamegt-is-a-type-in-ltcontainernamegt-and-cannot-be-used-as-an-expression"></a>'&lt;typename&gt;'是中的類型'&lt;containername&gt;'，無法當成運算式使用
正在使用的名稱之一已經宣告為類型名稱，不能用在運算式中。  
  
 **錯誤 ID:** BC30691  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   成員的唯一名稱。  
  
-   成員名稱請使用方括弧括住。
