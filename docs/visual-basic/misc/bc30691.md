# <a name="39lttypenamegt39-is-a-type-in-39ltcontainernamegt39-and-cannot-be-used-as-an-expression"></a>&#39;&lt;typename&gt; &#39;是中的型別&#39; &lt;containername&gt; &#39;和無法當成運算式使用
正在使用的名稱之一已經宣告為類型名稱，不能用在運算式中。  
  
 **錯誤 ID：** BC30691  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   成員的唯一名稱。  
  
-   成員名稱請使用方括弧括住。
