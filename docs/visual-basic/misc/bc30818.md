# <a name="function-is-not-declared"></a>'\<函式 >' 未宣告
'\<函式 >' 未宣告。 函式已移至 Microsoft.VisualBasic 命名空間。  
  
 **錯誤 ID:** BC30818  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   使用來自 `Microsoft.VisualBasic` 命名空間的適當函式。
