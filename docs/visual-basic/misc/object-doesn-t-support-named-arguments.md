# <a name="object-doesnt-support-named-arguments"></a>物件不支援具名引數
您嘗試存取的物件具有不支援具名引數的方法。  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 存取這個物件上的方法時，請依位置指定引數。 如需引數位置和類型的詳細資訊，請參閱物件的文件。
