# <a name="readonly-property-propertyname-cannot-be-the-target-of-an-assignment"></a>'ReadOnly' 屬性 '\<屬性名稱 >' 不可以是指派的目標
`ReadOnly` 屬性會發生在指派其值的內容中。 在執行期間，只會指派可寫入變數、屬性和陣列項目的值。  
  
 **錯誤 ID:** BC30098  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 從宣告變數的 `ReadOnly` 陳述式中移除 `Property` 關鍵字，或移除將值指派給它的陳述式。
