# <a name="nullable-modifier-cannot-be-specified-in-variable-declarations-with-39as-new39"></a>可為 null 的修飾詞不能指定變數的宣告與 &#39;為新 &#39;
在指定 `As New` 的變數宣告中已包括可為 Null 的類型修飾詞 (?)。 下列範例會產生這個錯誤：  
  
```vb  
Dim num? As New ExampleStructure  
```  
  
 **錯誤 ID︰** BC33109  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
1.  請從可為 Null 的變數宣告中移除 `New` 關鍵字 (如下列範例所示)：  
  
```vb  
Dim num? As ExampleStructure  
```
