# <a name="end-of-parameter-list-expected"></a>必須有參數清單的結尾
`ParamArray` 規格後面接著程序宣告中的一個或多個其他引數規格。  
  
 **錯誤 ID︰** BC30192  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請將 `ParamArray` 移到引數清單的結尾。
