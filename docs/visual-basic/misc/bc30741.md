# <a name="lttypename1gt-cannot-be-converted-to-lttypename2gt"></a>'&lt;typename1&gt;'無法轉換成'&lt;2&gt&gt;'
已嘗試在兩個不相容的類型之間進行類型轉換。  
  
 **錯誤 ID:** BC30741  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請確認正在轉換的類型。
