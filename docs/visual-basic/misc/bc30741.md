# 無法將 &#39;&lt;類型名稱1&gt;&#39; 轉換成 &#39;&lt;類型名稱2&gt;&#39;
已嘗試在兩個不相容的類型之間進行類型轉換。  
  
 **錯誤 ID︰**BC30741  
  
### 更正這個錯誤  
  
-   請確認正在轉換的類型。
