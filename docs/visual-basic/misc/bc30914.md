# <a name="name-cannot-expose-the-underlying-delegate-type-delegatetype-of-the-event-it-is-implementing-to-specifier1-type1-through-specifier2-type2"></a>'\<名稱 >' 無法公開 （expose) 的基礎委派類型'\<委派 >' 其實作之事件\<specifier1 > '\<type1 >' 透過\<specifier2 >'\<type2 >'
這個程式碼會在公用類別外部公開私用類型。  
  
 **錯誤 ID:** BC30914  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 宣告 `Public` 類型或使用另一種類型。
