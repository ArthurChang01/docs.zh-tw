# <a name="the-file-is-currently-closed"></a>目前已關閉檔案
嘗試執行的作業只能對開啟的檔案執行，但目前已關閉檔案。  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 請開啟檔案，然後再次嘗試這項作業。
