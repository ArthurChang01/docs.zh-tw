# <a name="modifier-is-not-valid-on-a-delegate-declaration"></a>'\<修飾詞 >' 在 Delegate 宣告中無效
您嘗試在 Delegate 宣告上使用修飾詞 (例如 `Shared`)。 委派是用來呼叫其他物件之方法的物件，方法是定義獲傳遞物件方法規格的建構函式。 在 Delegate 宣告上指定修飾詞無效。  
  
 **錯誤 ID:** BC30385  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
1.  請移除修飾詞。
