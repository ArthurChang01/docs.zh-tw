# <a name="39get39-statements-are-no-longer-supported"></a>&#39;Get &#39;不再支援陳述式
不再支援`Get` 陳述式。 檔案 I/O 功能可以在 `Microsoft.VisualBasic` 命名空間中使用。  
  
 `Get` 不受檔案作業的支援，只能用於屬性程序語法。  
  
 **錯誤 ID︰** BC30829  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
1.  使用 `System.IO`、 `FileSystemObject`和 [!INCLUDE[vbprvb](~/includes/vbprvb-md.md)] 執行階段函式的成員來執行檔案作業。
