# <a name="readonly-attribute-property-propertyfield-cannot-be-the-target-of-an-assignment"></a>'ReadOnly' 屬性 '\<propertyfield >' 不可以是指派的目標
已嘗試將值指派給屬性 (attribute) 中的 `ReadOnly` 屬性 (property)。  
  
 **錯誤 ID:** BC31501  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
1. 請移除屬性指派陳述式。  
  
2. 如果使用您所開發的屬性，請從屬性 (attribute) 的屬性 (property) 中移除 `ReadOnly` 或 `Shared` 修飾詞。
