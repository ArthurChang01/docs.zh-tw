# <a name="data-types-of-the-type-parameters-in-extension-method-methodname-defined-in-typename-cannot-be-inferred-from-these-arguments-because-they-do-not-convert-to-the-same-type"></a><span data-ttu-id="42271-102">無法從這些引數推斷在 ' typename ' 中定義之擴充方法 '\<方法名稱 > ' 中類型參數的資料類型, 因為它們不會轉換成相同的類型</span><span class="sxs-lookup"><span data-stu-id="42271-102">Data type(s) of the type parameter(s) in extension method '\<methodname>' defined in 'typename' cannot be inferred from these arguments because they do not convert to the same type</span></span>

<span data-ttu-id="42271-103">無法從這些引數推斷在 ' typename ' 中定義之擴充方法 '\<方法名稱 > ' 中類型參數的資料類型, 因為它們不會轉換成相同的類型。</span><span class="sxs-lookup"><span data-stu-id="42271-103">Data type(s) of the type parameter(s) in extension method '\<methodname>' defined in 'typename' cannot be inferred from these arguments because they do not convert to the same type.</span></span> <span data-ttu-id="42271-104">明確指定資料類型或許可以更正這個錯誤。</span><span class="sxs-lookup"><span data-stu-id="42271-104">Specifying the data type(s) explicitly might correct this error.</span></span>

<span data-ttu-id="42271-105">嘗試使用類型推斷，來判斷評估泛型擴充方法呼叫時之類型參數的資料類型。</span><span class="sxs-lookup"><span data-stu-id="42271-105">An attempt has been made to use type inference to determine the data type (or types) of the type parameter (or parameters) when evaluating a call to a generic extension method.</span></span> <span data-ttu-id="42271-106">編譯器找不到符合所有引數之條件約束的資料類型。</span><span class="sxs-lookup"><span data-stu-id="42271-106">The compiler could not find a data type that meets the constraints of all the arguments.</span></span> <span data-ttu-id="42271-107">因此，它已回報這個錯誤。</span><span class="sxs-lookup"><span data-stu-id="42271-107">Therefore, it reported this error.</span></span>

> [!NOTE]
> <span data-ttu-id="42271-108">當無法指定引數時 (例如，針對查詢運算式中的查詢運算子)，會顯示不含第二句的錯誤訊息。</span><span class="sxs-lookup"><span data-stu-id="42271-108">When specifying arguments is not an option (for example, for query operators in query expressions), the error message appears without the second sentence.</span></span>

<span data-ttu-id="42271-109">下列程式碼示範這個錯誤。</span><span class="sxs-lookup"><span data-stu-id="42271-109">The following code demonstrates the error.</span></span>

```vb
Option Strict Off
Module Module3
    Sub Main()

        Dim c1 As New Class1

        '' Not valid. Integer and Date do not convert to the same type.
        'c1.targetMethod(19, #3/4/2007#)

    End Sub

    <System.Runtime.CompilerServices.Extension()> _
    Sub targetMethod(Of T)(ByVal p0 As Class1, ByVal p1 As T, ByVal p2 As T)
    End Sub

    Class Class1
    End Class

End Module
```

## <a name="to-correct-this-error"></a><span data-ttu-id="42271-111">更正這個錯誤</span><span class="sxs-lookup"><span data-stu-id="42271-111">To correct this error</span></span>
