# <a name="property-value-is-not-valid"></a>屬性值無效
物件或控制項的其中一個屬性已設定為超出其範圍外部的值。  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請將屬性值變更為有效的設定。
