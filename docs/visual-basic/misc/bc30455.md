# <a name="argument-not-specified-for-parameter-39ltparameternamegt39-of-39ltmethodnamegt39"></a>未指定參數的引數&#39; &lt;parametername&gt; &#39;的&#39;&lt;方法名稱&gt;&#39;

 **錯誤 ID︰** BC30455  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請為指定的參數提供引數。
