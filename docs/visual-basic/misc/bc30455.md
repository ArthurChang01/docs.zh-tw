# <a name="argument-not-specified-for-parameter-ltparameternamegt-of-ltmethodnamegt"></a>未指定參數的引數 '&lt;parametername&gt;'的'&lt;methodname&gt;'
尚未提供必要參數的引數。  
  
 **錯誤 ID:** BC30455  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請為指定的參數提供引數。
