# <a name="the-file-39ltfilenamegt39-is-not-a-text-file"></a>檔案 &#39;&lt;filename&gt;&#39; 不是文字檔案
無法寫入指定的檔案，因為它是二進位檔。  
  
 **錯誤 ID︰** BC2015  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請檢查您提供的檔名和路徑是否正確。
