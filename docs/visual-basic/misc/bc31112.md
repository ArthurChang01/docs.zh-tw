# <a name="statement-cannot-appear-within-an-event-body"></a>陳述式不可出現在事件主體中
陳述式不可出現在事件主體中。 已假設是事件結尾。  
  
 在事件主體中不正確的陳述式出現在一個事件主體中。  
  
 **錯誤 ID︰** BC31112  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   在陳述式之前加入 `End Event` 。
