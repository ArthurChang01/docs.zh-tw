# <a name="branching-out-of-a-39finally39-is-not-valid"></a>分支&#39;最後&#39;無效
`GoTo` 區塊內的 `Finally` 陳述式分叉到區塊之外。 分叉進入或離開 `Catch` 或 `Finally` 區塊是無效的。  
  
 **錯誤 ID︰** BC30101  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請移除 `GoTo` 陳述式，並考慮使用決策或迴圈控制結構來實作程式邏輯。
