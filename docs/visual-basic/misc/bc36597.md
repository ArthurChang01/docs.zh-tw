# <a name="goto-to-label-39ltlabelnamegt39-cannot-be-used-within-lambda-expressions"></a>GoTo 標籤 &#39;&lt;labelname&gt;&#39; 不能用於 lambda 運算式
Lambda 運算式的主體必須是計算並傳回單一值的運算式。 不允許任何陳述式，包括 `GoTo` 陳述式。  
  
 **錯誤 ID︰** BC36597
