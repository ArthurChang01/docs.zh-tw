# <a name="39ltmathfunction1gt39-is-not-declared"></a>&#39;&lt;數學函式 1&gt&gt; &#39;未宣告
'\<數學函式 1&gt >' 未宣告。 函式已移至 System.Math 類別，現在稱為 '\<mathfunction2 >'。  
  
 Visual basic 在舊版中內建的數個函數都已移至<xref:System.Math?displayProperty=nameWithType>命名空間。 因此其功能可更廣泛地供所有程式設計語言使用。  
  
 **錯誤 ID︰** BC30819  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   使用中宣告的方法<xref:System.Math?displayProperty=nameWithType>。
