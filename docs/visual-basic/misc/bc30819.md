# <a name="39ltmathfunction1gt39-is-not-declared"></a>'&lt;mathfunction1&gt;' 未宣告
'\<mathfunction1 >' 未宣告。 函式已移至 System.Math 類別，而現在改稱為 '\<mathfunction2 >'。  
  
 已內建函式的幾個函式[!INCLUDE[vbprvb](../../csharp/programming-guide/concepts/linq/includes/vbprvb_md.md)]先前版本中都已移至<xref:System.Math?displayProperty=fullName>命名空間。</xref:System.Math?displayProperty=fullName> 因此其功能可更廣泛地供所有程式設計語言使用。  
  
 **錯誤 ID︰** BC30819  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   使用<xref:System.Math?displayProperty=fullName>.</xref:System.Math?displayProperty=fullName>中宣告的方法
