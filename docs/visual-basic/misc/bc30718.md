# <a name="39select39-statements-are-not-valid-in-the-immediate-window"></a>&#39;Select &#39;陳述式不是有效的即時運算視窗中
`Select` 陳述式只能用於原始程式碼中。  
  
 **錯誤 ID︰** BC30718  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請從偵錯程式碼中移除 `Select` 陳述式。
