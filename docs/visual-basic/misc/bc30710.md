# <a name="end-of-expression-expected"></a>必須是運算式結尾
本應完整的運算式結尾發現多餘的字元。  
  
 **錯誤 ID:** BC30710  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   確認物件或陳述式的語法。
