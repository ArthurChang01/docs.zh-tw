# <a name="optional-parameters-cannot-be-declared-as-the-type-39lttypegt39"></a>選擇性參數不能宣告類型為&#39;&lt;類型&gt;&#39;
選擇性參數不可為結構的資料類型。  
  
 **錯誤 ID：** BC30423  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
1.  若要將結構傳遞給選擇性參數，請將參數宣告為 `Object`。  
  
2.  使用 `CType` 將參數轉型成程序中的結構類型。
