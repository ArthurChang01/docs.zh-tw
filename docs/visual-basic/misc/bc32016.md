# <a name="ltprocedurenamegt-has-no-parameters-and-its-return-type-cannot-be-indexed"></a>'&lt;程序名稱&gt;' 沒有參數，以及其傳回型別無法編製索引
`Function` 或 `Sub` 程序的呼叫提供一個或多個引數，但程序不接受任何引數，而且其傳回類型 (如果是 `Function`) 不是陣列類型。  
  
 **錯誤 ID:** BC32016  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請移除程序呼叫中的引數。
