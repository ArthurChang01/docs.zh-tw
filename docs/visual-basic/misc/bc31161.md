# <a name="expected-closing---gt-for-xml-comment"></a>預期的結尾 '-&gt;' 的 XML 註解
XML 註解常值遺漏必要的結尾 '<' 字元。  
  
 **錯誤 ID:** BC31161  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請將必要的結尾 '-->' 字元加入 XML 註解常值的結尾。
