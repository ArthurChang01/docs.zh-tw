# <a name="conversion-from-39lttype1gt39-to-39lttype2gt39-cannot-occur-in-a-constant-expression"></a>從轉換&#39; &lt;type1&gt; &#39;至&#39; &lt;type2&gt; &#39;不能出現在常數運算式
`Const` 陳述式中的初始化運算式評估為無法轉換成常數之宣告類型的資料類型。  
  
 **錯誤 ID︰** BC30060  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
1.  請使用可轉換成針對常數所宣告之類型的資料類型的運算式，來初始化常數。
