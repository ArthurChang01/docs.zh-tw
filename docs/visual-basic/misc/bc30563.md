# <a name="39ltnamegt39-is-ambiguous-in-the-application-objects-39ltlistgt39"></a>'&lt;名稱&gt;'模稜兩可在應用程式物件'&lt;清單&gt;'
您提供的名稱模稜兩可，因此與另一個名稱發生衝突。 [!INCLUDE[vbprvb](../../csharp/programming-guide/concepts/linq/includes/vbprvb_md.md)] 編譯器並沒有任何衝突解決規則，您必須自行區分名稱。 錯誤訊息中提供的清單詳細說明名稱模稜兩可的應用程式物件。  
  
 **錯誤 ID：** BC30563  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   完整限定名稱。
