# 應用程式物件 &#39;&lt;清單&gt;&#39; 中的 &#39;&lt;名稱&gt;&#39; 模稜兩可
您提供的名稱模稜兩可，因此與另一個名稱發生衝突。[!INCLUDE[vbprvb](../../csharp/programming-guide/concepts/linq/includes/vbprvb-md.md)] 編譯器並沒有任何衝突解決規則，您必須自行區分名稱。 錯誤訊息中提供的清單詳細說明名稱模稜兩可的應用程式物件。  
  
 **錯誤 ID：**BC30563  
  
### 更正這個錯誤  
  
-   完整限定名稱。
