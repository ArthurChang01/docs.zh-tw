# <a name="name-is-ambiguous-in-the-application-objects-list"></a>'\<名稱 >' 模稜兩可的應用程式物件在\<清單 >'
您提供的名稱模稜兩可，因此與另一個名稱發生衝突。 Visual Basic 編譯器並沒有任何衝突解決規則中;您必須自行區分名稱。 錯誤訊息中提供的清單詳細說明名稱模稜兩可的應用程式物件。  
  
 **錯誤 ID:** BC30563  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 完整限定名稱。
