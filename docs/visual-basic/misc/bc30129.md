# <a name="assembly-attribute-attributename-is-not-valid-error"></a>組件屬性 '\<屬性名稱 >' 無效：\<錯誤 >
Common Language Runtime 不接受組件上的屬性。

**錯誤 ID:** BC30129

## <a name="to-correct-this-error"></a>更正這個錯誤

1. 請從宣告中移除屬性，或將它取代為對內容有效的屬性。

2. 請檢查引用的錯誤訊息，並採取適當的動作。
