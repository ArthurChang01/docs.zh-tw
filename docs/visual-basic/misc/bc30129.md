# <a name="assembly-attribute-39ltattributenamegt39-is-not-valid-lterrorgt"></a>組件屬性&#39; &lt;attributename&gt; &#39;無效：&lt;錯誤&gt;
Common Language Runtime 不接受組件上的屬性。

**錯誤 ID︰** BC30129

## <a name="to-correct-this-error"></a>更正這個錯誤

1. 請從宣告中移除屬性，或將它取代為對內容有效的屬性。

2. 請檢查引用的錯誤訊息，並採取適當的動作。
