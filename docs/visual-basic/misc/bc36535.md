# <a name="instance-members-and-39me39-cannot-be-used-in-a-query-expression"></a><span data-ttu-id="33476-102">執行個體成員和 &#39;我 &#39;不能在查詢運算式</span><span class="sxs-lookup"><span data-stu-id="33476-102">Instance members and &#39;Me&#39; cannot be used in a query expression</span></span>
<span data-ttu-id="33476-103">`Structure` 中的 LINQ 查詢包含 `Me` 或結構執行個體成員的參考。</span><span class="sxs-lookup"><span data-stu-id="33476-103">A LINQ query in a `Structure` includes a reference to `Me` or to an instance member of the structure.</span></span> <span data-ttu-id="33476-104">`Me` 內的查詢運算式不允許 `Structure`或執行個體成員的參考。</span><span class="sxs-lookup"><span data-stu-id="33476-104">References to `Me` or instance members are not allowed in query expressions within a `Structure`.</span></span>  
  
 <span data-ttu-id="33476-105">**錯誤 ID：** BC36535</span><span class="sxs-lookup"><span data-stu-id="33476-105">**Error ID:** BC36535</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="33476-106">更正這個錯誤</span><span class="sxs-lookup"><span data-stu-id="33476-106">To correct this error</span></span>  
  
1.  <span data-ttu-id="33476-107">建立執行個體成員複本或 `Me` 參考傳回值的複本，並在查詢運算式中使用此複本，如下例所示。</span><span class="sxs-lookup"><span data-stu-id="33476-107">Create a copy of the instance member or the value returned by the reference to `Me` and use the copy in the query expression, as shown in the following example.</span></span>  
  
```vb  
Structure SampleStructure  
    Public SearchValue As Integer  
  
    Public Sub SetSearchValue(ByVal number As Integer)  
      SearchValue = number  
    End Sub  
  
    Public Sub GetData()  
        Dim sv = SearchValue  
        Dim SampleData = New Integer() {1, 2, 3, 4}  
        Dim query = From number In SampleData _  
                    Where number < sv  
    End Sub  
End Structure  
```
