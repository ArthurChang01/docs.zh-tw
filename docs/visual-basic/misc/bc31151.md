# <a name="element-is-missing-an-end-tag"></a>項目遺漏結束標記
XML 項目常值遺漏 XML 結束標記或空白標記識別項 '/' (例如， `<tagName />`)。  
  
 **錯誤 ID︰** BC31151  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   加入 XML 標記的結束標記，或將空白標記識別項 '/' 加入 XML 標記。
