# <a name="39inherits39-can-appear-only-once-within-a-39class39-statement-and-can-only-specify-one-class"></a>&#39;繼承&#39;內只能出現一次&#39;類別&#39;陳述式，並且只能指定一個類別
多個 `Inherits` 陳述式出現在相同類別中，或 `Inherits` 陳述式指定多個類別。 類別只能繼承自一個基底類別。  
  
 **錯誤 ID︰** BC30121  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請移除任何額外的 `Inherits` 陳述式，並確定剩下的 `Inherits` 陳述式只指定一個基底類別。
