# <a name="type-in-a-module-cannot-be-declared-specifier"></a>模組中的類型不可以宣告為 '\<規範 >'
已在模組內宣告具有無效修飾詞的類型。  
  
 **錯誤 ID:** BC30735  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請不要對模組內宣告的類型使用 `Protected` 或 `Protected Friend` 修飾詞。
