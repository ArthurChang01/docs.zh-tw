# <a name="variables-in-modules-cannot-be-declared-39ltspecifiergt39"></a>在模組中的變數不可以宣告 &#39;&lt;規範&gt;&#39;
您已在 `MustInherit`陳述式的變數上使用規範 (例如 `Module` )。 模組絕不會執行個體化、不支援繼承，且不能實作介面。  
  
 **錯誤 ID︰** BC30593  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請移除規範。
