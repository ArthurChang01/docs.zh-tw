# <a name="39ltspecifiergt39-is-not-valid-on-an-enum-declaration"></a>&#39;&lt;規範&gt;&#39; 在 Enum 宣告中無效

 **錯誤 ID：** BC30396  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
1.  請移除規範。
