# <a name="namespace-or-type-name-has-already-been-imported"></a>命名空間或類型 '\<名稱 >' 已經匯入
嘗試多次匯入命名空間。  
  
 **錯誤 ID:** BC31051  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 移除多餘的 `Imports` 陳述式。
