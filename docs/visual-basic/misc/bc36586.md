# <a name="argument-not-specified-for-parameter-ltparameternamegt-of-extension-method-ltmethodnamegt-defined-in-lttypenamegt"></a>未指定參數的引數 '&lt;parametername&gt;'的擴充方法'&lt;methodname&gt;'中所定義的'&lt;typename&gt;'
尚未提供擴充方法中必要參數的引數。  
  
 **錯誤 ID:** BC36586  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請為指定的參數提供引數。
