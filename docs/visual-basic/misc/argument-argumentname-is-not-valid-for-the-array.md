# <a name="argument-argumentname-is-not-valid-for-the-array"></a>引數 '\<引數名稱 >' 不是有效的陣列
陣列包含無效的引數。  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請檢查運算式中的引數拼法。 拼錯的變數名稱可能會隱含地建立初始化為零的數值變數。
