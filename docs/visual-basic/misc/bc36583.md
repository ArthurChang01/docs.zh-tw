# <a name="parameter-39ltparameternamegt39-in-extension-method-39ltmethodnamegt39-defined-in-39lttypenamegt39-already-has-a-matching-omitted-argument"></a>參數&#39; &lt;parametername&gt; &#39;中擴充方法&#39; &lt;methodname&gt; &#39;中定義&#39; &lt;typename&gt; &#39;已經有相符的省略引數
擴充方法的程序呼叫依位置省略一個引數，然後依名稱提供引數。 例如，擴充方法 `ABC` 的下列呼叫會先忽略參數 `Y`的引數，然後再依名稱提供。  
  
```  
<Extension()> _  
Public Sub ABC(ByVal X As Integer, Optional ByVal Y As Byte = 0, _  
               Optional ByVal Z As Byte = 0)  
End Sub  
' . . .  
' Calling extension method ABC.  
Dim number As Integer  
' Not valid.  
' number.ABC(, 4, Y:=5)  
```  
  
 **錯誤 ID：** BC36583  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   依位置提供引數，或移除省略引數的逗號。
