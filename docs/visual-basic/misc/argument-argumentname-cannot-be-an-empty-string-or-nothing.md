# <a name="argument-ltargumentnamegt-cannot-be-an-empty-string-or-nothing"></a>引數&lt;引數名稱&gt;不能是空字串或 Nothing
此引數需要空字串 ("") 或 `Nothing`以外的值。 這可能是不正確計算值的結果。  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請為引數提供有效的值。
