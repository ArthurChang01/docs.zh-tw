# <a name="new-is-not-valid-in-this-context"></a>'New' 在此內容中無效
需要類型名稱的位置出現 `New` 關鍵字。 這個內容呼叫無法與 `New`合併的實值類型。  
  
 **錯誤 ID:** BC30200  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請移除 `New` 關鍵字。
