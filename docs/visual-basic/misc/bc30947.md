# <a name="suspend-request-is-pending"></a>暫止要求擱置中
在 Visual Studio 偵錯工具中，運算式指定程序呼叫，但會有暫止執行緒的要求。  
  
 偵錯工具不會嘗試在非使用中執行緒上呼叫程序。  
  
 **錯誤 ID:** BC30947  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 可能的話，請判斷執行緒暫止要求的來源，以防止它再次發生。  
  
- 終止並重新啟動偵錯。
