# <a name="error-embedding-win32-manifest-option-win32manifest-conflicts-with-nowin32manifest"></a>嵌入 Win32 資訊清單時發生錯誤：選項/win32manifest 與/nowin32manifest 衝突
已傳遞 `/win32manifest` 編譯器選項和 `/nowin32manifest` 編譯器選項給 Visual Basic 編譯器。 這些選項中只有一個可以傳遞給 Visual Basic 編譯器。  
  
 **錯誤 ID:** BC2033  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
1. 請移除 `/win32manifest` 編譯器選項或 `/nowin32manifest` 編譯器選項。
