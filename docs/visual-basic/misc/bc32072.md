# <a name="cannot-implement-interface-interfacename1-because-its-implementation-could-conflict-with-the-implementation-of-another-implemented-interface-interfacename2-for-some-type-arguments"></a><span data-ttu-id="c92a8-102">無法執行介面 '\<介面名稱 1> > ', 因為它的執行可能與某些類型引數的另\<一個實作為介面 ' 介面名稱 2> > ' 的執行發生衝突</span><span class="sxs-lookup"><span data-stu-id="c92a8-102">Cannot implement interface '\<interfacename1>' because its implementation could conflict with the implementation of another implemented interface '\<interfacename2>' for some type arguments</span></span>

<span data-ttu-id="c92a8-103">類別宣告包含指定兩個或多個介面的 `Implements` 陳述式，但至少有一個介面是泛型，且有兩個實作因為特定的類型引數值而發生衝突。</span><span class="sxs-lookup"><span data-stu-id="c92a8-103">A class declaration includes an `Implements` statement that specifies two or more interfaces, but at least one of the interfaces is generic and two of the implementations could conflict for certain values of type arguments.</span></span>

<span data-ttu-id="c92a8-104">下列陳述式可能會產生此錯誤。</span><span class="sxs-lookup"><span data-stu-id="c92a8-104">The following statements can generate this error.</span></span>

```vb
Public Interface iFace1
    Sub testSub(ByVal arg As String)
End Interface
Public Interface iFace2(Of t)
    Sub testSub(ByVal arg As t)
End Interface
Public Class testClass
    Implements iFace1, iFace2(Of String)
End Class
```

<span data-ttu-id="c92a8-105">因為 `iFace2` 使用 `String`建構，所以 `testClass` 必須實作具相同特徵標記的兩個版本 `testSub` 。</span><span class="sxs-lookup"><span data-stu-id="c92a8-105">Because `iFace2` is constructed using `String`, `testClass` must implement two versions of `testSub` with identical signatures.</span></span> <span data-ttu-id="c92a8-106">這樣會造成無法確定存取哪一個版本的情況。</span><span class="sxs-lookup"><span data-stu-id="c92a8-106">Doing so would produce an ambiguity about which version to access.</span></span>

<span data-ttu-id="c92a8-107">**錯誤識別碼:** BC32072</span><span class="sxs-lookup"><span data-stu-id="c92a8-107">**Error ID:** BC32072</span></span>

## <a name="to-correct-this-error"></a><span data-ttu-id="c92a8-108">更正這個錯誤</span><span class="sxs-lookup"><span data-stu-id="c92a8-108">To correct this error</span></span>

- <span data-ttu-id="c92a8-109">變更提供給泛型介面的類型引數，使其不發生衝突。</span><span class="sxs-lookup"><span data-stu-id="c92a8-109">Change the type argument supplied to the generic interface so that there is no conflict.</span></span>

  <span data-ttu-id="c92a8-110">\-或-</span><span class="sxs-lookup"><span data-stu-id="c92a8-110">\-or-</span></span>
