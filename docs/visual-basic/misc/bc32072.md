# <a name="cannot-implement-interface-39ltinterfacename1gt39-because-its-implementation-could-conflict-with-the-implementation-of-another-implemented-interface-39ltinterfacename2gt39-for-some-type-arguments"></a><span data-ttu-id="e09fd-102">無法實作介面 &#39;&lt;介面名稱 1>.<&gt;&#39;，因為它的實作可能與另一個實作的介面 &#39; 實作衝突&lt;介面名稱 2>&gt;&#39; 某些類型引數</span><span class="sxs-lookup"><span data-stu-id="e09fd-102">Cannot implement interface &#39;&lt;interfacename1&gt;&#39; because its implementation could conflict with the implementation of another implemented interface &#39;&lt;interfacename2&gt;&#39; for some type arguments</span></span>
<span data-ttu-id="e09fd-103">類別宣告包含指定兩個或多個介面的 `Implements` 陳述式，但至少有一個介面是泛型，且有兩個實作因為特定的類型引數值而發生衝突。</span><span class="sxs-lookup"><span data-stu-id="e09fd-103">A class declaration includes an `Implements` statement that specifies two or more interfaces, but at least one of the interfaces is generic and two of the implementations could conflict for certain values of type arguments.</span></span>  
  
 <span data-ttu-id="e09fd-104">下列陳述式可能會產生此錯誤。</span><span class="sxs-lookup"><span data-stu-id="e09fd-104">The following statements can generate this error.</span></span>  
  
```  
Public Interface iFace1  
    Sub testSub(ByVal arg As String)  
End Interface  
Public Interface iFace2(Of t)  
    Sub testSub(ByVal arg As t)  
End Interface  
Public Class testClass  
    Implements iFace1, iFace2(Of String)  
End Class  
```  
  
 <span data-ttu-id="e09fd-105">因為 `iFace2` 使用 `String`建構，所以 `testClass` 必須實作具相同特徵標記的兩個版本 `testSub` 。</span><span class="sxs-lookup"><span data-stu-id="e09fd-105">Because `iFace2` is constructed using `String`, `testClass` must implement two versions of `testSub` with identical signatures.</span></span> <span data-ttu-id="e09fd-106">這樣會造成無法確定存取哪一個版本的情況。</span><span class="sxs-lookup"><span data-stu-id="e09fd-106">Doing so would produce an ambiguity about which version to access.</span></span>  
  
 <span data-ttu-id="e09fd-107">**錯誤 ID：** BC32072</span><span class="sxs-lookup"><span data-stu-id="e09fd-107">**Error ID:** BC32072</span></span>
