# <a name="static-local-variable-ltvariablenamegt-is-already-declared"></a>靜態區域變數 '&lt;variablename&gt;' 已宣告
已經宣告具有相同名稱的另一個靜態區域變數。  
  
 **錯誤 ID:** BC31401  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
1.  移除多餘的靜態區域宣告。  
  
2.  為每個靜態區域變數指定唯一的名稱。
