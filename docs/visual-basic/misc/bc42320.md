# <a name="unable-to-include-xml-fragment-ltelementpathgt-of-file-ltcommentfilegt"></a>無法包含 XML 片段 '&lt;elementpath&gt;'的檔案'&lt;註解檔案&gt;'
XML 註解中的 `<include>` 標記包含不存在之項目的 XPath 運算式，或檔案包含格式不正確的 XML。  
  
 **錯誤 ID:** BC42320  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
1.  請確認檔案中的 XML 有效。  
  
2.  請確認 XML 片段存在於檔案中。
