# <a name="39finally39-can-only-appear-once-in-a-39try39-statement"></a>&#39;最後 &#39;只能出現一次在 &#39;再試一次 &#39;陳述式
`Finally` 用來完成 `Try...Catch...Finally` 區塊，因此它只能出現在區塊結尾一次。  
  
 **錯誤 ID︰** BC30381  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
1.  請找到並移除多餘的 `Finally`。
