# <a name="lttypenamegt-cannot-be-used-as-an-attribute-because-it-is-declared-mustinherit"></a>'&lt;typename&gt;' 無法用做為屬性中，因為它宣告為 'MustInherit'
自訂屬性類別不能宣告為 `MustInherit`。  
  
 **錯誤 ID:** BC31506  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請從自訂屬性移除 `MustInherit` 修飾詞。
