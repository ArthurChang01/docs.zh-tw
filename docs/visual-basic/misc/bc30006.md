# <a name="reference-required-to-module-modulename-containing-the-definition-for-event-eventname"></a>需要模組參考 '\<modulename >' 包含事件定義'\<事件名稱 >'
需要模組參考 '<`modulename`>' 包含事件定義' <`eventname`>'。 請在專案中加入一個參考。  
  
 此事件是在專案中未直接參考的模組中所定義。 Visual Basic 編譯器需要參考，以避免模稜兩可，以防多個模組中定義的事件時。  
  
 **錯誤 ID:** BC30006  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請在您的專案參考中包含未參考之模組的名稱。
