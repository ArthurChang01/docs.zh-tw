# <a name="reference-required-to-module-39ltmodulenamegt39-containing-the-definition-for-event-39lteventnamegt39"></a>需要模組參考&#39; &lt;modulename&gt; &#39;包含事件定義&#39;&lt;事件名稱&gt;&#39;
需要模組參考 '<`modulename`>' 包含事件定義' <`eventname`>'。 請在專案中加入一個參考。  
  
 此事件是在專案中未直接參考的模組中所定義。 Visual Basic 編譯器需要參考，以避免發生模稜兩可的情況在多個模組中定義事件。  
  
 **錯誤 ID︰** BC30006  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請在您的專案參考中包含未參考之模組的名稱。
