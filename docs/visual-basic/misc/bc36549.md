# <a name="anonymous-type-property-39ltpropertynamegt39-cannot-be-used-in-the-definition-of-a-lambda-expression-within-the-same-initialization-list"></a><span data-ttu-id="66e75-102">匿名類型屬性 &#39;&lt;propertyname&gt;&#39; 不能在相同的初始設定清單中 lambda 運算式的定義</span><span class="sxs-lookup"><span data-stu-id="66e75-102">Anonymous type property &#39;&lt;propertyname&gt;&#39; cannot be used in the definition of a lambda expression within the same initialization list</span></span>
<span data-ttu-id="66e75-103">匿名類型之初始設定清單中所定義的屬性不能是相同清單中 Lambda 運算式定義的一部分。</span><span class="sxs-lookup"><span data-stu-id="66e75-103">Properties defined in the initialization list of an anonymous type cannot be part of a lambda expression definition in the same list.</span></span> <span data-ttu-id="66e75-104">例如，在下列程式碼中，屬性 `Num` 不能包括在 `LambdaFun`的定義中。</span><span class="sxs-lookup"><span data-stu-id="66e75-104">For example, in the following code, property `Num` cannot be included in the definition of `LambdaFun`.</span></span>  
  
```vb  
' Not valid.  
'Dim anon = New With {.Num = 4, .LambdaFun = Function() .Num > 0}  
```  
  
 <span data-ttu-id="66e75-105">**錯誤 ID︰** BC36549</span><span class="sxs-lookup"><span data-stu-id="66e75-105">**Error ID:** BC36549</span></span>  

## <a name="to-correct-this-error"></a><span data-ttu-id="66e75-106">更正這個錯誤</span><span class="sxs-lookup"><span data-stu-id="66e75-106">To correct this error</span></span>  
  
1.  <span data-ttu-id="66e75-107">請考慮將匿名類型分割成兩個部分：</span><span class="sxs-lookup"><span data-stu-id="66e75-107">Consider splitting the anonymous type into two parts:</span></span>  
  
```vb  
Dim anon1 = New With {.Num = 4}  
Dim anon2 = New With {.LambdaFun = Function() anon1.Num > 0}  
' - or -  
Dim anon3 = New With {.lambdaFun = Function(n As Integer) n > 0}  
Console.WriteLine((anon2.LambdaFun)())  
Console.WriteLine(anon3.lambdaFun(anon1.Num))  
anon1.Num = -5  
Console.WriteLine((anon2.LambdaFun)())  
Console.WriteLine(anon3.lambdaFun(anon1.Num))  
```  
  
<span data-ttu-id="66e75-108">請注意，如果您將 `anon1.Num` 宣告為 `Key` 屬性，就無法變更其值。</span><span class="sxs-lookup"><span data-stu-id="66e75-108">Note that if you declare `anon1.Num` as a `Key` property, its value cannot be changed.</span></span>  
  
2.  <span data-ttu-id="66e75-109">替代方法是使用一般函式陳述式來存取匿名類型屬性：</span><span class="sxs-lookup"><span data-stu-id="66e75-109">An alternative is to use a regular function statement to access the anonymous type property:</span></span>  
  
```vb  
Function testNum(ByVal n As Integer) As Boolean  
    Return n > 0  
End Function  
Console.WriteLine(testNum(anon1.Num))  
```  
  
3.  <span data-ttu-id="66e75-110">同樣地，您可以使用在匿名類型外部定義的 Lambda 函式：</span><span class="sxs-lookup"><span data-stu-id="66e75-110">Similarly, you can use a lambda function that is defined outside the anonymous type:</span></span>  
  
```vb  
Dim lambdaFun1 = Function() anon1.Num > 0  
Dim lambdaFun2 = Function(n As Integer) n > 0  
```
