# <a name="value-of-type-39lttype1gt39-cannot-be-converted-to-39lttype2gt39-because-the-array-types-have-different-numbers-of-dimensions"></a>型別的值&#39; &lt;type1&gt; &#39;無法轉換成&#39; &lt;type2&gt; &#39;因為陣列類型的維度數目不同
您嘗試將某個陣列轉換成另一個具有不同維度數目的陣列。  
  
 **錯誤 ID：** BC30414  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請變更其中一個或兩個陣列，讓維度的數目相符。
