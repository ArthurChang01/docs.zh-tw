# <a name="property-39ltpropertynamegt39-is-39writeonly39"></a>屬性&#39; &lt;propertyname&gt; &#39;是&#39;WriteOnly&#39;
您已嘗試從宣告為 `WriteOnly`的屬性擷取值。  
  
 **錯誤 ID︰** BC30524  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請移除屬性宣告的 `WriteOnly` 指定名稱。
