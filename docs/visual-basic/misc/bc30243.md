# <a name="39ltmodifiergt39-is-not-valid-on-an-event-declaration"></a>&#39;&lt;修飾詞&gt;&#39;在事件宣告中無效
`Event` 陳述式包含無效的關鍵字，例如 `ReadOnly`。  
  
 **錯誤 ID︰** BC30243  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
1.  請從 `Event` 陳述式中移除無效的關鍵字。
