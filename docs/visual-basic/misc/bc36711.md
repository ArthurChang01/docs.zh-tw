# <a name="argument-matching-parameter-parametername-narrows-to-typename"></a>對應參數的引數 '\<參數名稱 >' 縮減到'\<類型名稱 >'
您已呼叫多載方法，但編譯器找不到可以呼叫且不用縮小轉換的方法。 *「縮小轉換」* (narrowing conversion) 會將值變更為資料類型，而可能無法保留一些可能的值。  
  
 **錯誤 ID:** BC36711
