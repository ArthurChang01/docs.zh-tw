# <a name="argument-matching-parameter-39ltparameternamegt39-narrows-to-39lttypenamegt39"></a>引數比對參數 &#39;&lt;parametername&gt;&#39; 若要以縮小 &#39;&lt;typename&gt;&#39;
您已呼叫多載方法，但編譯器找不到可以呼叫且不用縮小轉換的方法。 *「縮小轉換」* (narrowing conversion) 會將值變更為資料類型，而可能無法保留一些可能的值。  
  
 **錯誤 ID︰** BC36711
