# <a name="argument-matching-parameter-ltparameternamegt-narrows-to-lttypenamegt"></a>對應參數的引數 '&lt;parametername&gt;'縮減到'&lt;typename&gt;'
您已呼叫多載方法，但編譯器找不到可以呼叫且不用縮小轉換的方法。 *「縮小轉換」* (narrowing conversion) 會將值變更為資料類型，而可能無法保留一些可能的值。  
  
 **錯誤 ID:** BC36711
