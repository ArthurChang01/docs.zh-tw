# <a name="membername-cannot-be-declared-shadows-outside-of-a-class-structure-or-interface"></a>'\<成員名稱 >' 無法宣告為 'Shadows' 以外的類別、 結構或介面
`Shadows` 關鍵字出現在命名空間、模組或檔案層級的宣告中。 遮蔽只在可繼承自基底項目的程式設計項目內有意義。  
  
 **錯誤 ID:** BC32200  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請移除 `Shadows` 關鍵字，或移動類別、結構或介面內的宣告。
