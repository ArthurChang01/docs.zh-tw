# <a name="property-or-field-ltnamegt-does-not-have-a-valid-attribute-type"></a>屬性或欄位 '&lt;名稱&gt;' 沒有有效的屬性類型
您也可以使用屬性建構函式不接受有效的屬性類型，即使屬性可有其他適當的建構函式。  
  
 **錯誤 ID:** BC30659  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
1.  請檢查這個屬性的使用方式。
