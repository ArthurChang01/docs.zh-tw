# <a name="property-or-field-39ltnamegt39-does-not-have-a-valid-attribute-type"></a>屬性或欄位 &#39;&lt;名稱&gt;&#39; 不具備有效的屬性類型
您使用的屬性建構函式可能不接受有效的屬性類型，即使屬性可有其他適當的建構函式。  
  
 **錯誤 ID：** BC30659  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
1.  請檢查這個屬性的使用方式。
