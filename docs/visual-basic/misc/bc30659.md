# <a name="property-or-field-name-does-not-have-a-valid-attribute-type"></a>屬性或欄位 '\<name > ' 沒有有效的屬性類型
您可能使用不接受有效屬性類型的屬性函式, 即使該屬性可能有其他適當的函式也一樣。  
  
 **錯誤識別碼:** BC30659  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
1. 請檢查這個屬性的使用方式。
