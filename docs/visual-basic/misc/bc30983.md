# <a name="type-of-39ltvariablenamegt39-is-ambiguous-because-the-loop-bounds-and-the-step-variable-do-not-widen-to-the-same-type"></a><span data-ttu-id="db14f-102">類型 &#39;&lt;variablename&gt;&#39; 模稜兩可，因為迴圈繫結和 step 變數不會擴展為相同的型別</span><span class="sxs-lookup"><span data-stu-id="db14f-102">Type of &#39;&lt;variablename&gt;&#39; is ambiguous because the loop bounds and the step variable do not widen to the same type</span></span>
<span data-ttu-id="db14f-103">您的程式碼包含 `For...Next` 迴圈，其中，編譯器無法推斷迴圈控制變數的資料類型，因為下列條件為 true：</span><span class="sxs-lookup"><span data-stu-id="db14f-103">Your code contains a `For...Next` loop in which the compiler cannot infer a data type for the loop control variable because the following conditions is true:</span></span>  
  
-   <span data-ttu-id="db14f-104">未使用 `As` 子句指定迴圈控制變數的資料類型。</span><span class="sxs-lookup"><span data-stu-id="db14f-104">The data type of the loop control variable is not specified with an `As` clause.</span></span>  
  
-   <span data-ttu-id="db14f-105">迴圈繫結和 step 變數包含至少兩種資料類型。</span><span class="sxs-lookup"><span data-stu-id="db14f-105">The loop bounds and step variable contain at least two data types.</span></span>  
  
-   <span data-ttu-id="db14f-106">資料類型之間可能存在多種可能轉換。</span><span class="sxs-lookup"><span data-stu-id="db14f-106">More than one possible conversion exists between the data types.</span></span>  
  
-   <span data-ttu-id="db14f-107">候選項目之間沒有最佳類型，如此，迴圈控制變數的類型選擇模稜兩可。</span><span class="sxs-lookup"><span data-stu-id="db14f-107">There is no best type among the candidates, so that the choice of type for the loop control variable is ambiguous.</span></span>  
  
 <span data-ttu-id="db14f-108">例如，下列迴圈具有一個類型為 `Integer` 的迴圈繫結以及一個類型為 `UInteger`的迴圈繫結：</span><span class="sxs-lookup"><span data-stu-id="db14f-108">For example, the following loop has one loop bound of type `Integer` and one loop bound of type `UInteger`:</span></span>  
  
```vb  
Dim m As Integer = 1  
Dim n As UInteger = 10  
' Not valid.  
' For i = m To n  
    ' Loop processing.  
' Next  
```  
  
 <span data-ttu-id="db14f-109">`Integer` 與 `UInteger`之間存在可能轉換，而 `UInteger` 與 `Integer`之間存在可能轉換，但兩者都是縮小轉換，因此都不是最佳選擇。</span><span class="sxs-lookup"><span data-stu-id="db14f-109">There is a possible conversion between `Integer` and `UInteger`, and a possible conversion between `UInteger` and `Integer`, but both are narrowing conversions so neither is the best choice.</span></span>  
  
 <span data-ttu-id="db14f-110">在下一個範例中，加入類型為 `Double` 的第三個變數。</span><span class="sxs-lookup"><span data-stu-id="db14f-110">In the next example, a third variable of type `Double` is added.</span></span> <span data-ttu-id="db14f-111">`Integer` 和 `UInteger` 都具有 `Double`的標準擴展轉換，這會將 `Double` 的轉換設為最佳選擇。</span><span class="sxs-lookup"><span data-stu-id="db14f-111">Both `Integer` and `UInteger` have standard widening conversions to `Double`, which makes conversion to `Double` the best choice.</span></span> <span data-ttu-id="db14f-112">類型推斷會將資料類型 `i` 指派給迴圈控制變數 `Double`。</span><span class="sxs-lookup"><span data-stu-id="db14f-112">Type inference assigns to loop control variable `i` the data type `Double`.</span></span>  
  
```vb  
Dim stepVar As Double = 1  
' Valid.  
For i = m To n Step stepVar  
    ' Loop processing.  
Next  
```  
  
 <span data-ttu-id="db14f-113">**錯誤 ID︰** BC30983</span><span class="sxs-lookup"><span data-stu-id="db14f-113">**Error ID:** BC30983</span></span>
