# <a name="the--character-cannot-be-used-here"></a>'?' 字元不能用於此
'?' 字元可以用來指定可為 Null 的實值類型或結構。 它在其他情況下的使用受到限制。 例如，下列程式碼會造成這個例外狀況。  
  
```  
' Not valid.  
' #Const found = True?  
```  
  
 **錯誤 ID:** BC36637  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 請從宣告中移除 '?' 字元。
