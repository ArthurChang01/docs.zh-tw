# <a name="specifier-is-duplicated"></a>規範重複
關鍵字在相同的宣告中出現多次。  
  
 **錯誤 ID︰** BC30178  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請移除宣告中重複出現的關鍵字。
