# <a name="ltnamegt-is-ambiguous-between-declarations-in-modules-ltmodulename1gt-and-ltmodulename2gt"></a>'&lt;名稱&gt;'是在模組中的宣告之間模稜兩可 」&lt;modulename1&gt;'和'&lt;模組名稱 2&gt&gt;'
您提供的名稱模稜兩可，因此與另一個名稱發生衝突。 Visual Basic 編譯器並沒有任何衝突解決規則中;您必須自行區分名稱。  
  
 **錯誤 ID:** BC30562  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   完整限定名稱。
