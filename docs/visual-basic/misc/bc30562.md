# <a name="39ltnamegt39-is-ambiguous-between-declarations-in-modules-39ltmodulename1gt39-and-39ltmodulename2gt39"></a>&#39;&lt;名稱&gt;&#39; 在模組中的宣告 &#39; 中模稜兩可&lt;modulename1&gt;&#39; 和 &#39;&lt;modulename2&gt;&#39;
您提供的名稱模稜兩可，因此與另一個名稱發生衝突。 [!INCLUDE[vbprvb](~/includes/vbprvb-md.md)] 編譯器並沒有任何衝突解決規則，您必須自行區分名稱。  
  
 **錯誤識別碼：** BC30562  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   完整限定名稱。
