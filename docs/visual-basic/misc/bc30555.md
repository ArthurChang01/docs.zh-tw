# <a name="default-member-membername-is-not-a-property"></a>預設成員 '\<成員名稱 >' 不是屬性
您已嘗試宣告成員 `Default`。 不過， `Default` 關鍵字用來識別預設屬性。  
  
 **錯誤 ID:** BC30555  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 請確定您使用 `Default`來識別屬性，否則請移除 `Default` 關鍵字。
