# <a name="argument-ltargumentnamegt-cannot-be-converted-to-type-lttypenamegt"></a>引數 '&lt;argumentname&gt;'無法轉換為類型'&lt;typename&gt;'
轉換會嘗試將引數變更成無效的類型。  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   將類型變更為有效的類型。
