# <a name="argument-argumentname-cannot-be-converted-to-type-typename"></a>引數 '\<引數名稱 >' 無法轉換為類型'\<類型名稱 >'
轉換會嘗試將引數變更成無效的類型。  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   將類型變更為有效的類型。
