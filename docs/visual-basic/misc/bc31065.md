# <a name="set-parameter-cannot-be-declared-specifier"></a>'Set' 參數不可以宣告為 '\<規範 >'
在 `Set` 屬性程序中，指定的無效修飾詞具有參數。  
  
 **錯誤 ID:** BC31065  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請移除無效的修飾詞 (例如 `ByRef`、 `ParamArray`和 `Optional`)。
