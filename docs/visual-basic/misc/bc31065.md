# <a name="39set39-parameter-cannot-be-declared-39ltspecifiergt39"></a>&#39;設定&#39;參數不可宣告&#39;&lt;規範&gt;&#39;
在 `Set` 屬性程序中，指定的無效修飾詞具有參數。  
  
 **錯誤 ID︰** BC31065  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請移除無效的修飾詞 (例如 `ByRef`、 `ParamArray`和 `Optional`)。
