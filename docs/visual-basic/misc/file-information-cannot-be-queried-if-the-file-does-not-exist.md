# <a name="file-information-cannot-be-queried-if-the-file-does-not-exist"></a>如果檔案不存在，就無法查詢檔案資訊
已嘗試查詢 <xref:System.IO.FileInfo> 物件，以取得不存在檔案的相關資訊。 這通常是因為提供檔案名稱時發生錯誤。  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 確認您提供的檔案名稱正確。
