# <a name="ltargumentnamegt-is-not-a-parameter-of-ltmethoddeclarationgt"></a>'&lt;argumentname&gt;'不是參數的'&lt;methoddeclaration&gt;'
程序呼叫可提供程序未定義的其他引數。  
  
 **錯誤 ID:** BC30272  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請從引數清單中移除多餘的引數。
