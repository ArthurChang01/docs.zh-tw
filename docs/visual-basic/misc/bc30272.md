# <a name="39ltargumentnamegt39-is-not-a-parameter-of-39ltmethoddeclarationgt39"></a>&#39;&lt;引數名稱&gt;&#39; 不是參數 &#39;&lt;方法宣告&gt;&#39;
程序呼叫可提供程序未定義的其他引數。  
  
 **錯誤 ID︰** BC30272  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請從引數清單中移除多餘的引數。
