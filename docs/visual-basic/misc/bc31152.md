# <a name="xml-namespace-prefix-prefix-is-reserved-for-use-by-xml-and-the-namespace-uri-cannot-be-changed"></a>XML 命名空間前置字元 'prefix' 已保留給 XML 使用，且不能變更命名空間 URI
保留的 XML 命名空間前置字元 (例如 `xmlns`) 已用於 XML 常值。  
  
 **錯誤 ID:** BC31152  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   將 XML 命名空間前置字元變更為非保留值。
