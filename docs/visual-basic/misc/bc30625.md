# <a name="39module39-statement-must-end-with-a-matching-39end-module39"></a>&#39;模組&#39;陳述式必須搭配相對應的結尾&#39;結束模組&#39;
模組區塊的結尾必須`End Module`陳述式。  
  
 **錯誤 ID：** BC30625  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   將 `End Module` 陳述式加入 `Module` 區塊的結尾。
