# <a name="resource-name-39ltresourcenamegt39-cannot-be-used-more-than-once"></a>資源名稱 &#39;&lt;resourcename&gt;&#39; 無法使用超過一次
指定的資源名稱在動態連結程式庫或可執行檔中重複使用。  
  
 **錯誤 ID︰** BC31502  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   為資源指定唯一的名稱。
