# <a name="arguments-are-not-valid"></a>引數無效
正在將無效的引數傳遞給函式或副程式。  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
1.  請檢查引數，判斷無效的原因。  
  
2.  如果透過計算來提供引數，請檢查計算所處理的值。
