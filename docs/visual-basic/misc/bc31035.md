# <a name="interface-39ltinterfacenamegt39-is-not-implemented-by-this-class"></a>介面 &#39;&lt;interfacename&gt;&#39; 不由此類別實作
這個類別或結構的成員嘗試實作的成員，是類別或結構不會實作之介面的成員。  
  
 **錯誤 ID：** BC31035  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   將 `Implements` 陳述式加入類別或結構開頭，使其實作適當的介面。  
  
-   從導致此錯誤的成員中，移除 `Implements` 關鍵字。
