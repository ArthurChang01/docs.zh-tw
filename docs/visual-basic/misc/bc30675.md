# <a name="39synclock39-statement-must-end-with-a-matching-39end-synclock39"></a>&#39;SyncLock&#39;陳述式必須搭配相對應的結束&#39;End SyncLock&#39;
`SyncLock` 區塊以 `SyncLock` 關鍵字開頭並以 `End SyncLock` 建構結尾。  
  
 **錯誤 ID︰** BC30675  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   確定 `SyncLock` 區塊以 `End SyncLock` 建構結尾。
