# <a name="conversion-from-39double39-to-39date39-requires-calling-the-39datefromoadate39"></a>從轉換 &#39; 雙 &#39;若要 &#39; 日期 &#39;需要呼叫 &#39; 日期。FromOADate &#39;
您嘗試轉換`Date`值設定為`Double`值，而不使用，無法執行<xref:System.DateTime.FromOADate%2A?displayProperty=nameWithType>方法。  
  
 **錯誤 ID：** BC30533  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請使用 <xref:System.DateTime.FromOADate%2A> 方法來轉換值。
