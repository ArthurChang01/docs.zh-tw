# <a name="cannot-infer-a-common-type-for-the-first-and-second-operands-of-the-binary-39if39-operator"></a><span data-ttu-id="49e27-102">如果無法推斷一般類型的第一個和第二個運算元的二進位檔 &#39; &#39;運算子</span><span class="sxs-lookup"><span data-stu-id="49e27-102">Cannot infer a common type for the first and second operands of the binary &#39;If&#39; operator</span></span>
<span data-ttu-id="49e27-103">無法推斷二元 'If' 運算子的第一個和第二個運算元之一般類型。</span><span class="sxs-lookup"><span data-stu-id="49e27-103">Cannot infer a common type for the first and second operands of the binary 'If' operator.</span></span> <span data-ttu-id="49e27-104">一個運算元必須要能夠擴展轉換成另一個運算元。</span><span class="sxs-lookup"><span data-stu-id="49e27-104">One must have a widening conversion to the other.</span></span>  
  
 <span data-ttu-id="49e27-105">二元 `If` 運算子要求其中一個引數與另一個引數之間必須進行擴展轉換。</span><span class="sxs-lookup"><span data-stu-id="49e27-105">The binary `If` operator requires that there be a widening conversion between one of the arguments and the other argument.</span></span> <span data-ttu-id="49e27-106">例如，由於 `Integer` 與 `String`之間沒有任一方向的擴展轉換，因此下列程式碼會造成這個錯誤。</span><span class="sxs-lookup"><span data-stu-id="49e27-106">For example, because there is not a widening conversion in either direction between `Integer` and `String`, the following code causes this error.</span></span>  
  
```vb  
Dim first? As Integer  
Dim second As String = "First is Nothing"  
'' Not valid.  
' Console.WriteLine(If(first, second))  
```  
  
 <span data-ttu-id="49e27-107">**錯誤 ID︰** BC33110</span><span class="sxs-lookup"><span data-stu-id="49e27-107">**Error ID:** BC33110</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="49e27-108">更正這個錯誤</span><span class="sxs-lookup"><span data-stu-id="49e27-108">To correct this error</span></span>  
  
-   <span data-ttu-id="49e27-109">如果可能的話，請在您的程式碼中提供其中一個運算元的明確轉換：</span><span class="sxs-lookup"><span data-stu-id="49e27-109">Provide an explicit conversion for one of the operands, if that is possible in your code:</span></span>  
  
    ```  
    Console.WriteLine(If(first, CInt(second)))   
    ```  
  
-   <span data-ttu-id="49e27-110">使用不同的條件式建構重寫程式碼。</span><span class="sxs-lookup"><span data-stu-id="49e27-110">Rewrite the code by using a different conditional construction.</span></span>  
  
    ```  
    If first IsNot Nothing Then  
        Console.WriteLine(first)  
    Else  
        Console.WriteLine(second)  
    End If  
    ```
