# <a name="nesting-of-parentheses-in-lambda-expression39s-parameter-list-exceeds-maximum-allowable-depth"></a>Lambda 運算式中括號內的巢狀結構&#39;s 參數清單超過允許的最大深度
Lambda 運算式的參數清單中的巢狀括弧不能超過 128 的深度。  
  
 **錯誤 ID︰** BC36664
