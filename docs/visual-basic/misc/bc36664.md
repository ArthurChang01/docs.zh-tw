# <a name="nesting-of-parentheses-in-lambda-expression39s-parameter-list-exceeds-maximum-allowable-depth"></a>S 參數清單中 lambda 運算式 &#39; 括號內的巢狀結構超過允許的最大深度
Lambda 運算式的參數清單中的巢狀括弧不能超過 128 的深度。  
  
 **錯誤 ID︰** BC36664
