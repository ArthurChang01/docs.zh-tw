# <a name="39let39-and-39set39-assignment-statements-are-no-longer-supported"></a>&#39;可讓 &#39;和 &#39;設定 &#39;不再支援指派陳述式
`Let` 和 `Set` 不可以用於指派中。  
  
 **錯誤 ID︰** BC30807  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   指派陳述式中省略 `Let` 和 `Set` ；例如：  
  
     `X = 2`
