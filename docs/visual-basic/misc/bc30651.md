# <a name="property-parameters-cannot-be-declared-byref"></a>\<屬性 > 參數不可宣告為 'ByRef'
必須使用 `ByVal` 修飾詞來宣告屬性的參數。  
  
 **錯誤 ID:** BC30651  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 請將屬性參數定義為 `ByVal`。
