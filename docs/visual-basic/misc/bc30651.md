# <a name="ltpropertygt-parameters-cannot-be-declared-39byref39"></a>&lt;屬性&gt;參數不可宣告為 &#39;ByRef &#39;
必須使用 `ByVal` 修飾詞來宣告屬性的參數。  
  
 **錯誤 ID︰** BC30651  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請將屬性參數定義為 `ByVal`。
