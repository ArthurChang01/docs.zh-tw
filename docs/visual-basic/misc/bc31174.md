# XML 處理指示名稱 &#39;name&#39; 無效
為 XML 處理指令提供的名稱無效。  
  
 **錯誤 ID︰**BC31174  
  
### 更正這個錯誤  
  
-   更正 XML 處理指令名稱中的任何錯誤，或移除名稱。
