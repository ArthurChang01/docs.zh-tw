# <a name="39ltspecifiergt39-is-not-valid-on-a-withevents-declaration"></a>&#39;&lt;規範&gt;&#39;在 WithEvents 宣告中無效
`Dim` 陳述式包含關鍵字 (例如 `ReadOnly`)，與 `WithEvents` 關鍵字搭配使用時無效。  
  
 **錯誤 ID︰** BC30234  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
1.  請從 `Dim` 陳述式中移除無效的關鍵字。
