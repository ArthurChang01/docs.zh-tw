# <a name="automation-object-does-not-have-a-default-value"></a>Automation 物件沒有預設值
Visual Basic 無法判斷指定之物件的預設成員。  
  
### <a name="to-correct-the-error"></a>更正這個錯誤  
  
-   請檢查物件的文件，並提供屬性或方法的明確指定。
