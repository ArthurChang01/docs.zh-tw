# <a name="39interface39-must-end-with-a-matching-39end-interface39"></a>&#39;介面&#39;必須搭配相對應的結束&#39;結束介面&#39;
`Interface` 陳述式沒有相對應的 `End Interface` 陳述式。 您必須使用 `End Interface` 陳述式來結束介面定義。  
  
 **錯誤 ID︰** BC30253  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
1.  請確認已正確地格式化 `Interface` 成員。  
  
2.  將 `End Interface` 陳述式加到介面定義結尾。
