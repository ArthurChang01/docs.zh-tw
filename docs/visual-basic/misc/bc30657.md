# <a name="39ltprocedurenamegt39-has-a-return-type-that-is-not-supported-or-parameter-types-that-are-not-supported"></a>&#39;&lt;程序名稱&gt;&#39; 不支援的傳回型別或不支援的參數類型
宣告的程序具有 [!INCLUDE[vsprvs](~/includes/vsprvs-md.md)]無法代表的參數或傳回類型。  
  
 **錯誤 ID︰** BC30657  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   將傳回類型或參數定義為支援的類型。  
  
-   開發可描述這個程序所使用資料的新類型。
