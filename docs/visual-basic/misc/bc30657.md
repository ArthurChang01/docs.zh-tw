# <a name="39ltprocedurenamegt39-has-a-return-type-that-is-not-supported-or-parameter-types-that-are-not-supported"></a>&#39;&lt;程序名稱&gt;&#39;傳回的型別不支援或不支援的參數類型
使用參數或無法由 Visual Studio 的傳回型別宣告程序。  
  
 **錯誤 ID︰** BC30657  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   將傳回類型或參數定義為支援的類型。  
  
-   開發可描述這個程序所使用資料的新類型。
