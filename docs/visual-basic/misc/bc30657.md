# <a name="39ltprocedurenamegt39-has-a-return-type-that-is-not-supported-or-parameter-types-that-are-not-supported"></a>'&lt;m e&gt;' 具有不支援的傳回型別或不支援的參數型別
程序已宣告參數或傳回型別不能表示的[!INCLUDE[vsprvs](../../csharp/includes/vsprvs_md.md)]。  
  
 **錯誤 ID︰** BC30657  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   將傳回類型或參數定義為支援的類型。  
  
-   開發可描述這個程序所使用資料的新類型。
