# <a name="39return39-statement-in-a-function-get-or-operator-must-return-a-value"></a>&#39;傳回&#39;函式、 Get 或 Operator 中的陳述式必須傳回值
`Return` 陳述式必須用來將值傳回給呼叫的程序。 您不能使用 `Return` 陳述式本身來控制程式流程。  
  
 **錯誤 ID：** BC30654  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
1.  指定函式或程序可以傳回的值。  
  
2.  使用 `End` 陳述式讓程式結束目前的程序。
