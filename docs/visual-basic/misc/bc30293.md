# <a name="lterrorgt-39ltstructurename1gt39-contains-39ltstructurename2gt39"></a>&lt;錯誤&gt;: &#39;&lt;structurename1&gt;&#39; 包含 &#39;&lt;structurename2&gt;&#39;
結構將其本身指定為它自己的其中一個成員時，就會發生循環結構定義。  
  
 **錯誤 ID︰** BC30293  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   變更結構成員的名稱。
