# <a name="lterrorgt-ltstructurename1gt-contains-ltstructurename2gt"></a>&lt;錯誤&gt;: '&lt;結構名稱 1&gt&gt;'contains'&lt;結構名稱 2&gt&gt;'
結構將其本身指定為它自己的其中一個成員時，就會發生循環結構定義。  
  
 **錯誤 ID:** BC30293  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   變更結構成員的名稱。
