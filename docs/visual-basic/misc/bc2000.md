# <a name="initialization-error"></a>初始化錯誤
初始化程式時發生錯誤。 這可能是取消仍在進行中之安裝的結果。  
  
 **錯誤 ID︰** BC2000  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   重新嘗試安裝。
