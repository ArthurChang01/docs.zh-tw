# &#39;&lt;名稱&gt;&#39; 已經宣告為此方法的參數
已經針對這個方法宣告指定的參數名稱。  
  
 **錯誤 ID︰**BC30734  
  
### 更正這個錯誤  
  
-   重新命名參數，使其變成唯一的。
