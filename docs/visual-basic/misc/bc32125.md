# <a name="cannot-implement-39ltinterfacename1gtltmembernamegt39-because-its-implementation-could-conflict-with-the-implementation-for-39ltinterfacename2gtltmembernamegt39-for-some-type-arguments"></a><span data-ttu-id="a3569-102">無法實作 &#39;&lt;介面名稱 1>.<&gt;。&lt;membername&gt;&#39;，因為它實作發生衝突的實作 &#39;&lt;介面名稱 2>&gt;。&lt;membername&gt;&#39; 某些類型引數</span><span class="sxs-lookup"><span data-stu-id="a3569-102">Cannot implement &#39;&lt;interfacename1&gt;.&lt;membername&gt;&#39; because its implementation could conflict with the implementation for &#39;&lt;interfacename2&gt;.&lt;membername&gt;&#39; for some type arguments</span></span>
<span data-ttu-id="a3569-103">類別實作多個泛型介面，其中一個繼承另一個，還有兩個介面成員的實作可能和類型引數的特定值發生衝突。</span><span class="sxs-lookup"><span data-stu-id="a3569-103">A class implements more than one generic interface, one of which inherits from another, and two implementations of an interface member could conflict for certain values of type arguments.</span></span>  
  
 <span data-ttu-id="a3569-104">下列陳述式可能會產生此錯誤。</span><span class="sxs-lookup"><span data-stu-id="a3569-104">The following statements can generate this error.</span></span>  
  
```  
Public Interface iFace1(Of t)  
    Sub testSub()  
End Interface  
Public Interface iFace2(Of u)  
    Inherits iFace1(Of u)  
End Interface  
Public Class testClass(Of y, z)  
    Implements iFace1(Of y), iFace2(Of z)  
    Public Sub testSuby() Implements iFace1(Of y).testSub  
    End Sub  
    Public Sub testSubz() Implements iFace1(Of z).testSub  
    End Sub  
End Class  
```  
  
 <span data-ttu-id="a3569-105">因為 `iFace2` 使用自己的類型參數 ( `iFace1` ) 繼承了`u`，所以，如果相同的類型引數傳遞到 `testClass` 和 `iFace1.testSub` ， `y` 就會實作具有相同特徵標記的兩個版本的 `z`。</span><span class="sxs-lookup"><span data-stu-id="a3569-105">Because `iFace2` inherits from `iFace1` using its own type parameter (`u`), `testClass` would implement two versions of `iFace1.testSub` with identical signatures if the same type argument were passed to `y` and `z`.</span></span> <span data-ttu-id="a3569-106">這樣會造成不知該存取哪一個版本的情況。</span><span class="sxs-lookup"><span data-stu-id="a3569-106">This would produce an ambiguity about which version to access.</span></span>  
  
 <span data-ttu-id="a3569-107">**錯誤識別碼：** BC32125</span><span class="sxs-lookup"><span data-stu-id="a3569-107">**Error ID:** BC32125</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="a3569-108">更正這個錯誤</span><span class="sxs-lookup"><span data-stu-id="a3569-108">To correct this error</span></span>  
  
-   <span data-ttu-id="a3569-109">請變更介面的繼承結構，讓 `iFace1` 不能和兩個不同的類型引數一起實作。</span><span class="sxs-lookup"><span data-stu-id="a3569-109">Change the inheritance structure of the interfaces so that `iFace1` could not be implemented with two different type arguments.</span></span>  
  
     <span data-ttu-id="a3569-110">-或-</span><span class="sxs-lookup"><span data-stu-id="a3569-110">-or-</span></span>  
  
-   <span data-ttu-id="a3569-111">從 `Implements` 陳述式移除其中一個造成實作衝突的介面。</span><span class="sxs-lookup"><span data-stu-id="a3569-111">Remove from the `Implements` statement one of the interfaces resulting in the implementation conflict.</span></span>
