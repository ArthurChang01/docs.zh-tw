# <a name="character-character-0xlthexadecimalgt-is-not-allowed-in-xml-literal"></a>Character '' (0x&lt;十六進位&gt;) 中不允許 XML 常值

 **錯誤 ID:** BC31176  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請移除不支援的字元。
