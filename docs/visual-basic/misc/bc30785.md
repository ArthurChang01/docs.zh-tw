# <a name="parameter-specifier-is-duplicated"></a>參數規範重複
參數規範重複套用至參數。  
  
 **錯誤 ID:** BC30785  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 移除重複的規範。
