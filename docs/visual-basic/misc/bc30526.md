# <a name="property-ltpropertynamegt-is-readonly"></a>屬性 '&lt;propertyname&gt;' 是 'ReadOnly'
您已嘗試指派值給宣告為 `ReadOnly`的屬性。  
  
 **錯誤 ID:** BC30526  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請移除屬性宣告的 `ReadOnly` 指定名稱。
