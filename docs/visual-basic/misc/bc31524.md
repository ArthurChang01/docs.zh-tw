# <a name="39systemruntimeinteropservicesdllimportattribute39-cannot-be-applied-to-a-39get39-or-39set39"></a>&#39;System.Runtime.InteropServices.DllImportAttribute&#39;無法套用至&#39;取得&#39;或&#39;設定&#39;
`DllImportAttribute` 屬性已套用至 `Get` 或 `Set` 屬性程序。  
  
 **錯誤 ID︰** BC31524  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
1.  請從 `DllImportAttribute` 和 `Get` 屬性程序中移除 `Set` 。
