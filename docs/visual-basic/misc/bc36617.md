# <a name="aggregate-function-name-cannot-be-used-with-a-type-character"></a>彙總函式名稱不能配合類型字元使用
彙總函式名稱的結尾已加入類型字元。  
  
 **錯誤 ID︰** BC36617  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
1.  從彙總函式名稱中移除類型字元。
