# <a name="39ltnamegt39-is-not-a-member-of-39ltclassnamegt39"></a>&#39;&lt;名稱&gt;&#39;不是成員的&#39;&lt;類別名稱&gt;&#39;
您所提供的成員不是類別的成員。  
  
 **錯誤 ID：** BC30456  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
1.  請檢查成員名稱，確保它是正確的。  
  
2.  使用類別的實際成員。
