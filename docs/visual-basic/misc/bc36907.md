# <a name="extension-method-methodname-defined-in-modulename-is-not-generic-or-has-no-free-type-parameters-and-so-cannot-have-type-arguments"></a><span data-ttu-id="4cab5-102">擴充方法 '\<方法名稱 >' 中定義'\<模組名稱 >' 不是泛型 （或沒有可用的型別參數），因此不可有類型引數</span><span class="sxs-lookup"><span data-stu-id="4cab5-102">Extension method '\<methodname>' defined in '\<modulename>' is not generic (or has no free type parameters) and so cannot have type arguments</span></span>
<span data-ttu-id="4cab5-103">在擴充方法的呼叫中已指定類型引數，而擴充方法沒有泛型參數或沒有未指定類型的泛型參數。</span><span class="sxs-lookup"><span data-stu-id="4cab5-103">A type argument has been specified in a call to an extension method that either has no generic parameters or has no generic parameters for which a type is not already specified.</span></span> <span data-ttu-id="4cab5-104">例如，下列程式碼會造成這個錯誤。</span><span class="sxs-lookup"><span data-stu-id="4cab5-104">For example, the following code causes this error.</span></span>  
  
```vb  
' The extension method is not generic.  
<Extension()> _  
Sub Example(ByVal str As String)  
    ' Body of the Sub.  
End Sub  
```  
  
```vb  
Dim str = "hi"  
'' The call to Example specifies a type argument.  
'' Not valid.  
'str.Example(Of String)()  
```
