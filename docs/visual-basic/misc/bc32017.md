# <a name="comma--or-a-valid-expression-continuation-expected"></a>必須是逗號、')' 或有效的運算式接續符號
清單，例如程序定義程序呼叫中的引數清單，無法正確地分隔或終止其成員。 可能的一個原因是兩個清單成員只以空格分開。  
  
 **錯誤 ID:** BC32017  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請使用逗號 (,) 來分隔清單成員，並使用右括弧 ()) 來終止清單。
