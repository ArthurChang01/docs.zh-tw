# <a name="access-denied-to-name"></a>若要拒絕存取 '\<名稱 >'
裝置、檔案或磁碟無法存取，因為您的電腦安全性權限不足。  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
1.  請確定您具有必要的權限。
