# <a name="assembly-assemblyname-cannot-be-created-because-its-path-is-longer-than-259-characters"></a>組件 '\<組件名稱 >' 無法建立，因為它的路徑長度超過 259 個字元
這個組件的路徑超過 259 個字元的長度上限。  
  
 **錯誤 ID:** BC31518  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 使用較短的名稱，或選取較淺的檔案階層，來建立較短的路徑。
