# <a name="comma-or--expected-type-argument-list"></a>必須要有逗號或 ')' (類型引數清單)
類型引數清單包含未預期的項目。 請使用逗號 (`,`) 來分隔這類清單中的類型引數，並使用右括弧 (`)`) 來結束清單。  
  
 **錯誤 ID:** BC32094  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 加入逗號並繼續類型引數清單，或者加入右括弧字元來結束清單。
