# <a name="39sub-new39-cannot-be-declared-39overrides39"></a>&#39;Sub New&#39;不可以宣告為&#39;會覆寫&#39;
建構函式表示它會覆寫繼承的建構函式。 不可覆寫建構函式。  
  
 **錯誤 ID︰** BC30283  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請從 `Overrides` 宣告中移除 `Sub` 關鍵字。
