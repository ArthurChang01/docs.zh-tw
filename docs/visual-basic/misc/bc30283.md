# <a name="sub-new-cannot-be-declared-overrides"></a>'Sub New' 不可宣告為 'Overrides'
建構函式表示它會覆寫繼承的建構函式。 不可覆寫建構函式。  
  
 **錯誤 ID:** BC30283  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 請從 `Overrides` 宣告中移除 `Sub` 關鍵字。
