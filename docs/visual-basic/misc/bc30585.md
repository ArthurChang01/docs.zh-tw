# <a name="event-39lteventnamegt39-cannot-be-handled-because-it-is-not-accessible-from-39ltnamegt39"></a>事件&#39; &lt;eventname&gt; &#39;無法處理，因為它不是從存取&#39;&lt;名稱&gt;&#39;
您嘗試處理無法存取的事件。 例如，如果共用 `Handles` 變數，也必須共用處理事件的方法。  
  
 **錯誤 ID︰** BC30585  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
1.  確定事件可供存取。
