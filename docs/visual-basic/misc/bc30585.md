# <a name="event-eventname-cannot-be-handled-because-it-is-not-accessible-from-name"></a>無法處理\<事件 ' 事件名稱 > ', 因為它無法從 '\<name > ' 存取
您嘗試處理無法存取的事件。 例如，如果共用 `Handles` 變數，也必須共用處理事件的方法。  
  
 **錯誤識別碼:** BC30585  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
1. 確定事件可供存取。
