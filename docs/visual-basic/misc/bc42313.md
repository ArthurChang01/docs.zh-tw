# <a name="xml-comment-tag-39returns39-is-not-permitted-on-a-39writeonly39-property"></a>XML 註解標記 &#39; 傳回 &#39;上不允許 &#39;WriteOnly &#39;屬性
'WriteOnly' 屬性上不可使用 XML 註解標記 'returns'。 將會忽略 XML 註解。  
  
 唯寫屬性宣告的 XML 註解不能包含\<傳回 > 標記。  
  
 **錯誤 ID︰** BC42313  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   移除不支援\<傳回 > 標記。
