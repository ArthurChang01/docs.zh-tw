# <a name="ltpropertynamegt-implicitly-defines-ltmembernamegt-which-conflicts-with-a-member-of-the-same-name-in-lttypegt-lttypenamegt"></a>'&lt;propertyname&gt;'隱含定義'&lt;membername&gt;' 中, 同名成員互相衝突&lt;型別&gt;'&lt;typename&gt;'
類型成員名稱與為屬性隱含建立的成員名稱相衝突。 屬性會隱含建立數個與已定義成員發生衝突的隱含變數。  
  
 **錯誤 ID:** BC31060  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請重新命名明確宣告的成員，以移除命名衝突。
