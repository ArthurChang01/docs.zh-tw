# <a name="paramarray-cannot-be-applied-to-the-first-parameter-of-an-extension-method"></a><span data-ttu-id="c81d1-102">'ParamArray' 不能套用至擴充方法的第一個參數</span><span class="sxs-lookup"><span data-stu-id="c81d1-102">'ParamArray' cannot be applied to the first parameter of an extension method</span></span>
<span data-ttu-id="c81d1-103">'ParamArray' 不能套用至擴充方法的第一個參數。</span><span class="sxs-lookup"><span data-stu-id="c81d1-103">'ParamArray' cannot be applied to the first parameter of an extension method.</span></span> <span data-ttu-id="c81d1-104">第一個參數會指定要擴充的類型。</span><span class="sxs-lookup"><span data-stu-id="c81d1-104">The first parameter specifies which type to extend.</span></span>  
  
 <span data-ttu-id="c81d1-105">擴充方法的第一個參數會指定方法可擴充的資料類型。</span><span class="sxs-lookup"><span data-stu-id="c81d1-105">The first parameter of an extension method specifies the data type that the method extends.</span></span> <span data-ttu-id="c81d1-106">因此，第一個參數是必要項目，不得為選用項目。</span><span class="sxs-lookup"><span data-stu-id="c81d1-106">Therefore, the first parameter is required and cannot be optional.</span></span> <span data-ttu-id="c81d1-107">因為參數陣列會自動為選用項目，所以它不能作為擴充方法的第一個引數。</span><span class="sxs-lookup"><span data-stu-id="c81d1-107">Because a parameter array is automatically optional, it is not valid as the first argument of an extension method.</span></span>  
  
> [!NOTE]
>  <span data-ttu-id="c81d1-108">執行方法時，叫用方法的擴充資料類型執行個體會變成方法之第一個參數的引數。</span><span class="sxs-lookup"><span data-stu-id="c81d1-108">When the method is executed, the instance of the extended data type that invokes the method becomes the argument for the first parameter of the method.</span></span> <span data-ttu-id="c81d1-109">例如， `greeting` 中的執行個體 `greeting.Print()` 是擴充方法 `str`中第一個參數的引數 ( `Public Sub Print (ByVal str As String)`)。</span><span class="sxs-lookup"><span data-stu-id="c81d1-109">For example, the instance `greeting` in `greeting.Print()` is the argument for the first parameter, `str`, in extension method `Public Sub Print (ByVal str As String)`.</span></span>  
  
 <span data-ttu-id="c81d1-110">**錯誤 ID:** BC36554</span><span class="sxs-lookup"><span data-stu-id="c81d1-110">**Error ID:** BC36554</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="c81d1-111">更正這個錯誤</span><span class="sxs-lookup"><span data-stu-id="c81d1-111">To correct this error</span></span>  
  
- <span data-ttu-id="c81d1-112">如果參數陣列未指定您想要擴充的資料類型，請加入新的第一個參數來指定這個類型。</span><span class="sxs-lookup"><span data-stu-id="c81d1-112">If the parameter array does not specify the data type you want to extend, add a new first parameter that specifies this type.</span></span>  
  
    ```  
    <Extension()>  
    Public Sub AddTo(ByRef str As String, ByVal ParamArray addOns() As String)  
    ' Concatenate the strings in addOns to str.  
    End Sub  
    ```  
  
- <span data-ttu-id="c81d1-113">如果參數陣列指定您想要擴充的資料類型，請考慮將它變更為需要引數的一般陣列，而非參數陣列。</span><span class="sxs-lookup"><span data-stu-id="c81d1-113">If the parameter array does specify the data type you want to extend, consider changing it to a regular array, requiring an argument, instead of a parameter array.</span></span> <span data-ttu-id="c81d1-114">一般陣列可以進行擴充。</span><span class="sxs-lookup"><span data-stu-id="c81d1-114">Regular arrays can be extended.</span></span>  
  
    ```  
    <Extension()>  
    Public Function Sum(ByVal ints() As Integer) As Integer  
        Dim total As Integer = 0  
        For Each i As Integer In ints  
            total = total + i  
        Next i  
        Return total  
    End Function  
    ```
