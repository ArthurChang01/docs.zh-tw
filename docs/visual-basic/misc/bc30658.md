# <a name="property-ltpropertynamegt-with-no-parameters-cannot-be-found"></a>屬性 '&lt;propertyname&gt;' 找不到不含任何參數
正在使用的屬性具有需要一個或多個參數的屬性。  
  
 **錯誤 ID:** BC30658  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
1.  請指定屬性的參數。
