# <a name="index-indexnumber-for-dimension-dimensionnumber-is-out-of-range"></a>索引 '\<索引編號 >' 的維度'\<維度編號 >' je mimo rozsah
其中一個索引超出定義的範圍。  
  
 **錯誤 ID:** BC30702  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請變更程序的邏輯，使其不超出陣列界限。  
  
-   重新定義陣列，以允許較大範圍的值。
