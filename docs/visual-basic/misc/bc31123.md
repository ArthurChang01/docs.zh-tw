# <a name="39end-event39-must-be-preceded-by-a-matching-39custom-event39"></a>&#39;End Event&#39;之前必須搭配相對應的&#39;自訂事件&#39;
`End Event` 陳述式沒有相對應的 `Custom Event` 陳述式。 `End Event` 之前必須搭配相對應的 `Custom Event` 陳述式。  
  
 **錯誤 ID︰** BC31123  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請確定前面的 `Custom Event` 陳述式有效而且拼寫正確。
