# <a name="value-valuename1-cannot-be-converted-to-valuename2"></a>值 '\<值名稱 1&gt >' 無法轉換成'\<值名稱 2&gt >'
已嘗試在兩個不相容的類型之間進行類型轉換。  
  
 **錯誤 ID:** BC30742  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請確認正在轉換的類型。
