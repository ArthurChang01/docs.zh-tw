# <a name="value-ltvaluename1gt-cannot-be-converted-to-ltvaluename2gt"></a>值 '&lt;值名稱 1&gt&gt;'無法轉換成'&lt;值名稱 2&gt&gt;'
已嘗試在兩個不相容的類型之間進行類型轉換。  
  
 **錯誤 ID:** BC30742  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請確認正在轉換的類型。
