# <a name="specified-event-log-does-not-exist-on-this-machine"></a>指定的事件記錄檔不存在於這部電腦上
嘗試存取不存在於這部電腦上的事件記錄檔。  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請檢查您提供的檔名和路徑是否正確。
