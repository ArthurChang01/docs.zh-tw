# <a name="statement-cannot-appear-outside-of-a-method-bodymultiline-lambda"></a>陳述式不可以出現在方法主體/多行 Lambda 之外
可執行的陳述式必須在程序內使用。  
  
 **錯誤 ID:** BC30689  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 將陳述式移入程序或副程式。
