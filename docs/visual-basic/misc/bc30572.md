# <a name="alias-namespacename-is-already-declared"></a>別名\<命名空間名稱 >' 已宣告
您嘗試使用 `Imports` 匯入已宣告的命名空間。  
  
 **錯誤 ID:** BC30572  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 請檢查您嘗試匯入的命名空間。
