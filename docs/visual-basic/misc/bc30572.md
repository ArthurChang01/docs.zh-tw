# 別名 &#39;&lt;命名空間名稱&gt;&#39; 已經宣告
您嘗試使用 `Imports` 匯入已宣告的命名空間。  
  
 **錯誤 ID︰**BC30572  
  
### 更正這個錯誤  
  
-   請檢查您嘗試匯入的命名空間。
