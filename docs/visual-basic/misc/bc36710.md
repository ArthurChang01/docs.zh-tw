# <a name="extension-method-39ltmethodnamegt39-defined-in-39ltmodulenamegt39-does-not-have-a-signature-compatible-with-delegate-39ltdelegatenamegt39"></a>擴充方法&#39; &lt;methodname&gt; &#39;中定義&#39; &lt;modulename&gt; &#39;沒有與委派相容的簽章&#39;&lt;委派名稱&gt;&#39;
您嘗試使用之擴充方法和委派的簽章不符。 `Delegate` 陳述式定義委派類別的參數類型和傳回類型。 任何具有相容參數、參數類型和傳回類型的程序都可用來建立這個委派類型的執行個體。
