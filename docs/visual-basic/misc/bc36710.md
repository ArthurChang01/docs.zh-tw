# <a name="extension-method-ltmethodnamegt-defined-in-ltmodulenamegt-does-not-have-a-signature-compatible-with-delegate-ltdelegatenamegt"></a>擴充方法 '&lt;methodname&gt;'中所定義的'&lt;modulename&gt;'並沒有與委派相容的簽章'&lt;委派名稱&gt;'
您嘗試使用之擴充方法和委派的簽章不符。 `Delegate` 陳述式定義委派類別的參數類型和傳回類型。 任何具有相容參數、參數類型和傳回類型的程序都可用來建立這個委派類型的執行個體。  
  
 **錯誤 ID:** BC36710
