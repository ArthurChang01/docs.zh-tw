# <a name="extension-method-methodname-defined-in-modulename-does-not-have-a-signature-compatible-with-delegate-delegatename"></a>擴充方法 '\<方法名稱 >' 中定義'\<模組名稱 >' 沒有與委派相容的簽章 '\<委派名稱 >'
您嘗試使用之擴充方法和委派的簽章不符。 `Delegate` 陳述式定義委派類別的參數類型和傳回類型。 任何具有相容參數、參數類型和傳回類型的程序都可用來建立這個委派類型的執行個體。  
  
 **錯誤 ID:** BC36710
