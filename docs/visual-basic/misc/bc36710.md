# <a name="extension-method-39ltmethodnamegt39-defined-in-39ltmodulenamegt39-does-not-have-a-signature-compatible-with-delegate-39ltdelegatenamegt39"></a>擴充方法 &#39;&lt;methodname&gt;&#39; 定義中 （& s) #39;&lt;modulename&gt;&#39; 沒有與委派 &#39; 相容的簽章&lt;delegatename&gt;&#39;
