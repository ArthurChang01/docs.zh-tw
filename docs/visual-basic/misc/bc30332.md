# <a name="value-of-type-lttype1gt-cannot-be-converted-to-lttype2gt-because-lttype3gt-is-not-derived-from-lttype4gt"></a>類型的值 '&lt;type1&gt;'無法轉換成'&lt;type2&gt;'由於'&lt;類型 3>&gt;'不衍生自'&lt;type4&gt;'
在陣列項目的資料類型都不是參考類型的情況下，或在兩個陣列的項目類型之間無法進行轉換 (擴大或縮小) 的情況下，陳述式嘗試將陣列類型轉換成另一個陣列類型。  
  
 **錯誤 ID:** BC30332  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請確認用來判斷錯誤來源的兩個陣列項目的資料類型。
