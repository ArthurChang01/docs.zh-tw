# 您必須指定目前資料夾或其中一個子資料夾底下的路徑
指定的路徑不在目前目錄底下。 這可能是提供相對路徑時不正確剖析檔案路徑的結果。  
  
### 更正這個錯誤  
  
-   確定您提供的路徑正確。
