# <a name="argument-39ltargument1gt39-must-be-less-than-or-equal-to-the-length-of-argument-39ltargument2gt39"></a>引數 &#39;&lt;引數 1&gt;&#39; 必須是小於或等於引數的長度 &#39;&lt;引數 2&gt;&#39;
引數的長度超過允許的限制 (透過第二個引數的長度所決定)。  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
1.  請確定引數值的長度落在有效範圍內。  
  
2.  如果透過計算來提供引數，請檢查計算所處理的值。
