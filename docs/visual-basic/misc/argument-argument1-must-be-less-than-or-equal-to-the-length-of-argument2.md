# <a name="argument-ltargument1gt-must-be-less-than-or-equal-to-the-length-of-argument-ltargument2gt"></a>引數 '&lt;argument1&gt;'必須小於或等於引數的長度'&lt;引數 2&gt;'
引數的長度超過允許的限制 (透過第二個引數的長度所決定)。  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
1.  請確定引數值的長度落在有效範圍內。  
  
2.  如果透過計算來提供引數，請檢查計算所處理的值。
