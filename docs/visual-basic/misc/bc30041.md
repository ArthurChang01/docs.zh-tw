# <a name="maximum-number-of-errors-has-been-exceeded"></a>已經超過錯誤最大數目
編譯錯誤數目上限為 100。 編譯器會施加此限制，以減少錯誤報告的大小。  
  
 **錯誤 ID:** BC30041  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   檢閱此編譯所產生的錯誤並盡可能全數解決。 如果原始程式碼包含超過 100 個錯誤，您可能會在下次編譯中看到更多錯誤。
