# <a name="lttypenamegt-cannot-be-used-as-an-attribute-because-it-does-not-have-a-systemattributeusageattribute-attribute"></a>'&lt;typename&gt;' 無法用做為屬性中，因為它沒有 'System.AttributeUsageAttribute' 屬性
已嘗試使用的屬性，在宣告時未使用 `System.AttributeUsageAttribute` 來定義其使用方式。  
  
 **錯誤 ID:** BC31505  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
1.  自訂屬性必須是衍生自類別`System.Attribute`具有`AttributeUsageAttribute`套用的屬性。
