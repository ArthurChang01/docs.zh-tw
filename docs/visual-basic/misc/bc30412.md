# <a name="withevents-variables-must-have-an-as-clause"></a>'WithEvents' 變數必須有 'As' 子句
您未提供具有關鍵字 `As` 的 `WithEvents`子句。 請將變數宣告為可引發事件的特定類別。  
  
 **錯誤識別碼:** BC30412  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
1. 加入指定可引發事件之類別的 `As` 子句。
