# <a name="xml-attribute-39version39-must-be-the-first-attribute-in-xml-declaration"></a>XML 屬性&#39;版本&#39;必須是 XML 宣告中的第一個屬性
XML 文件常值的 `version` 屬性不是第一個指定的屬性。 根據 XML 1.0 規格， `version` 屬性必須是針對 XML 文件常值指定的第一個屬性。  
  
 **錯誤 ID︰** BC31156  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請重新定位 XML 文件常值的 `version` 屬性，讓它是第一個屬性。
