# <a name="the-file-is-currently-open-for-writing"></a>檔案目前開放寫入
已嘗試存取目前開放寫入的檔案。  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請關閉檔案，然後再次嘗試這項作業。
