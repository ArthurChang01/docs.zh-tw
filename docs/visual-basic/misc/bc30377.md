# <a name="method-ltmethodnamegt-is-already-declared-in-interface-ltinterfacenamegt"></a>方法 '&lt;methodname&gt;'已經在介面中宣告'&lt;interfacename&gt;'
您已第二次宣告介面方法，該介面方法重複。  
  
 **錯誤 ID:** BC30377  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請移除重複的宣告。
