# <a name="39default39-cannot-be-combined-with-39ltspecifiergt39"></a>&#39; 預設值 &#39;無法結合 &#39;&lt;規範&gt;&#39;
您嘗試結合 `Default` 關鍵字與規範 (例如 `Shared` 或 `Private`)，但此結合在這個內容中無效。  
  
 **錯誤 ID︰** BC30490  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   移除 `Default`。  
  
     -或-  
  
-   請移除衝突的規範。
