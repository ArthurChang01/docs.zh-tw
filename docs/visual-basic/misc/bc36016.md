# <a name="implicit-variable-39ltvariablenamegt39-is-invalid-because-of-39ltmessagegt39"></a>隱含變數&#39; &lt;variablename&gt; &#39;無效，因為&#39;&lt;訊息&gt;&#39;
在偵錯期間，已產生無效的隱含變數。  
  
 **錯誤 ID︰** BC36016  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
1.  請檢查指出的訊息中是否有原因和可能的補救方法。  
  
2.  如果錯誤持續發生，請收集情況的相關資訊，並通知 Microsoft 產品支援服務。
