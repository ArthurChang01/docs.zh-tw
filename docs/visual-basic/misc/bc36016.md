# <a name="implicit-variable-ltvariablenamegt-is-invalid-because-of-ltmessagegt"></a>隱含變數 '&lt;variablename&gt;'是無效，因為'&lt;訊息&gt;'
在偵錯期間，已產生無效的隱含變數。  
  
 **錯誤 ID:** BC36016  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
1.  請檢查指出的訊息中是否有原因和可能的補救方法。  
  
2.  如果錯誤持續發生，請收集情況的相關資訊，並通知 Microsoft 產品支援服務。
