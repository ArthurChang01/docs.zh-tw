# <a name="type-cannot-be-inherited-more-than-once"></a>'\<類型 >' 無法繼承一次以上
`Inherits` 陳述式重複。  
  
 **錯誤 ID:** BC30584  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 請移除多餘的 `Inherits` 陳述式。
