# <a name="39lttypegt39-cannot-be-inherited-more-than-once"></a>&#39;&lt;型別&gt;&#39;不能繼承一次以上
`Inherits` 陳述式重複。  
  
 **錯誤 ID︰** BC30584  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請移除多餘的 `Inherits` 陳述式。
