# <a name="lttypegt-cannot-be-inherited-more-than-once"></a>'&lt;型別&gt;' 無法繼承一次以上
`Inherits` 陳述式重複。  
  
 **錯誤 ID:** BC30584  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請移除多餘的 `Inherits` 陳述式。
