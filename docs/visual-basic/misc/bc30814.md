# <a name="39gosub39-statements-are-no-longer-supported"></a>&#39;GoSub&#39;不再支援陳述式
`GoSub` 不能用來呼叫程序。  
  
 **錯誤 ID︰** BC30814  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   直接呼叫程序，但不使用 `GoSub`；例如：  
  
    ```  
    CalculateInterest(Amount, Rate, Time)  
    ```
