# <a name="xml-comment-tag-lttaggt-appears-with-identical-attributes-more-than-once-in-the-same-xml-comment-block"></a>XML 註解標記 '&lt;標記&gt;' 具有相同屬性超過一次出現在相同的 XML 註解區塊
XML 註解標記\<標記 > 具有相同屬性超過一次出現在相同的 XML 註解區塊中。 將會忽略 XML 註解。  
  
 XML 註解包含多個具有相同屬性的標記。  
  
 **錯誤 ID:** BC42305  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請移除重複的標記。
