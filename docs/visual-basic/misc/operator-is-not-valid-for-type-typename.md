# <a name="operator-is-not-valid-for-type-lttypenamegt"></a>運算子不是有效的類型 '&lt;typename&gt;'
運算子不適用於這些類型。  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請確定運算子適用於所操作的類型。
