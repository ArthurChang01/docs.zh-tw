# <a name="instance-members-and-39me39-cannot-be-used-within-a-lambda-expression-in-structures"></a><span data-ttu-id="6d93c-102">執行個體成員和&#39;我&#39;不可用於結構中的 lambda 運算式</span><span class="sxs-lookup"><span data-stu-id="6d93c-102">Instance members and &#39;Me&#39; cannot be used within a lambda expression in structures</span></span>
<span data-ttu-id="6d93c-103">從結構內，您已定義 Lambda 運算式指向結構的執行個體成員或使用 `Me`。</span><span class="sxs-lookup"><span data-stu-id="6d93c-103">From within a structure, you have defined a lambda expression that refers to an instance member of the structure or uses `Me`.</span></span> <span data-ttu-id="6d93c-104">下列程式碼會示範這兩個無效的參考。</span><span class="sxs-lookup"><span data-stu-id="6d93c-104">The following code illustrates these two invalid references.</span></span>  
  
```vb  
Structure Structure1  
  
    Public InstanceMember As Integer  
  
    Public Function ExampleFun() As Integer  
        '' The error is caused by use of InstanceMember.  
        'Dim fun1 = Function() InstanceMember  
        '' The error is caused by use of Me.  
        'Dim fun2 = Function() Me.InstanceMember  
        'Return fun1()  
    End Function  
  
End Structure  
```

## <a name="to-correct-this-error"></a><span data-ttu-id="6d93c-106">更正這個錯誤</span><span class="sxs-lookup"><span data-stu-id="6d93c-106">To correct this error</span></span>  
  
-   <span data-ttu-id="6d93c-107">將執行個體成員指派給區域變數，並在您的陳述式中使用區域變數。</span><span class="sxs-lookup"><span data-stu-id="6d93c-107">Assign the instance member to a local variable, and use the local variable in your statement.</span></span>

```vb  
    Dim fun1 = Function() temp  
    Return fun1()  
End Function  
```
