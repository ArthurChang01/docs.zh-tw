# <a name="warning-number-ltnumbergt-for-the-option-ltoptionnamegt-is-either-not-configurable-or-not-valid"></a>警告編號 '&lt;數字&gt;'的選項'&lt;optionname&gt;' 無法設定或無效
警告 ID 無效。 ID 必須是大於 0 的整數值。  
  
 **錯誤 ID:** BC2026  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請檢查提供的警告 ID，確認它有效。
