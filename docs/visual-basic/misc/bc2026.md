# <a name="warning-number-39ltnumbergt39-for-the-option-39ltoptionnamegt39-is-either-not-configurable-or-not-valid"></a>警告編號&#39;&lt;數目&gt;&#39;選項&#39;&lt;選項名稱&gt;&#39;無法設定或無效
警告 ID 無效。 ID 必須是大於 0 的整數值。  
  
 **錯誤 ID︰** BC2026  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請檢查提供的警告 ID，確認它有效。
