# <a name="warning-number-number-for-the-option-optionname-is-either-not-configurable-or-not-valid"></a>警告編號 '\<數字 >' 的選項'\<選項名稱 >' 無法設定或無效
警告 ID 無效。 ID 必須是大於 0 的整數值。  
  
 **錯誤 ID:** BC2026  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請檢查提供的警告 ID，確認它有效。
