# <a name="39shared39-cannot-be-combined-with-39ltspecifiergt39-on-a-method-declaration"></a>&#39;共用&#39;無法結合&#39;&lt;規範&gt;&#39;方法宣告上
您嘗試合併 `Shared` 修飾詞與規範，例如 `Overridable`、 `NotOverridable`或 `MustOverride`，這種合併在方法宣告中無效。  
  
 **錯誤 ID︰** BC30501  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請移除規範。
