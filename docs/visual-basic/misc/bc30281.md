# <a name="structure-structurename-must-contain-at-least-one-instance-member-variable-or-event-declaration"></a>結構 '\<結構名稱 >' 必須包含至少一個執行個體成員變數或 Event 宣告
會出現功能性空白結構。 在 `Structure` 陳述式與 `End Structure` 陳述式之間至少必須宣告一個變數成員。  
  
 **錯誤 ID:** BC30281  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   將 `Dim` 陳述式或 `Event` 陳述式加入結構中。
