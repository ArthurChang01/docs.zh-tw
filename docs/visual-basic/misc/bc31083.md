# <a name="methods-in-a-module-cannot-implement-interface-members"></a>模組中的方法不可以實作介面成員
介面成員必須由類別或結構所實作。  
  
 **錯誤 ID︰** BC31083  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請從類別或結構實作介面成員。
