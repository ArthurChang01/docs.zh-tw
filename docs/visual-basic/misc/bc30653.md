# <a name="reference-required-to-module-39ltmodulenamegt39-containing-the-type-39ltmembernamegt39"></a>需要模組 &#39; 的參考&lt;modulename&gt;&#39; 包含類型 &#39;&lt;membername&gt;&#39;
需要模組參考 '\<模組名稱 >' 包含型別'\<成員名稱 >'。 請在專案中加入一個參考。  
  
 專案所含的類型取決於未包含在專案參考之組件的另一個類型。  
  
 **錯誤 ID︰** BC30653  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請在專案中加入相依類型的專案參考。
