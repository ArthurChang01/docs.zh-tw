# <a name="xml-attribute-39attribute139-must-appear-before-xml-attribute-39attribute239"></a>XML 屬性 &#39; attribute1 &#39;必須出現在 XML 屬性 &#39; attribute2 &#39; 之前
XML 文件常值中的屬性指定順序無效。 XML 文件常值的有效屬性順序是以 XML 1.0 規格為基礎。 例如，XML 文件常值的 `encoding` 屬性必須緊接在 `version` 屬性之後。  
  
 **錯誤 ID︰** BC31157  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請使用 XML 1.0 規格中指定的指導方針，來更新 XML 文件常值的屬性順序。
