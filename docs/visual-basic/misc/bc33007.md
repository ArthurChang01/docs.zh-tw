# <a name="39end-operator39-must-be-preceded-by-a-matching-39operator39"></a>&#39;End Operator&#39;之前必須搭配相對應的&#39;運算子&#39;
`End Operator` 陳述式的前面沒有配對的 `Operator` 宣告。  
  
 **錯誤識別碼：** BC33007  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   若 `End Operator` 陳述式為多餘，請予以移除。  
  
-   提供遺漏的 `Operator` 程序 (遺漏時)。  
  
-   請將 `End Operator` 陳述式移至程式碼中的適當位置。
