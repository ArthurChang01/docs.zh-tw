# <a name="properties-declared-readonly-cannot-have-a-set"></a>宣告為 'ReadOnly' 的屬性不能有 'Set'
`Set` 程序會寫入屬性的值。 無法寫入`ReadOnly` 屬性。  
  
 **錯誤 ID:** BC30022  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請從 `ReadOnly` 陳述式中移除 `Property` 關鍵字，或從屬性主體中移除 `Set` 程序。
