# <a name="39end-synclock39-must-be-preceded-by-a-matching-39synclock39"></a>&#39;End SyncLock &#39;之前必須搭配相對應的 &#39;SyncLock &#39;
`SyncLock` 區塊以 `SyncLock` 關鍵字開頭並以 `End``SyncLock` 建構結尾。  
  
 **錯誤 ID：** BC30674  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   確定 `SyncLock` 區塊以 `SyncLock` 陳述式開頭。
