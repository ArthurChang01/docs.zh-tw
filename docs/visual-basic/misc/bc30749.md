# <a name="unable-to-access-member"></a>無法存取成員
無法存取指定的成員。  
  
 **錯誤 ID︰** BC30749  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
1. 請確認已定義成員，而且成員名稱拼字正確。  
  
2. 嘗試存取模組中所宣告的其中一個成員。 在某些情況下，偵錯環境找不到成員，因為尚未載入宣告它們的模組。
