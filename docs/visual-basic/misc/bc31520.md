# <a name="unable-to-apply-security-attributes-to-name-error"></a>無法將安全性屬性套用至 '\<名稱 >':\<錯誤 >
套用安全性屬性時發生錯誤。 這可能是不正確使用安全性屬性所造成，也可能表示編譯器發生問題。  
  
 **錯誤 ID:** BC31520  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
1. 確認屬性的語法和引數。  
  
2. 再次編譯程式，看看錯誤是否重複發生。  
  
3. 如果錯誤持續發生，請儲存您的工作，然後重新啟動 Visual Studio。  
  
4. 如果錯誤再次發生，請重新安裝 Visual Basic。  
  
5. 如果錯誤在重新安裝之後持續發生，請通知 Microsoft 產品支援服務。
