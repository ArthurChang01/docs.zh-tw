# <a name="membername-is-already-declared-by-procedurename-which-was-generated-for-this-containername"></a>'\<成員名稱 >' 已經宣告'\<程序名稱 >'，其產生的這個\<容器名稱 >
已宣告成員的名稱與 Comom Language Runtime 所宣告的成員相同。  
  
 **錯誤 ID:** BC30733  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請變更要移除衝突的成員名稱。
