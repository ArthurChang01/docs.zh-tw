已宣告成員的名稱與 Comom Language Runtime 所宣告的成員相同。  
  
 **錯誤 ID:** BC30733  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請變更要移除衝突的成員名稱。
