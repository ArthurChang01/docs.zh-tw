# <a name="file-io-with-type-39lttypenamegt39-is-not-valid"></a>類型具有檔案 I/O &#39; &lt;typename&gt; &#39;無效
您正在嘗試將不支援的類型寫入檔案。  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請檢查結構以決定不受支援類型的發生位置，然後將它變更為有效的類型。
