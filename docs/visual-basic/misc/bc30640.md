# <a name="39option-explicit39-can-be-followed-only-by-39on39-or-39off39"></a>&#39; option Explicit &#39;後面只能接 &#39; 在 &#39;或 &#39; 關閉 &#39;
`Option Explicit` 陳述式必須指定 `On` 或 `Off`。 所有其他值都無效。  
  
 **錯誤 ID︰** BC30640  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請指定 `Option Explicit On` 或 `Option Explicit Off`。
