# <a name="paramarray-parameter-must-be-an-array"></a>ParamArray 參數必須為陣列
`ParamArray` 引數必須宣告為一維陣列類型。 宣告在 `ParamArray` 引數名稱之後未包含括弧。  
  
 **錯誤 ID:** BC30050  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請在 `ParamArray` 引數名稱之後加入一組括弧。
