# <a name="39set39-method-cannot-have-more-than-one-parameter"></a>&#39;設定 &#39;方法不可有多個參數
為屬性 `Set` 程序定義了多個參數。  
  
 **錯誤 ID︰** BC31063  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請從屬性 `Set` 程序移除任何多餘的參數。
