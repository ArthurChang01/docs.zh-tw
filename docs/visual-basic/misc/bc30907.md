<span data-ttu-id="9c2eb-103">繼承結構會導致巢狀類別之間發生循環相依性，也就是兩個類別彼此繼承。</span><span class="sxs-lookup"><span data-stu-id="9c2eb-103">An inheritance structure results in circular dependency among nested classes, that is, two classes inheriting from each other.</span></span>

    End Class
