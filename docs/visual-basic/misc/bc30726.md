# <a name="39with39-contexts-and-statements-are-not-valid-in-debug-windows"></a>&#39;與&#39;內容和陳述式不是有效偵錯視窗中
`With` 陳述式只能用於原始程式碼中。  
  
 **錯誤 ID︰** BC30726  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請從偵錯程式碼中移除 `With` 陳述式。
