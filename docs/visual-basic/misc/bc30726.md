# <a name="39with39-contexts-and-statements-are-not-valid-in-debug-windows"></a>'With' 陳述式和內容的有效值不在偵錯視窗
`With`陳述式僅能在原始程式碼中。  
  
 **錯誤 ID︰** BC30726  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請從偵錯程式碼中移除 `With` 陳述式。
