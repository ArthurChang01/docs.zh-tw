# <a name="leading--or--cannot-appear-in-a-constant-expression"></a>前置的 '.' 或 '!' 不能出現在常數運算式中
成員存取 （.） 和字典成員存取 （！） 需要一個運算式來指定包含該成員大部分的情況下，包括常數運算式的項目。 下列宣告無效。  
  
```  
' Not valid.  
Const c As String = .name  
```  
  
 **錯誤 ID:** BC30995  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 請指定包含您想要存取之成員的執行個體。
