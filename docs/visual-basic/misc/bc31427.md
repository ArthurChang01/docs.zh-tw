# <a name="syntax-error-in-conditional-compilation-expression"></a>條件式編譯運算式中的語法錯誤
條件式編譯運算式產生不明確的語法錯誤。  
  
 **錯誤 ID:** BC31427  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
1. 檢查文件中的條件式編譯，以及用於運算式的關鍵字。 將文件中的範例程式碼與產生這個錯誤的原始程式行進行比較。  
  
2. 如果無法指出錯誤的原因，請收集情況的相關資訊，並通知 Microsoft 產品支援服務。
