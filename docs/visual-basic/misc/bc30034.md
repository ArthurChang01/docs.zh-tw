# <a name="bracketed-identifier-is-missing-closing-3939"></a>以方括弧括住的識別項遺漏右&#39;]&#39;
逸出名稱中的括弧必須成對。  
  
 **錯誤 ID︰** BC30034  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請在逸出名稱的結尾放置一個右括弧。
