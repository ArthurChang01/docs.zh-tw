# <a name="39if39-block-must-end-with-a-matching-39end-if39"></a>&#39;#If&#39;區塊必須搭配相對應的結束&#39;#End If&#39;
`#If` 是條件式編譯指示詞。 `#If` 區塊不是透過 `#End If` 指示詞所終止。  
  
 **錯誤 ID︰** BC30012  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請將 `#End If` 指示詞加入條件式編譯區塊的結尾。
