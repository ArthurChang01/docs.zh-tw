# <a name="39end-sub39-expected"></a>&#39;End Sub&#39;預期
`End` 陳述式必須符合它所終止的區塊。 未正確地終止 `Sub` 程序定義。  
  
 **錯誤 ID︰** BC30026  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   將 `End Sub` 陳述式加入程序的結尾。
