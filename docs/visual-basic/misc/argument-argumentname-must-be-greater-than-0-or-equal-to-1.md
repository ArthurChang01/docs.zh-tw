# <a name="argument-39ltargumentnamegt39-must-be-greater-than-0-or-equal-to--1"></a>引數&#39;&lt;引數名稱&gt;&#39;必須大於 0 或等於-1
引數的值小於 0，但不等於 -1。  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
1.  請確定引數的值落在有效範圍內。  
  
2.  如果透過計算來提供引數，請檢查計算所處理的值。
