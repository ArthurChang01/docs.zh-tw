# <a name="field-fieldname-of-type-typename-is-readonly"></a>欄位 '\<欄位名稱 >' 的類型'\<類型名稱 >' 是 'ReadOnly'
您嘗試寫入已宣告為 `ReadOnly`的欄位。  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   變更欄位，使其不再宣告為 `ReadOnly`。  
  
-   -或-  
  
-   請勿嘗試寫入這個欄位。
