# <a name="39ltinterfacename1gt39-cannot-implement-39ltmethodnamegt39-because-there-is-no-matching-ltmethodgt-on-interface-39ltinterfacename2gt39"></a>&#39;&lt;介面名稱 1>.<&gt; &#39;無法實作&#39; &lt;methodname&gt; &#39;因為沒有相符&lt;方法&gt;介面上&#39; &lt;介面名稱 2>&gt;&#39;
您嘗試在從中衍生第一個介面的介面上沒有對應的方法時，於該介面上實作方法。  
  
 **錯誤識別碼︰** BC30401  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   確定您沒有拼錯方法的名稱，或提供了遺漏的方法。
