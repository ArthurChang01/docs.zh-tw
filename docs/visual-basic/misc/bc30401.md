# <a name="ltinterfacename1gt-cannot-implement-ltmethodnamegt-because-there-is-no-matching-ltmethodgt-on-interface-ltinterfacename2gt"></a>'&lt;介面名稱 1>.<&gt;'無法實作'&lt;methodname&gt;' 因為沒有相符&lt;方法&gt;介面上 '&lt;介面名稱 2&gt&gt;'
您嘗試在從中衍生第一個介面的介面上沒有對應的方法時，於該介面上實作方法。  
  
 **錯誤 ID:** BC30401  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   確定您沒有拼錯方法的名稱，或提供了遺漏的方法。
