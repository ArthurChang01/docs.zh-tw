# <a name="interfacename1-cannot-implement-methodname-because-there-is-no-matching-method-on-interface-interfacename2"></a>'\<介面名稱 1>.< >' 無法實作'\<方法名稱 >' 因為沒有相符\<方法 > 介面上 '\<介面名稱 2&gt >'
您嘗試在從中衍生第一個介面的介面上沒有對應的方法時，於該介面上實作方法。  
  
 **錯誤 ID:** BC30401  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   確定您沒有拼錯方法的名稱，或提供了遺漏的方法。
