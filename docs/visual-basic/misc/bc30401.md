# <a name="interfacename1-cannot-implement-methodname-because-there-is-no-matching-method-on-interface-interfacename2"></a>'\<介面名稱 1> > ' 無法執行 '\<方法名稱 > ', 因為在介面\<'\<介面名稱 2> > ' 上找不到相符的方法 >
您嘗試在從中衍生第一個介面的介面上沒有對應的方法時，於該介面上實作方法。  
  
 **錯誤識別碼:** BC30401  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 確定您沒有拼錯方法的名稱，或提供了遺漏的方法。
