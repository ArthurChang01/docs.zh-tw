# <a name="wrong-number-of-arguments-or-property-assignment-not-valid"></a>引數數目錯誤或屬性指派無效
嘗試進行無效的指派。  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
1.  檢查您提供的引數數目是否符合目標所需的數目。  
  
2.  檢查屬性指派。
