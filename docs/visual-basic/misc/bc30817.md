# <a name="39on-goto39-and-39on-gosub39-statements-are-no-longer-supported"></a>&#39; GoTo &#39;和 &#39; GoSub &#39;不再支援陳述式
搭配使用變數或運算式的值與 `GoTo` 和 `GoSub` 陳述式來控制程式流程，不再有效。  
  
 **錯誤 ID︰** BC30817  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請重構您的應用程式，以使用 `If...Then...Else` 或 `Case` 陳述式。
