# <a name="conversions-from-lttypename1gt-to-lttypename2gt-must-be-explicit"></a>從 '&lt;typename1&gt;'到'&lt;2&gt&gt;' 必須是明確
無法以隱含方式完成這項轉換。  
  
 **錯誤 ID:** BC30664  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   使用 `CType`或其中一個特定轉換函式，例如 `CInt` 或 `CDec`。
