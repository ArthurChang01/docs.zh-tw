# <a name="conversions-from-39lttypename1gt39-to-39lttypename2gt39-must-be-explicit"></a>從不 &#39;&lt;typename1&gt;&#39; 加入 &#39;&lt;2>&gt;&#39; 必須明確
無法以隱含方式完成這項轉換。  
  
 **錯誤 ID：** BC30664  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   使用 `CType`或其中一個特定轉換函式，例如 `CInt` 或 `CDec`。
