# <a name="typename-cannot-be-used-as-an-attribute-because-it-is-not-a-class"></a>'\<類型名稱 >' 無法用做為屬性中，因為它不是類別
您嘗試作為屬性使用的類型不是類別。  
  
 **錯誤 ID:** BC31503  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
1. 將自訂屬性定義為衍生自 `System.Attribute`的類別。
