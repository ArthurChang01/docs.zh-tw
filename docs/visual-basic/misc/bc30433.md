# <a name="methods-in-a-module-cannot-be-declared-39ltspecifiergt39"></a>在模組中的方法不可以宣告&#39;&lt;規範&gt;&#39;
您已經在 `Module` 陳述式內使用對方法無效的規範。 模組絕不會執行個體化、不支援繼承，且不能實作介面。  
  
 **錯誤識別碼︰** BC30433  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請移除規範。
