# <a name="39case39-cannot-follow-a-39case-else39-in-the-same-39select39-statement"></a>&#39;案例 &#39;不能跟在 &#39;其他情況 &#39;在相同 &#39;Select &#39;陳述式
如果沒有找到初始 `Case Else` 的相符項， `Case`陳述式導致執行陳述式。 在同一個 `Case` 區塊的 `Case Else` 之後找到 `Select` 陳述式。  
  
 **錯誤 ID︰** BC30321  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請將 `Case Else` 移至 `Case` 陳述式之後的適當位置。
