# <a name="argument-39ltargumentname1gt39-must-be-less-than-or-equal-the-length-of-argument-39ltargumentname2gt39"></a>引數 &#39;&lt;argumentname1&gt;&#39; 必須是小於或等於引數 &#39; 長度&lt;引數名稱 2&gt;&#39;
提供的引數長度超過它正在取代的引數長度。  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請讓第一個引數的長度小於或等於第二個引數的長度。
