# 引數 &#39;&lt;引數名稱1&gt;&#39; 必須小於或等於引數 &#39;&lt;引數名稱2 &gt;&#39; 的長度
提供的引數長度超過它正在取代的引數長度。  
  
### 更正這個錯誤  
  
-   請讓第一個引數的長度小於或等於第二個引數的長度。
