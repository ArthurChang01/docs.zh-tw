# <a name="argument-ltargumentname1gt-must-be-less-than-or-equal-the-length-of-argument-ltargumentname2gt"></a>引數 '&lt;引數名稱 1&gt&gt;' 必須是小於或等於引數的長度 '&lt;引數名稱 2&gt;'
提供的引數長度超過它正在取代的引數長度。  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請讓第一個引數的長度小於或等於第二個引數的長度。
