# <a name="out-of-memory-visual-basic-run-time-error"></a>記憶體不足 (Visual Basic 執行階段錯誤)
需要提供更多的記憶體。  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
1. 關閉不必要的應用程式、文件或開啟的原始程式檔。  
  
2. 將大型模組或程序分成較小項目。  
  
3. 消除常駐程式。  
  
4. 消除不必要的裝置驅動程式。  
  
5. 減少公用變數的數量。
