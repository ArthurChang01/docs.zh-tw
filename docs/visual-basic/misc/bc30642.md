# <a name="39optional39-and-39paramarray39-cannot-be-combined"></a>&#39;選擇性&#39;和&#39;ParamArray&#39;無法合併
`Optional` 與 `ParamArray` 修飾詞互斥。  
  
 **錯誤 ID︰** BC30642  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請指定 `ParamArray` 或 `Optional`，但兩者不能同時指定。
