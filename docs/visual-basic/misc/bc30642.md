# <a name="optional-and-paramarray-cannot-be-combined"></a>無法結合 'Optional' 和 'ParamArray'
`Optional` 與 `ParamArray` 修飾詞互斥。  
  
 **錯誤 ID:** BC30642  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請指定 `ParamArray` 或 `Optional`，但兩者不能同時指定。
