# <a name="operator-39operator39-is-not-defined-for-types-39type139-and-39type239"></a>運算子&#39;運算子&#39;未定義類型&#39;type1&#39;和&#39;type2&#39;
未定義類型 'type1' 和 'type2' 的運算子 'operator'。 您可以使用 'Value' 屬性取得的第一個元素的字串值 '\<parentElement >'。  
  
 二元或 `Boolean` 比較中已使用 XML 常值。  
  
 **錯誤 ID︰** BC31195  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   使用 XML 常值的 `Value` 屬性來進行二元或 `Boolean` 比較。
