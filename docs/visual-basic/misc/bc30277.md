# <a name="type-character-39ltcharacternamegt39-does-not-match-declared-data-type-39lttypegt39"></a>類型字元&#39;&lt;字元&gt;&#39;宣告的資料類型不符&#39;&lt;類型&gt;&#39;
將 `K$ = 10` 宣告為 `K` 時，變數是使用某種資料類型宣告，但參考時卻使用代表不相容資料類型的類型字元，例如 `Integer`。  
  
 **錯誤 ID：** BC30277  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   變更宣告變數的資料類型，或變更或移除參考中的類型字元。
