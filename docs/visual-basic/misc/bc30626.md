# <a name="namespace-statement-must-end-with-a-matching-end-namespace"></a>'Namespace' 陳述式之後必須搭配相對應的 'End Namespace'
`Namespace` 區塊的結尾必須是 `End Namespace` 陳述式。  
  
 **錯誤 ID:** BC30626  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   將 `End Namespace` 陳述式加入 `Namespace` 區塊的結尾。
