# <a name="39namespace39-statement-must-end-with-a-matching-39end-namespace39"></a>&#39;命名空間&#39;陳述式必須搭配相對應的結束&#39;結束命名空間&#39;
`Namespace` 區塊的結尾必須是 `End Namespace` 陳述式。  
  
 **錯誤 ID：** BC30626  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   將 `End Namespace` 陳述式加入 `Namespace` 區塊的結尾。
