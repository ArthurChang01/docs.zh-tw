# <a name="39region39-statement-must-end-with-a-matching-39end-region39"></a>&#39; #Region &#39;陳述式的結尾必須符合 &#39; #End 區域 &#39;
使用 `#Region` 指示詞來指定程式碼區段，以在使用 [!INCLUDE[vsprvs](~/includes/vsprvs-md.md)] 程式碼編輯器的大綱功能時展開或摺疊。 `#Region` 陳述式的開頭和結尾必須在相同的程式碼區塊中。  
  
 **錯誤 ID：** BC30681  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
1.  請在 `#End Region` 陳述式後面之程式碼的適當位置插入 `#Region` 。
