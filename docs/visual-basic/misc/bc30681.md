# <a name="39region39-statement-must-end-with-a-matching-39end-region39"></a>&#39;#Region&#39;陳述式必須搭配相對應的結束&#39;#End 區域&#39;
使用`#Region`指示詞來指定一段程式碼，您可以展開或摺疊時使用的 Visual Studio Code 編輯器的大綱功能。 `#Region` 陳述式的開頭和結尾必須在相同的程式碼區塊中。  
  
 **錯誤 ID：** BC30681  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
1.  請在 `#End Region` 陳述式後面之程式碼的適當位置插入 `#Region` 。
