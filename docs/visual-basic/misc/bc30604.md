# <a name="statement-cannot-appear-within-an-interface-body-visual-basic-error"></a>陳述式不能出現在介面主體內 (Visual Basic 錯誤)
發生未預期的語言建構。 假設遺漏 `End Interface` 建構，而且這個位置後面的所有原始程式碼都在介面外部。  
  
 **錯誤 ID:** BC30604  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
1.  請確認介面定義內使用程式碼語法。  
  
2.  請確定介面定義的結尾為 `End Interface` 建構。
