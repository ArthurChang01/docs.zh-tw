# <a name="ltmethodgt-39ltmethodname1gt39-must-be-declared-39overloads39-because-another-39ltmethodname2gt39-is-declared-39overloads39"></a>&lt;方法&gt; &#39; &lt;methodname1&gt; &#39;必須宣告&#39;多載&#39;因為另一個&#39; &lt;methodname2&gt; &#39;宣告&#39;多載&#39;
使用 `Overloads` 修飾詞宣告同名的另一個屬性或方法。  
  
 **錯誤 ID︰** BC31409  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
1.  將 `Overloads` 修飾詞加入屬性或方法。  
  
2.  從同名的所有屬性或方法中移除 `Overloads` 修飾詞。
