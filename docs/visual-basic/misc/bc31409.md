# &lt;方法&gt; &#39;&lt;方法名稱1&gt;&#39; 必須宣告為 &#39;Overloads&#39;，因為另一個 &#39;&lt;方法名稱2&gt;&#39; 已宣告為 &#39;Overloads&#39;
使用 `Overloads` 修飾詞宣告同名的另一個屬性或方法。  
  
 **錯誤 ID︰**BC31409  
  
### 更正這個錯誤  
  
1.  將 `Overloads` 修飾詞加入屬性或方法。  
  
2.  從同名的所有屬性或方法中移除 `Overloads` 修飾詞。
