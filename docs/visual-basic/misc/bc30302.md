# <a name="type-character-39ltcharactergt39-cannot-be-used-in-a-declaration-with-an-explicit-type"></a>型別字元 &#39;&lt;字元&gt;&#39; 不能在具有明確類型的宣告
明確指定類型時使用了類型字元，例如 `$`。  
  
 **錯誤 ID︰** BC30302  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請從宣告移除類型字元，或移除 `As <Type>` 子句。
