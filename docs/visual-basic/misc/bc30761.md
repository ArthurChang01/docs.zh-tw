# <a name="project-projectname-cannot-reference-project-projectname-because-projectname-directly-or-indirectly-references-projectname"></a>專案 '\<專案名稱 >' 無法參考專案'\<專案名稱 >' 由於 '\<專案名稱 >' 直接或間接參考'\<專案名稱 >'
專案包括第二個專案的參考。 第二個專案接著會包括參考它之專案的參考。  
  
 **錯誤 ID:** BC30761  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 請移除其中一個參考。
