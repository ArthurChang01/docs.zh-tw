# <a name="project-ltprojectnamegt-cannot-reference-project-ltprojectnamegt-because-ltprojectnamegt-directly-or-indirectly-references-ltprojectnamegt"></a>專案 '&lt;projectname&gt;'無法參考專案'&lt;projectname&gt;'由於'&lt;projectname&gt;'直接或間接參考'&lt;projectname&gt;'
專案包括第二個專案的參考。 第二個專案接著會包括參考它之專案的參考。  
  
 **錯誤 ID:** BC30761  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請移除其中一個參考。
