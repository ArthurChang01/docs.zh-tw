# <a name="project-39ltprojectnamegt39-cannot-reference-project-39ltprojectnamegt39-because-39ltprojectnamegt39-directly-or-indirectly-references-39ltprojectnamegt39"></a>專案&#39; &lt;projectname&gt; &#39;無法參考專案&#39; &lt;projectname&gt; &#39;因為&#39; &lt;projectname&gt; &#39;直接或間接參考&#39; &lt;projectname&gt;&#39;
專案包括第二個專案的參考。 第二個專案接著會包括參考它之專案的參考。  
  
 **錯誤 ID︰** BC30761  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請移除其中一個參考。
