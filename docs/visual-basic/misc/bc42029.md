# <a name="39catch39-block-never-reached-because-39ltname1gt39-inherits-from-39ltname2gt39"></a>&#39;Catch &#39;區塊永遠不會執行到，因為 &#39;&lt;name1&gt;&#39; 繼承自 &#39;&lt;name2&gt;&#39;
永遠不會執行到程式碼內的 `Catch` 區塊。  
  
 根據預設，這個訊息是一個警告。 如需隱藏警告或將警告視為錯誤的詳細資訊，請參閱[Visual Basic 中的 設定警告](/visualstudio/ide/configuring-warnings-in-visual-basic)。  
  
 **錯誤 ID︰** BC42029  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請移除多餘的 `Catch` 區塊。
