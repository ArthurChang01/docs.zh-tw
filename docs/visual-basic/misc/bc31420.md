# &#39;&lt;成員&gt;&#39; 與使用此名稱的保留成員產生衝突，此名稱已在所有列舉中隱含宣告
類型成員名稱與所有列舉中隱含宣告的成員名稱相衝突。  
  
 [!INCLUDE[vbprvb](../../csharp/programming-guide/concepts/linq/includes/vbprvb-md.md)] 編譯器會建立與所宣告之特定程式設計項目對應的隱含成員。 列舉會隱含地宣告成員 `value__member`。  
  
 **錯誤 ID︰**BC31420  
  
### 更正這個錯誤  
  
-   修改成員的名稱。
