# <a name="member-conflicts-with-the-reserved-member-by-this-name-that-is-implicitly-declared-in-all-enums"></a>'\<成員 >' 在所有列舉中隱含宣告同名的保留成員互相衝突
類型成員名稱與所有列舉中隱含宣告的成員名稱相衝突。  
  
 Visual Basic 編譯器會建立您宣告特定程式設計項目對應的隱含成員。 列舉會隱含地宣告成員 `value__member`。  
  
 **錯誤 ID:** BC31420  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   修改成員的名稱。
