# <a name="end-raiseevent-must-be-the-first-statement-on-a-line"></a>'End RaiseEvent' 必須是該行的第一個陳述式
`End RaiseEvent` 陳述式接在冒號 (:) 陳述式分隔符號的後面。 `End RaiseEvent` 必須是其原始程式碼行上的唯一陳述式。  
  
 **錯誤 ID:** BC31120  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 請將多個陳述式分成不同的行。
