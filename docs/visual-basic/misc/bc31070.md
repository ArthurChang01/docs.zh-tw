# <a name="class-in-an-interface-cannot-be-declared-39ltspecifiergt39"></a>介面中的類別不可以宣告 &#39;&lt;規範&gt;&#39;
指定的無效修飾詞具有在介面中宣告的類別。  
  
 **錯誤識別碼：** BC31070  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請移除修飾詞 (例如 `Public`、 `Private`、 `Shared`、 `Friend`、 `Protected`、 `Protected Friend`或 `Overridable`)。
