# <a name="keyword-is-valid-only-within-a-class"></a>'\<關鍵字 >' 無效，只在類別
與類別相關的關鍵字 (例如 `Me` 或 `MyClass`) 用在類別定義外部。  
  
 **錯誤 ID:** BC32002  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   如果使用關鍵字的程式碼牽涉到類別執行個體，請將它移至類別實作。  
  
-   如果使用關鍵字的程式碼未套用至類別，請移除無效的關鍵字。
