# <a name="xml-namespace-prefix-prefix-is-already-declared"></a>XML 命名空間前置詞 'prefix' 已宣告
`Imports` 陳述式所定義的 XML 命名空間前置詞，符合先前定義的 XML 命名空間前置詞。  
  
 **錯誤 ID:** BC30573  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 請針對 XML 命名空間前置詞使用唯一名稱。
