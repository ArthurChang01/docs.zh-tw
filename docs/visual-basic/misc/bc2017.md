# <a name="could-not-find-library-libraryname"></a>找不到程式庫 '\<程式庫名稱 >'
找不到指定的程式庫。  
  
 此問題可能會在從命令提示字元進行編譯，以及使用參考許多組件的回應檔 (.rsp) 時發生  
  
 **錯誤 ID:** BC2017  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請檢查程式庫名稱，確定它正確。
