# <a name="array-initializer-is-missing-number-elements"></a>陣列初始設定式遺漏\<數字 > 項目
您尚未在初始設定式中提供陣列的足夠項目。  
  
 **錯誤 ID:** BC30567  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 請提供遺漏的引數。  
  
- 使用 `ReDim` 變更陣列的大小。
