# <a name="39end-sub39-must-be-preceded-by-a-matching-39sub39"></a>&#39;End Sub &#39;之前必須搭配相對應的 &#39;Sub &#39;
`End Sub` 陳述式出現在您的程式碼中，但前面沒有對應的 `Sub` 程序定義。  
  
 **錯誤 ID︰** BC30429  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   若 `End Sub` 陳述式為多餘，請予以移除。  
  
-   提供遺漏的 `Sub` 程序 (遺漏時)。  
  
-   將 `End Sub` 移至程式碼中的適當位置。
