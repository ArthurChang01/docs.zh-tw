# <a name="39addhandler39-39removehandler39-and-39raiseevent39-method-parameters-cannot-be-declared-39ltmodifiergt39"></a>'AddHandler'，'RemoveHandler' 和 'RaiseEvent' 方法的參數不可以宣告為 '&lt;修飾詞&gt;'
不能使用 `AddHandler`或 `RemoveHandler`修飾詞來宣告 `RaiseEvent` 、 `Optional` 和 `ParamArray` 方法的參數。  
  
 `Optional` 或 `ParamArray` 修飾詞僅允許用在 `Declare`、 `Function`、 `Property`和 `Sub` 宣告的參數上。  
  
 **錯誤 ID︰** BC31138  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請從參數清單中移除 `Optional` 或 `ParamArray` 關鍵字。
