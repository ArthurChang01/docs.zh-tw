# <a name="39addhandler39-is-already-declared"></a>&#39;AddHandler&#39;已宣告
多個 `AddHandler` 宣告會出現在自訂事件宣告中。 `AddHandler` 宣告會宣告用來加入事件處理常式的程序。  
  
 **錯誤 ID︰** BC31127  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請移除多餘的 `AddHandler` 陳述式。
