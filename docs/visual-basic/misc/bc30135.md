# <a name="39synclock39-statements-are-not-valid-in-the-immediate-window"></a>&#39;SyncLock&#39;陳述式不是有效的即時運算視窗中
`SyncLock` 陳述式會同步處理執行緒，但在偵錯內容中則不允許執行。  
  
 **錯誤 ID︰** BC30135  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請不要在 [即時運算] `SyncLock`**視窗中發出** 陳述式。
