# <a name="insufficient-security-permissions-to-set-the-system-date"></a>設定系統日期的安全性權限不足
安全性權限不足，無法設定系統日期。  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請變更安全性權限，以允許設定系統日期。
