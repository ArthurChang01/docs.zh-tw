# <a name="sub-new-cannot-handle-events"></a>'Sub New' 無法處理事件
您嘗試結合 `Sub New` 與 `Handles`，這種做法無效。 在程序宣告結尾使用 `Handles` 關鍵字，讓它處理由使用 `WithEvents` 關鍵字宣告之物件變數引發的事件。  
  
 **錯誤 ID:** BC30497  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
1. 請勿在這個內容中使用 `New` 。
