# <a name="property-propertyname-cannot-be-set-to-an-empty-string-or-nothing"></a>屬性\<屬性名稱 > 不能設為空字串或 Nothing
此屬性需要空字串 ("") 或 `Nothing`以外的值。 這可能是不正確計算值的結果。  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 請提供有效的值。
