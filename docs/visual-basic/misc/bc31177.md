# <a name="white-space-may-not-appear-here"></a>空白字元不可出現在這裡
XML 項目常值名稱包含一或多個空格。  
  
 **錯誤 ID︰** BC31177  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請從 XML 項目常值名稱中移除一或多個空格。
