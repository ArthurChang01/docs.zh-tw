# <a name="compilation-failed-message"></a>編譯失敗：\<訊息 >
由於發生錯誤訊息中指定的情況，所以編譯失敗  
  
 **錯誤 ID:** BC2010  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請檢查錯誤訊息，以判斷失敗的確切原因。
