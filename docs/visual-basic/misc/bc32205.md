# <a name="withevents-variable-name-length-cannot-exceed-1019-characters"></a>WithEvents 變數名稱長度不能超過 1019 個字元
使用 `WithEvents` 宣告的每個變數名稱或識別項限制為 1019 個字元。  
  
 **錯誤 ID:** BC32205  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   減少識別項的長度。
