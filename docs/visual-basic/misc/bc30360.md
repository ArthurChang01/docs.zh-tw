# 屬性 &#39;&lt;屬性名稱&gt;&#39; 必須宣告為 &#39;Default&#39;，因為它會覆寫基底 &#39;&lt;名稱&gt;&#39; 上的預設屬性。
程式碼中的 `Property` 陳述式會覆寫基底類別、結構或介面上的 `Default` 屬性，因此必須宣告為 `Default`。  
  
 **錯誤 ID︰**BC30360  
  
### 更正這個錯誤  
  
-   宣告 `Default` 屬性。
