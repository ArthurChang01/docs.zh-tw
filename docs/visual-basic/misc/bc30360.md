# <a name="property-ltpropertynamegt-must-be-declared-default-because-it-overrides-a-default-property-on-the-base-ltnamegt"></a>屬性 '&lt;propertyname&gt;'必須宣告為 'Default' 因為其會覆寫基底上的預設屬性'&lt;名稱&gt;'
程式碼中的 `Property` 陳述式會覆寫基底類別、結構或介面上的 `Default` 屬性，因此必須宣告為 `Default`。  
  
 **錯誤 ID:** BC30360  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   宣告 `Default`屬性。
