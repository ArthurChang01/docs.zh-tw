# <a name="39end-namespace39-must-be-preceded-by-a-matching-39namespace39"></a>&#39;End Namespace&#39;之前必須搭配相對應的&#39;命名空間&#39;
命名空間區塊會以 `Namespace` 陳述式開頭並以 `End Namespace` 建構結尾。  
  
 **錯誤 ID：** BC30623  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   確定命名空間區塊以 `Namespace` 陳述式開頭。
