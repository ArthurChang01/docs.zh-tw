# <a name="end-namespace-must-be-preceded-by-a-matching-namespace"></a>'End Namespace' 之前必須搭配相對應的 'Namespace'
命名空間區塊會以 `Namespace` 陳述式開頭並以 `End Namespace` 建構結尾。  
  
 **錯誤 ID:** BC30623  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 確定命名空間區塊以 `Namespace` 陳述式開頭。
