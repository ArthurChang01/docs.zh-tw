# <a name="39redim39-statements-require-a-parenthesized-list-of-the-new-bounds-of-each-dimension-of-the-array"></a>&#39;ReDim&#39;陳述式需要每個陣列維度新界限的括號括住清單
您必須在 `ReDim` 陳述式中指定陣列的新大小。  
  
 **錯誤 ID：** BC30670  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   提供陣列的每個陣序大小；例如：  
  
    ```  
    ReDim arr(5, 6)  
    ```
