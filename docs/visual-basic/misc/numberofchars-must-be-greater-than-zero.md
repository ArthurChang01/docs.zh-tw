# NumberOfChars 必須大於零
使用 `TextFieldParser` 物件的 `PeekChars` 方法時，您必須提供大於 `0` 的 `NumberOfChars` 值。  
  
### 更正這個錯誤  
  
-   請將 `NumberOfChars` 變更為大於 `0` 的值。
