# <a name="string-length-exceeds-maximum-length-of-32767-characters-for-filesystem-apis"></a>字串長度超過 'FileSystem' API 容許長度 32767 個字元的上限
字串長度超過長度 32767 個字元的上限。  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
1.  請縮短字串。
