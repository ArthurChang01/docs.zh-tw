# <a name="argument-argumentname-is-nothing-or-empty"></a>引數 '\<引數名稱 >' 為 Nothing 或空白
引數名稱無效，因為它已初始化為 `Nothing` 或為空白。 引數必須具有特定資料類型和有效的名稱。  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請加入有效的引數名稱。
