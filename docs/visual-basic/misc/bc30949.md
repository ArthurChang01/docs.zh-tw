# <a name="array-initializer-cannot-be-specified-for-a-non-constant-dimension-use-the-empty-initializer-3939"></a><span data-ttu-id="cf824-102">陣列初始設定式無法指定非常數維度;使用空的初始設定式&#39;{}&#39;</span><span class="sxs-lookup"><span data-stu-id="cf824-102">Array initializer cannot be specified for a non constant dimension; use the empty initializer &#39;{}&#39;</span></span>
<span data-ttu-id="cf824-103">陣列初始化編譯時期未知的維度。</span><span class="sxs-lookup"><span data-stu-id="cf824-103">An array initializes a dimension that is not known at compile time.</span></span>  
  
 <span data-ttu-id="cf824-104">下列程式碼會產生這個錯誤。</span><span class="sxs-lookup"><span data-stu-id="cf824-104">The following code generates this error.</span></span>  
  
```  
Dim j As Integer  
Dim intArray As Integer = New Integer(1, j) {{0, 100}, {1,101}}  
```  
  
 <span data-ttu-id="cf824-105">下列程式碼可避免這個錯誤。</span><span class="sxs-lookup"><span data-stu-id="cf824-105">The following code avoids the error.</span></span>  
  
```  
Dim intArray As Integer = New Integer(1, j) {}  
For i As Integer = 0 To j  
    intArray(0, i) = i  
    intArray(1, i) = 100 + i  
Next i  
```  
  
 <span data-ttu-id="cf824-106">**錯誤 ID︰** BC30949</span><span class="sxs-lookup"><span data-stu-id="cf824-106">**Error ID:** BC30949</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="cf824-107">更正這個錯誤</span><span class="sxs-lookup"><span data-stu-id="cf824-107">To correct this error</span></span>  
  
-   <span data-ttu-id="cf824-108">如果可能的話，請指定陣列宣告中的常數維度。</span><span class="sxs-lookup"><span data-stu-id="cf824-108">If possible, specify a constant dimension in the array declaration.</span></span>  
  
-   <span data-ttu-id="cf824-109">如果您無法指定常數維度，則必須在非常數維度變成未知時，使用迴圈初始化陣列。</span><span class="sxs-lookup"><span data-stu-id="cf824-109">If you cannot specify a constant dimension, then you must initialize the array using a loop when the nonconstant dimension becomes known.</span></span>
