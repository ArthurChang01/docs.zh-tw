# <a name="cannot-create-an-instance-of-module-ltmodulenamegt"></a>無法建立模組的執行個體 '&lt;modulename&gt;'
模組僅能作為單一共用執行個體，因此無法建立額外的執行個體。  
  
 **錯誤 ID:** BC30166  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   將模組變更為類別，或將 `New` 子句中的模組取代為類別名稱。
