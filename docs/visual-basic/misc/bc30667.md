# ParamArray 參數必須宣告為 &#39;ByVal&#39;
`ParamArray` 參數不能使用 `ByRef` 修飾詞。  
  
 **錯誤識別碼：**BC30667  
  
### 更正這個錯誤  
  
-   使用 `ByVal` 修飾詞宣告 `ParamArray`。
