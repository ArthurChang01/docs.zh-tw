# <a name="paramarray-parameters-must-be-declared-byval"></a>ParamArray 參數必須宣告為 'ByVal'
`ParamArray` 參數不能使用 `ByRef` 修飾詞。  
  
 **錯誤 ID:** BC30667  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 使用 `ParamArray` 修飾詞宣告 `ByVal` 。
