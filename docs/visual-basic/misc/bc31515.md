# <a name="project-39ltprojectnamegt39-makes-an-indirect-reference-to-assembly-39ltassemblynamegt39-which-contains-39ltclassnamegt39"></a>專案 &#39;&lt;projectname&gt;&#39; 會間接參考組件 &#39;&lt;assemblyname&gt;&#39; 其中包含 &#39;&lt;classname&gt;&#39;
專案 '\<專案名稱 >' 會間接參考組件'\<assemblyname >' 包含 '\<類別名稱 >'。 將參考加入 '\<assemblyname >' 加入專案。  
  
 此專案包含相依於未包含在專案參考之組件的成員。  
  
 **錯誤 ID︰** BC31515  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   加入組件的參考，方法是在 **方案總管** 中，以滑鼠右鍵按一下 [參考]  ，然後按一下 [加入參考] 。
