# <a name="xml-comment-tag-lttaggt-is-not-permitted-on-a-ltelementgt-language-element"></a>XML 註解標記 '&lt;標記&gt;' 上不允許 '&lt;項目&gt;' 語言項目
XML 註解標記\<標記 > 上不可使用 '\<項目 >' 項目。 將會忽略 XML 註解。  
  
 語言項目不支援套用它的 XML 註解標記。  
  
 **錯誤 ID:** BC42306  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請移除不支援的 XML 註解標記。
