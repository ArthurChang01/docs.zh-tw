# <a name="-expected"></a>必須有 '('
程序呼叫中的引數清單開頭不是左括弧 (`(`) 字元。  
  
 **錯誤 ID:** BC30199  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 請在清單開頭加入左括弧字元。
