# BaudRate 必須大於 0
為 `My.Computer.Ports.OpenSerialPort` 方法指定的 `BaudRate` 引數必須大於零。  
  
### 更正這個錯誤  
  
-   將 `BaudRate` 引數的值變更為正數。
