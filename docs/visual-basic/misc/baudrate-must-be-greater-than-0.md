# <a name="baudrate-must-be-greater-than-0"></a>BaudRate 必須大於 0
為 `BaudRate` 方法指定的 `My.Computer.Ports.OpenSerialPort` 引數必須大於零。  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   將 `BaudRate` 引數的值變更為正數。
