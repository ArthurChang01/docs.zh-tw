# <a name="the-input-path-refers-to-a-file-but-ends-with-a-directory-separator-character"></a>輸入的路徑參考到檔案，但結果卻是目錄分隔符號字元
嘗試執行的作業需要提供檔名，但提供的名稱結果卻是目錄分隔符號字元 (`\`)。 這可能是檔案路徑剖析錯誤的結果。  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請檢查提供的檔案路徑確定是否正確。  
  
-   移除多餘的字元。
