# <a name="39ltnamespacenamegt39-is-ambiguous"></a>&#39;&lt;namespacename&gt;&#39; 模稜兩可
您提供的名稱模稜兩可，因此與另一個名稱發生衝突。 [!INCLUDE[vbprvb](~/includes/vbprvb-md.md)] 編譯器並沒有任何衝突解決規則，您必須自行區分名稱。  
  
 **錯誤 ID︰** BC30554  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   完整限定名稱。
