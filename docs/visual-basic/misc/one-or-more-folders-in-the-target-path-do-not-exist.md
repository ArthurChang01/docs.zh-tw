# 目標路徑中的一或多個資料夾不存在
目標路徑中所提供的一或多個目錄不存在。 這可能是不正確剖析路徑的結果，特別是提供相對路徑時。  
  
### 更正這個錯誤  
  
-   確定您提供的目標路徑正確。
