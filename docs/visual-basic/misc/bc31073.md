# <a name="name1-conflicts-with-a-type-by-the-same-name-defined-in-name2"></a>'\<name1 >' 與衝突\<類型 > 中定義的相同名稱'\<name2 >'
發現名稱與同名的類型衝突。  
  
 **錯誤 ID:** BC31073  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
1.  宣告唯一名稱。  
  
2.  使用方括號括住衝突的成員名稱。
