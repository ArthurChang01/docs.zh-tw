# <a name="ltname1gt-conflicts-with-a-lttypegt-by-the-same-name-defined-in-ltname2gt"></a>'&lt;name1&gt;' function.{0&lt;型別&gt;中所定義的相同名稱 '&lt;name2&gt;'
發現名稱與同名的類型衝突。  
  
 **錯誤 ID:** BC31073  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
1.  宣告唯一名稱。  
  
2.  使用方括號括住衝突的成員名稱。
