# <a name="operator-ltoperatornamegt-is-not-defined-for-types-lttype1gt-and-lttype2gt"></a>運算子 '&lt;運算子&gt;'未定義類型'&lt;type1&gt;'和'&lt;type2&gt;'
您嘗試使用二元運算子，執行對涉及的兩個傳回值程式碼項目無效的作業。  
  
 **錯誤 ID:** BC30452  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請檢查兩個項目，並確定它們相容。
