# <a name="operator-39ltoperatornamegt39-is-not-defined-for-types-39lttype1gt39-and-39lttype2gt39"></a>運算子 &#39;&lt;運算子名稱&gt;&#39; 未定義類型 &#39;&lt;type1&gt;&#39; 和 &#39;&lt;type2&gt;&#39;
您嘗試使用二元運算子，執行對涉及的兩個傳回值程式碼項目無效的作業。  
  
 **錯誤 ID：** BC30452  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請檢查兩個項目，並確定它們相容。
