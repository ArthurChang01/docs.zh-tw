# <a name="operator-operatorname-is-not-defined-for-types-type1-and-type2"></a>運算子 '\<運算子名稱 >' 未定義類型\<type1 >' 和'\<type2 >'
您嘗試使用二元運算子，執行對涉及的兩個傳回值程式碼項目無效的作業。  
  
 **錯誤 ID:** BC30452  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請檢查兩個項目，並確定它們相容。
