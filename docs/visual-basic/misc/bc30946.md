# <a name="stop-request-is-pending"></a>停止要求擱置中
在 Visual Studio 偵錯工具中，運算式指定程序呼叫，但會有停止執行緒的要求。  
  
 偵錯工具不會嘗試在非使用中執行緒上呼叫程序。  
  
 **錯誤 ID：** BC30946  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   可能的話，請判斷執行緒停止要求的來源，以防止它再次發生。  
  
-   終止並重新啟動偵錯。
