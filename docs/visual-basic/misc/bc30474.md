# <a name="ltmethod1gt-cannot-override-ltmethod2gt-because-it-is-a-declare-statement"></a>'&lt;method1&gt;'無法覆寫'&lt;method2&gt;' 因為它是 'Declare' 陳述式
您嘗試要覆寫基底類別名稱以 `Declare` 陳述式宣告的委派。  
  
 **錯誤 ID:** BC30474  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
1.  變更已覆寫的成員，讓它不是 `Declare` 陳述式。  
  
2.  請勿嘗試覆寫這個方法。
