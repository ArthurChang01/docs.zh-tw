# <a name="-recurse"></a><span data-ttu-id="00402-102">-recurse</span><span class="sxs-lookup"><span data-stu-id="00402-102">-recurse</span></span>
<span data-ttu-id="00402-103">編譯指定目錄或專案目錄之所有子目錄中的原始程式碼檔。</span><span class="sxs-lookup"><span data-stu-id="00402-103">Compiles source-code files in all child directories of either the specified directory or the project directory.</span></span>  
  
## <a name="syntax"></a><span data-ttu-id="00402-104">語法</span><span class="sxs-lookup"><span data-stu-id="00402-104">Syntax</span></span>  
  
```console  
-recurse:[dir\]file  
```  
  
## <a name="arguments"></a><span data-ttu-id="00402-105">引數</span><span class="sxs-lookup"><span data-stu-id="00402-105">Arguments</span></span>  
 `dir`  
 <span data-ttu-id="00402-106">選擇性。</span><span class="sxs-lookup"><span data-stu-id="00402-106">Optional.</span></span> <span data-ttu-id="00402-107">您想要開始搜尋的目錄。</span><span class="sxs-lookup"><span data-stu-id="00402-107">The directory in which you want the search to begin.</span></span> <span data-ttu-id="00402-108">如果未指定，則搜尋會從專案目錄開始。</span><span class="sxs-lookup"><span data-stu-id="00402-108">If not specified, the search begins in the project directory.</span></span>  
  
 `file`  
 <span data-ttu-id="00402-109">必要。</span><span class="sxs-lookup"><span data-stu-id="00402-109">Required.</span></span> <span data-ttu-id="00402-110">要搜尋的檔案。</span><span class="sxs-lookup"><span data-stu-id="00402-110">The file(s) to search for.</span></span> <span data-ttu-id="00402-111">允許萬用字元。</span><span class="sxs-lookup"><span data-stu-id="00402-111">Wildcard characters are allowed.</span></span>  
  
## <a name="remarks"></a><span data-ttu-id="00402-112">備註</span><span class="sxs-lookup"><span data-stu-id="00402-112">Remarks</span></span>  
 <span data-ttu-id="00402-113">您可以在檔案名中使用萬用字元，來編譯專案目錄中的所有相符檔案，而`-recurse`不使用。</span><span class="sxs-lookup"><span data-stu-id="00402-113">You can use wildcards in a file name to compile all matching files in the project directory without using `-recurse`.</span></span> <span data-ttu-id="00402-114">如果未指定輸出檔名稱，編譯器會在第一個處理的輸入檔案上以輸出檔案名為基礎。</span><span class="sxs-lookup"><span data-stu-id="00402-114">If no output file name is specified, the compiler bases the output file name on the first input file processed.</span></span> <span data-ttu-id="00402-115">這通常是在按字母順序查看時所編譯的檔案清單中的第一個檔案。</span><span class="sxs-lookup"><span data-stu-id="00402-115">This is generally the first file in the list of files compiled when viewed alphabetically.</span></span> <span data-ttu-id="00402-116">基於這個理由，最好是使用`-out`選項來指定輸出檔。</span><span class="sxs-lookup"><span data-stu-id="00402-116">For this reason, it is best to specify an output file using the `-out` option.</span></span>

 <span data-ttu-id="00402-120">下列命令會編譯目錄中的所有 Visual Basic `Test\ABC`檔案和其底下的任何目錄，然後產生`Test.ABC.dll`。</span><span class="sxs-lookup"><span data-stu-id="00402-120">The following command compiles all Visual Basic files in the `Test\ABC` directory and any directories below it, and then generates `Test.ABC.dll`.</span></span>
