# <a name="help--visual-basic"></a>/help、/? (Visual Basic)
顯示編譯器選項。  
  
## <a name="syntax"></a>語法  
  
```  
/help  
' -or-  
/?  
```  
  
## <a name="remarks"></a>備註  
 如果您在編譯中包含這個選項，會建立任何輸出檔，並不會進行編譯。  
  
> [!NOTE]
>  `/help`選項不可以從[!INCLUDE[vsprvs](../../../csharp/includes/vsprvs_md.md)]開發環境，它才可使用從命令列進行編譯。  
  
## <a name="example"></a>範例  
 下列程式碼會顯示從命令列說明。  
  
```  
vbc /help  
```
