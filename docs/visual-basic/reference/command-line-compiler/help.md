# <a name="-help---visual-basic"></a>-help、-? (Visual Basic)
顯示編譯器選項。  
  
## <a name="syntax"></a>語法  
  
```  
-help  
' -or-  
-?  
```  
  
## <a name="remarks"></a>備註  
 如果您在編譯中納入這個選項，會建立任何輸出檔案，並不會進行編譯。  
  
> [!NOTE]
>  `-help`選項不是從 Visual Studio 開發環境中使用; 只有在從命令列編譯時均可使用。  
  
## <a name="example"></a>範例  
 下列程式碼會顯示從命令列說明。  
  
```  
vbc -help  
```
