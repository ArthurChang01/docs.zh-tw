# <a name="visual-basic-intellisense-code-snippets"></a>Visual Basic IntelliSense 程式碼片段
[!INCLUDE[vbprvb](../../../csharp/programming-guide/concepts/linq/includes/vbprvb_md.md)]包含程式碼程式庫程式碼，稱為 IntelliSense 程式碼片段，即可在您的應用程式中插入幾個數百個部分所組成。 每個程式碼片段會執行完整的程式設計工作，例如建立自訂的例外狀況、 傳送電子郵件，或繪製圓形。 您可以將程式碼片段插入原始程式碼按幾下滑鼠。 您也可以建立您自己的程式碼片段符合商務需求，將它們新增至程式庫，然後在需要時加以使用。 如需詳細資訊，請參閱[程式碼片段](https://docs.microsoft.com/visualstudio/ide/code-snippets)。  
  
## <a name="a-productivity-tool-for-visual-basic-users"></a>Visual Basic 使用者產能工具  
 這些程式碼片段會增加產能減少您花在尋找程式碼範例的時間、 減少了解如何使用熟悉的功能，所需的時間和支援的程式碼重複使用。  
  
 程式碼程式庫支援下列功能︰  
  
-   插入至程式碼片段的其中一個**程式碼編輯器**。  
  
-   建立新的工作，您可以在專案中重複使用。  
  
-   建立新工作加入工作群組與同事分享。  
  
-   編輯工作。  
  
-   協力廠商下載更多工作。  
  
 這些程式碼區塊是可在整個[!INCLUDE[vsprvs](../../../csharp/includes/vsprvs_md.md)]:  
  
-   您可以透過快顯功能表加入**程式碼編輯器**。  
  
-   您可以將 XML 程式碼檔案，從**檔案總管**到原始程式檔。
