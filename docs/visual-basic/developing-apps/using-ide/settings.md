# <a name="visual-basic-settings"></a>Visual Basic 設定
Visual Basic 設定被用來最佳化使用者經驗，以及獲得最大的 Visual Basic 開發人員產能。 它屬於[Visual Studio 中自訂開發設定](http://msdn.microsoft.com/en-us/22c4debb-4e31-47a8-8f19-16f328d7dcd3)功能，可讓您自訂和儲存您的設定，在 Visual Studio 整合式開發環境 (IDE)。  
  
 您也可以連接埠，或重新載入您在其他電腦上的 IDE 設定。 如需詳細資訊，請參閱[How to︰ 電腦之間共用設定](http://msdn.microsoft.com/en-us/1131fb10-35c1-42da-9cd8-91aa3235b882)。

|功能|依預設會顯示？|備註|  
|---|---|---|    
|[類別檢視](https://docs.microsoft.com/visualstudio/ide/viewing-the-structure-of-code)|否|隱藏**隱藏的類型和成員**|  
|[命令視窗](https://docs.microsoft.com/visualstudio/ide/reference/command-window)|否||  
|**動態說明**視窗|否|不會顯示當您按 f1 鍵。|  
|[即時運算視窗](https://docs.microsoft.com/visualstudio/ide/reference/immediate-window)|[是]，偵錯開始時。||  
|[物件瀏覽器](https://docs.microsoft.com/visualstudio/ide/viewing-the-structure-of-code)|否|預設會顯示︰<br /><br /> 命名空間<br />-公用成員<br />-繼承的成員|  
|[輸出視窗](https://docs.microsoft.com/visualstudio/ide/reference/output-window)|是，建置開始時。||  
|[屬性視窗](https://docs.microsoft.com/visualstudio/ide/reference/properties-window)|是||  
|**方案總管**|是||  
|**伺服器總管**|否|顯示資料連接和可用的系統服務。|  
|[起始頁](https://docs.microsoft.com/visualstudio/ide/customizing-the-start-page-for-visual-studio)|是|會出現第一次啟動 IDE。 您可以加以自訂。|  
|**工作清單**|是，當編譯錯誤或警告存在。||  
|[Team Explorer](http://msdn.microsoft.com/library/1372e56c-b34f-42c2-b72c-94b57620c75c)|是|您可以使用 Team 總管 來存取您的 team 專案，並使用 Team Foundation 的應用程式生命週期管理功能。 這些包括版本控制、 組建、 專案管理、 整合式的報告和儀表板。|  
|[工具箱](https://docs.microsoft.com/visualstudio/ide/reference/toolbox)|是|依字母順序顯示控制項和元件。|  
  
 此外，下列項目都具有已啟用，當您選取特定行為**Visual Basic 開發設定**。  
  
## <a name="dialog-boxes"></a>對話方塊  
  
|功能|行為|
|---|---|  
|**新的專案**對話方塊|在電腦上安裝其他平台，則 IDE 會啟用下拉式方塊，可讓您指定目標平台。|  
|[選項對話方塊](https://docs.microsoft.com/visualstudio/ide/reference/options-dialog-box-visual-studio)|Visual Basic 可簡化**選項**頁面。 按一下 **顯示所有設定**核取方塊，請參閱完整的選項清單。|
