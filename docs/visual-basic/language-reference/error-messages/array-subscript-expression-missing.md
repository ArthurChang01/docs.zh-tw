# <a name="array-subscript-expression-missing"></a>遺漏陣列註標運算式
陣列初始化省去了一個或多個定義陣列界限的註標。 例如，此陳述式可能會包含運算式`myArray (5,5,,10)`，它將被排除的第三個註標。  
  
 **錯誤 ID:** BC30306  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   提供遺漏的註標。
