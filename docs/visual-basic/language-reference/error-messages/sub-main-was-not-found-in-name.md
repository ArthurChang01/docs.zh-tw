# <a name="sub-main-was-not-found-in-name"></a>'Sub Main' 找不到在 '\<名稱 >'
`Sub Main` 遺漏，或指定的位置不正確。  
  
 **錯誤 ID:** BC30420  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
1.  提供遺漏的 `Sub Main` 陳述式，或如果存在的話，將它移至程式碼中的適當位置。 如需詳細資訊`Sub Main`，請參閱 < [Visual Basic 中的 Main 程序](../../../visual-basic/programming-guide/program-structure/main-procedure.md)。  
  
2.  指定專案的啟動物件的位置**啟動表單**的方塊**專案設計工具**。
