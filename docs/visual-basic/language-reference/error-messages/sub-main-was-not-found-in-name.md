# <a name="39sub-main39-was-not-found-in-39ltnamegt39"></a>&#39;Sub Main &#39;找不到在 &#39;&lt;名稱&gt;&#39;
`Sub Main` 遺漏，或指定的位置不正確。  
  
 **錯誤 ID:** BC30420  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
1.  提供遺漏的 `Sub Main` 陳述式，或如果存在的話，將它移至程式碼中的適當位置。 如需有關`Sub Main`，請參閱[Main 程序，在 Visual Basic 中](../../../visual-basic/programming-guide/program-structure/main-procedure.md)。  
  
2.  指定專案的啟動物件的位置**啟動表單**方塊**專案設計工具**。
