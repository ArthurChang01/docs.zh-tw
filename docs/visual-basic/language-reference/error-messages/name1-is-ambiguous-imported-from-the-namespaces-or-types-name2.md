# <a name="39ltname1gt39-is-ambiguous-imported-from-the-namespaces-or-types-39ltname2gt39"></a>&#39;&lt;name1&gt;&#39; 模稜兩可，從命名空間或類型 &#39; 匯入&lt;name2&gt;&#39;
您提供的名稱模稜兩可，因此與另一個名稱發生衝突。 [!INCLUDE[vbprvb](~/includes/vbprvb-md.md)] 編譯器並沒有任何衝突解決規則，您必須自行區分名稱。  
  
 **錯誤 ID:** BC30561  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
1.  藉由移除命名空間匯入區分名稱。  
  
2.  完整限定名稱。
