# <a name="39ltname1gt39-is-ambiguous-imported-from-the-namespaces-or-types-39ltname2gt39"></a>'&lt;name1&gt;'是模稜兩可，匯入命名空間或類型從'&lt;name2&gt;'
您提供的名稱模稜兩可，因此與另一個名稱發生衝突。 [!INCLUDE[vbprvb](../../../csharp/programming-guide/concepts/linq/includes/vbprvb_md.md)] 編譯器並沒有任何衝突解決規則，您必須自行區分名稱。  
  
 **錯誤識別碼︰** BC30561  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
1.  釐清藉由移除命名空間匯入的名稱。  
  
2.  完整限定名稱。
