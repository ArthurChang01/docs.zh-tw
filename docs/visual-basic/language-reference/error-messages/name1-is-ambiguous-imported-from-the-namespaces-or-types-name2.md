# <a name="name1-is-ambiguous-imported-from-the-namespaces-or-types-name2"></a>'\<name1 >' 模稜兩可，從命名空間或類型的匯入\<name2 >'
您提供的名稱模稜兩可，因此與另一個名稱發生衝突。 Visual Basic 編譯器並沒有任何衝突解決規則中;您必須自行區分名稱。  
  
 **錯誤 ID:** BC30561  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
1.  藉由移除命名空間匯入，釐清的名稱。  
  
2.  完整限定名稱。
