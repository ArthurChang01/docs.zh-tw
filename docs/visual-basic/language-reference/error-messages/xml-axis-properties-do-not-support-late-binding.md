# <a name="xml-axis-properties-do-not-support-late-binding"></a>XML 軸屬性不支援晚期繫結
不具類型的物件已參考 XML 軸屬性。  
  
 **錯誤 ID:** BC31168  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請確定物件是一個強式型別<xref:System.Xml.Linq.XElement>之前參考 XML 軸屬性的物件。
