# <a name="unable-to-find-required-file-filename"></a>找不到必要的檔案 '\<檔案名稱 >'
Visual Studio 所需要的某個檔案已經遺失或損毀。  
  
 **錯誤 ID:** BC30655  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 請重新安裝 Visual Studio。
