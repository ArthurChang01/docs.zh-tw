# <a name="string-constants-must-end-with-a-double-quote"></a>字串常數必須以雙引號結尾
開頭與結尾加上引號的字串常數必須。  
  
 **ErrorID:** BC30648  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請確定字串常值結尾引號 （"）。 如果您貼上的其他文字編輯器中的值，請確定貼上的字元是有效的雙引號，而不是其中一個類似，例如 「 智慧 」 或 「 大 」 的引號字元 (") 或兩個單引號 （'）。
