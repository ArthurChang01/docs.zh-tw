# <a name="statement-is-not-valid-in-a-namespace"></a>命名空間中的陳述式無效
陳述式不能出現在命名空間層級。 唯一允許的命名空間層級的宣告是模組、 介面、 類別、 委派、 列舉和結構宣告。  
  
 **錯誤 ID:** BC30001  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   陳述式移至模組、 類別、 介面、 結構、 列舉或委派定義中的位置。
