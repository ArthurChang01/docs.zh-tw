# <a name="array-subscript-expression-missing"></a>遺漏陣列註標運算式
陣列初始化會離開定義陣列界限的一或多個注標。 例如，語句可能會包含運算式 `myArray (5,5,,10)`，這會留下第三個注標。  
  
 **錯誤識別碼：** BC30306  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 提供遺漏的注標。
