# <a name="internal-error-happened-at-ltlocationgt"></a>內部錯誤發生了&lt;位置&gt;
發生內部錯誤。 錯誤訊息中包含它發生的行。  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請確定這個錯誤，並非由`Error`陳述式或`Raise`方法; 如果它不是，請連絡 Microsoft 產品支援服務，並回報出現訊息。
