# <a name="internal-error-happened-at-location"></a>\<位置發生內部錯誤 >
發生內部錯誤。 錯誤訊息中會包含發生它的行。  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 請確定這個錯誤不是由 `Error` 語句或 `Raise` 方法所產生。如果不是，請聯絡 Microsoft 產品支援服務，以回報出現訊息的條件。
