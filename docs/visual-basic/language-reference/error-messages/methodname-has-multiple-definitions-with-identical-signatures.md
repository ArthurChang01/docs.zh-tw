# <a name="39ltmethodnamegt39-has-multiple-definitions-with-identical-signatures"></a>&#39;&lt;methodname&gt; &#39;有多個具相同簽章的定義
A`Function`或`Sub`程序宣告為上一個宣告中使用相同的程序名稱和引數清單。 其中一個可能的原因是嘗試多載，原始的程序。 多載程序必須有不同的引數清單。  
  
 **錯誤 ID:** BC30269  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   變更的程序名稱或引數清單中，或移除重複的宣告。
