# <a name="39class39-statement-must-end-with-a-matching-39end-class39"></a>&#39;類別 &#39;陳述式的結尾必須符合 &#39;End Class &#39;
`Class`用來起始`Class`區塊，因此它只能出現在區塊中，搭配相對應的開頭`End Class`陳述式結束區塊。 您有多餘`Class`陳述式，或您有未結束您`Class`區塊`End Class`。  
  
 **錯誤 ID:** BC30481  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   找到並移除不必要`Class`陳述式。  
  
-   結束`Class`搭配相對應的區塊`End Class`。
