    End Sub

 <span data-ttu-id="43920-105">在上述範例中，`x = 10` 指派語句會產生這個錯誤訊息。</span><span class="sxs-lookup"><span data-stu-id="43920-105">In the preceding example, the assignment statement `x = 10` generates this error message.</span></span> <span data-ttu-id="43920-106">這是因為共用程式嘗試存取執行個體變數。</span><span class="sxs-lookup"><span data-stu-id="43920-106">This is because a shared procedure is attempting to access an instance variable.</span></span>  
  
 <span data-ttu-id="43920-107">變數 `x` 是實例成員，因為它並未宣告為[Shared](../modifiers/shared.md)。</span><span class="sxs-lookup"><span data-stu-id="43920-107">The variable `x` is an instance member because it is not declared as [Shared](../modifiers/shared.md).</span></span> <span data-ttu-id="43920-108">類別的每個實例 `Sample` 都包含自己的個別變數 `x`。</span><span class="sxs-lookup"><span data-stu-id="43920-108">Each instance of class `Sample` contains its own individual variable `x`.</span></span> <span data-ttu-id="43920-109">當某個實例設定或變更 `x`的值時，不會影響任何其他實例中 `x` 的值。</span><span class="sxs-lookup"><span data-stu-id="43920-109">When one instance sets or changes the value of `x`, it does not affect the value of `x` in any other instance.</span></span>
  
 <span data-ttu-id="43920-110">不過，`SetX` 的程式會在類別 `Sample`的所有實例之間 `Shared`。</span><span class="sxs-lookup"><span data-stu-id="43920-110">However, the procedure `SetX` is `Shared` among all instances of class `Sample`.</span></span> <span data-ttu-id="43920-111">這表示它不會與類別的任何一個實例相關聯，而是獨立于個別實例運作。</span><span class="sxs-lookup"><span data-stu-id="43920-111">This means it is not associated with any one instance of the class, but rather operates independently of individual instances.</span></span> <span data-ttu-id="43920-112">因為它沒有與特定實例的連接，所以 `setX` 無法存取執行個體變數。</span><span class="sxs-lookup"><span data-stu-id="43920-112">Because it has no connection with a particular instance, `setX` cannot access an instance variable.</span></span> <span data-ttu-id="43920-113">它必須只在 `Shared` 變數上運作。</span><span class="sxs-lookup"><span data-stu-id="43920-113">It must operate only on `Shared` variables.</span></span> <span data-ttu-id="43920-114">當 `SetX` 設定或變更共用變數的值時，該新值可供類別的所有實例使用。</span><span class="sxs-lookup"><span data-stu-id="43920-114">When `SetX` sets or changes the value of a shared variable, that new value is available to all instances of the class.</span></span>
