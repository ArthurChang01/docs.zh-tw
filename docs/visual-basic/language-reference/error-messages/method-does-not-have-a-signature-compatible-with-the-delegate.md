# <a name="method-does-not-have-a-signature-compatible-with-the-delegate"></a>方法沒有與委派相容的簽章
方法的簽章與您嘗試使用的委派之間沒有相容性。 `Delegate` 陳述式定義委派類別的參數類型和傳回類型。 任何具有相容類型的參數相符且傳回類型可以用來建立這個委派類型的執行個體的程序。  
  
 **錯誤 ID**:BC36563
