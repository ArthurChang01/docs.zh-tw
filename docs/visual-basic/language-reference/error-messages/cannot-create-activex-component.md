# <a name="cannot-create-activex-component"></a>無法建立 ActiveX 元件
您嘗試在設計階段時將 ActiveX 控制項放至於表單上，或將表單加入其上有 ActiveX 控制項的專案，但登錄中找不到相關資訊。  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   登錄中的資訊可能已刪除或損毀。 重新安裝 ActiveX 控制項或連絡控制項廠商。
