# <a name="property-not-found"></a>找不到屬性
此物件不支援指定的屬性。  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
1. 檢查屬性名稱的拼字。  
  
2. 檢查物件的文件，請確定您未嘗試存取類似 「 文字 」 屬性，當物件實際上支援 「 標題 」 或類似名稱的屬性。
