# <a name="property-not-found"></a>找不到屬性
此物件不支援指定的屬性。  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
1.  請檢查屬性名稱的拼字。  
  
2.  請檢查物件的文件，請確定您未嘗試存取類似 「 文字 」 屬性，當物件實際上會支援 「 標題 」 或類似的具名屬性。
