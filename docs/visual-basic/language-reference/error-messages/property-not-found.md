# Property not found
[!INCLUDE[vs2017banner](../../../visual-basic/includes/vs2017banner.md)]

此物件不支援指定的屬性 \(Property\)。  
  
### 若要更正這個錯誤  
  
1.  請檢查屬性 \(Property\) 名稱的拼字。  
  
2.  請查閱物件的參考文件，以確定您並未在物件實際支援「標題」或其他類似具名屬性 \(Property\) 的情況下，嘗試存取如「文字」等屬性 \(Property\)。
