# <a name="property-array-index-is-not-valid"></a>無效的屬性陣列索引
提供的值不是有效的屬性陣列索引。  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   檢查元件的文件，藉此確定您的索引已指定之屬性的有效範圍內。
