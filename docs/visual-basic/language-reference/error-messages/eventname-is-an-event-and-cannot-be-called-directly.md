# <a name="39lteventnamegt39-is-an-event-and-cannot-be-called-directly"></a>&#39;&lt;eventname&gt;&#39; 事件，且不能直接呼叫
' <`eventname`>' 是個事件，並因此不能直接呼叫。 使用`RaiseEvent`陳述式來引發事件。  
  
 程序呼叫中指定事件的程序名稱。 事件處理常式是程序，但事件本身的信號裝置，必須引發和處理。  
  
 **錯誤 ID:** BC32022  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
1.  使用`RaiseEvent`對事件發出信號和叫用程序或處理程序的陳述式。
