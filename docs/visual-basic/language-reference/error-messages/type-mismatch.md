# <a name="type-mismatch-visual-basic"></a>類型不符 (Visual Basic)
您嘗試將值轉換成另一種類型不是有效的方式。  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
1.  請檢查並確定它是有效的指派。  
  
2.  請確定您未通過預期的單一屬性或值的程序物件。  
  
3.  請確定您未使用的模組或專案名稱必須是運算式。
