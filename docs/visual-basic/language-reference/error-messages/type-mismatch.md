# <a name="type-mismatch-visual-basic"></a>類型不符 (Visual Basic)
您嘗試以不正確方式，將值轉換成另一種類型。  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
1. 請檢查指派以確定它是有效的。  
  
2. 請確定您未將物件傳遞給需要單一屬性或值的程式。  
  
3. 請確定您未使用預期運算式的模組或專案名稱。
