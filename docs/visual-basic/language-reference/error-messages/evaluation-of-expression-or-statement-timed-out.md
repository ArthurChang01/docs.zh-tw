# Evaluation of expression or statement timed out
[!INCLUDE[vs2017banner](../../../visual-basic/includes/vs2017banner.md)]

運算式評估未及時完成。  
  
 **錯誤 ID：**BC30722  
  
### 若要更正這個錯誤  
  
1.  請確認輸入的程式碼是否正確。  
  
2.  請簡化運算式，使其花費較短的執行時間。
