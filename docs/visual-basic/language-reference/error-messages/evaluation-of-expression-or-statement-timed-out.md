# <a name="evaluation-of-expression-or-statement-timed-out"></a>運算式或陳述式評估已經逾時
運算式的評估未及時完成。  
  
 **錯誤 ID:** BC30722  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
1.  請確認輸入的程式碼正確無誤。  
  
2.  請簡化運算式，使它花較少的時間執行。
