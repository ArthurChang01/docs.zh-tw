# <a name="device-io-error"></a>裝置 I/O 錯誤
雖然您的程式所使用的裝置，例如印表機或磁碟機，就會發生輸入或輸出錯誤。  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 請確定裝置運作正常，，然後再重試此作業。
