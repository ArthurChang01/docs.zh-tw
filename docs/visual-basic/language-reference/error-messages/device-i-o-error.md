# <a name="device-io-error"></a>裝置 I/O 錯誤
您的程式已使用磁碟機或印表機等裝置時，就會發生輸入或輸出錯誤。  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請確定裝置運作正常，，然後再重試操作。
