# <a name="xml-entity-references-are-not-supported"></a>不支援 XML 實體參考
實體參考 (例如， `©`) 未定義在 XML 1.0 規格會包含做為 XML 常值的值。 只有`&`， `"`， `<`， `>`，和`'`支援 XML 實體參考 XML 常值中。  
  
 **錯誤 ID:** BC31180  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   移除不支援的實體參考。
