# <a name="cant-open-filename-for-writing"></a>無法開啟 '\<檔案名稱 >' 進行寫入
無法開啟指定的檔案進行寫入，可能因為它已經開啟。  
  
 **錯誤 ID:** BC2012  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
1.  關閉檔案，並重新開啟它。  
  
2.  檢查檔案的權限。
