# <a name="overflow-visual-basic-error"></a>溢位 (Visual Basic 錯誤)
常值代表的值超出資料類型，它被指派的限制。  
  
 **錯誤 ID:** BC30036  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   目標資料類型，請重寫常值，以符合該範圍，請參閱值的範圍。
