# <a name="overflow-visual-basic-error"></a>溢位 (Visual Basic 錯誤)
常值表示它已被指派的資料類型的限制以外的值。  
  
 **錯誤 ID:** BC30036  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   目標資料類型，請重寫常值以符合該範圍，請參閱位於值範圍。
