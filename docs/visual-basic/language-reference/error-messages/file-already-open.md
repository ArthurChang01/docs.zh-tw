# <a name="file-already-open"></a>檔案已經開啟
有時檔案必須先關閉另一個之前`FileOpen`或進行其他作業。 可能導致本錯誤的原因包括：  
  
-   循序輸出模式`FileOpen`已經開啟的檔案執行作業  
  
-   陳述式參考到開啟的檔案。  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
1.  關閉檔案再執行陳述式。
