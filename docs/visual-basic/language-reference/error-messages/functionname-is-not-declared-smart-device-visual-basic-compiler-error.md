# <a name="39ltfunctionnamegt39-is-not-declared-smart-devicevisual-basic-compiler-error"></a>&#39;&lt;functionname&gt;&#39; 不宣告 （智慧型裝置 /visual Basic 編譯器錯誤）
<`functionname`> 未宣告。 檔案 I/O 功能通常是在 `Microsoft.VisualBasic` 命名空間中，但是目標 .NET Compact Framework 版本並不支援它。  
  
 **錯誤 ID:** BC30766  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   執行檔案作業中所定義的函式`System.IO`命名空間。
