# <a name="some-subkeys-cannot-be-deleted"></a>無法刪除某些子機碼
嘗試刪除登錄機碼，但是作業失敗，因為無法刪除某些子機碼。 這通常是因為缺乏權限。  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 請確定您有足夠的權限，無法刪除指定的子機碼。
