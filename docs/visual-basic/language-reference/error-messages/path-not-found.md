# <a name="path-not-found"></a>找不到路徑
檔案存取或磁碟存取作業時，作業系統已找不到指定的路徑。 檔案的路徑包括磁碟機規格加上目錄和子目錄的必須周遊以找出檔案。 路徑可為相對或絕對路徑。  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請確認，然後重新指定路徑。
