# <a name="path-not-found"></a>找不到路徑
存取檔案或磁碟存取作業時，作業系統是找不到指定的路徑。 檔案的路徑包含磁碟機規格，加上的目錄和必須周遊以找出檔案的子目錄。 路徑可為相對或絕對路徑。  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 請確認，然後重新指定路徑。
