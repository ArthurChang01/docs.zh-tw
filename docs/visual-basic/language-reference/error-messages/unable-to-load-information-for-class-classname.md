# <a name="unable-to-load-information-for-class-classname"></a>無法載入類別 '\<classname > ' 的資訊
對無法使用的類別進行參考。  
  
 **錯誤識別碼：** BC30712  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
1. 請確認已定義類別，且您的名稱拼寫正確。  
  
2. 嘗試存取模組中所宣告的其中一個成員。 在某些情況下，偵錯環境找不到成員，因為尚未載入宣告它們的模組。
