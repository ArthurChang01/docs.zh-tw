# <a name="unable-to-load-information-for-class-classname"></a>無法載入類別資訊\<類別名稱 >'
參考了未提供的類別。  
  
 **錯誤 ID:** BC30712  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
1.  請確認已定義類別和名稱的拼字正確。  
  
2.  嘗試存取模組中所宣告的其中一個成員。 在某些情況下，偵錯環境找不到成員，因為尚未載入宣告它們的模組。
