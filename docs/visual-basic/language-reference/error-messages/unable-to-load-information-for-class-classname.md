# <a name="unable-to-load-information-for-class-39ltclassnamegt39"></a>無法載入類別的資訊&#39; &lt;classname&gt;&#39;
參考了未提供的類別。  
  
 **錯誤 ID:** BC30712  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
1.  請確認已定義類別和名稱的拼字正確。  
  
2.  嘗試存取模組中所宣告的其中一個成員。 在某些情況下，偵錯環境找不到成員，因為尚未載入宣告它們的模組。
