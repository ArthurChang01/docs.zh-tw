# File not found (Visual Basic Run-Time Error)
[!INCLUDE[vs2017banner](../../../visual-basic/includes/vs2017banner.md)]

在指定位置找不到檔案。  錯誤可能由下列原因所致：  
  
-   陳述式所參考的檔案不存在。  
  
-   已嘗試在動態連結程式庫 \(DLL\) 中呼叫程序，但是找不到 `Declare` 陳述式的 `Lib` 子句中所指定的程式庫。  
  
-   您嘗試開啟不存在的專案或載入不存在的文字檔。  
  
### 若要更正這個錯誤  
  
1.  請檢查檔案名稱的拼字和路徑描述。
