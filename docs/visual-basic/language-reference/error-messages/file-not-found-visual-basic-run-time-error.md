# <a name="file-not-found-visual-basic-run-time-error"></a>找不到檔案 (Visual Basic 執行階段錯誤)
指定 where，找不到檔案。 此錯誤有下列可能原因：  
  
-   陳述式參考到不存在的檔案。  
  
-   嘗試呼叫的程序中的動態連結程式庫 (DLL)，但在指定的程式庫`Lib`子句`Declare`找不到陳述式。  
  
-   您嘗試開啟專案或載入文字檔案不存在。  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
1.  請檢查拼字的檔案名稱和路徑規格。
