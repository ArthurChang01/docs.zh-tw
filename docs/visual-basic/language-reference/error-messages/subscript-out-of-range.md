# <a name="subscript-out-of-range-visual-basic"></a>陣列索引超出範圍 (Visual Basic)
陣列註標不正確，因為它超出允許的範圍。 維度的最低註標的值永遠為 0，而最高註標的值由`GetUpperBound`該維度的方法。  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   變更註標，所以它會落在有效範圍內。
