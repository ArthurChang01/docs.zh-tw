# <a name="identifier-expected"></a>必須是識別項
不是可辨識的宣告項目名稱的程式設計項目就會發生內容需要的項目名稱。 可能的原因是屬性已指定的位置以外的陳述式的開頭。  
  
 **錯誤識別碼︰** BC30203  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請確認該陳述式中的任何屬性全部放置在開頭處。  
  
-   驗證陳述式中的所有項目名稱拼寫正確。
