# <a name="identifier-expected"></a>必須是識別項
不是可辨識的宣告項目名稱的程式設計項目就會發生在內容需要的項目名稱。 一個可能的原因是屬性已指定的位置以外的陳述式開頭。  
  
 **錯誤 ID:** BC30203  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請確認該陳述式中的任何屬性全部放置在開頭。  
  
-   請確認陳述式中的所有項目名稱拼寫正確。
