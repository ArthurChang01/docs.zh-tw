# Expression does not produce a value
[!INCLUDE[vs2017banner](../../../visual-basic/includes/vs2017banner.md)]

您嘗試在產生值的內容中使用不產生值的運算式，例如在需要 `Function` 的內容中呼叫 `Sub`。  
  
 **錯誤 ID：**BC30491  
  
### 若要更正這個錯誤  
  
-   將運算式變更為產生值的運算式。
