# <a name="expression-does-not-produce-a-value"></a>運算式沒有產生值
您嘗試使用不會產生值產生內容，例如呼叫中的值的運算式`Sub`在內容中其中`Function`預期。  
  
 **錯誤 ID:** BC30491  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   將運算式變更為產生的值。
