# <a name="procedure-call-or-argument-is-not-valid-visual-basic"></a>無效的程序呼叫或引數 (Visual Basic)
無法完成呼叫的某些部分。  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   檢查允許的範圍引數以確定未安排超過允許的值。
