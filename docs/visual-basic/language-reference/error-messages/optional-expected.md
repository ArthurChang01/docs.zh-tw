# <a name="optional-expected"></a>必須是 'Optional'
在程序宣告中的選擇性引數的後面是必要的引數。 下列選擇性引數的每個引數也必須是選擇性的。  
  
 **錯誤 ID:** BC30202  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
1.  如果引數是必要，將它移至引數清單中前面的第一個選擇性引數。  
  
2.  引數是選擇性，如果使用`Optional`關鍵字。
