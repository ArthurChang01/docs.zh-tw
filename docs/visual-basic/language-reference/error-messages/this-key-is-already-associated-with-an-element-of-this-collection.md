# <a name="this-key-is-already-associated-with-an-element-of-this-collection"></a>此索引鍵已經和此集合的項目建立關聯性
指定的索引鍵的集合成員已經用來識別集合的另一個成員。 金鑰是字串中指定`Add`方法可唯一識別特定集合的成員。  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 這個成員使用不同的金鑰。
