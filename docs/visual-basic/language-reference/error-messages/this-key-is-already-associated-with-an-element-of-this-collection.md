# <a name="this-key-is-already-associated-with-an-element-of-this-collection"></a>此索引鍵已經和此集合的項目建立關聯性
指定的索引鍵的集合成員已識別的另一個集合的成員。 索引鍵是字串中指定`Add`方法，以唯一識別特定集合的成員。  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   此成員使用不同的索引鍵。
