# <a name="expression-too-complex"></a>運算式太複雜
浮點運算式中包含太多巢狀的子運算式。  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 將運算式分割為多個獨立的運算式，必要時，若要避免錯誤發生。
