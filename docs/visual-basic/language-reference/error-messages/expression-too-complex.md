# <a name="expression-too-complex"></a>運算式太複雜
浮點運算式中包含太多巢狀的子運算式。  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   若要避免錯誤發生視分成許多個別的運算式的運算式。
