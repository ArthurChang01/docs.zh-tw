# <a name="unable-to-write-output-to-memory"></a>無法將輸出寫入至記憶體
將輸出寫至記憶體時發生問題。  
  
 **錯誤 ID:** BC31020  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
1.  再次編譯程式，看看錯誤是否重複發生。  
  
2.  如果錯誤繼續發生，請儲存您的工作，然後重新啟動 [!INCLUDE[vsprvs](~/includes/vsprvs-md.md)]。  
  
3.  如果問題重複發生，請重新安裝 [!INCLUDE[vbprvb](~/includes/vbprvb-md.md)]。  
  
4.  如果錯誤在重新安裝之後持續發生，請通知 Microsoft 產品支援服務。
