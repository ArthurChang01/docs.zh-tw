# <a name="constant-expression-not-representable-in-type-39lttypenamegt39"></a>無法顯示在型別常數運算式&#39;&lt;類型名稱&gt;&#39;
您嘗試評估常數，不符合目標類型，通常是因為它超出範圍。  
  
 **錯誤 ID:** BC30439  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
1.  將目標類型變更為可處理常數的其中一個。
