# <a name="constant-expression-not-representable-in-type-39lttypenamegt39"></a>型別中無法顯示常數運算式&#39; &lt;typename&gt;&#39;
您嘗試評估常數所無法容納到目標類型，通常是因為它超出範圍。  
  
 **錯誤 ID:** BC30439  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
1.  將目標類型變更為可以處理的常數。
