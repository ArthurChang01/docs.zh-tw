# <a name="object-required-visual-basic"></a>必須提供物件 (Visual Basic)
屬性和方法的參考通常會需要明確的物件識別碼。 以下是這種情況。  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
1.  請檢查參考的物件屬性或方法具有有效的物件限定詞。 如果您沒有提供的話，請指定物件限定詞。  
  
2.  物件限定詞的拼字檢查，並確定該物件為的程式在其中您要參考它的組件中可見。  
  
3.  如果主應用程式提供路徑**開啟舊檔**命令，檢查引數是否正確。  
  
4.  請檢查物件的文件，並確定動作有效。
