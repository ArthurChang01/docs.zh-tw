# <a name="too-many-files"></a>檔案太多
多個檔案已在根目錄中建立比作業系統許可，或是已開啟多個檔案中指定的數目比**檔案 =**中您的組態設定。SYS 檔案。  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
1.  如果您的程式已開啟、 關閉或儲存檔案的根目錄中，變更您的程式，使其使用子目錄。  
  
2.  中指定的檔案數目增加您**檔案 =**中您的組態設定。SYS 檔案，然後重新啟動電腦。
