# <a name="property-or-method-not-found"></a>找不到屬性或方法
未定義參考的物件方法或物件屬性。  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   您可能拼錯的物件名稱。 若要查看哪些屬性和方法所定義的物件，顯示 物件瀏覽器。 選取適當的物件程式庫，以檢視可用的屬性和方法的清單。
