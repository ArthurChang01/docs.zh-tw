# <a name="property-or-method-not-found"></a>找不到屬性或方法
未定義的參考的物件的方法或物件屬性。  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   您可能會有拼字錯誤的物件名稱。 若要查看物件所定義的屬性和方法，顯示 物件瀏覽器。 選取適當的物件程式庫，以檢視可用的屬性及方法的清單。
