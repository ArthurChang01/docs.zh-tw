# <a name="unable-to-create-strong-named-assembly-from-key-file-filename-error"></a>無法從金鑰檔建立強式名稱組件 '\<檔案名稱 >':\<錯誤 >
無法從指定的金鑰檔建立的強式名稱組件。  
  
 **錯誤 ID:** BC31026  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
1.  請確認已指定正確的金鑰檔，而且不會鎖定由其他應用程式。
