無法從指定的金鑰檔建立強式名稱組件。  
  
 **錯誤 ID:** BC31026  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
1.  請確認已指定正確的金鑰檔，且不會鎖定另一個應用程式。
