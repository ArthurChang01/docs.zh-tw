# <a name="unable-to-create-strong-named-assembly-from-key-file-39ltfilenamegt39-lterrorgt"></a>無法建立強式名稱組件金鑰檔案從&#39;&lt;檔名&gt;&#39;:&lt;錯誤&gt;
無法從指定的金鑰檔建立的強式名稱組件。  
  
 **錯誤 ID:** BC31026  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
1.  請確認已指定正確的金鑰檔，而且不會鎖定由其他應用程式。
