# <a name="queries-visual-basic"></a>查詢 (Visual Basic)
[!INCLUDE[vbprvb](~/includes/vbprvb-md.md)]可讓您建立[!INCLUDE[vbteclinqext](~/includes/vbteclinqext-md.md)]您的程式碼中的運算式。  
  
## <a name="in-this-section"></a>本章節內容  
 [Aggregate 子句](../../../visual-basic/language-reference/queries/aggregate-clause.md)  
 描述`Aggregate`子句，可套用至集合的一個或多個彙總函式。  
  
 [Distinct 子句](../../../visual-basic/language-reference/queries/distinct-clause.md)  
 描述`Distinct`子句，以消除重複的值，查詢結果中目前的範圍變數的值限制。  
  
 [From 子句](../../../visual-basic/language-reference/queries/from-clause.md)  
 描述`From`子句，指定的集合和查詢的範圍變數。  
  
 [Group By 子句](../../../visual-basic/language-reference/queries/group-by-clause.md)  
 描述`Group By`子句，讓群組查詢結果的項目，而且可用來將彙總函式套用至每個群組。  
  
 [Group Join 子句](../../../visual-basic/language-reference/queries/group-join-clause.md)  
 描述`Group Join`子句，其中兩個將集合合併成單一階層式實體集合。  
  
 [Join 子句](../../../visual-basic/language-reference/queries/join-clause.md)  
 描述`Join`子句，結合成單一集合的兩個集合。  
  
 [Let 子句](../../../visual-basic/language-reference/queries/let-clause.md)  
 描述`Let`子句，計算值，並將它指派給新的變數，在查詢中。  
  
 [Order By 子句](../../../visual-basic/language-reference/queries/order-by-clause.md)  
 描述`Order By`子句，指定查詢中的資料行的排序次序。  
  
 [Select 子句](../../../visual-basic/language-reference/queries/select-clause.md)  
 描述`Select`子句，會宣告一組查詢的範圍變數。  
  
 [Skip 子句](../../../visual-basic/language-reference/queries/skip-clause.md)  
 描述`Skip`子句，可略過指定的集合中的項目數目，然後傳回其餘項目。  
  
 [Skip While 子句](../../../visual-basic/language-reference/queries/skip-while-clause.md)  
 描述`Skip While`子句，略過集合中的項目，只要指定的條件是`true`然後傳回其餘項目。  
  
 [Take 子句](../../../visual-basic/language-reference/queries/take-clause.md)  
 描述`Take`子句，從集合開頭傳回指定的連續的項目數。  
  
 [Take While 子句](../../../visual-basic/language-reference/queries/take-while-clause.md)  
 描述`Take While`子句，其中包含集合中的項目，只要指定的條件是`true`並略過其餘項目。  
  
 [Where 子句](../../../visual-basic/language-reference/queries/where-clause.md)  
 描述`Where`子句，指定查詢的篩選條件。
