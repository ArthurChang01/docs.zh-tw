# <a name="properties-visual-basic"></a>屬性 (Visual Basic)
此頁面會列出 Visual Basic 模組的成員的屬性。 其他屬於 Visual Basic 的特定物件的屬性會列在[物件](../../visual-basic/language-reference/objects/index.md)。  
  
## <a name="visual-basic-properties"></a>Visual Basic 屬性  
  
|||  
|---|---|  
|<xref:Microsoft.VisualBasic.DateAndTime.DateString%2A>|傳回或設定`String`值，表示目前的日期，根據您的系統。|  
|<xref:Microsoft.VisualBasic.DateAndTime.Now%2A>|傳回`Date`值，其中包含目前的日期和時間，根據您的系統。|  
|<xref:Microsoft.VisualBasic.Globals.ScriptEngine%2A>|傳回`String`表示現用執行階段使用。|  
|<xref:Microsoft.VisualBasic.Globals.ScriptEngineBuildVersion%2A>|傳回`Integer`包含現用執行階段使用的組建版本號碼。|  
|<xref:Microsoft.VisualBasic.Globals.ScriptEngineMajorVersion%2A>|傳回`Integer`包含目前使用中的執行階段的主要版本號碼。|  
|<xref:Microsoft.VisualBasic.Globals.ScriptEngineMinorVersion%2A>|傳回`Integer`包含目前使用中的執行階段的次要版本號碼。|  
|<xref:Microsoft.VisualBasic.DateAndTime.TimeOfDay%2A>|傳回或設定`Date`值，其中包含根據您的系統的目前時間。|  
|<xref:Microsoft.VisualBasic.DateAndTime.Timer%2A>|傳回`Double`午夜算所經過的秒數表示。|  
|<xref:Microsoft.VisualBasic.DateAndTime.TimeString%2A>|傳回或設定`String`值，表示一天，根據您的系統目前的時間。|  
|<xref:Microsoft.VisualBasic.DateAndTime.Today%2A>|傳回或設定`Date`值，其中包含根據您的系統目前的日期。|
