# <a name="attributes-visual-basic"></a>屬性 (Visual Basic)
[!INCLUDE[vbprvb](~/includes/vbprvb-md.md)]提供數個屬性可讓物件交互操作與 unmanaged 程式碼，並可讓模組名稱不能存取的模組成員的一個屬性。 下表列出所使用的屬性[!INCLUDE[vbprvb](~/includes/vbprvb-md.md)]。  
  
|||  
|---|---|  
|<xref:Microsoft.VisualBasic.ComClassAttribute>|指示編譯器將可讓類別公開為 COM 物件的中繼資料。|  
|<xref:Microsoft.VisualBasic.HideModuleNameAttribute>|允許該模組成員僅限定性條件所需的模組存取。|  
|<xref:Microsoft.VisualBasic.VBFixedArrayAttribute>|表示結構或非區域變數中的陣列應該視為固定長度的陣列。|  
|<xref:Microsoft.VisualBasic.VBFixedStringAttribute>|表示應該視為字串，如同固定長度。|
