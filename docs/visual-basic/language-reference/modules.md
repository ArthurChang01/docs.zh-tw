# <a name="modules-visual-basic"></a>模組 (Visual Basic)
[!INCLUDE[vbprvb](../../csharp/programming-guide/concepts/linq/includes/vbprvb_md.md)]提供數個模組，讓您簡化一般工作，包括管理字串，執行數學計算，取得系統資訊、 執行檔案和目錄的作業，在您的程式碼中等等。 下表列出所提供的模組[!INCLUDE[vbprvb](../../csharp/programming-guide/concepts/linq/includes/vbprvb_md.md)]。  
  
|||  
|---|---|  
|<xref:Microsoft.VisualBasic.Constants></xref:Microsoft.VisualBasic.Constants>|包含其他的常數。 這些常數可以在程式碼中任何地方使用。|  
|<xref:Microsoft.VisualBasic.ControlChars></xref:Microsoft.VisualBasic.ControlChars>|包含常數控制字元來列印及顯示文字。|  
|<xref:Microsoft.VisualBasic.Conversion></xref:Microsoft.VisualBasic.Conversion>|包含數字字串、 數字的字串和一個資料輸入到另一個十進位數字轉換成其他的基底的成員。|  
|<xref:Microsoft.VisualBasic.DateAndTime></xref:Microsoft.VisualBasic.DateAndTime>|包含成員，取得目前的日期或時間、 執行日期計算傳回日期或時間，設定日期或時間，或處理程序的持續時間的時間。|  
|<xref:Microsoft.VisualBasic.ErrObject></xref:Microsoft.VisualBasic.ErrObject>|包含執行階段錯誤和方法引發，或清除錯誤的相關資訊。|  
|<xref:Microsoft.VisualBasic.FileSystem></xref:Microsoft.VisualBasic.FileSystem>|包含執行檔案、 目錄或資料夾，以及系統作業的成員。|  
|<xref:Microsoft.VisualBasic.Financial></xref:Microsoft.VisualBasic.Financial>|包含用來執行財務計算程序。|  
|<xref:Microsoft.VisualBasic.Globals></xref:Microsoft.VisualBasic.Globals>|包含目前的指令碼引擎版本的相關資訊。|  
|<xref:Microsoft.VisualBasic.Information></xref:Microsoft.VisualBasic.Information>|包含的成員，傳回、 測試，或確認資訊，例如陣列的大小、 類型名稱等等。|  
|<xref:Microsoft.VisualBasic.Interaction></xref:Microsoft.VisualBasic.Interaction>|包含與物件、 應用程式和系統互動的成員。|  
|<xref:Microsoft.VisualBasic.Strings></xref:Microsoft.VisualBasic.Strings>|包含執行字串作業，例如重新格式化字串，搜尋字串，取得字串，長度等等的成員。|  
|<xref:Microsoft.VisualBasic.VBMath></xref:Microsoft.VisualBasic.VBMath>|包含成員執行數學運算。|
