# <a name="module-ltkeywordgt-visual-basic"></a>模組&lt;關鍵字&gt;(Visual Basic)
指定的原始程式檔開頭的屬性會套用到目前的組件模組。  
  
## <a name="remarks"></a>備註  
 許多屬性適用於個別的程式設計元素，例如類別或屬性。 您套用這類屬性所附加的屬性區塊角括號內 (`< >`)，直接至宣告陳述式。  
  
 如果屬性屬於下列項目不僅目前組件模組，您將屬性區塊放在原始程式檔的開頭，然後找出具有屬性`Module`關鍵字。 它會套用至整個組件，如果您使用[組件](../../../visual-basic/language-reference/modifiers/assembly.md)關鍵字。  
  
 `Module`修飾詞不相同[Module 陳述式](../../../visual-basic/language-reference/statements/module-statement.md)。
