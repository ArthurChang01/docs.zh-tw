# <a name="module-ltkeywordgt-visual-basic"></a>模組&lt;關鍵字&gt;(Visual Basic)
指定原始程式檔的開頭將屬性套用至目前組件模組。  
  
## <a name="remarks"></a>備註  
 許多屬性都屬於個別程式設計項目，例如類別或屬性。 藉由附加角括號內的屬性區塊套用這類屬性 (`< >`)，來宣告陳述式。  
  
 如果屬性屬於下列項目不僅至目前組件模組，您將屬性區塊放在原始程式檔的開頭，並找出具有屬性`Module`關鍵字。 適用於整個組件時，如果您使用[組件](../../../visual-basic/language-reference/modifiers/assembly.md)關鍵字。  
  
 `Module`修飾詞不是相同[Module 陳述式](../../../visual-basic/language-reference/statements/module-statement.md)。
