# <a name="assembly-visual-basic"></a>Assembly (Visual Basic)
指定原始程式檔的開頭將屬性套用至整個組件。  
  
## <a name="remarks"></a>備註  
 許多屬性都屬於個別程式設計項目，例如類別或屬性。 藉由附加角括號內的屬性區塊套用這類屬性 (`< >`)，來宣告陳述式。  
  
 如果屬性屬於下列項目不僅整個組件，您將屬性區塊放在原始程式檔的開頭，並找出具有屬性`Assembly`關鍵字。 適用於目前的組件模組時，如果您使用[模組](../../../visual-basic/language-reference/modifiers/module-keyword.md)關鍵字。  
  
 您也可以套用屬性至組件在 AssemblyInfo.vb 檔案中，在此情況下不需要主要原始程式檔中使用屬性區塊。
