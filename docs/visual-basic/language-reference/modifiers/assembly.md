# <a name="assembly-visual-basic"></a>Assembly (Visual Basic)
指定的原始程式檔開頭的屬性會套用到整個組件。  
  
## <a name="remarks"></a>備註  
 許多屬性適用於個別的程式設計元素，例如類別或屬性。 您套用這類屬性所附加的屬性區塊角括號內 (`< >`)，直接至宣告陳述式。  
  
 如果屬性屬於下列項目不僅整個組件，您將屬性區塊放在原始程式檔的開頭，然後找出具有屬性`Assembly`關鍵字。 它會套用至目前的組件模組，如果您使用[模組](../../../visual-basic/language-reference/modifiers/module-keyword.md)關鍵字。  
  
 您也可以套用屬性至組件在 AssemblyInfo.vb 檔案中，在此情況下您沒有主要原始程式檔中使用屬性區塊。
