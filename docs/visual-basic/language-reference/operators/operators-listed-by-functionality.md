## <a name="categories-of-operators"></a>類別目錄的運算子  
  
|運算子|描述|  
|---------------|-----------------|  
|[算術運算子](../../../visual-basic/language-reference/operators/arithmetic-operators.md)|這些運算子會執行數學計算。|  
|[指派運算子](../../../visual-basic/language-reference/operators/assignment-operators.md)|這些運算子會執行指派作業。|  
|[比較運算子](../../../visual-basic/language-reference/operators/comparison-operators.md)|這些運算子會執行比較。|  
|[串連運算子](../../../visual-basic/language-reference/operators/concatenation-operators.md)|這些運算子會合併字串。|  
|[邏輯/位元運算子](../../../visual-basic/language-reference/operators/logical-bitwise-operators.md)|這些運算子執行邏輯運算。|  
|[位元移位運算子](../../../visual-basic/language-reference/operators/bit-shift-operators.md)|這些運算子會執行算術移位的位元模式。|  
|[其他運算子](../../../visual-basic/language-reference/operators/miscellaneous-operators.md)|這些運算子會執行其他作業。|
