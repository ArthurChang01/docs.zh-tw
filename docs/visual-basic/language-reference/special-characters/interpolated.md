# <a name="-visual-basic-reference"></a>$ （Visual Basic 參考）

將字串常值識別為[字串插值](../../programming-guide/language-features/strings/interpolated-strings.md)。 字串插值是類似範本的字串，其中包含常值文字及「插入運算式」。 解析字串插值時 (例如在指派陳述式或方法呼叫中)，其插入運算式會以運算式在結果字串中的字串表示取代。 字串插值會以 .NET Framework 支援的[複合格式字串](../../../standard/base-types/composite-format.md)取代。
