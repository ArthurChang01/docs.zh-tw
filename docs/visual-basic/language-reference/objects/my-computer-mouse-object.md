# <a name="mycomputermouse-object"></a>My.Computer.Mouse 物件
提供屬性，以取得本機電腦上所安裝之滑鼠格式和設定的相關資訊。  
  
## <a name="remarks"></a>備註  
 如需方法和 `My.Computer.Mouse` 物件屬性的資訊，請參閱 <xref:Microsoft.VisualBasic.Devices.Mouse>。  
  
 如需詳細資訊，請參閱[存取滑鼠](../../../visual-basic/developing-apps/programming/computer-resources/accessing-the-mouse.md)。  
  
## <a name="requirements"></a>需求  
 **命名空間︰** <xref:Microsoft.VisualBasic.Devices>  
  
 **類別︰** <xref:Microsoft.VisualBasic.Devices.Mouse>  
  
 **元件：** Visual Basic 執行時間程式庫（在 Microsoft 中）
