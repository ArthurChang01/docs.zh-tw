# <a name="mycomputerclipboard-object"></a>My.Computer.Clipboard 物件
提供用於操作剪貼簿的方法。  
  
## <a name="remarks"></a>備註  
 如需方法和 `My.Computer.Clipboard` 物件屬性的資訊，請參閱 <xref:Microsoft.VisualBasic.MyServices.ClipboardProxy>。  
  
 如需詳細資訊，請參閱 <<c0> [ 要儲存資料，以及從剪貼簿讀取](../../../visual-basic/developing-apps/programming/computer-resources/storing-data-to-and-reading-from-the-clipboard.md)。  
  
> [!NOTE]
>  您也可以使用的方法<xref:System.Windows.Forms.Clipboard?displayProperty=nameWithType>類別來操作剪貼簿。  
  
## <a name="requirements"></a>需求  
 **命名空間︰** <xref:Microsoft.VisualBasic.MyServices>  
  
 **類別：** <xref:Microsoft.VisualBasic.MyServices.ClipboardProxy> (可存取<xref:System.Windows.Forms.Clipboard>)  
  
 **組件：** Visual Basic Runtime Library (位於 Microsoft.VisualBasic.dll)
