 如需方法和 `My.Computer.Clipboard` 物件屬性的資訊，請參閱 <xref:Microsoft.VisualBasic.MyServices.ClipboardProxy>。  
  
 如需詳細資訊，請參閱 <<c0> [ 要儲存資料，以及從剪貼簿讀取](../../../visual-basic/developing-apps/programming/computer-resources/storing-data-to-and-reading-from-the-clipboard.md)。
