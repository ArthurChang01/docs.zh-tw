# <a name="mylog-object"></a>My.Log 物件
提供屬性和事件和例外狀況資訊寫入應用程式的記錄檔接聽程式的方法。  
  
## <a name="remarks"></a>備註  
 如需方法和屬性`My.Log`物件，請參閱<xref:Microsoft.VisualBasic.Logging.AspLog>。  
  
 `My.Log`物件可供 ASP.NET 應用程式只使用。 用戶端應用程式，使用[My.Application.Log 物件](../../../visual-basic/language-reference/objects/my-application-log-object.md)。
