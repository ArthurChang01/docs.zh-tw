# <a name="mylog-object"></a>My.Log 物件
提供屬性和事件和例外狀況資訊寫入應用程式的記錄檔接聽程式的方法。  
  
## <a name="remarks"></a>備註  
 如需方法和 `My.Log` 物件屬性的資訊，請參閱 <xref:Microsoft.VisualBasic.Logging.AspLog>。  
  
 `My.Log`物件是適用於 ASP.NET 應用程式。 用戶端應用程式，使用[My.Application.Log 物件](../../../visual-basic/language-reference/objects/my-application-log-object.md)。  
  
## <a name="requirements"></a>需求  
 **命名空間︰** <xref:Microsoft.VisualBasic.Logging>  
  
 **類別︰** <xref:Microsoft.VisualBasic.Logging.AspLog>  
  
 **組件：** Visual Basic Runtime Library (位於 Microsoft.VisualBasic.dll)
