# <a name="option-ltkeywordgt-statement"></a>選項&lt;關鍵字&gt;陳述式
引進指定套用至整個原始程式檔的編譯器選項的陳述式。  
  
## <a name="remarks"></a>備註  
 編譯器選項可以控制是否必須明確地宣告所有變數，縮小類型轉換是否必須明確，否則成文字或二進位的數量是否比較字串。  
  
 `Option` 關鍵字可用於以下內容：  
  
 [Option Compare 陳述式](../../../visual-basic/language-reference/statements/option-compare-statement.md)  
  
 [Option Explicit 陳述式](../../../visual-basic/language-reference/statements/option-explicit-statement.md)  
  
 [Option Infer 陳述式](../../../visual-basic/language-reference/statements/option-infer-statement.md)  
  
 [Option Strict 陳述式](../../../visual-basic/language-reference/statements/option-strict-statement.md)
