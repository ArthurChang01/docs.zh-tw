# <a name="in-clause-visual-basic"></a>In 子句 (Visual Basic)
指定迴圈變數来周遊的群組`For Each`迴圈中，或指定的集合中查詢`From`， `Join`，或`Group Join`子句。

 `In`關鍵字可以使用下列內容：
