# <a name="type-providers"></a><span data-ttu-id="4ced7-103">型別提供者</span><span class="sxs-lookup"><span data-stu-id="4ced7-103">Type Providers</span></span>

<span data-ttu-id="4ced7-104">F# 型別提供者是一個元件，該元件會提供類型、屬性和方法讓您在程式中使用。</span><span class="sxs-lookup"><span data-stu-id="4ced7-104">An F# type provider is a component that provides types, properties, and methods for use in your program.</span></span> <span data-ttu-id="4ced7-105">型別提供者會產生所謂**提供的型別**，這由 F # 編譯器產生，而且外部資料來源為基礎。</span><span class="sxs-lookup"><span data-stu-id="4ced7-105">Type Providers generate what are known as **Provided Types**, which are generated by the F# compiler and are based on an external data source.</span></span>

<span data-ttu-id="4ced7-106">比方說，F # 型別提供者的 SQL 可以產生代表資料表和關聯式資料庫中的資料行的類型。</span><span class="sxs-lookup"><span data-stu-id="4ced7-106">For example, an F# Type Provider for SQL can generate types representing tables and columns in a relational database.</span></span> <span data-ttu-id="4ced7-107">事實上，這是什麼[SQLProvider](https://fsprojects.github.io/SQLProvider/)型別提供者。</span><span class="sxs-lookup"><span data-stu-id="4ced7-107">In fact, this is what the [SQLProvider](https://fsprojects.github.io/SQLProvider/) Type Provider does.</span></span>

<span data-ttu-id="4ced7-108">提供類型取決於型別提供者的輸入參數。</span><span class="sxs-lookup"><span data-stu-id="4ced7-108">Provided Types depend on input parameters to a Type Provider.</span></span> <span data-ttu-id="4ced7-109">這類的輸入可以是範例資料來源 （例如 JSON 結構描述檔案），直接指向外部服務或資料來源的連接字串的 URL。</span><span class="sxs-lookup"><span data-stu-id="4ced7-109">Such input can be a sample data source (such as a JSON schema file), a URL pointing directly to an external service, or a connection string to a data source.</span></span> <span data-ttu-id="4ced7-110">型別提供者也可以確保只會視情況下，展開類型的群組也就是說，它們會展開如果實際上您的程式所參考的型別。</span><span class="sxs-lookup"><span data-stu-id="4ced7-110">A Type Provider can also ensure that groups of types are only expanded on demand; that is, they are expanded if the types are actually referenced by your program.</span></span> <span data-ttu-id="4ced7-111">這樣就能以強類型的方式視需要直接整合大型的資訊空間，例如線上資料市場。</span><span class="sxs-lookup"><span data-stu-id="4ced7-111">This allows for the direct, on-demand integration of large-scale information spaces such as online data markets in a strongly typed way.</span></span>

## <a name="generative-and-erased-type-providers"></a><span data-ttu-id="4ced7-112">富有生產力和它們的型別提供者</span><span class="sxs-lookup"><span data-stu-id="4ced7-112">Generative and Erased Type Providers</span></span>

<span data-ttu-id="4ced7-113">型別提供者有兩種形式： 富有生產力和清除。</span><span class="sxs-lookup"><span data-stu-id="4ced7-113">Type Providers come in two forms: Generative and Erased.</span></span>

<span data-ttu-id="4ced7-114">富有生產力的型別提供者會產生可以寫成.NET 類型，會產生這些組件的類型。</span><span class="sxs-lookup"><span data-stu-id="4ced7-114">Generative Type Providers produce types that can be written as .NET types into the assembly in which they are produced.</span></span> <span data-ttu-id="4ced7-115">這可讓他們從其他組件中的程式碼取用。</span><span class="sxs-lookup"><span data-stu-id="4ced7-115">This allows them to be consumed from code in other assemblies.</span></span> <span data-ttu-id="4ced7-116">這表示資料來源的型別的表示通常必須是另一個則是可行的方法是使用.NET 型別代表。</span><span class="sxs-lookup"><span data-stu-id="4ced7-116">This means that the typed representation of the data source must generally be one that is feasible to represent with .NET types.</span></span>

<span data-ttu-id="4ced7-117">清除的型別提供者產生僅供在組件或專案，從產生的類型。</span><span class="sxs-lookup"><span data-stu-id="4ced7-117">Erasing Type Providers produce types that can only be consumed in the assembly or project they are generated from.</span></span> <span data-ttu-id="4ced7-118">型別都是暫時的;也就是說，它們不會寫入至組件，而且不能使用的其他組件中的程式碼。</span><span class="sxs-lookup"><span data-stu-id="4ced7-118">The types are ephemeral; that is, they are not written into an assembly and cannot be consumed by code in other assemblies.</span></span> <span data-ttu-id="4ced7-119">它們可以包含*延遲*成員，可讓您將使用提供的型別可能是無限的資訊空間中。</span><span class="sxs-lookup"><span data-stu-id="4ced7-119">They can contain *delayed* members, allowing you to use provided types from a potentially infinite information space.</span></span> <span data-ttu-id="4ced7-120">它們可用於使用大型且相互連結的資料來源的一小部分。</span><span class="sxs-lookup"><span data-stu-id="4ced7-120">They are useful for using a small subset of a large and interconnected data source.</span></span>

## <a name="commonly-used-type-providers"></a><span data-ttu-id="4ced7-121">常用的型別提供者</span><span class="sxs-lookup"><span data-stu-id="4ced7-121">Commonly used Type Providers</span></span>

<span data-ttu-id="4ced7-122">下列的廣泛使用程式庫包含型別提供者針對不同的使用者：</span><span class="sxs-lookup"><span data-stu-id="4ced7-122">The following widely-used libraries contain Type Providers for different uses:</span></span>

- <span data-ttu-id="4ced7-123">[FSharp.Data](https://fsharp.github.io/FSharp.Data/)的 JSON、 XML、 CSV 及 HTML 文件格式和資源，包含型別提供者。</span><span class="sxs-lookup"><span data-stu-id="4ced7-123">[FSharp.Data](https://fsharp.github.io/FSharp.Data/) includes Type Providers for JSON, XML, CSV, and HTML document formats and resources.</span></span>
- <span data-ttu-id="4ced7-124">[根據 SQLProvider](https://fsprojects.github.io/SQLProvider/)提供強型別存取關聯資料庫物件的對應和 F # LINQ 對這些資料來源的查詢。</span><span class="sxs-lookup"><span data-stu-id="4ced7-124">[SQLProvider](https://fsprojects.github.io/SQLProvider/) provides strongly-typed access to relation databases through object mapping and F# LINQ queries against these data sources.</span></span>
- <span data-ttu-id="4ced7-125">[FSharp.Data.SqlClient](https://fsprojects.github.io/FSharp.Data.SqlClient/)編譯時期的型別提供者的一組簽入內嵌的 F # 中的 T-SQL。</span><span class="sxs-lookup"><span data-stu-id="4ced7-125">[FSharp.Data.SqlClient](https://fsprojects.github.io/FSharp.Data.SqlClient/) has a set of type providers for compile-time checked embedding of T-SQL in F#.</span></span>
- <span data-ttu-id="4ced7-126">[Azure 儲存體類型提供者](https://fsprojects.github.io/AzureStorageTypeProvider/)提供 Azure Blob、 資料表和佇列，可讓您存取這些資源，而不需要指定為字串，在整個程式的資源名稱的類型。</span><span class="sxs-lookup"><span data-stu-id="4ced7-126">[Azure Storage Type provider](https://fsprojects.github.io/AzureStorageTypeProvider/) provides types for Azure Blobs, Tables, and Queues, allowing you to access these resources without needing to specify resource names as strings throughout your program.</span></span>
- <span data-ttu-id="4ced7-127">[FSharp.Data.GraphQL](https://fsprojects.github.io/FSharp.Data.GraphQL/index.html)包含**GraphQLProvider**，可由 URL 指定 GraphQL server 為基礎的類型。</span><span class="sxs-lookup"><span data-stu-id="4ced7-127">[FSharp.Data.GraphQL](https://fsprojects.github.io/FSharp.Data.GraphQL/index.html) contains the **GraphQLProvider**, which provides types based on a GraphQL server specified by URL.</span></span>

<span data-ttu-id="4ced7-128">必要時，您可以[建立您自己的自訂型別提供者](creating-a-type-provider.md)，或參考已由其他人建立的型別提供者。</span><span class="sxs-lookup"><span data-stu-id="4ced7-128">Where necessary, you can [create your own custom type providers](creating-a-type-provider.md), or reference type providers that have been created by others.</span></span> <span data-ttu-id="4ced7-129">例如，假設您的組織的資料服務提供大量且不斷增加的具名資料集，各資料集都有自己的穩定資料結構描述。</span><span class="sxs-lookup"><span data-stu-id="4ced7-129">For example, assume your organization has a data service providing a large and growing number of named data sets, each with its own stable data schema.</span></span> <span data-ttu-id="4ced7-130">您可能選擇建立一個類型提供者，以強類型方式讀取結構描述並對程式設計人員呈現最新的可用資料集。</span><span class="sxs-lookup"><span data-stu-id="4ced7-130">You may choose to create a type provider that reads the schemas and presents the latest available data sets to the programmer in a strongly typed way.</span></span>
