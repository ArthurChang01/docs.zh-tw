# <a name="getting-started-with-f"></a>開始使用 F# #

開始使用 F# 的主要方法有三種：

若您是在 Windows 上，請參閱[在 Visual Studio 中開始使用 F#](getting-started-visual-studio.md)。

若您想在 Windows、Linux 或 macOS 上以輕量級 IDE 撰寫程式碼，請參閱[在 Visual Studio Code 中透過 Ionide 開始使用 F#](getting-started-vscode.md)。

若您偏好使用任何作業系統的命令列，請參閱[透過命令列開始使用 F #](getting-started-command-line.md)。
