# <a name="f-guide"></a>F# 指南

F # 是一種功能的程式設計語言.NET 上執行。  除了支援的函式程式設計建構，它也會有物件的程式設計功能。  此混合式功能性程式設計與物件導向的功能，讓 F # 實用的語言，完成任何工作。

## <a name="if-youre-new-to-f"></a>如果您剛接觸到 F # #

如果您剛接觸到 F #，以開始[教學課程的 F #](tour.md)取得語言的概觀，以及其程式設計概念。  如果您使用 Visual Studio，教學課程專案範本也包含相同的內容。

## <a name="if-youre-experienced-with-f"></a>如果您是熟悉 F # #

如果您知道您的工作環境 F # 中，或想要深入了解特定語言建構時，請參閱[語言參考](language-reference/index.md)。  它是所有的 F # 語言功能的完整指南。

此外， [F # 核心程式庫參考](https://msdn.microsoft.com/visualfsharpdocs/conceptual/fsharp-core-library-reference)是了解相關的 FSharp.Core，屬於的 F # 核心程式庫的絕佳資源。

## <a name="the-f-software-foundation"></a>F# Software Foundation (F# 軟體基金會)

雖然 Microsoft 是主要的 F # 語言和其工具開發人員，但 F # 也支援獨立的 foundation、 F # 軟體基礎 (FSSF)。

F# Software Foundation 的使命是促進、保護和推展 F# 程式設計語言，並支援和促進 F# 程式設計人員國際社群的多元發展。

若要進一步了解並參與，請參閱 [fsharp.org](http://fsharp.org)。
