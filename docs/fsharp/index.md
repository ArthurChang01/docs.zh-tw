## <a name="the-f-software-foundation"></a>F# Software Foundation (F# 軟體基金會)

雖然 Microsoft 是F#語言的主要開發人員和其在 Visual Studio 中的工具F# , 但也是由F#軟體基礎 (FSSF) 獨立基礎所支援。

F# Software Foundation 的使命是促進、保護和推展 F# 程式設計語言，並支援和促進 F# 程式設計人員國際社群的多元發展。

若要進一步了解並參與，請參閱 [fsharp.org](https://fsharp.org)。這是免費的聯結, 而基礎F#開發人員的網路則是您不想錯過的事!
