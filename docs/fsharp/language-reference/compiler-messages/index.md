# <a name="f-compiler-messages"></a>F#編譯器訊息

本節詳細說明F#編譯器會針對特定結構發出的編譯器錯誤和警告。 預設的錯誤集可以透過下列方式來變更：

- 使用 `-warnaserror+` 編譯器選項，將特定警告視為錯誤，

- 使用 `-nowarn` 編譯器選項忽略特定警告

如果此區段尚未記錄特定的警告或錯誤，請移至此頁面的結尾，並傳送包含錯誤數位或文字的意見反應。
