# <a name="indexed-properties"></a><span data-ttu-id="cd3a9-103">索引屬性</span><span class="sxs-lookup"><span data-stu-id="cd3a9-103">Indexed Properties</span></span>

<span data-ttu-id="cd3a9-104">在定義可抽象化已排序資料的類別時，有時會很有説明，提供該資料的索引存取權，而不會公開基礎的執行。</span><span class="sxs-lookup"><span data-stu-id="cd3a9-104">When defining a class that abstracts over ordered data, it can sometimes be helpful to provide indexed access to that data without exposing the underlying implementation.</span></span> <span data-ttu-id="cd3a9-105">這是使用 `Item` 成員來完成。</span><span class="sxs-lookup"><span data-stu-id="cd3a9-105">This is done with the `Item` member.</span></span>

## <a name="syntax"></a><span data-ttu-id="cd3a9-106">語法</span><span class="sxs-lookup"><span data-stu-id="cd3a9-106">Syntax</span></span>

```fsharp
// Indexed property that can be read and written to
member self-identifier.Item
    with get(index-values) =
        get-member-body
    and set index-values values-to-set =
        set-member-body

// Indexed property can only be read
member self-identifier.Item
    with get(index-values) =
        get-member-body

// Indexed property that can only be set
member self-identifier.Item
    with set index-values values-to-set =
        set-member-body
```

## <a name="indexed-properties-with-multiple-index-values"></a><span data-ttu-id="cd3a9-121">具有多個索引值的索引屬性</span><span class="sxs-lookup"><span data-stu-id="cd3a9-121">Indexed Properties with multiple index values</span></span>

<span data-ttu-id="cd3a9-122">已編制索引的屬性可以有一個以上的索引值。</span><span class="sxs-lookup"><span data-stu-id="cd3a9-122">Indexed properties can have more than one index value.</span></span> <span data-ttu-id="cd3a9-123">在此情況下，使用屬性時，值會以逗號分隔。</span><span class="sxs-lookup"><span data-stu-id="cd3a9-123">In that case, the values are separated by commas when the property is used.</span></span> <span data-ttu-id="cd3a9-124">這類屬性中的 set 方法必須有兩個擴充引數，第一個是包含索引鍵的元組，而第二個是要設定的值。</span><span class="sxs-lookup"><span data-stu-id="cd3a9-124">The set method in such a property must have two curried arguments, the first of which is a tuple containing the keys, and the second of which is the value to set.</span></span>

<span data-ttu-id="cd3a9-125">下列程式碼示範如何使用具有多個索引值的索引屬性。</span><span class="sxs-lookup"><span data-stu-id="cd3a9-125">The following code demonstrates the use of an indexed property with multiple index values.</span></span>

```fsharp
open System.Collections.Generic

/// Basic implementation of a sparse matrix based on a dictionary
type SparseMatrix() =
    let table = new Dictionary<(int * int), float>()
    member _.Item
        // Because the key is comprised of two values, 'get' has two index values
        with get(key1, key2) = table.[(key1, key2)]

        // 'set' has two index values and a new value to place in the key's position
        and set (key1, key2) value = table.[(key1, key2)] <- value

let sm = new SparseMatrix()
for i in 1..1000 do
    sm.[i, i] <- float i * float i
```
