# <a name="mlnet-guide"></a>ML.NET 指南

ML.NET 是免費的跨平台開放原始碼機器學習服務架構，可供您建置量身打造的機器學習解決方案，以及與您的 .NET 應用程式相整合。 此指南提供有關使用 ML.NET 的資源。

如需 ML.NET 的詳細資訊，請參閱張貼在.NET 部落格上的 [ML.NET 簡介：經實證的跨平台開放原始碼機器學習服務架構](https://blogs.msdn.microsoft.com/dotnet/2018/05/07/introducing-ml-net-cross-platform-proven-and-open-source-machine-learning-framework/)。

## <a name="get-started"></a>開始使用

若要開始使用 ML.NET，請參閱 [Iris Petal Prediction 快速入門](https://www.microsoft.com/net/learn/apps/machine-learning-and-ai/ml-dotnet/get-started)，或其他更深入的[教學課程](tutorials/index.md)。

如果您是機器學習服務的新手，也可檢閱[機器學習服務基本概念](resources/basics.md)，可找到能協助您的機器學習服務資源。

## <a name="how-the-mlnet-guide-is-organized"></a>ML.NET 指南的編排方式

ML.NET 指南包含數個章節。 您可以按順序閱讀，或直接跳至您最感興趣的部分。

**[教學課程](tutorials/index.md)**

在本節中，您可以找到逐步教學課程，能引導您完成建置量身打造之機器學習服務的模型，用於一般的開發人員案例。

**[字彙](resources/glossary.md)**

提供機器學習服務術語及其定義的清單。
