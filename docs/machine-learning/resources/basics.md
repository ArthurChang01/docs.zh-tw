# <a name="machine-learning-basics"></a>機器學習服務基本概念

本文提供機器學習之快速簡介的一些基本資源。

## <a name="what-is-machine-learning"></a>什麼是機器學習服務？

機器學習服務是資料科學技術，可讓電腦使用現有的資料預測未來的行為、結果與趨勢。 使用機器學習服務，電腦會學習而不必明確程式化。

來自機器學習服務的預報或預測可讓應用程式及裝置更聰明。 當您在線上購物時，機器學習服務可根據您已購買的商品，協助建議您可能會想要的其他產品。 當您刷卡時，機器學習服務會比較此交易與交易資料庫，並協助偵測詐騙。 當您的掃地機器人在房間吸塵時，機器學習服務可協助它判斷工作是否已完成。
