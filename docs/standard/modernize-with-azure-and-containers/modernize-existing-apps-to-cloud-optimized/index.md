# <a name="modernize-existing-net-apps-to-cloud-optimized-applications"></a>現代化雲端最佳化應用程式的現有.NET 應用程式

> 願景：現代化雲端最佳化應用程式以讓您可以更快交付並降低應用程式傳遞成本大幅改善部署靈活度，現有的.NET Framework 應用程式。

若要充分利用雲端和新技術 (例如容器)，您應該至少將現有 .NET 應用程式局部現代化。 最後，將企業應用程式現代化可降低整體擁有成本。

局部現代化應用程式不一定表示完全移轉及改造。 您可以一開始將重要但容易現代化現有的應用程式的現代化。 您可以使用任何 Windows 和 IIS 相依性，來維護以現有 .NET Framework 版本撰寫的目前程式碼基底。 圖 4-1 反白顯示如何以雲端最佳化應用程式都位於 Azure 的應用程式現代化成熟模型。
