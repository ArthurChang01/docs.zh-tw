# <a name="modernize-existing-net-apps-to-cloud-optimized-applications"></a>現代化雲端最佳化應用程式現有的.NET 應用程式

> 願景： 現代化雲端最佳化應用程式，以大幅提升部署的靈活度，讓您可以出貨速度及較低的應用程式的傳遞成本您現有的.NET Framework 應用程式。

若要充分利用雲端和新技術 (例如容器)，您應該至少將現有 .NET 應用程式局部現代化。 最後，將企業應用程式現代化可降低整體擁有成本。

部分種應用程式不一定表示的完整移轉和 rearchitecture。 您可以一開始現代化重要但容易現代化現有的應用程式。 您可以使用任何 Windows 和 IIS 相依性，來維護以現有 .NET Framework 版本撰寫的目前程式碼基底。 圖 4-1 會反白顯示方式的雲端最佳化應用程式都放置在 Azure 應用程式現代化成熟度模型。
