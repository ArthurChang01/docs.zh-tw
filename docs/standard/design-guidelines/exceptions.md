# <a name="design-guidelines-for-exceptions"></a>例外狀況的設計方針
例外狀況處理與以傳回值為基礎的錯誤報表有許多優點。 良好的架構設計可協助應用程式開發人員實現例外狀況的優點。 本節討論例外狀況的優點，並提供有效使用它們的指導方針。
