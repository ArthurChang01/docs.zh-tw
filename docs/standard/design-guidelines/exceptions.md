# 例外狀況的設計指導方針
例外狀況處理有許多優點傳回值為基礎的錯誤報告。 良好的架構設計可協助應用程式開發人員了解例外狀況的優點。 本節討論例外狀況的優點，並提供有效地使用這些指導方針。
