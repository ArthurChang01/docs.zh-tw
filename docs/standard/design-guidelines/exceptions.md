# <a name="design-guidelines-for-exceptions"></a>例外狀況的設計方針
例外狀況處理有傳回值為基礎的錯誤報告的許多優點。 良好的架構設計可協助應用程式開發人員實現優勢的例外狀況。 本節討論這些優點的例外狀況，並提供有效地使用這些指導方針。
