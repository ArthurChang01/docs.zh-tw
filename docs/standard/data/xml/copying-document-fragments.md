# <a name="copying-document-fragments"></a>複製文件片段
您可以建立 **XmlDocumentFragment** 節點，然後將節點加入其下。 當 **XmlDocumentFragment** 以 **InsertNode** 方法插入時，不會複製 **XmlDocumentFragment** 節點，但是它的子節點會插入 XML 文件物件模型 (DOM) 中。
