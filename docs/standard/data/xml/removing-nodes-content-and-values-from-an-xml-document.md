# <a name="removing-nodes-content-and-values-from-an-xml-document"></a>從 XML 文件移除節點、內容和值
一旦 XML 文件物件模型 (DOM) 在記憶體中，即可從樹狀目錄移除節點，或從特定節點型別移除內容及值。 如需如何從文件移除分葉節點或整個節點樹狀子目錄的詳細資訊，請參閱[從 DOM 移除節點](../../../../docs/standard/data/xml/removing-nodes-from-the-dom.md)。 如需如何在項目上移除屬性的詳細資訊，請參閱[移除 DOM 中項目節點的屬性](../../../../docs/standard/data/xml/removing-attributes-from-an-element-node-in-the-dom.md)。 如需移除節點內容但在樹狀目錄中保留節點的詳細資訊，請參閱[移除 DOM 中的節點內容](../../../../docs/standard/data/xml/removing-node-content-in-the-dom.md)。
