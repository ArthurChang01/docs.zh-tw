 <span data-ttu-id="fbe3b-113">如需將轉換套用至關聯式資料的詳細資訊，請參閱[將 XSLT 轉換套用至 DataSet](../../../../docs/framework/data/adonet/dataset-datatable-dataview/applying-an-xslt-transform-to-a-dataset.md)。</span><span class="sxs-lookup"><span data-stu-id="fbe3b-113">For more information on applying a transform to relational data, see [Applying an XSLT Transform to a DataSet](../../../../docs/framework/data/adonet/dataset-datatable-dataview/applying-an-xslt-transform-to-a-dataset.md).</span></span>
