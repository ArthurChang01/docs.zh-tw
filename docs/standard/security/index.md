# <a name="security-in-net"></a>.NET 的安全性
Common language runtime 和.NET 提供許多實用的類別和服務，可讓開發人員輕鬆地撰寫安全程式碼，以及讓系統管理員自訂權限授與程式碼，使其可以存取受保護的資源。 此外，執行階段和.NET 提供實用的類別和服務，方便使用密碼編譯和以角色為基礎的安全性。
