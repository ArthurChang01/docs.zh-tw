# <a name="building-documentation-with-mdoc"></a>使用 mdoc 建置文件

mdoc 是 Managed 參考文件的文件建置系統。 透過您的組件反映，mdoc 就會建立您受管理參考文件的基本格式，而這份文件補充所編寫的內容。
