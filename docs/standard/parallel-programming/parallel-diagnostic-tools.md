# Parallel Diagnostic Tools
[!INCLUDE[vs_dev10_ext](../../../includes/vs-dev10-ext-md.md)] 可更廣泛地支援多執行緒應用程式的偵錯和分析。  
  
## 偵錯  
 Visual Studio 偵錯工具加入了新的視窗來偵錯平行應用程式。  如需詳細資訊，請參閱下列主題：  
  
-   [使用平行堆疊視窗](../Topic/Using%20the%20Parallel%20Stacks%20Window.md)  
  
-   [使用工作視窗](../Topic/Using%20the%20Tasks%20Window.md)  
  
-   [逐步解說：偵錯平行應用程式](../Topic/Walkthrough:%20Debugging%20a%20Parallel%20Application.md).  
  
## 分析  
 \[並行視覺化檢視\] 報表可讓您將平行程式中的執行緒彼此互動的方式，以及它們與系統上其他處理序中的執行緒互動的方式視覺化。  如需詳細資訊，請參閱[並行視覺化檢視](../Topic/Concurrency%20Visualizer.md)。
