# <a name="for-further-reading-parallel-programming"></a>進一步閱讀 (平行程式設計)

下列資源包含 .NET 中平行程式設計的其他相關資訊：

- [平行程式設計模式：了解及套用平行模式與 .NET Framework 4](https://www.microsoft.com/download/details.aspx?id=19222) \(英文\) 文件描述常見的平行模式，以及利用那些模式開發平行元件的最佳做法。

- [針對多核架構解構與協調的設計模式](https://msdn.microsoft.com/library/ff963553.aspx) \(英文\) 電子書說明使用 .NET Framework 4 中所引入平行程式設計支援的平行程式設計模式。

- [使用 .NET 的平行程式設計](https://blogs.msdn.microsoft.com/pfxteam/) \(英文\) 部落格包含許多有關 .NET 中平行程式設計的深度文章。

- [使用 .NET Framework 進行平行程式設計的範例](https://code.msdn.microsoft.com/ParExtSamples)網頁包含了許多示範中級和進階平行程式設計技術的範例。
