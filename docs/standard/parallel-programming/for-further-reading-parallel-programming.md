# For Further Reading (Parallel Programming)
下列資源包含 .NET Framework 之平行程式設計的其他資訊：  
  
-   MSDN 網站上的[平行計算開發人員中心](http://go.microsoft.com/fwlink/?LinkID=160570) \(英文\) 具有相關連結，可連至原生 C\+\+ 與 .NET Framework 之平行程式設計的最新內容與論壇貼文。  
  
-   MSDN 部落格網站上的 [.NET 平行程式設計部落格](http://go.microsoft.com/fwlink/?LinkID=169627)\(英文\)，有許多與 .NET Framework 的平行程式設計有關的深度文章。  
  
-   MSDN 部落格網站上的[並行視覺化檢視部落格](http://go.microsoft.com/fwlink/?LinkID=169630) \(英文\) 討論了某些 [!INCLUDE[vs_dev10_long](../../../includes/vs-dev10-long-md.md)] 版本所包含的新型效能分析工具。  
  
-   MSDN 論壇網站上的[平行擴充功能論壇](http://go.microsoft.com/fwlink/?LinkID=169628) \(英文\)，是平行程式設計相關問題的問答園地。  
  
-   MSDN Code Gallery 網站上的[平行擴充範例](http://go.microsoft.com/fwlink/?LinkID=165717) \(英文\) 網頁包含了許多示範中級和進階平行程式設計技術的範例。
