# <a name="custom-timespan-format-strings"></a><span data-ttu-id="bf4fb-102">自訂 TimeSpan 格式字串</span><span class="sxs-lookup"><span data-stu-id="bf4fb-102">Custom TimeSpan Format Strings</span></span>
<span data-ttu-id="bf4fb-103"><xref:System.TimeSpan> 格式字串會定義從格式化作業產生之 <xref:System.TimeSpan> 值的字串表示。</span><span class="sxs-lookup"><span data-stu-id="bf4fb-103">A <xref:System.TimeSpan> format string defines the string representation of a <xref:System.TimeSpan> value that results from a formatting operation.</span></span> <span data-ttu-id="bf4fb-104">自訂格式字串會由一或多個自訂的 <xref:System.TimeSpan> 格式規範搭配任意數目的常值字元所組成。</span><span class="sxs-lookup"><span data-stu-id="bf4fb-104">A custom format string consists of one or more custom <xref:System.TimeSpan> format specifiers along with any number of literal characters.</span></span> <span data-ttu-id="bf4fb-105">任何字串只要不是[標準 TimeSpan 格式字串](../../../docs/standard/base-types/standard-timespan-format-strings.md)，就會被解譯為自訂的 <xref:System.TimeSpan> 格式字串。</span><span class="sxs-lookup"><span data-stu-id="bf4fb-105">Any string that is not a [standard TimeSpan format string](../../../docs/standard/base-types/standard-timespan-format-strings.md) is interpreted as a custom <xref:System.TimeSpan> format string.</span></span>  
  
> [!IMPORTANT]
>  <span data-ttu-id="bf4fb-106">自訂 <xref:System.TimeSpan> 格式規範不包含預留位置分隔符號，例如分隔天與小時、小時與分鐘或是秒與小數秒的符號。</span><span class="sxs-lookup"><span data-stu-id="bf4fb-106">The custom <xref:System.TimeSpan> format specifiers do not include placeholder separator symbols, such as the symbols that separate days from hours, hours from minutes, or seconds from fractional seconds.</span></span> <span data-ttu-id="bf4fb-107">相反地，這些符號必須包含在自訂格式字串中作為字串常值。</span><span class="sxs-lookup"><span data-stu-id="bf4fb-107">Instead, these symbols must be included in the custom format string as string literals.</span></span> <span data-ttu-id="bf4fb-108">例如，`"dd\.hh\:mm"` 會定義句號 (.) 作為天與小時之間的分隔符號，並定義冒號 (:) 作為小時與分鐘之間的分隔符號。</span><span class="sxs-lookup"><span data-stu-id="bf4fb-108">For example, `"dd\.hh\:mm"` defines a period (.) as the separator between days and hours, and a colon (:) as the separator between hours and minutes.</span></span>  
>   
>  <span data-ttu-id="bf4fb-109">自訂 <xref:System.TimeSpan> 格式規範也不包括讓您區分正負時間間隔的正負號。</span><span class="sxs-lookup"><span data-stu-id="bf4fb-109">Custom <xref:System.TimeSpan> format specifiers also do not include a sign symbol that enables you to differentiate between negative and positive time intervals.</span></span> <span data-ttu-id="bf4fb-110">若要包括正負號，您必須使用條件邏輯建構格式字串。</span><span class="sxs-lookup"><span data-stu-id="bf4fb-110">To include a sign symbol, you have to construct a format string by using conditional logic.</span></span> <span data-ttu-id="bf4fb-111">[其他字元](#Other)一節中將提供範例。</span><span class="sxs-lookup"><span data-stu-id="bf4fb-111">The [Other Characters](#Other) section includes an example.</span></span>  
  
 <span data-ttu-id="bf4fb-112">產生 <xref:System.TimeSpan> 值的字串表示時，會藉由呼叫 <xref:System.TimeSpan.ToString%2A?displayProperty=nameWithType> 方法的多載，以及藉由支援複合格式設定的方法 (例如 <xref:System.String.Format%2A?displayProperty=nameWithType>)，來產生。</span><span class="sxs-lookup"><span data-stu-id="bf4fb-112">The string representations of <xref:System.TimeSpan> values are produced by calls to the overloads of the <xref:System.TimeSpan.ToString%2A?displayProperty=nameWithType> method, and by methods that support composite formatting, such as <xref:System.String.Format%2A?displayProperty=nameWithType>.</span></span> <span data-ttu-id="bf4fb-113">如需詳細資訊，請參閱[格式化類型](../../../docs/standard/base-types/formatting-types.md)和[複合格式設定](../../../docs/standard/base-types/composite-formatting.md)。</span><span class="sxs-lookup"><span data-stu-id="bf4fb-113">For more information, see [Formatting Types](../../../docs/standard/base-types/formatting-types.md) and [Composite Formatting](../../../docs/standard/base-types/composite-formatting.md).</span></span> <span data-ttu-id="bf4fb-114">下列範例說明如何在格式設定作業中使用自訂格式自串。</span><span class="sxs-lookup"><span data-stu-id="bf4fb-114">The following example illustrates the use of custom format strings in formatting operations.</span></span>
