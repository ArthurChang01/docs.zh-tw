# <a name="manipulating-strings-in-net"></a>操作在.NET 中的字串
.NET 提供大量的常式集合，讓您能夠有效率建立、比較和修改字串，並迅速剖析大量文字和資料以搜尋、移除和取代文字模式。

 [.NET 規則運算式](../../../docs/standard/base-types/regular-expressions.md)  
 提供有關.NET 規則運算式，包括語言項目、 規則運算式行為和範例的詳細的資訊。  
  
 [基本字串作業](../../../docs/standard/base-types/basic-string-operations.md)
