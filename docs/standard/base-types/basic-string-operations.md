# .NET Framework 中的基本字串作業
應用程式通常是以根據使用者輸入建構訊息的方式來回應使用者。  例如，網站就經常以包含使用者姓名的特殊問候語來回應新登入的使用者。  <xref:System.String?displayProperty=fullName> 和 <xref:System.Text.StringBuilder?displayProperty=fullName> 類別中有幾個方法可讓您以動態方式建構要顯示在使用者介面中的自訂字串。  這些方法也可幫助您執行一些基本的字串作業，例如從位元組陣列建立新字串、比較字串值，以及修改現有字串。
