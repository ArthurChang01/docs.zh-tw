# <a name="basic-string-operations"></a>基本字串作業

應用程式通常是以根據使用者輸入建構訊息的方式來回應使用者。 例如，網站就經常以包含使用者姓名的特殊問候語來回應新登入的使用者。 [System.String](xref:System.String) 和 [System.Text.StringBuilder](xref:System.Text.StringBuilder) 類別中有幾個方法可讓您以動態方式建構要顯示在使用者介面中的自訂字串。 這些方法也可協助您執行一些基本字串作業，例如從位元組陣列建立新的字串、比較字串值，以及修改現有的字串。
