# <a name="how-to-convert-numeric-user-input-in-web-controls-to-numbers"></a><span data-ttu-id="bbd73-102">如何：將使用者輸入 Web 控制項的數值轉換成數字</span><span class="sxs-lookup"><span data-stu-id="bbd73-102">How to: Convert Numeric User Input in Web Controls to Numbers</span></span>
<span data-ttu-id="bbd73-103">由於網頁可在世界上的任何角落顯示，所以，使用者可以幾乎不限數量的格式來將數值資料輸入至 <xref:System.Web.UI.WebControls.TextBox> 控制項。</span><span class="sxs-lookup"><span data-stu-id="bbd73-103">Because a Web page can be displayed anywhere in the world, users can input numeric data into a <xref:System.Web.UI.WebControls.TextBox> control in an almost unlimited number of formats.</span></span> <span data-ttu-id="bbd73-104">因此，判斷網頁使用者的地區設定和文化特性就變得非常重要。</span><span class="sxs-lookup"><span data-stu-id="bbd73-104">As a result, it is very important to determine the locale and culture of the Web page's user.</span></span> <span data-ttu-id="bbd73-105">當您剖析使用者輸入時，可以接著套用使用者地區設定和文化特性所定義的格式設定慣例。</span><span class="sxs-lookup"><span data-stu-id="bbd73-105">When you parse user input, you can then apply the formatting conventions defined by the user's locale and culture.</span></span>  
  
### <a name="to-convert-numeric-input-from-a-web-textbox-control-to-a-number"></a><span data-ttu-id="bbd73-106">將來自 Web TextBox 控制項的數值輸入轉換為數字</span><span class="sxs-lookup"><span data-stu-id="bbd73-106">To convert numeric input from a Web TextBox control to a number</span></span>  
  
1.  <span data-ttu-id="bbd73-107">判斷是否已填入由 <xref:System.Web.HttpRequest.UserLanguages%2A?displayProperty=nameWithType> 屬性所傳回的字串陣列。</span><span class="sxs-lookup"><span data-stu-id="bbd73-107">Determine whether the string array returned by the <xref:System.Web.HttpRequest.UserLanguages%2A?displayProperty=nameWithType> property is populated.</span></span> <span data-ttu-id="bbd73-108">如果不要，請繼續進行步驟 6。</span><span class="sxs-lookup"><span data-stu-id="bbd73-108">If it is not, continue to step 6.</span></span>  
  
2.  <span data-ttu-id="bbd73-109">如果由 <xref:System.Web.HttpRequest.UserLanguages%2A> 屬性所傳回的字串陣列已填入，請擷取它的第一個元素。</span><span class="sxs-lookup"><span data-stu-id="bbd73-109">If the string array returned by the <xref:System.Web.HttpRequest.UserLanguages%2A> property is populated, retrieve its first element.</span></span> <span data-ttu-id="bbd73-110">第一個元素代表使用者預設或慣用的語言和地區。</span><span class="sxs-lookup"><span data-stu-id="bbd73-110">The first element indicates the user's default or preferred language and region.</span></span>  
  
3.  <span data-ttu-id="bbd73-111">藉由呼叫 <xref:System.Globalization.CultureInfo.%23ctor%28System.String%2CSystem.Boolean%29?displayProperty=nameWithType> 建構函式來將代表使用者慣用文化特性的 <xref:System.Globalization.CultureInfo> 物件具現化。</span><span class="sxs-lookup"><span data-stu-id="bbd73-111">Instantiate a <xref:System.Globalization.CultureInfo> object that represents the user's preferred culture by calling the <xref:System.Globalization.CultureInfo.%23ctor%28System.String%2CSystem.Boolean%29?displayProperty=nameWithType> constructor.</span></span>  
  
4.  <span data-ttu-id="bbd73-112">針對您想要轉換使用者輸入的目標數值類型呼叫 `TryParse` 或 `Parse` 方法。</span><span class="sxs-lookup"><span data-stu-id="bbd73-112">Call either the `TryParse` or the `Parse` method of the numeric type that you want to convert the user's input to.</span></span> <span data-ttu-id="bbd73-113">搭配 `provider` 參數使用 `TryParse` 或 `Parse` 方法的多載，然後將下列任一項傳遞給它：</span><span class="sxs-lookup"><span data-stu-id="bbd73-113">Use an overload of the `TryParse` or the `Parse` method with a `provider` parameter, and pass it either of the following:</span></span>  
  
    -   <span data-ttu-id="bbd73-114">在步驟 3 中建立的 <xref:System.Globalization.CultureInfo> 物件。</span><span class="sxs-lookup"><span data-stu-id="bbd73-114">The <xref:System.Globalization.CultureInfo> object created in step 3.</span></span>  
  
    -   <span data-ttu-id="bbd73-115">由步驟 3 中所建立 <xref:System.Globalization.CultureInfo> 物件之 <xref:System.Globalization.CultureInfo.NumberFormat%2A> 屬性所傳回的 <xref:System.Globalization.NumberFormatInfo> 物件。</span><span class="sxs-lookup"><span data-stu-id="bbd73-115">The <xref:System.Globalization.NumberFormatInfo> object that is returned by the <xref:System.Globalization.CultureInfo.NumberFormat%2A> property of the <xref:System.Globalization.CultureInfo> object created in step 3.</span></span>  
  
5.  <span data-ttu-id="bbd73-116">如果轉換失敗，請針對由 <xref:System.Web.HttpRequest.UserLanguages%2A> 屬性所傳回之字串陣列中的每個剩餘元素，重複執行步驟 2 到 4。</span><span class="sxs-lookup"><span data-stu-id="bbd73-116">If the conversion fails, repeat steps 2 through 4 for each remaining element in the string array returned by the <xref:System.Web.HttpRequest.UserLanguages%2A> property.</span></span>  
  
6.  <span data-ttu-id="bbd73-117">如果轉換仍然失敗，或由 <xref:System.Web.HttpRequest.UserLanguages%2A> 屬性所傳回的字串陣列是空的，請使用不因文化特性而異的方式來剖析字串，這是由 <xref:System.Globalization.CultureInfo.InvariantCulture%2A?displayProperty=nameWithType> 屬性所傳回。</span><span class="sxs-lookup"><span data-stu-id="bbd73-117">If the conversion still fails or if the string array returned by the <xref:System.Web.HttpRequest.UserLanguages%2A> property is empty, parse the string by using the invariant culture, which is returned by the <xref:System.Globalization.CultureInfo.InvariantCulture%2A?displayProperty=nameWithType> property.</span></span>
