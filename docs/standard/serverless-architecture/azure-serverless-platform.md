# <a name="azure-serverless-platform"></a>Azure 無伺服器平台

Azure 無伺服器平台包含 Azure Functions、 Logic Apps 和 Event Grid。 這些服務搭配運作，以及無數與連線的其他資源。 無伺服器平台會從資料庫和儲存體所需的一切運作，以分析和機器學習/人工智慧的智慧。

您也可以使用 Application Insights 中，擷取診斷追蹤和遙測的無伺服器平台。 Application Insights 可用於所有類型的應用程式 (傳統型、 行動、 或 web) 以及無伺服器的實作。 下圖中視覺化平台：

![Azure 無伺服器平台](./media/azure-serverless-platform.png)
