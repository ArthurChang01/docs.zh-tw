# <a name="choosing-between-net-core-and-net-framework-for-docker-containers"></a>針對 Docker 容器在 .NET Core 和 .NET Framework 之間進行選擇

有兩個支援的實作，可使用 .NET 來建置伺服器端容器化 Docker 應用程式：[.NET Framework](https://www.microsoft.com/net/download/framework) 及 [.NET Core](https://www.microsoft.com/net/download/core)。 這兩者共用許多 .NET Standard 元件，而您可以在這兩者間共用程式碼。 不過，兩者間有一些基本差異，而您使用的實作取決於您想要完成的目標。 本節提供每個實作選擇時機的指引。
