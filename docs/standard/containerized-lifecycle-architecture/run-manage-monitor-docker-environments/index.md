# <a name="run-manage-and-monitor-docker-production-environments"></a>執行、管理和監視 Docker 生產環境

願景：企業應用程式需要具備高可用性和高延展性執行IT 部門需要能夠管理和監視環境和應用程式本身。

容器化 Docker 應用程式生命週期中的最後一個部分集中於如何在可擴充、高可用性 (HA) 生產環境中執行、管理和監視應用程式。

您在生產環境 （基礎結構架構和平台技術） 中執行容器化的應用程式的方式是非常相關，並根據所選架構和開發平台本電子書第 1 章所述。

本章檢查特定產品和技術，從 Microsoft 和其他廠商，可用來有效率地執行可調整的 HA 分散式應用程式，以及您可以管理和監視它們從 IT 觀點的方式。
