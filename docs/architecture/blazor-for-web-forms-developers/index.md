## <a name="introduction"></a><span data-ttu-id="b8870-126">簡介</span><span class="sxs-lookup"><span data-stu-id="b8870-126">Introduction</span></span>

<span data-ttu-id="b8870-127">.NET 透過 ASP.NET 提供長時間支援的 web 應用程式開發，這是一組完整的架構和工具，可用來建立任何類型的 web 應用程式。</span><span class="sxs-lookup"><span data-stu-id="b8870-127">.NET has long supported web app development through ASP.NET, a comprehensive set of frameworks and tools for building any kind of web app.</span></span> <span data-ttu-id="b8870-128">ASP.NET 有自己的 web 架構和技術歷程，從傳統的 Active Server Pages （ASP）開始一直回到一步。</span><span class="sxs-lookup"><span data-stu-id="b8870-128">ASP.NET has its own lineage of web frameworks and technologies starting all the way back with classic Active Server Pages (ASP).</span></span> <span data-ttu-id="b8870-129">ASP.NET Web Forms、ASP.NET MVC、ASP.NET Web Pages 和最近 ASP.NET Core 的架構，可提供有效率且強大的方式來建立*伺服器呈現*的 Web 應用程式，其中 UI 內容是在伺服器上動態產生，以回應 HTTP發送.</span><span class="sxs-lookup"><span data-stu-id="b8870-129">Frameworks like ASP.NET Web Forms, ASP.NET MVC, ASP.NET Web Pages, and more recently ASP.NET Core, provide a productive and powerful way to build *server-rendered* web apps, where UI content is dynamically generated on the server in response to HTTP requests.</span></span> <span data-ttu-id="b8870-130">每個 ASP.NET 架構都會已經考慮到不同的物件和應用程式建立原理。</span><span class="sxs-lookup"><span data-stu-id="b8870-130">Each ASP.NET framework caters to a different audience and app building philosophy.</span></span> <span shipped with the original release of the .NET Framework and enabled web development using many of the patterns familiar to desktop developers, like reusable UI controls with simple event provide a way to run code that executed in the user's browser.</span></span> <span data-ttu-id="b8870-133">若要這麼做，需要撰寫 JavaScript，並使用多年來的任何 JavaScript 架構和工具： jQuery、挖式、角度、反應等等。</span><span class="sxs-lookup"><span data-stu-id="b8870-133">To do that requires writing JavaScript and using any of the many JavaScript frameworks and tools that have phased in and out of popularity over the years: jQuery, Knockout, Angular, React, and so on.</span></span>

<span data-ttu-id="b8870-134">[Blazor](https://blazor.net)是新的 web 架構，可變更使用 .net 建立 web 應用程式時可能發生的情況。</span><span class="sxs-lookup"><span data-stu-id="b8870-134">[Blazor](https://blazor.net) is a new web framework that changes what is possible when building web apps with .NET.</span></span> <span data-ttu-id="b8870-135">Blazor 是以為基礎的用戶端 web UI 架構C# ，而不是 JavaScript。</span><span class="sxs-lookup"><span data-stu-id="b8870-135">Blazor is a client-side web UI framework based on C# instead of JavaScript.</span></span> <span data-ttu-id="b8870-136">透過 Blazor，您可以在中C#撰寫用戶端邏輯和 UI 元件、將它們編譯成一般的 .net 元件，然後使用稱為 WebAssembly 的新 open web standard，直接在瀏覽器中執行它們。</span><span class="sxs-lookup"><span data-stu-id="b8870-136">With Blazor you can write your client-side logic and UI components in C#, compile them into normal .NET assemblies, and then run them directly in the browser using a new open web standard called WebAssembly.</span></span> <span data-ttu-id="b8870-137">或者，Blazor 可以在伺服器上執行 .NET UI 元件，並處理與瀏覽器的即時連線流暢地的所有 UI 互動。</span><span class="sxs-lookup"><span data-stu-id="b8870-137">Or alternatively, Blazor can run your .NET UI components on the server and handle all UI interactions fluidly over a real-time connection with the browser.</span></span> <span data-ttu-id="b8870-138">當與伺服器上執行的 .NET 配對時，Blazor 會啟用 .NET 的完整堆疊 網頁程式開發。</span><span class="sxs-lookup"><span data-stu-id="b8870-138">When paired with .NET running on the server, Blazor enables full-stack web development with .NET.</span></span> <span data-ttu-id="b8870-139">雖然 Blazor 與 ASP.NET Web form 共用許多共通性，例如擁有可重複使用的元件模型，以及處理使用者事件的簡單方法，但它也是以 .NET Core 的基礎為基礎，以提供現代化且高效能的 Web 開發體驗。</span><span class="sxs-lookup"><span data-stu-id="b8870-139">While Blazor shares many commonalities with ASP.NET Web Forms, like having a reusable component model and a simple way to handle user events, it also builds on the foundations of .NET Core to provide a modern and high performance web development experience.</span></span>

<span data-ttu-id="b8870-140">本書介紹 ASP.NET Web Forms 開發人員，以熟悉且方便的方式 Blazor。</span><span class="sxs-lookup"><span data-stu-id="b8870-140">This book introduces ASP.NET Web Forms developers to Blazor in a way that is familiar and convenient.</span></span> <span data-ttu-id="b8870-141">它會以 ASP.NET Web form 中的類似概念來平行引進 Blazor 概念，同時說明可能較不熟悉的新概念。</span><span class="sxs-lookup"><span data-stu-id="b8870-141">It introduces Blazor concepts in parallel with analogous concepts in ASP.NET Web Forms while also explaining new concepts that may be less familiar.</span></span> <span data-ttu-id="b8870-142">其中涵蓋了各種主題和疑慮，包括元件撰寫、路由、版面配置、設定和安全性。</span><span class="sxs-lookup"><span data-stu-id="b8870-142">It covers a broad range of topics and concerns including component authoring, routing, layout, configuration, and security.</span></span> <span data-ttu-id="b8870-143">雖然本書的內容主要是用來啟用新的開發，但它也涵蓋了將現有的 ASP.NET Web form 遷移至 Blazor 的指導方針和策略，以供您想要將現有的應用程式現代化。</span><span class="sxs-lookup"><span data-stu-id="b8870-143">And while the content of this book is primarily for enabling new development, it also covers guidelines and strategies for migrating existing ASP.NET Web Forms to Blazor for when you want to modernize an existing app.</span></span>
