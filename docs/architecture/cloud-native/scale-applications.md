# <a name="scaling-cloud-native-applications"></a>調整雲端原生應用程式

[!INCLUDE [book-preview](../../../includes/book-preview.md)]

移至雲端裝載環境最常被標榜的優點之一，就是可調整規模。 擴充性，或能夠讓應用程式接受額外的使用者負載，而不需要過度降低每個使用者的效能，最常見的做法是將應用程式分成幾個小部分，讓每個專案都能獲得所需的任何資源。 在本章中，我們將介紹可讓雲端原生應用程式調整以符合使用者需求的技術。 這些技術包括：

- 容器
- 協調器
- 無伺服器運算
