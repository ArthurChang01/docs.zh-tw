# <a name="scaling-cloud-native-applications"></a>調整雲端原生應用程式

[!INCLUDE [book-preview](../../../includes/book-preview.md)]

移至雲端裝載環境最常被標榜的優點之一，就是可調整規模。 擴充性，或能夠讓應用程式接受額外的使用者負載，而不會影響每位使用者的效能。 最常見的做法是將應用程式分解成可提供所需資源的小部分。 雲端廠商可隨時隨地和世界各地提供大規模的擴充性。

 在本章中，我們會討論可讓雲端原生應用程式調整以符合使用者需求的技術。 這些技術包括：

- 容器
- Orchestrator
- 無伺服器運算
