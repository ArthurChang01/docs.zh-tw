# <a name="when-to-deploy-windows-containers-in-your-on-premises-iaas-vm-infrastructure"></a>在內部部署 IaaS VM 基礎結構中部署 Windows 容器的時機

- 您的組織可能尚未準備好遷移到雲，或者由於業務原因可能無法遷移到雲。 但是，您仍然可以在自己的資料中心中使用 Windows 容器的好處。

- 您可能還有其他專案正在本地使用，當您嘗試遷移到雲時，這些工件可能會減慢速度。 例如，具有本地 Windows 伺服器活動目錄或任何其他本地資產的安全或身份驗證依賴項。

- 如果立即開始使用 Windows 容器，則可以從更好的位置進行分階段遷移到雲。 Windows 容器正在成為任何雲的部署單元，沒有鎖定。
