# <a name="modernize-existing-net-apps-to-cloud-optimized-applications"></a>將現有 .NET 應用程式現代化成雲端最佳化應用程式

> 願景：將現有的 .NET Framework 應用程式現代化到雲端優化的應用程式，大幅提升您的部署靈活性，讓您可以更快交付並降低應用程式的交付成本。

若要充分利用雲端和新技術 (例如容器)，您應該至少將現有 .NET 應用程式局部現代化。 最後，將企業應用程式現代化可降低整體擁有成本。

將應用程式局部現代化，未必需要全面移轉及重新架構。 剛開始將現有的應用程式現代化時，可以利用匯入加快現代化的速度。 您可以使用任何 Windows 和 IIS 相依性，來維護以現有 .NET Framework 版本撰寫的目前程式碼基底。 圖 4-1 反白顯示 Azure 應用程式現代化成熟模型中，雲端最佳化的定位。
