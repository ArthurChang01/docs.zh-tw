<span data-ttu-id="6537a-191">下列範例*函數. json*檔案會定義觸發程式和系結：</span><span class="sxs-lookup"><span data-stu-id="6537a-191">The following example *functions.json* file defines a trigger and a binding:</span></span>

```json
{
  "bindings": [
    {
      "name": "myBlob",
      "type": "blobTrigger",
      "direction": "in",
      "path": "images/{name}",
      "connection": "AzureWebJobsStorage"
    },
    {
      "name": "$return",

    }
  ],
  "disabled": false
```
