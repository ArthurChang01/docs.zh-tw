## <a name="remoting"></a><span data-ttu-id="e1414-120">遠端處理</span><span class="sxs-lookup"><span data-stu-id="e1414-120">Remoting</span></span>

## <a name="code-access-security-cas"></a><span data-ttu-id="e1414-131">程式碼存取安全性 (CAS)</span><span class="sxs-lookup"><span data-stu-id="e1414-131">Code Access Security (CAS)</span></span>
