## <a name="increase-the-maximum-open-file-limit-net-core-versions-before-net-core-sdk-202"></a><span data-ttu-id="57ca1-127">增加開啟的檔案上限 (.NET Core SDK 2.0.2 之前的 .NET Core 版本)</span><span class="sxs-lookup"><span data-stu-id="57ca1-127">Increase the maximum open file limit (.NET Core versions before .NET Core SDK 2.0.2)</span></span> 

<span data-ttu-id="57ca1-128">在較舊的 .NET Core 版本中 (.NET Core SDK 2.0.2 之前)，macOS 的預設開啟檔案限制可能對一些 .NET Core 工作負載而言不足，例如還原專案或執行單元測試。</span><span class="sxs-lookup"><span data-stu-id="57ca1-128">In older .NET Core versions (before .NET Core SDK 2.0.2), the default open file limit on macOS may not be sufficient for some .NET Core workloads, such as restoring projects or running unit tests.</span></span>

<span data-ttu-id="57ca1-129">您可以遵循下列步驟來增加此限制：</span><span class="sxs-lookup"><span data-stu-id="57ca1-129">You can increase this limit by following these steps:</span></span>

1. <span data-ttu-id="57ca1-130">使用文字編輯器建立新檔案 _/Library/LaunchDaemons/limit.maxfiles.plist_，並儲存含此內容的檔案：</span><span class="sxs-lookup"><span data-stu-id="57ca1-130">Using a text editor, create a new file _/Library/LaunchDaemons/limit.maxfiles.plist_, and save the file with this content:</span></span>

```xml
<?xml version="1.0" encoding="UTF-8"?>
<!DOCTYPE plist PUBLIC "-//Apple//DTD PLIST 1.0//EN"
        "http://www.apple.com/DTDs/PropertyList-1.0.dtd">
<plist version="1.0">
  <dict>
    <key>Label</key>
    <string>limit.maxfiles</string>
    <key>ProgramArguments</key>
    <array>
      <string>launchctl</string>
      <string>limit</string>
      <string>maxfiles</string>
      <string>2048</string>
      <string>4096</string>
    </array>
    <key>RunAtLoad</key>
    <true/>
    <key>ServiceIPC</key>
    <false/>
  </dict>
</plist>
```

2. <span data-ttu-id="57ca1-131">在終端機視窗中，執行下列命令：</span><span class="sxs-lookup"><span data-stu-id="57ca1-131">In a terminal window, run the following command:</span></span>

```console

```

3. <span data-ttu-id="57ca1-132">重新啟動 Mac，以套用這些設定。</span><span class="sxs-lookup"><span data-stu-id="57ca1-132">Reboot your Mac to apply these settings.</span></span>
