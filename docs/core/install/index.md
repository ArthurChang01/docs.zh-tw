# <a name="download-and-install-net-core"></a>下載並安裝 .NET Core

本文提供如何下載和安裝 .NET Core 的相關資訊。 .NET Core 有兩個部分：用來執行應用程式的**運行**時間，以及用來建立應用程式的**SDK** 。 SDK 包含執行時間。

- 如果您是需要 .NET Core 才能執行應用程式的使用者，請參閱[安裝 .Net Core 運行](runtime.md)時間。

- 如果您是需要 .NET Core 來建立應用程式的開發人員，請參閱[安裝 .NET Core SDK](sdk.md)。

> [!NOTE]
> .NET Core SDK 包含執行時間。

## <a name="dependencies"></a>相依性

Windows、Linux 和 macOS 上都支援 .NET Core。 如需完整的需求清單，請參閱[支援的作業系統](dependencies.md)。
