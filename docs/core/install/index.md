# <a name="download-and-install-net-core"></a>下載並安裝 .NET 核心

本文提供有關如何下載和安裝 .NET Core 的資訊。 .NET Core 有兩個部分，用於運行應用程式的**運行時**和用於創建應用的**SDK。** SDK 包括運行時。

- 如果您是需要 .NET Core 來運行應用的使用者，請參閱安裝[.NET 核心運行時](runtime.md)。

- 如果您是需要 .NET Core 來創建應用程式的開發人員，請參閱[安裝 .NET 核心 SDK](sdk.md)。

> [!NOTE]
> .NET 核心 SDK 包括運行時。

## <a name="dependencies"></a>相依性

.NET 核心在 Windows、Linux 和 macOS 上受支援。 有關需求的完整清單，請參閱[支援的作業系統](dependencies.md)。
