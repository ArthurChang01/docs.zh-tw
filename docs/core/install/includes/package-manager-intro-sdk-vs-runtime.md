如果要開發 .NET Core 應用，請安裝 SDK（包括運行時）。 或者，如果您只需要運行應用，請安裝運行時。 如果要安裝運行時，我們建議您安裝[ASP.NET核心運行時](#install-the-aspnet-core-runtime)，因為它包括 .NET Core 和 ASP.NET核心運行時。

如果已安裝 SDK 或運行時，請使用`dotnet --list-sdks`和`dotnet --list-runtimes`命令查看已安裝的版本。 有關詳細資訊，請參閱[如何檢查 .NET Core 是否已安裝](../how-to-detect-installed-versions.md?pivots=os-linux)。
