## <a name="github-issues-and-announcements"></a>GitHub 發佈和公告

您還可以查看在以下 GitHub 存儲庫中詳細介紹了 .NET Core 中引入的突發更改的單個問題：

- 對於 .NET 核心[，dotnet/docs](https://github.com/dotnet/docs/issues?q=is%3Aissue+label%3Abreaking-change)存儲庫。
- 對於ASP.NET核心[，aspnet/通知存儲庫](https://github.com/aspnet/Announcements/issues?q=is%3Aissue+is%3Aopen+label%3A%22Breaking+change%22+label%3A3.0.0)。
- 對於實體框架核心，[點網/efcore](https://github.com/dotnet/efcore/issues?q=is%3Aopen+is%3Aissue+label%3Abreaking-change)存儲庫。
