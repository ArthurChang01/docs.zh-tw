# <a name="net-core-breaking-changes"></a>.NET Core 的重大變更

下列版本選取器會針對 .NET Core、ASP.NET Core 和 EF Core 提供已篩選的重大變更清單。 目標版本不只包括主要版本，例如 .NET Core 2.2 和 .NET Core 3.0，也包含預覽版本（例如 .NET Core 3.0 Preview 7）。

您也可以在下列 GitHub 存放庫中，查看詳細說明 .NET Core 3.0 中所引進之重大變更的個別問題：

- 針對 .NET Core，則為[dotnet/](https://github.com/dotnet/docs/issues?q=is%3Aopen+is%3Aissue+label%3Abreaking-change)檔存放庫。
- 若為 ASP.NET Core，則為[aspnet/公告](https://github.com/aspnet/Announcements/issues?q=is%3Aissue+is%3Aopen+label%3A%22Breaking+change%22+label%3A3.0.0)存放庫。
- 若為 Entity Framework Core，則為[aspnet/microsoft.entityframeworkcore](https://github.com/aspnet/EntityFrameworkCore/issues?q=is%3Aopen+is%3Aissue+label%3Abreaking-change)存放庫。
