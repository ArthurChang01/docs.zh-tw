# <a name="what-is-net-for-apache-spark"></a>什麼是 .NET for Apache Spark？

[Apache Spark](https://spark.apache.org/)是一般用途的分散式處理引擎, 用於分析大型資料集, 通常是 tb 或數 pb 的資料。 適用于 Apache Spark 的 .NET 讓開發人員可以存取 Apache Spark, 將 Spark 的威力帶入您的應用程式, 並支援您熟悉的語言。

有C#了F#和, 您就可以存取:

* 使用結構化資料的資料框架和 SparkSQL。
* 用於處理串流資料的 Spark 結構化串流。

適用于 Apache Spark 的 .NET 符合 .NET Standard, 這是在 .NET 程式中通用的 .NET Api 正式規格。 這表示您可以在撰寫 .NET 程式碼的任何位置使用 .NET 來 Apache Spark, 讓您可以重複使用您已經擁有的 .NET 開發人員所擁有的所有知識、技能、程式碼和程式庫。

適用于 Apache Spark 的 .NET 會使用 .NET Core 在 Windows、Linux 和 macOS 上執行。 它也會使用 .NET Framework 在 Windows 上執行。 您可以將應用程式部署至所有主要雲端提供者, 包括 Azure HDInsight Spark、Amazon EMR Spark、Azure Databricks 和 AWS 上的 Databricks。

## <a name="net-foundation"></a>.NET Foundation

適用于 Apache Spark 專案的 .NET 是[.Net Foundation](https://www.dotnetfoundation.org/)的一部分。

## <a name="contributions"></a>基值

適用于 Apache Spark 小組的 .NET 鼓勵貢獻, 包括 GitHub 問題和提取要求。 首先, 尋找現有的[問題](https://github.com/dotnet/spark/issues)。 如果找不到現有的問題, 請[開啟新的問題](https://github.com/dotnet/spark/issues?utf8=%E2%9C%93&q=is%3Aissue+is%3Aopen+)。
