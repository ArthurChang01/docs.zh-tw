# <a name="constants-unmanaged-api-reference"></a>常數 (Unmanaged API 參考)
本主題描述的語言類型、 語言廠商和定義於 CorSym.idl 中的文件類型常數。  
  
## <a name="language-type-constants"></a>語言類型常數  
 下表顯示的語言類型的常數，表示識別程式設計語言的 Guid。  
  
|符號|描述|  
|------------|-----------------|  
|CorSym_LanguageType_C|表示 C 語言。|  
|CorSym_LanguageType_CPlusPlus|表示C++語言。|  
|CorSym_LanguageType_CSharp|表示C#語言。|  
|CorSym_LanguageType_Basic|表示基本的語言。|  
|CorSym_LanguageType_Java|表示 Java 語言。|  
|CorSym_LanguageType_Cobol|表示的 COBOL 語言。|  
|CorSym_LanguageType_Pascal|表示的 Pascal 語言。|  
|CorSym_LanguageType_ILAssembly|表示 Microsoft intermediate language (MSIL) 組件程式碼。|  
|CorSym_LanguageType_JScript|表示 JScript 語言。|  
|CorSym_LanguageType_SMC|表示的 SMC 語言。|  
|CorSym_LanguageType_MCPlusPlus|表示C++啟用適用於.NET Framework 的語言。|  
  
## <a name="language-vendor-constants"></a>語言廠商常數  
 下表顯示的語言廠商的常數，表示識別程式設計語言廠商的 Guid。  
  
|符號|描述|  
|------------|-----------------|  
|CorSym_LanguageVendor_Microsoft|表示 Microsoft。|  
  
## <a name="document-type-constants"></a>文件類型常數  
 下表顯示文件類型的常數，表示識別文件類型的 Guid。  
  
|符號|描述|  
|------------|-----------------|  
|CorSym_DocumentType_Text|表示文字文件。|  
|CorSym_DocumentType_MC|表示非文字文件。|
