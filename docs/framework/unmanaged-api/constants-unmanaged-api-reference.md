# <a name="constants-unmanaged-api-reference"></a>常數 (Unmanaged API 參考)
本主題描述的語言類型、 語言廠商和定義於 CorSym.idl 中的文件類型常數。  
  
## <a name="language-type-constants"></a>語言類型常數  
 下表顯示類型的常數，表示 Guid，用於識別程式設計語言的語言。  
  
|符號|描述|  
|------------|-----------------|  
|CorSym_LanguageType_C|表示 C 語言。|  
|CorSym_LanguageType_CPlusPlus|表示 c + + 語言。|  
|CorSym_LanguageType_CSharp|表示 C# 語言。|  
|CorSym_LanguageType_Basic|表示基本語言。|  
|CorSym_LanguageType_Java|表示 Java 語言。|  
|CorSym_LanguageType_Cobol|表示 COBOL 語言。|  
|CorSym_LanguageType_Pascal|表示依照 pascal 命名法的語言。|  
|CorSym_LanguageType_ILAssembly|表示 Microsoft intermediate language (MSIL) 的組譯程式碼。|  
|CorSym_LanguageType_JScript|表示 JScript 語言。|  
|CorSym_LanguageType_SMC|表示讓 SMC 語言。|  
|CorSym_LanguageType_MCPlusPlus|表示啟用.NET Framework 的 c + + 語言。|  
  
## <a name="language-vendor-constants"></a>語言廠商常數  
 下表顯示的語言廠商常數，代表用來識別程式設計的語言廠商 Guid。  
  
|符號|描述|  
|------------|-----------------|  
|CorSym_LanguageVendor_Microsoft|表示 Microsoft。|  
  
## <a name="document-type-constants"></a>文件類型常數  
 下表顯示文件類型的常數，表示用來識別文件類型的 Guid。  
  
|符號|描述|  
|------------|-----------------|  
|CorSym_DocumentType_Text|表示文字文件。|  
|CorSym_DocumentType_MC|表示非文字文件。|
