# 常數 (Unmanaged API 參考)
本主題描述的語言類型、 語言廠商和定義於 CorSym.idl 中的文件類型常數。  
  
## 語言類型常數  
 下表列出語言型別常數，表示識別程式語言的 Guid。  
  
|符號|描述|  
|--------|--------|  
|CorSym\_LanguageType\_C|表示 C 語言。|  
|CorSym\_LanguageType\_CPlusPlus|表示 c \+ \+ 語言。|  
|CorSym\_LanguageType\_CSharp|表示 C\# 語言。|  
|CorSym\_LanguageType\_Basic|表示基本的語言。|  
|CorSym\_LanguageType\_Java|表示 Java 語言。|  
|CorSym\_LanguageType\_Cobol|表示 COBOL 語言。|  
|CorSym\_LanguageType\_Pascal|表示依照 pascal 命名法的語言。|  
|CorSym\_LanguageType\_ILAssembly|表示 Microsoft 中繼語言 \(MSIL\) 的組譯程式碼。|  
|CorSym\_LanguageType\_JScript|表示 JScript 語言。|  
|CorSym\_LanguageType\_SMC|表示讓 SMC 語言。|  
|CorSym\_LanguageType\_MCPlusPlus|表示啟用.NET Framework 的 c \+ \+ 語言。|  
  
## 語言廠商常數  
 下表列出語言廠商常數，表示識別程式語言廠商的 Guid。  
  
|符號|描述|  
|--------|--------|  
|CorSym\_LanguageVendor\_Microsoft|表示 Microsoft。|  
  
## 文件類型常數  
 下表顯示文件類型的常數，表示識別文件類型的 Guid。  
  
|符號|描述|  
|--------|--------|  
|CorSym\_DocumentType\_Text|表示文字文件。|  
|CorSym\_DocumentType\_MC|表示非文字文件。|
