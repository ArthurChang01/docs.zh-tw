# <a name="iceegengetilsection-method"></a>ICeeGen::GetIlSection 方法
取得指定的控制代碼所參考的基底中繼語言程式碼區段。  
  
 這個方法已經過時，不應使用。  
  
## <a name="syntax"></a>語法  
  
```  
HRESULT GetIlSection (  
    [in] HCEESECTION  *section  
);  
```  
  
## <a name="parameters"></a>參數  
 `section`  
 [in]一節，以控制代碼。
