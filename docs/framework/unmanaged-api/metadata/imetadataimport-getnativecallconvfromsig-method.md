# <a name="imetadataimportgetnativecallconvfromsig-method"></a>IMetaDataImport::GetNativeCallConvFromSig 方法
取得由指定簽章指標代表的方法之原生呼叫慣例。  
  
## <a name="syntax"></a>語法  
  
```  
HRESULT GetNativeCallConvFromSig (  
   [in]  void const  *pvSig,  
   [in]  ULONG       cbSig,  
   [out] ULONG       *pCallConv  
);  
```  
  
#### <a name="parameters"></a>參數  
 `pvSig`  
 [in]這個方法傳回的呼叫慣例的中繼資料簽章指標。  
  
 `cbSig`  
 [in]以位元組為單位的大小`pvSig`。  
  
 `pCallConv`  
 [out]原生呼叫慣例指標。
