# <a name="imetadataemitgettokenfromsig-method"></a>IMetaDataEmit::GetTokenFromSig 方法
取得指定之中繼資料簽章的 token。  
  
## <a name="syntax"></a>語法  
  
```cpp  
HRESULT GetTokenFromSig (   
    [in]  PCCOR_SIGNATURE pvSig,   
    [in]  ULONG       cbSig,   
    [out] mdSignature *pmsig   
);  
```  
  
## <a name="parameters"></a>參數  
 `pvSig`  
 在要保存並儲存的簽章。  
  
 `cbSig`  
 在`pvSig`中的位元組計數。  
  
 `pmsig`  
 脫銷指派的 `mdSignature` token。
