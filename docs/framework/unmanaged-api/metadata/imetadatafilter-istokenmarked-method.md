# <a name="imetadatafilteristokenmarked-method"></a>IMetaDataFilter::IsTokenMarked 方法
獲取指示指定中繼資料權杖是否已標記為已處理的值。  
  
## <a name="syntax"></a>語法  
  
```cpp  
HRESULT IsTokenMarked (  
    [in]  mdToken  tk,
    [out] BOOL     *pIsMarked  
);  
```  
  
## <a name="parameters"></a>參數  
 `tk`  
 [在]要檢查處理標記的權杖。  
  
 `pIsMarked`  
 [出]`true`已處理的值`tk`;如果已處理，則為否則`false`。
