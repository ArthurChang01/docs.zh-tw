# <a name="imetadatafilteristokenmarked-method"></a>IMetaDataFilter::IsTokenMarked 方法
取得值，指出指定的中繼資料語彙基元是否已標示為處理。  
  
## <a name="syntax"></a>語法  
  
```  
HRESULT IsTokenMarked (  
    [in]  mdToken  tk,   
    [out] BOOL     *pIsMarked  
);  
```  
  
#### <a name="parameters"></a>參數  
 `tk`  
 [in]要檢查處理標記的語彙基元。  
  
 `pIsMarked`  
 [out]值是`true`如果`tk`已處理，否則為`false`。
