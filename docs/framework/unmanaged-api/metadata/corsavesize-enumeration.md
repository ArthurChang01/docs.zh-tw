# <a name="corsavesize-enumeration"></a>CorSaveSize 列舉
包含值，這些值表示在查詢儲存作業的大小時所需的精確度等級。  
  
## <a name="syntax"></a>語法  
  
```  
typedef enum CorSaveSize {  
    cssAccurate                = 0x0000,   
    cssQuick                   = 0x0001,   
    cssDiscardTransientCAs     = 0x0002  
} CorSaveSize;  
```  
  
## <a name="members"></a>成員  
  
|成員|描述|  
|------------|-----------------|  
|`cssAccurate`|指定傳回的值應該完全一樣。|  
|`cssQuick`|指定應該估計的傳回值。|  
|`cssDiscardTransientCAs`|指定應移除捨棄的型別。|
