# <a name="corsavesize-enumeration"></a>CorSaveSize 列舉
包含值，這些值表示在查詢儲存作業的大小時所需的精確度等級。  
  
## <a name="syntax"></a>語法  
  
```cpp  
typedef enum CorSaveSize {  
    cssAccurate                = 0x0000,
    cssQuick                   = 0x0001,
    cssDiscardTransientCAs     = 0x0002  
} CorSaveSize;  
```  
  
## <a name="members"></a>成員  
  
|member|描述|  
|------------|-----------------|  
|`cssAccurate`|指定傳回值應精確。|  
|`cssQuick`|指定應估計傳回值。|  
|`cssDiscardTransientCAs`|指定應刪除可丟棄的類型。|
