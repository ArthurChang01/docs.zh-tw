# <a name="imetadatatablesgetcolumn-method"></a>IMetaDataTables::GetColumn 方法
取得包含指定之資料行和指定的資料表中資料列的儲存格的值的指標。  
  
## <a name="syntax"></a>語法  
  
```  
HRESULT GetColumn (   
    [in]  ULONG   ixTbl,  
    [in]  ULONG   ixCol,  
    [in]  ULONG   rid,  
    [out] ULONG   *pVal  
);  
```  
  
#### <a name="parameters"></a>參數  
 `ixTbl`  
 [in]資料表的索引。  
  
 `ixCol`  
 [in]資料表中資料行的索引。  
  
 `rid`  
 [in]資料表中的資料列索引。  
  
 `pVal`  
 [out]在儲存格中值的指標。
