# <a name="imetadatatablesgetcolumninfo-method"></a>IMetaDataTables::GetColumnInfo 方法
指定資料表中取得指定之資料行的相關資料。  
  
## <a name="syntax"></a>語法  
  
```  
HRESULT GetColumnInfo (   
    [in]  ULONG        ixTbl,  
    [in]  ULONG        ixCol,  
    [out] ULONG        *poCol,  
    [out] ULONG        *pcbCol,  
    [out] ULONG        *pType,  
    [out] const char   **ppName  
);  
```  
  
#### <a name="parameters"></a>參數  
 `ixTbl`  
 [in]所需的資料表索引。  
  
 `ixCol`  
 [in]所需的資料行的索引。  
  
 `poCol`  
 [out]指向的資料列中的資料行的位移。  
  
 `pcbCol`  
 [out]大小 （位元組），資料行的指標。  
  
 `pType`  
 [out]資料行中值的類型的指標。  
  
 `ppName`  
 [out]指標的資料行名稱的指標。
