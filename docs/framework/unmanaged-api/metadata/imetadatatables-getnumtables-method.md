# <a name="imetadatatablesgetnumtables-method"></a>IMetaDataTables::GetNumTables 方法
取得目前範圍中的資料表數目`IMetaDataTables`執行個體。  
  
## <a name="syntax"></a>語法  
  
```  
HRESULT GetNumTables (  
    [out]  ULONG   *pcTables  
);  
```  
  
#### <a name="parameters"></a>參數  
 `pcTables`  
 [out]目前的執行個體範圍中的資料表數目指標。
