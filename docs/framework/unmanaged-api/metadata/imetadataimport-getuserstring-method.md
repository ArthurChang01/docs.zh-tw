# <a name="imetadataimportgetuserstring-method"></a>IMetaDataImport::GetUserString 方法
取得指定中繼資料語彙基元所代表的常值字串。  
  
## <a name="syntax"></a>語法  
  
```  
HRESULT GetUserString (  
   [in]   mdString    stk,  
   [out]  LPWSTR      szString,  
   [in]   ULONG       cchString,  
   [out]  ULONG       *pchString  
);  
```  
  
## <a name="parameters"></a>參數  
 `stk`  
 [in]要傳回的相關聯的字串的字串權杖。  
  
 `szString`  
 [out]要求字串的複本。  
  
 `cchString`  
 [in]所要求的寬字元的大小上限`szString`。  
  
 `pchString`  
 [out]傳回寬字元大小`szString`。
