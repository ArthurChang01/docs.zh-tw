# <a name="imetadataimportgetuserstring-method"></a>IMetaDataImport::GetUserString 方法
取得指定中繼資料語彙基元所代表的常值字串。  
  
## <a name="syntax"></a>語法  
  
```  
HRESULT GetUserString (  
   [in]   mdString    stk,  
   [out]  LPWSTR      szString,  
   [in]   ULONG       cchString,  
   [out]  ULONG       *pchString  
);  
```  
  
#### <a name="parameters"></a>參數  
 `stk`  
 [in]傳回相關聯的字串的字串語彙基元。  
  
 `szString`  
 [out]要求字串的複本。  
  
 `cchString`  
 [in]要求的寬字元的最大大小`szString`。  
  
 `pchString`  
 [out]傳回的寬字元大小`szString`。
