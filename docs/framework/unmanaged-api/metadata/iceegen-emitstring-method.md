# <a name="iceegenemitstring-method"></a>ICeeGen::EmitString 方法
指定的字串發出到程式碼基底。  
  
 這個方法已經過時，不應使用。  
  
## <a name="syntax"></a>語法  
  
```  
HRESULT EmitString (  
    [in]  LPWSTR    lpString,  
    [out] ULONG     *RVA  
);  
```  
  
#### <a name="parameters"></a>參數  
 `lpString`  
 [in]要發出的字串。  
  
 `RVA`  
 [out]發出的字串相對虛擬位址。
