# <a name="imetadatafilterunmarkall-method"></a>IMetaDataFilter::UnmarkAll 方法
處理標記移除目前的中繼資料範圍內的所有權杖。  
  
## <a name="syntax"></a>語法  
  
```cpp  
HRESULT UnmarkAll ();  
```  
  
## <a name="parameters"></a>參數  
 `UnmarkAll` 不接受任何參數。
