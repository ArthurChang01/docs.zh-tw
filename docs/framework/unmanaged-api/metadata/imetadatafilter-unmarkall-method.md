# <a name="imetadatafilterunmarkall-method"></a>IMetaDataFilter::UnmarkAll 方法
從目前中繼資料範圍中的所有標記移除處理標記。  
  
## <a name="syntax"></a>語法  
  
```cpp  
HRESULT UnmarkAll ();  
```  
  
## <a name="parameters"></a>參數  
 `UnmarkAll` 不接受任何參數。
