# <a name="cornativelinkflags-enumeration"></a>CorNativeLinkFlags 列舉
提供連結器在連結機器碼時所使用的旗標值。  
  
## <a name="syntax"></a>語法  
  
```cpp  
typedef enum  
{  
    nlfNone         = 0x00,  
    nlfLastError    = 0x01,  
    nlfNoMangle     = 0x02,  
    nlfMaxValue     = 0x03  
} CorNativeLinkFlags;  
```  
  
## <a name="members"></a>Members  
  
|成員|描述|  
|------------|-----------------|  
|`nlfNone`|Indicates no flags.|  
|`nlfLastError`|Indicates a `setLastError` keyword.|  
|`nlfNoMangle`|Indicates a `nomangle` keyword.|  
|`nlfMaxValue`|未使用。|
