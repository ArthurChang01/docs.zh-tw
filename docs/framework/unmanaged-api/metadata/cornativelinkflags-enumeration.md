# <a name="cornativelinkflags-enumeration"></a>CorNativeLinkFlags 列舉
提供連結器在連結機器碼時所使用的旗標值。  
  
## <a name="syntax"></a>語法  
  
```  
typedef enum  
{  
    nlfNone         = 0x00,  
    nlfLastError    = 0x01,  
    nlfNoMangle     = 0x02,  
    nlfMaxValue     = 0x03  
} CorNativeLinkFlags;  
```  
  
## <a name="members"></a>成員  
  
|成員|描述|  
|------------|-----------------|  
|`nlfNone`|表示沒有旗標。|  
|`nlfLastError`|指出`setLastError`關鍵字。|  
|`nlfNoMangle`|指出`nomangle`關鍵字。|  
|`nlfMaxValue`|未使用。|
