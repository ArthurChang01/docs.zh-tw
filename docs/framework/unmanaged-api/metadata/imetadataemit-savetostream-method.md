# <a name="imetadataemitsavetostream-method"></a>IMetaDataEmit::SaveToStream 方法
將所有的中繼資料儲存至指定在目前範圍中`IStream`。  
  
## <a name="syntax"></a>語法  
  
```  
HRESULT SaveToStream (   
    [in]  IStream     *pIStream,  
    [in]  DWORD       dwSaveFlags  
);  
```  
  
#### <a name="parameters"></a>參數  
 `pIStream`  
 [in]若要將儲存至可寫入的資料流。  
  
 `dwSaveFlags`  
 [in]保留。 必須是零。
