# <a name="imetadataemitsavetostream-method"></a>IMetaDataEmit::SaveToStream 方法
將當前作用域中的所有中繼資料保存到指定的`IStream`。  
  
## <a name="syntax"></a>語法  
  
```cpp  
HRESULT SaveToStream (
    [in]  IStream     *pIStream,  
    [in]  DWORD       dwSaveFlags  
);  
```  
  
## <a name="parameters"></a>參數  
 `pIStream`  
 [在]要保存到的可寫流。  
  
 `dwSaveFlags`  
 [in] 保留。 必須為零。
