# <a name="imetadatatablesgetblobheapsize-method"></a>IMetaDataTables::GetBlobHeapSize 方法
獲取二進位大型物件 （BLOB） 堆的大小（以位元組為單位）。  
  
## <a name="syntax"></a>語法  
  
```cpp  
HRESULT GetBlobHeapSize (  
    [out] ULONG     *pcbBlobs  
);
```  
  
## <a name="parameters"></a>參數  
 `pcbBlobs`  
 [出]指向 BLOB 堆的大小（以位元組為單位）的指標。
