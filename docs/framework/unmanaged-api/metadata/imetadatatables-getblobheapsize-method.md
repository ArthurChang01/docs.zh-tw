# <a name="imetadatatablesgetblobheapsize-method"></a>IMetaDataTables::GetBlobHeapSize 方法
取得大小，單位為位元組的二進位大型物件 (BLOB) 堆積。  
  
## <a name="syntax"></a>語法  
  
```  
HRESULT GetBlobHeapSize (  
    [out] ULONG     *pcbBlobs  
);   
```  
  
#### <a name="parameters"></a>參數  
 `pcbBlobs`  
 [out]大小，以位元組為單位，對 BLOB 堆積的指標。
