# <a name="imetadataemitdefineuserstring-method"></a>IMetaDataEmit::DefineUserString 方法
取得指定的常值字串的中繼資料語彙基元。  
  
## <a name="syntax"></a>語法  
  
```  
HRESULT DefineUserString (   
    [in]  LPCWSTR     szString,   
    [in]  ULONG       cchString,   
    [out] mdString    *pstk   
);  
```  
  
## <a name="parameters"></a>參數  
 `szString`  
 [in]要儲存的使用者字串。  
  
 `cchString`  
 [in]中的寬字元的計數`szString`。  
  
 `pstk`  
 [out]指派的字串權杖。
