# <a name="imetadatatablesgetblob-method"></a>IMetaDataTables::GetBlob 方法
取得二進位大型物件 (BLOB) 的指標，在指定的資料行索引。  
  
## <a name="syntax"></a>語法  
  
```  
HRESULT GetBlob (  
    [in]  ULONG          ixBlob,  
    [out] ULONG          *pcbData,  
    [out] const void     **ppData  
);  
```  
  
## <a name="parameters"></a>參數  
 `ixBlob`  
 [in]要取得的記憶體位址`ppData`。  
  
 `pcbData`  
 [out]指標的大小，以位元組為單位的`ppData`。  
  
 `ppData`  
 [out]擷取二進位資料指標的指標。
