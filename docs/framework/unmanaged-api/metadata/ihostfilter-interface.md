# <a name="ihostfilter-interface"></a>IHostFilter 介面
提供方法來指出將會處理指定的權杖。  
  
> [!NOTE]
> `IHostFilter`介面必須由用戶端執行。  
  
## <a name="methods"></a>方法  
  
|方法|說明|  
|------------|-----------------|  
|[MarkToken 方法](../../../../docs/framework/unmanaged-api/metadata/ihostfilter-marktoken-method.md)|表示將會處理指定的元資料標記。|
