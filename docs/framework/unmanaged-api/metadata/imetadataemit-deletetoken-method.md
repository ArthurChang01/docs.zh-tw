# <a name="imetadataemitdeletetoken-method"></a>IMetaDataEmit::DeleteToken 方法
從目前的中繼資料範圍中刪除指定的語彙基元。  
  
## <a name="syntax"></a>語法  
  
```  
HRESULT DeleteToken (   
    [in]  mdToken     tkObj   
);  
```  
  
#### <a name="parameters"></a>參數  
 `tkObj`  
 [in]要刪除的語彙基元。
