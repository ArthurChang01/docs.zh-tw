# <a name="imetadataemitdeletetoken-method"></a>IMetaDataEmit::DeleteToken 方法
從當前中繼資料作用域中刪除指定的權杖。  
  
## <a name="syntax"></a>語法  
  
```cpp  
HRESULT DeleteToken (
    [in]  mdToken     tkObj
);  
```  
  
## <a name="parameters"></a>參數  
 `tkObj`  
 [在]要刪除的權杖。
