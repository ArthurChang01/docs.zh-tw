# <a name="imetadataemitsetmethodimplflags-method"></a>IMetaDataEmit::SetMethodImplFlags 方法
設置或更新指定權杖引用的繼承方法實現的中繼資料簽名。  
  
## <a name="syntax"></a>語法  
  
```cpp  
HRESULT SetMethodImplFlags (
    [in]  mdMethodDef   md,
    [in]  DWORD         dwImplFlags
);  
```  
  
## <a name="parameters"></a>參數  
 `md`  
 [在]要更改的方法的權杖。  
  
 `dwImplFlags`  
 [在][CorMethodImpl](../../../../docs/framework/unmanaged-api/metadata/cormethodimpl-enumeration.md)枚舉的值的組合，用於指定方法實現要素。
