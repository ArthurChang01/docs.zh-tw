# <a name="imetadataemit2savedelta-method"></a>IMetaDataEmit2::SaveDelta 方法
將目前的編輯和繼續工作階段的變更儲存至指定的檔案。  
  
## <a name="syntax"></a>語法  
  
```  
HRESULT SaveDelta (  
    [in] LPCWSTR     szFile,   
    [in] DWORD       dwSaveFlags  
);  
```  
  
## <a name="parameters"></a>參數  
 `szFile`  
 [in]用來儲存檔案名稱會變更。  
  
 `dwSaveFlags`  
 [in] 保留。 必須是零。
