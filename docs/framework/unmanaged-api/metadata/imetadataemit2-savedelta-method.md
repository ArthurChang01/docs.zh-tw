# <a name="imetadataemit2savedelta-method"></a>IMetaDataEmit2::SaveDelta 方法
將當前編輯和繼續會話的更改保存到指定的檔。  
  
## <a name="syntax"></a>語法  
  
```cpp  
HRESULT SaveDelta (  
    [in] LPCWSTR     szFile,
    [in] DWORD       dwSaveFlags  
);  
```  
  
## <a name="parameters"></a>參數  
 `szFile`  
 [在]要保存的檔案名將發生更改。  
  
 `dwSaveFlags`  
 [in] 保留。 必須為零。
