# <a name="imetadatatablesgetnextstring-method"></a>IMetaDataTables::GetNextString 方法
獲取當前表列中下一個字串的索引。  
  
## <a name="syntax"></a>語法  
  
```cpp  
HRESULT GetNextString (
    [in]  ULONG   ixString,  
    [out] ULONG   *pNext  
);  
```  
  
## <a name="parameters"></a>參數  
 `ixString`  
 [在]字串表列中的索引值。  
  
 `pNext`  
 [出]指向列中下一個字串的索引的指標。
