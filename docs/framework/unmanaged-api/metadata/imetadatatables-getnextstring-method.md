# <a name="imetadatatablesgetnextstring-method"></a>IMetaDataTables::GetNextString 方法
取得目前的資料表資料行中的下一個字串的索引。  
  
## <a name="syntax"></a>語法  
  
```  
HRESULT GetNextString (   
    [in]  ULONG   ixString,  
    [out] ULONG   *pNext  
);  
```  
  
#### <a name="parameters"></a>參數  
 `ixString`  
 [in]字串資料表資料行的索引值。  
  
 `pNext`  
 [out]索引資料行中的下一個字串的指標。
