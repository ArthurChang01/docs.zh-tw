# <a name="iceegengetsectiondatalen-method"></a>ICeeGen::GetSectionDataLen 方法
取得指定的區段長度。  
  
 這個方法已經過時，不應使用。  
  
## <a name="syntax"></a>語法  
  
```  
HRESULT GetSectionDataLen (  
    [in]  HCEESECTION  section,  
    [out] ULONG        *dataLen  
);  
```  
  
## <a name="parameters"></a>參數  
 `section`  
 [in][資料] 區段將擷取其長度的情況下。  
  
 `dataLen`  
 [out]指定的區段傳回的長度。  
  
## <a name="remarks"></a>備註  
 呼叫`GetSectionDataLen`只有當您有未處理的其他方法的特殊區段需求。
