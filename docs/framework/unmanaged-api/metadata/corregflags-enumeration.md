# <a name="corregflags-enumeration"></a>CorRegFlags 列舉
提供用於註冊安裝模組或複合映像時的旗標值。  
  
## <a name="syntax"></a>語法  
  
```  
typedef enum   
{  
    regNoCopy  = 0x00000001,  
    regConfig  = 0x00000002,  
    regHasRefs = 0x00000004  
} CorRegFlags;  
```  
  
## <a name="members"></a>成員  
  
|成員|描述|  
|------------|-----------------|  
|`regNoCopy`|指定的檔案不會複製到目的地。|  
|`regConfig`|指定模組或組合的組態。|  
|`regHasRefs`|指定此模組或複合類別的參考。|
