# <a name="corregflags-enumeration"></a>CorRegFlags 列舉
提供安裝模組或複合映射時用於註冊的標誌值。  
  
## <a name="syntax"></a>語法  
  
```cpp  
typedef enum
{  
    regNoCopy  = 0x00000001,  
    regConfig  = 0x00000002,  
    regHasRefs = 0x00000004  
} CorRegFlags;  
```  
  
## <a name="members"></a>成員  
  
|member|描述|  
|------------|-----------------|  
|`regNoCopy`|指定不應將檔案複製到目標。|  
|`regConfig`|指定模組或複合是配置。|  
|`regHasRefs`|指定模組或複合具有類引用。|
