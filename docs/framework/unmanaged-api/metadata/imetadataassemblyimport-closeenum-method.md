# <a name="imetadataassemblyimportcloseenum-method"></a>IMetaDataAssemblyImport::CloseEnum 方法

## <a name="syntax"></a>語法  
  
```cpp  
void CloseEnum (  
    [in] HCORENUM     hEnum  
);  
```  
  
## <a name="parameters"></a>參數  
 `hEnum`  
 [in]若要關閉列舉型別執行個體。
