# <a name="imetadatatablesgetstring-method"></a>IMetaDataTables::GetString 方法
取得字串中指定索引處與資料表資料行目前參考的範圍內。  
  
## <a name="syntax"></a>語法  
  
```  
HRESULT GetString (   
    [in]  ULONG       ixString,  
    [out] const char  **ppString  
);  
```  
  
#### <a name="parameters"></a>參數  
 `ixString`  
 [in]要開始搜尋下一個值索引。  
  
 `ppString`  
 [out]指標的指標，傳回的字串值。
