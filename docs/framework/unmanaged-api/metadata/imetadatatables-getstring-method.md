# <a name="imetadatatablesgetstring-method"></a>IMetaDataTables::GetString 方法
取得字串的指定索引處與資料表資料行中目前參考的範圍。  
  
## <a name="syntax"></a>語法  
  
```  
HRESULT GetString (   
    [in]  ULONG       ixString,  
    [out] const char  **ppString  
);  
```  
  
## <a name="parameters"></a>參數  
 `ixString`  
 [in]要開始搜尋下一個值的索引。  
  
 `ppString`  
 [out]要傳回的字串值的指標的指標。
