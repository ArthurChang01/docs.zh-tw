# <a name="imetadatatablesgetstring-method"></a>IMetaDataTables::GetString 方法
從當前引用作用域中的表列獲取指定索引處的字串。  
  
## <a name="syntax"></a>語法  
  
```cpp  
HRESULT GetString (
    [in]  ULONG       ixString,  
    [out] const char  **ppString  
);  
```  
  
## <a name="parameters"></a>參數  
 `ixString`  
 [在]開始搜索下一個值的索引。  
  
 `ppString`  
 [出]指向返回的字串值的指標。
