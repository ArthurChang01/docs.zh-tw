# <a name="imetadatavalidate-interface"></a>IMetaDataValidate 介面
提供方法來驗證中繼資料的簽章。  
  
## <a name="methods"></a>方法  
  
|方法|描述|  
|------------|-----------------|  
|[ValidateMetaData 方法](../../../../docs/framework/unmanaged-api/metadata/imetadatavalidate-validatemetadata-method.md)|驗證目前中繼資料範圍內物件的中繼資料簽章。|  
|[ValidatorInit 方法](../../../../docs/framework/unmanaged-api/metadata/imetadatavalidate-validatorinit-method.md)|設定會指定目前中繼資料範圍內模組類型的旗標，並註冊對於驗證錯誤的指定回呼方法。|
