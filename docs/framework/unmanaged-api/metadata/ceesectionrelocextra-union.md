# <a name="ceesectionrelocextra-union"></a>CeeSectionRelocExtra 等位
代表可由為位址位移[ICeeGen](../../../../docs/framework/unmanaged-api/metadata/iceegen-interface.md)重新配置區段的介面。  
  
## <a name="syntax"></a>語法  
  
```cpp  
typedef union  {  
    USHORT highAdj;  
} CeeSectionRelocExtra;  
```  
  
## <a name="members"></a>成員  
  
|成員|描述|  
|------------|-----------------|  
|`highAdj`|區段上方的地址調整。|
