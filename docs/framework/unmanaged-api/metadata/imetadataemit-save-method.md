# <a name="imetadataemitsave-method"></a>IMetaDataEmit::Save 方法
將當前作用域中的所有中繼資料保存到指定位址的檔。  
  
## <a name="syntax"></a>語法  
  
```cpp  
HRESULT Save (
    [in]  LPCWSTR     szFile,
    [in]  DWORD       dwSaveFlags  
);  
```  
  
## <a name="parameters"></a>參數  
 `wzFile`  
 [在]要保存到的檔的名稱。 如果此值為空，記憶體中的副本將保存到使用的最後一個位置。  
  
 `dwSaveFlags`  
 [in] 保留。 必須為零。
