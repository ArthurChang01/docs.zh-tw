# <a name="imetadataemitsave-method"></a>IMetaDataEmit::Save 方法
在指定的位址的檔案目前範圍中儲存所有的中繼資料。  
  
## <a name="syntax"></a>語法  
  
```  
HRESULT Save (   
    [in]  LPCWSTR     szFile,   
    [in]  DWORD       dwSaveFlags  
);  
```  
  
#### <a name="parameters"></a>參數  
 `wzFile`  
 [in]若要將儲存至檔案的名稱。 如果這個值是 null，記憶體中複本會儲存到最後一個使用的位置。  
  
 `dwSaveFlags`  
 [in]保留。 必須是零。
