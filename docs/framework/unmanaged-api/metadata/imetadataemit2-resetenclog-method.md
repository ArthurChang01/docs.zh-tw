# <a name="imetadataemit2resetenclog-method"></a>IMetaDataEmit2::ResetENCLog 方法
Resets the edit-and-continue log and starts a new session.  
  
## <a name="syntax"></a>語法  
  
```cpp  
HRESULT ResetENCLog ();  
```  
  
## <a name="parameters"></a>參數  
 `ResetENCLog` takes no parameters.
