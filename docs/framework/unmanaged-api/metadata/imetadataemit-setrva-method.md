# <a name="imetadataemitsetrva-method"></a>IMetaDataEmit::SetRVA 方法
設定指定之方法的相對虛擬位址。  
  
## <a name="syntax"></a>語法  
  
```  
HRESULT SetRVA (  
    [in]  mdMethodDef  md,   
    [in]  ULONG        ulRVA   
);  
```  
  
#### <a name="parameters"></a>參數  
 `md`  
 [in]目標方法或方法實作語彙基元。  
  
 `ulRVA`  
 [in]程式碼或資料區的位址。
