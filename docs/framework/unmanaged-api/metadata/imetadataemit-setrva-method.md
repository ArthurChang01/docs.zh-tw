# <a name="imetadataemitsetrva-method"></a>IMetaDataEmit::SetRVA 方法
設置指定方法的相對虛擬位址。  
  
## <a name="syntax"></a>語法  
  
```cpp  
HRESULT SetRVA (  
    [in]  mdMethodDef  md,
    [in]  ULONG        ulRVA
);  
```  
  
## <a name="parameters"></a>參數  
 `md`  
 [在]目標方法或方法實現的權杖。  
  
 `ulRVA`  
 [在]代碼或資料區域的位址。
