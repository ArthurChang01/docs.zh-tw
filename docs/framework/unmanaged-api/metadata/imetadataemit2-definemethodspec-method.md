# <a name="imetadataemit2definemethodspec-method"></a>IMetaDataEmit2::DefineMethodSpec 方法
創建方法的泛型實例，並獲取定義中的權杖。  
  
## <a name="syntax"></a>語法  
  
```cpp  
HRESULT DefineMethodSpec (  
    [in]  mdToken           tkParent,
    [in]  PCCOR_SIGNATURE   pvSigBlob,
    [in]  ULONG             cbSigBlob,
    [out] mdMethodSpec      *pmi  
);  
```  
  
## <a name="parameters"></a>參數  
 `tkParent`  
 [在]用於創建泛型實例的方法的權杖。 權杖必須為 類型`mdMethodDef`或`mdMemberRef`。  
  
 `pvSigBlob`  
 [在]指向方法的二進位 COM+ 簽名的指標。  
  
 `cbSibBlob`  
 [在]的大小（以位元組為單位）的大小`pvSigBlob`。  
  
 `pmi`  
 [出]方法的中繼資料簽名定義的權杖。
