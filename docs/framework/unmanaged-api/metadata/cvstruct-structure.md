# <a name="cvstruct-structure"></a>CVStruct 結構
包含在安裝模組或複合映像時，所使用的資訊。  
  
## <a name="syntax"></a>語法  
  
```cpp  
typedef struct {  
    short Major;  
    short Minor;  
    short Sub;  
    short Build;  
} CVStruct;  
```

|微幅|次要版本組建編號。|
