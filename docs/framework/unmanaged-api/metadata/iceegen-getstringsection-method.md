# <a name="iceegengetstringsection-method"></a>ICeeGen::GetStringSection 方法
取得指定的控制代碼所參考的程式碼區段的字串表示。  
  
 這個方法已經過時，不應使用。  
  
## <a name="syntax"></a>語法  
  
```  
HRESULT GetStringSection (  
    [in, out] HCEESECTION     *section  
);  
```  
  
#### <a name="parameters"></a>參數  
 `section`  
 [in、 out]程式碼區段之控制代碼。
