# <a name="imetadataemitsetfieldrva-method"></a>IMetaDataEmit::SetFieldRVA 方法
為指定權杖引用的欄位的相對虛擬位址設置全域變數值。  
  
## <a name="syntax"></a>語法  
  
```cpp  
HRESULT SetFieldRVA (
    [in]  mdFieldDef  fd,
    [in]  ULONG       ulRVA
);  
```  
  
## <a name="parameters"></a>參數  
 `fd`  
 [在]目標欄位的權杖。  
  
 `ulRVA`  
 [在]代碼或資料區域的位址。
