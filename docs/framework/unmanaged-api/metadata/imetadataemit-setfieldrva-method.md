# <a name="imetadataemitsetfieldrva-method"></a>IMetaDataEmit::SetFieldRVA 方法
針對指定之標記所參考的欄位，設定其相對虛擬位址的全域變數值。  
  
## <a name="syntax"></a>語法  
  
```cpp  
HRESULT SetFieldRVA (   
    [in]  mdFieldDef  fd,   
    [in]  ULONG       ulRVA   
);  
```  
  
## <a name="parameters"></a>參數  
 `fd`  
 在目標欄位的 token。  
  
 `ulRVA`  
 在程式碼或資料區域的位址。
