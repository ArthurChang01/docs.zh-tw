# <a name="imetadataemitsetfieldrva-method"></a>IMetaDataEmit::SetFieldRVA 方法
設定全域變數的值，指定語彙基元所參考之欄位的相對虛擬位址。  
  
## <a name="syntax"></a>語法  
  
```  
HRESULT SetFieldRVA (   
    [in]  mdFieldDef  fd,   
    [in]  ULONG       ulRVA   
);  
```  
  
#### <a name="parameters"></a>參數  
 `fd`  
 [in]目標欄位的語彙基元。  
  
 `ulRVA`  
 [in]程式碼或資料區域的位址。
