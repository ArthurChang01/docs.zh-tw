# <a name="iceegengetsectioncreate-method"></a>ICeeGen::GetSectionCreate 方法
產生並取得使用指定的名稱和旗標值的程式碼區段。  
  
 這個方法已經過時，不應使用。  
  
## <a name="syntax"></a>語法  
  
```  
HRESULT GetSectionCreate (  
    [in]  const char     *name,  
    [in]  DWORD          flags,  
    [out] HCEESECTION    *section  
);  
```  
  
#### <a name="parameters"></a>參數  
 `name`  
 [in]指定要建立的區段名稱的字串指標。  
  
 `flags`  
 [in]指定選項的旗標。  
  
 `section`  
 [out]指標，新建的程式碼區段。  
  
## <a name="remarks"></a>備註  
 呼叫`GetSectionCreate`只有當您有不由其他方法處理的特殊區段需求。
