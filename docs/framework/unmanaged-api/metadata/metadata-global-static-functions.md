# <a name="metadata-global-static-functions"></a>中繼資料全域靜態函式
從.NET Framework 4 的文件`_Cor*`， `CoInitialize*`， `CoUninitialize*`，和`CoEEShutDownCOM`函式，適用於裝載，它們位於[已被取代 CLR 裝載函式](../../../../docs/framework/unmanaged-api/hosting/deprecated-clr-hosting-functions.md)。 沒有與中繼資料相關的全域靜態函式。
