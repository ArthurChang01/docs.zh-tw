# <a name="imetadatainfo-interface"></a>IMetaDataInfo 介面
提供方法，以取得關於將中繼資料從磁碟上檔案對應到記憶體的資訊。  
  
## <a name="methods"></a>方法  
  
|方法|描述|  
|------------|-----------------|  
|[GetFileMapping 方法](../../../../docs/framework/unmanaged-api/metadata/imetadatainfo-getfilemapping-method.md)|取得記憶體區域的對應的檔，以及對應的種類。|
