# <a name="iceegengetmethodbuffer-method"></a>ICeeGen::GetMethodBuffer 方法
取得方法的適當大小的緩衝區，在指定的相對虛擬位址。  
  
 這個方法已經過時，不應使用。  
  
## <a name="syntax"></a>語法  
  
```  
HRESULT GetMethodBuffer (  
    [in]  ULONG       RVA,  
    [out] UCHAR       **lpBuffer  
);  
```  
  
#### <a name="parameters"></a>參數  
 `RVA`  
 [in]要傳回緩衝區方法的相對虛擬位址。  
  
 `lpBuffer`  
 [out]傳回緩衝區的指標。
