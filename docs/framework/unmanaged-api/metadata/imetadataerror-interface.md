# <a name="imetadataerror-interface"></a>IMetaDataError 介面
提供回呼機制，中繼資料合併期間回報錯誤。  
  
> [!NOTE]
>  `IMetaDataError`介面必須由用戶端實作。  
  
## <a name="methods"></a>方法  
  
|方法|描述|  
|------------|-----------------|  
|[OnError 方法](../../../../docs/framework/unmanaged-api/metadata/imetadataerror-onerror-method.md)|提供中繼資料合併期間所發生之錯誤的通知。|
