# <a name="imetadatatablesgetguidheapsize-method"></a>IMetaDataTables::GetGuidHeapSize 方法
取得大小，以位元組為單位，GUID 堆積。  
  
## <a name="syntax"></a>語法  
  
```  
HRESULT GetGuidHeapSize (  
    [out] ULONG   *pcbGuids  
);  
```  
  
## <a name="parameters"></a>參數  
 `pcbGuids`  
 [out]大小 （位元組），GUID 堆積的指標。
