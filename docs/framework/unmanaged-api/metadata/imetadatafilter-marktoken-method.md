# <a name="imetadatafiltermarktoken-method"></a>IMetaDataFilter::MarkToken 方法
設定值，指出指定的中繼資料語彙基元已處理。  
  
## <a name="syntax"></a>語法  
  
```cpp  
HRESULT MarkToken (  
    [in] mdToken   tk  
);  
```  
  
## <a name="parameters"></a>參數  
 `tk`  
 [in]要標記為已處理的權杖。
