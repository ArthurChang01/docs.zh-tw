# <a name="imetadataemitdeleteclasslayout-method"></a>IMetaDataEmit::DeleteClassLayout 方法
終結類別配置中繼資料簽章的指定語彙基元所代表的類型。  
  
## <a name="syntax"></a>語法  
  
```  
HRESULT DeleteClassLayout (  
    [in]  mdTypeDef   td  
);  
```  
  
#### <a name="parameters"></a>參數  
 `td`  
 [in]`mdTypeDef`代表刪除類別配置類型的中繼資料語彙基元。
