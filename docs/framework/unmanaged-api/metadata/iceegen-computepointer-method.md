# <a name="iceegencomputepointer-method"></a>ICeeGen::ComputePointer 方法
判斷指定的程式碼區段的緩衝區。  
  
 這個方法已經過時，不應使用。  
  
## <a name="syntax"></a>語法  
  
```  
HRESULT ComputePointer (  
    [in]  HCEESECTION  section,  
    [in]  ULONG        RVA,   
    [out] UCHAR        **lpBuffer  
);  
```  
  
## <a name="parameters"></a>參數  
 `section`  
 [in]要傳回之緩衝區的程式碼區段。  
  
 `RVA`  
 [in]要取得的指標方法的相對虛擬位址。  
  
 `lpBuffer`  
 [out]傳回緩衝區的指標。
