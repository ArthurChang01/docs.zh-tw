# <a name="imetadataimportresetenum-method"></a>IMetaDataImport::ResetEnum 方法
重設指定列舉程式至指定位置。  
  
## <a name="syntax"></a>語法  
  
```cpp  
HRESULT ResetEnum (  
   [in] HCORENUM    hEnum,
   [in] ULONG       ulPos  
);  
```  
  
## <a name="parameters"></a>參數  
 `hEnum`  
 [在]要重置的枚舉器。  
  
 `ulPos`  
 [在]放置枚舉器的新位置。
