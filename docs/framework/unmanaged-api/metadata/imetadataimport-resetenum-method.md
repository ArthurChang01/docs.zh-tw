# <a name="imetadataimportresetenum-method"></a>IMetaDataImport::ResetEnum 方法
重設指定列舉程式至指定位置。  
  
## <a name="syntax"></a>語法  
  
```cpp  
HRESULT ResetEnum (  
   [in] HCORENUM    hEnum,   
   [in] ULONG       ulPos  
);  
```  
  
## <a name="parameters"></a>參數  
 `hEnum`  
 [in] The enumerator to reset.  
  
 `ulPos`  
 [in] The new position at which to place the enumerator.
