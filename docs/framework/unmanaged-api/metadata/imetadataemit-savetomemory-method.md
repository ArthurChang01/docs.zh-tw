# <a name="imetadataemitsavetomemory-method"></a>IMetaDataEmit::SaveToMemory 方法
將當前作用域中的所有中繼資料保存到指定的記憶體區域。  
  
## <a name="syntax"></a>語法  
  
```cpp  
HRESULT SaveToMemory (
    [out]  void        *pbData,
    [in]   ULONG       cbData
);  
```  
  
## <a name="parameters"></a>參數  
 `pbData`  
 [出]開始寫入中繼資料的位址。  
  
 `cbData`  
 [在]已分配記憶體的大小（以位元組為單位）。
