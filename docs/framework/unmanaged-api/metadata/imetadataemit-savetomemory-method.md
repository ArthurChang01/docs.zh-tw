# <a name="imetadataemitsavetomemory-method"></a>IMetaDataEmit::SaveToMemory 方法
指定的記憶體區域目前範圍中儲存所有的中繼資料。  
  
## <a name="syntax"></a>語法  
  
```  
HRESULT SaveToMemory (   
    [out]  void        *pbData,   
    [in]   ULONG       cbData   
);  
```  
  
#### <a name="parameters"></a>參數  
 `pbData`  
 [out]要開始寫入中繼資料位址。  
  
 `cbData`  
 [in]以位元組為單位配置的記憶體大小。
