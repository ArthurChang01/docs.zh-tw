# <a name="metadata-unmanaged-api-reference"></a>中繼資料 (Unmanaged API 參考)
中繼資料應用程式開發介面可讓編譯器之類的用戶端無需由 Common Language Runtime (CLR) 載入類型，即可產生或存取元件的中繼資料。
