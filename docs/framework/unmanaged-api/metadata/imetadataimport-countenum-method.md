# <a name="imetadataimportcountenum-method"></a>IMetaDataImport::CountEnum 方法
擷取所指定的列舉值的列舉中取得的項目數。  
  
## <a name="syntax"></a>語法  
  
```  
HRESULT CountEnum (  
   [in]  HCORENUM    hEnum,   
   [out] ULONG       *pulCount  
);  
```  
  
#### <a name="parameters"></a>參數  
 `hEnum`  
 [in]列舉值控制代碼。  
  
 `pulCount`  
 [out]列舉的元素數目。  
  
## <a name="remarks"></a>備註  
 所指定的控制代碼`hEnum`取自先前`Enum`*名稱*呼叫 (例如， [imetadataimport:: Enumtypedefs](../../../../docs/framework/unmanaged-api/metadata/imetadataimport-enumtypedefs-method.md))。
