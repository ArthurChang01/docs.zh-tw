# <a name="imetadataimportisglobal-method"></a>IMetaDataImport::IsGlobal 方法
取得一個值，用來表示指定中繼資料語彙基元所代表的欄位、方法或類型值是否具有全域範圍。  
  
## <a name="syntax"></a>語法  
  
```  
HRESULT IsGlobal (  
   [in]  mdToken     pd,  
   [out] int         *pbGlobal  
);  
```  
  
#### <a name="parameters"></a>參數  
 `pd`  
 [in]中繼資料語彙基元，表示類型、 欄位或方法。  
  
 `pbGlobal`  
 [out] 1 如果物件具有全域範圍。否則為 0 （零）。
