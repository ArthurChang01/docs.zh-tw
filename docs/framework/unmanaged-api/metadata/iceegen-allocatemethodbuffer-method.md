# <a name="iceegenallocatemethodbuffer-method"></a>ICeeGen::AllocateMethodBuffer 方法
建立方法，指定大小的緩衝區，並取得方法的相對虛擬位址。  
  
 這個方法已經過時，不應使用。  
  
## <a name="syntax"></a>語法  
  
```  
HRESULT AllocateMethodBuffer (   
    [in]  ULONG    cchBuffer,   
    [out] UCHAR    **lpBuffer,  
    [out] ULONG    *RVA  
);  
```  
  
#### <a name="parameters"></a>參數  
 `cchBuffer`  
 [in]要建立之緩衝區的長度。  
  
 `lpBuffer`  
 [out]傳回的緩衝區。  
  
 `RVA`  
 [out]方法的相對虛擬位址。
