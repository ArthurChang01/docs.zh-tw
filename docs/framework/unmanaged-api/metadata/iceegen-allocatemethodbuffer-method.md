# <a name="iceegenallocatemethodbuffer-method"></a>ICeeGen::AllocateMethodBuffer 方法
為方法創建指定大小的緩衝區，並獲取方法的相對虛擬位址。  
  
 此方法已過時，不應使用。  
  
## <a name="syntax"></a>語法  
  
```cpp  
HRESULT AllocateMethodBuffer (
    [in]  ULONG    cchBuffer,
    [out] UCHAR    **lpBuffer,  
    [out] ULONG    *RVA  
);  
```  
  
## <a name="parameters"></a>參數  
 `cchBuffer`  
 [在]要創建的緩衝區的長度。  
  
 `lpBuffer`  
 [出]返回的緩衝區。  
  
 `RVA`  
 [出]方法的相對虛擬位址。
