# <a name="imetadataimportisvalidtoken-method"></a>IMetaDataImport::IsValidToken 方法
取得一個值，用來表示指定語彙基元是否包含程式碼物件的有效參考。  
  
## <a name="syntax"></a>語法  
  
```  
BOOL IsValidToken (  
   [in] mdToken     tk  
);  
```  
  
#### <a name="parameters"></a>參數  
 `tk`  
 [in]若要檢查的參考有效性語彙基元。  
  
## <a name="return-value"></a>傳回值  
 `true`如果`tk`目前範圍內是有效的中繼資料語彙基元。 否則為 `false`。
