# <a name="corsymvarflag-enumeration"></a>CorSymVarFlag 列舉
指出變數是否為編譯器所產生。  
  
## <a name="syntax"></a>語法  
  
```  
typedef enum CorSymVarFlag   
{  
    VAR_IS_COMP_GEN = 1  
} CorSymVarFlag;  
```  
  
## <a name="members"></a>成員  
  
|成員|描述|  
|------------|-----------------|  
|`VAR_IS_COMP_GEN`|指出指定的變數是由編譯器產生。|  
  
## <a name="requirements"></a>需求  
 **標頭：** 於 CorSym.idl CorSym.h
