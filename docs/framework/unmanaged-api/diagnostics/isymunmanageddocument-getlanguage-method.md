# <a name="isymunmanageddocumentgetlanguage-method"></a>ISymUnmanagedDocument::GetLanguage 方法
取得這份文件的語言識別碼  
  
## <a name="syntax"></a>語法  
  
```  
HRESULT GetLanguage(  
    [out, retval]  GUID*  pRetVal);  
```  
  
## <a name="parameters"></a>參數  
 `pRetVal`  
 [out]接收的語言識別碼的變數指標。  
  
## <a name="return-value"></a>傳回值  
 如果方法成功為 S_OK。
