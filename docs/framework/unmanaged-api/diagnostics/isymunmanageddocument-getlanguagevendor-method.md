# <a name="isymunmanageddocumentgetlanguagevendor-method"></a>ISymUnmanagedDocument::GetLanguageVendor 方法
取得此檔的語言廠商。  
  
## <a name="syntax"></a>語法  
  
```cpp  
HRESULT GetLanguageVendor(  
    [out, retval]  GUID*  pRetVal);  
```  
  
## <a name="parameters"></a>參數  
 `pRetVal`  
 脫銷接收語言廠商之變數的指標。  
  
## <a name="return-value"></a>傳回值  
 如果方法成功，則 S_OK。
