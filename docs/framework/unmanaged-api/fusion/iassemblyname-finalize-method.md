# <a name="iassemblynamefinalize-method"></a>IAssemblyName::Finalize 方法
這可讓[IAssemblyName](../../../../docs/framework/unmanaged-api/fusion/iassemblyname-interface.md)物件釋放資源並呼叫其解構函式之前執行其他清除作業。  
  
## <a name="syntax"></a>語法  
  
```  
HRESULT Finalize ();  
```
