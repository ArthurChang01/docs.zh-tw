# <a name="getappidauthority-function"></a>GetAppIdAuthority 函式

## <a name="syntax"></a>語法  
  
```  
HRESULT GetAppIdAuthority (  
    [out] IAppIdAuthority **ppIAppIdAuthority  
 );  
```  
  
#### <a name="parameters"></a>參數  
 `ppIAppIdAuthority`  
 [out]傳回`IAppIdAuthority`指標。
