# <a name="iassemblyenum-interface"></a>IAssemblyEnum 介面
表示陣列的列舉程式`IAssemblyName`物件。  
  
## <a name="methods"></a>方法  
  
|方法|描述|  
|------------|-----------------|  
|[Clone 方法](../../../../docs/framework/unmanaged-api/fusion/iassemblyenum-clone-method.md)|建立這個的淺層複製`IAssemblyEnum`物件。|  
|[GetNextAssembly 方法](../../../../docs/framework/unmanaged-api/fusion/iassemblyenum-getnextassembly-method.md)|取得下一個指標`IAssemblyName`包含在此`IAssemblyEnum`物件。|  
|[Reset 方法](../../../../docs/framework/unmanaged-api/fusion/iassemblyenum-reset-method.md)|重設這個`IAssemblyEnum`起始位置的物件。|
