# <a name="iassemblyenum-interface"></a>IAssemblyEnum 介面
表示 `IAssemblyName` 物件陣列的列舉值。  
  
## <a name="methods"></a>方法  
  
|方法|描述|  
|------------|-----------------|  
|[Clone 方法](iassemblyenum-clone-method.md)|建立這個 `IAssemblyEnum` 物件的淺層複本。|  
|[GetNextAssembly 方法](iassemblyenum-getnextassembly-method.md)|取得包含在這個 `IAssemblyEnum` 物件中的下一個 `IAssemblyName` 的指標。|  
|[Reset 方法](iassemblyenum-reset-method.md)|將這個 `IAssemblyEnum` 物件重設為其開始位置。|
