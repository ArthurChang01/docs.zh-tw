# <a name="iassemblynamesetproperty-method"></a>IAssemblyName::SetProperty 方法
設定指定的屬性的識別項所參考之屬性的值。  
  
## <a name="syntax"></a>語法  
  
```  
HRESULT SetProperty (  
    [in] DWORD  PropertyId,  
    [in] LPVOID pvProperty,  
    [in] DWORD  cbProperty  
);  
```  
  
#### <a name="parameters"></a>參數  
 `PropertyId`  
 [in]屬性會設定其值唯一識別項。  
  
 `pvProperty`  
 [in]要用來設定所參考之屬性的值`PropertyId`。  
  
 `cbProperty`  
 [in]大小，以位元組為單位的`pvProperty`。
