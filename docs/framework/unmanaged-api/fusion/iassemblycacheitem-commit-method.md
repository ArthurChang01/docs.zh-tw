# <a name="iassemblycacheitemcommit-method"></a>IAssemblyCacheItem::Commit 方法
將緩存的程式集引用提交到記憶體。  
  
## <a name="syntax"></a>語法  
  
```cpp  
HRESULT Commit (  
    [in] DWORD dwFlags,
    [out, optional] ULONG *pulDisposition  
);  
```  
  
## <a name="parameters"></a>參數  
 `dwFlags`  
 [在]在 Fusion.idl 中定義的標誌。  
  
 `pulDisposition`  
 [退出，可選]指示操作結果的值。
