# <a name="iassemblycacheitemcommit-method"></a>IAssemblyCacheItem::Commit 方法
認可記憶體的快取元件參考。  
  
## <a name="syntax"></a>語法  
  
```cpp  
HRESULT Commit (  
    [in] DWORD dwFlags,   
    [out, optional] ULONG *pulDisposition  
);  
```  
  
## <a name="parameters"></a>參數  
 `dwFlags`  
 在在融合 .idl 中定義的旗標。  
  
 `pulDisposition`  
 [out，optional]值，指出作業的結果。
