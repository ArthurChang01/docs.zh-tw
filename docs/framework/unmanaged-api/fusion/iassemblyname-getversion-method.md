# <a name="iassemblynamegetversion-method"></a>IAssemblyName::GetVersion 方法
取得這個[IAssemblyName](iassemblyname-interface.md)物件所參考之元件的版本資訊。  
  
## <a name="syntax"></a>語法  
  
```cpp  
HRESULT GetVersion (  
    [out] LPDWORD pdwVersionHi,  
    [out] LPDWORD pdwVersionLow  
);  
```  
  
## <a name="parameters"></a>參數  
 `pdwVersionHi`  
 脫銷版本的高32位。  
  
 `pdwVersionLow`  
 脫銷版本的低32位。
