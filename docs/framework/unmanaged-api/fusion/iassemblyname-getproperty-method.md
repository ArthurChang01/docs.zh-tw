# <a name="iassemblynamegetproperty-method"></a>IAssemblyName::GetProperty 方法
取得指定的屬性的識別項所參考之屬性的指標。  
  
## <a name="syntax"></a>語法  
  
```  
HRESULT GetProperty (  
    [in]      DWORD    PropertyId,  
    [out]     LPVOID   pvProperty,  
    [in, out] LPDWORD  pcbProperty  
);  
```  
  
#### <a name="parameters"></a>參數  
 `PropertyId`  
 [in]要求屬性的唯一識別項。  
  
 `pvProperty`  
 [out]傳回的屬性資料。  
  
 `pcbProperty`  
 [in、 out]大小，以位元組為單位的`pvProperty`。
