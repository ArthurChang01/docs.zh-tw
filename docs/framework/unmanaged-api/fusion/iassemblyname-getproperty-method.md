# <a name="iassemblynamegetproperty-method"></a>IAssemblyName::GetProperty 方法
取得指定的屬性識別碼所參考之屬性的指標。  
  
## <a name="syntax"></a>語法  
  
```cpp  
HRESULT GetProperty (  
    [in]      DWORD    PropertyId,  
    [out]     LPVOID   pvProperty,  
    [in, out] LPDWORD  pcbProperty  
);  
```  
  
## <a name="parameters"></a>參數  
 `PropertyId`  
 在所要求之屬性的唯一識別碼。  
  
 `pvProperty`  
 脫銷傳回的屬性資料。  
  
 `pcbProperty`  
 [in、out]`pvProperty`的大小（以位元組為單位）。
