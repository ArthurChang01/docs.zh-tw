# <a name="igchostcollect-method"></a>IGCHost::Collect 方法
強制發生指定的層代，不論目前的記憶體回收集合的狀態。  
  
## <a name="syntax"></a>語法  
  
```  
HRESULT Collect (  
    [in] LONG Generation  
);  
```  
  
## <a name="parameters"></a>參數  
 `Generation`  
 [in]要執行記憶體回收層代。 -1 值表示所有層代將會進行記憶體回收。
