# <a name="_corimageunloading-function"></a>_CorImageUnloading 函式
卸載託管模組映射時通知載入程式。  
  
 此函數並未實作。 如果調用，它將返回E_NOTIMPL。  
  
## <a name="syntax"></a>語法  
  
```cpp  
STDAPI (VOID) _CorImageUnloading(
   [in] PVOID* ImageBase  
);  
```  
  
## <a name="parameters"></a>參數  
 `ImageBase`  
 [在]指向要卸載的圖像的起始位置的指標。
