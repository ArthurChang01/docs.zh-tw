# <a name="corimageunloading-function"></a>_CorImageUnloading 函式
卸載 managed 的模組映像時，請通知載入器。  
  
 未實作此函式。 如果呼叫，它會傳回 E_NOTIMPL。  
  
## <a name="syntax"></a>語法  
  
```  
STDAPI (VOID) _CorImageUnloading(   
   [in] PVOID* ImageBase  
);  
```  
  
#### <a name="parameters"></a>參數  
 `ImageBase`  
 [in]要卸載之影像的起始位置指標。
