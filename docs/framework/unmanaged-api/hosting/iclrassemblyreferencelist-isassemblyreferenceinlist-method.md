# <a name="iclrassemblyreferencelistisassemblyreferenceinlist-method"></a>ICLRAssemblyReferenceList::IsAssemblyReferenceInList 方法
取得值，指出是否在清單中的組件參考提供的指標。  
  
## <a name="syntax"></a>語法  
  
```  
HRESULT IsAssemblyReferenceInList (  
    [in] IUnknown *pName  
);  
```  
  
#### <a name="parameters"></a>參數  
 `pName`  
 [in]要搜尋的組件介面指標。 有效值為型別的`IAssemblyName`或`IReferenceIdentity`。  
  
## <a name="return-value"></a>傳回值  
  
|HRESULT|描述|  
|-------------|-----------------|  
|S_OK|字串會出現在清單中。|  
|S_FALSE|字串不在清單中。|  
|E_FAIL|發生未知的嚴重失敗。 方法會傳回 E_FAIL 之後，就不再可在處理程序中使用通用語言執行平台。 裝載方法的後續呼叫會傳回 HOST_E_CLRNOTAVAILABLE。|
