# <a name="cormarkthreadinthreadpool-function"></a>CorMarkThreadInThreadPool 函式
標記目前正在執行的執行緒集區執行緒，以便執行 managed 程式碼。 從.NET Framework 2.0 版開始，此函式沒有任何作用。 它不是必要項目，並可以從您的程式碼中移除。此函式已被取代[!INCLUDE[net_v40_long](../../../../includes/net-v40-long-md.md)]。  
  
## <a name="syntax"></a>語法  
  
```  
void CorMarkThreadInThreadPool ();  
```
