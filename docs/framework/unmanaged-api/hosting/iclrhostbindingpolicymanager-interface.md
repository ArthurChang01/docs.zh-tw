# <a name="iclrhostbindingpolicymanager-interface"></a>ICLRHostBindingPolicyManager 介面
提供方法，讓主機評估目前的系結原則，並傳達指定元件的原則變更。  
  
## <a name="methods"></a>方法  
  
|方法|描述|  
|------------|-----------------|  
|[EvaluatePolicy 方法](../../../../docs/framework/unmanaged-api/hosting/iclrhostbindingpolicymanager-evaluatepolicy-method.md)|代表主機評估系結原則。|  
|[ModifyApplicationPolicy 方法](../../../../docs/framework/unmanaged-api/hosting/iclrhostbindingpolicymanager-modifyapplicationpolicy-method.md)|修改指定元件的系結原則，並建立新版本的原則。|
