# <a name="couninitializecor-function"></a>CoUninitializeCor 函式
`CoUninitializeCor` 已經過時。  
  
## <a name="syntax"></a>語法  
  
```  
STDAPI_(void) CoUninitializeCor(void);  
```  
  
## <a name="remarks"></a>備註  
 Common language runtime 無法從處理序卸載。 若要完全移除執行中處理序的執行階段，您必須關閉該程序。
