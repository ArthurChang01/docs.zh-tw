# <a name="imanagedobjectgetserializedbuffer-method"></a>IManagedObject::GetSerializedBuffer 方法
取得這個 managed 物件的字串表示。  
  
## <a name="syntax"></a>語法  
  
```cpp  
HRESULT GetSerializedBuffer (  
    [out] BSTR *pBSTR  
);  
```  
  
## <a name="parameters"></a>參數  
 `pBSTR`  
 脫銷序列化物件之字串的指標。  
  
## <a name="remarks"></a>備註  
 `GetSerializedBuffer` 方法會序列化物件，以便將其封送處理至用戶端。
