# <a name="hosting-interfaces"></a>裝載介面
本節描述 unmanaged 介面主機可用來將 common language runtime (CLR) 整合到他們的應用程式。  
  
 .NET Framework 2.0 版的裝載介面會取代.NET Framework 1.0 和 1.1 版介面。 有兩個集合的介面之間有顯著的差異。 混用它們可能會造成非預期的行為，並不建議使用。  
  
 .NET Framework 3.0 和 3.5 版使用.NET Framework 2.0 版裝載介面，並不會產生任何裝載的功能。  
  
 .NET Framework 4 裝載介面會取代.NET Framework 2.0 介面。
