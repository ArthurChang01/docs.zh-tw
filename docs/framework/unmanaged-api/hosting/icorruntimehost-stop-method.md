# <a name="icorruntimehoststop-method"></a>ICorRuntimeHost::Stop 方法
停止目前的處理序的執行階段中的程式碼執行。  
  
## <a name="syntax"></a>語法  
  
```  
HRESULT Stop ();  
```  
  
## <a name="return-value"></a>傳回值  
  
|HRESULT|描述|  
|-------------|-----------------|  
|S_OK|此作業成功。|  
|S_FALSE|作業無法完成。|  
|E_FAIL|發生不明、 重大失敗。 如果方法會傳回 E_FAIL，common language runtime (CLR) 不再使用舊處理序中。 任何裝載 api 的後續呼叫會傳回 HOST_E_CLRNOTAVAILABLE。|  
|HOST_E_CLRNOTAVAILABLE|不到程序中，載入 CLR 或 CLR 處於的狀態不能在其中執行 managed 程式碼，或程序呼叫成功。|  
  
## <a name="remarks"></a>備註  
 通常不需要呼叫`Stop`方法，因為程式碼就會停止執行處理序結束時。  
  
> [!NOTE]
>  若要在呼叫之後`Stop`，CLR 無法重新初始化成相同的程序。
