# <a name="iapartmentcallback-interface"></a>IApartmentCallback 介面
提供方法來進行回呼的 apartment 中。 *Apartment*是這些物件共用相同的執行緒存取需求的程序內的邏輯容器。  
  
## <a name="methods"></a>方法  
  
|方法|描述|  
|------------|-----------------|  
|[DoCallback 方法](../../../../docs/framework/unmanaged-api/hosting/iapartmentcallback-docallback-method.md)|執行指定的函式中的 apartment。|
