# <a name="iobjecthandle-interface"></a>IObjectHandle 介面
提供方法，以解除包裝的傳值方式封送處理物件，從間接取值。  
  
## <a name="methods"></a>方法  
  
|方法|描述|  
|------------|-----------------|  
|[Unwrap 方法](../../../../docs/framework/unmanaged-api/hosting/iobjecthandle-unwrap-method.md)|間接取值的傳值方式封送處理物件會解除包裝。|
