# <a name="iobjecthandle-interface"></a>IObjectHandle 介面

|方法|描述|  
|------------|-----------------|  
|[Unwrap 方法](../../../../docs/framework/unmanaged-api/hosting/iobjecthandle-unwrap-method.md)|從間接取值解除封送處理傳值物件。|
