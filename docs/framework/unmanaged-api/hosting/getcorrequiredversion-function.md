# <a name="getcorrequiredversion-function"></a>GetCORRequiredVersion 函式
取得所需的 common language runtime （CLR）版本號碼。  
  
 此函式在 .NET Framework 4 中已被取代。  
  
## <a name="syntax"></a>語法  
  
```cpp  
HRESULT GetCORRequiredVersion (  
    [out] LPWSTR   pbuffer,  
    [in]  DWORD    cchBuffer,  
    [out] DWORD   *dwLength  
);  
```  
  
## <a name="parameters"></a>參數  
 `pbuffer`  
 脫銷包含指定版本號碼之字串的緩衝區。  
  
 `cchBuffer`  
 在緩衝區的大小（以位元組為單位）。  
  
 `dwLength`  
 脫銷緩衝區中傳回的位元組數目。
