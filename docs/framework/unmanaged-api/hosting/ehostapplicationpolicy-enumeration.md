# <a name="ehostapplicationpolicy-enumeration"></a>EHostApplicationPolicy 列舉
指出如何修改[IHostAssemblyManager](../../../../docs/framework/unmanaged-api/hosting/ihostassemblymanager-interface.md)介面物件。 這個列舉型別已被取代。  
  
## <a name="syntax"></a>語法  
  
```  
typedef enum _HostApplicationPolicy {  
    HOST_APPLICATION_BINDING_POLICY = 1  
} EHostApplicationPolicy;  
```
