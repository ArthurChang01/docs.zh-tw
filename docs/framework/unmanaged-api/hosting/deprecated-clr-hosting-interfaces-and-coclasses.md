# <a name="deprecated-clr-hosting-interfaces-and-coclasses"></a>已被取代的 CLR 裝載介面和 Coclass
本節描述 unmanaged 介面主機可用來整合 common language runtime (CLR) 在.NET Framework 1.0 和 1.1 版到其應用程式。 這些介面提供的主機，若要設定及執行階段載入處理序的方法。  
  
## <a name="in-this-section"></a>本節內容  
 IAppDomainSetup  
 提供方法來設定主機<xref:System.AppDomain>。  
  
 [ICeeFileGen 類別](../../../../docs/framework/unmanaged-api/hosting/iceefilegen-class.md)  
 （已過時）提供建立原生可攜式執行檔 (PE) 的功能。  
  
 [ICorRuntimeHost 介面](../../../../docs/framework/unmanaged-api/hosting/icorruntimehost-interface.md)  
 提供方法來進行 CLR 設定主應用程式。
