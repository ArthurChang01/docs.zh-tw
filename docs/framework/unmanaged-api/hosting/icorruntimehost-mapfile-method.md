# <a name="icorruntimehostmapfile-method"></a>ICorRuntimeHost::MapFile 方法
將指定的檔案對應到記憶體中。 這個方法已過時。  
  
## <a name="syntax"></a>語法  
  
```  
HRESULT MapFile(  
    [in]  HANDLE    hFile,  
    [out] HMODULE*  hMapAddress  
);  
```  
  
#### <a name="parameters"></a>參數  
 `hFile`  
 [in]要對應的檔案控制代碼。  
  
 `hMapAddress`  
 [out]要開始進行對應的檔案開始的記憶體位址。
