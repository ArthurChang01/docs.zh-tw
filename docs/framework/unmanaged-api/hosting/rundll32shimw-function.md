# <a name="rundll32shimw-function"></a>RunDll32ShimW 函式
執行指定命令。  
  
 此函式已被取代，在.NET Framework 4。  
  
## <a name="syntax"></a>語法  
  
```cpp  
HRESULT RunDll32ShimW (  
    [in] HWND        hwnd,  
    [in] HINSTANCE   hinst,  
    [in] LPCWSTR     lpszCmdLine,  
    [in] int         nCmdShow  
);  
```  
  
## <a name="parameters"></a>參數  
 `hwnd`  
 [in]將在其中顯示命令輸出的視窗控制代碼。  
  
 `hinst`  
 [in]包含命令的文件庫控制代碼。  
  
 `lpszCmdLine`  
 [in]字串，指定要執行的命令。  
  
 `nCmdShow`  
 [in]整數，指定 [輸出] 視窗的顯示模式。
