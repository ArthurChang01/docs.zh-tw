# <a name="rundll32shimw-function"></a>RunDll32ShimW 函式
執行指定命令。  
  
 此函式中已被取代[!INCLUDE[net_v40_long](../../../../includes/net-v40-long-md.md)]。  
  
## <a name="syntax"></a>語法  
  
```  
HRESULT RunDll32ShimW (  
    [in] HWND        hwnd,  
    [in] HINSTANCE   hinst,  
    [in] LPCWSTR     lpszCmdLine,  
    [in] int         nCmdShow  
);  
```  
  
#### <a name="parameters"></a>參數  
 `hwnd`  
 [in]會顯示命令輸出的視窗控制代碼。  
  
 `hinst`  
 [in]包含命令的文件庫的控制代碼。  
  
 `lpszCmdLine`  
 [in]字串，指定要執行的命令。  
  
 `nCmdShow`  
 [in]整數，指定 [輸出] 視窗的顯示模式。
