# <a name="ivalidator-interface"></a>IValidator 介面
提供方法來驗證的可攜式執行檔 (PE) 映像和報告驗證錯誤。  
  
## <a name="methods"></a>方法  
  
|方法|描述|  
|------------|-----------------|  
|Validate|驗證指定的 PE 或 Microsoft intermediate language (MSIL) 檔案。|  
|FormatEventInfo|取得對應至指定的驗證錯誤的錯誤訊息。|
