# <a name="ecustomdumpitemkind-enumeration"></a>ECustomDumpItemKind 列舉
保留供未來的延伸模組[CustomDumpItem](../../../../docs/framework/unmanaged-api/hosting/customdumpitem-structure.md)結構。  
  
## <a name="syntax"></a>語法  
  
```  
typedef enum {  
    DUMP_ITEM_None = 0  
} ECustomDumpItemKind;  
```  
  
## <a name="members"></a>成員  
  
|成員|描述|  
|------------|-----------------|  
|`DUMP_ITEM_None`|保留供未來使用。|
