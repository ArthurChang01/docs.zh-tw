# <a name="ecustomdumpitemkind-enumeration"></a>ECustomDumpItemKind 列舉
保留以供未來[CustomDumpItem](../../../../docs/framework/unmanaged-api/hosting/customdumpitem-structure.md)結構的延伸。  
  
## <a name="syntax"></a>語法  
  
```cpp  
typedef enum {  
    DUMP_ITEM_None = 0  
} ECustomDumpItemKind;  
```  
  
## <a name="members"></a>Members  
  
|成員|描述|  
|------------|-----------------|  
|`DUMP_ITEM_None`|保留供未來使用。|
