# <a name="iclrtaskmanagersetlocale-method"></a>ICLRTaskManager::SetLocale 方法
通知主機已修改的地區設定識別碼 （這會對應至地理文化特性和語言） 目前正在執行的工作上值的 common language runtime (CLR)。  
  
## <a name="syntax"></a>語法  
  
```  
HRESULT SetLocale (  
    [in] LCID lcid  
);  
```  
  
#### <a name="parameters"></a>參數  
 `lcid`  
 [in]可對應至新指派的地理文化特性和語言的地區設定識別碼值。  
  
## <a name="return-value"></a>傳回值  
  
|HRESULT|描述|  
|-------------|-----------------|  
|S_OK|方法成功傳回。|  
|HOST_E_CLRNOTAVAILABLE|CLR 尚未載入到處理程序，或 CLR 正在中它無法執行 managed 程式碼，或成功地處理呼叫的狀態。|  
|HOST_E_TIMEOUT|呼叫已逾時。|  
|HOST_E_NOT_OWNER|呼叫端未擁有鎖定。|  
|HOST_E_ABANDONED|事件已取消時封鎖的執行緒或 fiber 等候它。|  
|E_FAIL|發生未知的嚴重失敗。 方法會傳回 E_FAIL CLR 已不再可用的處理序內。 裝載方法的後續呼叫會傳回 HOST_E_CLRNOTAVAILABLE。|  
  
## <a name="remarks"></a>備註  
 `SetLocale` 可讓主應用程式執行的地區設定同步處理可能會有任何機制。
