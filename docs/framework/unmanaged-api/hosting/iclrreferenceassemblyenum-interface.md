# <a name="iclrreferenceassemblyenum-interface"></a>ICLRReferenceAssemblyEnum 介面
提供方法，讓主應用程式管理的檔案或使用組件識別資料，而不需要建立，或了解這些識別的 common language runtime (CLR)，內部資料流所參考的組件集。
