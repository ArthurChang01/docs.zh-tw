# <a name="iactiononclrevent-interface"></a>IActionOnCLREvent 介面
提供[iactiononclrevent:: Onevent](../../../../docs/framework/unmanaged-api/hosting/iactiononclrevent-onevent-method.md)方法，用於執行所使用的呼叫已註冊的事件回呼[iclroneventmanager:: Registeractiononevent](../../../../docs/framework/unmanaged-api/hosting/iclroneventmanager-registeractiononevent-method.md)方法。  
  
## <a name="methods"></a>方法  
  
|方法|描述|  
|------------|-----------------|  
|[OnEvent 方法](../../../../docs/framework/unmanaged-api/hosting/iactiononclrevent-onevent-method.md)|執行已註冊的事件回呼。|
