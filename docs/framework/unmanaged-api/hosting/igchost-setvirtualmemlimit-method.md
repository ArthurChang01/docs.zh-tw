# <a name="igchostsetvirtualmemlimit-method"></a>IGCHost::SetVirtualMemLimit 方法
設定執行時間虛擬記憶體的大小上限。  
  
## <a name="syntax"></a>語法  
  
```cpp  
HRESULT SetVirtualMemLimit (  
    [in] SIZE_T sztMaxVirtualMemMB  
);  
```  
  
## <a name="parameters"></a>參數  
 `sztMaxVirtualMemMB`  
 在執行時間虛擬記憶體的大小上限（以 mb 為單位）。  
  
## <a name="remarks"></a>備註  
 可以動態變更執行時間虛擬記憶體的大小上限。
