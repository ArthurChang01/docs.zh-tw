# <a name="iobjecthandleunwrap-method"></a>IObjectHandle::Unwrap 方法
解除包裝從間接取值的傳值方式封送處理物件。  
  
## <a name="syntax"></a>語法  
  
```  
HRESULT Unwrap (  
    [out, retval] VARIANT *ppv  
);  
```  
  
#### <a name="parameters"></a>參數  
 `ppv`  
 [out]要未包裝的物件指標。
