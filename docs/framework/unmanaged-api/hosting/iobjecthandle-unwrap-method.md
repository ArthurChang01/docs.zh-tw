# <a name="iobjecthandleunwrap-method"></a>IObjectHandle::Unwrap 方法
從間接取值解除封送處理傳值物件。  
  
## <a name="syntax"></a>語法  
  
```cpp  
HRESULT Unwrap (  
    [out, retval] VARIANT *ppv  
);  
```  
  
## <a name="parameters"></a>參數  
 `ppv`  
 脫銷要解除包裝之物件的指標。
