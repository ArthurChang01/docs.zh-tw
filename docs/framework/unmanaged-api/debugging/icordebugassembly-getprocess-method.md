# <a name="icordebugassemblygetprocess-method"></a>ICorDebugAssembly::GetProcess 方法
取得這個 ICorDebugAssembly 執行個體正在執行的程序的介面指標。  
  
## <a name="syntax"></a>語法  
  
```  
HRESULT GetProcess (  
    [out] ICorDebugProcess **ppProcess  
);  
```  
  
#### <a name="parameters"></a>參數  
 `ppProcess`  
 [out]ICorDebugProcess 介面代表程序的指標。
