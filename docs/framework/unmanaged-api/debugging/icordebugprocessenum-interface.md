# <a name="icordebugprocessenum-interface"></a>ICorDebugProcessEnum Interface
會執行 ICorDebugEnum 方法並列舉 ICorDebugProcess 陣列。  
  
## <a name="methods"></a>方法  
  
|方法|描述|  
|------------|-----------------|  
|[Next 方法](icordebugprocessenum-next-method.md)|從列舉中取得指定數目的 `ICorDebugProcess` 實例，從目前位置開始。|
