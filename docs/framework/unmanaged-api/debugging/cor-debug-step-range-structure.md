# <a name="cordebugsteprange-structure"></a>COR_DEBUG_STEP_RANGE 結構
包含程式碼範圍的位移資訊。  
  
 這個結構由[icordebugstepper:: Steprange](../../../../docs/framework/unmanaged-api/debugging/icordebugstepper-steprange-method.md)方法。  
  
## <a name="syntax"></a>語法  
  
```  
typedef struct {  
    ULONG32 startOffset;  
    ULONG32 endOffset;  
} COR_DEBUG_STEP_RANGE;  
```  
  
## <a name="members"></a>成員  
  
|成員|說明|  
|------------|-----------------|  
|`startOffset`|範圍開頭的位移。|  
|`endOffset`|範圍結尾的位移。|
