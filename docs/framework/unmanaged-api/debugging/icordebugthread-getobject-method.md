# <a name="icordebugthreadgetobject-method"></a>ICorDebugThread::GetObject 方法
取得 common language runtime (CLR) 執行緒的介面指標。  
  
## <a name="syntax"></a>語法  
  
```  
HRESULT GetObject (  
    [out] ICorDebugValue   **ppObject  
);  
```  
  
#### <a name="parameters"></a>參數  
 `ppObject`  
 [out]ICorDebugValue 介面物件，表示在 CLR 執行緒的位址指標。
