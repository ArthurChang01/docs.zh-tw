# <a name="icordebugenumerateprocesses-method"></a>ICorDebug::EnumerateProcesses 方法
取得正在進行偵錯的處理程序中的列舉值。  
  
## <a name="syntax"></a>語法  
  
```  
HRESULT EnumerateProcesses (  
    [out] ICorDebugProcessEnum      **ppProcess  
);  
```  
  
#### <a name="parameters"></a>參數  
 `ppProcess`  
 ICorDebugProcessEnum 物件的列舉程式進行偵錯的處理程序的位址指標。
