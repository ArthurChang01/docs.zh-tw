# <a name="icordebugchaingetprevious-method"></a>ICorDebugChain::GetPrevious 方法
取得執行緒中先前的框架鏈結。  
  
## <a name="syntax"></a>語法  
  
```  
HRESULT GetPrevious (  
    [out] ICorDebugChain     **ppChain  
);  
```  
  
#### <a name="parameters"></a>參數  
 `ppChain`  
 [out]ICorDebugChain 物件，表示先前的這個執行緒的框架鏈結的位址指標。 如果這個鏈結是第一個鏈結`ppChain`為 null。
