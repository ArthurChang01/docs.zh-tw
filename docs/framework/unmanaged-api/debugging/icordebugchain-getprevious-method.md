# <a name="icordebugchaingetprevious-method"></a>ICorDebugChain::GetPrevious 方法
取得執行緒先前的框架鏈結。  
  
## <a name="syntax"></a>語法  
  
```cpp  
HRESULT GetPrevious (  
    [out] ICorDebugChain     **ppChain  
);  
```  
  
## <a name="parameters"></a>參數  
 `ppChain`  
 [out]ICorDebugChain 物件，表示這個執行緒的畫面格的上一個鏈結的位址指標。 如果這個鏈結的第一個鏈結，`ppChain`為 null。
