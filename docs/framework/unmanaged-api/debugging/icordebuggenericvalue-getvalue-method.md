# <a name="icordebuggenericvaluegetvalue-method"></a>ICorDebugGenericValue::GetValue 方法
將此泛型的值複製到指定的緩衝區。  
  
## <a name="syntax"></a>語法  
  
```cpp  
HRESULT GetValue (  
    [out] void     *pTo  
);  
```  
  
## <a name="parameters"></a>參數  
 `pTo`  
 [out]此 ICorDebugGenericValue 物件所代表的值的指標。 值可能是簡單類型或參考類型 （也就是指標）。
