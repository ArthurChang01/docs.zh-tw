# <a name="icordebugframegetchain-method"></a>ICorDebugFrame::GetChain 方法
取得此框架所屬之鏈的指標。  
  
## <a name="syntax"></a>語法  
  
```cpp  
HRESULT GetChain (  
    [out] ICorDebugChain     **ppChain  
);  
```  
  
## <a name="parameters"></a>參數  
 `ppChain`  
 脫銷ICorDebugChain 物件位址的指標，表示包含此框架的鏈。
