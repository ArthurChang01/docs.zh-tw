# <a name="icordebugframegetchain-method"></a>ICorDebugFrame::GetChain 方法
取得此範圍是一部分的鏈結的指標。  
  
## <a name="syntax"></a>語法  
  
```cpp  
HRESULT GetChain (  
    [out] ICorDebugChain     **ppChain  
);  
```  
  
## <a name="parameters"></a>參數  
 `ppChain`  
 [out]ICorDebugChain 物件，表示包含此框架鏈結的位址指標。
