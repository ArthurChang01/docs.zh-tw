# <a name="icordebugframegetchain-method"></a>ICorDebugFrame::GetChain 方法
取得此框架就是鏈結的指標。  
  
## <a name="syntax"></a>語法  
  
```  
HRESULT GetChain (  
    [out] ICorDebugChain     **ppChain  
);  
```  
  
#### <a name="parameters"></a>參數  
 `ppChain`  
 [out]ICorDebugChain 物件，代表包含此框架鏈結的位址指標。
