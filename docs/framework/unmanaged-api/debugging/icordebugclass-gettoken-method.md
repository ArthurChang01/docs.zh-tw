# <a name="icordebugclassgettoken-method"></a>ICorDebugClass::GetToken 方法
取得參考此類別定義的 `TypeDef` 中繼資料 token。  
  
## <a name="syntax"></a>語法  
  
```cpp  
HRESULT GetToken (  
    [out] mdTypeDef          *pTypeDef  
);  
```  
  
## <a name="parameters"></a>參數  
 `pTypeDef`  
 脫銷參考此類別定義之 `mdTypeDef` token 的指標。
