# <a name="icordebugclassgettoken-method"></a>ICorDebugClass::GetToken 方法
取得`TypeDef`參考此類別定義的中繼資料語彙基元。  
  
## <a name="syntax"></a>語法  
  
```  
HRESULT GetToken (  
    [out] mdTypeDef          *pTypeDef  
);  
```  
  
## <a name="parameters"></a>參數  
 `pTypeDef`  
 [out]指標`mdTypeDef`參考此類別定義的語彙基元。
