# <a name="icordebugvaluebreakpointgetvalue-method"></a>ICorDebugValueBreakpoint::GetValue 方法

## <a name="syntax"></a>語法  
  
```  
HRESULT GetValue (  
    [out] ICorDebugValue   **ppValue  
);  
```  
  
#### <a name="parameters"></a>參數  
 `ppValue`  
 [out]位址指標`ICorDebugValue`物件。
