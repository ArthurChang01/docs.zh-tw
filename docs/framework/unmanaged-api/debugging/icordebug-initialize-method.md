# <a name="icordebuginitialize-method"></a>ICorDebug::Initialize 方法
初始化 `ICorDebug` 物件。  
  
## <a name="syntax"></a>語法  
  
```  
HRESULT Initialize ();  
```  
  
## <a name="remarks"></a>備註  
 偵錯工具必須呼叫`Initialize`在建立服務的時間來初始化偵錯。 必須在任何其他方法之前呼叫此方法`ICorDebug`呼叫。
