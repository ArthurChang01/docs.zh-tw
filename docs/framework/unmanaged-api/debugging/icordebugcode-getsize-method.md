# <a name="icordebugcodegetsize-method"></a>ICorDebugCode::GetSize 方法
取得大小，以位元組為單位，此 「 ICorDebugCode"所表示的二進位程式碼。  
  
## <a name="syntax"></a>語法  
  
```cpp  
HRESULT GetSize (  
    [out] ULONG32    *pcBytes  
);  
```  
  
## <a name="parameters"></a>參數  
 `pcBytes`  
 [out]指標大小 （位元組），二進位檔的程式碼這個`ICorDebugCode`物件表示。
