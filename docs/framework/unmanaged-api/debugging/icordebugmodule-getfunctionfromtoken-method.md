# <a name="icordebugmodulegetfunctionfromtoken-method"></a>ICorDebugModule::GetFunctionFromToken 方法
取得指定中繼資料語彙基元的函式。  
  
## <a name="syntax"></a>語法  
  
```  
HRESULT GetFunctionFromToken(  
    [in] mdMethodDef methodDef,  
    [out] ICorDebugFunction **ppFunction  
);  
```  
  
#### <a name="parameters"></a>參數  
 `methodDef`  
 [in]A`mdMethodDef`參考函式的中繼資料的中繼資料語彙基元。  
  
 `ppFunction`  
 [out]ICorDebugFunction 介面物件，表示函式的位址指標。  
  
## <a name="remarks"></a>備註  
 `GetFunctionFromToken`如果傳入的值，方法會傳回 CORDBG_E_FUNCTION_NOT_IL HRESULT`methodDef`未參考 Microsoft intermediate language (MSIL) 方法。
