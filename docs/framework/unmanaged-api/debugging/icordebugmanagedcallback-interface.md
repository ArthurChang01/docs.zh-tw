 <span data-ttu-id="3b94b-161">序列化，相同的執行緒中呼叫和以同步處理的狀態中的程序呼叫所有的回呼。</span><span class="sxs-lookup"><span data-stu-id="3b94b-161">All callbacks are serialized, called in the same thread, and called with the process in the synchronized state.</span></span>
