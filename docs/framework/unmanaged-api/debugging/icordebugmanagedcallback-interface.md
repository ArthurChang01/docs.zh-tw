 <span data-ttu-id="9740b-161">所有回呼都會序列化，並在相同的執行緒中呼叫，並在進程處於已同步處理狀態時呼叫。</span><span class="sxs-lookup"><span data-stu-id="9740b-161">All callbacks are serialized, called in the same thread, and called with the process in the synchronized state.</span></span>
