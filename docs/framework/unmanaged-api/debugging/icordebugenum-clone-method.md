# <a name="icordebugenumclone-method"></a>ICorDebugEnum::Clone 方法
建立這個 ICorDebugEnum 物件的複本。  
  
## <a name="syntax"></a>語法  
  
```  
HRESULT Clone (  
    [out] ICorDebugEnum **ppEnum  
);  
```  
  
## <a name="parameters"></a>參數  
 `ppEnum`  
 [out]位址指標`ICorDebugEnum`物件，這個複本`ICorDebugEnum`物件。
