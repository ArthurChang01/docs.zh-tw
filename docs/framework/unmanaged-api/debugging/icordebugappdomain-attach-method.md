# <a name="icordebugappdomainattach-method"></a>ICorDebugAppDomain::Attach 方法
將偵錯工具附加至應用程式定義域中。  
  
## <a name="syntax"></a>語法  
  
```  
HRESULT Attach ();  
```  
  
## <a name="remarks"></a>備註  
 偵錯工具必須連接到接收事件，並啟用偵錯之應用程式定義域的應用程式定義域中。
