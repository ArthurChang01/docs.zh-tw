# <a name="icordebugarrayvaluegetelementtype-method"></a>ICorDebugArrayValue::GetElementType 方法
取得值，指出陣列中元素的簡單類型。  
  
## <a name="syntax"></a>語法  
  
```  
HRESULT GetElementType (  
    [out] CorElementType  *pType  
);  
```  
  
## <a name="parameters"></a>參數  
 `pType`  
 [out]CorElementType 列舉型別，表示型別值的指標。
