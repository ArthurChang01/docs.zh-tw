# <a name="icordebugappdomain4-interface"></a>ICorDebugAppDomain4 介面
以邏輯方式延伸 ICorDebugAppDomain 介面，以從 COM 可呼叫包裝函式取得 Managed 物件。  
  
## <a name="method"></a>方法  
  
|方法|描述|  
|------------|-----------------|  
|[GetObjectForCCW 方法](icordebugappdomain4-getobjectforccw-method.md)|從 COM 可呼叫包裝函式 (CCW) 指標取得 Managed 物件。|
