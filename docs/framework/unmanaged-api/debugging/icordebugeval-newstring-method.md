# <a name="icordebugevalnewstring-method"></a>ICorDebugEval::NewString 方法
會配置新的字串執行個體具有指定的內容。  
  
## <a name="syntax"></a>語法  
  
```  
HRESULT NewString (  
    [in] LPCWSTR   string  
);  
```  
  
#### <a name="parameters"></a>參數  
 `string`  
 [in]字串內容的指標。  
  
## <a name="remarks"></a>備註  
 字串，永遠會建立所在的執行緒目前正在執行的應用程式定義域中。
