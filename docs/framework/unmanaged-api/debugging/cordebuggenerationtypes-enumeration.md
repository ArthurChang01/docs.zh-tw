# <a name="cordebuggenerationtypes-enumeration"></a>CorDebugGenerationTypes 列舉
指定 Managed 堆積上的記憶體區域產生方式。  
  
## <a name="syntax"></a>語法  
  
```  
typedef enum CorDebugGenerationTypes {  
    CorDebug_Gen0 = 0,  
    CorDebug_Gen1 = 1,  
    CorDebug_Gen2 = 2,  
    CorDebug_LOH  = 3,  
} CorDebugRegionTypes;  
```  
  
## <a name="members"></a>成員  
  
|成員名稱|描述|  
|-----------------|-----------------|  
|`CorDebug_Gen0`|層代 0。|  
|`CorDebug_Gen1`|層代 1。|  
|`CorDebug_Gen2`|層代 2。|  
|`CorDebug_LOH`|大型物件堆積。|
