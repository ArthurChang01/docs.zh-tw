# <a name="icordebugmodulegetassembly-method"></a>ICorDebugModule::GetAssembly 方法
取得針對此模組包含組件。  
  
## <a name="syntax"></a>語法  
  
```  
HRESULT GetAssembly(  
    [out] ICorDebugAssembly **ppAssembly  
);  
```  
  
#### <a name="parameters"></a>參數  
 `ppAssembly`  
 [out]ICorDebugAssembly 物件，代表包含此模組的組件的指標。
