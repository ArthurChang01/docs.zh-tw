# <a name="icordebugevalisactive-method"></a>ICorDebugEval::IsActive 方法
取得值，指出這個 ICorDebugEval 物件目前是否正在執行。  
  
## <a name="syntax"></a>語法  
  
```  
HRESULT IsActive (  
    [out] BOOL              *pbActive  
);  
```  
  
## <a name="parameters"></a>參數  
 `pbActive`  
 [out]值，指出這項評估是否在作用中的指標。
