# <a name="icordebugthreadgetregisterset-method"></a>ICorDebugThread::GetRegisterSet 方法
取得與這個 ICorDebugThread 物件的使用中部分相關聯的暫存器集的介面指標。  
  
## <a name="syntax"></a>語法  
  
```cpp  
HRESULT GetRegisterSet (  
    [out] ICorDebugRegisterSet **ppRegisters  
);  
```  
  
## <a name="parameters"></a>參數  
 `ppRegisters`  
 [out]位址指標[ICorDebugRegisterSet](../../../../docs/framework/unmanaged-api/debugging/icordebugregisterset-interface.md)介面的物件，代表註冊為此執行緒的使用中部分。
