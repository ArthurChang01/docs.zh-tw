# <a name="icordebugchaingetnext-method"></a>ICorDebugChain::GetNext 方法
取得執行緒的下一個框架鏈。  
  
## <a name="syntax"></a>語法  
  
```cpp  
HRESULT GetNext (  
    [out] ICorDebugChain     **ppChain  
);  
```  
  
## <a name="parameters"></a>參數  
 `ppChain`  
 脫銷ICorDebugChain 物件位址的指標，表示執行緒的下一個框架鏈。 如果這個鏈是最後一個鏈，`ppChain` 是 null。
