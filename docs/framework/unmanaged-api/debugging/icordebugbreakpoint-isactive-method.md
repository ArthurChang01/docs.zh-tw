# <a name="icordebugbreakpointisactive-method"></a>ICorDebugBreakpoint::IsActive 方法
取得值，指出是否此`ICorDebugBreakpoint`為作用中。  
  
## <a name="syntax"></a>語法  
  
```  
HRESULT IsActive (  
    [out] BOOL *pbActive  
);  
```  
  
#### <a name="parameters"></a>參數  
 `pbActive`  
 [out]`true`此中斷點為作用中; 否則如果`false`。
