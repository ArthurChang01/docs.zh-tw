# <a name="icordebugbreakpointisactive-method"></a>ICorDebugBreakpoint::IsActive 方法
取得值，指出是否此`ICorDebugBreakpoint`作用中。  
  
## <a name="syntax"></a>語法  
  
```  
HRESULT IsActive (  
    [out] BOOL *pbActive  
);  
```  
  
## <a name="parameters"></a>參數  
 `pbActive`  
 [out]`true`作用中; 否則即為此中斷點是否`false`。
