# <a name="icordebugtypegetrank-method"></a>ICorDebugType::GetRank 方法
取得陣列型別中的維度數目。  
  
## <a name="syntax"></a>語法  
  
```  
HRESULT GetRank (  
    [out] ULONG32   *pnRank  
);  
```  
  
#### <a name="parameters"></a>參數  
 `pnRank`  
 [out]維度的數目指標。
