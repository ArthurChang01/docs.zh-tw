# <a name="icordebugtypegetrank-method"></a>ICorDebugType::GetRank 方法
取得陣列類型中的維度數目。  
  
## <a name="syntax"></a>語法  
  
```cpp  
HRESULT GetRank (  
    [out] ULONG32   *pnRank  
);  
```  
  
## <a name="parameters"></a>參數  
 `pnRank`  
 脫銷維度數目的指標。
