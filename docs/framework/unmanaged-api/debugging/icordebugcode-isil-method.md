# <a name="icordebugcodeisil-method"></a>ICorDebugCode::IsIL 方法
取得值，指出是否此"ICorDebugCode 」 表示已編譯的 Microsoft 中繼語言 (MSIL) 中的程式碼。  
  
## <a name="syntax"></a>語法  
  
```  
HRESULT IsIL (  
    [out] BOOL       *pbIL  
);  
```  
  
#### <a name="parameters"></a>參數  
 `pbIL`  
 [out]`true`如果這個`ICorDebugCode`代表已編譯的 MSIL 中; 否則程式碼`false`。
