# <a name="icordebugclassgetmodule-method"></a>ICorDebugClass::GetModule 方法
取得定義此類別的模組。  
  
## <a name="syntax"></a>語法  
  
```cpp  
HRESULT GetModule (  
    [out] ICorDebugModule    **pModule  
);  
```  
  
## <a name="parameters"></a>參數  
 `pModule`  
 [out]ICorDebugModule 物件，表示這個類別的定義模組的位址指標。
