# <a name="icordebugprocessgetid-method"></a>ICorDebugProcess::GetID 方法
取得處理序的作業系統 (OS) 識別碼。  
  
## <a name="syntax"></a>語法  
  
```  
HRESULT GetID([out] DWORD *pdwProcessId);  
```  
  
#### <a name="parameters"></a>參數  
 `pdwProcessId`  
 [out]處理序的唯一識別碼。
