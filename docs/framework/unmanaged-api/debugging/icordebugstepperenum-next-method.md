# <a name="icordebugstepperenumnext-method"></a>ICorDebugStepperEnum::Next 方法
取得指定的 ICorDebugStepper 執行個體的數目從列舉型別，從目前位置開始。  
  
## <a name="syntax"></a>語法  
  
```cpp  
HRESULT Next(  
    [in] ULONG  celt,  
    [out, size_is(celt), length_is(*pceltFetched)]  
        ICorDebugStepper *steppers[],  
    [out] ULONG *pceltFetched  
);  
```  
  
## <a name="parameters"></a>參數  
 `celt`  
 [in]數目`ICorDebugStepper`要擷取的執行個體。  
  
 `steppers`  
 [out]指標的陣列，其中每一個指向`ICorDebugStepper`物件。  
  
 `pceltFetched`  
 [out]數目的指標`ICorDebugStepper`實際傳回的執行個體。 此值可能為 null 如果`celt`是其中一個。
