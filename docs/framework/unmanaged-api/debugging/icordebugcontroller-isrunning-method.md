# <a name="icordebugcontrollerisrunning-method"></a>ICorDebugController::IsRunning 方法
取得值，指出是否在處理程序中的執行緒目前正在自由。  
  
## <a name="syntax"></a>語法  
  
```  
HRESULT IsRunning (  
    [out] BOOL *pbRunning  
);  
```  
  
#### <a name="parameters"></a>參數  
 `pbRunning`  
 [out]值的指標`true`程序中的執行緒執行自由; 否則如果`false`。
