# <a name="icordebugcontrollerisrunning-method"></a>ICorDebugController::IsRunning 方法
取得值，指出是否在程序中的執行緒目前正在自由。  
  
## <a name="syntax"></a>語法  
  
```  
HRESULT IsRunning (  
    [out] BOOL *pbRunning  
);  
```  
  
#### <a name="parameters"></a>參數  
 `pbRunning`  
 [out]為值的指標`true`自由; 否則執行程序中的執行緒如果`false`。
