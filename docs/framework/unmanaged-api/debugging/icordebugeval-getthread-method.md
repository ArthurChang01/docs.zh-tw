# <a name="icordebugevalgetthread-method"></a>ICorDebugEval::GetThread 方法
取得這項評估在執行或是將執行的執行緒。  
  
## <a name="syntax"></a>語法  
  
```  
HRESULT GetThread (  
    [out] ICorDebugThread   **ppThread  
);  
```  
  
#### <a name="parameters"></a>參數  
 `ppThread`  
 [out]ICorDebugThread 物件，表示執行緒的位址指標。
