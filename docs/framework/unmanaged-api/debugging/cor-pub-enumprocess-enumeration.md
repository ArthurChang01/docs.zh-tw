# <a name="corpubenumprocess-enumeration"></a>COR_PUB_ENUMPROCESS 列舉
識別所要列舉的類型。  
  
## <a name="syntax"></a>語法  
  
```  
typedef enum {  
    COR_PUB_MANAGEDONLY    = 0x00000001  
} COR_PUB_ENUMPROCESS;  
```  
  
## <a name="members"></a>成員  
  
|成員名稱|說明|  
|-----------------|-----------------|  
|`COR_PUB_MANAGEDONLY`|受管理的程序。|  
  
## <a name="remarks"></a>備註  
 目前版本的 unmanaged 偵錯 API 列舉只受管理的處理程序。
