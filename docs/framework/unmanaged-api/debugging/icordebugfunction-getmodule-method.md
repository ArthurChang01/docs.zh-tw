# <a name="icordebugfunctiongetmodule-method"></a>ICorDebugFunction::GetModule 方法
取得定義此函式的模組。  
  
## <a name="syntax"></a>語法  
  
```  
HRESULT GetModule (  
    [out] ICorDebugModule **ppModule  
);  
```  
  
#### <a name="parameters"></a>參數  
 `ppModule`  
 [out]ICorDebugModule 物件，表示此函式定義所在模組的位址指標。
