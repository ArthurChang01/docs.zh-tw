# <a name="logswitchcallreason-enumeration"></a>LogSwitchCallReason 列舉
指出在切換偵錯/追蹤時所執行的作業。  
  
## <a name="syntax"></a>語法  
  
```cpp  
typedef enum LogSwitchCallReason {  
    SWITCH_CREATE,  
    SWITCH_MODIFY,  
    SWITCH_DELETE  
} LogSwitchCallReason;  
```  
  
## <a name="members"></a>成員  
  
|成員|描述|  
|------------|-----------------|  
|`SWITCH_CREATE`|建立偵錯/追蹤參數。|  
|`SWITCH_MODIFY`|偵錯/追蹤交換器已修改。|  
|`SWITCH_DELETE`|偵錯/追蹤參數已被刪除。|
