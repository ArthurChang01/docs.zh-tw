# <a name="icordebugcodegetaddress-method"></a>ICorDebugCode::GetAddress 方法
取得此"ICorDebugCode 」 介面代表的程式碼片段的相對虛擬位址 (RVA)。  
  
## <a name="syntax"></a>語法  
  
```  
HRESULT GetAddress (  
    [out] CORDB_ADDRESS *pStart  
);  
```  
  
#### <a name="parameters"></a>參數  
 `pStart`  
 [out]程式碼區段的 RVA 指標。
