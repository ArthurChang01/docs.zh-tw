# <a name="icordebugcodegetaddress-method"></a>ICorDebugCode::GetAddress 方法
取得此「ICorDebugCode」介面所代表之程式碼區段的相對虛擬位址（RVA）。  
  
## <a name="syntax"></a>語法  
  
```cpp  
HRESULT GetAddress (  
    [out] CORDB_ADDRESS *pStart  
);  
```  
  
## <a name="parameters"></a>參數  
 `pStart`  
 脫銷程式碼區段之 RVA 的指標。
