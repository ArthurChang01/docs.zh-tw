# <a name="icorpublishprocessenumnext-method"></a>ICorPublishProcessEnum::Next 方法
從目前的資料指標位置開始，取得集合中指定的進程數目。  
  
## <a name="syntax"></a>語法  
  
```cpp  
HRESULT Next (  
    [in] ULONG celt,  
    [out, size_is(celt), length_is(*pceltFetched)]  
        ICorPublishProcess **objects,  
    [out] ULONG *pceltFetched  
);  
```  
  
## <a name="parameters"></a>參數  
 `celt`  
 在要抓取的進程數目。  
  
 `objects`  
 脫銷已抓取[ICorPublishProcess](../../../../docs/framework/unmanaged-api/debugging/icorpublishprocess-interface.md)物件之陣列的指標，每個物件都代表一個進程。  
  
 `pceltFetched`  
 脫銷實際傳回之進程數目的指標。 如果 `celt` 是一個，這個值可能會是 null。
