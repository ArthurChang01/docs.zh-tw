# <a name="icordebugappdomaingetprocess-method"></a>ICorDebugAppDomain::GetProcess 方法
取得包含在應用程式定義域的處理序。  
  
## <a name="syntax"></a>語法  
  
```  
HRESULT GetProcess (  
    [out] ICorDebugProcess   **ppProcess  
);  
```  
  
#### <a name="parameters"></a>參數  
 `ppProcess`  
 [out]ICorDebugProcess 物件代表處理程序的位址指標。
