# <a name="icordebugcodeenum-interface"></a>ICorDebugCodeEnum 介面

會執行 "ICorDebugEnum" 方法，並列舉 "ICorDebugCode" 陣列。  
  
## <a name="methods"></a>方法  
  
|方法|描述|  
|------------|-----------------|  
|[Next 方法](icordebugcodeenum-next-method.md)|從列舉中取得指定數目的 `ICorDebugCode` 實例，從目前位置開始。|
