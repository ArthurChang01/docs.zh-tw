# <a name="debugging-unmanaged-api-reference"></a>偵錯 (Unmanaged API 參考)
偵錯 API 可讓偵錯工具偵錯在 Common Language Runtime (CLR) 環境中執行的程式碼。 要偵錯的程式碼可以是 CLR 支援的任何類型程式碼。
