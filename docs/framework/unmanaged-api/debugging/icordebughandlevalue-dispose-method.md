# <a name="icordebughandlevaluedispose-method"></a>ICorDebugHandleValue::Dispose 方法
釋放這個 ICorDebugHandleValue 物件所參考的控制碼，而不明確釋放介面指標。  
  
## <a name="syntax"></a>語法  
  
```cpp  
HRESULT Dispose ();  
```
