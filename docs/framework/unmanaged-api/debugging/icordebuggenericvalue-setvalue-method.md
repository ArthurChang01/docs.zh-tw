# <a name="icordebuggenericvaluesetvalue-method"></a>ICorDebugGenericValue::SetValue 方法
從指定的緩衝區複製的新值。  
  
## <a name="syntax"></a>語法  
  
```  
HRESULT SetValue (  
    [in] void      *pFrom  
);  
```  
  
#### <a name="parameters"></a>參數  
 `pFrom`  
 [in]要從中複製值的緩衝區指標。  
  
## <a name="remarks"></a>備註  
 對於參考型別，這個值會是參考，而不是內容。
