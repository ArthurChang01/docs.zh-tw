# <a name="codechunkinfo-structure1"></a>CodeChunkInfo 結構 1
代表記憶體中的單一程式碼區塊。  
  
## <a name="syntax"></a>語法  
  
```  
typedef struct _CodeChunkInfo {  
    CORDB_ADDRESS startAddr;  
    ULONG32       length;  
} CodeChunkInfo;  
```  
  
## <a name="members"></a>成員  
  
|成員|描述|  
|------------|-----------------|  
|`startAddr`|A`CORDB_ADDRESS`值，指定區塊的起始位址。|  
|`length`|以位元組為單位，區塊的大小。|  
  
## <a name="remarks"></a>備註  
 單一的程式碼區塊是物件的屬於程式碼，例如函式的原生程式碼的區域。
