# <a name="icordebugboxvaluegetobject-method"></a>ICorDebugBoxValue::GetObject 方法
取得 boxed 的值。  
  
## <a name="syntax"></a>語法  
  
```  
HRESULT GetObject (  
    [out] ICorDebugObjectValue **ppObject  
);  
```  
  
## <a name="parameters"></a>參數  
 `ppObject`  
 [out]ICorDebugObjectValue 物件，表示 boxed 的值的位址指標。
