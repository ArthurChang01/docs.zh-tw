# <a name="icordebugthreadgetprocess-method"></a>ICorDebugThread::GetProcess 方法

## <a name="syntax"></a>語法  
  
```  
HRESULT GetProcess (  
    [out] ICorDebugProcess   **ppProcess

```

 `ppProcess`
