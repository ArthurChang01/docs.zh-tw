# <a name="icordebugthreadgetprocess-method"></a>ICorDebugThread::GetProcess 方法
取得處理程序的此 ICorDebugThread 一部分的介面指標。  
  
## <a name="syntax"></a>語法  
  
```  
HRESULT GetProcess (  
    [out] ICorDebugProcess   **ppProcess  
);  
```  
  
#### <a name="parameters"></a>參數  
 `ppProcess`  
 [out]ICorDebugProcess 介面物件，表示處理程序的位址指標。
