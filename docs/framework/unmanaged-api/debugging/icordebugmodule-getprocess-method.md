# <a name="icordebugmodulegetprocess-method"></a>ICorDebugModule::GetProcess 方法
取得包含此模組的處理序。  
  
## <a name="syntax"></a>語法  
  
```  
HRESULT GetProcess (  
    [out] ICorDebugProcess **ppProcess  
);  
```  
  
## <a name="parameters"></a>參數  
 `ppProcess`  
 [out]ICorDebugProcess 物件，表示包含此模組的程序的位址指標。
