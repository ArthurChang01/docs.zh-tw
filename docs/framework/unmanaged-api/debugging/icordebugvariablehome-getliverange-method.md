# <a name="icordebugvariablehomegetliverange-method"></a>IcorDebugVariableHome::GetLiveRange 方法
取得原生的此變數是即時的範圍。  
  
## <a name="syntax"></a>語法  
  
```  
HRESULT GetLiveRange(  
    [out] ULONG32* pStartOffset,  
    [out] ULONG32 *pEndOffset  
);  
```  
  
#### <a name="parameters"></a>參數  
 `pStartOffset`  
 [out]變數在第一個即時邏輯的位移。  
  
 `pEndOffset`  
 [out]變數在最後一個即時點之後，立即邏輯的位移。
