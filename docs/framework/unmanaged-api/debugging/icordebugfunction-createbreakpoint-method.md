# <a name="icordebugfunctioncreatebreakpoint-method"></a>ICorDebugFunction::CreateBreakpoint 方法

## <a name="syntax"></a>語法  
  
```  
HRESULT CreateBreakpoint (  
    [out] ICorDebugFunctionBreakpoint **ppBreakpoint

```  
  
#### <a name="parameters"></a>參數  
 `ppBreakpoint`
