# <a name="iclrdatatargetsettlsvalue-method"></a>ICLRDataTarget::SetTLSValue 方法
執行緒區域儲存區 (TLS) 的目標處理序中指定的執行緒中設定的值。 這個方法是由通用語言執行平台 (CLR) 資料存取服務呼叫。  
  
## <a name="syntax"></a>語法  
  
```  
HRESULT SetTLSValue (  
    [in] ULONG32            threadID,  
    [in] ULONG32            index,  
    [in] CLRDATA_ADDRESS    value  
);  
```  
  
#### <a name="parameters"></a>參數  
 `threadID`  
 [in]目標處理序中執行緒的作業系統識別項。  
  
 `index`  
 [in]位置索引。 此值必須是指定之執行緒本機存放區中的有效索引。  
  
 `value`  
 [in]A`CLRDATA_ADDRESS`值，指定要放在給定的 TLS 位置的值。  
  
## <a name="remarks"></a>備註  
 此方法是由偵錯應用程式的作者來實作。
