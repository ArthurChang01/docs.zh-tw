# <a name="iclrdatatargetsettlsvalue-method"></a>ICLRDataTarget::SetTLSValue 方法
設定執行緒區域儲存區 (TLS) 的目標處理序中指定的執行緒中的值。 這個方法是由通用語言執行平台 (CLR) 資料存取服務呼叫。  
  
## <a name="syntax"></a>語法  
  
```  
HRESULT SetTLSValue (  
    [in] ULONG32            threadID,  
    [in] ULONG32            index,  
    [in] CLRDATA_ADDRESS    value  
);  
```  
  
## <a name="parameters"></a>參數  
 `threadID`  
 [in]目標處理序中的執行緒作業系統識別項。  
  
 `index`  
 [in]位置的索引。 此值必須是執行緒的指定本機存放區中的有效索引。  
  
 `value`  
 [in]A`CLRDATA_ADDRESS`值，指定要放置在指定的 TLS 位置的值。  
  
## <a name="remarks"></a>備註  
 此方法是由偵錯應用程式的作者來實作。
