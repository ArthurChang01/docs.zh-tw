# <a name="icordebugprocess6getcode-method"></a>ICorDebugProcess6::GetCode 方法
取得特定程式碼位址之 Managed 程式碼的相關資訊。  
  
## <a name="syntax"></a>語法  
  
```cpp  
HRESULT GetCode(  
    [in] CORDB_ADDRESS codeAddress,
    [out] ICorDebugCode **ppCode);  
```  
  
## <a name="parameters"></a>參數  
 `codeAddress`  
 [在]指定託管程式碼片段的起始位址[CORDB_ADDRESS](../../../../docs/framework/unmanaged-api/common-data-types-unmanaged-api-reference.md)值。  
  
 `ppCode`  
 [出]指向表示託管程式碼片段的"ICorDebugCode"物件位址的指標。  
  
## <a name="remarks"></a>備註  
  
> [!NOTE]
> 這個方法僅適用於 .NET Native。
