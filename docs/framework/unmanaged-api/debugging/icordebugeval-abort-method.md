# <a name="icordebugevalabort-method"></a>ICorDebugEval::Abort 方法
中止目前正在這個 ICorDebugEval 物件的計算。  
  
## <a name="syntax"></a>語法  
  
```cpp  
HRESULT Abort ();  
```  
  
## <a name="remarks"></a>備註  
 如果評估為巢狀，而且不是最新`Abort`方法可能會失敗。
