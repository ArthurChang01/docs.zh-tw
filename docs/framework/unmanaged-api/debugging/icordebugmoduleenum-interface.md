# <a name="icordebugmoduleenum-interface"></a>ICorDebugModuleEnum 介面

會執行 ICorDebugEnum 方法, 並列舉 ICorDebugModule 陣列。  
  
## <a name="methods"></a>方法  
  
|方法|描述|  
|------------|-----------------|  
|[Next 方法](../../../../docs/framework/unmanaged-api/debugging/icordebugmoduleenum-next-method.md)|從列舉中取得指定`ICorDebugModule`的實例數目, 從目前位置開始。|  
  
## <a name="remarks"></a>備註  
  
> [!NOTE]
> 這個介面不支援跨電腦或跨處理序的遠端呼叫。
