# <a name="icordebugbreakpointenum-interface"></a>ICorDebugBreakpointEnum 介面

會執行 ICorDebugEnum 方法, 並列舉 ICorDebugBreakpoint 陣列。  
  
## <a name="methods"></a>方法  
  
|方法|描述|  
|------------|-----------------|  
|[Next 方法](../../../../docs/framework/unmanaged-api/debugging/icordebugbreakpointenum-next-method.md)|從列舉中取得指定`ICorDebugBreakpoint`的實例數目, 從目前位置開始。|  
  
## <a name="remarks"></a>備註  
  
> [!NOTE]
> 這個介面不支援跨電腦或跨處理序的遠端呼叫。
