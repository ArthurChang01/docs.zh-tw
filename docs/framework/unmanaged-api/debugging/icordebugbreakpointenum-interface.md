# <a name="icordebugbreakpointenum-interface"></a>ICorDebugBreakpointEnum 介面

實作 ICorDebugEnum 方法，並列舉 ICorDebugBreakpoint 陣列。  
  
## <a name="methods"></a>方法  
  
|方法|描述|  
|------------|-----------------|  
|[Next 方法](../../../../docs/framework/unmanaged-api/debugging/icordebugbreakpointenum-next-method.md)|取得指定的數目`ICorDebugBreakpoint`從列舉型別，從目前位置開始的執行個體。|
