# <a name="icorpublishenumskip-method"></a>ICorPublishEnum::Skip 方法
將游標移往前列舉中所指定的項目數。  
  
## <a name="syntax"></a>語法  
  
```cpp  
HRESULT Skip (  
    [in] ULONG   celt  
);  
```  
  
## <a name="parameters"></a>參數  
 `celt`  
 [in]用來將游標向前移動的項目數目。
