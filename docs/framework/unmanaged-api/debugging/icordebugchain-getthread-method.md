# <a name="icordebugchaingetthread-method"></a>ICorDebugChain::GetThread 方法
取得實體的執行緒，此呼叫鏈結的一部分。  
  
## <a name="syntax"></a>語法  
  
```  
HRESULT GetThread (  
    [out] ICorDebugThread    **ppThread  
);  
```  
  
## <a name="parameters"></a>參數  
 `ppThread`  
 [out]指向 ICorDebugThread 物件，表示實體執行緒的這個呼叫鏈結是的一部分。
