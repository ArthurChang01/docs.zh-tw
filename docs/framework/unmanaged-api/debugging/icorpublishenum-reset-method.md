# <a name="icorpublishenumreset-method"></a>ICorPublishEnum::Reset 方法
將游標移至列舉的開頭。  
  
## <a name="syntax"></a>語法  
  
```  
HRESULT Reset ();  
```  
  
#### <a name="parameters"></a>參數  
 `Reset`不接受任何參數。
