# <a name="icordebugappdomainenumeratebreakpoints-method"></a>ICorDebugAppDomain::EnumerateBreakpoints 方法
取得應用程式定義域中所有使用中的中斷點列舉值。  
  
## <a name="syntax"></a>語法  
  
```cpp  
HRESULT EnumerateBreakpoints (  
    [out] ICorDebugBreakpointEnum   **ppBreakpoints  
);  
```  
  
## <a name="parameters"></a>參數  
 `ppBreakpoints`  
 [out]ICorDebugBreakpointEnum 物件，應用程式定義域中的所有作用中的中斷點的列舉值的位址指標。  
  
## <a name="remarks"></a>備註  
 列舉值會包含所有類型的中斷點，包括函式中斷點和資料中斷點。
