# <a name="icordebugstringvalue-interface"></a>ICorDebugStringValue 介面
適用于字串值之 ICorDebugHeapValue 的子類別。  
  
## <a name="methods"></a>方法  
  
|方法|描述|  
|------------|-----------------|  
|[GetLength 方法](../../../../docs/framework/unmanaged-api/debugging/icordebugstringvalue-getlength-method.md)|取得此 `ICorDebugStringValue`所參考字串中的字元數。|  
|[GetString 方法](../../../../docs/framework/unmanaged-api/debugging/icordebugstringvalue-getstring-method.md)|取得此 `ICorDebugStringValue`所參考的字串。|  
  
## <a name="remarks"></a>備註
