# <a name="icordebugframegetcode-method"></a>ICorDebugFrame::GetCode 方法
取得此堆疊框架相關聯的程式碼的指標。  
  
## <a name="syntax"></a>語法  
  
```  
HRESULT GetCode (  
    [out] ICorDebugCode      **ppCode  
);  
```  
  
#### <a name="parameters"></a>參數  
 `ppCode`  
 [out]ICorDebugCode 物件，表示這個畫面格相關聯的程式碼的位址指標。
