# <a name="icordebugarrayvaluegetcount-method"></a>ICorDebugArrayValue::GetCount 方法
取得陣列中的項目總數。  
  
## <a name="syntax"></a>語法  
  
```  
HRESULT GetCount (  
    [out] ULONG32 *pnCount  
);  
```  
  
## <a name="parameters"></a>參數  
 `pnCount`  
 [out]指標的陣列中的項目總數。
