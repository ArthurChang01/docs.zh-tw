# <a name="icordebugstepperenum-interface"></a>ICorDebugStepperEnum 介面
實作 ICorDebugEnum 方法，並列舉 ICorDebugStepper 陣列。  
  
## <a name="methods"></a>方法  
  
|方法|描述|  
|------------|-----------------|  
|[Next 方法](../../../../docs/framework/unmanaged-api/debugging/icordebugstepperenum-next-method.md)|取得指定的數目`ICorDebugStepper`從列舉型別，從目前位置開始的執行個體。|  
  
## <a name="remarks"></a>備註  
  
> [!NOTE]
>  這個介面不支援跨電腦或跨處理序的遠端呼叫。
