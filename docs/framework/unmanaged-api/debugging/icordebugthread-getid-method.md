# <a name="icordebugthreadgetid-method"></a>ICorDebugThread::GetID 方法
取得此 ICorDebugThread 的使用中部分的目前作業系統識別項。  
  
## <a name="syntax"></a>語法  
  
```  
HRESULT GetID (  
    [out] DWORD *pdwThreadId  
);  
```  
  
#### <a name="parameters"></a>參數  
 `pdwThreadId`  
 [out]執行緒的識別項。  
  
## <a name="remarks"></a>備註  
 作業系統識別項期間執行的處理序中，有可能變更，而且可以對不同的執行緒不同的值。
