# <a name="icordebugprocess3-interface"></a>ICorDebugProcess3 介面
控制自訂偵錯工具通知。  
  
## <a name="methods"></a>方法  
  
|方法|描述|  
|------------|-----------------|  
|[SetEnableCustomNotification 方法](icordebugprocess3-setenablecustomnotification-method.md)|啟用和停用指定類型的自訂偵錯工具通知。|  
  
## <a name="remarks"></a>備註  
 這個介面會以邏輯方式擴充 ICorDebugProcess 和 ICorDebugProcess2 介面。  
  
> [!NOTE]
> 這個介面不支援跨電腦或跨處理序的遠端呼叫。
