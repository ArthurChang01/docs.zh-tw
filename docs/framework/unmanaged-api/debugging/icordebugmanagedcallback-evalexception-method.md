# <a name="icordebugmanagedcallbackevalexception-method"></a>ICorDebugManagedCallback::EvalException 方法
告知偵錯工具評估已結束，發生未處理的例外狀況。  
  
## <a name="syntax"></a>語法  
  
```  
HRESULT EvalException (  
    [in] ICorDebugAppDomain *pAppDomain,  
    [in] ICorDebugThread    *pThread,  
    [in] ICorDebugEval      *pEval  
);  
```  
  
#### <a name="parameters"></a>參數  
 `pAppDomain`  
 [in]代表應用程式定義域中結束評估 ICorDebugAppDomain 物件的指標。  
  
 `pThread`  
 [in]表示評估已終止執行緒 ICorDebugThread 物件的指標。  
  
 `pEval`  
 [in]表示執行評估的程式碼 ICorDebugEval 物件的指標。
