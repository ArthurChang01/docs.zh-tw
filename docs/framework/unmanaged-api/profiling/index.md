 [分析概觀](../../../../docs/framework/unmanaged-api/profiling/profiling-overview.md) 說明 CLR 提供來支援在 .NET Framework 環境中進行分析的服務與介面。

 [CLR 分析工具與 Windows Store 應用程式](../../../../docs/framework/unmanaged-api/profiling/clr-profilers-and-windows-store-apps.md) 討論如何移植取用 CLR 分析 API 的診斷工具移植以成功搭配 Windows Store 應用程式使用。

 [分析全域靜態函式](../../../../docs/framework/unmanaged-api/profiling/profiling-global-static-functions.md) 說明分析 API 所使用的非受控全域靜態函式。

 [分析列舉](../../../../docs/framework/unmanaged-api/profiling/profiling-enumerations.md) 說明分析 API 所使用的非受控列舉。

 [分析結構](../../../../docs/framework/unmanaged-api/profiling/profiling-structures.md) 說明分析 API 所使用的非受控結構。
