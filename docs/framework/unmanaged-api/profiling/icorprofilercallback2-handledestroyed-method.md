# <a name="icorprofilercallback2handledestroyed-method"></a>ICorProfilerCallback2::HandleDestroyed 方法
通知記憶體回收控制代碼已終結的程式碼剖析工具。  
  
## <a name="syntax"></a>語法  
  
```  
HRESULT HandleDestroyed(  
    [in] GCHandleID handleId);  
```  
  
#### <a name="parameters"></a>參數  
 `handleId`  
 [in]進行記憶體回收控制代碼的識別碼。
