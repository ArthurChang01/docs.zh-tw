# <a name="iclrprofiling-interface"></a>ICLRProfiling 介面

## <a name="methods"></a>方法  
  
|方法|描述|  
|------------|-----------------|  
|[AttachProfiler 方法](../../../../docs/framework/unmanaged-api/profiling/iclrprofiling-attachprofiler-method.md)|將指定的程式碼剖析工具附加至指定的處理序。|
