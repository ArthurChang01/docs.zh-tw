# <a name="icorprofilercallback2threadnamechanged-method"></a>ICorProfilerCallback2::ThreadNameChanged 方法
通知程式碼分析工具，執行緒的名稱已變更。  
  
## <a name="syntax"></a>語法  
  
```cpp  
HRESULT ThreadNameChanged(  
    [in] ThreadID threadId,  
    [in] ULONG cchName,  
    [in] WCHAR name[]);  
```  
  
## <a name="parameters"></a>參數  
 `threadId`  
 在執行緒的識別碼。  
  
 `cchName`  
 在執行緒新名稱的長度。  
  
 `name`  
 在執行緒的新名稱。 名稱不是以 null 結束。
