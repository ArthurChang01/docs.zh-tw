# <a name="icorprofilerobjectenumreset-method"></a>ICorProfilerObjectEnum::Reset 方法
Moves this enumerator's cursor to the starting position of the sequence.  
  
## <a name="syntax"></a>語法  
  
```cpp  
HRESULT Reset ();  
```
