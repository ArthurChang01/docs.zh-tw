# <a name="corprfcodeinfo-structure"></a>COR_PRF_CODE_INFO 結構
代表儲存在記憶體中的一個機器碼連續區塊。  
  
## <a name="syntax"></a>語法  
  
```  
typedef struct _COR_PRF_CODE_INFO {  
    UINT_PTR startAddress;  
    SIZE_T size;  
} COR_PRF_CODE_INFO;  
```  
  
## <a name="members"></a>成員  
  
|成員|描述|  
|------------|-----------------|  
|`startAddress`|程式碼的連續區塊的開始位址。|  
|`size`|區塊大小。|
