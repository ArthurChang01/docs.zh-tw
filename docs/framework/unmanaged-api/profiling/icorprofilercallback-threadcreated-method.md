# <a name="icorprofilercallbackthreadcreated-method"></a>ICorProfilerCallback::ThreadCreated 方法
通知分析工具已建立一個執行緒。  
  
## <a name="syntax"></a>語法  
  
```  
HRESULT ThreadCreated(  
    [in] ThreadID threadId);   
```  
  
#### <a name="parameters"></a>參數  
 `threadId`  
 [in]已建立的執行緒識別碼。  
  
## <a name="remarks"></a>備註  
 `threadId`值是立即有效。
