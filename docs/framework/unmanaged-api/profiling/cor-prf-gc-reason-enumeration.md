# <a name="cor_prf_gc_reason-enumeration"></a>COR_PRF_GC_REASON 列舉
指出正在發生之記憶體回收的原因。  
  
## <a name="syntax"></a>語法  
  
```cpp  
typedef enum {  
    COR_PRF_GC_INDUCED = 1,  
    COR_PRF_GC_OTHER = 0  
} COR_PRF_GC_REASON;  
```  
  
## <a name="members"></a>Members  
  
|成員|描述|  
|------------|-----------------|  
|`COR_PRF_GC_INDUCED`|垃圾收集是由 <xref:System.GC.Collect%2A> 方法所引發。|  
|`COR_PRF_GC_OTHER`|這是未指定的原因。|
