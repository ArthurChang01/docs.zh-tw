# <a name="corprfgcreason-enumeration"></a>COR_PRF_GC_REASON 列舉
指出正在發生之記憶體回收的原因。  
  
## <a name="syntax"></a>語法  
  
```cpp  
typedef enum {  
    COR_PRF_GC_INDUCED = 1,  
    COR_PRF_GC_OTHER = 0  
} COR_PRF_GC_REASON;  
```  
  
## <a name="members"></a>成員  
  
|成員|說明|  
|------------|-----------------|  
|`COR_PRF_GC_INDUCED`|記憶體回收所引起的<xref:System.GC.Collect%2A>方法。|  
|`COR_PRF_GC_OTHER`|未指定原因。|
