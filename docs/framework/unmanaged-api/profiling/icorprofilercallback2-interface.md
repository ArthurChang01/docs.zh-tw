## <a name="remarks"></a><span data-ttu-id="a270d-128">備註</span><span class="sxs-lookup"><span data-stu-id="a270d-128">Remarks</span></span>  
 <span data-ttu-id="a270d-129">CLR 會呼叫 `ICorProfilerCallback` （或 `ICorProfilerCallback2`）介面中的方法，以便在分析工具已訂閱的事件發生時，通知分析工具。</span><span class="sxs-lookup"><span data-stu-id="a270d-129">The CLR calls a method in the `ICorProfilerCallback` (or `ICorProfilerCallback2`) interface to notify the profiler when an event, to which the profiler had subscribed, occurs.</span></span> <span data-ttu-id="a270d-130">這是 CLR 用來與程式碼 profiler 通訊的主要回呼介面。</span><span class="sxs-lookup"><span data-stu-id="a270d-130">This is the primary callback interface through which the CLR communicates with the code profiler.</span></span>  
  
 <span data-ttu-id="a270d-131">程式碼分析工具必須執行 `ICorProfilerCallback` 介面的方法。</span><span class="sxs-lookup"><span data-stu-id="a270d-131">A code profiler must implement the methods of the `ICorProfilerCallback` interface.</span></span> <span data-ttu-id="a270d-132">針對 .NET Framework 2.0 和更新版本，分析工具也必須執行 `ICorProfilerCallback2` 方法。</span><span class="sxs-lookup"><span data-stu-id="a270d-132">For the .NET Framework 2.0 and later versions, the profiler must also implement the `ICorProfilerCallback2` methods.</span></span> <span data-ttu-id="a270d-133">每個方法的執行都必須傳回 HRESULT，其值為 [成功] 或 [失敗時 E_FAIL] S_OK。</span><span class="sxs-lookup"><span data-stu-id="a270d-133">Each method implementation must return an HRESULT having a value of S_OK on success or E_FAIL on failure.</span></span> <span data-ttu-id="a270d-134">目前，CLR 會忽略[ICorProfilerCallback：： ObjectReferences](../../../../docs/framework/unmanaged-api/profiling/icorprofilercallback-objectreferences-method.md)以外的每個回呼所傳回的 HRESULT。</span><span class="sxs-lookup"><span data-stu-id="a270d-134">Currently, the CLR ignores the HRESULT that is returned by each callback except [ICorProfilerCallback::ObjectReferences](../../../../docs/framework/unmanaged-api/profiling/icorprofilercallback-objectreferences-method.md).</span></span>  
  
 <span data-ttu-id="a270d-135">程式碼分析工具必須在 Microsoft Windows 登錄（它是執行 `ICorProfilerCallback` 和 `ICorProfilerCallback2` 介面的 COM 物件）中註冊。</span><span class="sxs-lookup"><span data-stu-id="a270d-135">A code profiler must register in the Microsoft Windows registry, its COM object that implements the `ICorProfilerCallback` and `ICorProfilerCallback2` interfaces.</span></span> <span data-ttu-id="a270d-136">程式碼分析工具會透過呼叫[ICorProfilerInfo：： SetEventMask](../../../../docs/framework/unmanaged-api/profiling/icorprofilerinfo-seteventmask-method.md)，訂閱想要接收通知的事件。</span><span class="sxs-lookup"><span data-stu-id="a270d-136">A code profiler subscribes to the events for which it wants to receive notification by calling [ICorProfilerInfo::SetEventMask](../../../../docs/framework/unmanaged-api/profiling/icorprofilerinfo-seteventmask-method.md).</span></span> <span data-ttu-id="a270d-137">這通常會在分析工具的[ICorProfilerCallback：： Initialize](../../../../docs/framework/unmanaged-api/profiling/icorprofilercallback-initialize-method.md)的執行中完成。</span><span class="sxs-lookup"><span data-stu-id="a270d-137">This is usually done in the profiler's implementation of [ICorProfilerCallback::Initialize](../../../../docs/framework/unmanaged-api/profiling/icorprofilercallback-initialize-method.md).</span></span> <span data-ttu-id="a270d-138">然後，分析工具就能夠在事件即將發生或發生在執行中的執行時間進程時，從執行時間接收通知。</span><span class="sxs-lookup"><span data-stu-id="a270d-138">The profiler is then able to receive notification from the runtime when an event is about to occur or has just occurred in an executing runtime process.</span></span>  
  
> [!NOTE]
> <span data-ttu-id="a270d-139">Profiler 會註冊單一 COM 物件。</span><span class="sxs-lookup"><span data-stu-id="a270d-139">The profiler registers a single COM object.</span></span> <span data-ttu-id="a270d-140">如果分析工具的目標是 .NET Framework 版本1.0 或1.1，則該 COM 物件只需要執行 `ICorProfilerCallback`的方法。</span><span class="sxs-lookup"><span data-stu-id="a270d-140">If the profiler is targeting .NET Framework version 1.0 or 1.1, that COM object need only implement the methods of `ICorProfilerCallback`.</span></span> <span data-ttu-id="a270d-141">如果它的目標是 .NET Framework 2.0 版和更新版本，則 COM 物件也必須執行 `ICorProfilerCallback2`的方法。</span><span class="sxs-lookup"><span data-stu-id="a270d-141">If it is targeting .NET Framework version 2.0 and later, the COM object must also implement the methods of `ICorProfilerCallback2`.</span></span>
