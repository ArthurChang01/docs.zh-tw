# <a name="itypelibresolver-interface"></a>ITypeLibResolver 介面
提供[ResolveTypeLib 方法](resolvetypelib-method.md)，它會解析型別程式庫的檔案路徑。  
  
## <a name="methods"></a>方法  
  
|方法|描述|  
|------------|-----------------|  
|[ResolveTypeLib 方法](resolvetypelib-method.md)|藉由傳回完整的路徑，解析類型程式庫的簡單名稱。|
