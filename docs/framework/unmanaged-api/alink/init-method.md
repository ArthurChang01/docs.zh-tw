# <a name="init-method"></a>Init 方法
準備實作物件[IALink 介面](../../../../docs/framework/unmanaged-api/alink/ialink-interface.md)供使用。  
  
## <a name="syntax"></a>語法  
  
```  
HRESULT Init(  
    IMetaDataDispenserEx* pDispenser,  
    IMetaDataError* pErrorHandler  
) PURE;  
```  
  
#### <a name="parameters"></a>參數  
 `pDispenser`  
 [IMetaDataDispenserEx 介面](../../../../docs/framework/unmanaged-api/metadata/imetadatadispenserex-interface.md)指標的中繼資料分配程式。  
  
 `pErrorHandler`  
 [IMetaDataError 介面](../../../../docs/framework/unmanaged-api/metadata/imetadataerror-interface.md)選擇性錯誤處理介面指標。  
  
## <a name="return-value"></a>傳回值  
 如果方法成功則傳回 S_OK。  
  
## <a name="requirements"></a>需求  
 需要 alink.h
