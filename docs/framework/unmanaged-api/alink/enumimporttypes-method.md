# <a name="enumimporttypes-method"></a>EnumImportTypes 方法
列舉每個範圍中的每個型別。  
  
## <a name="syntax"></a>語法  
  
```  
HRESULT EnumImportTypes(  
    HALINKENUM   hEnum,  
    DWORD        dwMax,  
    mdTypeDef    aTypeDefs[],  
    DWORD*       pdwCount  
) PURE;  
```  
  
#### <a name="parameters"></a>參數  
 `hEnum`  
 列舉值的控制代碼。  
  
 `dwMax`  
 擷取類型的最大數目。  
  
 `aTypeDefs`  
 將收到輸入權杖，不能超過`dwMax`。  
  
 `pdwCount`  
 接收的類型中實際數目`aTypeDefs`。  
  
## <a name="return-value"></a>傳回值  
 如果方法成功則傳回 S_OK。  
  
## <a name="requirements"></a>需求  
 需要 alink.h
