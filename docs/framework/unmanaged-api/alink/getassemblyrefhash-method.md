# <a name="getassemblyrefhash-method"></a>GetAssemblyRefHash 方法
擷取指定的組件的雜湊 blob。  
  
## <a name="syntax"></a>語法  
  
```  
HRESULT GetAssemblyRefHash(  
    mdToken FileToken,  
    const void** ppvHash,  
    DWORD* pcbHash  
) PURE;  
```  
  
#### <a name="parameters"></a>參數  
 `FileToken`  
 要雜湊會參考的組件識別碼。  
  
 `ppvHash`  
 接收所產生的雜湊 blob。  
  
 `pcbHash`  
 接收大小，以位元組為單位的雜湊 blob。  
  
## <a name="return-value"></a>傳回值  
 如果方法成功，則會傳回 S_OK。  
  
## <a name="requirements"></a>需求  
 需要 alink.h
