# <a name="closeassembly-method"></a>CloseAssembly 方法
完成組件作業。 開始新的組件或未繫結的模組之前呼叫這個方法。  
  
## <a name="syntax"></a>語法  
  
```  
HRESULT CloseAssembly(  
    mdAssembly AssemblyID  
) PURE;  
```  
  
#### <a name="parameters"></a>參數  
 `AssemblyID`  
 組件的識別碼。  
  
## <a name="return-value"></a>傳回值  
 如果方法成功，則會傳回 S_OK。  
  
## <a name="requirements"></a>需求  
 需要 alink.h。
