# <a name="setnonassemblyflags-method"></a>SetNonAssemblyFlags 方法
設定不是組件特定的旗標。  
  
## <a name="syntax"></a>語法  
  
```  
HRESULT SetNonAssemblyFlags(  
    AssemblyFlags afFlags  
) PURE;  
```  
  
#### <a name="parameters"></a>參數  
 `afFlags`  
 ALink 旗標。  
  
## <a name="return-value"></a>傳回值  
 如果方法成功則傳回 S_OK。  
  
## <a name="requirements"></a>需求  
 需要 alink.h
