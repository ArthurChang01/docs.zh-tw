# <a name="enumcustomattributes-method"></a>EnumCustomAttributes 方法
抓取元件層級的自訂屬性。  
  
## <a name="syntax"></a>語法  
  
```cpp  
HRESULT EnumCustomAttributes(  
    HALINKENUM hEnum,  
    mdToken tkType,  
    mdCustomAttribute rCustomValues[],  
    ULONG cMax,  
    ULONG* pcCustomValues  
) PURE;  
```  
  
## <a name="parameters"></a>參數  
 `hEnum`  
 列舉值的控制碼。  
  
 `tkType`  
 要列舉的屬性類型。 `mdTokenNill`用於所有屬性。  
  
 `rCustomValues`  
 接收自訂屬性標記。  
  
 `cMax`  
 指定陣列的`rCustomValues`大小。  
  
 `pcCustomValues`  
 選擇性地接收權杖值的計數。  
  
## <a name="return-value"></a>傳回值  
 如果方法成功，則傳回 S_OK。  
  
## <a name="requirements"></a>需求  
 需要 alink. h
