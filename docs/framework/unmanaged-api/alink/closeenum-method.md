# <a name="closeenum-method"></a>CloseEnum 方法
關閉指定的列舉型別，並釋放相關聯的資源。  
  
## <a name="syntax"></a>語法  
  
```  
HRESULT CloseEnum(  
    HALINKENUM hEnum  
) PURE;  
```  
  
#### <a name="parameters"></a>參數  
 `hEnum`  
 要關閉的列舉控制代碼。  
  
## <a name="return-value"></a>傳回值  
 如果方法成功則傳回 S_OK。  
  
## <a name="requirements"></a>需求  
 需要 alink.h
