# <a name="closeenum-method"></a>CloseEnum 方法
關閉指示的列舉，並釋出相關聯的資源。  
  
## <a name="syntax"></a>語法  
  
```cpp  
HRESULT CloseEnum(  
    HALINKENUM hEnum  
) PURE;  
```  
  
## <a name="parameters"></a>參數  
 `hEnum`  
 要關閉之列舉的控制碼。  
  
## <a name="return-value"></a>傳回值  
 如果方法成功，則傳回 S_OK。  
  
## <a name="requirements"></a>需求  
 需要 alink. h
