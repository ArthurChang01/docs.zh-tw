## <a name="delegates"></a><span data-ttu-id="e4d69-108">委派</span><span class="sxs-lookup"><span data-stu-id="e4d69-108">Delegates</span></span>

- <span data-ttu-id="e4d69-121">建立委派時，會對委派建立者的授權集執行委派目標方法上的安全性連結要求。</span><span class="sxs-lookup"><span data-stu-id="e4d69-121">When a delegate is created, security link demands on the delegate target method are performed against the grant set of the delegate creator.</span></span>  <span data-ttu-id="e4d69-122">無法滿足安全性動作會導致 <xref:System.Security.SecurityException>。</span><span class="sxs-lookup"><span data-stu-id="e4d69-122">Failure to satisfy the security action results in a <xref:System.Security.SecurityException>.</span></span>

## <a name="link-demands-and-wrappers"></a><span data-ttu-id="e4d69-126">連結要求和包裝函式</span><span class="sxs-lookup"><span data-stu-id="e4d69-126">Link demands and wrappers</span></span>  
 <span data-ttu-id="e4d69-127">我們已在安全性基礎結構中增強連結要求的特殊保護案例，但它仍然是您的程式碼中潛在弱點的來源。</span><span class="sxs-lookup"><span data-stu-id="e4d69-127">A special protection case with link demands has been strengthened in the security infrastructure, but it is still a source of possible weakness in your code.</span></span>
