# <a name="net-framework-guide"></a>.NET Framework 指南

> [!NOTE]
> 此 .NET Framework 內容集包含 .NET Framework 4.5 到 4.8 版的資訊。 要下載 .NET 框架，請參閱[安裝 .NET 框架](./install/guide-for-developers.md)。 有關 .NET 框架中的新功能和更改的清單，請參閱[.NET 框架中的新增功能](./whats-new/index.md)和更改。 如需支援平台的清單，請參閱 [.NET Framework 系統需求](./get-started/system-requirements.md)。 有關 .NET 框架的舊版本的文檔，請參閱[.NET 早期版本文檔](https://docs.microsoft.com/previous-versions/dotnet/)。

.NET 框架是一個開發平臺，用於構建 Web、Windows、Windows 伺服器和 Microsoft Azure 的應用。 這是由通用語言執行平台 (CLR) 和 .NET Framework 類別庫 (包括各種功能且支援許多產業標準) 所組成。

.NET Framework 提供許多服務，包括記憶體管理、類型和記憶體安全、安全性、網路和應用程式部署。 它提供了簡單好用的資料結構和 API，以抽取出較低層級的 Windows 作業系統。 您可以將不同的程式設計語言與 .NET 框架一起使用，包括 C#、F# 和視覺化基本。

有關 .NET 框架的一般介紹，請參閱[入門](./get-started/index.md)。 有關 .NET Framework 的體系結構和關鍵功能的介紹，請參閱[概述](./get-started/overview.md)。

.NET 框架可用於 Docker 和[Windows 容器](/virtualization/windowscontainers/about/)。

## <a name="installation"></a>安裝

.NET 框架附帶 Windows，它使您能夠運行 .NET 框架應用程式。 您可能需要比 Windows 版本附帶的更高版本的 .NET 框架版本。 有關詳細資訊，請參閱在[Windows 上安裝 .NET 框架](./install/index.md)。

要瞭解如何修復 .NET 框架安裝（如果您在安裝過程中遇到錯誤），請參閱修復[.NET 框架](./install/repair.md)。

有關下載 .NET 框架的更多詳細資訊，請參閱[為開發人員安裝 .NET 框架](./install/guide-for-developers.md)。
