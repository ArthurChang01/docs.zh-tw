### <a name="step-1-define-the-sessionful-wcf-service-interface-and-implementation"></a><span data-ttu-id="9b4ae-184">步驟 1：定義工作階段 WCF 服務介面與實作</span><span class="sxs-lookup"><span data-stu-id="9b4ae-184">Step 1: Define the Sessionful WCF service interface and implementation</span></span>
