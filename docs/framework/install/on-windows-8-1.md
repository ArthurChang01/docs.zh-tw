# <a name="install-the-net-framework-on-windows-81-and-windows-server-2012-r2"></a>在 Windows 8.1 和 Windows Server 2012 R2 上安裝.NET Framework

.NET Framework 才能在 Windows 上執行許多應用程式。 您可以使用下列指示來安裝它。 您可能已到達此頁面上之後嘗試執行的應用程式，並在您的電腦上看到下列對話方塊。

![無法啟動此應用程式](./media/this-application-could-not-be-started.png)

這些指示將協助您安裝您所需要的.NET Framework 版本。 [.NET Framework 4.7.1](https://www.microsoft.com/en-us/download/details.aspx?id=56115&desc=dotnet47)是最新版本。 Windows 8.1 和 Windows Server 2012 R2 支援並包含[Windows 10 年秋季建立者更新](https://www.microsoft.com/software-download/windows10)和[Windows Server 2016 版本 1709年](https://docs.microsoft.com/windows-server/get-started/get-started-with-1709)。

## <a name="net-framework-471"></a>.NET framework 4.7.1

* [下載.NET Framework 4.7.1](https://www.microsoft.com/net/framework/versions/net471?utm_source=ms-docs&utm_medium=referral)

[.NET Framework 4.7.1](https://www.microsoft.com/en-us/download/details.aspx?id=56115&desc=dotnet47)可以用來執行建置.NET Framework 4.0 或更新版本的應用程式。

## <a name="net-framework-35"></a>.NET Framework 3.5

請依照指示安裝[Windows 8.1 及 10 的.NET Framework 3.5](dotnet-35-windows-10.md)。

.NET Framework 3.5 支援針對 .NET Framework 1.0 到 3.5 建置的應用程式。

## <a name="help"></a>說明

如果您無法安裝正確的 .NET Framework 版本，可以[連絡 Microsoft 以取得協助](mailto:dotnet-install-help@service.microsoft.com?subject=Install-Help)。
