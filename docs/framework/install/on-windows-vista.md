# <a name="install-the-net-framework-on-windows-vista-and-windows-server-2008"></a>在 Windows Vista 和 Windows Server 2008 上安裝.NET Framework

.NET Framework 才能在 Windows 上執行許多應用程式。 您可以使用下列指示來安裝它。 您可能已到達此頁面上之後嘗試執行的應用程式，並在您的電腦上看到下列對話方塊。

![無法啟動此應用程式](./media/this-application-could-not-be-started.png)

這些指示將協助您安裝您所需要的.NET Framework 版本。 [.NET Framework 4.7.1](https://www.microsoft.com/en-us/download/details.aspx?id=56115&desc=dotnet47)是最新版本。 不支援在 Windows Vista 和 Windows Server 2008 上。 其隨附於[Windows 10 年秋季建立者更新](https://www.microsoft.com/software-download/windows10)和[Windows Server 2016 版本 1709年](https://docs.microsoft.com/windows-server/get-started/get-started-with-1709)。

## <a name="net-framework-46"></a>.NET Framework 4.6

[.NET Framework 4.6](https://www.microsoft.com/en-us/download/details.aspx?id=48130&e6b34bbe-475b-1abd-2c51-b5034bcdd6d2=True)是最新支援.NET Framework 版本 Windows Vista 和 Windows Server 2008 上。

[.NET Framework 4.6](https://www.microsoft.com/en-us/download/details.aspx?id=48130&e6b34bbe-475b-1abd-2c51-b5034bcdd6d2=True)支援建置之.NET Framework 4.0 到 4.6 的應用程式。

## <a name="net-framework-35"></a>.NET Framework 3.5

您可以在 Windows Vista 上安裝 [.NET Framework 3.5](http://go.microsoft.com/fwlink/?LinkID=213834&dotnetdocs)。

.NET Framework 3.5 支援針對 .NET Framework 1.0 到 3.5 建置的應用程式。
