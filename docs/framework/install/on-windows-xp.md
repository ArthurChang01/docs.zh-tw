# <a name="install-the-net-framework-on-windows-xp-and-windows-server-2003"></a>在 Windows XP 和 Windows Server 2003 上安裝.NET Framework

> [!NOTE]
> Microsoft 已不再支援 Windows XP。 我們建議您升級為 Windows 10，而且會包含最新版本的.NET framework 支援。 本文件僅供用來作為有幫助的疑難排解指南。

.NET Framework 才能在 Windows 上執行許多應用程式。 您可以使用下列指示來安裝它。 您可能已到達此頁面上之後嘗試執行的應用程式，並在您的電腦上看到下列對話方塊。

![無法啟動此應用程式](./media/this-application-could-not-be-started.png)

這些指示將協助您安裝您所需要的.NET Framework 版本。 [.NET Framework 4.7.1](https://www.microsoft.com/en-us/download/details.aspx?id=56115&desc=dotnet47)是最新版本。 不支援在 Windows XP 和 Windows Server 2003 上。 其隨附於[Windows 10 年秋季建立者更新](https://www.microsoft.com/software-download/windows10)和[Windows Server 2016 版本 1709年](https://docs.microsoft.com/windows-server/get-started/get-started-with-1709)。

## <a name="net-framework-403"></a>.NET Framework 4.0.3

[.NET Framework 4.0.3](http://go.microsoft.com/fwlink/?LinkID=213834)是最新支援.NET Framework 版本 Windows XP 和 Windows Server 2003。 .NET Framework 4.0.3 會要求必須先安裝 [.NET Framework 4](http://go.microsoft.com/fwlink/?LinkID=213834)。 這兩個 .NET Framework 版本都已不再受 Microsoft 支援。

## <a name="net-framework-4"></a>.NET Framework 4

您可以在 Windows XP 上安裝 [.NET Framework 4](http://go.microsoft.com/fwlink/?LinkID=213834&dotnetdocs)。 Microsoft 已不再支援它。

## <a name="net-framework-35"></a>.NET Framework 3.5

您可以在 Windows XP 上安裝 [.NET Framework 3.5](http://go.microsoft.com/fwlink/?LinkID=213834&dotnetdocs)。

.NET Framework 3.5 可用來執行針對 .NET Framework 1.0 到 3.5 建置的應用程式。
