<span data-ttu-id="709cc-103">.NET Client Profile 是 .NET Framework 的子集，隨 [!INCLUDE[net_v40_short](../../../includes/net-v40-short-md.md)] (含) 以前版本提供，並已針對用戶端應用程式最佳化。</span><span class="sxs-lookup"><span data-stu-id="709cc-103">The .NET Client Profile is a subset of the .NET Framework, which was provided with [!INCLUDE[net_v40_short](../../../includes/net-v40-short-md.md)] and earlier versions and was optimized for client applications.</span></span> <span data-ttu-id="709cc-104">.NET Framework 是 Windows、Windows Phone 和 Microsoft Azure 的開發平台，並提供 Managed 應用程式執行環境和 .NET Framework 類別庫。</span><span class="sxs-lookup"><span data-stu-id="709cc-104">The .NET Framework is a development platform for Windows, Windows Phone and Microsoft Azure and provides a managed app execution environment and the .NET Framework class library.</span></span> <span data-ttu-id="709cc-105">[!INCLUDE[net_v40_short](../../../includes/net-v40-short-md.md)] (含) 以前版本提供了兩種部署選項：完整 .NET Framework 和 Client Profile。</span><span class="sxs-lookup"><span data-stu-id="709cc-105">The [!INCLUDE[net_v40_short](../../../includes/net-v40-short-md.md)] and earlier versions provided two deployment options: the full .NET Framework and the Client Profile.</span></span> <span data-ttu-id="709cc-106">Client Profile 提供比完整 .NET Framework 更快速的部署方式，產生的應用程式安裝套件也更小。</span><span class="sxs-lookup"><span data-stu-id="709cc-106">The Client Profile enabled faster deployment and smaller app installation packages than the full .NET Framework.</span></span>  
  
 <span data-ttu-id="709cc-107">從 [!INCLUDE[net_v45](../../../includes/net-v45-md.md)] 開始，Client Profile 已經停止提供，並且將只提供完整的可轉散發套件。</span><span class="sxs-lookup"><span data-stu-id="709cc-107">Starting with the [!INCLUDE[net_v45](../../../includes/net-v45-md.md)], the Client Profile has been discontinued and only the full redistributable package is available.</span></span> <span data-ttu-id="709cc-108">[!INCLUDE[net_v45](../../../includes/net-v45-md.md)] 提供的最佳化像是較小的下載量和較快的部署速度，已滿足了對個別部署套件的需求。</span><span class="sxs-lookup"><span data-stu-id="709cc-108">Optimizations provided by the [!INCLUDE[net_v45](../../../includes/net-v45-md.md)], such as smaller download size and faster deployment, have eliminated the need for a separate deployment package.</span></span> <span data-ttu-id="709cc-109">單一可轉散發套件可以使安裝程序順暢執行，並且簡化應用程式的部署選項。</span><span class="sxs-lookup"><span data-stu-id="709cc-109">The single redistributable streamlines the installation process and simplifies your app's deployment options.</span></span>  
  
 <span data-ttu-id="709cc-110">不過，如果您是以 .NET Framework 4 或 3.5 為目標，而且想要進一步了解 Client Profile 以及使用時機，請參閱 .NET Framework 4 文件中的 [.NET Framework Client Profile](https://docs.microsoft.com/previous-versions/dotnet/netframework-4.0/cc656912%28v=vs.100%29)。</span><span class="sxs-lookup"><span data-stu-id="709cc-110">However, if you are targeting the .NET Framework 4 or 3.5 and want to learn more about the Client Profile and when to use it, see [.NET Framework Client Profile](https://docs.microsoft.com/previous-versions/dotnet/netframework-4.0/cc656912%28v=vs.100%29) in the .NET Framework 4 documentation.</span></span>  
  
 <span data-ttu-id="709cc-111">當您安裝 [!INCLUDE[net_v45](../../../includes/net-v45-md.md)] 時，.NET Framework 4 Client Profile 會更新為 .NET Framework 的完整版。</span><span class="sxs-lookup"><span data-stu-id="709cc-111">When you install the [!INCLUDE[net_v45](../../../includes/net-v45-md.md)], the .NET Framework 4 Client Profile is updated to the full version of the .NET Framework.</span></span> <span data-ttu-id="709cc-112">如需安裝 [!INCLUDE[net_v45](../../../includes/net-v45-md.md)] 的資訊，請參閱[安裝適用於開發人員的 .NET Framework](../../../docs/framework/install/guide-for-developers.md)。</span><span class="sxs-lookup"><span data-stu-id="709cc-112">For information about installing the [!INCLUDE[net_v45](../../../includes/net-v45-md.md)], see [Install the .NET Framework for developers](../../../docs/framework/install/guide-for-developers.md).</span></span>
