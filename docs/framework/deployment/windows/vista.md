# <a name="installing-the-net-framework-on-windows-vista"></a>在 Windows Vista 上安裝 .NET Framework

您可能需要 .NET Framework，才能在 Windows Vista 上執行應用程式。 您可以使用下列指示來協助您。 這些指示取決於您所需的 .NET Framework 版本。

## <a name="net-framework-461-and-later"></a>.NET Framework 4.6.1 和更新版本

您無法在 Windows Vista 上安裝 .NET Framework 4.6.1 或更新版本。 取而代之的是，您必須升級至較新的 Windows 版本，才能使用 .NET Framework 4.6.1 或更新版本。

.NET Framework 4.6.2 隨附於 [Windows 10 年度更新](https://www.microsoft.com/software-download/windows10)。

## <a name="net-framework-46"></a>.NET Framework 4.6

您可以在 Windows Vista 上安裝 [.NET Framework 4.6](http://go.microsoft.com/fwlink/?LinkID=213834&dotnetdocs)。

## <a name="net-framework-35"></a>.NET Framework 3.5

您可以在 Windows Vista 上安裝 [.NET Framework 3.5](http://go.microsoft.com/fwlink/?LinkID=213834&dotnetdocs)。

.NET Framework 3.5 可用來執行針對 .NET Framework 1.0 到 3.5 建置的應用程式。

## <a name="help"></a>說明

如果您無法安裝正確的 .NET Framework 版本，可以[連絡 Microsoft 以取得協助](mailto:dotnet-install-help@microsoft.com?subject=Help installing .NET Framework)。
