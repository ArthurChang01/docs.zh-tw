# <a name="installing-the-net-framework-on-windows"></a>在 Windows 上安裝 .NET Framework

下列文件描述如何安裝每個 Windows 版本的 .NET Framework。

## <a name="version-differences"></a>版本差異

.NET Framework 隨附於 Windows。 每個 Windows 版本包含單一的 .NET Framework 版本，但這可能不是您需要的那一個版本。 若您想要針對指定版本的 Windows 使用不同版本的 .NET Framework，則必須加以安裝。

一般而言 .NET Framework 較高版本將執行針對 .NET Framework 較低版本所建置的應用程式。 例如，.NET Framework 4.6.2 將執行針對 .NET Framework 4.0 所建置的應用程式。 這個規則只有一個例外。 針對 .NET Framework 1.0 到 3.5 所建置的應用程式可以在 .NET Framework 3.5 上執行，但不能在 .NET Framework 4.0 或更新版本上執行。
