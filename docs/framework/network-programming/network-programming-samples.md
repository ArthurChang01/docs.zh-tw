# 網路程式設計範例
本節包含描述和連結 <xref:System.Net>、 <xref:System.Net.Cache>、 <xref:System.Net.Configuration>、 <xref:System.Net.Mail><xref:System.Net.Mime><xref:System.Net.NetworkInformation>、、、、、 <xref:System.Net.PeerToPeer><xref:System.Net.Security><xref:System.Net.Sockets>和相關使用命名空間中的類別下載的網路程式設計範例。  
  
## 在本節中  
 [下載進度指示器技術範例](http://go.microsoft.com/fwlink/?LinkID=179556)  
 示範如何顯示檔案下載的進度。  
  
 [FTP 用戶端技術範例](http://go.microsoft.com/fwlink/?LinkID=179557)  
 示範如何從 FTP 伺服器上載和下載檔案。  
  
 [HttpListener 技術範例](http://go.microsoft.com/fwlink/?LinkID=179558)  
 顯示如何處理 HTTP 要求從應用程式內。  
  
 [HttpListener ASPX 主應用程式範例](http://go.microsoft.com/fwlink/?LinkID=179560)  
 示範如何使用 <xref:System.Net.HttpListener> 類別的功能建立路由呼叫裝載 ASP.NET 應用程式的 HTTP 伺服器。  
  
 [Mailer 技術範例](http://go.microsoft.com/fwlink/?LinkID=179561)  
 示範如何從用戶端應用程式傳送電子郵件訊息。  
  
 [NetStat 工具技術範例](http://go.microsoft.com/fwlink/?LinkID=179562)  
 示範如何使用 NCLNetStat 網路資訊工具。  
  
 [網路資訊技術範例](http://go.microsoft.com/fwlink/?LinkID=179564)  
 顯示如何監視和顯示網路資訊。  
  
 [Ping 用戶端技術範例](http://go.microsoft.com/fwlink/?LinkID=179565)  
 示範可以 Ping 遠端主機的用戶端應用程式。  
  
 [WebClient 技術範例](http://go.microsoft.com/fwlink/?LinkID=179566)  
 示範如何使用執行一般作業，例如檔案或資料上載或下載。  
  
 [安全資料流範例](http://go.microsoft.com/fwlink/?LinkID=179567)  
 顯示如何使用安全資料流通訊在用戶端和伺服器之間。  
  
 [IPv6 Sockets 範例](http://go.microsoft.com/fwlink/?LinkID=179568)  
 在啟用 IPv6 時，示範如何使用通訊端。  
  
 [FTP 總管技術範例](http://go.microsoft.com/fwlink/?LinkID=179569)  
 示範如何列出 FTP 伺服器的內容。  
  
 [Socket 效能技術範例](http://go.microsoft.com/fwlink/?LinkID=179570)  
 在 <xref:System.Net.Sockets.Socket> 說明如何使用類別內的增強功能會使用非同步網路 I\/O 以達到最高效能的伺服器應用程式。  
  
 [PeerToPeer 技術範例](http://go.microsoft.com/fwlink/?LinkID=179571)  
 在 <xref:System.Net.PeerToPeer> 說明如何使用命名空間內的新類別來註冊及發行對等名稱然後再解析對等名稱。
