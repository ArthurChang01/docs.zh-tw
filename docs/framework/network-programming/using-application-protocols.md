# 使用應用程式通訊協定
.NET Framework 支援常用的網際網路應用程式通訊協定。  本節包含有關使用 [HTTP](../../../docs/framework/network-programming/http.md)、 [TCP](../../../docs/framework/network-programming/tcp-udp.md)和 [UDP](../../../docs/framework/network-programming/tcp-udp.md) 通訊協定，以及資訊的相關資訊。 [Windows Sockets](../../../docs/framework/network-programming/sockets.md) 使用介面實作自訂通訊協定。
