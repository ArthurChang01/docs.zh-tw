# 使用應用程式通訊協定
.NET Framework 支援常用的網際網路應用程式通訊協定。 本節包含有關使用 [HTTP](../../../docs/framework/network-programming/http.md)、"TCP"和 "UDP" 通訊協定的資訊，以及有關使用 [Windows Sockets](../../../docs/framework/network-programming/sockets.md) 介面以實作自訂通訊協定的資訊。
