# 通訊端程式碼範例
下列程式碼範例示範如何使用 <xref:System.Net.Sockets.Socket> 類別當做用戶端連接至遠端 Web 服務和做為接聽來自遠端用戶端的連接的伺服器。  
  
## 在本節中  
 [同步用戶端通訊端範例](../../../docs/framework/network-programming/synchronous-client-socket-example.md)  
 示範如何實作連接至伺服器並顯示伺服器所傳回資料的同步 <xref:System.Net.Sockets.Socket> 用戶端。  
  
 [同步伺服器通訊端範例](../../../docs/framework/network-programming/synchronous-server-socket-example.md)  
 示範如何實作可接受來自用戶端的連接，並回應支援從用戶端接收的資料同步處理 <xref:System.Net.Sockets.Socket> 伺服器。  
  
 [非同步用戶端通訊端範例](../../../docs/framework/network-programming/asynchronous-client-socket-example.md)  
 示範如何實作連接至伺服器並顯示伺服器傳回資料的非同步 <xref:System.Net.Sockets.Socket> 用戶端。  
  
 [非同步伺服器通訊端範例](../../../docs/framework/network-programming/asynchronous-server-socket-example.md)  
 示範如何實作可接受來自用戶端的連接，並回應支援從用戶端接收資料的非同步 <xref:System.Net.Sockets.Socket> 伺服器。
