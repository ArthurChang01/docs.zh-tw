# 適用於 Windows 市集應用程式的 .NET 的 MEF 命名空間
<xref:System.Composition?displayProperty=fullName> 及其子命名空間包含用於開發具備 Managed Extensibility Framework \(MEF\) 之可延伸 [!INCLUDE[win8_appname_long](../../../includes/win8-appname-long-md.md)] 應用程式的型別。  這些命名空間是 [!INCLUDE[win8](../../../includes/win8-md.md)] 作業系統之 [!INCLUDE[net_win8_profile](../../../includes/net-win8-profile-md.md)] 子集的一部分。  
  
 這些命名空間不是隨 .NET Framework 一起散發之核心類別庫的一部分。  若要安裝這些命名空間，請在 [!INCLUDE[vs_dev11_long](../../../includes/vs-dev11-long-md.md)] 中開啟您的專案，選擇 \[**專案**\] 功能表中的 \[**管理 NuGet 封裝**\]，並在線上搜尋 Microsoft.Composition 封裝。  
  
-   <xref:System.Composition?displayProperty=fullName> 會提供構成 [!INCLUDE[win8_appname_long](../../../includes/win8-appname-long-md.md)] 應用程式核心 MEF 的類別。  
  
-   <xref:System.Composition.Convention?displayProperty=fullName> 會提供支援搭配慣例組態模型使用 MEF 的型別。  
  
-   <xref:System.Composition.Hosting?displayProperty=fullName> 會提供對主應用程式開發人員非常有用的 MEF 型別。  
  
-   <xref:System.Composition.Hosting.Core?displayProperty=fullName> 會提供組合引擎內部使用的 MEF 型別。  
  
 如需 [!INCLUDE[net_win8_profile](../../../includes/net-win8-profile-md.md)] 以及它所包含的命名空間與型別清單的詳細資訊，請在《Windows 開發人員中心》參閱 [適用於 Windows 市集應用程式的 .NET 概觀](http://go.microsoft.com/fwlink/p/?LinkID=238312) 。
