# <a name="using-regions"></a>使用區域
GDI +<xref:System.Drawing.Region>類別可讓您定義自訂形狀。 圖形可以組成線條、 多邊形和曲線。  
  
 測試和裁剪，則會叫用兩個區域的一般使用方式。 點擊測試，可決定是否在特定區域的畫面按一下滑鼠。 裁剪為繪圖特定區域限制。
