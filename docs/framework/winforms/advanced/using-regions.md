# <a name="using-regions"></a>使用區域
[!INCLUDE[ndptecgdiplus](../../../../includes/ndptecgdiplus-md.md)] <xref:System.Drawing.Region>類別可讓您定義自訂形狀。 圖形可以組成線條、 多邊形和曲線。  
  
 測試和結束時間裁剪時，會叫用兩種常見用途區域。 點擊測試，可決定是否將螢幕的某些區域中按下滑鼠。 裁剪為特定地區繪圖限制。
