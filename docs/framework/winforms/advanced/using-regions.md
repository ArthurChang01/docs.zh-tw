# 使用區域
[!INCLUDE[ndptecgdiplus](../../../../includes/ndptecgdiplus-md.md)] <xref:System.Drawing.Region> 類別可讓您定義自訂形狀。  形狀可由線條、多邊形和曲線組成。  
  
 區域的兩個常見用途是點擊測試 \(Hit Testing\) 和裁剪 \(Clipping\)。  點擊測試可決定是否已在螢幕的某個區域按了滑鼠鍵。  裁剪會將繪圖限制在某個區域內。

 示範如何使用 <xref:System.Drawing.Region> 以執行點擊測試 \(Hit Test\)。  
  
 [如何：使用區域的裁剪](../../../../docs/framework/winforms/advanced/how-to-use-clipping-with-a-region.md)  
 解說如何設定 <xref:System.Drawing.Graphics> 物件的裁剪區域 \(Clipping Region\)。
