# 使用畫筆繪製線條和形狀
使用 [!INCLUDE[ndptecgdiplus](../../../../includes/ndptecgdiplus-md.md)] `Pen` 物件繪製線段、曲線和圖案的外框。  在這個章節中，除非特別指明只代表線段，否則*線條*代表任何線段、曲線和圖案的外框。  設定畫筆用來控制色彩、寬度、對齊的屬性以及使用該畫筆繪製的線條樣式。
