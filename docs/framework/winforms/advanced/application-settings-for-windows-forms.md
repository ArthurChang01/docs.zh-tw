# <a name="application-settings-for-windows-forms"></a>Windows Form 的應用程式設定
Windows Form 的應用程式設定功能可讓您輕鬆地在用戶端上建立、儲存及維護自訂應用程式和使用者偏好設定。 利用應用程式設定，您不只可以儲存應用程式資料 (例如資料庫連接字串)，也可以儲存使用者專屬資料 (例如工具列位置和最近使用的清單)。  
  
## <a name="in-this-section"></a>本節內容  
 [應用程式設定概觀](application-settings-overview.md)  
 討論如何代表您的應用程式和使用者來建立及儲存設定資料。  
  
 [應用程式設定架構](application-settings-architecture.md)  
 描述應用程式設定功能的運作方式，並且瀏覽架構的進階功能 (例如已群組的設定和設定索引鍵)。  
  
 [應用程式設定屬性](application-settings-attributes.md)  
 列出並描述可套用至應用程式設定包裝函式類別或其設定屬性 (Property) 的屬性 (Attribute)。  
  
 [Application Settings for Custom Controls](application-settings-for-custom-controls.md)  
 討論在第三方應用程式中裝載時，要讓自訂控制項能夠保存應用程式設定所須執行的作業。  
  
 [如何：建立應用程式設定](how-to-create-application-settings.md)  
 示範如何建立新的應用程式設定，這些設定會在兩個應用程式工作階段之間保存。  
  
 [如何：驗證應用程式設定](how-to-validate-application-settings.md)  
 示範如何在保存應用程式設定之前先行驗證。
