# 使用筆刷填滿形狀
[!INCLUDE[ndptecgdiplus](../../../../includes/ndptecgdiplus-md.md)] <xref:System.Drawing.Brush> 物件是用來填滿封閉式形狀的內部。  [!INCLUDE[ndptecgdiplus](../../../../includes/ndptecgdiplus-md.md)] 定義幾種填滿樣式：純色、規劃圖樣、影像材質以及色彩漸層。  
  
## 在本節中

 [如何：以規劃圖樣填滿形狀](../../../../docs/framework/winforms/advanced/how-to-fill-a-shape-with-a-hatch-pattern.md)  
 示範如何使用規劃圖樣筆刷填滿形狀。  
  
 [如何：使用影像材質填滿圖案](../../../../docs/framework/winforms/advanced/how-to-fill-a-shape-with-an-image-texture.md)  
 解說如何使用材質筆刷填滿形狀。  
  
 [如何：使用影像並排顯示圖案](../../../../docs/framework/winforms/advanced/how-to-tile-a-shape-with-an-image.md)
