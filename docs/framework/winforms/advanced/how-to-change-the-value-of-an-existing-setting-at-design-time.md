# <a name="how-to-change-the-value-of-an-existing-setting-at-design-time"></a>如何：在設計階段的現有設定的值變更
您可以使用 Visual Studio 來編輯您的專案中的現有設定的值。  
  
### <a name="to-change-the-value-of-an-existing-setting-at-design-time-in-c"></a>若要變更現有的設定，在 C 中的設計階段值\#
  
1.  在 **方案總管**，展開**屬性**在專案的節點。  
  
2.  按兩下您要在其中加入新的設定.settings 檔案。 此檔案的預設名稱是 Settings.settings。  
  
3.  在 設定設計工具中，找到您要變更值，並在 值 欄位中輸入新值的設定。  
  
### <a name="to-change-the-value-of-an-existing-setting-at-design-time-in-visual-basic"></a>若要變更現有的設定，在設計階段，在 Visual Basic 中的值  
  
1.  在 **方案總管**，以滑鼠右鍵按一下您的專案節點，然後選擇**屬性**。  
  
2.  在 [**屬性**頁面上，選取**設定**] 索引標籤。  
  
3.  在 設定設計工具中，找到您要變更值，並在 值 欄位中輸入新值的設定。
