# 如何：在設計階段變更現有設定的值
您可以使用 Visual Studio 編輯專案中現有的設定值。  
  
### 若要在 C\# 中的設計階段變更現有設定的值  
  
1.  在 \[**方案總管**\] 中，展開專案的 \[**屬性**\] 節點。  
  
2.  按兩下您要在其中新增設定的 .settings 檔案。  此檔案的預設名稱是 Settings.settings。  
  
3.  在 \[設定設計工具\] 中，尋找您要變更其值的設定，並在 \[值\] 一欄中輸入新值。  
  
### 若要在 Visual Basic 中的設計階段變更現有設定的值  
  
1.  在 \[**方案總管**\] 中，以滑鼠右鍵按一下專案節點，然後選擇 \[**屬性**\]。  
  
2.  在 \[**屬性**\] 頁面中，選取 \[**設定**\] 索引標籤。  
  
3.  在 \[設定設計工具\] 中，尋找您要變更其值的設定，並在 \[值\] 一欄中輸入新值。
