# <a name="how-to-create-a-new-setting-at-design-time"></a>如何：在設計階段建立新設定
您可以使用 設定設計工具，在設計階段建立新的設定。 設定設計工具是格線樣式介面，可讓您建立新的設定，並指定這些設定的屬性。 您必須指定名稱、 值、 類型和您的新設定的範圍。 一旦建立的設定時，就是在程式碼中存取。  
  
### <a name="to-create-a-new-setting-at-design-time-in-c"></a>若要在 C# 中的設計階段建立新的設定  
  
1.  在**方案總管 中**，依序展開**屬性**您的專案節點。  
  
2.  按兩下您要加入新的設定.settings 檔案。 這個檔案的預設名稱是 Settings.settings。  
  
3.  在 設定設計工具中，設定名稱、 值、 類型和您的設定範圍。 每個資料列都代表單一的設定。  
  
### <a name="to-create-a-new-setting-at-design-time-in-visual-basic"></a>若要在 Visual Basic 中的設計階段建立新的設定  
  
1.  在**方案總管] 中**，以滑鼠右鍵按一下您的專案節點，然後選擇 [**屬性**。  
  
2.  在**屬性**頁面上，選取**設定** 索引標籤。  
  
3.  在 設定設計工具中，設定名稱、 值、 類型和您的設定範圍。 每個資料列都代表單一的設定。
