# <a name="about-gdi-managed-code"></a>關於 GDI+ Managed 程式碼

GDI + 是 Windows 作業系統中提供二維向量圖形、影像和印刷樣式的部分。 GDI + 藉由加入新功能和優化現有的功能，改善了 GDI （舊版 Windows 隨附的圖形裝置介面）。

GDI + managed 類別介面（一組包裝函式）是 .NET Framework 的一部分，這是建立、部署和執行 XML Web Service 和其他應用程式的環境。

本節提供使用受控碼之程式設計人員的 GDI + API 的相關資訊。
