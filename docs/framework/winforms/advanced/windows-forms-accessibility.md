# Windows Forms 協助工具
Windows Forms 的協助工具功能可讓您將應用程式提供給各式各樣的使用者。  
  
## 在本節中  
 [逐步解說：建立 Windows 架構的可及性應用程式](../../../../docs/framework/winforms/advanced/walkthrough-creating-an-accessible-windows-based-application.md)  
 描述為了提高可及性應支援的所有功能。  
  
## 參考  
 <xref:Accessibility>  
 此命名空間包含一些與協助工具相關的類別。  
  
 <xref:System.Windows.Forms.AccessibleObject>  
 提供資訊，讓協助工具應用程式能用於為殘障使用者調整應用程式的使用者介面 \(UI\)。
