# 將影像重新著色
重新著色是指調整影像色彩的程序。  將某個色彩變成另一個色彩、調整某個色彩相對於另一個色彩的濃度、調整所有色彩的亮度和對比，以及將色彩轉換為灰階等，這些都是重新著色的例子。
