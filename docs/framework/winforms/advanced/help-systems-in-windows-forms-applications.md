# Windows Form 應用程式中的說明系統
身為應用程式開發人員的您，最重要的貢獻之一就是提供一套能夠滿足所有使用者需求的說明系統，  因為當使用者感到疑惑、無所適從時，他們會求助於說明系統。  您只要使用 [HelpProvider 元件](../../../../docs/framework/winforms/controls/helpprovider-component-windows-forms.md)，便能夠很輕鬆地在 Windows 架構應用程式中提供說明系統。  
  
## 不同類型的說明  
 Windows Form <xref:System.Windows.Forms.HelpProvider> 元件用於將 HTML HELP 1.x 說明檔 \(可能是以 HTML Help Workshop 產生的 .chm 檔，或是 .htm 檔\) 與您的 Windows 架構應用程式建立關聯。  <xref:System.Windows.Forms.HelpProvider> 元件可用來在 Windows Form 或特定控制項上提供控制項的即時線上說明。  此外，<xref:System.Windows.Forms.HelpProvider> 元件可在特定區域中開啟說明檔，例如目錄的主頁、索引或搜尋功能。  如需 <xref:System.Windows.Forms.HelpProvider> 元件的一般資訊，請參閱 [HelpProvider 元件概觀](../../../../docs/framework/winforms/controls/helpprovider-component-overview-windows-forms.md)。  如需如何使用 <xref:System.Windows.Forms.HelpProvider> 元件在 Windows Form 上顯示快顯說明的詳細資訊，請參閱 [如何：顯示快顯說明](../../../../docs/framework/winforms/advanced/how-to-display-pop-up-help.md)。  如需使用 <xref:System.Windows.Forms.ToolTip> 元件以顯示特定控制項說明的詳細資訊，請參閱 [使用工具提示來顯示控制項說明](../../../../docs/framework/winforms/advanced/control-help-using-tooltips.md)。  
  
 您可使用 HTML Help Workshop 產生 HTML Help 1.x 檔案。  如需有關 HTML 說明的詳細資訊，請參閱＜HTML Help Workshop＞或 MSDN 中其他的＜HTML 說明＞主題。
