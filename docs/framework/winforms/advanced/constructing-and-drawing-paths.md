# 建構和繪製路徑
路徑是指可當成單一單位來進行操作和繪製的一系列基本圖形 \(直線、矩形、曲線、文字和類似圖形\)。  路徑可為分開放和封閉的*圖形*。  圖形可包含幾個基本類型。  
  
 您可以呼叫 <xref:System.Drawing.Graphics> 類別的 <xref:System.Drawing.Graphics.DrawPath%2A> 方法來繪製路徑，也可以呼叫 <xref:System.Drawing.Graphics> 類別的 <xref:System.Drawing.Graphics.FillPath%2A> 方法來填滿路徑。
