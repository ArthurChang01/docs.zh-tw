# 使用應用程式設定和使用者設定
從 .NET Framework 2.0 開始，您可以建立和存取在應用程式執行工作階段之間保留的值。  這些值稱為「*設定*」\(Setting\)。  設定代表使用者偏好設定，或應用程式必須使用的寶貴資訊。  例如，您可以建立一系列的設定，儲存應用程式之色彩配置的使用者偏好設定。  或者，也可以儲存指定應用程式使用之資料庫的連接字串。  設定可讓您保留對於程式碼外部之應用程式而言相當重要的資訊，以及建立可儲存個別使用者偏好設定的設定檔。
