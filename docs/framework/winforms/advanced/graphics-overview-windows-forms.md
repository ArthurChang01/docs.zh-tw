# 圖形概觀 (Windows Form)
[!INCLUDE[ndptecgdiplus](../../../../includes/ndptecgdiplus-md.md)]是一個圖形裝置介面，可讓程式設計人員撰寫與裝置無關的應用程式。  [!INCLUDE[ndptecgdiplus](../../../../includes/ndptecgdiplus-md.md)] 的服務是由一組 Managed 類別所提供。  
  
## 在本節中  
 [圖形概觀](../../../../docs/framework/winforms/advanced/overview-of-graphics.md)  
 提供 [!INCLUDE[ndptecgdiplus](../../../../includes/ndptecgdiplus-md.md)] 的一般簡介。  
  
 [圖形服務的三個分類](../../../../docs/framework/winforms/advanced/three-categories-of-graphics-services.md)  
 描述使用 [!INCLUDE[ndptecgdiplus](../../../../includes/ndptecgdiplus-md.md)] 進行程式設計的三個分類。  
  
 [圖形介面的結構](../../../../docs/framework/winforms/advanced/structure-of-the-graphics-interface.md)  
 描述 [!INCLUDE[ndptecgdiplus](../../../../includes/ndptecgdiplus-md.md)] 的 Managed 類別介面。  
  
## 參考  
 <xref:System.Drawing>  
 提供對 [!INCLUDE[ndptecgdiplus](../../../../includes/ndptecgdiplus-md.md)] 基本圖形功能的存取。  
  
 <xref:System.Drawing.Drawing2D>  
 提供進階的 2D 和向量圖形功能。  
  
 <xref:System.Drawing.Imaging>  
 提供進階的 [!INCLUDE[ndptecgdiplus](../../../../includes/ndptecgdiplus-md.md)] 影像處理功能。  
  
 <xref:System.Drawing.Text>  
 提供進階的 [!INCLUDE[ndptecgdiplus](../../../../includes/ndptecgdiplus-md.md)] 印刷功能。  
  
 <xref:System.Drawing.Printing>  
 提供和列印相關的服務。  
  
 <xref:System.Windows.Forms.TextRenderer>  
 提供 [!INCLUDE[ndptecgdi](../../../../includes/ndptecgdi-md.md)] 文字繪製和測量功能。
