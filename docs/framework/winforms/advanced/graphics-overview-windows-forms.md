# <a name="graphics-overview-windows-forms"></a>圖形概觀 (Windows Form)
GDI + 是圖形裝置介面，可讓程式設計人員撰寫與裝置無關的應用程式。 GDI + 的服務會透過一組受管理的類別公開。  
  
## <a name="in-this-section"></a>本節內容  
 [圖形概觀](overview-of-graphics.md)  
 提供的一般簡介 GDI +。  
  
 [圖形服務的三個分類](three-categories-of-graphics-services.md)  
 描述使用 GDI + 進行程式設計所組成的三個類別。  
  
 [圖形介面的結構](structure-of-the-graphics-interface.md)  
 描述 managed 的類別介面的 GDI +。

 <xref:System.Drawing>  
 提供對 GDI + 基本繪圖功能的存取。  
  
 <xref:System.Drawing.Drawing2D>  
 提供進階二維和向量圖形功能。  
  
 <xref:System.Drawing.Imaging>  
 提供進階的 GDI + 影像處理功能。  
  
 <xref:System.Drawing.Text>  
 提供進階的 GDI + 印刷樣式功能。  
  
 <xref:System.Drawing.Printing>  
 提供列印相關服務。  
  
 <xref:System.Windows.Forms.TextRenderer>  
 提供 GDI 文字繪圖中並正在測量功能。
