# 整合 Windows Form 中的使用者說明
在建置 Windows 應用程式時，說明系統是一項基本但常常會被忽略的部分，而這是使用者有疑問時尋求協助的地方。  Windows Form 支援兩種不同類型的說明，都是透過 [HelpProvider 元件](../../../../docs/framework/winforms/controls/helpprovider-component-windows-forms.md) 提供。  第一個說明會將使用者指向 HTML 或者是 HTML Help 1.*x* 或更新版本的說明檔。  第二個說明會在個別控制項上顯示簡短的「這是什麼？」類型的說明；這在對話方塊中特別有用。  相同表單上可以同時使用兩種說明類型。  
  
 此外，[ToolTip 元件](../../../../docs/framework/winforms/controls/tooltip-component-windows-forms.md) 可以用來為 Windows Form 上控制項提供個別說明。
