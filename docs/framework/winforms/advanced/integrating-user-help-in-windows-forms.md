# <a name="integrating-user-help-in-windows-forms"></a>整合 Windows Form 中的使用者說明
建立以 Windows 為基礎之應用程式的重要但經常被忽略的層面是說明系統，因為這是使用者在混淆時提供協助的地方。 Windows Forms 支援兩種不同類型的說明，分別由[HelpProvider 元件](../controls/helpprovider-component-windows-forms.md)提供。 第一種方式是將使用者指向 HTML 或 HTML Help 1 的說明檔。*x*或更大的格式。 第二個可以在個別控制項上顯示簡短的「這是什麼」類型的說明;這在對話方塊中特別有用。 這兩種類型的說明都可以在相同的表單上使用。  
  
 此外，[工具提示元件](../controls/tooltip-component-windows-forms.md)可以用來為 Windows Forms 上的控制項提供個別的協助。
