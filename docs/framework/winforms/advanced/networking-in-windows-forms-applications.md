# Windows Form 應用程式中的網路
.NET Framework 提供的類別可以用來顯示 Web 網頁、下載 Web 內容、與檔案傳輸通訊協定 \(File Transfer Protocol，FTP\) 站台互動以及使用 Web 服務，這使得在應用程式中建置網路功能變得相當容易。  下列資源將協助您了解 .NET Framework 的網路技術，以及如何將它們整合到 Windows Form 中。  
  
## 參考  
 <xref:System.Net>  
 .NET Framework 中處理網路連接的類別的根命名空間。  
  
 <xref:System.Net.WebClient>  
 以程式設計方式擷取 Web 或 HTTP 內容的實用類別。  
  
 <xref:System.Net.FtpWebRequest>  
 以 FTP 擷取和傳送檔案的類別。  
  
 <xref:System.Windows.Forms.WebBrowser>  
 包含在 Windows 中的 `WebBrowser` 控制項的 Managed 包裝函式類別。
