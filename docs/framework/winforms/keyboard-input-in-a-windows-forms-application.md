# Windows Form 應用程式中的鍵盤輸入
Windows Form 包含讓您回應按下的特定按鍵的標準鍵盤事件，同時也提供您在應用程式、表單和控制項層次上攔截、修改和使用按下的按鍵。  
  
## 在本節中  
 [鍵盤輸入的運作方式](../../../docs/framework/winforms/how-keyboard-input-works.md)  
 描述如何處理鍵盤訊息，並將它轉換成鍵盤事件。  
  
 [使用鍵盤事件](../../../docs/framework/winforms/using-keyboard-events.md)  
 提供關於鍵盤事件型別的資訊，以及鍵盤事件處理常式所收到的資訊。  
  
 [如何：將鍵盤輸入修改為標準控制項](../../../docs/framework/winforms/how-to-modify-keyboard-input-to-a-standard-control.md)  
 顯示程式碼範例，示範如何在按鍵值到達控制項之前加以修改。  
  
 [如何：判斷所按的輔助按鍵為何](../../../docs/framework/winforms/how-to-determine-which-modifier-key-was-pressed.md)  
 示範如何得知在按下 SHIFT、ALT 或 CTRL 的同時是否按下另一個按鍵。  
  
 [如何：處理表單層級的鍵盤輸入](../../../docs/framework/winforms/how-to-handle-keyboard-input-at-the-form-level.md)  
 顯示程式碼範例，示範如何在按鍵到達控制項之前加以攔截。
