# <a name="mainmenu-component-windows-forms"></a>MainMenu 元件 (Windows Form)
> [!NOTE]
> 雖然 `MenuStrip` 和 `ContextMenuStrip` 將功能取代並加入舊版的 `MainMenu` 和 <xref:System.Windows.Forms.ContextMenu> 控制項，但如果您選擇，`MainMenu` 和 <xref:System.Windows.Forms.ContextMenu> 會保留供回溯相容性及未來使用。  
  
 Windows Forms `MainMenu` 元件會在執行時間顯示功能表。
