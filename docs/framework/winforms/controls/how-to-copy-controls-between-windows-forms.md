# 如何：在 Windows Form 之間複製控制項
控制項可以複製到同一個表單內、複製到專案內的另一個表單中或複製到剪貼簿以供其他方案使用。  
  
### 若要複製控製項  
  
1.  選取控制項，並且從 \[**編輯**\] 功能表中選擇 \[**複製**\]。  
  
 此時這個控制項會貼到任何可以接受這個控制項型別的表單。  此外，該控制項已經加入至剪貼簿中。
