# <a name="colordialog-component-windows-forms"></a>ColorDialog 元件 (Windows Form)
Windows 表單<xref:System.Windows.Forms.ColorDialog>元件是一個預先配置的對話方塊，允許使用者從調色板中選擇顏色，並將自訂顏色添加到該調色板。 這是您在其他 Windows 應用程式中看到可選取色彩的相同對話方塊。 只要在 Windows 應用程式中使用這個控制項，便不需要設定自己的對話方塊。
