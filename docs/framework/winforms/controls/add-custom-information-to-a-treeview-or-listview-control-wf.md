<span data-ttu-id="4d9b8-103">您可以在 Windows Forms <xref:System.Windows.Forms.TreeView> 控制項或 <xref:System.Windows.Forms.ListView> 控制項中的衍生專案中建立衍生節點。</span><span class="sxs-lookup"><span data-stu-id="4d9b8-103">You can create a derived node in a Windows Forms <xref:System.Windows.Forms.TreeView> control or a derived item in a <xref:System.Windows.Forms.ListView> control.</span></span> <span data-ttu-id="4d9b8-104">衍生可讓您新增您所需的任何欄位，以及新增自訂方法和建構函式來處理它們。</span><span class="sxs-lookup"><span data-stu-id="4d9b8-104">Derivation allows you to add any fields you require, as well as custom methods and constructors for handling them.</span></span> <span data-ttu-id="4d9b8-105">這項功能的其中一個用途是將 Customer 物件附加至每個樹狀節點或清單項目。</span><span class="sxs-lookup"><span data-stu-id="4d9b8-105">One use of this feature is to attach a Customer object to each tree node or list item.</span></span> <span data-ttu-id="4d9b8-106">這裡的範例適用于 <xref:System.Windows.Forms.TreeView> 控制項，但相同的方法也可用於 <xref:System.Windows.Forms.ListView> 控制項。</span><span class="sxs-lookup"><span data-stu-id="4d9b8-106">The examples here are for a <xref:System.Windows.Forms.TreeView> control, but the same approach can be used for a <xref:System.Windows.Forms.ListView> control.</span></span>  
  
### <a name="to-derive-a-tree-node"></a><span data-ttu-id="4d9b8-107">衍生樹狀節點</span><span class="sxs-lookup"><span data-stu-id="4d9b8-107">To derive a tree node</span></span>  
  
- <span data-ttu-id="4d9b8-108">建立新的節點類別，衍生自 <xref:System.Windows.Forms.TreeNode> 類別，其中包含用來記錄檔案路徑的自訂欄位。</span><span class="sxs-lookup"><span data-stu-id="4d9b8-108">Create a new node class, derived from the <xref:System.Windows.Forms.TreeNode> class, which has a custom field to record a file path.</span></span>  
  
    ```vb  
    Class myTreeNode  
       Inherits TreeNode  
  
       Public FilePath As String  
  
       Sub New(ByVal fp As String)  
          MyBase.New()  
          FilePath = fp  
          Me.Text = fp.Substring(fp.LastIndexOf("\"))  
       End Sub  
    End Class  
    ```  
  
    ```csharp  
    class myTreeNode : TreeNode  
    {  
       public string FilePath;  
  
       public myTreeNode(string fp)  
       {  
          FilePath = fp;  
          this.Text = fp.Substring(fp.LastIndexOf("\\"));  
       }  
    }  
    ```  
  
    ```cpp  
    ref class myTreeNode : public TreeNode  
    {  
    public:  
       System::String ^ FilePath;  
  
       myTreeNode(System::String ^ fp)  
       {  
          FilePath = fp;  
          this->Text = fp->Substring(fp->LastIndexOf("\\"));  
       }  
    };  
    ```
