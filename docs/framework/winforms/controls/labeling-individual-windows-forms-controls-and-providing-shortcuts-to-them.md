# 標記個別 Windows Form 控制項並提供其捷徑
加入 Windows Form 的控制項具有可用來進一步特製化使用者經驗的屬性和方法。  對於設計良好的 Windows 應用程式而言，依使用者的需要自訂使用者介面是非常重要的。
