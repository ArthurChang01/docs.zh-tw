# <a name="groupbox-control-overview-windows-forms"></a>GroupBox 控制項概觀 (Windows Form)
Windows Forms <xref:System.Windows.Forms.GroupBox> 控制項是用來為其他控制項提供可識別的群組。 一般來說，您會使用群組方塊來依函式細分表單。 例如，您的訂單表單可能會指定郵寄選項，例如要使用的貨運公司。 將群組方塊中的所有選項分組可讓使用者具有邏輯視覺提示，而且在設計階段，所有控制項都可以輕鬆地移動，而當您移動單一 <xref:System.Windows.Forms.GroupBox> 控制項時，也會移動其包含的所有控制項。  
  
 群組方塊的標題是由 <xref:System.Windows.Forms.GroupBox.Text%2A> 屬性所定義。 如需詳細資訊，請參閱[如何：設定 Windows Forms 控制項所顯示的文字](how-to-set-the-text-displayed-by-a-windows-forms-control.md)。  
  
## <a name="groupbox-and-panel"></a>分組和麵板  
 <xref:System.Windows.Forms.GroupBox> 控制項類似于 <xref:System.Windows.Forms.Panel> 控制項;不過，只有 [<xref:System.Windows.Forms.GroupBox>] 控制項會顯示標題，而且只有 [<xref:System.Windows.Forms.Panel>] 控制項才可以有捲軸。
