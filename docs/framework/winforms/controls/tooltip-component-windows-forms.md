# <a name="tooltip-component-windows-forms"></a>ToolTip 元件 (Windows Form)
當使用者指向控制項時，Windows Form <xref:System.Windows.Forms.ToolTip> 元件會顯示文字。 工具提示可以與任何控制項產生關聯。 使用此控制項的範例：為了節省表單空間，您可以在按鈕上顯示小圖示，並使用工具提示來說明按鈕的功能。
