# LinkLabel 控制項 (Windows Form)
Windows Form `LinkLabel` 控制項可讓您將 Web 樣式的連結加入 Windows Form 應用程式。  在原本 <xref:System.Windows.Forms.Label> 控制項所能作用的任何內容上，您都可以使用 `LinkLabel` 控制項來代替；您也可以將部分文字設定為物件或 Web 網頁的連結。
