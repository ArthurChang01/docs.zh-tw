# TableLayoutPanel 控制項 (Windows Form)
<xref:System.Windows.Forms.TableLayoutPanel> 控制項會在格線中排列其內容。  由於配置會在設計階段和執行階段執行，因此當應用程式環境變更時，配置也會隨著動態變更。  這可讓面板中的控制項按比例調整大小，以便回應變更 \(例如，由於當地語系化所造成的父控制項調整大小或文字長度變更\)。
