# <a name="button-control-windows-forms"></a>Button 控制項 (Windows Form)
Windows Form `Button` 控制項可讓使用者按一下以執行動作。 `Button` 控制項可以顯示文字和影像。 按一下按鈕時，按鈕看起來就像被推入又釋放。

 [如何：回應 Windows Form Button 按一下動作](how-to-respond-to-windows-forms-button-clicks.md)  
 說明 Windows Form 上按鈕的最基本用法。  
  
 [如何：將 Windows Form 按鈕指定為接受按鈕](how-to-designate-a-windows-forms-button-as-the-accept-button.md)  
 說明如何指定 `Button` 控制項為接受按鈕，也稱為預設按鈕。  
  
 [如何：將 Windows Form 按鈕指定為取消按鈕](how-to-designate-a-windows-forms-button-as-the-cancel-button.md)  
 說明如何指定 `Button` 控制項為取消按鈕，每當使用者按下 ESC 鍵時就會被按下。
