     <span data-ttu-id="54147-124">下列程式碼示範如何建立 <xref:System.Windows.Forms.Timer> 及其 `Interval` 屬性集。</span><span class="sxs-lookup"><span data-stu-id="54147-124">The following code shows the creation of a <xref:System.Windows.Forms.Timer> with its `Interval` property set.</span></span>

    ```vb

    End Sub
    ```

    ```csharp
    public void createTimer()
    {
       System.Windows.Forms.Timer timerKeepTrack = new
           System.Windows.Forms.Timer();
       timerKeepTrack.Interval = 1000;
    ```

    ```cpp
       void createTimer()

          System::Windows::Forms::Timer^ timerKeepTrack = gcnew
    ```
