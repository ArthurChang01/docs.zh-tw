# ErrorProvider 元件 (Windows Form)
Windows Form 的 `ErrorProvider` 元件能夠以不干擾使用者的方式顯示錯誤狀況。  它通常會和使用者的表單輸入驗證一起使用，或在資料集中顯示錯誤。
