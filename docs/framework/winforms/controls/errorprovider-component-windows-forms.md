# <a name="errorprovider-component-windows-forms"></a>ErrorProvider 元件 (Windows Form)
Windows Form`ErrorProvider`元件可用來讓使用者不會造成干擾的方式有不對勁的錯誤。 通常用於搭配驗證表單上的使用者輸入或顯示資料集內的錯誤。
