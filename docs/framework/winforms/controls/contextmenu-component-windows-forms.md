# ContextMenu 元件 (Windows Form)
> [!NOTE]
>  雖然 `MenuStrip` 和 `ContextMenuStrip` 會取代和加入功能至舊版的 <xref:System.Windows.Forms.MainMenu> 和 <xref:System.Windows.Forms.ContextMenu> 控制項，但是會保留 <xref:System.Windows.Forms.MainMenu> 和 <xref:System.Windows.Forms.ContextMenu> 以提供回溯相容性 \(Backward Compatibility\) 和未來使用 \(如果您選擇要用\)。  
  
 Windows Form `ContextMenu` 元件是用來為使用者提供易於存取且與所選取物件關聯的常用命令捷徑功能表。  捷徑功能表中的項目通常是出現在應用程式其他地方的主功能表項目子集。  捷徑功能表通常都可以按一下滑鼠右鍵取得。  在 Windows Form 中，它們與其他控制項有關聯。
