# <a name="pagesetupdialog-component-windows-forms"></a>PageSetupDialog 元件 (Windows Form)
Windows 表單<xref:System.Windows.Forms.PageSetupDialog>元件是一個預先配置的對話方塊，用於設置頁面詳細資訊，以便在基於 Windows 的應用程式中列印。 在基於 Windows 的應用程式中使用它作為使用者設置頁面首選項以代替配置自己的對話方塊的簡單解決方案。 您可以允許使用者設置邊框和邊距調整、標題和頁腳以及縱向與橫向。 藉由標準 Windows 對話方塊，建立使用者可立即熟悉基本功能的應用程式。
