# <a name="monthcalendar-control-windows-forms"></a>MonthCalendar 控制項 (Windows Form)
Windows Form`MonthCalendar`控制項提供直覺式圖形介面，以便讓使用者檢視並設定日期資訊。 控制項會顯示方格，包含編號的日期的月份，在下一周天數的資料行中排列。 您可以在月份標題的任一邊的箭號按鈕，即可選取不同的月份。 不同於類似<xref:System.Windows.Forms.DateTimePicker>控制項，您可以選取範圍的日期與這個控制項; 不過，<xref:System.Windows.Forms.DateTimePicker>控制項可讓您設定的時間以及日期。
