# MonthCalendar 控制項 (Windows Form)
Windows Form `MonthCalendar` 控制項為使用者提供直覺式圖形介面，以檢視和設定日期資訊。  控制項會顯示包含本月編號日期的方格，以每週日期的方式成行排列。  您可以按一下月份標題兩側的箭頭按鈕來選取不同的月份。  與類似的 <xref:System.Windows.Forms.DateTimePicker> 控制項不一樣的是，您可以使用這個控制項來選取一個日期範圍；但是，<xref:System.Windows.Forms.DateTimePicker> 控制項允許您設定時間和日期。
