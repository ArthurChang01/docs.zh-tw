# <a name="monthcalendar-control-windows-forms"></a>MonthCalendar 控制項 (Windows Form)
Windows Forms `MonthCalendar` 控制項呈現直覺的圖形化介面，讓使用者能夠查看和設定日期資訊。 控制項會顯示一個方格，其中包含月份的編號日期，並以資料行排列在一周的哪幾天之下。 您可以按一下月份標題任一側的箭號按鈕，以選取不同的月份。 不同于類似的 <xref:System.Windows.Forms.DateTimePicker> 控制項，您可以使用此控制項選取某個日期範圍。不過，<xref:System.Windows.Forms.DateTimePicker> 控制項可讓您設定時間和日期。
