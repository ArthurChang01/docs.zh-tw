# <a name="layout-in-windows-forms-controls"></a>Windows Form 控制項中的配置
對許多應用程式而言，控制項在表單上的精確位置是高優先順序。 <xref:System.Windows.Forms?displayProperty=nameWithType>命名空間會提供您許多版面配置工具，來完成這項作業。
