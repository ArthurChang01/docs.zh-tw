# ToolStripContainer 控制項
<xref:System.Windows.Forms.ToolStrip> 控制項使用 <xref:System.Windows.Forms.ToolStripContainer> 來進行內建浮動定位 \(共用停駐時工具區域內的水平或垂直空間\)。  
  
 本節中的主題描述您可以用來將 <xref:System.Windows.Forms.ToolStripContainer> 功能建置在應用程式中的概念和技術。

 提供描述 Windows Form <xref:System.Windows.Forms.ToolStripContainer> 控制項的用途和主要概念的主題。  
  
 [如何：將 ToolStripContainer 加入表單](../../../../docs/framework/winforms/controls/how-to-add-a-toolstripcontainer-to-a-form.md)  
 示範如何將 <xref:System.Windows.Forms.ToolStripContainer> 加入應用程式，以及將控制項加入 <xref:System.Windows.Forms.ToolStripContainer> 的特定面板。  
  
 [如何：將控制項加入 ToolStripContentPanel](../../../../docs/framework/winforms/controls/how-to-add-a-control-to-a-toolstripcontentpanel.md)
