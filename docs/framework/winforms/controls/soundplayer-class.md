# SoundPlayer 類別
`SoundPlayer` 類別可讓您輕鬆地在應用程式中包含音效。  
  
 您也可以使用 <xref:System.Media.SystemSounds> 類別來播放常見的系統音效，包括嗶聲。  
  
## 在本節中  
 [SoundPlayer 類別概觀](../../../../docs/framework/winforms/controls/soundplayer-class-overview.md)  
 介紹類別以及其常用的屬性、方法和事件。  
  
 [如何：播放 Windows Form 中的音效](../../../../docs/framework/winforms/controls/how-to-play-a-sound-from-a-windows-form.md)  
 提供程式碼來播放經由檔案路徑、 UNC 路徑或 HTTP 路徑指定的音效。  
  
 [如何：播放 Windows Form 中的嗶聲](../../../../docs/framework/winforms/controls/how-to-play-a-beep-from-a-windows-form.md)  
 提供播放電腦嗶聲音效的程式碼。  
  
 [如何：播放內嵌在 Windows Form 資源中的音效](../../../../docs/framework/winforms/controls/how-to-play-a-sound-embedded-in-a-resource-from-a-windows-form.md)  
 提供從資源播放音效的程式碼。  
  
 [如何：播放 Windows Form 中的系統音效](../../../../docs/framework/winforms/controls/how-to-play-a-system-sound-from-a-windows-form.md)  
 提供程式碼來播放系統音效中的一個音效。  
  
 [如何：在 Windows Form 中非同步地載入音效](../../../../docs/framework/winforms/controls/how-to-load-a-sound-asynchronously-within-a-windows-form.md)  
 提供從 URL 非同步載入音效並播放的程式碼。  
  
 [如何：在 Windows Form 上循環播放音效](../../../../docs/framework/winforms/controls/how-to-loop-a-sound-playing-on-a-windows-form.md)
