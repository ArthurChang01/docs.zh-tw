# <a name="soundplayer-class"></a>SoundPlayer 類別
`SoundPlayer` 類別可讓您輕鬆地在應用程式中包含音效。  
  
 您也可以使用 <xref:System.Media.SystemSounds> 類別來播放常見的系統音效，包括嗶聲。

 [操作說明：播放內嵌在 Windows Forms 資源中的音效](../../../../docs/framework/winforms/controls/how-to-play-a-sound-embedded-in-a-resource-from-a-windows-form.md)
