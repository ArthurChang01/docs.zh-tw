## <a name="bound-mode-and-virtual-mode"></a><span data-ttu-id="84f35-106">系結模式和虛擬模式</span><span class="sxs-lookup"><span data-stu-id="84f35-106">Bound Mode and Virtual Mode</span></span>

## <a name="supplementing-bound-mode"></a><span data-ttu-id="84f35-110">補充系結模式</span><span class="sxs-lookup"><span data-stu-id="84f35-110">Supplementing Bound Mode</span></span>

 <span data-ttu-id="84f35-113">由於未系結的資料行是在資料來源之外，因此資料來源的排序作業會忽略它們。</span><span class="sxs-lookup"><span data-stu-id="84f35-113">Because unbound columns are outside the data source, they are ignored by the data source's sorting operations.</span></span> <span data-ttu-id="84f35-114">因此，當您在混合模式中啟用排序時，您必須在本機快取中管理未系結的資料，並執行虛擬模式，讓 <xref:System.Windows.Forms.DataGridView> 控制項與它互動。</span><span class="sxs-lookup"><span data-stu-id="84f35-114">Therefore, when you enable sorting in mixed mode, you must manage the unbound data in a local cache and implement virtual mode to let the <xref:System.Windows.Forms.DataGridView> control interact with it.</span></span>  
  
 <span data-ttu-id="84f35-115">如需使用虛擬模式來維護未系結資料行中之值的詳細資訊，請參閱 <xref:System.Windows.Forms.DataGridViewCheckBoxColumn.ThreeState%2A?displayProperty=nameWithType> 屬性和 <xref:System.Windows.Forms.DataGridViewComboBoxColumn?displayProperty=nameWithType> 類別參考主題中的範例。</span><span class="sxs-lookup"><span data-stu-id="84f35-115">For more information about using virtual mode to maintain the values in unbound columns, see the examples in the <xref:System.Windows.Forms.DataGridViewCheckBoxColumn.ThreeState%2A?displayProperty=nameWithType> property and <xref:System.Windows.Forms.DataGridViewComboBoxColumn?displayProperty=nameWithType> class reference topics.</span></span>  
  
## <a name="replacing-bound-mode"></a><span data-ttu-id="84f35-116">取代系結模式</span><span class="sxs-lookup"><span data-stu-id="84f35-116">Replacing Bound Mode</span></span>

 <span data-ttu-id="84f35-120">如需在即時案例中使用虛擬模式的詳細資訊，請參閱[在 Windows Forms DataGridView 控制項中，使用即時資料載入來執行虛擬模式](implementing-virtual-mode-jit-data-loading-in-the-datagrid.md)。</span><span class="sxs-lookup"><span data-stu-id="84f35-120">For more information about using virtual mode in a just-in-time scenario, see [Implementing Virtual Mode with Just-In-Time Data Loading in the Windows Forms DataGridView Control](implementing-virtual-mode-jit-data-loading-in-the-datagrid.md).</span></span>
