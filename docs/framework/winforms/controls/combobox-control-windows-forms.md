# <a name="combobox-control-windows-forms"></a>ComboBox 控制項 (Windows Form)
Windows Form`ComboBox`控制項用來在下拉式清單方塊中顯示資料。 根據預設，`ComboBox`控制項出現在兩個部分： 上半部是文字方塊，可讓使用者輸入的清單項目。 第二個部分是清單方塊會顯示，使用者可以從中選取一個項目清單。
