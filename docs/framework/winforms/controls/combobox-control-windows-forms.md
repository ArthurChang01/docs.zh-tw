# <a name="combobox-control-windows-forms"></a>ComboBox 控制項 (Windows Form)
Windows Forms `ComboBox` 控制項是用來在下拉式方塊中顯示資料。 根據預設，`ComboBox` 控制項會出現在兩個部分中：最上方的部分是允許使用者輸入清單專案的文字方塊。 第二個部分是清單方塊，其中顯示使用者可以從中選取一個專案的清單。
