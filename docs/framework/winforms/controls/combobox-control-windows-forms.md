# ComboBox 控制項 (Windows Form)
Windows Form `ComboBox` 控制項是用來在下拉式清單方塊中顯示資料。  根據預設，`ComboBox` 控制項會顯示於兩個部分：上層部分是可讓使用者輸入清單項目的文字方塊。  第二個部分是顯示項目清單的清單方塊，使用者可從中選取一個項目。
