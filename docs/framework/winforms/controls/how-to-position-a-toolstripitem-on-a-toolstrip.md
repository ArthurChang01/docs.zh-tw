# <a name="how-to-position-a-toolstripitem-on-a-toolstrip"></a>如何：在 ToolStrip 上放置 ToolStripItem
您可以移動，或新增<xref:System.Windows.Forms.ToolStripItem>到左邊或右邊<xref:System.Windows.Forms.ToolStrip>。  
  
### <a name="to-move-or-add-a-toolstripitem-to-the-left-side-of-a-toolstrip"></a>若要移動或 ToolStripItem 加入 ToolStrip 的左半部。  
  
1.  設定<xref:System.Windows.Forms.ToolStripItemAlignment>屬性<xref:System.Windows.Forms.ToolStripItem>至<xref:System.Windows.Forms.ToolStripItemAlignment.Left>。  
  
### <a name="to-move-or-add-a-toolstripitem-to-the-right-side-of-a-toolstrip"></a>若要移動或 ToolStripItem 加入 ToolStrip 右邊  
  
1.  設定<xref:System.Windows.Forms.ToolStripItemAlignment>屬性<xref:System.Windows.Forms.ToolStripItem>至<xref:System.Windows.Forms.ToolStripItemAlignment.Right>。
