# <a name="how-to-position-a-toolstripitem-on-a-toolstrip"></a>HOW TO：將 ToolStripItem 放在 ToolStrip 上
您可以移動，或新增<xref:System.Windows.Forms.ToolStripItem>向左或右側<xref:System.Windows.Forms.ToolStrip>。  
  
### <a name="to-move-or-add-a-toolstripitem-to-the-left-side-of-a-toolstrip"></a>若要移動或加入 ToolStrip 的左下方的 prvku ToolStripItem  
  
1. 設定<xref:System.Windows.Forms.ToolStripItemAlignment>的屬性<xref:System.Windows.Forms.ToolStripItem>至<xref:System.Windows.Forms.ToolStripItemAlignment.Left>。  
  
### <a name="to-move-or-add-a-toolstripitem-to-the-right-side-of-a-toolstrip"></a>若要移動或加入 ToolStrip 右側 prvku ToolStripItem  
  
1. 設定<xref:System.Windows.Forms.ToolStripItemAlignment>的屬性<xref:System.Windows.Forms.ToolStripItem>至<xref:System.Windows.Forms.ToolStripItemAlignment.Right>。
