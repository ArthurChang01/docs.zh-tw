# 如何：在 ToolStrip 上放置 ToolStripItem
您可以將 <xref:System.Windows.Forms.ToolStripItem> 移動或加入至 <xref:System.Windows.Forms.ToolStrip> 的左邊或右邊。  
  
### 若要將 ToolStripItem 移動或加入至 ToolStrip 的左邊  
  
1.  將 <xref:System.Windows.Forms.ToolStripItem> 的 <xref:System.Windows.Forms.ToolStripItemAlignment> 屬性設為 <xref:System.Windows.Forms.ToolStripItemAlignment>。  
  
### 若要將 ToolStripItem 移動或加入至 ToolStrip 的右邊  
  
1.  將 <xref:System.Windows.Forms.ToolStripItem> 的 <xref:System.Windows.Forms.ToolStripItemAlignment> 屬性設為 <xref:System.Windows.Forms.ToolStripItemAlignment>。
