# CheckedListBox 控制項概觀 (Windows Form)
Windows Form <xref:System.Windows.Forms.CheckedListBox> 控制項擴充了 <xref:System.Windows.Forms.ListBox> 控制項。  它幾乎可進行清單方塊可做的每件事，也可在清單中的項目旁顯示核取記號。  這兩種控制項的其他不同之處在於：該核取清單方塊只支援 <xref:System.Windows.Forms.DrawMode?displayProperty=fullName>，且只能選取一個或零個項目。  請注意，選取的項目在表單中會反白顯示，而這與核取的項目不同。  
  
 核取清單方塊可具有在設計階段時使用 **String Collection Editor** 加入的項目，或者可在執行階段時使用 <xref:System.Windows.Forms.CheckedListBox.Items%2A> 屬性從集合動態地加入其項目。  如需詳細資訊，請參閱 [如何：從 Windows Form 的 ComboBox、ListBox 或 CheckedListBox 控制項加入或移除項目](../../../../docs/framework/winforms/controls/add-and-remove-items-from-a-wf-combobox.md)。
