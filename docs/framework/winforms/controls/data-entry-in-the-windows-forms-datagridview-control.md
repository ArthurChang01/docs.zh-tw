# <a name="data-entry-in-the-windows-forms-datagridview-control"></a>Windows Form DataGridView 控制項中的資料輸入
`DataGridView`控制項提供數個功能，可讓您變更使用者如何新增或修改控制項中的資料。 比方說，您可以使資料輸入更有效率地藉由提供預設值的新資料列和發生錯誤時警示使用者。
