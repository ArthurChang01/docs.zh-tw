# 如何：對齊 Windows Form 上的多個控制項
若要將 Windows 架構應用程式使用者介面 \(UI\) 的配置標準化，您可以使用單一命令放置控制項群組。  
  
> [!NOTE]
>  根據您目前使用的設定或版本，您所看到的對話方塊與功能表指令可能會與 \[說明\] 中描述的不同。  若要變更設定，請從 \[**工具**\] 功能表中選擇 \[**匯入和匯出設定**\]。  如需詳細資訊，請參閱 [Customizing Development Settings in Visual Studio](http://msdn.microsoft.com/zh-tw/22c4debb-4e31-47a8-8f19-16f328d7dcd3)。  
  
### 若要對齊表單上多個控制項  
  
1.  打開包含控制項的表單，這些控制項是您想要放置在 \[**Windows Form 設計工具**\] 內的。  
  
2.  選取您要對齊的控制項，如此所選取的第一個控制項會是主要控制項，其餘控制項則與之對齊。  
  
3.  在 \[**格式**\] 功能表上指向 \[**對齊**\]，並按一下七個可選用的選項之一。
