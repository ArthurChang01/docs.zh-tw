# <a name="textbox-control-windows-forms"></a>TextBox 控制項 (Windows Form)
Windows Form 文字方塊用來從使用者取得輸入，或顯示的文字。 `TextBox`控制項通常使用於可編輯的文字，雖然它可以也成為唯讀狀態。 文字方塊可以顯示多行、 文字換行到控制項的大小和加入基本的格式。 `TextBox`控制項可讓文字在控制項中輸入或顯示的單一格式。
