# <a name="how-to-set-the-image-displayed-by-a-windows-forms-control-using-the-designer"></a>作法：使用設計工具設定 Windows Forms 控制項所顯示的影像

數個 Windows Forms 控制項可以顯示影像。 影像可以是說明控制項用途的圖示, 例如代表**儲存**命令之按鈕上的磁片圖示。 或者, 圖示可以是背景影像, 為控制項提供您想要的外觀。

## <a name="to-set-the-image-displayed-by-a-control"></a>若要設定控制項所顯示的影像

1. 在 [**屬性**] 視窗中, 選取控制項的 [**影像**] 或 [ **BackgroundImage** ] 屬性, 然後選取![省略號 (Visual Studio](./media/visual-studio-ellipsis-button.png)中的省略號按鈕) 以顯示 [**選取資源**]對話方塊。

2. 選取您想要顯示的影像。
