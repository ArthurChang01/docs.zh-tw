# 如何：使用 BindingSource ResetItem 方法引發變更通知
變更、加入或刪除項目時，控制項的某些資料來源並不會引發變更通知。  利用 <xref:System.Windows.Forms.BindingSource> 元件，您可以繫結至這類資料來源，並從您的程式碼引發變更通知。
