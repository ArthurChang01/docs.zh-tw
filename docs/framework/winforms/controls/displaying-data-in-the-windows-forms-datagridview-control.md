# 在 Windows Form DataGridView 控制項中顯示資料
`DataGridView` 控制項是用來顯示來自各種外部資料來源的資料。  或者，您也可以加控制項中加入資料列和資料行，並且手動在控制項中填入資料。  
  
 當您將控制項繫結至資料來源時，可以根據資料來源的結構描述自動產生資料行。  如果這些資料行不符合您的需求，您可以將它們隱藏、移除或重新排列。  您也可以加入未繫結的資料行，來顯示不是來自資料來源的補充資料。  
  
 此外，您可以使用標準格式 \(例如貨幣格式\) 來顯示資料，也可以者自訂顯示格示，以您想要的方式呈現資料 \(例如變更負數的背景色彩，或是以對應的影像取代字串值\)。
