# <a name="displaying-data-in-the-windows-forms-datagridview-control"></a>在 Windows Form DataGridView 控制項中顯示資料
`DataGridView`控制項用來顯示各種不同的外部資料來源的資料。 或者，您可以將資料列和資料行加入至控制項，並手動填入資料。  
  
 當您將控制項繫結至資料來源時，您可以產生會自動根據資料來源的結構描述的資料行。 如果只是依您想要時，才不會出現這些資料行，您可以隱藏、 移除或重新排列它們。 您也可以加入未繫結的資料行，顯示不是來自資料來源的補充資料。  
  
 此外，您可以顯示資料，可使用標準格式 （例如貨幣格式），或者您可以自訂顯示格式來呈現您的資料，但您需要 （例如變更背景色彩為負數，或取代字串值與對應的映像）。
