# <a name="radiobutton-control-windows-forms"></a>RadioButton 控制項 (Windows Form)
Windows Forms `RadioButton` 控制項會向使用者呈現兩個或多個互斥選項的集合。 雖然選項按鈕和核取方塊的功能可能類似，但有一項重要的差異：當使用者選取選項按鈕時，也無法選取相同群組中的其他選項按鈕。
