# <a name="radiobutton-control-windows-forms"></a>RadioButton 控制項 (Windows Form)
Windows Form`RadioButton`控制項呈現給使用者的一組的兩個或多個互斥的選項。 選項按鈕和核取方塊可能看起來運作同樣地，雖然還有一項重要差異： 當使用者選取的選項按鈕，不能也選取相同的群組中的其他選項按鈕。
