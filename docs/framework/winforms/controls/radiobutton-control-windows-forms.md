# RadioButton 控制項 (Windows Form)
Windows Form `RadioButton` 控制項提供使用者一組兩個或多個互斥 \(Mutually Exclusive\) 選項。  選項按鈕和核取方塊的功能雖然類似，但有一個很重要的差異：當使用者選取一個選項按鈕時，便無法選取相同群組中的其他選項按鈕。
