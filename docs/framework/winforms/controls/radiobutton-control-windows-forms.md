# <a name="radiobutton-control-windows-forms"></a>RadioButton 控制項 (Windows Form)
Windows Form`RadioButton`控制項向使用者顯示一組兩個或多個互斥。 選項按鈕和核取方塊看起來可能類似函式，而是一項重要差異： 無法選取以及當使用者選取選項按鈕，在相同群組中的其他選項按鈕。
