# FlowLayoutPanel 控制項 (Windows Form)
`FlowLayoutPanel` 控制項會以水平或垂直的流動方向來排列其內容。  其內容可以從某一資料列換行至下一個資料列，或從某一資料行換行至下一個資料行。  此外，也可裁剪其內容而不換行。  
  
 本節中的主題描述可讓您將 `FlowLayoutPanel` 功能建置在應用程式中的概念和技術。
