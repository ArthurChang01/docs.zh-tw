# <a name="hscrollbar-and-vscrollbar-controls-windows-forms"></a>HScrollBar 和 VScrollBar 控制項 (Windows Form)
Windows Form 捲軸控制項可用來在應用程式或控制項內以水平或垂直方式捲動，能夠輕鬆瀏覽一長串的項目或大量的資訊。 捲軸是 Windows 介面中的常見項目。
