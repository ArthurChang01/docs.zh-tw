# <a name="listview-control-windows-forms"></a>ListView 控制項 (Windows Form)
Windows Form `ListView` 控制項顯示具有圖示的項目清單。 若要建立像 Windows 檔案總管右窗格的使用者介面，您可以使用清單檢視。
