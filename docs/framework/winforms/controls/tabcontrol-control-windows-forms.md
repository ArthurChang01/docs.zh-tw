# <a name="tabcontrol-control-windows-forms"></a>TabControl 控制項 (Windows Form)
Windows Form `TabControl` 顯示多個索引標籤，像是在筆記本中的分隔線或在檔案櫃中一組資料夾的標籤。 索引標籤可以包含圖片和其他控制項。 使用 `TabControl` 建立屬性頁。
