# DataGrid 控制項 (Windows Form)
> [!NOTE]
>  <xref:System.Windows.Forms.DataGridView> 控制項會取代 `DataGrid` 控制項並加入其他功能，不過您也可以選擇保留 `DataGrid` 控制項，以提供回溯相容性及未來使用。  如需詳細資訊，請參閱 [Windows Form DataGridView 和 DataGrid 控制項之間的差異](../../../../docs/framework/winforms/controls/differences-between-the-windows-forms-datagridview-and-datagrid-controls.md)。  
  
 Windows Form `DataGrid` 控制項會提供 [!INCLUDE[vstecado](../../../../includes/vstecado-md.md)] 資料集的使用者介面，以顯示表格式資料並允許更新資料來源。  
  
 當 `DataGrid` 控制項設定了有效資料來源時，控制項就會自動填入 \(Populate\)，並根據資料的形式來建立資料行和資料列。  `DataGrid` 控制項可用來顯示單一資料表或一組資料表間的階層式關聯性。
