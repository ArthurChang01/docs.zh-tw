# MenuStrip 控制項 (Windows Form)
此控制項會將應用程式的命令分組，並使其容易存取。  
  
## 在本節中  
 [MenuStrip 控制項概觀](../../../../docs/framework/winforms/controls/menustrip-control-overview-windows-forms.md)

 示範如何在父視窗功能表上建立一份所有作用中的子表單清單。

 描述如何停用整個功能表或個別的功能表命令。  
  
 [如何：隱藏 ToolStripMenuItems](../../../../docs/framework/winforms/controls/how-to-hide-toolstripmenuitems.md)  
 描述如何隱藏整個功能表或個別的功能表命令。  
  
 [如何：將 MenuStrip 插入至 MDI 下拉式功能表](../../../../docs/framework/winforms/controls/how-to-insert-a-menustrip-into-an-mdi-drop-down-menu-windows-forms.md)  
 描述如何設定數個屬性，從 MDI 子功能表中插入功能表項目的群組到 MDI 父功能表的下拉式部分。  
  
 [如何：從 MDI 下拉式功能表移除 ToolStripMenuItem](../../../../docs/framework/winforms/controls/how-to-remove-a-toolstripmenuitem-from-an-mdi-drop-down-menu-windows-forms.md)
