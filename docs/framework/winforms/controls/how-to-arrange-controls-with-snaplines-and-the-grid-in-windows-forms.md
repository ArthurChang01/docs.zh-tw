# 如何：使用 Windows Form 中的對齊線和格線排列控制項
使用 [!INCLUDE[vsprvs](../../../../includes/vsprvs-md.md)] 的配置功能，您可以精確地指出控制項放置在表單上的位置。  加入至表單或在表單上移動的控制項可自動對齊至 Windows Form 設計工具格線的資料列和資料行，或者您可以使用對齊線功能來對齊控制項。  
  
> [!NOTE]
>  根據您目前使用的設定或版本，您所看到的對話方塊與功能表指令可能會與 \[說明\] 中描述的不同。  若要變更設定，請從 \[**工具**\] 功能表中選擇 \[**匯入和匯出設定**\]。  如需詳細資訊，請參閱 [Customizing Development Settings in Visual Studio](http://msdn.microsoft.com/zh-tw/22c4debb-4e31-47a8-8f19-16f328d7dcd3)。  
  
### 若要讓控制項貼齊格線  
  
-   在 Windows Form 設計工具 \[**選項**\] 對話方塊中選取 \[**SnapToGrid**\] 配置模式。  
  
     如需詳細資訊，請參閱[General, Windows Forms Designer, Options Dialog Box](http://msdn.microsoft.com/zh-tw/8dd170af-72f0-4212-b04b-034ceee92834)。  所有的控制項現在沿著格線上的點對齊。  
  
     您可以藉由鎖定控制項到定位，將個別控制項貼齊到格線上。  然而，當它們被鎖定時，它們不能夠移動或改變大小。  如需鎖定控制項的詳細資訊，請參閱 [如何：鎖定 Windows Form 的控制項](../../../../docs/framework/winforms/controls/how-to-lock-controls-to-windows-forms.md)。  
  
### 若要使用對齊線來對齊控制項  
  
-   在 Windows Form 設計工具 \[**選項**\] 對話方塊中選取 \[**對齊線**\] 配置模式。  
  
     如需詳細資訊，請參閱[逐步解說：使用對齊線排列 Windows Form 上的控制項](../../../../docs/framework/winforms/controls/walkthrough-arranging-controls-on-windows-forms-using-snaplines.md)。  您現在可以使用對齊線來對齊並排列表單上的控制項。
