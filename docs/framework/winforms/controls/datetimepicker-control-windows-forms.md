# <a name="datetimepicker-control-windows-forms"></a>DateTimePicker 控制項 (Windows Form)
Windows Form`DateTimePicker`控制項可讓使用者從清單中的日期或時間選取單一項目。 用來代表日期時，它會出現在兩個部分： 下拉式清單中的文字，以及一個方格，其中會顯示當您按一下清單旁邊的向下箭號表示的日期。
