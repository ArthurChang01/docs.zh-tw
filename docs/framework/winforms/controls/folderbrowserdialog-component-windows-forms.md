# FolderBrowserDialog 元件 (Windows Form)
Windows Form 的 `FolderBrowserDialog` 元件會顯示介面，使用者可以用來瀏覽和選取資料夾，或是建立新資料夾。  它是 [OpenFileDialog 元件](../../../../docs/framework/winforms/controls/openfiledialog-component-windows-forms.md) 的補充元件，用來瀏覽和選取檔案。

 [如何：使用 Windows Form FolderBrowserDialog 元件選擇資料夾](../../../../docs/framework/winforms/controls/how-to-choose-folders-with-the-windows-forms-folderbrowserdialog-component.md)
