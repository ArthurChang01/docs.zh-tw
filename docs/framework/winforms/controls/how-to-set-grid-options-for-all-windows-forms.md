# <a name="how-to-set-grid-options-for-all-windows-forms"></a>作法：設定所有 Windows Forms 的格線選項

當您在 Visual Studio 開發環境中使用時, 可以針對在 Windows Form 設計工具中使用的所有表單和專案設定喜好設定。

## <a name="set-global-windows-forms-options"></a>設定全域 Windows Forms 選項

1. 在 Visual Studio 中, 從 [**工具**] 功能表選取 [**選項**]。

2. 在 [**選項**] 對話方塊的左窗格中, 按一下 [ **Windows Form 設計工具**]。

   在右窗格的 [**版面配置設定**] 標題下, 您可以針對您所建立的所有新表單設定預設方格設定。 這些設定包括格線大小、控制項是否貼齊其中, 以及是否預設為開啟。 此外, 您可以在**對齊**和**對齊線**版面配置模式之間進行選取。 如需對齊線的詳細資訊[, 請參閱逐步解說:使用對齊線](walkthrough-arranging-controls-on-windows-forms-using-snaplines.md)排列 Windows Forms 上的控制項。
