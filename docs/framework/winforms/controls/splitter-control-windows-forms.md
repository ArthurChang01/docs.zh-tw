# Splitter 控制項 (Windows Form)
Windows Form `Splitter` 控制項的用途在於執行階段重新調整停駐控制項的大小。  `Splitter` 控制項通常用於內含控制項的表單上，這些控制項會顯示各種長度的資料 \(如 Windows 檔案總管\)，其資料窗格在不同時間會顯示不同寬度的資訊。  
  
> [!NOTE]
>  雖然 <xref:System.Windows.Forms.SplitContainer> 會取代並且將功能加入至之前版本的 `Splitter` 控制項；不過，也可以選擇保留 `Splitter`，以提供回溯相容性 \(Backward Compatibility\) 以及供未來使用。
