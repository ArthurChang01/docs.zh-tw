# Windows Form 控制項
在您設計和修改 Windows Form 應用程式的使用者介面時，您將需要加入、調整和放置控制項。  控制項是包含在表單物件中的物件。  每一種控制項型別都有其個別的屬性、方法和事件的集合，以便適應各個特定的用途。  您可以在設計工具中操作控制項並寫入程式碼，以便在執行階段動態加入控制項。
