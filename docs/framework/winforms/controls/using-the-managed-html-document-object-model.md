# 使用 Managed HTML 文件物件模型
Managed HTML 文件物件模型 \(DOM\) 會根據 Internet Explorer 所公開之 HTML 類別的 [!INCLUDE[dnprdnshort](../../../../includes/dnprdnshort-md.md)] 來提供包裝函式。  請使用這些類別來管理 <xref:System.Windows.Forms.WebBrowser> 控制項內所裝載的 HTML 網頁，或者從頭開始建置新頁面。  
  
## 在本節中  
 [如何：存取 Managed HTML 文件物件模型](../../../../docs/framework/winforms/controls/how-to-access-the-managed-html-document-object-model.md)  
 描述如何從 Windows Form 應用程式或從 Internet Explorer 內所裝載的 <xref:System.Windows.Forms.UserControl> 取得有效的 <xref:System.Windows.Forms.HtmlDocument> 執行個體。  
  
 [如何：存取 Managed HTML 文件物件模型中的 HTML 原始檔](../../../../docs/framework/winforms/controls/how-to-access-the-html-source-in-the-managed-html-document-object-model.md)  
 描述如何取得原始、未經修改的 HTML 原始檔，以及如何取得反映 DOM 目前狀態的「即時」原始檔。  
  
 [如何：變更 Managed HTML 文件物件模型中的項目樣式](../../../../docs/framework/winforms/controls/how-to-change-styles-on-an-element-in-the-managed-html-document-object-model.md)  
 描述如何管理用來控制項目視覺顯示的樣式。  
  
 [存取 Managed HTML 文件物件模型中的框架](../../../../docs/framework/winforms/controls/accessing-frames-in-the-managed-html-document-object-model.md)  
 描述什麼是框架和框架組，以及如何存取框架的 DOM。  
  
 [存取 Managed HTML 文件物件模型上未公開的成員](../../../../docs/framework/winforms/controls/accessing-unexposed-members-on-the-managed-html-document-object-model.md)  
 描述如何存取沒有 Managed 對等用法的基礎 DOM 成員。
