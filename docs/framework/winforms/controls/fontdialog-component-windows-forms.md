# FontDialog 元件 (Windows Form)
Windows Form <xref:System.Windows.Forms.FontDialog> 元件是預先設定的對話方塊。  它與 Windows 作業系統所公開的 \[**字型**\] 對話方塊是相同的對話方塊。  該元件是繼承自 <xref:System.Windows.Forms.CommonDialog> 類別 \(Class\)。  
  
## 在本節中  
 [FontDialog 元件概觀](../../../../docs/framework/winforms/controls/fontdialog-component-overview-windows-forms.md)  
 簡介 <xref:System.Windows.Forms.FontDialog> 元件的一般概念，您可以使用該元件來顯示設定前的對話方塊。  使用者可以使用對話方塊來管理字型及其設定。  
  
 [如何：使用 FontDialog 元件顯示字型清單](../../../../docs/framework/winforms/controls/how-to-show-a-font-list-with-the-fontdialog-component.md)  
 說明如何在執行階段透過 <xref:System.Windows.Forms.FontDialog> 元件的執行個體選擇字型。
