# <a name="how-to-split-a-window-horizontally-using-the-designer"></a>HOW TO：使用設計工具水平分割視窗

這個範例會建立分割<xref:System.Windows.Forms.SplitContainer> , 以水準分隔控制項。

> [!NOTE]
> <xref:System.Windows.Forms.SplitContainer>控制項<xref:System.Windows.Forms.SplitContainer.Orientation%2A>的屬性會決定分隔器的方向, 而不是控制項本身。

## <a name="to-split-a-window-horizontally"></a>水準分割視窗

在 [**屬性**] 視窗中, <xref:System.Windows.Forms.SplitContainer.Orientation%2A>將<xref:System.Windows.Forms.SplitContainer>控制項的屬性設<xref:System.Windows.Forms.Orientation.Horizontal>為。
