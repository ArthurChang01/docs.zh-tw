# <a name="toolstrippanel-control"></a>ToolStripPanel 控制項
`ToolStripPanel` 控制項可讓共用的停駐時工具區域內的水平或垂直空間，並排列<xref:System.Windows.Forms.ToolStrip>控制當您不需要四個面板和中間面板<xref:System.Windows.Forms.ToolStripContainer>。  
  
 本節中的主題說明您可以用來將 `ToolStripPanel` 功能建置在應用程式中的概念和技術。
