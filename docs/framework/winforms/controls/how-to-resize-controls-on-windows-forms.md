# <a name="how-to-resize-controls-on-windows-forms"></a>如何：重新調整 Windows Form 上控制項的大小
您也可以調整個別控制項，以及您可以多個調整控制項的大小相同或不同的種類，例如<xref:System.Windows.Forms.Button>和<xref:System.Windows.Forms.GroupBox>控制項。  
  
### <a name="to-resize-a-control"></a>若要調整控制項大小  
  
1.  按一下控制項，以調整大小並拖曳其中一個八個調整大小控點。  
  
    > [!NOTE]
    >  選取的控制項，然後按住 SHIFT 鍵可調整控制項的一個像素大小一次按下方向鍵。 按住 SHIFT 與 CTRL 鍵，調整大幅度控制項的大小，請按下或向右箭號鍵。  
  
### <a name="to-resize-multiple-controls-on-a-form"></a>若要在表單上的多個控制項調整大小  
  
1.  按住 CTRL 或 SHIFT 鍵，然後選取您要調整大小的控制項。 您選取的第一個控制項的大小用於其他控制項。  
  
2.  在**格式**功能表上，選擇**成相同大小**，然後選取四個選項的其中一個。 前三個命令會變更以符合第一個選取的控制項控制項的大小。
