# 如何：重新調整 Windows Form 上控制項的大小
您可以調整個別控制項的大小，也可以調整相同或不同種類之多重控制項的大小，例如 <xref:System.Windows.Forms.Button> 和 <xref:System.Windows.Forms.GroupBox> 控制項。  
  
### 若要調整控制項的大小  
  
1.  按一下要調整大小的控制項，然後拖曳八個縮放控點 \(Sizing Handle\) 其中之一。  
  
    > [!NOTE]
    >  選取控制項，然後在按住 SHIFT 鍵時按下方向鍵，控制項的大小就可一次調整一個像素。  按住 SHIFT 和 CTRL 鍵不放，然後再按向下鍵或向右鍵，即可大幅調整控制項的大小。  
  
### 若要調整表單上多個控制項的大小  
  
1.  按住 CTRL 或 SHIFT 鍵不放，並選取想要調整大小的控制項。  其他控制項將使用第一個選取的控制項之大小。  
  
2.  從 \[**格式**\] 功能表選擇 \[**設定成相同大小**\]，然後從四個選項中選取一項。  前三個命令會改變控制項的大小以符合第一個選取的控制項。
