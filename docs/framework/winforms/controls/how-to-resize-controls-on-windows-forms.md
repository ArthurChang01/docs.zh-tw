### <a name="to-resize-a-control"></a><span data-ttu-id="67425-104">調整控制項的大小</span><span class="sxs-lookup"><span data-stu-id="67425-104">To resize a control</span></span>

2. <span data-ttu-id="67425-111">在 [**格式**] 功能表上, 選擇 [**設成相同大小**], 然後選取四個選項的其中一個。</span><span class="sxs-lookup"><span data-stu-id="67425-111">On the **Format** menu, choose **Make Same Size**, and select one of the four options.</span></span> <span data-ttu-id="67425-112">前三個命令會變更控制項的維度, 以符合第一個選取的控制項。</span><span class="sxs-lookup"><span data-stu-id="67425-112">The first three commands change the dimensions of the controls to match the first-selected control.</span></span>
