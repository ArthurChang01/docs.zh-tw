# <a name="imagelist-component-windows-forms"></a>ImageList 元件 (Windows Form)
Windows Form `ImageList` 元件可用來儲存影像，然後可以用控制項來顯示這些影像。 影像清單可讓您針對單一且一致的影像目錄來撰寫程式碼。
