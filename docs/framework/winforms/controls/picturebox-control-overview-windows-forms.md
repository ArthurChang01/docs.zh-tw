# <a name="picturebox-control-overview-windows-forms"></a>PictureBox 控制項概觀 (Windows Form)
Windows Form<xref:System.Windows.Forms.PictureBox>控制項用來顯示點陣圖、 GIF、 JPEG、 中繼檔或圖示格式的圖形。  
  
## <a name="key-properties-and-methods"></a>索引鍵屬性和方法  
 顯示的圖片由<xref:System.Windows.Forms.PictureBox.Image%2A>屬性，可以在執行階段或設計階段設定。 您也可以指定映像設定<xref:System.Windows.Forms.PictureBox.ImageLocation%2A>屬性，然後將以同步方式使用的映像載入<xref:System.Windows.Forms.PictureBox.Load%2A>方法或使用以非同步方式<xref:System.Windows.Forms.PictureBox.LoadAsync%2A>方法。 <xref:System.Windows.Forms.PictureBox.SizeMode%2A>屬性控制的映像和控制如何彼此配合。 如需詳細資訊，請參閱[如何： 修改的大小或位置的圖片，在執行階段](../../../../docs/framework/winforms/controls/how-to-modify-the-size-or-placement-of-a-picture-at-run-time-windows-forms.md)。
