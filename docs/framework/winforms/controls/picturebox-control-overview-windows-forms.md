# <a name="picturebox-control-overview-windows-forms"></a>PictureBox 控制項概觀 (Windows Form)
Windows Form<xref:System.Windows.Forms.PictureBox>控制項用來顯示點陣圖、 GIF、 JPEG、 中繼檔或圖示格式的圖形。  
  
## <a name="key-properties-and-methods"></a>索引鍵屬性和方法  
 取決於顯示的圖片<xref:System.Windows.Forms.PictureBox.Image%2A>屬性，可以在執行階段，或在設計階段設定。 您也可以藉由設定來指定映像<xref:System.Windows.Forms.PictureBox.ImageLocation%2A>屬性，然後載入以同步方式使用的映像<xref:System.Windows.Forms.PictureBox.Load%2A>方法，或以非同步方式使用<xref:System.Windows.Forms.PictureBox.LoadAsync%2A>方法。 <xref:System.Windows.Forms.PictureBox.SizeMode%2A>屬性可讓您控制的映像和控制如何彼此配合。 如需詳細資訊，請參閱[如何：在執行階段修改的大小或位置的圖片](how-to-modify-the-size-or-placement-of-a-picture-at-run-time-windows-forms.md)。
