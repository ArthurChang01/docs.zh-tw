# <a name="trackbar-control-windows-forms"></a>TrackBar 控制項 (Windows Form)
Windows Forms `TrackBar` 控制項（有時也稱為「滑杆」控制項）是用來流覽大量資訊，或用來以視覺方式調整數值設定。 `TrackBar` 控制項有兩個部分： thumb （也稱為滑杆）和刻度標記。 捲動方塊是可以調整的部分。 其位置會對應至 <xref:System.Windows.Forms.TrackBar.Value%2A> 屬性。 刻度是以固定間隔分隔的視覺指示器。 追蹤列會以您指定的增量移動，而且可以水準或垂直對齊。 追蹤列的範例使用是用來設定資料指標閃爍頻率或滑鼠速度。
