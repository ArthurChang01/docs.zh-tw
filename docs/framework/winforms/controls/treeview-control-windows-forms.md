# TreeView 控制項 (Windows Form)
Windows Form `TreeView` 控制項會顯示節點的階層，就像 Windows 作業系統中 Windows 檔案總管功能左窗格顯示檔案和資料夾的方式。  
  
## 在本節中  
 [TreeView 控制項概觀](../../../../docs/framework/winforms/controls/treeview-control-overview-windows-forms.md)  
 說明此控制項是什麼，並說明其重要功能與屬性。  
  
 [如何：使用 Windows Form TreeView 控制項加入和移除節點](../../../../docs/framework/winforms/controls/how-to-add-and-remove-nodes-with-the-windows-forms-treeview-control.md)  
 說明如何在樹狀檢視中加入和移除節點。  
  
 [如何：將自訂資訊加入 TreeView 或 ListView 控制項 \(Windows Form\)](../../../../docs/framework/winforms/controls/add-custom-information-to-a-treeview-or-listview-control-wf.md)  
 說明如何在清單檢視中衍生項目，或在樹狀檢視中衍生節點，以加入您需要的任何欄位、方法或建構函式。  
  
 [如何：判斷按下哪個 TreeView 節點](../../../../docs/framework/winforms/controls/how-to-determine-which-treeview-node-was-clicked-windows-forms.md)  
 說明如何判斷已按下樹狀檢視中的哪個節點，讓應用程式能夠適當地回應。  
  
 [如何：逐一查看 Windows Form TreeView 控制項的所有節點](http://msdn.microsoft.com/zh-tw/427f8928-ebcf-4beb-887f-695b905d5134)  
 說明如何檢查樹狀檢視中的每個節點。  
  
 [如何：設定 Windows Form TreeView 控制項的圖示](http://msdn.microsoft.com/zh-tw/c14ddcc0-e5a6-4c21-a2d5-6799fd491781)  
 說明如何顯示樹狀檢視的節點圖示。  
  
 [如何：將捷徑功能表附加至 TreeView 節點](../../../../docs/framework/winforms/controls/how-to-attach-a-shortcut-menu-to-a-treeview-node.md)
