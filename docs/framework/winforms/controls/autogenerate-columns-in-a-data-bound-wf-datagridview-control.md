# <a name="how-to-autogenerate-columns-in-a-data-bound-windows-forms-datagridview-control"></a>HOW TO：自動產生資料繫結 Windows Form DataGridView 控制項中的資料行
下列程式碼範例示範如何顯示資料行中的繫結的資料來源從<xref:System.Windows.Forms.DataGridView>控制項。 當<xref:System.Windows.Forms.DataGridView.AutoGenerateColumns%2A>屬性值是`true`（預設值），<xref:System.Windows.Forms.DataGridViewColumn>建立每個資料行中的資料來源資料表。  
  
 如果<xref:System.Windows.Forms.DataGridView>控制項已有資料行，當您設定<xref:System.Windows.Forms.DataGridViewComboBoxColumn.DataSource%2A>屬性、 現有的繫結資料行相較於資料來源中的資料行，而相符項目時予以儲存。 未繫結的資料行，一律會保留。 會移除資料來源中的沒有相符項目是繫結的資料行。 控制項中的沒有相符項目是資料來源中的資料行產生新<xref:System.Windows.Forms.DataGridViewColumn>物件，會新增至結尾<xref:System.Windows.Forms.DataGridView.Columns%2A>集合。
