# ProgressBar 控制項 (Windows Form)
> [!IMPORTANT]
>  <xref:System.Windows.Forms.ToolStripProgressBar> 控制項會取代 <xref:System.Windows.Forms.ProgressBar> 控制項並加入其他功能，不過您也可以選擇保留 <xref:System.Windows.Forms.ProgressBar> 控制項，以提供回溯相容性及未來使用。  
  
 Windows Form <xref:System.Windows.Forms.ProgressBar> 控制項藉由在水平列 \(Horizontal Bar\) 中顯示適當數目的矩形來指出動作的進度。  當動作完成後，水平列將被填滿。  進度列通常用來讓使用者瞭解完成長時間動作所需等待的時間，例如當載入大型檔案時。
