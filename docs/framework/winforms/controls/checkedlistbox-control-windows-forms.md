# <a name="checkedlistbox-control-windows-forms"></a>CheckedListBox 控制項 (Windows Form)
Windows Forms `CheckedListBox` 控制項會顯示專案清單（例如 <xref:System.Windows.Forms.ListBox> 控制項），而且也可以在清單中的專案旁顯示核取記號。
