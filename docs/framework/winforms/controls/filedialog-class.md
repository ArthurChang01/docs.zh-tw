# FileDialog 類別
Windows Form <xref:System.Windows.Forms.FileDialog> 類別是 <xref:System.Windows.Forms.OpenFileDialog> 和 <xref:System.Windows.Forms.SaveFileDialog> 元件常見的基底類別。  您可以變更 <xref:System.Windows.Forms.FileDialog> 類別，這會根據執行應用程式的 Windows 版本而影響這些對話方塊的外觀與行為。
