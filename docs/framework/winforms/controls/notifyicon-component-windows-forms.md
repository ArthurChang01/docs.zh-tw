# NotifyIcon 元件 (Windows Form)
Windows Form `NotifyIcon` 元件在工作列的狀態通知區域中顯示圖示，供在背景中執行的處理序之用，除此以外並沒有使用者介面。  防毒程式即為範例之一，我們可在工作列的狀態告知區按一下圖示來存取。
