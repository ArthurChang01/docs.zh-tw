# <a name="notifyicon-component-windows-forms"></a>NotifyIcon 元件 (Windows Form)
Windows Form`NotifyIcon`元件會顯示在工作列的狀態通知區域圖示處理程序在背景執行，否則不會有使用者介面。 防毒保護程式即為一例，您可以在工作列的狀態通知區域中，按一下圖示來存取這個程式。
