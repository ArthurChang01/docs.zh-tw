# <a name="notifyicon-component-windows-forms"></a>NotifyIcon 元件 (Windows Form)
Windows Forms `NotifyIcon` 元件會在工作列的狀態通知區域中顯示圖示，針對在背景中執行且不會有使用者介面的進程。 防毒保護程式即為一例，您可以在工作列的狀態通知區域中，按一下圖示來存取這個程式。
