# ToolStripStatusLabel 控制項概觀
<xref:System.Windows.Forms.ToolStripStatusLabel> 是 <xref:System.Windows.Forms.StatusStrip> 的標籤 \(Label\)。  如同 <xref:System.Windows.Forms.Label> 或 <xref:System.Windows.Forms.ToolStripLabel>，<xref:System.Windows.Forms.ToolStripStatusLabel> 會為文字、影像或兩者提供不可點選的顯示區域。  <xref:System.Windows.Forms.ToolStripStatusLabel> 已由 <xref:System.Windows.Forms.StatusStrip> 所裝載。  
  
### 重要的 ToolStripStatusLabel 成員  
  
|名稱|描述|  
|--------|--------|  
|<xref:System.Windows.Forms.ToolStripStatusLabel.Spring%2A>|取得或設定值，指出 <xref:System.Windows.Forms.ToolStripStatusLabel> 是否會在調整表單大小時自動填滿 <xref:System.Windows.Forms.StatusStrip> 上的可用空間。|  
|<xref:System.Windows.Forms.ToolStripStatusLabel.BorderSides%2A>|取得或設定值，表示 <xref:System.Windows.Forms.ToolStripStatusLabel> 的哪一邊會顯示框線。|  
|<xref:System.Windows.Forms.ToolStripStatusLabel.BorderStyle%2A>|取得或設定 <xref:System.Windows.Forms.ToolStripStatusLabel> 的框線樣式。|
