# <a name="toolstripstatuslabel-control-overview"></a>ToolStripStatusLabel 控制項概觀
<xref:System.Windows.Forms.ToolStripStatusLabel>當做標籤<xref:System.Windows.Forms.StatusStrip>。 像是<xref:System.Windows.Forms.Label>或<xref:System.Windows.Forms.ToolStripLabel>，則<xref:System.Windows.Forms.ToolStripStatusLabel>提供文字、 影像或同時不可點選式的顯示區域。 <xref:System.Windows.Forms.ToolStripStatusLabel>由裝載<xref:System.Windows.Forms.StatusStrip>。  
  
### <a name="important-toolstripstatuslabel-members"></a>重要 ToolStripStatusLabel 成員  
  
|名稱|描述|  
|----------|-----------------|  
|<xref:System.Windows.Forms.ToolStripStatusLabel.Spring%2A>|取得或設定值，指出是否<xref:System.Windows.Forms.ToolStripStatusLabel>會自動填滿可用空間，在<xref:System.Windows.Forms.StatusStrip>調整表單大小時|  
|<xref:System.Windows.Forms.ToolStripStatusLabel.BorderSides%2A>|取得或設定值，這個值，指出哪一個側邊<xref:System.Windows.Forms.ToolStripStatusLabel>顯示框線。|  
|<xref:System.Windows.Forms.ToolStripStatusLabel.BorderStyle%2A>|取得或設定的框線樣式<xref:System.Windows.Forms.ToolStripStatusLabel>。|
