# ToolStripPanel 控制項概觀
<xref:System.Windows.Forms.ToolStripPanel> 提供了放置和浮動定位 <xref:System.Windows.Forms.ToolStrip>、<xref:System.Windows.Forms.MenuStrip> 和 <xref:System.Windows.Forms.StatusStrip> 控制項的單一區域。  多個 <xref:System.Windows.Forms.ToolStrip> 可以根據 <xref:System.Windows.Forms.ToolStripPanel> 當中的 <xref:System.Windows.Forms.ToolStripPanelRow.Orientation%2A> 來垂直或水平疊置。  
  
### 重要 ToolStripPanel 成員  
  
|名稱|描述|  
|--------|--------|  
|<xref:System.Windows.Forms.ToolStripPanel.Orientation%2A>|取得或設定值，指出 <xref:System.Windows.Forms.ToolStripPanel> 的水平或垂直方向。|  
|<xref:System.Windows.Forms.ToolStripPanel.Renderer%2A>|取得或設定 <xref:System.Windows.Forms.ToolStripRenderer>，用來自訂 <xref:System.Windows.Forms.ToolStripPanel> 的外觀。|  
|<xref:System.Windows.Forms.ToolStripPanel.RenderMode%2A>|取得或設定套用至 <xref:System.Windows.Forms.ToolStripPanel> 的繪製樣式。|  
|<xref:System.Windows.Forms.ToolStripPanel.RowMargin%2A>|取得或設定 <xref:System.Windows.Forms.ToolStripPanelRow> 和 <xref:System.Windows.Forms.ToolStripPanel> 之間的間距，以像素為單位。|  
|<xref:System.Windows.Forms.ToolStripPanel.Rows%2A>|取得這個 <xref:System.Windows.Forms.ToolStripPanel> 中的 <xref:System.Windows.Forms.ToolStripPanelRow>。|  
|<xref:System.Windows.Forms.ToolStripPanel.Join%2A>|加入 <xref:System.Windows.Forms.ToolStrip> 至 <xref:System.Windows.Forms.ToolStripPanel> 中。|
