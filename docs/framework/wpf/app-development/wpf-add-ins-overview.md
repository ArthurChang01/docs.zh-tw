<span data-ttu-id="360da-130">最後，開發強固的增益集模型是重要的工作。</span><span class="sxs-lookup"><span data-stu-id="360da-130">Ultimately, developing a robust add-in model is a non-trivial undertaking.</span></span> <span data-ttu-id="360da-131">基於這個理由，.NET Framework 提供基礎結構建置增益集模型。</span><span class="sxs-lookup"><span data-stu-id="360da-131">For this reason, the .NET Framework provides an infrastructure for building add-in models.</span></span>
