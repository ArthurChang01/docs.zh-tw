# 裝載 WPF 應用程式
WPF XAML 瀏覽器應用程式 \(XBAP\) 是豐富型用戶端應用程式，可以部署到 Web 伺服器並在瀏覽器中啟動。  WPF 主應用程式 \(PresentationHost.exe\) 會註冊為 XBAP 和 XAML 檔案適用的 Shell 和 MIME 處理常式。  因此，Internet Explorer 知道在 XBAP 啟動時要啟動 WPF 主應用程式。  Firefox 使用者可以安裝 Firefox 附加元件，讓 Firefox 也可以裝載 XBAP。  若要將 XBAP 裝載於其他瀏覽器或獨立應用程式中，可以使用 WPF 提供的原生瀏覽器裝載 API。
