# <a name="windows-in-wpf-applications"></a>WPF 應用程式中的視窗
使用者透過 windows 的應用程式與互動。 視窗的基本目的是裝載，並顯示內容。 視窗主機取決於資料類型的應用程式上運作，其中可以包含媒體、 Extensible Application Markup Language (XAML) 頁面、 網頁、 文件、 資料庫資料表和記錄，以及系統資訊的內容類型。
