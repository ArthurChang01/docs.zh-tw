# WPF 應用程式中的視窗
使用者是透過視窗與應用程式進行互動。  視窗的基本用途是用來裝載 \(Host\) 和顯示內容。  而視窗裝載的內容類型是取決於應用程式操作的資料型別，可以是媒體、可延伸應用程式標記語言 \(XAML\) 頁面、網頁、文件、資料庫資料表和記錄，以及系統資訊。
