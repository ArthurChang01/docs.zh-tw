# <a name="building-and-deploying-wpf-applications"></a>建置和部署 WPF 應用程式
建置和部署模型可讓您建置和部署應用程式，在本機和遠端電腦上，包括下列：  
  
-   MSBuild:.NET 建置系統位於 Microsoft.Build.Tasks.Windows 命名空間。  
  
-   資源： 使用 UI 資源。  
  
-   ClickOnce 部署：.NET 發行與部署系統。
