# <a name="building-and-deploying-wpf-applications"></a>建置和部署 WPF 應用程式
組建和部署模型提供在本機和遠端建立及部署應用程式的功能，包括下列各項：  
  
- MSBuild：位於 Windows 命名空間的 .NET 組建系統。  
  
- 資源：使用 UI 資源。  
  
- ClickOnce 部署： .NET 發行和部署系統。
