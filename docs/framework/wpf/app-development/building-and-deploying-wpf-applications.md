# 建置和部署 WPF 應用程式
建置和部署模型提供在本機和從遠端建置及部署的功能，包括：  
  
-   MSBuild：在 Microsoft.Build.Tasks.Windows 命名空間 \(Namespace\) 內的 .NET 建置系統。  
  
-   資源：使用 UI 資源。  
  
-   ClickOnce 部署：.NET 發行和部署系統。
