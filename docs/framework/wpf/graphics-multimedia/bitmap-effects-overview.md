# <a name="bitmap-effects-overview"></a><span data-ttu-id="74f24-102">點陣圖效果概觀</span><span class="sxs-lookup"><span data-stu-id="74f24-102">Bitmap Effects Overview</span></span>
<span data-ttu-id="74f24-103">點陣圖效果可讓設計人員和開發人員来套用至視覺效果呈現 Windows Presentation Foundation (WPF) 內容。</span><span class="sxs-lookup"><span data-stu-id="74f24-103">Bitmap effects enable designers and developers to apply visual effects to rendered Windows Presentation Foundation (WPF) content.</span></span> <span data-ttu-id="74f24-104">例如，點陣圖效果可讓您輕鬆地套用<xref:System.Windows.Media.Effects.DropShadowBitmapEffect>效果或模糊效果影像或按鈕。</span><span class="sxs-lookup"><span data-stu-id="74f24-104">For example, bitmap effects allow you to easily apply a <xref:System.Windows.Media.Effects.DropShadowBitmapEffect> effect or a blur effect to an image or a button.</span></span>

## <a name="wpf-bitmap-effects"></a><span data-ttu-id="74f24-109">WPF 點陣圖效果</span><span class="sxs-lookup"><span data-stu-id="74f24-109">WPF Bitmap Effects</span></span>  
 <span data-ttu-id="74f24-110">點陣圖效果 (<xref:System.Windows.Media.Effects.BitmapEffect>物件) 是簡單的像素處理作業。</span><span class="sxs-lookup"><span data-stu-id="74f24-110">Bitmap effects (<xref:System.Windows.Media.Effects.BitmapEffect> object) are simple pixel processing operations.</span></span> <span data-ttu-id="74f24-111">點陣圖效果會採用<xref:System.Windows.Media.Imaging.BitmapSource>作為輸入，並產生新<xref:System.Windows.Media.Imaging.BitmapSource>之後套用效果，例如模糊或陰影。</span><span class="sxs-lookup"><span data-stu-id="74f24-111">A bitmap effect takes a <xref:System.Windows.Media.Imaging.BitmapSource> as an input and produces a new <xref:System.Windows.Media.Imaging.BitmapSource> after applying the effect, such as a blur or drop shadow.</span></span> <span data-ttu-id="74f24-112">每個點陣圖效果會公開可控制篩選的屬性，這類<xref:System.Windows.Media.Effects.BlurBitmapEffect.Radius%2A>的<xref:System.Windows.Media.Effects.BlurBitmapEffect>。</span><span class="sxs-lookup"><span data-stu-id="74f24-112">Each bitmap effect exposes properties that can control the filtering properties, such as <xref:System.Windows.Media.Effects.BlurBitmapEffect.Radius%2A> of <xref:System.Windows.Media.Effects.BlurBitmapEffect>.</span></span>  
  
 <span data-ttu-id="74f24-113">特殊案例，在[!INCLUDE[TLA2#tla_winclient](../../../../includes/tla2sharptla-winclient-md.md)]，效果可以為屬性上設定即時<xref:System.Windows.Media.Visual>物件，例如<xref:System.Windows.Controls.Button>或<xref:System.Windows.Controls.TextBox>。</span><span class="sxs-lookup"><span data-stu-id="74f24-113">As a special case, in [!INCLUDE[TLA2#tla_winclient](../../../../includes/tla2sharptla-winclient-md.md)], effects can be set as properties on live <xref:System.Windows.Media.Visual> objects, such as a <xref:System.Windows.Controls.Button> or <xref:System.Windows.Controls.TextBox>.</span></span> <span data-ttu-id="74f24-114">像素處理會在執行階段套用並進行轉譯。</span><span class="sxs-lookup"><span data-stu-id="74f24-114">The pixel processing is applied and rendered at run-time.</span></span> <span data-ttu-id="74f24-115">在此情況下，在轉譯時<xref:System.Windows.Media.Visual>會自動轉換為其<xref:System.Windows.Media.Imaging.BitmapSource>對等和做為輸入<xref:System.Windows.Media.Effects.BitmapEffect>。</span><span class="sxs-lookup"><span data-stu-id="74f24-115">In this case, at the time of rendering, a <xref:System.Windows.Media.Visual> is automatically converted to its <xref:System.Windows.Media.Imaging.BitmapSource> equivalent and is fed as input to the <xref:System.Windows.Media.Effects.BitmapEffect>.</span></span> <span data-ttu-id="74f24-116">輸出會取代<xref:System.Windows.Media.Visual>物件的預設轉譯行為。</span><span class="sxs-lookup"><span data-stu-id="74f24-116">The output replaces the <xref:System.Windows.Media.Visual> object's default rendering behavior.</span></span> <span data-ttu-id="74f24-117">這就是為什麼<xref:System.Windows.Media.Effects.BitmapEffect>物件會強制在軟體中轉譯只也就是沒有視覺效果上的硬體加速時套用效果的視覺效果。</span><span class="sxs-lookup"><span data-stu-id="74f24-117">This is why <xref:System.Windows.Media.Effects.BitmapEffect> objects force visuals to render in software only i.e. no hardware acceleration on visuals when effects are applied.</span></span>  
  
-   <span data-ttu-id="74f24-118"><xref:System.Windows.Media.Effects.BlurBitmapEffect> 模擬物件出現失焦。</span><span class="sxs-lookup"><span data-stu-id="74f24-118"><xref:System.Windows.Media.Effects.BlurBitmapEffect> simulates an object that appears out-of-focus.</span></span>  
  
-   <span data-ttu-id="74f24-119"><xref:System.Windows.Media.Effects.OuterGlowBitmapEffect> 建立物件周圍的色彩的光暈。</span><span class="sxs-lookup"><span data-stu-id="74f24-119"><xref:System.Windows.Media.Effects.OuterGlowBitmapEffect> creates a halo of color around the perimeter of an object.</span></span>  
  
-   <span data-ttu-id="74f24-120"><xref:System.Windows.Media.Effects.DropShadowBitmapEffect> 建立物件後方的陰影。</span><span class="sxs-lookup"><span data-stu-id="74f24-120"><xref:System.Windows.Media.Effects.DropShadowBitmapEffect> creates a shadow behind an object.</span></span>  
  
-   <span data-ttu-id="74f24-121"><xref:System.Windows.Media.Effects.BevelBitmapEffect> 建立根據指定的曲線影像表面的斜面。</span><span class="sxs-lookup"><span data-stu-id="74f24-121"><xref:System.Windows.Media.Effects.BevelBitmapEffect> creates a bevel which raises the surface of an image according to a specified curve.</span></span>  
  
-   <span data-ttu-id="74f24-122"><xref:System.Windows.Media.Effects.EmbossBitmapEffect> 建立的凹凸貼圖<xref:System.Windows.Media.Visual>以便從人工光源創造深度和紋理的印象。</span><span class="sxs-lookup"><span data-stu-id="74f24-122"><xref:System.Windows.Media.Effects.EmbossBitmapEffect> creates a bump mapping of a <xref:System.Windows.Media.Visual> to give the impression of depth and texture from an artificial light source.</span></span>  
  
> [!NOTE]
>  [!INCLUDE[TLA2#tla_wpf](../../../../includes/tla2sharptla-wpf-md.md)]<span data-ttu-id="74f24-123"> 點陣圖效果是在軟體模式中轉譯。</span><span class="sxs-lookup"><span data-stu-id="74f24-123"> bitmap effects are rendered in software mode.</span></span> <span data-ttu-id="74f24-124">套用效果的任何物件也會在軟體中轉譯。</span><span class="sxs-lookup"><span data-stu-id="74f24-124">Any object that applies an effect will also be rendered in software.</span></span> <span data-ttu-id="74f24-125">對大型視覺物件使用點陣圖效果或為點陣圖效果的屬性建立動畫時，效能會降低最多。</span><span class="sxs-lookup"><span data-stu-id="74f24-125">Performance is degraded the most when using Bitmap effects on large visuals or animating properties of a Bitmap effect.</span></span> <span data-ttu-id="74f24-126">這並不是說您不應該以此種方式使用點陣圖效果，而是您應特別小心並徹底測試，以確保使用者能得到預期的體驗。</span><span class="sxs-lookup"><span data-stu-id="74f24-126">This is not to say that you should not use Bitmap effects in this way at all, but you should use caution and test thoroughly to ensure that your users are getting the experience you expect.</span></span>
