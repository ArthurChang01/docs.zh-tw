# 繪圖
<xref:System.Windows.Media.Drawing> 物件是用來有效繪製圖案、影像或文字。  當使用 <xref:System.Windows.Media.DrawingBrush> 進行繪製，或使用 <xref:System.Windows.Media.Visual> 物件進行程式設計時，便會用到繪圖。
