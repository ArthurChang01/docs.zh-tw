# 資料繫結 (WPF)
Windows Presentation Foundation \(WPF\) 資料繫結 \(Data Binding\) 在資料的展示和互動上，提供應用程式簡單而一致的方式。  項目可以使用 Common Language Runtime \(CLR\) 物件和 XML 的格式，繫結到各種資料來源的資料。
