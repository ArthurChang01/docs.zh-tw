# WPF 中的 XAML
[!INCLUDE[TLA#tla_xaml](../../../../includes/tlasharptla-xaml-md.md)] 是用於宣告式應用程式設計的標記語言。  [!INCLUDE[TLA#tla_winclient](../../../../includes/tlasharptla-winclient-md.md)] 會實作 XAML 處理器，並提供 XAML 語言支援。  [!INCLUDE[TLA2#tla_winclient](../../../../includes/tla2sharptla-winclient-md.md)] 型別的實作，即是為了針對 XAML 表示提供必要的型別支援。一般而言，您可以使用 XAML 標記建立 [!INCLUDE[TLA2#tla_winclient](../../../../includes/tla2sharptla-winclient-md.md)] 應用程式大部分的 UI。
