# 資源 (WPF)
資源是可以在不同的地方，在您的應用程式中重複使用的物件。  WPF 支援不同類型的資源。  這些資源是主要的兩種類型的資源: XAML 資源和資源資料檔。  XAML 資源的範例包括 \[筆刷和樣式。  資源資料檔案是應用程式所需的非可執行檔的資料。
