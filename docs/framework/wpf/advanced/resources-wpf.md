# <a name="resources-wpf"></a>資源 (WPF)
資源是可在應用程式中不同位置重複使用的物件。 WPF 支援不同類型的資源。 這些資源主要是兩種類型的資源： XAML 資源和資源資料檔。 XAML 資源的範例包括筆刷和樣式。 資源資料檔是應用程式所需的無法執行資料檔案。
