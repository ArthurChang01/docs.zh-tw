 <span data-ttu-id="0a1ff-133">如需如何在自訂類別中實作通道/事件反昇輸入路由事件組的詳細資訊，請參閱[建立自訂路由事件](how-to-create-a-custom-routed-event.md)。</span><span class="sxs-lookup"><span data-stu-id="0a1ff-133">For more information about how to implement tunnel/bubble input routed event pairs in a custom class, see [Create a Custom Routed Event](how-to-create-a-custom-routed-event.md).</span></span>
