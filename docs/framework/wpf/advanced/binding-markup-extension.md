# <a name="binding-markup-extension"></a><span data-ttu-id="4539f-102">繫結標記延伸</span><span class="sxs-lookup"><span data-stu-id="4539f-102">Binding Markup Extension</span></span>
<span data-ttu-id="4539f-103">將屬性值延遲為數據系結的值, 建立中繼運算式物件, 並在執行時間解讀套用至元素及其系結的資料內容。</span><span class="sxs-lookup"><span data-stu-id="4539f-103">Defers a property value to be a data-bound value, creating an intermediate expression object and interpreting the data context that applies to the element and its binding at run time.</span></span>  
  
## <a name="binding-expression-usage"></a><span data-ttu-id="4539f-104">系結運算式使用方式</span><span class="sxs-lookup"><span data-stu-id="4539f-104">Binding Expression Usage</span></span>  
  
```  
<object property="{Binding}" .../>  
-or-  
<object property="{Binding  bindProp1=value1[, bindPropN=valueN]*}" ...  
/>  
-or-  
<object property="{Binding path}" .../>  
-or  
<object property="{Binding path[, bindPropN=valueN]*}" .../>  
```
