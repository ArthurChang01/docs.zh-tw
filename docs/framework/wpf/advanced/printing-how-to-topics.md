# 列印 HOW TO 主題
本節中的主題示範如何使用 [!INCLUDE[TLA#tla_winclient](../../../../includes/tlasharptla-winclient-md.md)] 中的列印功能與列印系統管理功能，以及新增的 [!INCLUDE[TLA#tla_xps](../../../../includes/tlasharptla-xps-md.md)] 列印路徑。  
  
## 在本節中  
 [叫用列印對話方塊](../../../../docs/framework/wpf/advanced/how-to-invoke-a-print-dialog.md)  
 使用 XAML 標記宣告 [!INCLUDE[TLA#tla_win](../../../../includes/tlasharptla-win-md.md)] 列印對話方塊物件，以及使用程式碼從 [!INCLUDE[TLA#tla_wpf](../../../../includes/tlasharptla-wpf-md.md)] 應用程式內叫用 \(Invoke\) 對話方塊的指示。  
  
 [複製印表機](../../../../docs/framework/wpf/advanced/how-to-clone-a-printer.md)  
 如何使用現有列印佇列的相同屬性，以安裝第二個列印佇列的指示。  
  
 [診斷問題列印工作](../../../../docs/framework/wpf/advanced/how-to-diagnose-problematic-print-job.md)  
 使用列印佇列與列印工作的屬性，以診斷未進行列印之列印工作的指示。  
  
 [得知列印工作是否可在此時列印](../../../../docs/framework/wpf/advanced/how-to-discover-whether-a-print-job-can-be-printed-at-this-time-of-day.md)  
 使用列印佇列與列印工作的屬性，以程式設計的方式設定應在一天中的何時執行列印工作的指示。  
  
 [列舉列印佇列的子集](../../../../docs/framework/wpf/advanced/how-to-enumerate-a-subset-of-print-queues.md)  
 產生具有特定特性之印表機清單的指示。  
  
 [NOTINBUILD: How to: Extend the Print Schema and Create New Print System Classes](http://msdn.microsoft.com/zh-tw/a3600218-1ea5-478a-9853-6560464f2885)  
 廣泛討論 Managed 程式碼的程式設計人員應如何使用新增或非一般的功能，將列印結構描述擴充至專業的列印裝置或印表機。  
  
 [取得列印系統物件屬性但不使用反映](../../../../docs/framework/wpf/advanced/how-to-get-print-system-object-properties-without-reflection.md)  
 如何在執行階段探索列印系統物件的屬性及其類型的指示。  
  
 [以程式設計方式列印 XPS 檔](../../../../docs/framework/wpf/advanced/how-to-programmatically-print-xps-files.md)  
 不需使用[!INCLUDE[TLA#tla_ui](../../../../includes/tlasharptla-ui-md.md)] 即可快速列印 [!INCLUDE[TLA#tla_xps](../../../../includes/tlasharptla-xps-md.md)] 檔案的指示。  
  
 [從遠端調查印表機的狀態](../../../../docs/framework/wpf/advanced/how-to-remotely-survey-the-status-of-printers.md)  
 建立公用程式以檢查印表機有無夾紙或其他問題的指示。  
  
 [驗證和合併 PrintTickets](../../../../docs/framework/wpf/advanced/how-to-validate-and-merge-printtickets.md)  
 檢查列印票證是否有效以及是否要求任何印表機不支援項目的指示。
