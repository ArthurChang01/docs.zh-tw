# <a name="setfakeactivewindow-function-wpf-unmanaged-api-reference"></a>SetFakeActiveWindow 函式（WPF 非受控 API 參考）
這個 API 支援 Windows Presentation Foundation （WPF）基礎結構，但不適合直接從程式碼使用。  
  
 由適用于 Windows 管理的 Windows Presentation Foundation （WPF）基礎結構使用。  
  
## <a name="syntax"></a>語法  
  
```cpp  
void __stdcall SetFakeActiveWindow(  
   HWND hwnd  
)  
```  
  
## <a name="parameters"></a>參數  
 hwnd  
 視窗控制碼。
