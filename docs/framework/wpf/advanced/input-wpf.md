# <a name="input-wpf"></a>輸入 (WPF)
Windows Presentation Foundation (WPF) 包含支援數種類型的輸入。 此輸入包含文字、 觸控、 滑鼠、 命令、 焦點、 觸控、 拖放，以及數位筆跡。 本章節描述在 WPF 中輸入的相關主題。
