# 最佳化 WPF 應用程式效能
本節是給尋求改善應用程式效能之方法的 [!INCLUDE[TLA#tla_winclient](../../../../includes/tlasharptla-winclient-md.md)] 應用程式開發人員做為參考。  如果您是剛剛接觸 [!INCLUDE[TLA#tla_avalonwinfx](../../../../includes/tlasharptla-avalonwinfx-md.md)] 和 [!INCLUDE[TLA2#tla_winclient](../../../../includes/tla2sharptla-winclient-md.md)] 的開發人員，應該先熟悉這兩個平台。  本節需要這兩個平台的操作知識，並且是針對已能夠執行應用程式的程式設計人員所撰寫。  
  
> [!NOTE]
>  本節中提供的效能資料是根據在 2.8 GHz PC \(配備 512 RAM 和 ATI Radeon 9700 圖形卡\) 上執行的 [!INCLUDE[TLA2#tla_winclient](../../../../includes/tla2sharptla-winclient-md.md)] 應用程式。
