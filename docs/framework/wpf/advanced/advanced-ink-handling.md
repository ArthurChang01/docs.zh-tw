# 筆墨進階處理
[!INCLUDE[TLA2#tla_winclient](../../../../includes/tla2sharptla-winclient-md.md)] 隨附於 <xref:System.Windows.Controls.InkCanvas> 中，您可以將這個項目放在應用程式中，可以立即開始收集和顯示筆墨。  但是，如果 <xref:System.Windows.Controls.InkCanvas> 控制項的控制層級不足，您可以使用 <xref:System.Windows.Input.StylusPlugIns> 來自訂用於收集和呈現筆墨的類別，將控制項維持在較高的層級。  
  
 <xref:System.Windows.Input.StylusPlugIns> 類別會提供一種機制，可以對 <xref:System.Windows.Input.Stylus> 輸入實作低層級的控制項，並動態呈現筆墨。  <xref:System.Windows.Input.StylusPlugIns.StylusPlugIn> 類別提供一種機制，讓您實作自訂行為並套用至手寫筆裝置所傳入的資料流以達到最佳效能。  <xref:System.Windows.Input.StylusPlugIns.DynamicRenderer> 是一種特殊的 <xref:System.Windows.Input.StylusPlugIns.StylusPlugIn>，可讓您自訂即時動態呈現筆墨資料，也就是說 <xref:System.Windows.Input.StylusPlugIns.DynamicRenderer> 會在 <xref:System.Windows.Input.StylusPoint> 資料產生時立即繪製數位筆墨，所以看起來就像「順著」手寫筆裝置出現一樣。
