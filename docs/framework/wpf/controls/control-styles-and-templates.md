# 控制項的樣式和範本
Windows Presentation Foundation \(WPF\) 中的控制項具有 <xref:System.Windows.Controls.ControlTemplate>，其中包含這個控制項的[視覺化樹狀目錄](GTMT)。  您可以透過修改控制項的 <xref:System.Windows.Controls.ControlTemplate>，變更該控制項的結構和外觀。  您無法只取代控制項的視覺化樹狀目錄的某一部分，若要變更控制項的視覺化樹狀目錄，您必須將控制項的 <xref:System.Windows.Controls.Control.Template%2A> 屬性設定為完整的新 <xref:System.Windows.Controls.ControlTemplate>。  
  
 桌面主題決定所使用的資源字典。  桌面主題決定所使用的資源字典。  若要取得桌面主題的資源字典，請參閱[預設 WPF 主題](http://go.microsoft.com/fwlink/?LinkID=158252) \(英文\)。  
  
 下表說明資源字典檔案名稱及對應的桌面主題。  
  
|主題檔案|桌面主題|  
|----------|----------|  
|Classic.xaml|Windows XP 作業系統上的傳統 Windows 外觀 \(源自 Windows 95、Windows 98 和 Windows 2000\)。|  
|Luna.NormalColor.xaml|Windows XP 上的預設藍色佈景主題。|  
|Luna.Homestead.xaml|Windows XP 上的橄欖色佈景主題。|  
|Luna.Metallic.xaml|Windows XP 上的銀色佈景主題。|  
|Royale.NormalColor.xaml|Windows XP Media Center Edition 作業系統上的預設佈景主題。|  
|Aero.NormalColor.xaml|Windows Vista 作業系統上的預設佈景主題。|
