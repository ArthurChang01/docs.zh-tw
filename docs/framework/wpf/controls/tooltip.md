# ToolTip
工具提示是一個小型快顯視窗，會在使用者將滑鼠指標暫停在某個項目 \(如 <xref:System.Windows.Controls.Button>\) 上時出現。  
  
 下圖顯示滑鼠指標指向 \[**Close**\] <xref:System.Windows.Controls.Button>，這會顯示其識別 <xref:System.Windows.Controls.ToolTip>。
