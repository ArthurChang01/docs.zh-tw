# 功能表
<xref:System.Windows.Controls.Menu> 是一種控制項，可將命令或事件處理常式 \(Event Handler\) 的關聯項目以階層方式組織。  每個 <xref:System.Windows.Controls.Menu> 都可以包含多個 <xref:System.Windows.Controls.MenuItem> 控制項。  每個 <xref:System.Windows.Controls.MenuItem> 都可叫用 \(Invoke\) 命令或叫用 `Click` 事件處理常式。  <xref:System.Windows.Controls.MenuItem> 也可以有多個 <xref:System.Windows.Controls.MenuItem> 項目做為子系，以形成子功能表。  
  
 下圖顯示功能表控制項的三種不同狀態。  預設狀態是沒有任何裝置 \(例如滑鼠指標\) 停留在 <xref:System.Windows.Controls.Menu> 上方。  焦點狀態發生於滑鼠指標停留在 <xref:System.Windows.Controls.Menu> 上時，而按下狀態則發生於在 <xref:System.Windows.Controls.Menu> 上按一下滑鼠按鈕時。
