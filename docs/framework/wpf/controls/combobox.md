# ComboBox
<xref:System.Windows.Controls.ComboBox> 控制項提供使用者一份選項清單。  此清單會隨著控制項展開和摺疊而顯示和隱藏。  清單的預設狀態為摺疊，只會顯示一個選項。  使用者可按一下按鈕來檢視完整的選項清單。
