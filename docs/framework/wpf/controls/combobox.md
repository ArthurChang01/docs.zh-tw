# <a name="combobox"></a>ComboBox
<xref:System.Windows.Controls.ComboBox>控制向使用者呈現選項的清單。 清單會顯示和隱藏控制項展開和摺疊。 在其預設狀態下，清單會摺疊，顯示只有一個選擇。 使用者按一下按鈕以查看完整的選項清單。
