# RepeatButton
<xref:System.Windows.Controls.Primitives.RepeatButton> 與 <xref:System.Windows.Controls.Button> 類似。  不過，<xref:System.Windows.Controls.Primitives.RepeatButton> 項目可讓您控制 <xref:System.Windows.Controls.Primitives.ButtonBase.Click> 事件發生的時機和方式。  
  
 下圖顯示重複按鈕控制項之 Default、PointerFocused 和 Pressed 三種狀態的範例。  第一個按鈕顯示 <xref:System.Windows.Controls.Primitives.RepeatButton> 的預設狀態。  第二個按鈕顯示當滑鼠指標停留於按鈕上使其取得焦點時，按鈕外觀的變化。  最後一個按鈕顯示當使用者在控制項上按下滑鼠按鍵時，<xref:System.Windows.Controls.Primitives.RepeatButton> 的外觀。
