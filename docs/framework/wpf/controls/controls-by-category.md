# 按分類區隔控制項

 按鈕是最基本的一種使用者介面控制項。  應用程式通常會在使用者按下按鈕時，執行 <xref:System.Windows.Controls.Primitives.ButtonBase.Click> 事件中的某項工作。  
  
-   <xref:System.Windows.Controls.Button>  
  
-   <xref:System.Windows.Controls.Primitives.RepeatButton>
