# FlowDocumentPageViewer
<xref:System.Windows.Controls.FlowDocumentPageViewer> 控制項可用於以每頁為基礎來檢視 <xref:System.Windows.Documents.FlowDocument> 內容。  這與 <xref:System.Windows.Controls.FlowDocumentScrollViewer> 相反，它會在捲動檢視器中呈現 <xref:System.Windows.Documents.FlowDocument> 內容。
