# 樣式和範本
Windows Presentation Foundation \(WPF\) 設計樣式和樣板化是一組功能套件 \(樣式、樣板、觸發程序和腳本\)，可以讓應用程式、文件或使用者介面 \(UI\) 的設計人員創造視覺效果出色的應用程式，並標準化其成品的特定外觀。
