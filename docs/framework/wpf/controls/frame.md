# Frame
<xref:System.Windows.Controls.Frame> 控制項支援在內容中進行內容巡覽。  <xref:System.Windows.Controls.Frame> 可以藉由 <xref:System.Windows.Window>、<xref:System.Windows.Navigation.NavigationWindow>、<xref:System.Windows.Controls.Page>、<xref:System.Windows.Controls.UserControl>、<xref:System.Windows.Documents.FlowDocument> 等根項目裝載，也可以裝載於根項目下的內容樹狀目錄中做為資料島。
