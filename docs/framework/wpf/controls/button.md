# 按鈕
<xref:System.Windows.Controls.Button> 控制項會回應滑鼠、鍵盤、手寫筆或其他輸入裝置的使用者輸入，並引發 <xref:System.Windows.Controls.Primitives.ButtonBase.Click> 事件。  <xref:System.Windows.Controls.Button> 是基本[!INCLUDE[TLA#tla_ui](../../../../includes/tlasharptla-ui-md.md)] 元件，可以包含簡單的內容，例如文字，也可以包含複雜的內容，例如影像和 <xref:System.Windows.Controls.Panel> 控制項。
