# 開始使用 WCF Data Services
本節的主題將透過說明基礎技術，幫助您快速了解 [!INCLUDE[ssODataFull](../../../../includes/ssodatafull-md.md)] 及如何使用 [!INCLUDE[ssAstoria](../../../../includes/ssastoria-md.md)] 來公開及取用 [!INCLUDE[ssODataShort](../../../../includes/ssodatashort-md.md)] 摘要。  本節包括概念性內容及[快速入門](../../../../docs/framework/data/wcf/quickstart-wcf-data-services.md)教學課程。  
  
## 本章節內容  
 下列主題將描述如何使用 [!INCLUDE[ssAstoria](../../../../includes/ssastoria-md.md)] 來建立資料服務。  
  
 [將資料公開為服務](../../../../docs/framework/data/wcf/exposing-your-data-as-a-service-wcf-data-services.md)  
 描述使用 [!INCLUDE[ssAstoria](../../../../includes/ssastoria-md.md)] 建立資料服務所需的步驟。  
  
 [存取資料服務資源](../../../../docs/framework/data/wcf/accessing-data-service-resources-wcf-data-services.md)  
 描述如何使用 [!INCLUDE[ssODataShort](../../../../includes/ssodatashort-md.md)] 摘要。  
  
 [使用用戶端應用程式中的資料服務](../../../../docs/framework/data/wcf/using-a-data-service-in-a-client-application-wcf-data-services.md)  
 描述如何在 .NET Framework 用戶端應用程式中使用 [!INCLUDE[ssODataShort](../../../../includes/ssodatashort-md.md)] 摘要。  
  
 [快速入門](../../../../docs/framework/data/wcf/quickstart-wcf-data-services.md)  
 示範如何建立及存取 [!INCLUDE[ssODataShort](../../../../includes/ssodatashort-md.md)] 型簡單服務，這個服務會公開以 Northwind 範例資料庫為基礎的摘要。  
  
 [應用程式案例](../../../../docs/framework/data/wcf/application-scenarios-wcf-data-services.md)  
 強調 [!INCLUDE[ssAstoria](../../../../includes/ssastoria-md.md)] 所支援的一組核心 [!INCLUDE[ssODataShort](../../../../includes/ssodatashort-md.md)] 案例。  
  
 [資源](../../../../docs/framework/data/wcf/wcf-data-services-resources.md)  
 提供 [!INCLUDE[ssAstoria](../../../../includes/ssastoria-md.md)] 和 [!INCLUDE[ssODataShort](../../../../includes/ssodatashort-md.md)] 資源的連結。
