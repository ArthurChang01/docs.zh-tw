# <a name="getting-started-with-wcf-data-services"></a>WCF 資料服務使用者入門
本節中的主題可協助您快速瞭解[!INCLUDE[ssODataFull](../../../../includes/ssodatafull-md.md)] ，並說明基礎[!INCLUDE[ssAstoria](../../../../includes/ssastoria-md.md)]技術，如何使用[!INCLUDE[ssODataShort](../../../../includes/ssodatashort-md.md)]來公開和取用摘要。 本節包含概念內容和[快速入門](quickstart-wcf-data-services.md)教學課程。  
  
## <a name="in-this-section"></a>本節內容  
 下列主題將描述如何使用 [!INCLUDE[ssAstoria](../../../../includes/ssastoria-md.md)] 來建立資料服務。  
  
 [將資料當作服務公開](exposing-your-data-as-a-service-wcf-data-services.md)  
 描述使用 [!INCLUDE[ssAstoria](../../../../includes/ssastoria-md.md)] 建立資料服務所需的步驟。  
  
 [存取資料服務資源](accessing-data-service-resources-wcf-data-services.md)  
 說明如何使用[!INCLUDE[ssODataShort](../../../../includes/ssodatashort-md.md)]摘要。  
  
 [在用戶端應用程式中使用資料服務](using-a-data-service-in-a-client-application-wcf-data-services.md)  
 描述如何在 .NET Framework 用戶端[!INCLUDE[ssODataShort](../../../../includes/ssodatashort-md.md)]應用程式中使用摘要。  
  
 [快速入門](quickstart-wcf-data-services.md)  
 示範如何建立和存取以 Northwind 範例[!INCLUDE[ssODataShort](../../../../includes/ssodatashort-md.md)]資料庫為基礎公開摘要的簡單服務。  
  
 [應用程式案例](application-scenarios-wcf-data-services.md)  
 強調 [!INCLUDE[ssODataShort](../../../../includes/ssodatashort-md.md)] 所支援的一組核心 [!INCLUDE[ssAstoria](../../../../includes/ssastoria-md.md)] 案例。  
  
 [資源](wcf-data-services-resources.md)
