# <a name="-equals-entity-sql"></a>= (等號) (Entity SQL)
比較兩個運算式是否相等。  
  
## <a name="syntax"></a>語法  
  
```  
expression = expression  
or   
expression == expression  
```  
  
## <a name="arguments"></a>引數  
 `expression`  
 任何有效的運算式。 兩個運算式都必須有可隱含轉換的資料型別。  
  
## <a name="result-types"></a>結果型別  
 如果左運算式等於右運算式則為`true` ；否則為 `false`。  
  
## <a name="remarks"></a>備註  
 == 運算子就相當於 =。
