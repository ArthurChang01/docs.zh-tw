# <a name="-less-than-entity-sql"></a>\< (小於) (Entity SQL)
比較兩個運算式來判斷左運算式的值是否小於右運算式。  
  
## <a name="syntax"></a>語法  
  
```  
expression < expression  
```  
  
## <a name="arguments"></a>引數  
 `expression`  
 任何有效的運算式。 兩個運算式都必須有可隱含轉換的資料型別。  
  
## <a name="result-types"></a>結果型別  
 `true` 左的運算式是否小於右邊的運算式; 的值否則， `false`。
