# <a name="-greater-than-entity-sql"></a>> （大於）（Entity SQL）
比較兩個運算式來判斷左運算式的值是否大於右運算式。  
  
## <a name="syntax"></a>語法  
  
```sql  
expression > expression  
```  
  
## <a name="arguments"></a>引數  
 `expression`  
 任何有效的運算式。 兩個運算式都必須有可隱含轉換的資料型別。  
  
## <a name="result-types"></a>結果型別  
 如果左運算式的值大於右運算式則為`true` ；否則為 `false`。
