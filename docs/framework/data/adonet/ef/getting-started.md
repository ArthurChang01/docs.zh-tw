# <a name="getting-started-entity-framework"></a>使用者入門 (Entity Framework)
ADO.NET [!INCLUDE[adonet_ef](../../../../../includes/adonet-ef-md.md)]支援以資料為中心的應用程式和服務, 並提供一個平臺, 可針對從邏輯關聯式層級將抽象層級提升為概念層級的資料進行程式設計。 [!INCLUDE[adonet_ef](../../../../../includes/adonet-ef-md.md)] 讓開發人員能在更高的抽象層級處理資料，利用這種方式支援不需依賴任何特定資料儲存引擎或關聯式結構描述的程式碼。 如需詳細資訊, 請參閱[Entity Framework 總覽](overview.md)。  
  
 若要快速開始使用最新版本的 Entity Framework, 請參閱[入門](https://go.microsoft.com/fwlink/?LinkId=235280)
