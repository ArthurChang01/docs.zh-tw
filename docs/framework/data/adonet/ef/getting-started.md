# <a name="getting-started-entity-framework"></a>使用者入門 (Entity Framework)
ADO.NET Entity Framework 支援以資料為中心的應用程式和服務，並針對從邏輯關聯式層級將抽象層級提升為概念層級的資料，提供設計平臺。 藉由讓開發人員能夠在更高的抽象層級處理資料，Entity Framework 支援與任何特定資料儲存引擎或關聯式結構描述無關的程式碼。 如需詳細資訊，請參閱[Entity Framework 總覽](overview.md)。  
  
 若要快速開始使用最新版本的 Entity Framework，請參閱[開始使用 Entity Framework 6](/ef/ef6/get-started)。
