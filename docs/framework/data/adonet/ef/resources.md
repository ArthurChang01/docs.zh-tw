# <a name="entity-framework-resources"></a>Entity Framework 資源
下列外部資源提供建立 Entity Framework 應用程式的資訊和支援。  
  
 [資料開發人員中心](http://go.microsoft.com/fwlink/?LinkId=213876)  
 使用 Microsoft 技術進行資料開發的中心位置。  
  
 [ADO.NET 小組部落格](http://go.microsoft.com/fwlink/?LinkId=91905)  
 包含 ADO.NET 特性與功能最新消息與討論的部落格。  
  
 [Entity Framework 設計部落格](http://go.microsoft.com/fwlink/?LinkId=186888)

 [Entity Framework 常見問題集](http://go.microsoft.com/fwlink/?LinkID=213877)
