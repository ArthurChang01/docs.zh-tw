# 使用物件
[!INCLUDE[adonet_ef](../../../../../includes/adonet-ef-md.md)] 可讓您查詢、插入、更新及刪除資料 \(這些資料以實體類型執行個體的具型別 Common Language Runtime \(CLR\) 物件來表示\)。這些實體類型代表在概念模型中定義的實體。  [!INCLUDE[adonet_ef](../../../../../includes/adonet-ef-md.md)] 會將概念模型中定義的實體和關聯性對應至資料來源。  [!INCLUDE[adonet_ef](../../../../../includes/adonet-ef-md.md)] 提供了可進行下列作業的機能：將資料來源傳回的資料具體化為物件、追蹤對物件所做的變更、處理並行、將物件變更傳播回資料來源，以及將物件繫結至控制項。  
  
 如需在最新版 Entity Framework 中使用物件的詳細資訊，請參閱[使用物件](http://go.microsoft.com/fwlink/?LinkId=235289)。
