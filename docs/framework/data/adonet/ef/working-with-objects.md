# <a name="working-with-objects"></a>處理物件
[!INCLUDE[adonet_ef](../../../../../includes/adonet-ef-md.md)] 可讓您查詢、插入、更新及刪除資料 (這些資料表示為實體型別執行個體的具型別 Common Language Runtime (CLR) 物件)。 這些實體類型代表在概念模型中定義的實體。 [!INCLUDE[adonet_ef](../../../../../includes/adonet-ef-md.md)] 會將概念模型中定義的實體和關聯性對應至資料來源。 [!INCLUDE[adonet_ef](../../../../../includes/adonet-ef-md.md)]提供執行下列功能： 具體化為物件資料來源傳回的資料; 追蹤對物件所做的變更; 處理並行; 物件將變更傳播回資料來源，並將物件繫結至控制項。  
  
 如需有關使用最新版的 Entity Framework，請參閱中的物件[使用物件](http://go.microsoft.com/fwlink/?LinkId=235289)。
