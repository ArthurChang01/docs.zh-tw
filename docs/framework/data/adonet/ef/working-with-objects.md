# <a name="working-with-objects"></a>使用物件
此 Entity Framework 可讓您查詢、插入、更新和刪除資料，這是以實體類型實例的具類型 common language runtime （CLR）物件表示。 這些實體類型代表在概念模型中定義的實體。 Entity Framework 會將概念模型中定義的實體和關聯性對應至資料來源。 Entity Framework 提供可執行下列工作的功能：具體化從資料來源傳回的資料做為物件;追蹤對物件所做的變更;處理並行;將物件變更傳播回資料來源;和會將物件系結至控制項。  
  
 如需在最新 Entity Framework 版本中使用物件的詳細資訊，請參閱使用[物件](https://docs.microsoft.com/previous-versions/gg696163(v=vs.103))。
