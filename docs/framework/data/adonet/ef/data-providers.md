本節提供支援 Entity Framework 之資料提供者的相關資訊。  
  
## <a name="in-this-section"></a>本節內容  
 [Entity Framework 的 EntityClient 提供者](entityclient-provider-for-the-entity-framework.md)  
 描述 EntityClient 資料提供者 (Data Provider)。 這個提供者會將針對資料模型的查詢轉換成標準命令樹。 然後，此命令可供適用於 Entity Framework 的 .NET Framework 資料提供者取用。

 描述支援與 SQL Server 資料庫搭配使用之 Entity Framework 的 .NET Framework Data Provider。
