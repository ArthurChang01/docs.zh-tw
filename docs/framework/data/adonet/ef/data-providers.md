# Entity Framework 資料提供者
本節提供支援 [!INCLUDE[adonet_ef](../../../../../includes/adonet-ef-md.md)] 之資料提供者的資訊。  
  
## 在本節中  
 [Entity Framework 的 EntityClient 提供者](../../../../../docs/framework/data/adonet/ef/entityclient-provider-for-the-entity-framework.md)  
 描述 EntityClient 資料提供者 \(Data Provider\)。  這個提供者會將針對資料模型的查詢轉換成標準命令樹。  然後，此命令可供適用於 [!INCLUDE[adonet_ef](../../../../../includes/adonet-ef-md.md)] 的 [!INCLUDE[dnprdnshort](../../../../../includes/dnprdnshort-md.md)] 資料提供者取用。  
  
 [Entity Framework 函式的 SqlClient](../../../../../docs/framework/data/adonet/ef/sqlclient-for-the-entity-framework.md)  
 描述支援 [!INCLUDE[adonet_ef](../../../../../includes/adonet-ef-md.md)] 搭配 [!INCLUDE[ssNoVersion](../../../../../includes/ssnoversion-md.md)] 資料庫使用的 [!INCLUDE[dnprdnshort](../../../../../includes/dnprdnshort-md.md)] 資料提供者。
