<span data-ttu-id="24bda-103">除了查詢單一資料表以外，您也可以在 [!INCLUDE[linq_dataset](../../../../includes/linq-dataset-md.md)] 中執行跨資料表查詢。</span><span class="sxs-lookup"><span data-stu-id="24bda-103">In addition to data-ttu-id="24bda-104">這是使用*聯結*。</span><span class="sxs-lookup"><span data-stu-id="24bda-104">This is done by using a *join*.</span></span> <span data-ttu-id="24bda-105">聯結是指某個資料來源中的物件與另一個資料來源中共用相同屬性之物件的關聯，例如產品或連絡人識別碼。</span><span class="sxs-lookup"><span data-stu-id="24bda-105">A join is the association of objects in one data source with class="sxs-lookup"><span data-stu-id="24bda-106">In object-oriented programming, relationships between objects are relatively easy to navigate because each object has a member that references another object.</span></span> <span data-ttu-id="24bda-107">不過，在外部資料庫資料表中，瀏覽關聯性就沒有這麼直接。</span><span class="sxs-lookup"><span data-stu-id="24bda-107">In external database tables, however, navigating order.</span></span>  
  
 <span data-ttu-id="24bda-111">[!INCLUDE[vbteclinqext](../../../../includes/vbteclinqext-md.md)] Framework 提供兩個聯結運算子，<xref:System.Linq.Enumerable.Join%2A>和<xref:System.Linq.Enumerable.GroupJoin%2A>。</span><span class="sxs-lookup"><span data-stu-id="24bda-111">The [!INCLUDE[vbteclinqext](../../../../includes/vbteclinqext-md.md)] framework provides two join operators, <xref:System.Linq.Enumerable.Join%2A> and <xref:System.Linq.Enumerable.GroupJoin%2A>.</span></span> <span data-ttu-id="24bda-112">這些運算子會執行*聯結 （equi-join）*： 也就是聯結來比對兩個資料來源只當它們的索引鍵是否相等。</span><span class="sxs-lookup"><span data-stu-id="24bda-112">These operators perform *equi-joins*: that is, joins that match two data sources only when their keys are equal.</span></span> <span data-ttu-id="24bda-113">(相較之下，[!INCLUDE[tsql](../../../../includes/tsql-md.md)] 支援 `equals` 以外的聯結運算子，例如 `less than` 運算子)。</span><span class="sxs-lookup"><span data-stu-id="24bda-113">(By contrast, [!INCLUDE[tsql](../../../../includes/tsql-md.md)] supports join operators other than `equals`, such as the `less than` operator.)</span></span>  
  
 <span data-ttu-id="24bda-114">在關聯式資料庫詞彙中，<xref:System.Linq.Enumerable.Join%2A> 會實作內部聯結 (Inner Join)。</span><span class="sxs-lookup"><span data-stu-id="24bda-114">In relational database terms, <xref:System.Linq.Enumerable.Join%2A> implements an inner join.</span></span> <span data-ttu-id="24bda-115">內部聯結是一種聯結類型，而這種聯結只會傳回在相對資料集內部具有相符項目的物件。</span><span class="sxs-lookup"><span data-stu-id="24bda-115">An inner join is a type of join in which only those objects that have a match in the opposite data set are returned.</span></span>  
  
 <span data-ttu-id="24bda-116"><xref:System.Linq.Enumerable.GroupJoin%2A>運算子具有關聯式資料庫堆規定中沒有直接的對等; 它們必須實作內部聯結和左方外部聯結的超集。</span><span class="sxs-lookup"><span data-stu-id="24bda-116">The <xref:System.Linq.Enumerable.GroupJoin%2A> operators have no direct equivalent in relational database terms; they implement a superset of inner joins and left outer joins.</span></span> <span data-ttu-id="24bda-117">左方外部聯結是傳回的第一個 （左） 集合，每個元素的聯結，即使它在第二個集合中有任何相互關聯的項目。</span><span class="sxs-lookup"><span data-stu-id="24bda-117">A left outer join is a join that returns each element of the first (left) collection, even if it has no correlated elements in the second collection.</span></span>
