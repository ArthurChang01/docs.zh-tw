# 以方法為基礎的查詢語法範例：彙總運算子 (LINQ to DataSet)

 此主題中的範例將使用 AdventureWorks 範例資料庫中的 Contact、Address、Product、SalesOrderHeader 和 SalesOrderDetail 資料表。
