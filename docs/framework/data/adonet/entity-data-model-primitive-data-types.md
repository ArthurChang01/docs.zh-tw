<span data-ttu-id="061ad-103">實體資料模型 (EDM) 支援一組抽象的基本資料類型 (例如字串、布林值、Int32 等等), 可用來定義概念模型中的[屬性](../../../../docs/framework/data/adonet/property.md)。</span><span class="sxs-lookup"><span data-stu-id="061ad-103">The Entity Data Model (EDM) supports a set of abstract primitive data types (such as String, Boolean, Int32, and so on) that are used to define [properties](../../../../docs/framework/data/adonet/property.md) in a conceptual model.</span></span> <span data-ttu-id="061ad-104">這些基本資料型別是實際基本資料型別的 Proxy，無論在 SQL Server 資料庫或 Common Language Runtime (CLR) 等儲存或裝載環境中皆可支援。</span><span class="sxs-lookup"><span data-stu-id="061ad-104">These primitive data types are proxies for actual primitive data types that are supported in the storage or hosting environment, such as a SQL Server database or the common language runtime (CLR).</span></span> <span data-ttu-id="061ad-105">EDM 不會定義基本資料型別作業或慣例的語意，這些語意是由儲存或裝載環境定義的。</span><span class="sxs-lookup"><span data-stu-id="061ad-105">The EDM does not define the semantics of operations or conversions over primitive data types; these semantics are defined by the storage or hosting environment.</span></span> <span data-ttu-id="061ad-106">一般來說，EDM 中的基本資料型別對應於儲存或裝載環境中相對應的基本資料型別。</span><span class="sxs-lookup"><span data-stu-id="061ad-106">Typically, primitive data types in the EDM are mapped to corresponding primitive data types in the storage or hosting environment.</span></span> <span data-ttu-id="061ad-107">如需 Entity Framework 如何將 EDM 中的基本型別對應至 SQL Server 資料類型的詳細資訊, 請參閱[Entity FrameworkTypes 的 SqlClient](../../../../docs/framework/data/adonet/ef/sqlclient-for-ef-types.md)。</span><span class="sxs-lookup"><span data-stu-id="061ad-107">For information about how the Entity Framework maps primitive types in the EDM to SQL Server data types, see [SqlClient for Entity FrameworkTypes](../../../../docs/framework/data/adonet/ef/sqlclient-for-ef-types.md).</span></span>  
  
> [!NOTE]
> <span data-ttu-id="061ad-108">EDM 不支援基本資料型別集合。</span><span class="sxs-lookup"><span data-stu-id="061ad-108">The EDM does not support collections of primitive data types.</span></span>  
  
 <span data-ttu-id="061ad-109">如需 EDM 中結構化資料類型的詳細資訊, 請參閱[實體類型](../../../../docs/framework/data/adonet/entity-type.md)和[複雜類型](../../../../docs/framework/data/adonet/complex-type.md)。</span><span class="sxs-lookup"><span data-stu-id="061ad-109">For information about structured data types in the EDM, see [entity type](../../../../docs/framework/data/adonet/entity-type.md) and [complex type](../../../../docs/framework/data/adonet/complex-type.md).</span></span>
