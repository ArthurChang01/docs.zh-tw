# <a name="download-sample-databases-linq-to-dataset"></a>下載範例資料庫(LINQ 到資料集)

LINQ 到 DataSet 文件中的範例和演練使用 AdventureWorks 範例資料庫。 您可以免費下載此範例資料庫。 流覽到[AdventureWorks 範例資料庫](https://github.com/Microsoft/sql-server-samples/releases/tag/adventureworks)以下載資料庫。 然後,按照該頁上的說明進行附加或還原資料庫。
  
## <a name="sql-server-express"></a>SQL Server Express

LINQ 到 DataSet 部分中的範例和演練使用 SQL Server 2005 作為資料存儲,但您可以修改它們以改用[SQL Server Express。](https://go.microsoft.com/fwlink/?linkid=866658) SQL Server Express 是免費的,您可以使用應用程式重新分發它。
