 <span data-ttu-id="afab2-111">每個 [!INCLUDE[dnprdnshort](../../../../includes/dnprdnshort-md.md)] 資料提供者都有它自己的 `Transaction` 物件，以執行本機交易。</span><span class="sxs-lookup"><span data-stu-id="afab2-111">Each of the [!INCLUDE[dnprdnshort](../../../../includes/dnprdnshort-md.md)] data providers has its own `Transaction` object for performing local transactions.</span></span> <span data-ttu-id="afab2-112">如果您需要在 SQL Server 資料庫中執行異動，請選擇 <xref:System.Data.SqlClient> 異動。</span><span class="sxs-lookup"><span data-stu-id="afab2-112">If you require a transaction to be performed in a SQL Server database, select a <xref:System.Data.SqlClient> transaction.</span></span> <span data-ttu-id="afab2-113">若為 Oracle 異動，請使用 <xref:System.Data.OracleClient> 提供者。</span><span class="sxs-lookup"><span data-stu-id="afab2-113">For an Oracle transaction, use the <xref:System.Data.OracleClient> provider.</span></span> <span data-ttu-id="afab2-114">此外，還有<xref:System.Data.Common.DbTransaction>適用於撰寫需要交易的提供者獨立程式碼的類別。</span><span class="sxs-lookup"><span data-stu-id="afab2-114">In addition, there is a <xref:System.Data.Common.DbTransaction> class that is available for writing provider-independent code that requires transactions.</span></span>
