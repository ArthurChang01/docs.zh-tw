# <a name="query-examples"></a>查詢範例
本節提供一般[!INCLUDE[vbtecdlinq](../../../../../../includes/vbtecdlinq-md.md)]查詢的C# Visual Basic 和範例。 使用 Visual Studio 的開發人員可以在範例一節中提供的範例解決方案中找到更多範例。 如需詳細資訊，請參閱[範例](samples.md)。  
  
> [!IMPORTANT]
> 在[!INCLUDE[vbtecdlinq](../../../../../../includes/vbtecdlinq-md.md)]檔的程式碼範例中，通常會使用 db。 假設*db*是*Northwind*類別的實例，它會繼承自<xref:System.Data.Linq.DataContext>。
