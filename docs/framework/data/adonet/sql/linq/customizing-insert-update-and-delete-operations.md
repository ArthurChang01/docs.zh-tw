# <a name="customizing-insert-update-and-delete-operations"></a>自訂插入、更新和刪除作業

> [!NOTE]
>  如果您使用 Visual Studio，您可以使用[!INCLUDE[vs_ordesigner_long](../../../../../../includes/vs-ordesigner-long-md.md)]來自訂插入、 更新和刪除動作。
