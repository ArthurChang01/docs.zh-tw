# Boolean 資料型別
除了不會轉譯最少運算行為以外，在 Common Language Runtime \(CLR\) 中布林運算子會如預期運作。  例如，Visual Basic `AndAlso` 運算子的行為方式如同 `And` 運算子。  而 C\# `&&` 運算子的行為方式則如同 `&` 運算子。  
  
 [!INCLUDE[vbtecdlinq](../../../../../../includes/vbtecdlinq-md.md)] 支援下列運算子。  
  
|Visual Basic|C\#|  
|------------------|---------|  
|[And Operator](../Topic/And%20Operator%20\(Visual%20Basic\).md)|[& 運算子](../Topic/&%20Operator%20\(C%23%20Reference\).md)|  
|[AndAlso Operator](../Topic/AndAlso%20Operator%20\(Visual%20Basic\).md)|[&& 運算子](../Topic/&&%20Operator%20\(C%23%20Reference\).md)|  
|[Or 運算子](../Topic/Or%20Operator%20\(Visual%20Basic\).md)|[&#124; 運算子](../Topic/%7C%20Operator%20\(C%23%20Reference\).md)|  
|[OrElse Operator](../Topic/OrElse%20Operator%20\(Visual%20Basic\).md)|[&#124;&#124; 運算子](../Topic/%7C%7C%20Operator%20\(C%23%20Reference\).md)|  
|[Xor Operator](../Topic/Xor%20Operator%20\(Visual%20Basic\).md)|[^ 運算子](../Topic/%5E%20Operator%20\(C%23%20Reference\).md)|  
|[Not Operator](../Topic/Not%20Operator%20\(Visual%20Basic\).md)|[\! 運算子](../Topic/!%20Operator%20\(C%23%20Reference\).md)|
