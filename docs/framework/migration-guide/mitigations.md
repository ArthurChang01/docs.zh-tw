# <a name="mitigate-new-behaviors-in-net-framework-46-and-later"></a>減輕 .NET 架構 4.6 及更高版本中的新行為

本節包含描述 .NET 框架 4.6 及更高版本中引入的行為的緩解措施的文章。

如果新行為導致應用出現問題或不受歡迎,請考慮減輕其緩解。 在某些情況下,您還可以在面向舊版本的 .NET Framework 但正在 .NET 框架 4.6 或更高版本上運行的應用上*啟用*新行為。
