# <a name="httpwebrequest_httpresponse-field"></a>HttpWeb請求。\_Http 回應欄位

`HttpWebRequest._HttpResponse`是<xref:System.Net.HttpWebResponse>包含 HTTP 要求的 HTTP 回應詳細資訊。 它可以直到`null`收到 HTTP 回應。

## <a name="syntax"></a>語法
  
```csharp  
internal HttpWebResponse _HttpResponse
```

> [!WARNING]
> 該`HttpWebRequest._HttpResponse`欄位是內部欄位，不應直接用於代碼。
>
> 在任何情況下，Microsoft 都不支援在生產應用程式中使用此欄位。

## <a name="requirements"></a>需求

**命名空間：**<xref:System.Net>

**裝配：** 系統（系統中）

**.NET 框架版本：** 自 2.0 起可用。
