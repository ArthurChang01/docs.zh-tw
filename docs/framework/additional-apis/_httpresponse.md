# <a name="httpwebrequesthttpresponse-field"></a>HttpWebRequest。\_HttpResponse 欄位

`HttpWebRequest._HttpResponse` 是<xref:System.Net.HttpWebResponse>包含來自 HTTP 要求的 HTTP 回應詳細資料。 它可以是`null`直到收到 HTTP 回應。

## <a name="syntax"></a>語法
  
```csharp  
internal HttpWebResponse _HttpResponse
```

> [!WARNING]
> `HttpWebRequest._HttpResponse`欄位是內部，而且不會直接在您的程式碼中使用它們。
> 
> Microsoft 不支援在生產環境應用程式中任何情況下使用此欄位。

## <a name="requirements"></a>需求

**命名空間︰** <xref:System.Net>

**組件：** （在 System.dll) 的系統

**.NET framework 版本：** 自 2.0 起可用。
