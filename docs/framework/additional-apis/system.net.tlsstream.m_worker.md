# <a name="tlsstreamm_worker-field"></a>TlsStream. m_Worker 欄位

表示 SSL 資料流程的狀態。

## <a name="syntax"></a>語法

```csharp
private SslState m_Worker;
```

## <a name="field-value"></a>域值

`System.Net.Security.SslState`  
SSL 資料流程的狀態。

## <a name="remarks"></a>備註

> [!WARNING]
> [`TlsStream.m_Worker`] 欄位是私用的，不適合直接在程式碼中使用。
>
> 在任何情況下，Microsoft 不支援在生產應用程式中使用此欄位。

## <a name="requirements"></a>需求

**命名空間︰** <xref:System.Net>

**元件：** 系統（在 System .dll 中）

**.NET Framework 版本：** 自2.0 開始提供。
