# <a name="sqlstreamchars-constructor"></a>SqlStreamChars 建構函式

初始化 `SqlStreamChars` 類別的新執行個體。 包含此函數的元件與 SQLAccess 具有 friend 關聯性。 其目的是要供 SQL Server 使用。 若為其他資料庫，請使用該資料庫所提供的裝載機制。

```csharp
protected SqlStreamChars ();
```

## <a name="remarks"></a>備註

> [!WARNING]
> @No__t-0 的函式會受到保護，而且不適合直接在程式碼中使用。
>
> 在任何情況下，Microsoft 不支援在實際執行應用程式中使用此函數。
