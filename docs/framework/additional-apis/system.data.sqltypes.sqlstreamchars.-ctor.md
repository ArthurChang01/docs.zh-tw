# <a name="sqlstreamchars-constructor"></a>SqlStreamChars 建構函式

初始化 `SqlStreamChars` 類別的新執行個體。 包含這個建構函式的組件有 SQLAccess.dll friend 關聯性。 它適用於 SQL server。 其他資料庫，請使用該資料庫所提供的主控機制。

```csharp
protected SqlStreamChars ();
```

## <a name="remarks"></a>備註

> [!WARNING]
> `SqlStreamChars`建構函式受到保護，而不是直接在您的程式碼中使用。
>
> Microsoft 不支援在生產環境應用程式中任何情況下使用此欄位。
