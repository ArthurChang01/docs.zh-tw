# <a name="datamemberfieldeditor-class"></a>DataMemberFieldEditor 類別

提供使用者介面來編輯資料系結物件的屬性，方法是列出要從中選取值來源的 `DataSource` 物件的所有屬性。  
  
## <a name="syntax"></a>語法
  
```csharp
internal class DataMemberFieldEditor : UITypeEditor
```

> [!WARNING]
> `DataMemberFieldEditor` 類別是內部的，而且不適合直接在程式碼中使用。
> 
> 在任何情況下，Microsoft 不支援在生產應用程式中使用此類別。
