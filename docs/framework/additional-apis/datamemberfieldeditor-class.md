# <a name="datamemberfieldeditor-class"></a>資料成員欄位編輯器類

通過列出要從中選擇值源的物件的所有屬性，`DataSource`提供用於編輯資料繫結物件屬性的使用者介面。  
  
## <a name="syntax"></a>語法
  
```csharp
internal class DataMemberFieldEditor : UITypeEditor
```

> [!WARNING]
> 該`DataMemberFieldEditor`類是內部的，不應直接在代碼中使用。
>
> 在任何情況下，Microsoft 都不支援在生產應用程式中使用此類。

**.NET 框架版本：** 自 2.0 起可用。
