# <a name="servicepointm_connectiongrouplist-field"></a>服務點.m\_連接組清單欄位

`ServicePoint.m_ConnectionGroupList`是連接<xref:System.Collections.Hashtable>組，每個組都持有<xref:System.Net.ServicePoint>URI 的連接。

## <a name="syntax"></a>語法
  
```csharp  
private Hashtable m_ConnectionGroupList
```

> [!WARNING]
> 該`ServicePoint.m_ConnectionGroupList`欄位是私有的，不應直接用於代碼。
>
> 在任何情況下，Microsoft 都不支援在生產應用程式中使用此欄位。

## <a name="requirements"></a>需求

**命名空間：**<xref:System.Net>

**裝配：** 系統（系統中）

**.NET 框架版本：** 自 2.0 起可用。
