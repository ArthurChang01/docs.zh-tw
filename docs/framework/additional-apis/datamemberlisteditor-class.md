# <a name="datamemberlisteditor-class"></a>DataMemberListEditor 類別

提供下拉式使用者介面，用於編輯資料系結物件（具有非 null `DataSource` 屬性的物件）的屬性，方法是列出 `DataSource` 物件的所有屬性，以從中選取值來源。  
  
## <a name="syntax"></a>語法
  
```csharp  
internal class DataMemberListEditor : UITypeEditor
```

> [!WARNING]
> `DataMemberListEditor` 類別是內部的，而且不適合直接在程式碼中使用。
> 
> 在任何情況下，Microsoft 不支援在生產應用程式中使用此類別。
