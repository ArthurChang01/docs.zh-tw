# <a name="datamemberlisteditor-class"></a>資料成員清單編輯器類

通過列出`DataSource`物件的所有屬性以從中選擇值源，提供用於編輯資料繫結物件（具有非 null`DataSource`屬性的物件）屬性的下拉使用者介面。  
  
## <a name="syntax"></a>語法
  
```csharp  
internal class DataMemberListEditor : UITypeEditor
```

> [!WARNING]
> 該`DataMemberListEditor`類是內部的，不應直接在代碼中使用。
>
> 在任何情況下，Microsoft 都不支援在生產應用程式中使用此類。

**.NET 框架版本：** 自 2.0 起可用。
