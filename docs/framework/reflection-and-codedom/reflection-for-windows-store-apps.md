 <span data-ttu-id="777e4-116">在適用于 Windows 8.x 存放區應用程式的 .NET 中，您可以使用會傳回 <xref:System.Collections.Generic.IEnumerable%601> 集合的反映屬性，而不是傳回陣列的方法。</span><span class="sxs-lookup"><span data-stu-id="777e4-116">In the .NET for Windows 8.x Store apps, you use the reflection properties that return <xref:System.Collections.Generic.IEnumerable%601> collections instead of methods that return arrays.</span></span> <span data-ttu-id="777e4-117">反映內容可以針對大型組件或類型實作這些集合的消極式周遊。</span><span class="sxs-lookup"><span data-stu-id="777e4-117">Reflection contexts can implement lazy traversal of these collections for large assemblies or types.</span></span>
