# <a name="ui-automation-specification-and-community-promise"></a>UI 自動化規格和社群承諾
UI 自動化可讓您以程式設計方式存取桌面上大部分的使用者介面（UI）專案，啟用輔助技術產品（例如螢幕閱讀程式），將 UI 的相關資訊提供給使用者，以及透過標準輸入以外的方式來操作 UI.
