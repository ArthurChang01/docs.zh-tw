# UI Automation Specification and Community Promise
UI 自動化讓您能夠用程式存取桌面上大部分的使用者介面 \(UI\) 項目，啟用輔助技術產品 \(例如螢幕助讀員\)，以便為使用者提供 UI 的相關資訊，以及使用標準輸入以外的方式來操作 UI。
