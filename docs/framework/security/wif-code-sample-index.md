# <a name="wif-code-sample-index"></a>WIF 程式碼範例索引
Windows Identity Foundation 4.5 的程式碼範例如下：  
  
-   [ClaimsAwareWebApp](http://go.microsoft.com/fwlink/?LinkID=248405) - 此範例將示範驗證外部化 (對 Visual Studio 11 的身分識別與存取工具中的本機測試安全性權杖服務) 在傳統 ASP.NET 應用程式上 (相對於網站) 的基本用法。  
  
-   [ClaimsAwareWebService](http://go.microsoft.com/fwlink/?LinkID=248406) - 此範例示範驗證外部化在傳統 WCF 服務上的基本用法。  
  
-   [ClaimsAwareMvcApplication](http://go.microsoft.com/fwlink/?LinkID=248407) - 此範例示範如何將 WIF 與 MVC 整合，包括非地毯式保護以及接受將表單驗證重新導向至登入控制器外的程式碼。  
  
-   [ClaimsAwareWebFarm](http://go.microsoft.com/fwlink/?LinkID=248408) - 此範例示範伺服陣列就緒的工作階段快取 (相對於 tokenreplycache)，讓您能夠透過參考使用工作階段，而非交換大型 Cookie。 還示範了一種更簡單的方法，用來保護伺服陣列中的 Cookie。  
  
-   [ClaimsAwareFormsAuthentication](http://go.microsoft.com/fwlink/?LinkID=248409) - 這個非常簡單的範例示範在 .NET 4.5 中，不論您驗證使用者的方式為何，都能取得主體中的宣告。  
  
-   [ClaimsBasedAuthorization](http://go.microsoft.com/fwlink/?LinkID=248410) - 此範例示範如何使用 CLaimsAuthorizationManager 類別和 ClaimsAuthorizationModule 來套用您自己的授權原則。  
  
-   [FederationMetadata](http://go.microsoft.com/fwlink/?LinkID=248411) – 此範例示範中繼資料文件的動態產生 (在自訂的 STS 上) 和動態使用 (在信賴憑證者應用程式上)。  
  
-   [CustomToken](http://go.microsoft.com/fwlink/?LinkID=248412) – 此範例示範如何建置自訂的簡單 Web 權杖 (SWT) 權杖類型。
