# <a name="custom-tokens"></a>自訂權杖
雖然 Windows Communication Foundation （WCF）原本就支援 x.509 憑證、安全性內容權杖、Kerberos 票證和使用者名稱權杖做為認證，但它有足夠的彈性，可讓您使用自己的自訂認證。  
  
## <a name="in-this-section"></a>本節內容  
 [如何：建立自訂權杖](how-to-create-a-custom-token.md)  
 說明如何使用 <xref:System.IdentityModel.Tokens.SecurityToken> 類別來建立自訂安全性權杖，以及如何將它與自訂安全性權杖提供者和驗證器整合。
