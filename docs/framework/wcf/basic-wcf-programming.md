# <a name="basic-wcf-programming"></a>基本 WCF 程式設計
本節提供建立 Windows Communication Foundation （WCF）應用程式的基本概念。  
  
## <a name="in-this-section"></a>本章節內容  
 [基本程式設計週期](basic-programming-lifecycle.md)  
 描述設計、建立和部署 WCF 服務和用戶端應用程式的生命週期。  
  
 [設計與實作服務](designing-and-implementing-services.md)  
 說明如何設計和實作服務合約、選擇訊息交換模式、指定錯誤合約，以及其他服務的基本部分。  
  
 [設定服務](configuring-services.md)  
 描述如何設定 WCF 服務以支援合約需求、自訂本機執行時間行為，以及指示發行服務的位址。  
  
 [裝載服務](hosting-services.md)  
 說明在應用程式中裝載服務的基本概念。  
  
 [建置用戶端](building-clients.md)  
 描述如何從服務取得中繼資料、將其轉換成 WCF 用戶端程式代碼、處理安全性問題，以及建立、設定和裝載 WCF 用戶端。  
  
 [擴充性簡介](introduction-to-extensibility.md)  
 說明如何擴充 WCF 來建立自訂解決方案。  
  
 [WCF 疑難排解快速入門](wcf-troubleshooting-quickstart.md)  
 說明某些最常發生的問題、解決這些問題的方法，以及如何尋找更多相關問題的資訊。  
  
 [WCF 與 ASP.NET Web API](wcf-and-aspnet-web-api.md)  
 討論這兩項技術、彼此的關聯性和使用時機。
