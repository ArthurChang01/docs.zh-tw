# <a name="windows-communication-foundation-bindings"></a>Windows Communication Foundation 繫結
系結會指定 Windows Communication Foundation （WCF）服務端點與其他端點通訊的方式。 繫結的最基本功能，就是必須指定要使用的傳輸 (例如，HTTP 或 TCP)。 此外，您也可以透過繫結程序來設定其他特性，例如安全性與異動支援。  
  
## <a name="in-this-section"></a>本章節內容  
 [WCF 繫結概觀](bindings-overview.md)  
 概述 WCF 系結的用途、系統提供哪些系結，以及您可以如何定義或修改這些系結。  
  
 [系統提供的繫結](system-provided-bindings.md)  
 WCF 包含的系結清單。 這些繫結涵蓋了大部分的安全性與訊息模式需求。  
  
 [使用繫結設定服務與用戶端](using-bindings-to-configure-services-and-clients.md)  
 WCF 系結包含用戶端必須用來連接到服務端點的重要資訊。  
  
 [設定服務的繫結](configuring-bindings-for-wcf-services.md)  
 組態可讓系統管理員和安裝程式自訂服務端點的繫結。
