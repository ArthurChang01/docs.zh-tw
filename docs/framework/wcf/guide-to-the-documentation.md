# 文件指南
此處提供有關 [!INCLUDE[indigo1](../../../includes/indigo1-md.md)] 文件的指引。連結文件是建議的起點，並且根據特定興趣和專業等級分群組。  
  
 若要安裝獨立版本的文件和說明檢視器，請下載 [Microsoft Windows SDK v 7.1](http://go.microsoft.com/fwlink/?LinkID=194146&clcid=0x409)。  
  
## Windows Communication Foundation 程式設計的新功能  
  
-   如果您在使用 [!INCLUDE[indigo2](../../../includes/indigo2-md.md)] 進行程式設計方面是新手，並且只是想要檢視能夠運作的範例應用程式，請參閱 [Windows Communication Foundation 範例](../../../docs/framework/wcf/samples/index.md)中列出的主題。  
  
-   如需教學課程說明建立 [!INCLUDE[indigo2](../../../includes/indigo2-md.md)] 服務與用戶端的基本步驟，請參閱[快速入門教學課程](../../../docs/framework/wcf/getting-started-tutorial.md)。  
  
-   如果您對 [!INCLUDE[indigo2](../../../includes/indigo2-md.md)] 之後的概念有興趣，請參閱[概觀說明](../../../docs/framework/wcf/conceptual-overview.md)章節中的主題。  
  
-   若要檢視用戶端\/伺服器安全性組態的圖形範例，請參閱[常見的安全性案例](../../../docs/framework/wcf/feature-details/common-security-scenarios.md)。  
  
## 程式設計深入說明  
  
-   如果您已經準備好開始開發應用程式，請參閱[基本 WCF 程式設計](../../../docs/framework/wcf/basic-wcf-programming.md)。  
  
-   如果您在尋找有關特定功能或 [!INCLUDE[indigo2](../../../includes/indigo2-md.md)] 功能的指引，請參閱 [WCF 功能詳細資料](../../../docs/framework/wcf/feature-details/index.md)中的主題。  
  
-   如果您想要延伸或自訂 [!INCLUDE[indigo2](../../../includes/indigo2-md.md)] 以滿足需求，請參閱[延伸 WCF](../../../docs/framework/wcf/extending/extending-wcf.md)。  
  
-   如需有關協助建立與偵錯 [!INCLUDE[indigo2](../../../includes/indigo2-md.md)] 應用程式的工具資訊，請參閱 [Windows Communication Foundation 工具](../../../docs/framework/wcf/tools.md)。  
  
-   使用 XML 檔案設定組態是設計 [!INCLUDE[indigo2](../../../includes/indigo2-md.md)] 服務與用戶端程式的主要方法。如需組態檔中所使用之 XML 項目的參考文件，請參閱 [WCF 組態結構描述](../../../docs/framework/configure-apps/file-schema/wcf/index.md)。

## 搭配其他技術使用 Windows Communication Foundation  
  
-   若要建立與 ASP.NET 用戶端通訊的服務，請參閱 [HOW TO：將 WCF 服務設為與 ASP.NET Web 服務用戶端交互操作](../../../docs/framework/wcf/feature-details/config-wcf-service-with-aspnet-web-service.md)。  
  
-   在[將 .NET 遠端處理應用程式移轉到 WCF](../../../docs/framework/wcf/feature-details/migrating-net-remoting-applications-to-wcf.md)中會解釋與 [!INCLUDE[dnprdnshort](../../../includes/dnprdnshort-md.md)] 遠端處理整合。  
  
-   若要整合現有 COM\+ 應用程式與 [!INCLUDE[indigo2](../../../includes/indigo2-md.md)] 服務或用戶端，請參閱[整合 COM\+ 應用程式概觀](../../../docs/framework/wcf/feature-details/integrating-with-com-plus-applications-overview.md)。  
  
-   若要整合現有 COM 應用程式與 [!INCLUDE[indigo2](../../../includes/indigo2-md.md)] 服務或用戶端，請參閱[整合 COM 應用程式](../../../docs/framework/wcf/feature-details/integrating-with-com-applications.md)。  
  
-   若要整合現有 MSMQ 應用程式與 [!INCLUDE[indigo2](../../../includes/indigo2-md.md)] 服務或用戶端，請參閱 [HOW TO：與 WCF 端點交換佇列訊息](../../../docs/framework/wcf/feature-details/how-to-exchange-queued-messages-with-wcf-endpoints.md)和 [HOW TO：與 WCF 端點和訊息佇列應用程式交換訊息](../../../docs/framework/wcf/feature-details/how-to-exchange-messages-with-wcf-endpoints-and-message-queuing-applications.md)。  
  
-   若要使用 Internet Information Services \(IIS\) 主控服務，請參閱[裝載服務](../../../docs/framework/wcf/hosting-services.md)。  
  
-   若要使用 [!INCLUDE[indigo2](../../../includes/indigo2-md.md)] 取用 Web Services Extensions \(WSE\) 3.0 服務，請參閱 [HOW TO：存取 WSE 3.0 服務](../../../docs/framework/wcf/feature-details/how-to-access-a-wse-3-0-service-with-a-wcf-client.md)。
