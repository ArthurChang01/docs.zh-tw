# <a name="com-integration"></a>COM+ 整合
此主題會列出 COM+ 整合產生的所有例外狀況。  
  
## <a name="exception-list"></a>例外狀況清單  
  
|資源程式碼|資源字串|  
|-------------------|---------------------|  
|CannotResolveTypeForParamInMessageDescription|無法解析所指定命名空間中指定之參數的型別。|  
|NoEqualSignFound|指定的關鍵字後面沒有等號。 請確定每個關鍵字後面都加上一個等號與一個值。|  
|UdtNotFoundInAssembly|找不到指定的使用者定義型別。 請確定已註冊並指定正確的型別與型別程式庫。|
