# <a name="core-communications-named-pipe-transport-channels"></a>核心通訊：具名管道傳輸通道
本主題會列出由具名管道傳輸通道產生的所有例外狀況 (Exception)。  
  
## <a name="exception-list"></a>例外狀況清單  
  
|資源程式碼|資源字串|  
|-------------------|---------------------|  
|PipeCantCloseWithPendingWrite|寫入管道為擱置中狀態時，無法關閉管道。|  
|PipeReadPending|管道已有進行中的讀取作業。|  
|PipeShutdownReadError|管道「關閉」指示器的讀取作業失敗。|  
|PipeShutdownWriteError|管道「關閉」指示器的寫入作業失敗。|  
|PipeWritePending|管道已有進行中的寫入作業。|
