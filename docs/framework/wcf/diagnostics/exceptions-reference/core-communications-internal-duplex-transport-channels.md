# <a name="core-communications-internal-duplex-transport-channels"></a>核心通訊：內部雙工傳輸通道
本主題會列出所有由 Windows Communication Foundation (WCF) 內部雙工傳輸通道產生的例外狀況。  
  
## <a name="exception-list"></a>例外狀況清單  
  
|資源程式碼|資源字串|  
|-------------------|---------------------|  
|MessageMustHaveViaOrToSetForSendingOnServerSideCompositeDuplexChannels|若要在伺服器複合雙工通道上傳送訊息，該訊息必須已設定 'Via' 屬性或 'To' 標頭。|
