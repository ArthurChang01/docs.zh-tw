# <a name="4821---synchronizationcontextsettonull"></a>4821 - SynchronizationContextSetToNull
## <a name="properties"></a>屬性  
  
|||  
|-|-|  
|ID|4821|  
|關鍵字|探索|  
|層級|資訊|  
|通道|Microsoft-Windows-Application Server-Applications/Debug|  
  
## <a name="description"></a>描述  
 當在初始化尋找作業之前，SynchronizationContext 已由 DiscoveryClientChannel 設定為 null 時，就會發出此事件。  
  
## <a name="message"></a>訊息  
 在初始化尋找作業之前，SynchronizationContext 已由 DiscoveryClientChannel 設定為 null。
