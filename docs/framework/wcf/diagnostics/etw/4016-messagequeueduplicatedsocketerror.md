# 4016 - MessageQueueDuplicatedSocketError
## 屬性  
  
|||  
|-|-|  
|ID|4016|  
|關鍵字|ActivationServices|  
|層級|錯誤|  
|通道|Microsoft\-Windows\-Application Server\-Applications\/Analytic|  
  
## 描述  
 當訊息佇列複製通訊端失敗時，就會發出此事件。  
  
## 訊息  
 訊息佇列無法複製通訊端。
