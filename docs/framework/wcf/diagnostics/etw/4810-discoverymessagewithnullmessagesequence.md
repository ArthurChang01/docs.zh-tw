|||  
|-|-|  
|ID|4810|  
|關鍵字|探索|  
|層級|警告|

## <a name="description"></a>描述  
 當發現訊息已被 DiscoveryClient 丟棄時，此事件發出，因為它沒有 DiscoveryMessageSequence 屬性。  
  
## <a name="message"></a>訊息  
 messageId='%2' 的 %1 訊息已遭 DiscoveryClient 丟棄，因為它沒有 DiscoveryMessageSequence 屬性。
