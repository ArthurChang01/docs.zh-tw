|||  
|-|-|  
|ID|4808|  
|關鍵字|探索|  
|層級|警告|

## <a name="description"></a>描述  
 當探索訊息因為 ReplyTo 地址無效而遭丟棄時，就會發出此事件。  
  
## <a name="message"></a>訊息  
 %1 訊息已遭丟棄，因為它沒有任何內容。
