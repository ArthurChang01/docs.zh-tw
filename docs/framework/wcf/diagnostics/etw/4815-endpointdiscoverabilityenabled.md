# 4815 - EndpointDiscoverabilityEnabled
## 屬性  
  
|||  
|-|-|

|關鍵字|探索|  
|層級|資訊|  
|通道|Microsoft\-Windows\-Application Server\-Applications\/Debug|  
  
## 描述  
 啟用端點的探索能力時，就會發出此事件。
