# 3826 - RoutingServiceUsingExistingTransaction
## 屬性  
  
|||

## 描述  
 路由服務會使用現有的交易傳送訊息時，會發出此事件。  
  
## 訊息  
 路由服務會使用現有的交易 '%1' 傳送訊息。
