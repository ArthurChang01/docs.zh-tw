|||  
|-|-|  
|識別碼|3826|  
|關鍵字|RoutingServices|  
|層級|詳細資訊|

## <a name="description"></a>描述  
 路由服務會使用現有的異動傳送訊息時，會發出此事件。  
  
## <a name="message"></a>訊息  
 路由服務會使用現有的異動 '%1' 傳送訊息。
