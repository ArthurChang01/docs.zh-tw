## <a name="properties"></a>屬性  
  
|||  
|-|-|  
|ID|3306|  
|關鍵字|ServiceModel|  
|層級|資訊|  
|通道|Microsoft-Windows-Application Server-Applications/Debug|  
  
## <a name="description"></a>描述  
 當通道處理站因為快取已達其限制而過時超出快取範圍時，會發出此事件。  
  
## <a name="message"></a>訊息  
 通道處理站已過時超出快取範圍，因為快取已達其 '%1' 的限制。
