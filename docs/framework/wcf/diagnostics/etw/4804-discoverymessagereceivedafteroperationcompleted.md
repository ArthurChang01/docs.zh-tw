# 4804 - DiscoveryMessageReceivedAfterOperationCompleted
## 屬性  
  
|||  
|-|-|  
|ID|4804|  
|關鍵字|探索|  
|層級|資訊|  
|通道|Microsoft\-Windows\-Application Server\-Applications\/Debug|  
  
## 描述  
 在作業完成後並收到探索訊息時，就會發出此事件。  
  
## 訊息  
 messageId\='%2' 的 %1 訊息已由 DiscoveryClient 丟棄，因為對應的 %3 作業已完成。
