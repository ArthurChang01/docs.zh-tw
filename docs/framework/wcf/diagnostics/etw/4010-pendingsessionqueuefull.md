# 4010 - PendingSessionQueueFull
## 屬性  
  
|||  
|-|-|  
|ID|4010|  
|關鍵字|ActivationServices|  
|層級|警告|  
|通道|Microsoft\-Windows\-Application Server\-Applications\/Analytic|  
  
## 描述  
 當工作階段分派由於擱置中的工作階段佇列已滿而失敗時，就會發出此事件。  
  
## 訊息  
 '%1' 的工作階段分派失敗，因為擱置中的工作階段佇列已填滿 '%2' 擱置中的項目。
