# <a name="3821---routingserviceclosefailed"></a>3821 - RoutingServiceCloseFailed
## <a name="properties"></a>屬性  
  
|||  
|-|-|  
|識別碼|3821|  
|關鍵字|RoutingServices|  
|層級|警告|  
|通道|Microsoft-Windows-Application Server-Applications/Debug|  
  
## <a name="description"></a>描述  
 如果路由服務在關閉傳出用戶端時失敗，就會發出此事件。  
  
## <a name="message"></a>訊息  
 關閉傳出用戶端 '%1' 時路由服務失敗。
