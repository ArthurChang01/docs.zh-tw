# 3803 - RoutingServiceProcessingFailure
## 屬性  
  
|||  
|-|-|  
|ID|3803|  
|關鍵字|RoutingServices|  
|層級|錯誤|  
|通道|Microsoft\-Windows\-Application Server\-Applications\/Debug|  
  
## 描述  
 當處理端點上的訊息時路由服務失敗，就會發出此事件。  
  
## 訊息  
 處理位址 '%1' 的端點上的訊息時，路由服務失敗。
