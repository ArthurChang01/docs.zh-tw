# 4014 - MessageQueueRegisterFailed
## 屬性  
  
|||  
|-|-|  
|ID|4014|  
|關鍵字|ActivationServices|  
|層級|錯誤|  
|通道|Microsoft\-Windows\-Application Server\-Applications\/Analytic|  
  
## 描述  
 當訊息佇列登錄失敗時，就會發出此事件。  
  
## 訊息  
 URI：'%1' 的訊息佇列登錄失敗，狀態為：'%2'。
