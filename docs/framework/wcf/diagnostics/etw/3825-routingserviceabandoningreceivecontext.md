## <a name="properties"></a>屬性  
  
|||  
|-|-|  
|ID|3825|  
|關鍵字|RoutingServices|  
|層級|警告|  
|通道|Microsoft-Windows-Application Server-Applications/Debug|  
  
## <a name="description"></a>描述  
 此事件於路由服務為指定 ID 的訊息呼叫 ReceiveContext.Abandon 時發出。  
  
## <a name="message"></a>訊息  
 路由服務正在為 ID 為 '%1' 的訊息呼叫 ReceiveContext.Abandon。
