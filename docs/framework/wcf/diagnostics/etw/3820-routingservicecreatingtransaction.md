# 3820 - RoutingServiceCreatingTransaction
## 屬性  
  
|||  
|-|-|  
|ID|3820|  
|關鍵字|RoutingServices|  
|層級|資訊|  
|通道|Microsoft\-Windows\-Application Server\-Applications\/Debug|  
  
## 描述  
 當路由服務為處理訊息已建立新交易時，就會發出此事件。  
  
## 訊息  
 路由服務為處理訊息，已建立 ID 為 '%1' 的新交易。
