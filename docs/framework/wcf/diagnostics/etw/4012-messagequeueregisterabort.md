# 4012 - MessageQueueRegisterAbort
## 屬性  
  
|||  
|-|-|  
|ID|4012|  
|關鍵字|ActivationServices|  
|層級|錯誤|  
|通道|Microsoft\-Windows\-Application Server\-Applications\/Analytic|  
  
## 描述  
 當訊息佇列登錄中止時，就會發出此事件。  
  
## 訊息  
 URI：'%2' 的訊息佇列登錄已中止，狀態為：'%1'。
