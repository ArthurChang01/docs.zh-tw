# <a name="3312---messagereadbyencoder"></a>3312 - MessageReadByEncoder
## <a name="properties"></a>屬性  
  
|||  
|-|-|  
|識別碼|3312|  
|關鍵字|通道|  
|層級|資訊|  
|通道|Microsoft-Windows-Application Server-Applications/Debug|  
  
## <a name="description"></a>描述  
 當編碼器讀取訊息時，就會發出此事件。  
  
## <a name="message"></a>Message  
 編碼器讀取大小為 '%1' 位元組的訊息。
