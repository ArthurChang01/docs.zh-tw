# 62326 - HttpHandlerPickedForUrl
## 屬性  
  
|||  
|-|-|  
|ID|62326|  
|關鍵字|WebHost|  
|層級|資訊|  
|通道|Microsoft\-Windows\-Application Server\-Applications\/Debug|  
  
## 描述  
 系統挑選 HTTP 處理常式來服務對這個 URL 提出的所有要求時，就會發出此事件。  
  
## 訊息  
 URL '%1' 裝載了具有根元素型別 '%2' 的 XAML 文件。  系統挑選 HTTP 處理常式型別 '%3' 來服務對這個 URL 提出的所有要求。
