# 3314 - SessionIdleTimeout
## 屬性  
  
|||  
|-|-|  
|ID|3314|  
|關鍵字|ServiceModel|  
|層級|錯誤|  
|通道|Microsoft\-Windows\-Application Server\-Applications\/Analytic|  
  
## 描述  
 當工作階段因閒置通道而中止時，就會發出此事件。  
  
## 訊息  
 工作階段正在中止閒置的 URI 通道：'%1'。
