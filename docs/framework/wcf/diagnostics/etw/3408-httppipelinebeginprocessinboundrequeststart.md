|||  
|-|-|  
|識別碼|3408|  
|關鍵字|HTTP|  
|層級|詳細資訊|

## <a name="description"></a>描述  
 當 Http 訊息處理常式開始以非同步方式處理輸入要求時，就會發出此事件。
