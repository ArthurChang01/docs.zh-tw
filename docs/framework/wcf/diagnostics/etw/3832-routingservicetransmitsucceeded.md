# 3832 - RoutingServiceTransmitSucceeded
## 屬性  
  
|||  
|-|-|  
|ID|3832|  
|關鍵字|RoutingServices|  
|層級|資訊|  
|通道|Microsoft\-Windows\-Application Server\-Applications\/Debug|  
  
## 描述  
 當路由服務成功傳送具有指定 ID 的訊息時，就會發出此事件。  
  
## 訊息  
 路由服務成功地將 ID：'%1 的訊息 \[作業 %2\] 傳輸至 '%3'。
