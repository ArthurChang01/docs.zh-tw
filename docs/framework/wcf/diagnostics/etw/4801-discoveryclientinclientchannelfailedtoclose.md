# <a name="4801---discoveryclientinclientchannelfailedtoclose"></a>4801 - DiscoveryClientInClientChannelFailedToClose
## <a name="properties"></a>屬性  
  
|||  
|-|-|  
|ID|4801|  
|關鍵字|探索|  
|層級|警告|  
|通道|Microsoft-Windows-Application Server-Applications/Debug|  
  
## <a name="description"></a>描述  
 當在 DiscoveryClientChannel 中建立的 DiscoveryClient 無法關閉，並因而中止時，就會發出此事件。  
  
## <a name="message"></a>訊息  
 在 DiscoveryClientChannel 中建立的 DiscoveryClient 無法關閉，因此已經中止。
