# 4600 - MessageLogEventSizeExceeded
## 屬性  
  
|||  
|-|-|  
|ID|4600|  
|關鍵字|WCFMessageLogging|  
|層級|警告|  
|通道|Microsoft\-Windows\-Application Server\-Applications\/Debug|  
  
## 描述  
 當訊息因為超過 ETW 事件大小而無法記錄時，就會發出此事件。  
  
## 訊息  
 無法記錄訊息，因為它已超過 ETW 事件大小
