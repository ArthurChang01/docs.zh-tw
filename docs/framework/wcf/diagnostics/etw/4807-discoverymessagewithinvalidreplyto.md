# 4807 - DiscoveryMessageWithInvalidReplyTo
## 屬性  
  
|||  
|-|-|  
|ID|4807|  
|關鍵字|探索|  
|層級|警告|  
|通道|Microsoft\-Windows\-Application Server\-Applications\/Debug|  
  
## 描述  
 當探索訊息因為 ReplyTo 地址無效而遭丟棄時，就會發出此事件。  
  
## 訊息  
 messageId\='%1' 的探索要求訊息已遭丟棄，因為它的 ReplyTo 地址無效。
