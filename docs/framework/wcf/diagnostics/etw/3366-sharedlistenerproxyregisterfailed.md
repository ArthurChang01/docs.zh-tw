# 3366 - SharedListenerProxyRegisterFailed
## 屬性  
  
|||  
|-|-|  
|ID|3366|  
|關鍵字|ActivationServices|  
|層級|錯誤|  
|通道|Microsoft\-Windows\-Application Server\-Applications\/Analytic|  
  
## 描述  
 當 SharedListenerProxy 登錄失敗時，就會發出此事件。  
  
## 訊息  
 SharedListenerProxy 登錄失敗，狀態為 '%1'。
