# 3827 - RoutingServiceTransmitFailed
## 屬性  
  
|||  
|-|-|  
|ID|3827|  
|關鍵字|RoutingServices|  
|層級|警告|  
|通道|Microsoft\-Windows\-Application Server\-Applications\/Debug|  
  
## 描述  
 當路由服務傳輸失敗時，就會發出此事件。  
  
## 訊息  
 傳送至 '%1' 時，路由服務失敗。
