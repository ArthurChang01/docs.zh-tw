# 4805 - DiscoveryMessageWithInvalidContent
## 屬性  
  
|||  
|-|-|  
|ID|4805|  
|關鍵字|探索|  
|層級|警告|  
|通道|Microsoft\-Windows\-Application Server\-Applications\/Debug|  
  
## 描述  
 當探索訊息包含無效的內容時，就會發出此事件。  
  
## 訊息  
 一項 messageId\='%2' 的 %1 訊息已遭丟棄，因為它的內容無效。
