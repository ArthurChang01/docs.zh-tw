# 1443 - MaxRetryCyclesExceededMsmq
## 屬性  
  
|||  
|-|-|  
|ID|1443|  
|關鍵字|配額|  
|層級|錯誤|  
|通道|Microsoft\-Windows\-Application Server\-Applications\/Analytic|  
  
## 描述  
 當 MSMQ 訊息上的重試循環超出上限時，就會發出此事件。  
  
## 訊息  
 在 ID 為 '%1' 之 MSMQ 訊息上已超過最大重試循環
