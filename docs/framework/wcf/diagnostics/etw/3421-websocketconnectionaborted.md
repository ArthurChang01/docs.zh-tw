# 3421 - WebSocketConnectionAborted
## 屬性  
  
|||  
|-|-|  
|ID|3421|  
|關鍵字|HTTP|  
|層級|錯誤|  
|通道|Microsoft\-Windows\-Application Server\-Applications\/Analytic|  
  
## 描述  
 當 WebSocket 連線中止時，就會發出此事件。  
  
## 訊息  
 WebSocketId：%1 連接已中止。
