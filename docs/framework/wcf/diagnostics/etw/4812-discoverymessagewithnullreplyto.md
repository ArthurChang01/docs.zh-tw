# 4812 - DiscoveryMessageWithNullReplyTo
## 屬性  
  
|||  
|-|-|  
|ID|4812|  
|關鍵字|探索|  
|層級|警告|  
|通道|Microsoft\-Windows\-Application Server\-Applications\/Debug|  
  
## 描述  
 當探索訊息因為沒有 ReplyTo 地址而已被 DiscoveryClient 丟棄時，就會發出此事件。  
  
## 訊息  
 messageId\='%1' 的探索要求訊息已遭丟棄，因為它沒有 ReplyTo 地址。
