# 4814 - EndpointDiscoverabilityDisabled
## 屬性  
  
|||  
|-|-|  
|ID|4814|  
|關鍵字|探索|  
|層級|資訊|  
|通道|Microsoft\-Windows\-Application Server\-Applications\/Debug|  
  
## 描述  
 停用端點的探索能力時，就會發出此事件。  
  
## 訊息  
 EndpointAddress\='%1' 且 ListenUri\='%2' 的端點可搜尋性已遭停用。
