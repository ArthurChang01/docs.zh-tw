# <a name="3430---websocketuseversionfromclientwebsocketfactory"></a>3430 - WebSocketUseVersionFromClientWebSocketFactory
## <a name="properties"></a>屬性  
  
|||  
|-|-|  
|識別碼|3430|  
|關鍵字|HTTP|  
|層級|詳細資訊|  
|通道|Microsoft-Windows-Application Server-Applications/Debug|  
  
## <a name="description"></a>描述  
 使用來自用戶端 WebSocket 工廠的 WebSocketVersion 時，就會發出此事件。  
  
## <a name="message"></a>訊息  
 正在使用來自型別為 '%1' 的用戶端 WebSocket 工廠的 WebSocketVersion。
