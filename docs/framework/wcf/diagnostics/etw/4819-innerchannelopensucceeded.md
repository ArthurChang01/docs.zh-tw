|||  
|-|-|

|關鍵字|探索|  
|層級|資訊|

## <a name="description"></a>描述  
 當 DiscoveryClientChannel 成功找到端點並用其開啟通道時，就會發出此事件。  
  
## <a name="message"></a>訊息  
 DiscoveryClientChannel 成功發現一個端點並使用它開啟通道。 用戶端已使用 EndpointAddress='%1' 和 Via='%2' 連接至服務。
