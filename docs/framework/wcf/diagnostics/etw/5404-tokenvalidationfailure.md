# 5404 - TokenValidationFailure
## 屬性  
  
|||  
|-|-|  
|ID|5404|  
|關鍵字|安全性|  
|層級|錯誤|  
|通道|Microsoft\-Windows\-Application Server\-Applications\/Debug|  
  
## 描述  
 當 SecurityToken 驗證失敗時，就會發出此事件。  
  
## 訊息  
 SecurityToken \(類型 '%1' 和 ID '%2'\) 驗證失敗。  %3
