# 4809 - DiscoveryMessageWithNullMessageId
## 屬性  
  
|||  
|-|-|  
|ID|4809|  
|關鍵字|探索|  
|層級|警告|  
|通道|Microsoft\-Windows\-Application Server\-Applications\/Debug|  
  
## 描述  
 當探索訊息因為訊息標頭未包含必要的 MessageId 屬性而遭丟棄時，會發出此事件。  
  
## 訊息  
 %1 訊息已遭丟棄，因為訊息標頭未包含必要的 MessageId 屬性。
