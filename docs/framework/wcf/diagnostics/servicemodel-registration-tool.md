# <a name="servicemodel-registration-tool"></a>ServiceModel 註冊工具
ServiceModel 註冊工具是一種命令列工具，可讓您在單一電腦上管理 ServiceModel 的註冊。 如需此工具的詳細資訊，請參閱[System.servicemodel 註冊工具（servicemodelreg.exe .exe）](../servicemodelreg-exe.md)。
