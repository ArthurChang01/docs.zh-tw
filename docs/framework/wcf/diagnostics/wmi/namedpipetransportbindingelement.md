# <a name="namedpipetransportbindingelement"></a>NamedPipeTransportBindingElement
NamedPipeTransportBindingElement  
  
## <a name="syntax"></a>語法  
  
```  
class NamedPipeTransportBindingElement : ConnectionOrientedTransportBindingElement  
{  
  NamedPipeConnectionPoolSettings ConnectionPoolSettings;  
};  
```  
  
## <a name="methods"></a>方法  
 NamedPipeTransportBindingElement 類別沒有定義任何方法。  
  
## <a name="properties"></a>屬性  
 NamedPipeTransportBindingElement 類別具有下列屬性：  
  
### <a name="connectionpoolsettings"></a>ConnectionPoolSettings  
 資料型別：NamedPipeConnectionPoolSettings  
  
 存取類型：唯讀  
  
 連線集區設定。
