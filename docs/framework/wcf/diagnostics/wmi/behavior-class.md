# Behavior 類別
## 語法  
  
```  
class Behavior  
{  
  string Type;  
};  
```  
  
## 方法  
 Behavior 類別不會定義任何方法。  
  
## 屬性  
 Behavior 類別具有下列屬性：  
  
### 類型  
 資料型別：字串  
  
 存取類型：唯讀  
  
 行為類型。  
  
## 需求  
  
|MOF|於 Servicemodel.mof 中宣告。|  
|---------|-----------------------------|  
|命名空間|於 root\\ServiceModel 中定義|
