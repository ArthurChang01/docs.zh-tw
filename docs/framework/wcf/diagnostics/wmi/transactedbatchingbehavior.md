# TransactedBatchingBehavior
TransactedBatchingBehavior  
  
## 語法  
  
```  
class TransactedBatchingBehavior : Behavior  
{  
  sint32 MaxBatchSize;  
};  
```  
  
## 方法  
 TransactedBatchingBehavior 類別並未定義任何方法。  
  
## 屬性  
 TransactedBatchingBehavior 類別具有下列屬性：  
  
### MaxBatchSize  
 資料型別：sint32  
  
 存取類型：唯讀  
  
 交易批次處理的批次大小上限。
