# PeerResolverBindingElement
PeerResolverBindingElement  
  
## 語法  
  
```  
class PeerResolverBindingElement : BindingElement  
{  
  string ReferralPolicy;  
};  
```  
  
## 方法  
 PeerResolverBindingElement 類別不會定義任何方法。  
  
## 屬性  
 PeerResolverBindingElement 類別有下列屬性：  
  
### ReferralPolicy  
 資料型別：字串  
  
 存取類型：唯讀  
  
 決定如何在對等之間共用轉介。
