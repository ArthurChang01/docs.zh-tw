# TransactionFlowAttribute
TransactionFlowAttribute  
  
## 語法  
  
```  
class TransactionFlowAttribute : Behavior  
{  
  string TransactionFlowOption;  
};  
```  
  
## 方法  
 TransactionFlowAttribute 類別並未定義任何方法。  
  
## 屬性  
 TransactionFlowAttribute 類別具有下列屬性：  
  
### TransactionFlowOption  
 資料型別：字串  
  
 存取類型：唯讀  
  
 代表交易是否正在進行。
