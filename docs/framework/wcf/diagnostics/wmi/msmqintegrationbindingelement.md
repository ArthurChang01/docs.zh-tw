# MsmqIntegrationBindingElement
MsmqIntegrationBindingElement  
  
## 語法  
  
```  
class MsmqIntegrationBindingElement : MsmqBindingElementBase  
{  
  string SerializationFormat;  
};  
```  
  
## 方法  
 MsmqIntegrationBindingElement 類別不會定義任何方法。  
  
## 屬性  
 MsmqIntegrationBindingElement 類別具有下列屬性：  
  
### SerializationFormat  
 資料型別：字串  
  
 存取類型：唯讀  
  
 繫結用來序列化訊息的格式。
