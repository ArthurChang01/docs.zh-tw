# PeerTransportSecuritySettings
PeerTransportSecuritySettings  
  
## 語法  
  
```  
class PeerTransportSecuritySettings  
{  
  string CredentialType;  
};  
```  
  
## 方法  
 PeerTransportSecuritySettings 類別並未定義任何方法。  
  
## 屬性  
 PeerTransportSecuritySettings 類別具有下列屬性：  
  
### CredentialType  
 資料型別：字串  
  
 存取類型：唯讀  
  
 對等安全性項目的傳輸認證類型。
