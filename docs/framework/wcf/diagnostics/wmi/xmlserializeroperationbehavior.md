# XmlSerializerOperationBehavior
XmlSerializerOperationBehavior  
  
## 語法  
  
```  
class XmlSerializerOperationBehavior : Behavior  
{  
  string Style;  
  string Use;  
};  
```  
  
## 方法  
 XmlSerializerOperationBehavior 類別不會定義任何方法。  
  
## 屬性  
 XmlSerializerOperationBehavior 類別具有下列屬性：  
  
### Style  
 資料型別：字串  
  
 存取類型：唯讀  
  
 定義 SOAP 訊息的樣式。  
  
### Use  
 資料型別：字串  
  
 存取類型：唯讀  
  
 指定 SOAP 編碼樣式。

|命名空間|於 root\\ServiceModel 中定義|
