# MessageEncodingBindingElement
MessageEncodingBindingElement  
  
## 語法  
 `class MessageEncodingBindingElement : BindingElement`  
  
 `{`  
  
 `string MessageVersion;`  
  
 `};`  
  
## 方法  
 MessageEncodingBindingElement 類別不會定義任何方法。  
  
## 屬性  
 MessageEncodingBindingElement 類別具有下列屬性：  
  
### MessageVersion  
 資料型別：字串  
  
 存取類型：唯讀  
  
 使用這個繫結所傳送訊息的 SOAP 版本。
