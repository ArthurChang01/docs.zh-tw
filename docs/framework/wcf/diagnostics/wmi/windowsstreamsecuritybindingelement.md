# WindowsStreamSecurityBindingElement
WindowsStreamSecurityBindingElement  
  
## 語法  
  
```  
class WindowsStreamSecurityBindingElement : BindingElement  
{  
  string ProtectionLevel;  
};  
```  
  
## 方法  
 WindowsStreamSecurityBindingElement 類別並未定義任何方法。  
  
## 屬性  
 WindowsStreamSecurityBindingElement 類別具有下列屬性：  
  
### ProtectionLevel  
 資料型別：字串  
  
 存取類型：唯讀  
  
 TCP 資料流的保護層級。
