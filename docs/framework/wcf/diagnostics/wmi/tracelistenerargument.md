# <a name="tracelistenerargument"></a>TraceListenerArgument
TraceListenerArgument  
  
## <a name="syntax"></a>語法  
  
```  
class TraceListenerArgument  
{  
  string Name;  
  string Value;  
};  
```  
  
## <a name="methods"></a>方法  
 TraceListenerArgument 類別不會定義任何方法。  
  
## <a name="properties"></a>屬性  
 TraceListenerArgument 類別有下列屬性：  
  
### <a name="name"></a>名稱  
 資料型別：字串  
  
 存取類型：唯讀  
  
 引數的名稱。  
  
### <a name="value"></a>值  
 資料型別：字串  
  
 存取類型：唯讀  
  
 引數的值。
