# <a name="thumbprintnotfound"></a>ThumbPrintNotFound
識別碼:152  
  
 嚴重性：錯誤  
  
 類別：TransactionBridge  
  
## <a name="description"></a>描述  
 這個事件表示找不到具有該特定指紋的身分識別憑證。 此事件會列出處理序名稱和處理序識別碼。
