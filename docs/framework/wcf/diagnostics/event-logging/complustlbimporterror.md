# <a name="complustlbimporterror"></a>ComPlusTLBImportError
識別碼：132  
  
 嚴重性：錯誤  
  
 分類：ServiceModel  
  
## <a name="description"></a>描述  
 這個事件表示當匯入型別程式庫時發生錯誤。 此事件會列出 IID、型別程式庫識別碼、例外狀況、處理序名稱和處理序識別碼。
