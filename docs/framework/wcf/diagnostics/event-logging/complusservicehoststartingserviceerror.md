# <a name="complusservicehoststartingserviceerror"></a>ComPlusServiceHostStartingServiceError
識別碼:130  
  
 嚴重性：Error  
  
 CategoryServiceModel  
  
## <a name="description"></a>描述  
 這個事件表示在啟動 COM+ 服務時發生錯誤。 此事件會列出 AppID、ClsID、Surrogate、例外狀況、處理序名稱和處理序識別碼。
