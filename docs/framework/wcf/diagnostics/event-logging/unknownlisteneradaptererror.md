# <a name="unknownlisteneradaptererror"></a>UnknownListenerAdapterError
識別碼:161  
  
 嚴重性：Error  
  
 CategorySMSvcHost  
  
## <a name="description"></a>說明  
 這個事件表示處理 Windows Process Activation Service (WAS) 通知時接聽項配接器中發生意外的錯誤。 已結束此處理序。 此事件會列出通訊協定識別碼、例外狀況、處理序名稱和處理序識別碼。
