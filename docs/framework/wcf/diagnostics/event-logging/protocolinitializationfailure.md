# <a name="protocolinitializationfailure"></a>ProtocolInitializationFailure
識別碼:142  
  
 嚴重性：錯誤  
  
 類別：TransactionBridge  
  
## <a name="description"></a>描述  
 此事件表示 WS-Atomic Transaction 通訊協定服務未能初始化。 因此，會停用 WS-Atomic 異動功能。 此事件會列出通訊協定識別碼、通訊協定名稱、例外狀況、處理序名稱與處理序識別碼。 。
