# <a name="protocolstopfailure"></a>ProtocolStopFailure
Id: 147  
  
 嚴重性：錯誤  
  
 分類：TransactionBridge  
  
## <a name="description"></a>描述  
 這個事件表示 WS-Atomic 異動通訊協定服務無法停止。 此事件會列出通訊協定識別碼、通訊協定名稱、例外狀況、處理序名稱與處理序識別碼。
