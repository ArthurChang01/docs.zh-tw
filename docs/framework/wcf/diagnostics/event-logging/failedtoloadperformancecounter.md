# <a name="failedtoloadperformancecounter"></a>FailedToLoadPerformanceCounter
識別碼：118  
  
 嚴重性：錯誤  
  
 分類：ServiceModel  
  
## <a name="description"></a>描述  
 這個事件指出效能計數器沒有載入。 此事件會列出類別名稱、計數器名稱、執行個體名稱、例外狀況、處理序名稱和處理序識別碼。
