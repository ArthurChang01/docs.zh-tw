# <a name="failedtosetuptracing"></a>FailedToSetupTracing
識別碼:100  
  
 嚴重性：錯誤  
  
 類別：追蹤  
  
## <a name="description"></a>描述  
 嘗試設定追蹤失敗。 已停用追蹤。  
  
 此事件會列出例外狀況、處理序名稱和處理序識別碼。
