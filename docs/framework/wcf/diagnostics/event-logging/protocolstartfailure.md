# <a name="protocolstartfailure"></a>ProtocolStartFailure
識別碼：143  
  
 嚴重性：錯誤  
  
 分類：TransactionBridge  
  
## <a name="description"></a>描述  
 此事件表示 WS-Atomic 交易通訊協定服務無法啟動。 因此，會停用 WS-Atomic 異動功能。 此事件會列出通訊協定識別碼、通訊協定名稱、例外狀況、處理序名稱與處理序識別碼。 。
