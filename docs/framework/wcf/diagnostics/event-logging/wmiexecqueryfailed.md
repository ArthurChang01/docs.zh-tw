# <a name="wmiexecqueryfailed"></a>WmiExecQueryFailed
識別碼:124  
  
 嚴重性：錯誤  
  
 類別：ServiceModel  
  
## <a name="description"></a>描述  
 此事件表示 WMI ExecQuery 要求尚未處理。 此事件會列出例外狀況、處理序名稱和處理序識別碼。
