# <a name="protocolrecoverycomplete"></a>ProtocolRecoveryComplete
識別碼:150  
  
 嚴重性：錯誤  
  
 類別：TransactionBridge  
  
## <a name="description"></a>描述  
 此事件表示 WS-Atomic Transaction 通訊協定服務已成功完成啟動與復原。 此事件會列出通訊協定識別碼、通訊協定名稱、處理序名稱和處理序識別碼。
