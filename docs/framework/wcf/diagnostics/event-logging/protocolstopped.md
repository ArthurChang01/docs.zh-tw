# <a name="protocolstopped"></a>ProtocolStopped
識別碼：151  
  
 嚴重性：錯誤  
  
 分類：TransactionBridge  
  
## <a name="description"></a>描述  
 這個事件指出 WS-Atomic 異動通訊協定服務已停止。 此事件會列出通訊協定識別碼、通訊協定名稱、處理序名稱和處理序識別碼。 。
