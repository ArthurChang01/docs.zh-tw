# <a name="piiloggingon"></a>PiiLoggingOn
Id: 107  
  
 嚴重性：錯誤  
  
 分類：追蹤  
  
## <a name="description"></a>描述  
 這個事件表示已開啟 PII 記錄。 將記錄敏感資料。 此事件會列出處理序名稱和處理序識別碼。
