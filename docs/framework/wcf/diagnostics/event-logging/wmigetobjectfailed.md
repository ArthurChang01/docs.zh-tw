# <a name="wmigetobjectfailed"></a>WmiGetObjectFailed
Id: 120  
  
 嚴重性：錯誤  
  
 分類：ServiceModel  
  
## <a name="description"></a>描述  
 此事件表示 WMI GetObject 要求尚未處理。 此事件會列出 WMI 物件、例外狀況、處理序名稱與處理序識別碼。
