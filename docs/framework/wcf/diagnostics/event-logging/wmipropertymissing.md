# <a name="wmipropertymissing"></a>WmiPropertyMissing
識別碼:129  
  
 嚴重性：錯誤  
  
 類別：ServiceModel  
  
## <a name="description"></a>描述  
 這個事件會指出發生類型不符。 指定的 CIM 類別沒有特定型別的指定屬性。 此事件會列出處理序名稱和處理序識別碼。
