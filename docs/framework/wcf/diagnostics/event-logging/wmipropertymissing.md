# WmiPropertyMissing
識別碼：129  
  
 嚴重性：錯誤  
  
 分類：ServiceModel  
  
## 描述  
 這個事件表示發生型別不符。指定的 CIM 類別沒有特定型別的指定屬性。此事件會列出處理序名稱和處理序識別碼。
