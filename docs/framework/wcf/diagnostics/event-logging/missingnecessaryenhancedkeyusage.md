# <a name="missingnecessaryenhancedkeyusage"></a>MissingNecessaryEnhancedKeyUsage
Id: 157  
  
 嚴重性：錯誤  
  
 分類：TransactionBridge  
  
## <a name="description"></a>描述  
 此事件表示，包含特定主體名稱和指紋的身分識別憑證並未在其 EnhancedKeyUsages 中提供特定資料。 此事件會列出處理序名稱和處理序識別碼。
