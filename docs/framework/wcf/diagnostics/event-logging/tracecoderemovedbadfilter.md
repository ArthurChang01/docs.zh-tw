# <a name="tracecoderemovedbadfilter"></a>TraceCodeRemovedBadFilter
識別碼:114  
  
 嚴重性：Error  
  
 CategoryServiceModel  
  
## <a name="description"></a>說明  
 這個事件表示存取訊息本文的不正確訊息記錄篩選器已經移除。 此事件會列出 XPath、處理序名稱和處理序識別碼。
