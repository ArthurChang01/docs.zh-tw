# ParticipantRecoveryLogEntryCreationFailure
識別碼：141  
  
 嚴重性：錯誤  
  
 分類：TransactionBridge  
  
## 描述  
 這個事件指出，無法產生修復記錄項目供參與者登記。將會中止該交易。此事件會列出交易識別碼、登記識別碼、例外狀況、處理序名稱和處理序識別碼。
