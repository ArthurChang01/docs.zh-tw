# MessageAuthenticationSuccess
ID：169  
  
 嚴重性：告知性  
  
 分類：SecurityAudit  
  
## 描述  
 這個事件表示訊息驗證成功。此事件會列出服務、動作、用戶端身分識別和活動識別碼。
