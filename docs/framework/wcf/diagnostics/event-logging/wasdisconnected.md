# <a name="wasdisconnected"></a>WasDisconnected
識別碼:162  
  
 嚴重性：Error  
  
 CategorySMSvcHost  
  
## <a name="description"></a>描述  
 這個事件表示 Windows Process Activation (WAS) 已中斷連線。 此事件會列出 HRESULT、處理序名稱與處理序識別碼。
