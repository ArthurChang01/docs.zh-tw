# <a name="wmiexecmethodfailed"></a>WmiExecMethodFailed
Id: 125  
  
 嚴重性：錯誤  
  
 分類：ServiceModel  
  
## <a name="description"></a>描述  
 此事件表示 WMI ExecMethod 要求尚未處理。 此事件會列出例外狀況、處理序名稱和處理序識別碼。
