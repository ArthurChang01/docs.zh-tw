# <a name="performancecounterinitializationfailure"></a>PerformanceCounterInitializationFailure
Id: 149  
  
 嚴重性：錯誤  
  
 分類：TransactionBridge  
  
## <a name="description"></a>描述  
 這個事件表示效能計數器無法初始化。 此事件會列出計數器名稱、例外狀況、處理序名稱與處理序識別碼。
