# <a name="messagequeueduplicatedsocketleak"></a>MessageQueueDuplicatedSocketLeak
Id: 165  
  
 嚴重性：錯誤  
  
 分類：SMSvcHost  
  
## <a name="description"></a>描述  
 此事件指出分派重複的通訊端時發生錯誤。 處理序中的這個處理常式已流失。 此事件會列出來源、例外狀況、處理序名稱與處理序識別碼。
