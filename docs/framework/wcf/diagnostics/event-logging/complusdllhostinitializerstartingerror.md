# <a name="complusdllhostinitializerstartingerror"></a>ComPlusDllHostInitializerStartingError
識別碼:131  
  
 嚴重性：Error  
  
 CategoryServiceModel  
  
## <a name="description"></a>說明  
 這個事件指出，在啟動 DllHost 初始設定式時發生錯誤。 此事件會列出 AppID、例外狀況、處理序名稱和處理序識別碼。
