# FatalUnexpectedStateMachineEvent
ID：137  
  
 嚴重性：錯誤  
  
 分類：TransactionBridge  
  
## 描述  
 這個事件表示狀態電腦嘗試處理非預期的事件。這個事件視為嚴重錯誤，而且其中還會列出交易識別碼、狀態電腦、目前狀態、記錄、事件名稱、事件詳細資料、處理序名稱及處理序識別碼。
