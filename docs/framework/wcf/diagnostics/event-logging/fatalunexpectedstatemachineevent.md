# <a name="fatalunexpectedstatemachineevent"></a>FatalUnexpectedStateMachineEvent
識別碼:137  
  
 嚴重性：錯誤  
  
 類別：TransactionBridge  
  
## <a name="description"></a>描述  
 這個事件表示狀態電腦嘗試處理非預期的事件。 這個事件視為嚴重錯誤， 而且其中會列出異動識別碼、狀態電腦、目前狀態、記錄、事件名稱、事件詳細資料、處理序名稱及處理序識別碼。
