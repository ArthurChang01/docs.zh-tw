# <a name="wmiputinstancefailed"></a>WmiPutInstanceFailed
Id: 121  
  
 嚴重性：錯誤  
  
 分類：ServiceModel  
  
## <a name="description"></a>描述  
 這個事件表示 WMI PutInstance 要求尚未處理。 此事件會列出 WMI 物件、例外狀況、處理序名稱與處理序識別碼。
