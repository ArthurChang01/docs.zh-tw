# <a name="securitynegotiationsuccess"></a>SecurityNegotiationSuccess
識別碼： 171  
  
 嚴重性：告知性  
  
 分類：SecurityAudit  
  
## <a name="description"></a>描述  
 這個事件表示安全性交涉成功，並列出服務名稱、行動、用戶端識別、ActivityId 和交涉。
