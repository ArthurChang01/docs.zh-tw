# InvariantAssertionFailed
識別碼：106  
  
 嚴重性：錯誤  
  
 分類：追蹤  
  
## 描述  
 此事件表示非變異判斷提示為 false。此事件會列出訊息、處理序名稱和處理序識別碼。
