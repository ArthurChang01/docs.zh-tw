# <a name="serviceauthorizationfailure"></a>ServiceAuthorizationFailure
識別碼:168  
  
 嚴重性：錯誤  
  
 類別：SecurityAudit  
  
## <a name="description"></a>描述  
 這個事件表示服務授權失敗。 此事件會列出服務、動作、用戶端身分識別、AuthorizationContext、活動識別碼及 ServiceAuthorizationManager。
