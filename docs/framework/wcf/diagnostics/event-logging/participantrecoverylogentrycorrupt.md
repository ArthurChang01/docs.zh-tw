# ParticipantRecoveryLogEntryCorrupt
識別碼：138  
  
 嚴重性：錯誤  
  
 分類：TransactionBridge  
  
## 描述  
 這個事件指出，參與者修復記錄項目已毀損，且無法還原序列化。這個錯誤可能會造成資料遺失。事件會列出交易識別碼、修復資料 \(以 Base64 編碼\)、例外狀況、處理序名稱和處理序識別碼。
