# <a name="wmiadmintypemismatch"></a>WmiAdminTypeMismatch
識別碼:128  
  
 嚴重性：錯誤  
  
 類別：ServiceModel  
  
## <a name="description"></a>描述  
 這個事件會指出發生類型不符。 指定的 CIM 類別屬性無法接受特定型別的值。 此事件會列出處理序名稱和處理序識別碼。
