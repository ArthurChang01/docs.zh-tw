# FailedToCreateMessageLoggingTraceSource
ID：115  
  
 嚴重性：錯誤  
  
 分類：ServiceModel  
  
## 描述  
 這個事件表示訊息記錄無法建立追蹤來源。此事件會列出例外狀況、處理序名稱和處理序識別碼。
