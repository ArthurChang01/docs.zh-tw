# <a name="securitynegotiationfailure"></a>SecurityNegotiationFailure
識別碼： 172  
  
 嚴重性：錯誤  
  
 分類：SecurityAudit  
  
## <a name="description"></a>描述  
 這個事件表示安全性交涉失敗。 此事件會列出服務、動作、用戶端識別、活動識別碼和交涉。
