# WebHostUnhandledException
識別碼：109  
  
 嚴重性：錯誤  
  
 分類：ServiceModel  
  
## 描述  
 這個事件會指出已發生 WebHost 未處理例外狀況。此事件會列出寄件者資訊、例外狀況、處理序名稱和處理序識別碼。
