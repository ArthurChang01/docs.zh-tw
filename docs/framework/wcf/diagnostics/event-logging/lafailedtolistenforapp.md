# <a name="lafailedtolistenforapp"></a>LAFailedToListenForApp
識別碼:160  
  
 嚴重性：Error  
  
 CategorySMSvcHost  
  
## <a name="description"></a>說明  
 這個事件表示在嘗試接聽指定的網站時，指定之通訊協定的指定啟用服務中發生錯誤，因此暫時停用該網站的通訊協定。 此事件會列出 URL、狀態、例外狀況、進程名稱和處理序識別碼。
