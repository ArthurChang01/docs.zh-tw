# <a name="bindingerror"></a>BindingError
識別碼:159  
  
 嚴重性：Error  
  
 CategorySMSvcHost  
  
## <a name="description"></a>說明  
 這個事件表示剖析特定網站的特定繫結時發生錯誤，因此已為該網站暫時停用此通訊協定。 此事件會列出繫結、來源、例外狀況、處理序名稱與處理序識別碼。
