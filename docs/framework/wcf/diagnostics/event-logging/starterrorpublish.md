# <a name="starterrorpublish"></a>StartErrorPublish
識別碼:158  
  
 嚴重性：錯誤  
  
 類別：SMSvcHost  
  
## <a name="description"></a>描述  
 這個事件表示無法啟動 NT 服務。 也無法發行接聽端點。 此事件會列出處理序名稱和處理序識別碼。
