# <a name="sslnoprivatekey"></a>SslNoPrivateKey
識別碼:154  
  
 嚴重性：錯誤  
  
 類別：TransactionBridge  
  
## <a name="description"></a>描述  
 此事件表示包含特定主體名稱與指紋的身分識別憑證並未包含私密金鑰。 此事件會列出處理序名稱和處理序識別碼。
