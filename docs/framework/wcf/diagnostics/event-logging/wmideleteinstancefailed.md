# <a name="wmideleteinstancefailed"></a>WmiDeleteInstanceFailed
識別碼:122  
  
 嚴重性：Error

## <a name="description"></a>說明  
 此事件表示 WMI DeleteInstance 要求尚未處理。 此事件會列出例外狀況、處理序名稱和處理序識別碼。
