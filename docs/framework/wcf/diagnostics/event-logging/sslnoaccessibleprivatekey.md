# <a name="sslnoaccessibleprivatekey"></a>SslNoAccessiblePrivateKey
Id: 155  
  
 嚴重性：錯誤  
  
 分類：TransactionBridge  
  
## <a name="description"></a>描述  
 此事件表示包含特定主體名稱與指紋的識別憑證並未包含可存取之私密金鑰。 此事件會列出處理序名稱和處理序識別碼。
