# <a name="transportauthenticationfailure"></a>TransportAuthenticationFailure
識別碼:174  
  
 嚴重性：錯誤  
  
 類別：SecurityAudit  
  
## <a name="description"></a>描述  
 這個事件指出，傳輸驗證已失敗、列出用戶端身分識別以及活動識別碼。
