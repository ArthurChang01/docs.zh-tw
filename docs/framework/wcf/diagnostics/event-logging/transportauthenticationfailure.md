# TransportAuthenticationFailure
識別碼：174  
  
 嚴重性：錯誤  
  
 分類：SecurityAudit  
  
## 描述  
 這個事件指出，傳輸驗證已失敗、列出用戶端身分識別以及活動識別碼。
