# FailedToInitializeTraceSource
識別碼：101  
  
 嚴重性：錯誤  
  
 分類：追蹤  
  
## 描述  
 未初始化追蹤來源。  已停用追蹤。  此事件會列出例外狀況、處理序名稱和處理序識別碼。
