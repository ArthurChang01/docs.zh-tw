# TransportAuthenticationSuccess
ID：173  
  
 嚴重性：告知性  
  
 分類：SecurityAudit  
  
## 描述  
 這個事件表示傳輸驗證成功。此事件會列出服務、用戶端身分識別和活動 ID。
