# <a name="serviceauthorizationsuccess"></a>ServiceAuthorizationSuccess
Id: 176  
  
 嚴重性：告知性  
  
 分類：SecurityAudit  
  
## <a name="description"></a>描述  
 此事件表示服務授權成功。 此事件會列出服務、動作、用戶端識別、授權內容、活動 ID和 ServiceAuthorizationManager。
