# <a name="webhostfailedtolisten"></a>WebHostFailedToListen
識別碼:112  
  
 嚴重性：錯誤  
  
 類別：ServiceModel  
  
## <a name="description"></a>描述  
 這個事件表示當嘗試接聽指定的 URL 時發生錯誤。 這個背景工作處理序已終止。 此事件會列出例外狀況、處理序名稱和處理序識別碼。
