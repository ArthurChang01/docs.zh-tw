# <a name="wmiunregistrationfailed"></a>WmiUnregistrationFailed
識別碼:127  
  
 嚴重性：錯誤  
  
 類別：ServiceModel  
  
## <a name="description"></a>描述  
 這個事件表示未移除註冊 WMI 提供者。 此事件會列出 WMI 物件、錯誤、處理序名稱與處理序識別碼。
