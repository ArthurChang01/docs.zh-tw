# <a name="wmiregistrationfailed"></a>WmiRegistrationFailed
Id: 126  
  
 嚴重性：錯誤  
  
 分類：ServiceModel  
  
## <a name="description"></a>描述  
 此事件表示尚未註冊 WMI 提供者。 此事件會列出 WMI 物件、錯誤、處理序名稱與處理序識別碼。
