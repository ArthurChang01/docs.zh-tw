# ProtocolRecoveryCompleteFailure
識別碼：145  
  
 嚴重性：錯誤  
  
 分類：TransactionBridge  
  
## 描述  
 此事件表示 WS\-Atomic Transaction 通訊協定服務無法完成啟動和修復。因此，會停用 WS\-Atomic 交易功能。此事件會列出通訊協定識別碼、通訊協定名稱、例外狀況、處理序名稱和處理序識別碼。
