# 部署服務
本主題將說明如何將 [!INCLUDE[indigo1](../../../../includes/indigo1-md.md)] 應用程式部署到執行階段環境。  
  
## 為您的應用程式選擇裝載環境  
 [!INCLUDE[indigo2](../../../../includes/indigo2-md.md)] 服務是設計為在支援 Managed 程式碼的 Windows 處理序中執行的。如果要成為作用中的服務，必須將服務裝載在建立及控制其內容和存留時間的執行階段環境中。裝載選項的範圍從在最簡單的主控台應用程式到伺服器環境 \(如 Windows 服務、Internet Information Services \(IIS\)\) 內執行，或在由 Windows Activation Service \(WAS\) 管理的背景工作處理序中執行。如果要檢閱您的 [!INCLUDE[indigo2](../../../../includes/indigo2-md.md)] 應用程式的不同裝載選項，請參閱[裝載服務](../../../../docs/framework/wcf/hosting-services.md)。
