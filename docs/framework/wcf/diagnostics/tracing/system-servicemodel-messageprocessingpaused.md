# <a name="systemservicemodelmessageprocessingpaused"></a>System.ServiceModel.MessageProcessingPaused
System.ServiceModel.MessageProcessingPaused  
  
## <a name="description"></a>描述  
 已在處理訊息時切換執行緒。  
  
 訊息處理可能會因為下列原因而暫停：  
  
-   ConcurrencyMode 是單一或可重新進入，且服務正在處理另一個訊息。  
  
-   異動已啟用且服務正在處理另一個異動。  
  
-   同步處理內容不是最新的。
