# Microsoft.Transactions.TransactionBridge.RegisterParticipant
WS\-Atomic 交易通訊協定服務已順利為控制通訊協定註冊參與者。  
  
## 描述  
 當本機交易管理員已成功認可來自參與者的 2pc 通訊協定註冊要求時，則會進行追蹤。
