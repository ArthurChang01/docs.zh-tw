# <a name="systemservicemodelchannelspnrpregisteredaddresses"></a>System.ServiceModel.Channels.PnrpRegisteredAddresses
Pnrp Resolver 已註冊本機節點的位址。  
  
## <a name="description"></a>描述  
 在 PNRP 服務成功註冊本機節點的位址之後，就會發生此追蹤。
