# Microsoft.Transactions.TransactionBridge.RecoveredCoordinatorInvalidMetadata
在協調者修復記錄項目中找到的端點參照，包含無效或不相容的中繼資料。  
  
## 描述  
 系統無法為已修復的協調者登記建立 Proxy。  
  
## 疑難排解  
 如果問題持續存在，請連絡 Microsoft。
