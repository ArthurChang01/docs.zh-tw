# System.ServiceModel.Channels.PnrpResolvedAddresses
Pnrp 解析程式已完成解析作業。  
  
## 描述  
 這個警告追蹤會指出 Pnrp 解析程式找不到任何位址。這項追蹤會在 PNRP 解析程式的解析作業期間發生。
