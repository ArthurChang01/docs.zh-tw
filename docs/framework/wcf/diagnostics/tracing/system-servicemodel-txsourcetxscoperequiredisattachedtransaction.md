# System.ServiceModel.TxSourceTxScopeRequiredIsAttachedTransaction
從 InstanceContext 交易接收到指定之作業的指定交易。  
  
## 描述  
 如果提供給使用者程式碼的交易是由此叫用之前呼叫的 AutoComplete\=false 方法所留下，則會進行追蹤。
