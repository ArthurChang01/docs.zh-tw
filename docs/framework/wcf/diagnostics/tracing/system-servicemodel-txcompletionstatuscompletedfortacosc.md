# System.ServiceModel.TxCompletionStatusCompletedForTACOSC
此特定交易已經在工作階段關閉時完成。  
  
## 描述  
 在目前作用中的工作階段已關閉且 `TransactionAutoCompleteOnSessionClose` 是設定為 `true` 時追蹤。
