# System.ServiceModel.TxCompletionStatusCompletedForError
由於未處理的執行例外狀況，所以指定之作業的指定交易已完成。  
  
## 描述  
 如果在嘗試完成目前交易期間發生錯誤，則會進行追蹤。這個情況會發生在回覆或錯誤傳送至呼叫者之前。  
  
## 疑難排解  
 檢查追蹤訊息，找出例外狀況訊息和任何可執行動作的項目。
