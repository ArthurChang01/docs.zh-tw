# Microsoft.Transactions.TransactionBridge.RegistrationCoordinatorFaulted
WS\-AT 通訊協定服務已從協調器接收到回應登錄訊息的錯誤。  
  
## 描述  
 如果本機 TransactionManager 因為傳回的錯誤而無法向它的上層 TransactionManager 登錄，則會進行追蹤。  
  
## 疑難排解  
 針對傳回的錯誤檢查追蹤訊息。
