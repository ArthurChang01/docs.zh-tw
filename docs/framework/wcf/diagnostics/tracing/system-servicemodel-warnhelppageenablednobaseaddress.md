# <a name="systemservicemodelwarnhelppageenablednobaseaddress"></a>System.ServiceModel.WarnHelpPageEnabledNoBaseAddress
System.ServiceModel.WarnHelpPageEnabledNoBaseAddress  
  
## <a name="description"></a>描述
