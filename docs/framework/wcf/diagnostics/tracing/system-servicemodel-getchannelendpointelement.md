# System.ServiceModel.GetChannelEndpointElement
System.ServiceModel.GetChannelEndpointElement  
  
## 描述  
 在資訊層建構通道處理站時，會發出此追蹤。  該追蹤會提供用戶端進行通訊之端點的說明 \(例如，遠端位址、繫結、合約名稱\)。
