# <a name="systemservicemodelchannelspeerneighbornotfound"></a>System.ServiceModel.Channels.PeerNeighborNotFound
在芳鄰清單上找不到這個芳鄰，因此無法執行該芳鄰上的作業。  
  
## <a name="description"></a>描述  
 當連接至芳鄰或關閉芳鄰時，就會發生這個錯誤。
