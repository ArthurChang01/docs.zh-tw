# System.ServiceModel.ServiceOperationMissingReplyContext
System.ServiceModel.ServiceOperationMissingReplyContext  
  
## 描述  
 雙向作業的 ReplyContext 為 null。追蹤資料會顯示下列訊息：「回覆作業沒有 IRequestContext 可用於回覆。」  
  
 這個追蹤對於在開發期間偵錯十分有用。
