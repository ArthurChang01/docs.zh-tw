# Microsoft.Transactions.TransactionBridge.RegistrationCoordinatorResponseInvalidMetadata
WS\-Atomic Transaction 通訊協定服務從協調器收到 RegisterResponse 訊息，內含的端點參照有無效或不相容的中繼資料。  
  
## 描述  
 當本機交易管理員嘗試向其上層交易管理員註冊，而該上層傳回的 RegisterResponse 訊息內為無效位址時進行追蹤。
