# System.ServiceModel.Channels.PrematureDatagramEof
System.ServiceModel.Channels.PrematureDatagramEof  
  
## 描述  
 已從資料包 \(Datagram\) 通道接收 null 訊息 \(表示通道結尾\)，但通道仍在「已開啟」狀態中。  這表示資料包通道中發生錯誤，而且已永久終止解多工器接收迴圈。
