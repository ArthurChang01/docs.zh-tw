# System.Runtime.Serialization.FactoryTypeNotFound
System.Runtime.Serialization.FactoryTypeNotFound  
  
## 描述  
 還原序列化程式不知道還原序列化期間要使用的類型。  請考慮將在 XML 中指定為處理站類型新增至已知的類型清單，例如藉由使用 KnownTypeAttribute 屬性來新增。
