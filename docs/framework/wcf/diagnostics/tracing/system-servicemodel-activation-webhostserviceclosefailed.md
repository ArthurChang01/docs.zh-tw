# System.ServiceModel.Activation.WebHostServiceCloseFailed
服務無法順利關閉且中止時發生。  
  
## 描述  
 此錯誤碼只會出現在記錄檔中。該錯誤碼通常會指出程式設計錯誤，例如，嘗試在已呼叫中止之後關閉服務。  
  
## 疑難排解  
 請檢查應用程式原始程式碼。
