# <a name="tracing"></a>追蹤
Windows Communication Foundation (WCF) 會提供錯誤監視與分析應用程式檢測和診斷資料。 您可以不使用偵錯工具而改用追蹤，以了解應用程式行為以及發生錯誤的原因。 您也可以將各元件的錯誤與處理相互關聯，以提供端對端的經驗。  
  
 WCF 會輸出下列進行診斷追蹤資料：  
  
- 所有應用程式元件的處理過程里程碑追蹤，例如作業呼叫、程式碼例外狀況、警告和其他重大處理事件。  
  
- 追蹤功能故障時出現的 Windows 錯誤事件。
