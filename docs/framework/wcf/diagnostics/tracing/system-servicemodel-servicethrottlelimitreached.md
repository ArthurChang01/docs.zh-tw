# System.ServiceModel.ServiceThrottleLimitReached
System.ServiceModel.ServiceThrottleLimitReached  
  
## 描述  
 已達到其中一個 MaxConcurrentCalls\/Instances\/Sessions 節流閥，而且有訊息正在佇列中。  
  
 您可以使用這項追蹤，根據預期和實際的生產負載調整節流值。
