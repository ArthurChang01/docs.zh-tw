# Microsoft.Transactions.TransactionBridge.RegisterCoordinator
WS\-Atomic 交易通訊協定服務已順利對控制通訊協定註冊其協調器。  
  
## 描述  
 當本機 TransactionManager 已順利註冊其上層 TransactionManage 時進行追蹤
