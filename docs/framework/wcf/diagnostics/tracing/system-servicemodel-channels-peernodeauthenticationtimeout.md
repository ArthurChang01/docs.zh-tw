# System.ServiceModel.Channels.PeerNodeAuthenticationTimeout
在指定的時間限制內，安全性交換失敗。  
  
## 描述  
 在嘗試建立安全的芳鄰連線時，發生這個追蹤。  
  
## 疑難排解  
 請確認所有芳鄰都有適當的安全性認證，而且會在 1 分鐘內回應所有訊息。
