# System.ServiceModel.Channels.PeerNodeOpenFailed
嘗試開啟芳鄰連線失敗。  
  
## 描述  
 這個追蹤發生在嘗試建立芳鄰連線之後。  可能的原因包括解析程式沒有回應、繫結設定錯誤，或暫時性的網路失敗。  
  
## 疑難排解  
 請確定兩個節點都具備有效的位址，並且正確設定。
