# System.ServiceModel.DroppedAMessage
System.ServiceModel.DroppedAMessage  
  
## 描述  
 已接收回應訊息，但此訊息沒有未處理的要求。[!INCLUDE[indigo1](../../../../../includes/indigo1-md.md)] 執行階段已捨棄訊息。  
  
 這個追蹤可以用來偵錯遺失的訊息。
