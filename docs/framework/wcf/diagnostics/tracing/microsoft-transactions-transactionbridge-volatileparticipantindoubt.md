# Microsoft.Transactions.TransactionBridge.VolatileParticipantInDoubt
WS\-AT 通訊協定服務已收到「已準備」或「重新執行」訊息，但均來自無法辨識的變動參與者。錯誤已傳回給參與者，宣告交易結果不確定。  
  
## 描述  
 在本機交易管理員收到來自其已忘記的變動登記之「已準備」或「重新執行」訊息時追蹤。  
  
## 疑難排解  
 調查來自變動參與者的晚期訊息潛在原因。
