# System.ServiceModel.Channels.PeerFloodedMessageNotPropagated
PeerNode 已接收訊息，但沒有傳播至芳鄰。  
  
## 描述  
 此追蹤發生在處理傳入的湧入訊息期間。  
  
## 疑難排解  
 檢查訊息傳播篩選條件是否存在。
