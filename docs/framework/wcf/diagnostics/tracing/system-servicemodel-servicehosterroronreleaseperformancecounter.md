# System.ServiceModel.ServiceHostErrorOnReleasePerformanceCounter
System.ServiceModel.ServiceHostErrorOnReleasePerformanceCounter  
  
## 描述  
 呼叫 ReleasePerformanceCounters 時，ServiceHost 發生錯誤。釋放效能計數器時，可以使用這項追蹤來判斷錯誤。
