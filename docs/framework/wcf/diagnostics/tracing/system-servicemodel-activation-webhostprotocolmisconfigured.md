# System.ServiceModel.Activation.WebHostProtocolMisconfigured
通訊協定在 Windows Process Activation Service \(WAS\) 中的設定不正確。  
  
## 描述  
 這個錯誤會在傳輸通訊協定設定不正確時發生。  
  
## 疑難排解  
 檢查並修正 web.config 中的傳輸組態參數。
