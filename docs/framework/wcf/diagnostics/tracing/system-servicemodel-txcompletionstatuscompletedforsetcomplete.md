# System.ServiceModel.TxCompletionStatusCompletedForSetComplete
由於呼叫 SetTransactionComplete，已完成指定之作業的指定交易。  
  
## 描述  
 追蹤使用者是否已明確決定要透過 OperationContext 來完成目前交易。
