# System.ServiceModel.ServiceOperationMissingReply
System.ServiceModel.ServiceOperationMissingReply  
  
## 描述  
 要求\/回覆作業不包含回覆訊息。  
  
 這是一種雙向作業；然而，並未包含任何回覆可傳送至用戶端。在開發期間您可以使用這項追蹤進行偵錯。
