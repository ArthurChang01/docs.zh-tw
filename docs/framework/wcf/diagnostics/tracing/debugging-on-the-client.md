# <a name="debugging-on-the-client"></a>用戶端偵錯
若要讓使用者撰寫的用戶端應用程式更容易您[!INCLUDE[indigo2](../../../../../includes/indigo2-md.md)]服務，您可以加入[ \<serviceDebug >](../../../../../docs/framework/configure-apps/file-schema/wcf/servicedebug.md)服務到您的服務組態檔的行為。 這個行為可以用來發行說明頁，並將傳回給用戶端的 SOAP 錯誤詳細資料中的 Managed 例外狀況資訊傳回。
