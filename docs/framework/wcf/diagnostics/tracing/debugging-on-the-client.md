# 用戶端偵錯
若要更方便使用者為您的 [!INCLUDE[indigo2](../../../../../includes/indigo2-md.md)] 服務撰寫用戶端應用程式，您可以將 [\<serviceDebug\>](../../../../../docs/framework/configure-apps/file-schema/wcf/servicedebug.md) 服務行為加入服務的組態檔中。這個行為可以用來發行說明頁，並將傳回給用戶端的 SOAP 錯誤詳細資料中的 Managed 例外狀況資訊傳回。
