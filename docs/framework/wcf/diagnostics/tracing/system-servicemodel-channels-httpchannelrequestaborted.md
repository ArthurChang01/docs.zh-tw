# System.ServiceModel.Channels.HttpChannelRequestAborted
無法透過 HTTP 傳送要求訊息。  
  
## 描述  
 這個警告層級追蹤代表 HTTP 通道要求已中止。  此追蹤裡的其他資訊將顯示原因或傳遞至 Abort 方法的訊息。
