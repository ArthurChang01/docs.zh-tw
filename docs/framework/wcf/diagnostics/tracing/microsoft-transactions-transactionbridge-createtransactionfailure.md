# Microsoft.Transactions.TransactionBridge.CreateTransactionFailure
交易無法建立。  
  
## 描述  
 這個追蹤是在 MSDTC 無法建立交易時產生的。  原因可能是資源不足、記錄檔空間不足或其他錯誤。  
  
## 疑難排解  
 請檢查追蹤訊息內的狀態字串，以判斷是否有任何可執行動作的項目存在。
