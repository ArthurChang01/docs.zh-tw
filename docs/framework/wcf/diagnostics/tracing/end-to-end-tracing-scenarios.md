# 端對端追蹤案例
本節包含的主題將說明使用追蹤的各種不同案例。  
  
 活動追蹤是為 Reply\/Response、OneWay 和 Duplex 案例實作的，這些案例會使用下列各項  
  
-   同步或非同步  
  
-   BasicHttpBinding、netNamedPipeBinding、netTcpBinding、wsHttpBinding 或 netMsmqBinding  
  
-   COM Service  
  
-   使用者活動識別碼傳播
