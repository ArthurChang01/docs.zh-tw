# System.ServiceModel.Channels.MessageSent
System.ServiceModel.Channels.MessageSent  
  
## 描述  
 已透過通道傳送訊息。  
  
 如果在資訊層級或詳細資訊層級啟用 `System.ServiceModel` 追蹤來源，則當傳送 [!INCLUDE[indigo2](../../../../../includes/indigo2-md.md)] 訊息時，會發出此追蹤。必須有這個追蹤，才能在活動圖形檢視中看到訊息相互關聯箭頭。
