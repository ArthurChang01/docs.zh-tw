# Microsoft.Transactions.TransactionBridge.EnlistmentIdentityCheckFailed
傳入訊息的傳送者身分識別與衍生自登記之端點參考的身分識別不符。  
  
## 描述  
 如果訊息安全性驗證失敗的話，則進行追蹤。  
  
## 疑難排解  
 確保所有參與交易的電腦皆已正確設定交易安全性。
