# System.ServiceModel.Channels.HttpsClientCertificateInvalid
用戶端憑證無效。  
  
## 描述  
 這個追蹤表示發現 HTTPS 接聽項視用戶端提供的憑證為無效。HTTPS 接聽項嘗試使用這個憑證驗證用戶端的真實性。如果裝載服務的伺服器無法辨識其憑證授權單位，憑證可能無效。
