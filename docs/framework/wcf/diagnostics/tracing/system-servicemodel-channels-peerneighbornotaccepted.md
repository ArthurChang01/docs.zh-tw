# System.ServiceModel.Channels.PeerNeighborNotAccepted
未接受對等芳鄰。  
  
## 描述  
 當處理與芳鄰的連線時，就會發生此追蹤。可能原因包含不相符的安全性設定、超過的逾時、芳鄰無法接受連線，或暫時性網路問題。  
  
## 疑難排解  
 檢查是否存在不相符的安全性設定、超過的逾時、芳鄰無法接受連線，或暫時性網路問題。
