# <a name="calls-duration"></a>呼叫持續時間
計數器名稱:呼叫持續時間  
  
## <a name="description"></a>說明  
 這個作業呼叫的平均持續時間。 平均持續時間是根據這個方程式來計算:(N1-N0)/(D1-D0)。  
  
> [!WARNING]
> 在非同步 WCF 服務上使用時, 呼叫持續時間計數器一律會傳回-1。
