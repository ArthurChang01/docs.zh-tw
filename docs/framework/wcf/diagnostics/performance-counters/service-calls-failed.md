# <a name="service-calls-failed"></a>服務：失敗的呼叫數
計數器名稱：失敗的呼叫數。  
  
## <a name="description"></a>描述  
 此服務中含有未處理之例外狀況的呼叫數。 每當此服務含有未處理的例外狀況時，此計數器就會遞增。
