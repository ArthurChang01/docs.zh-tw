# <a name="service-calls-faulted"></a>服務：發生錯誤的呼叫數
計數器名稱：發生錯誤的呼叫。  
  
## <a name="description"></a>描述  
 針對已傳回錯誤之此服務的呼叫數。 在 Windows Communication Foundation (WCF) 應用程式，服務方法通訊，使用 SOAP 錯誤訊息的處理錯誤資訊。 SOAP 錯誤是包含在服務作業之中繼資料內的訊息類型，因此會建立錯誤合約，而用戶端可使用該合約來讓其執行作業更強固或更具互動性。 由於 SOAP 錯誤會以 XML 表單的形式傳達給用戶端，所以其互通性很高。
