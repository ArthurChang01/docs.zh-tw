# <a name="queue-dropped-messages"></a>佇列捨棄的訊息
計數器名稱：卸除已排入佇列的訊息。  
  
## <a name="description"></a>描述  
 此服務佇列傳輸所捨棄的訊息數目。  
  
 請參閱[有害訊息處理](https://go.microsoft.com/fwlink/?LinkId=96233)如需何時捨棄訊息的詳細資訊。
