# <a name="queue-dropped-messages"></a>佇列捨棄的訊息
計數器名稱：Queued Messages Dropped。  
  
## <a name="description"></a>描述  
 此服務佇列傳輸所捨棄的訊息數目。  
  
 有關郵件刪除時間的詳細資訊，請參閱[毒消息處理](../../feature-details/poison-message-handling.md)。
