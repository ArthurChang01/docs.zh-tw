# <a name="queue-dropped-messages"></a>佇列捨棄的訊息
計數器名稱：Queued Messages Dropped。  
  
## <a name="description"></a>描述  
 此服務佇列傳輸所捨棄的訊息數目。  
  
 請參閱[有害訊息處理](http://go.microsoft.com/fwlink/?LinkId=96233)如需有關時就會捨棄訊息。
