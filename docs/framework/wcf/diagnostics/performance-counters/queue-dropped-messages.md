# 佇列捨棄的訊息
計數器名稱：Queued Messages Dropped。  
  
## 描述  
 此服務佇列傳輸所捨棄的訊息數目。  
  
 如需何時捨棄訊息的詳細資訊，請參閱[有害訊息處理](http://go.microsoft.com/fwlink/?LinkId=96233) \(本頁面可能為英文\)。
