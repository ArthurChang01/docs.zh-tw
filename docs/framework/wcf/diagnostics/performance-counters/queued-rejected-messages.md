# 佇列拒絕的訊息數
計數器名稱：拒絕的佇列訊息。  
  
## 描述  
 此服務佇列傳輸所拒絕的訊息數量。  
  
 關於訊息拒絕時的詳細資訊，請參閱 [有害訊息處理](http://go.microsoft.com/fwlink/?LinkID=96233) \(本頁面可能為英文\)。
