# <a name="queued-rejected-messages"></a>佇列拒絕的訊息數
計數器名稱：拒絕的佇列訊息。  
  
## <a name="description"></a>描述  
 此服務佇列傳輸所拒絕的訊息數量。  
  
 有關郵件被拒絕時間的詳細資訊，請參閱[毒消息處理](../../feature-details/poison-message-handling.md)。
