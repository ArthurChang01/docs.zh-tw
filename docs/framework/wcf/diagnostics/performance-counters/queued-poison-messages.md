# 佇列的有害訊息
計數器名稱：佇列的有害訊息。  
  
## 描述  
 此服務佇列傳輸標示為有害的訊息數目。  
  
 如需訊息何時標示為有害的詳細資訊，請參閱[有害訊息處理](http://go.microsoft.com/fwlink/?LinkID=96233) \(本頁面可能為英文\)。
