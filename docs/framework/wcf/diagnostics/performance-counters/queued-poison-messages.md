# <a name="queued-poison-messages"></a>佇列的有害訊息
計數器名稱：佇列的有害訊息。  
  
## <a name="description"></a>描述  
 此服務佇列傳輸標示為有害的訊息數目。  
  
 請參閱[有害訊息處理](http://go.microsoft.com/fwlink/?LinkID=96233)如需有關當訊息標示為有害。
