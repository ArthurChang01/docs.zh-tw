# <a name="service-calls-duration"></a>服務： 呼叫持續時間
計數器名稱： 呼叫持續時間  
  
## <a name="description"></a>描述  
 這個服務呼叫的平均持續時間。 平均持續時間是根據此方程式計算而來：(N1-N0)/(D1-D0)。
