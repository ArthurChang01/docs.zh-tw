# <a name="transacted-operations-committed-per-second"></a>每秒認可的交易作業數
計數器名稱：每秒認可的交易作業數。  
  
## <a name="description"></a>描述  
 此服務每秒所認可的異動作業數。  
  
 這個計數器的效能計數器型別是[PERF_COUNTER_COUNTER](https://go.microsoft.com/fwlink/?LinkID=94649)，使用下列公式來計算其值。  
  
 (N 1 - N 0 ) / ( (D 1 -D 0 ) / F)
