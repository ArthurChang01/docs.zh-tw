# <a name="endpoint-calls-failed"></a>端點：失敗的呼叫數
計數器名稱：失敗的呼叫數。  
  
## <a name="description"></a>描述  
 此端點中包含未處理例外狀況的呼叫數。 每當此端點包含未處理的例外狀況時，此計數器就會遞增。
