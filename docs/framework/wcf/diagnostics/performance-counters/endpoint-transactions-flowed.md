# <a name="endpoint-transactions-flowed"></a>端點：流動的異動數
計數器名稱：流動的異動數。  
  
## <a name="description"></a>描述  
 流動至此端點上作業的交易數。 每當傳送給端點的訊息中有交易識別碼存在時，此計數器就會遞增。
