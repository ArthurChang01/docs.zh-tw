# <a name="feedback-and-community"></a>意見與社群
我們非常感謝您的意見與想法的 Windows Communication Foundation (WCF) 或 CardSpace 文件。 在 SDK 中每一頁的底部都有一句「請將您對這個主題的意見傳送至 Microsoft」。 請按一下 [意見] 連結，提供您對於特定主題的看法或建議。  
  
 如果您要尋找的詳細資訊，或想要與有興趣使用 WCF 的人分享技巧，請嘗試[MSDN Windows Communication Foundation 論壇](https://go.microsoft.com/fwlink/?LinkId=186463)。
