# <a name="transport-extensibility"></a>傳輸擴充性
本節包含示範自訂傳輸在 Windows Communication Foundation (WCF) 的範例。  
  
## <a name="in-this-section"></a>本節內容  
 [UDP 啟用](../../../../docs/framework/wcf/samples/udp-activation.md)  
 示範如何擴充[傳輸：UDP](../../../../docs/framework/wcf/samples/transport-udp.md)範例，以支援使用 Windows Process Activation Service (WAS) 處理序啟動。  
  
 [傳輸：自訂跨 UDP 異動](../../../../docs/framework/wcf/samples/transport-custom-transactions-over-udp-sample.md)  
 示範如何擴充 UDP 傳輸範例以支援自訂異動流程，並示範 <xref:System.ServiceModel.Channels.TransactionMessageProperty> 屬性的使用方式。  
  
 [傳輸：WSE 3.0 TCP 互通性](../../../../docs/framework/wcf/samples/transport-wse-3-0-tcp-interoperability.md)  
 此範例示範如何實作 TCP 雙工工作階段做為自訂的 Windows Communication Foundation (WCF) 傳輸。
