# <a name="web"></a>Web
本節包含 Web 裝載的範例。  
  
## <a name="in-this-section"></a>本節內容  
 [進階格式選取](../../../../docs/framework/wcf/samples/advanced-format-selection.md)  
 示範如何擴充 WCF Web HTTP 程式設計模型，以支援新的傳出回應格式。  
  
 [自動格式選取](../../../../docs/framework/wcf/samples/automatic-format-selection.md)  
 示範如何啟用自動格式選取 （XML 或 JSON） 與 WCF Web HTTP 程式設計模型，以及如何明確設定作業碼中的格式。  
  
 [基本 HTTP 服務](../../../../docs/framework/wcf/samples/basic-http-service.md)  
 示範如何實作 HTTP、 RPC 服務，也就是一般稱為"POX"(Plain Old XML) 服務 – 使用 WCF Web HTTP 程式設計模型。  
  
 [基本資源服務](../../../../docs/framework/wcf/samples/basic-resource-service.md)  
 示範如何實作以 HTTP 為基礎的服務使用 WCF Web HTTP 程式設計模型會公開客戶的一組支援擷取、 加入、 刪除和取代作業。  
  
 [AspNetRouteIntegration](../../../../docs/framework/wcf/samples/aspnetrouteintegration.md)  
 示範如何裝載使用 ASP.NET 路由之 WCF Web HTTP 服務。  
  
 [條件式 Get 和 Put](../../../../docs/framework/wcf/samples/conditional-get-and-put.md)  
 示範如何使用新的條件式擷取和更新應用程式開發介面 WCF Web HTTP 程式設計模型。  
  
 [SOAP 和 HTTP 端點](../../../../docs/framework/wcf/samples/soap-and-http-endpoints.md)  
 示範如何實作以 RPC 為基礎的服務，並將它公開在 SOAP 格式和"Plain Old XML"(POX) 格式使用 WCF Web HTTP 程式設計模型。  
  
 [ASP.NET 快取整合](../../../../docs/framework/wcf/samples/aspnet-caching-integration.md)  
 示範如何使用 ASP.NET 輸出快取搭配 WCF Web HTTP 程式設計模型。  
  
 [UriTemplate](../../../../docs/framework/wcf/samples/uritemplate-sample.md)  
 示範與 <xref:System.UriTemplate> 相關的主要概念。  
  
 [UriTemplate 資料表](../../../../docs/framework/wcf/samples/uritemplate-table-sample.md)  
 示範 <xref:System.UriTemplateTable> 類別如何提供可用來處理 <xref:System.UriTemplate> 執行個體集合的字典式關聯表結構。  
  
 [UriTemplate 資料表發送器](../../../../docs/framework/wcf/samples/uritemplate-table-dispatcher-sample.md)  
 示範使用 <xref:System.UriTemplateTable> 建立的基本分派引擎，這是 <xref:System.UriTemplateTable> 類別的常用案例。
