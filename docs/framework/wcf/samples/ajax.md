# <a name="ajax"></a>AJAX
本節包含示範非同步 JavaScript 與 XML 功能的範例。  
  
## <a name="in-this-section"></a>本節內容  
 [JSONP](../../../../docs/framework/wcf/samples/jsonp.md)  
 示範如何在 WCF REST 服務中支援 JSON with Padding (JSONP)。  
  
 [JSON 序列化](../../../../docs/framework/wcf/samples/json-serialization.md)  
 示範如何使用 <xref:System.Runtime.Serialization.Json.DataContractJsonSerializer> 來序列化與還原序列化採用「JavaScript 物件標記法」(JavaScript Object Notation，JSON) 格式的資料。  
  
 [基本 AJAX 服務](../../../../docs/framework/wcf/samples/basic-ajax-service.md)  
 示範如何使用 WCF 來建立基本 ASP.NET Asynchronous JavaScript 與 XML (AJAX) 服務。  
  
 [使用 HTTP POST 的 AJAX 服務](../../../../docs/framework/wcf/samples/ajax-service-using-http-post.md)  
 示範如何使用 WCF 來建立[!INCLUDE[vstecasp](../../../../includes/vstecasp-md.md)]Asynchronous JavaScript and XML (AJAX) 服務會使用 HTTP POST。  
  
 [無組態的 AJAX 服務](../../../../docs/framework/wcf/samples/ajax-service-without-configuration.md)  
 示範如何使用 WCF 來建立基本 ASP.NET Asynchronous JavaScript 與 XML (AJAX) 服務。  
  
 [使用複雜類型的 AJAX 服務](../../../../docs/framework/wcf/samples/ajax-service-using-complex-types-sample.md)  
 示範如何使用 WCF 來建立 ASP.NET Asynchronous JavaScript 與 XML (AJAX) 服務，建立複雜型別的執行個體。  
  
 [使用 JSON 和 XML 的 AJAX 服務](../../../../docs/framework/wcf/samples/ajax-service-with-json-and-xml-sample.md)  
 示範如何使用 WCF 來建立會傳回 JavaScript 物件標記法 (JSON) 或 XML 資料的 Asynchronous JavaScript and XML (AJAX) 服務。
