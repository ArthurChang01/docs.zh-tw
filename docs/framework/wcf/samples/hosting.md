# <a name="hosting"></a>裝載
本節包含示範裝載 Windows Communication Foundation (WCF) 服務的範例。  
  
## <a name="in-this-section"></a>本節內容  
 [Windows 處理序啟用](../../../../docs/framework/wcf/samples/windows-process-activation.md)  
 示範透過網路通訊協定啟動服務。  
  
 [以組態為基礎的啟用](../../../../docs/framework/wcf/samples/configuration-based-activation.md)  
 示範 WCF 服務的啟用，而不需要.svc 檔案。  
  
 [SystemWebRouting 整合範例](../../../../docs/framework/wcf/samples/systemwebrouting-integration-sample.md)  
 示範裝載層與 <xref:System.Web.Routing> 命名空間中的類別整合。  
  
 [ASP.NET 相容性](../../../../docs/framework/wcf/samples/aspnet-compatibility.md)  
 示範如何啟用[!INCLUDE[vstecasp](../../../../includes/vstecasp-md.md)]WCF 中的相容性模式。  
  
 [使用內嵌程式碼的 IIS 裝載](../../../../docs/framework/wcf/samples/iis-hosting-using-inline-code.md)  
 示範如何實作由 Internet Information Services (IIS) 裝載的服務，此時服務程式碼內嵌在 .svc 檔中並且視需要進行編譯。  
  
 [Windows 服務主機](../../../../docs/framework/wcf/samples/windows-service-host.md)  
 示範在 managed Windows 服務中裝載的 WCF 服務。  
  
 [自我裝載](../../../../docs/framework/wcf/samples/self-host.md)  
 示範如何在主控台應用程式中實作自我裝載的服務。
