# Internet Information Service 裝載指示
若要執行由網際網路資訊服務 \(IIS\) 裝載的範例，您必須確定 IIS 已正確安裝並正在執行中。  
  
### 若要在 Windows Server 2008 R2 上安裝 IIS 7.5  
  
1.  選取 \[**伺服器管理員**\] 中的 \[**角色**\]。 按一下 \[**角色摘要**\] 下的 \[**新增角色**\]。  
  
2.  按一下 \[**下一步**\] 以顯示 \[**選取伺服器角色**\] 對話方塊。  
  
3.  從 \[**角色**\] 清單中選取 \[**應用程式伺服器**\]，然後按兩次 \[**下一步**\] 以顯示應用程式伺服器角色的 \[**選取角色服務**\] 對話方塊。  
  
4.  選取 \[Web 伺服器 \(IIS\)\] 核取方塊。  如果提示您安裝其他的角色服務和功能，請按一下 \[**新增所需的功能**\]。  按兩次 \[**下一步**\] 以顯示 Web 伺服器 \(IIS\) 角色的 \[**選取角色服務**\] 對話方塊。  
  
5.  依序展開 \[**管理工具**\] 和 \[**IIS 6 管理相容性**\]。  選取 \[**IIS 6 指令碼工具**\]。  如果提示您安裝其他的角色服務和功能，請按一下 \[**新增所需的角色服務**\]。  按 \[**下一步**\]。  
  
6.  如果選項的摘要正確，請按一下 \[**安裝**\]。  
  
7.  當安裝完成時，按一下 \[**關閉**\]。  
  
### 若要在 Windows 7 上安裝 IIS 7.5 版  
  
1.  按一下 \[**開始**\]，然後按一下 \[**控制台**\]。  
  
2.  開啟 \[**程式**\] 群組。  
  
3.  按一下 \[**程式和功能**\] 下的 \[**開啟或關閉 Windows 功能**\]。  
  
4.  \[**使用者帳戶控制**\] 對話方塊隨即顯示。  按一下 \[**繼續**\]。  
  
5.  這時會顯示 \[**Windows 功能**\]。  展開標記為 \[**網際網路資訊服務**\] 的項目。  
  
6.  展開標記為 \[**全球資訊網服務**\] 的項目。  
  
7.  展開標記為 \[**應用程式開發功能**\] 的項目。  
  
8.  請確定已選取下列項目：  
  
    1.  **.NET 擴充性**  
  
    2.  **ASP.NET**  
  
    3.  **ISAPI 延伸**  
  
    4.  **ISAPI 篩選器**  
  
9. 在標記為 \[**全球資訊網服務**\] 的項目下，展開 \[**一般 HTTP 功能**\]。  
  
10. 確定已選取 \[**靜態內容**\]。  
  
11. 在標記為 \[**全球資訊網服務**\] 的項目下，展開 \[**安全性**\]。  
  
12. 確定已選取 \[**Windows 驗證**\]。  
  
13. 展開 \[**Internet Information Services**\] 目錄下，標記為 \[**Web 管理工具**\] 的項目，然後選取 \[**IIS 管理主控台**\]。  
  
14. 展開標記為 \[**IIS 6 管理相容性**\] 的項目，然後選取 \[**IIS 6 指令碼工具**\]。  
  
15. 展開 \[**Internet Information Services**\] 目錄下，標記為 \[**Microsoft .NET Framework 3.5.1**\] 的項目，然後選取 \[**Windows Communication Foundation Http 啟動**\]。  
  
16. 按一下 \[**確定**\]。

### 在 Windows Server 2003 上安裝 IIS 6.0 版  
  
1.  從 \[管理您的伺服器\]，按一下 \[新增或移除角色\]，然後按 \[下一步\]。  
  
2.  在 \[**伺服器角色**\] 清單中選取 \[**應用程式伺服器 \(IIS，ASP.NET\)**\]，然後按一下 \[**下一步**\]。  
  
3.  選取 \[啟用 ASP.NET\] 核取方塊，然後按 \[下一步\]。  
  
4.  如果選項的摘要正確，請按一下 \[下一步\]。  
  
### 若要在已安裝 Service Pack 2 和 Service Pack 3 的 Windows XP 上安裝 IIS 5.1 版  
  
1.  在 \[控制台\] 中，按一下 \[**新增或移除程式**\]。  
  
2.  按一下 \[**新增或移除程式**\] 對話方塊中的 \[**新增\/移除 Windows 元件**\]。  
  
3.  在 \[**Windows 元件精靈**\] 中，選取 \[**網際網路資訊服務 \(IIS\)**\] 核取方塊，然後按一下 \[**下一步**\]。  
  
4.  如果顯示 \[**需要檔案**\] 對話方塊，請插入作業系統安裝光碟，接著瀏覽至 i386 資料夾，然後按一下 \[**確定**\]。  
  
5.  完成安裝時，請按一下 \[**完成**\]。  
  
6.  關閉 \[**新增或移除程式**\] 對話方塊，然後關閉 \[**控制台**\]。  
  
### 確認 IIS 和 ASP.NET 安裝  
  
1.  將本主題結尾處的 HTML 檔儲存在根 \\InetPub\\wwwroot 目錄，並重新命名為 Default.aspx。  
  
2.  開啟瀏覽器視窗。  
  
3.  在位址方塊中輸入 `http://localhost/Default.aspx`，然後按 ENTER 鍵。  
  
4.  這時應該會出現包含文字 "Hello World" 的網頁。  
  
> [!NOTE]
>  每次您安裝新版本的 [!INCLUDE[dnprdnshort](../../../../includes/dnprdnshort-md.md)] 時，就必須重新註冊 aspnet\_isapi 做為 IIS 的 Web 服務延伸。  若要這樣做，請發出 `aspnet_regiis –I –enable` 命令。
