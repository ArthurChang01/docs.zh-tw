# 訊息佇列整合
本節包含示範訊息佇列與 [!INCLUDE[indigo1](../../../../includes/indigo1-md.md)] 間之整合的範例。  
  
## 在本節中  
 [訊息佇列至 Windows Communication Foundation](../../../../docs/framework/wcf/samples/message-queuing-to-wcf.md)  
 示範訊息佇列 \(MSMQ\) 應用程式如何將 MSMQ 訊息傳送至 [!INCLUDE[indigo2](../../../../includes/indigo2-md.md)] 服務。  
  
 [自訂 Demux](../../../../docs/framework/wcf/samples/custom-demux.md)  
 示範 MSMQ 訊息標頭如何對應至不同的服務作業，讓使用 <xref:System.ServiceModel.MsmqIntegration.MsmqIntegrationBinding> 的 [!INCLUDE[indigo2](../../../../includes/indigo2-md.md)] 服務不會受限於使用一項服務作業。  
  
 [Windows Communication Foundation 至訊息佇列](../../../../docs/framework/wcf/samples/wcf-to-message-queuing.md)  
 示範 [!INCLUDE[indigo2](../../../../includes/indigo2-md.md)] 應用程式如何將訊息傳送至訊息佇列 \(MSMQ\) 應用程式。  
  
 [訊息相互關聯](../../../../docs/framework/wcf/samples/message-correlation.md)  
 示範訊息佇列 \(MSMQ\) 應用程式如何傳送 MSMQ 訊息至 [!INCLUDE[indigo2](../../../../includes/indigo2-md.md)] 服務，以及要求\/回應案例中的寄件者與接收者應用程式之間的訊息如何相互關聯。
