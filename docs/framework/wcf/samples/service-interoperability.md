# <a name="service-interoperability"></a>服務互通性
本節包含示範 Windows Communication Foundation (WCF) 與其他服務技術之間的互動的範例。  
  
## <a name="in-this-section"></a>本節內容  
 [利用 COM 用戶端使用 WCF Moniker](../../../../docs/framework/wcf/samples/using-the-wcf-moniker-with-com-clients.md)  
 示範如何使用 WCF 服務 moniker 將 Web 服務整合至 COM 架構開發環境，例如 Microsoft Office Visual Basic for Applications (Office VBA) 或 Visual Basic 6.0。  
  
 [含 WCF 服務的 ASMX 用戶端](../../../../docs/framework/wcf/samples/asmx-client-with-a-wcf-service.md)  
 示範如何建立使用 WCF 服務，然後再從非 WCF 用戶端，例如 ASMX 用戶端存取服務。
