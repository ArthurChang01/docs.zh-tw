# <a name="custom-binding"></a>自訂繫結
本節包含示範使用端點項目之自訂繫結屬性的範例。

 [命令式自訂繫結](../../../../docs/framework/wcf/samples/custom-binding-imperative.md)  
 示範如何撰寫命令式程式碼來定義和使用自訂繫結，而不使用組態檔或產生 WCF 用戶端。
