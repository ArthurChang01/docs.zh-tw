# 路由服務
本節包含示範如何使用 [!INCLUDE[indigo1](../../../../includes/indigo1-md.md)] 路由器的範例。  
  
## 在本節中  
 [路由服務的 Hello World](../../../../docs/framework/wcf/samples/hello-world-with-the-routing-service.md)  
 示範如何使用 [!INCLUDE[indigo2](../../../../includes/indigo2-md.md)] 路由器。  
  
 [橋接及錯誤處理](../../../../docs/framework/wcf/samples/bridging-and-error-handling.md)  
 示範如何將 [!INCLUDE[indigo2](../../../../includes/indigo2-md.md)] 路由服務用於使用不同繫結之用戶端和服務間的橋接通訊。  
  
 [進階篩選](../../../../docs/framework/wcf/samples/advanced-filters.md)
