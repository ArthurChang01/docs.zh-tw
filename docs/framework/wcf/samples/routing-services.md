# <a name="routing-services"></a>路由服務
本節包含示範如何使用 Windows Communication Foundation (WCF) 路由器的範例。  
  
## <a name="in-this-section"></a>本節內容  
 [路由服務的 Hello World](../../../../docs/framework/wcf/samples/hello-world-with-the-routing-service.md)  
 示範如何使用 WCF 路由器。  
  
 [橋接和錯誤處理](../../../../docs/framework/wcf/samples/bridging-and-error-handling.md)  
 示範 WCF 路由服務如何用於通訊的用戶端和服務使用不同繫結之間的橋接。  
  
 [進階篩選](../../../../docs/framework/wcf/samples/advanced-filters.md)  
 示範如何調整標準的 WCF 計算機範例，以便使用路由服務進行通訊。  
  
 [動態重新組態](../../../../docs/framework/wcf/samples/dynamic-reconfiguration.md)  
 示範如何在執行階段以動態方式重新設定路由服務。  
  
 [進階錯誤處理](../../../../docs/framework/wcf/samples/advanced-error-handling.md)
