# 發行之權杖的訊息安全性
發行的權杖用戶端可在 [!INCLUDE[indigo1](../../../../includes/indigo1-md.md)] 中啟用聯合安全性案例。  
  
 聯合安全性可在獨立網域之間建立信任關係。每個網域都會集中進行安全性權杖服務中的驗證與授權作業。當您在兩個此類服務之間建立起信任關係時，每個網域的成員都能使用另一個網域的服務。  
  
 如需聯合案例的詳細說明，請參閱[聯合與發行的權杖](../../../../docs/framework/wcf/feature-details/federation-and-issued-tokens.md)。
