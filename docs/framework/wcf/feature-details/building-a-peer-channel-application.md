# <a name="building-a-peer-channel-application"></a>建置對等通道應用程式

## <a name="in-this-section"></a>本節內容  
 [將 NetTcpBinding 應用程式轉換為對等通道應用程式](../../../../docs/framework/wcf/feature-details/converting-a-nettcpbinding-application-to-a-peer-channel-application.md)

 [限制訊息散發](../../../../docs/framework/wcf/feature-details/limiting-message-distribution.md)
