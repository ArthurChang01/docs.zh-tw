# WCF 新聞訂閱
[!INCLUDE[indigo1](../../../../includes/indigo1-md.md)] 提供的支援可讓您輕鬆使用 Atom、RSS 或其他自訂格式的新聞訂閱摘要，並且可讓您讀取和建立這些摘要，以及在服務端點上公開 \(Expose\) 這些摘要。  本節的各主題會詳細描述新聞訂閱的程式設計模型。  
  
## 在本節中  
 [WCF 新聞訂閱概觀](../../../../docs/framework/wcf/feature-details/wcf-syndication-overview.md)  
 提供 [!INCLUDE[indigo2](../../../../includes/indigo2-md.md)] 提供之新聞訂閱支援的概觀。  
  
 [新聞訂閱架構](../../../../docs/framework/wcf/feature-details/architecture-of-syndication.md)  
 描述物件模型中的類別以及新聞訂閱的可擴充性。  
  
 [WCF 新聞訂閱物件模型對應到 Atom 和 RSS 的方式](../../../../docs/framework/wcf/feature-details/how-the-wcf-syndication-object-model-maps-to-atom-and-rss.md)  
 描述摘要在 WCF 新聞訂閱物件模型中的呈現方式，以及轉換至 RSS 和 Atom 摘要的方法。  
  
 [HOW TO：建立基本 RSS 摘要](../../../../docs/framework/wcf/feature-details/how-to-create-a-basic-rss-feed.md)  
 顯示如何建立可產生基本 RSS 摘要的服務。  
  
 [HOW TO：建立基本 Atom 摘要](../../../../docs/framework/wcf/feature-details/how-to-create-a-basic-atom-feed.md)  
 顯示如何建立可產生基本 ATOM 摘要的服務。  
  
 [HOW TO：將摘要同時公開為 Atom 和 RSS](../../../../docs/framework/wcf/feature-details/how-to-expose-a-feed-as-both-atom-and-rss.md)  
 顯示如何建立可產生 ATOM 和 RSS 相同摘要的服務。  
  
 [新聞訂閱擴充性](../../../../docs/framework/wcf/feature-details/syndication-extensibility.md)
