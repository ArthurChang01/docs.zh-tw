# <a name="integrating-with-com-applications"></a>整合 COM 應用程式
Windows Communication Foundation (WCF) 服務使用 WCF 服務 moniker 可以直接整合您現有的程式碼中。 服務 Moniker 可以從多種 COM 架構開發環境中使用，例如 Office VBA、Visual Basic 6.0 或 Visual C++ 6.0。
