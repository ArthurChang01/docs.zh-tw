## <a name="multiple-base-addresses"></a><span data-ttu-id="3378f-109">多個基底位址</span><span class="sxs-lookup"><span data-stu-id="3378f-109">Multiple Base Addresses</span></span>

 <span data-ttu-id="3378f-113">當裝載在 IIS 下的 WCF 服務，IIS 會為您的虛擬目錄，包含應用程式的 URI 為基礎建立一個基底位址。</span><span class="sxs-lookup"><span data-stu-id="3378f-113">When hosting a WCF service under IIS, IIS creates one base address for you based on the URI to the virtual directory that contains the application.</span></span> <span data-ttu-id="3378f-114">您可以使用 Internet Information Services 管理員，加入其他使用相同通訊協定的基底位址，以便在網站中加入一個或多個繫結。</span><span class="sxs-lookup"><span data-stu-id="3378f-114">You can add additional base addresses that use the same protocol by using Internet Information Services Manager to add one or more bindings to your Web site.</span></span> <span data-ttu-id="3378f-115">為每個繫結指定通訊協定 (HTTP 或 HTTPS)、IP 位址、連接埠和主機名稱。</span><span class="sxs-lookup"><span data-stu-id="3378f-115">For each binding specify a protocol (HTTP or HTTPS), an IP address, a port, and a host name.</span></span> <span data-ttu-id="3378f-116">如需使用 Internet Information Services 管理員的詳細資訊，請參閱 < [IIS 管理員 (IIS 7)](https://go.microsoft.com/fwlink/?LinkId=164057)。</span><span class="sxs-lookup"><span data-stu-id="3378f-116">For more information about using Internet Information Services Manager, see [IIS Manager (IIS 7)](https://go.microsoft.com/fwlink/?LinkId=164057).</span></span> <span data-ttu-id="3378f-117">如需新增繫結至站台的詳細資訊，請參閱[建立網站 (IIS 7)](https://go.microsoft.com/fwlink/?LinkId=164060)</span><span class="sxs-lookup"><span data-stu-id="3378f-117">For more information about adding bindings to a site, see [Create a Web Site (IIS 7)](https://go.microsoft.com/fwlink/?LinkId=164060)</span></span>
