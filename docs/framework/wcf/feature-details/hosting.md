# <a name="hosting"></a>裝載
本節中各主題描述服務裝載。 服務可以裝載網際網路資訊服務 (IIS)、 Windows Process Activation Service (WAS)、 Windows Server AppFabric、 Windows 服務，或由受管理的應用程式，這個選項通常稱為*自我裝載*。  
  
 從未受信任的主機執行服務或任何延伸項目都會破壞安全性，這點請您務必注意。
