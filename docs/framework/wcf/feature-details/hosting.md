# 裝載
本節中各主題描述服務裝載。服務可由 Internet Information Services \(IIS\)、Windows 處理序啟用服務 \(WAS\)、Windows Server AppFabric、Windows 服務或由 Managed 應用程式來裝載，而最後這個選項通常稱為「*自我裝載*」\(Self Hosting\)。  
  
 從未受信任的主機執行服務或任何延伸項目都會破壞安全性，這點請您務必注意。
