# <a name="metadata"></a>中繼資料
Windows Communication Foundation (WCF) 提供基礎結構的匯出、 發行、 擷取與匯入服務中繼資料。 WCF 服務會使用中繼資料，說明如何與服務的端點互動，以便 Svcutil.exe 之類的工具，可以自動產生用戶端程式碼，來存取服務。
