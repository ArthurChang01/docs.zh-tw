# <a name="servicemodel-attributes-and-servicedescription-reference"></a><span data-ttu-id="0a2b4-102">ServiceModel 屬性與 ServiceDescription 參考</span><span class="sxs-lookup"><span data-stu-id="0a2b4-102">ServiceModel Attributes and ServiceDescription Reference</span></span>
<span data-ttu-id="0a2b4-103">*描述樹狀目錄*是類型的階層架構 (開頭為<xref:System.ServiceModel.Description.ServiceDescription?displayProperty=nameWithType>類別)，合起來描述服務的各個層面。</span><span class="sxs-lookup"><span data-stu-id="0a2b4-103">The *description tree* is the hierarchy of types (starting with the <xref:System.ServiceModel.Description.ServiceDescription?displayProperty=nameWithType> class) that together describe every aspect of a service.</span></span> <span data-ttu-id="0a2b4-104">Windows Communication Foundation (WCF) 會使用描述樹狀目錄來建置有效的服務執行階段，以發佈 Web 服務描述語言 (WSDL)、 XML 結構描述定義語言 (XSD) 和用戶端可用來服務有關的原則判斷提示 （中繼資料）若要連線，並使用服務，以及產生各種程式碼和組態檔值的表示法描述樹狀結構。</span><span class="sxs-lookup"><span data-stu-id="0a2b4-104">Windows Communication Foundation (WCF) uses a description tree to build a valid service runtime, to publish Web Services Description Language (WSDL), XML Schema definition language (XSD), and policy assertions (metadata) about the service that clients can use to connect to and use the service, and to generate various code and configuration file representations of the description tree values.</span></span>  
  
 <span data-ttu-id="0a2b4-105">本主題說明如何從服務合約取得與合約有關的屬性，以及如何實作這些屬性並新增到描述樹狀目錄中。</span><span class="sxs-lookup"><span data-stu-id="0a2b4-105">This topic describes how contract-related properties are obtained from the service contract, and how they are implemented and added to the description tree.</span></span> <span data-ttu-id="0a2b4-106">在某些情況下，屬性值會轉換成行為屬性，然後行為會插入描述樹狀目錄中。</span><span class="sxs-lookup"><span data-stu-id="0a2b4-106">In some cases, attribute values are converted into behavior properties and behavior is then inserted into the description tree.</span></span> <span data-ttu-id="0a2b4-107">如需有關如何將描述樹狀目錄值轉換成中繼資料的詳細資訊，請參閱[ServiceDescription 與 WSDL 參考](../../../../docs/framework/wcf/feature-details/servicedescription-and-wsdl-reference.md)。</span><span class="sxs-lookup"><span data-stu-id="0a2b4-107">For more information about how the description tree values are converted into metadata, see [ServiceDescription and WSDL Reference](../../../../docs/framework/wcf/feature-details/servicedescription-and-wsdl-reference.md).</span></span>
