## <a name="untyped-message-programming"></a><span data-ttu-id="2565c-115">不具型別的訊息程式設計</span><span class="sxs-lookup"><span data-stu-id="2565c-115">Untyped Message Programming</span></span>  
 <span data-ttu-id="2565c-116">直接使用不具型別的訊息物件時，您必須明確設定不具型別訊息上的屬性，以將其序列化為 JSON。</span><span class="sxs-lookup"><span data-stu-id="2565c-116">When working directly with untyped Message objects, you must explicitly set the properties on the untyped message to serialize it as JSON.</span></span> <span data-ttu-id="2565c-117">下列程式碼片段示範如何執行這項操作。</span><span class="sxs-lookup"><span data-stu-id="2565c-117">The following code snippet shows how to do this.</span></span>  
  
```  
 Message response = Message.CreateMessage(  
                  MessageVersion.None,    // No SOAP message version  
                             "*",                     // SOAP action, ignored since this is JSON  
                             "Response string: JSON format specified", // Message body  
                             new DataContractJsonSerializer(typeof(string))); // Specify DataContractJsonSerializer  
      response.Properties.Add( WebBodyFormatMessageProperty.Name,   
                    new WebBodyFormatMessageProperty(WebContentFormat.Json)); // Use JSON format  
```
