# <a name="wcf-and-net-framework-client-profile"></a>WCF 和 .NET Framework Client Profile
[!INCLUDE[arrowhead](../../../includes/arrowhead-md.md)] 是專為不需要整個架構的用戶端的完整.NET framework 的輕量型版本。 此用戶端架構並未完整支援 Windows Communication Foundation。  
  
## <a name="wcf-features-supported-by-the-net-framework-client-profile"></a>支援的.NET Framework Client Profile 的 WCF 功能  
 下列為 [!INCLUDE[arrowhead](../../../includes/arrowhead-md.md)] 支援的 Windows Communication Foundation 功能：  
  
- 除了 Cardspace 和 Web 裝載之外，支援所有的 WCF。  
  
- 支援遠端 TCP/IP 通道。  
  
- 不支援 ASMX (Web 服務)。
