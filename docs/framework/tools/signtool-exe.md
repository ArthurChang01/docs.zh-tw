## <a name="examples"></a><span data-ttu-id="ed956-305">範例</span><span class="sxs-lookup"><span data-stu-id="ed956-305">Examples</span></span>  
 <span data-ttu-id="ed956-306">下列命令會將目錄檔 MyCatalogFileName.cat 加入至系統元件和驅動程式資料庫。</span><span class="sxs-lookup"><span data-stu-id="ed956-306">The following command adds the catalog file MyCatalogFileName.cat to the system component and driver database.</span></span> <span data-ttu-id="ed956-307">如有必要防止取代名為 `/u` 的現有目錄檔案，`MyCatalogFileName.cat` 選項會產生一個唯一的名稱。</span><span class="sxs-lookup"><span data-stu-id="ed956-307">The `/u` option generates a unique name if necessary to prevent replacing an existing catalog file named `MyCatalogFileName.cat`.</span></span>  
  
```  
signtool catdb /v /u MyCatalogFileName.cat  
```  
  
 <span data-ttu-id="ed956-308">下列命令會使用最佳憑證自動簽署檔案。</span><span class="sxs-lookup"><span data-stu-id="ed956-308">The following command signs a file automatically by using the best certificate.</span></span>  
  
```  
signtool sign /a MyFile.exe  
```  
  
 <span data-ttu-id="ed956-309">下列命令使用儲存在受密碼保護之 PFX 檔中的憑證存放區，對檔案進行數位簽署。</span><span class="sxs-lookup"><span data-stu-id="ed956-309">The following command digitally signs a file by using a certificate stored in a password-protected PFX file.</span></span>

  
 <span data-ttu-id="ed956-310">下列命令會對檔案進行數位簽署和時間戳記。</span><span class="sxs-lookup"><span data-stu-id="ed956-310">The following command digitally signs and time-stamps a file.</span></span> <span data-ttu-id="ed956-311">用於簽署檔案的憑證存放在 PFX 檔中。</span><span class="sxs-lookup"><span data-stu-id="ed956-311">The certificate used to sign the file is stored in a PFX file.</span></span>

  
 <span data-ttu-id="ed956-312">下列命令會使用位於主旨名稱為 `My` 之 `My Company Certificate` 存放區中的憑證來簽署檔案。</span><span class="sxs-lookup"><span data-stu-id="ed956-312">The following command signs a file by using a certificate located in the `My` store that has a subject name of `My Company Certificate`.</span></span>

  
 <span data-ttu-id="ed956-313">下列命令會簽署 ActiveX 控制項，並在提示使用者安裝該控制項時，提供由 Internet Explorer 顯示的資訊。</span><span class="sxs-lookup"><span data-stu-id="ed956-313">The following command signs an ActiveX control and provides information that is displayed by Internet Explorer when the user is prompted to install the control.</span></span>

 <span data-ttu-id="ed956-314">下列命令會為已數位簽署的檔案加上時間戳記。</span><span class="sxs-lookup"><span data-stu-id="ed956-314">The following command time-stamps a file that has already been digitally signed.</span></span>
