# 4207 - MaximumRetriesExceededForSqlCommand
## 屬性  
  
|||  
|-|-|  
|ID|4207|  
|關鍵字|Quota、WFInstanceStore|  
|層級|資訊|  
|通道|Microsoft\-Windows\-Application Server\-Applications\/Debug|  
  
## 描述  
 表示 SQL 提供者要放棄重試 SQL 命令，因為已經執行了重試次數上限。  
  
## 訊息  
 放棄重試 SQL 命令，因為已經執行了重試次數上限。
