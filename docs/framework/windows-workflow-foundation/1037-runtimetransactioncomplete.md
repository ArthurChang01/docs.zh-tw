# <a name="1037---runtimetransactioncomplete"></a>1037 - RuntimeTransactionComplete
## <a name="properties"></a>屬性  
  
|||  
|-|-|  
|識別碼|1037|  
|關鍵字|WFRuntime|  
|層級|詳細資訊|  
|通道|Microsoft-Windows-Application Server-Applications/Debug|  
  
## <a name="description"></a>描述  
 表示執行階段異動已完成。  
  
## <a name="message"></a>訊息  
 執行階段異動已完成，狀態為 '%1'。
