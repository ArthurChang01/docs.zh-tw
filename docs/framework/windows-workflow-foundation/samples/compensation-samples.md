# <a name="compensation-samples"></a>補償 (範例)
本節包含相關的補償中 Windows Workflow Foundation (WF) 範例。  
  
## <a name="in-this-section"></a>本節內容  
 [可補償的活動](../../../../docs/framework/windows-workflow-foundation/samples/compensable-activity-sample.md)  
 示範如何使用 <xref:System.Activities.Statements.CompensableActivity> 活動，定義一般執行期間針對給定動作所執行的工作，以及稍後如有需要，為了補償該動作所必須執行的工作。  
  
 [自訂補償](../../../../docs/framework/windows-workflow-foundation/samples/custom-compensation-sample.md)  
 示範如何使用 <xref:System.Activities.Statements.CompensableActivity> 及其補償處理常式，定義自訂的補償邏輯。  
  
 [組態](../../../../docs/framework/windows-workflow-foundation/samples/confirmation.md)  
 示範四個有關 <xref:System.Activities.Statements.CompensableActivity> 和確認用法的一般案例。  
  
 [可補償活動上的取消處理常式](../../../../docs/framework/windows-workflow-foundation/samples/cancellation-handler-on-compensable-activity.md)  
 示範如何在 <xref:System.Activities.Statements.CompensableActivity> 使用取消處理常式。
