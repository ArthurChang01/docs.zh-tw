# <a name="tracking"></a>追蹤
本節包含的範例會示範 Windows Workflow Foundation （WF）中的工作流程追蹤。

## <a name="in-this-section"></a>本章節內容
 [自訂追蹤](custom-tracking.md)示範如何建立自訂追蹤參與者，以及將追蹤資料的內容寫入主控台。

 將[事件追蹤到 Windows 中的事件追蹤](tracking-events-into-event-tracing-in-windows.md)示範如何在工作流程服務上啟用 [!INCLUDE[wf1](../../../../includes/wf1-md.md)] 追蹤，並在 Windows 事件追蹤（ETW）中發出追蹤事件。

 [SQL 追蹤](sql-tracking.md)示範如何撰寫自訂的 SQL 追蹤參與者，將追蹤記錄寫入至 SQL 資料庫。
