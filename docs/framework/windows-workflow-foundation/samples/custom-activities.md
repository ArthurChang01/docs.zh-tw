# <a name="custom-activities"></a>自訂活動
本節包含示範如何建立自訂活動中 Windows Workflow Foundation (WF) 範例。  
  
## <a name="in-this-section"></a>本節內容  
 [Code-Bodied](../../../../docs/framework/windows-workflow-foundation/samples/code-bodied.md)  
 包含以程式碼宣告方式建立的自訂活動範例。  
  
 [複合](../../../../docs/framework/windows-workflow-foundation/samples/composite.md)  
 包含由其他活動所組成的自訂活動範例。  
  
 [自訂活動設計工具](../../../../docs/framework/windows-workflow-foundation/samples/custom-activity-designers.md)  
 包含使用自訂設計工具以顯示在工作流程設計工具中的活動範例。  
  
 [驗證](../../../../docs/framework/windows-workflow-foundation/samples/validation.md)  
 包含示範設計階段活動驗證的範例。
