# <a name="code-bodied"></a>Code-Bodied
本節包含以程式碼宣告方式建立的自訂活動範例。  
  
## <a name="in-this-section"></a>本章節內容  
 [書籤](../../../../docs/framework/windows-workflow-foundation/samples/bookmarks.md)  
 這個範例示範如何撰寫自訂活動，建立可接收外部輸入的書籤。  
  
 [使用原生活動的自訂複合](../../../../docs/framework/windows-workflow-foundation/samples/custom-composite-using-native-activity.md)  
 這個範例示範如何撰寫 <xref:System.Activities.NativeActivity>，它會排程其他 <xref:System.Activities.Activity> 物件來控制工作流程的執行流程。  
  
 [執行屬性](../../../../docs/framework/windows-workflow-foundation/samples/execution-properties.md)  
 這個範例示範如何使用定義及使用自訂活動中的執行屬性。  
  
 [公開及叫用 ActivityActions](../../../../docs/framework/windows-workflow-foundation/samples/exposing-and-invoking-activityactions.md)  
 這個範例示範如何開發有 <xref:System.Activities.ActivityAction> 的自訂活動。  
  
 [在活動中使用 AsyncCodeActivityContext](../../../../docs/framework/windows-workflow-foundation/samples/using-asyncoperationcontext-in-an-activity-sample.md)  
 這個範例示範如何開發自訂的 <xref:System.Activities.CodeActivity>，該活動會使用 <xref:System.Activities.AsyncCodeActivityContext> 在工作流程之外以非同步方式執行工作。 
  
 [Hello World 自訂活動](../../../../docs/framework/windows-workflow-foundation/samples/hello-world-custom-activity.md)  
 這個範例示範 [!INCLUDE[wf](../../../../includes/wf-md.md)] 的數個關鍵功能，包括如何建立簡單的自訂活動。  
  
 [動態引數](../../../../docs/framework/windows-workflow-foundation/samples/dynamic-arguments.md)  
 這個範例示範如何實作活動，由活動消費者 (而不是活動作者) 定義其引數。
