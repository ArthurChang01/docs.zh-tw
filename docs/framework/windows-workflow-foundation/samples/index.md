# Windows Workflow (WF) 範例
這些範例會針對 [!INCLUDE[wf](../../../../includes/wf-md.md)] 的各種層面提供指示。  
  
 這些範例可從 [Microsoft 下載中心](http://go.microsoft.com/fwlink/?LinkId=150780)下載。  
  
> [!NOTE]
>  這些範例是使用 [!INCLUDE[vs2010](../../../../includes/vs2010-md.md)] 和 [!INCLUDE[netfx40_short](../../../../includes/netfx40-short-md.md)] 建立的，但是與 [!INCLUDE[vs_current_long](../../../../includes/vs-current-long-md.md)] 和 [!INCLUDE[netfx_current_short](../../../../includes/netfx-current-short-md.md)] 相容。其他以 [!INCLUDE[netfx_current_short](../../../../includes/netfx-current-short-md.md)] 撰寫的 [!INCLUDE[wf2](../../../../includes/wf2-md.md)] 範例可以在 [Codeplex](http://aka.ms/WF45Samples) 中找到。  
  
## 在本節中  
 [應用程式](../../../../docs/framework/windows-workflow-foundation/samples/application.md)  
 提供與工作流程應用程式相關的範例。  
  
 [基本](../../../../docs/framework/windows-workflow-foundation/samples/basic.md)  
 提供示範基本 [!INCLUDE[wf](../../../../includes/wf-md.md)] 功能的範例。
