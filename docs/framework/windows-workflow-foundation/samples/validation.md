# <a name="validation"></a>驗證
本節包含示範活動驗證的範例。  
  
## <a name="in-this-section"></a>本節內容  
 [外部活動驗證](../../../../docs/framework/windows-workflow-foundation/samples/external-activity-validation.md)  
 這個範例示範如何將驗證邏輯加入至不是您撰寫的內建活動。  
  
 [基本驗證](../../../../docs/framework/windows-workflow-foundation/samples/basic-validation.md)  
 示範 `CreateProduct` 活動，驗證其 `Cost` 引數小於或等於其 `Price` 引數。  
  
 [OverloadGroups](../../../../docs/framework/windows-workflow-foundation/samples/overloadgroups.md)  
 示範 `CreateLocation` 活動，這個活動有必要引數和選擇性引數，可讓使用者選擇提供兩組不同引數的其中一組。  
  
 [活動關聯性驗證](../../../../docs/framework/windows-workflow-foundation/samples/activity-relationships-validation.md)  
 示範如何驗證活動之間的關聯性。
