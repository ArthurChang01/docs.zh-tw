# 4213 - RunnableInstancesDetectionError
## 屬性  
  
|||  
|-|-|  
|ID|4213|  
|關鍵字|WFInstanceStore|  
|層級|錯誤|  
|通道|Microsoft\-Windows\-Application Server\-Applications\/Debug|  
  
## 描述  
 偵測可執行的執行個體失敗，因為發生例外狀況  
  
## 訊息  
 偵測可執行的執行個體失敗，因為發生下列例外狀況
