# <a name="2025---internalcachemetadatastop"></a>2025 - InternalCacheMetadataStop
## <a name="properties"></a>屬性  
  
|||  
|-|-|  
|ID|2025|  
|關鍵字|WFRuntime|  
|層級|詳細資訊|  
|通道|Microsoft-Windows-Application Server-Applications/Debug|
