# 連接字串與連接字串名稱
**Connection String**屬性會指定 SQL 工作流程執行個體存放區應用於連接持續性資料庫的連接字串。這個參數是選擇性參數。**Connection String Name**屬性會指定 SQL 工作流程執行個體存放區應用於連接持續性資料庫之具名連接字串的名稱。這個參數是選擇性參數。如果您不希望 SQL 工作流程執行個體存放區使用預設的具名連接字串 **DefaultSqlWorkflowInstanceStoreConnectionString**，請指定連接字串名稱屬性或連接字串屬性的值。
