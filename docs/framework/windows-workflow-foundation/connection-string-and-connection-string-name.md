# <a name="connection-string-and-connection-string-name"></a>連接字串與連接字串名稱
**連接字串**屬性會指定 SQL 工作流程執行個體存放區應該用來連接至持續性資料庫的連接字串。 這個參數是選擇性參數。 **連接字串名稱**屬性會指定 SQL 工作流程執行個體存放區應該用來連接至持續性資料庫的具名的連接字串名稱。 這個參數是選擇性參數。 如果您不想要使用預設的具名連接字串的 SQL 工作流程執行個體存放區，您應指定連接字串名稱屬性或連接字串屬性的值**DefaultSqlWorkflowInstanceStoreConnectionString**.
