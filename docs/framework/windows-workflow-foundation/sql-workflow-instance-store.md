 <span data-ttu-id="42a58-114">Windows Server App Fabric 提供其本身的執行個體存放區和工具，可簡化執行個體存放區的設定和使用。</span><span class="sxs-lookup"><span data-stu-id="42a58-114">Windows Server App Fabric provides its own instance store and tooling to simplify the configuration and use of the instance store.</span></span> <span data-ttu-id="42a58-115">如需詳細資訊，請參閱 < 請參閱[Windows Server App Fabric 執行個體存放區](https://go.microsoft.com/fwlink/?LinkId=201201)。</span><span class="sxs-lookup"><span data-stu-id="42a58-115">For more information, see see [Windows Server App Fabric Instance Store](https://go.microsoft.com/fwlink/?LinkId=201201).</span></span> <span data-ttu-id="42a58-116">如需有關 App Fabric SQL Server 持續性資料庫，請參閱[App Fabric SQL Server 持續性資料庫](https://go.microsoft.com/fwlink/?LinkId=201202)</span><span class="sxs-lookup"><span data-stu-id="42a58-116">For more information about the App Fabric SQL Server Persistence Database see [App Fabric SQL Server Persistence Database](https://go.microsoft.com/fwlink/?LinkId=201202)</span></span>
