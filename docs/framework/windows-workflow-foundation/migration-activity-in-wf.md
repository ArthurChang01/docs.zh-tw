# 在 WF 中移轉活動
[!INCLUDE[netfx_current_long](../../../includes/netfx-current-long-md.md)] 提供 <xref:> System.Activities.Statements.Interop?qualifyHint=False&autoUpgrade=True 活動，用於執行衍生自 <xref:System.WorkflowModel.Activities.Activity> 為基礎的工作流程內 <xref:System.Activities.Activity>。 [!INCLUDE[crdefault](../../../includes/crdefault-md.md)]  [移轉指引](../../../docs/framework/windows-workflow-foundation//migration-guidance.md) 一節。  
  
> [!NOTE]
>   <xref:System.Activities.Statements.Interop> 活動不會顯示在工作流程設計工具的工具箱除非工作流程的專案具有其 **目標 Framework** 設定設為 **.Net Framework 4** 或更高版本。
