# <a name="using-custom-activity-designers-and-templates"></a>使用自訂活動設計工具及範本
本節包含的主題描述如何建立自訂活動設計工具和自訂活動範本。  
  
## <a name="in-this-section"></a>本節內容  
 [如何：建立自訂活動設計工具](how-to-create-a-custom-activity-designer.md)  
 描述如何在工作流程提供的設計工具不適合進行設計工作時，建立自訂的活動設計工具。  
  
 [如何：建立自訂活動範本](how-to-create-a-custom-activity-template.md)  
 描述如何使用自訂活動範本預先設定活動，如此使用者就不需要個別建立每個活動，以及手動設定其所有屬性和其他設定。  
  
 [使用 ModelItem 編輯內容](using-the-modelitem-editing-context.md)  
 說明如何利用 ModelItem 編輯內容的功能，讓設計工具與主機互動。  
  
 [將自訂活動屬性繫結至設計工具控制項](binding-a-custom-activity-property-to-a-designer-control.md)  
 說明如何將 listview 控制項繫結至設計工具中的活動屬性。
