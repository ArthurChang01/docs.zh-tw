# 使用自訂活動設計工具及範本
本節包含的主題描述如何建立自訂活動設計工具和自訂活動範本。  
  
## 在本節中  
 [HOW TO：建立自訂活動設計工具](../../../docs/framework/windows-workflow-foundation//how-to-create-a-custom-activity-designer.md)  
 描述如何在工作流程提供的設計工具不適合進行設計工作時，建立自訂的活動設計工具。  
  
 [HOW TO：建立自訂活動範本](../../../docs/framework/windows-workflow-foundation//how-to-create-a-custom-activity-template.md)  
 描述如何使用自訂活動範本預先設定活動，如此使用者就不需要個別建立每個活動，以及手動設定其所有屬性和其他設定。  
  
 [使用 ModelItem 編輯內容](../../../docs/framework/windows-workflow-foundation//using-the-modelitem-editing-context.md)  
 描述如何使用編輯 ModelItem 編輯內容的功能，使設計工具與主機互動。  
  
 [將自訂活動屬性繫結至設計工具控制項](../../../docs/framework/windows-workflow-foundation//binding-a-custom-activity-property-to-a-designer-control.md)  
 描述如何將 listview 控制項繫結至設計工具中的活動屬性。
