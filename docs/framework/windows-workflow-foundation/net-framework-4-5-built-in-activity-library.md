# <a name="net-framework-45-built-in-activity-library"></a>.NET Framework 4.5 內建活動程式庫

[!INCLUDE[netfx_current_long](../../../includes/netfx-current-long-md.md)] 包含具有擴充功能的新活動程式庫。 活動的類型包含下列項目：

- [控制流程](../../../docs/framework/windows-workflow-foundation/control-flow-activities-in-wf.md)

- [流程圖](../../../docs/framework/windows-workflow-foundation/flowchart-activities-in-wf.md)

- [狀態機器](../../../docs/framework/windows-workflow-foundation/state-machine-activities-in-wf.md)

- [傳訊活動](../../../docs/framework/wcf/feature-details/messaging-activities.md)

- [執行階段](../../../docs/framework/windows-workflow-foundation/runtime-activities-in-wf.md)

- [Primitives](../../../docs/framework/windows-workflow-foundation/primitives-activities-in-wf.md)

- [異動](../../../docs/framework/windows-workflow-foundation/transaction-activities-in-wf.md)

- [集合](../../../docs/framework/windows-workflow-foundation/collection-activities-in-wf.md)

- [錯誤處理](../../../docs/framework/windows-workflow-foundation/error-handling-activities-in-wf.md)

- [移轉](../../../docs/framework/windows-workflow-foundation/migration-activity-in-wf.md)

內建活動程式庫中的活動可以組成工作流程，或與自訂活動一起使用。 內建活動程式庫中的活動是密封的，它們不是透過繼承以建立新功能。
