# 擴充 Windows Workflow Foundation
下面章節描述如何在 [!INCLUDE[vs2010](../../../includes/vs2010-md.md)] 外部的重新裝載環境中，使用自訂活動和設計工具來擴充 [!INCLUDE[wf](../../../includes/wf-md.md)]。  
  
## 在本節中  
 [自訂工作流程設計經驗](../../../docs/framework/windows-workflow-foundation//customizing-the-workflow-design-experience.md)  
 指出用來設計自訂活動及重新裝載 [!INCLUDE[wfd1](../../../includes/wfd1-md.md)] 的案例如何在 [!INCLUDE[netfx40_short](../../../includes/netfx40-short-md.md)] 中大幅簡化。現在開發和部署工作都變得比較輕鬆而且更有彈性，因為新的活動設計工具程式設計模型是根據 [!INCLUDE[avalon1](../../../includes/avalon1-md.md)] 所建置。
