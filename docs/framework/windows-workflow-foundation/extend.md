# <a name="extending-windows-workflow-foundation"></a>擴充 Windows Workflow Foundation
下一節說明如何在 Visual Studio 2010 以外的重新裝載環境中，使用自訂活動和設計工具來擴充 Windows Workflow Foundation （WF）。

## <a name="in-this-section"></a>本章節內容
 [自訂工作流程設計經驗](customizing-the-workflow-design-experience.md)指出設計自訂活動和重新裝載 [!INCLUDE[wfd1](../../../includes/wfd1-md.md)] 的案例，已在 .NET Framework 4 中大幅簡化。 開發和部署現在更容易且更具彈性，因為新的活動設計工具程式設計模型是以 Windows Presentation Foundation （WPF）為基礎建立的。
