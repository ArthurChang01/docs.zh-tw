# <a name="2028---cacherootmetadatastop"></a>2028 - CacheRootMetadataStop

|||  
|-|-|  
|ID|2028|  
|關鍵字|WFRuntime|  
|層級|詳細資訊|  
|通道|Microsoft-Windows-Application Server-Applications/Debug|
