# <a name="rehosting-the-workflow-designer"></a>重新裝載工作流程設計工具
Windows 工作流程設計工具可以在 Visual Studio 2012 以外的環境中重新裝載，以建立、修改和監視工作流程。

 <xref:System.Activities.Presentation.WorkflowDesigner> 型別是畫布、屬性方格與其他項目的包裝函式，並公開基本程式設計模型以處理大部分的設計工具重新裝載實例。 在 Windows Presentation Foundation （WPF）應用程式內裝載 <xref:System.Activities.Presentation.WorkflowDesigner> 是工作流程設計工具常見的重新裝載案例。
