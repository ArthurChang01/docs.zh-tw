# <a name="2024---internalcachemetadatastart"></a>2024 - InternalCacheMetadataStart
## <a name="properties"></a>屬性  
  
|||  
|-|-|  
|ID|2024|  
|關鍵字|WFRuntime|  
|層級|詳細資訊|  
|通道|Microsoft-Windows-Application Server-Applications/Debug|
