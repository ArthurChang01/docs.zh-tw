# 2578 - TryCatchExceptionFromCatchOrFinally
## 屬性  
  
|||  
|-|-|  
|ID|2578|  
|關鍵字|WFActivities|  
|層級|警告|  
|通道|Microsoft\-Windows\-Application Server\-Applications\/Debug|  
  
## 描述  
 表示 Catch 或 Finally 活動已擲回例外狀況。  
  
## 訊息  
 與 TryCatch 活動 '%1' 關聯的 Catch 或 Finally 活動擲回例外狀況。
