# <a name="configuring-cryptography-classes"></a>設定密碼編譯類別
Windows 軟體開發套件 (SDK) 可讓電腦系統管理員，若要設定的預設密碼編譯演算法和.NET Framework 和適當地撰寫的應用程式使用的演算法實作。  比方說，企業有它自己的密碼編譯演算法的實作可以在 Windows SDK 中進行而不是實作的預設值隨附該實作。 雖然永遠都可以選擇使用加密的受控應用程式，明確繫結至特定的實作，但建議他們使用密碼編譯組態系統建立密碼編譯物件。
