# 組態區段結構描述
組態區段結構描述 \(Schema\) 包含在組態檔中放置自訂設定的項目。  
  
 [\<configuration\>](../../../../docs/framework/configure-apps/file-schema/configuration-element.md)  
  
 [\<configSections\>](http://msdn.microsoft.com/zh-tw/8a5cbc84-0257-4c2e-80a9-a064fe7c896b)  
  
 [\<clear\>](http://msdn.microsoft.com/zh-tw/b0ce3c91-89fa-4bb5-a995-30e21a61036b)  
  
 [\<remove\>](http://msdn.microsoft.com/zh-tw/6e2b2d45-113d-4d10-9f8e-0b15fc86bb98)  
  
 [\<section\>](http://msdn.microsoft.com/zh-tw/51c9d658-c275-4ab2-aa66-c5d5b5289570)  
  
 [\<sectionGroup\>](http://msdn.microsoft.com/zh-tw/45fbeb5c-26c7-4c68-b732-b3f1e62c53bf)  
  
|項目|描述|  
|--------|--------|  
|[configSections 的 clear 項目 \(一般設定結構描述\)](http://msdn.microsoft.com/zh-tw/b0ce3c91-89fa-4bb5-a995-30e21a61036b)|清除所有先前定義的區段和區段群組。|  
|[configSections 項目 \(一般設定結構描述\)](http://msdn.microsoft.com/zh-tw/8a5cbc84-0257-4c2e-80a9-a064fe7c896b)|包含組態區段和命名空間宣告。|  
|[configSections 的 remove 項目 \(一般設定結構描述\)](http://msdn.microsoft.com/zh-tw/6e2b2d45-113d-4d10-9f8e-0b15fc86bb98)|移除預先定義的區段或區段群組。|  
|[configSections 的 section 項目 \(一般設定結構描述\)](http://msdn.microsoft.com/zh-tw/51c9d658-c275-4ab2-aa66-c5d5b5289570)|包含組態區段宣告。|  
|[configSections 的 sectionGroup 項目 \(一般設定結構描述\)](http://msdn.microsoft.com/zh-tw/45fbeb5c-26c7-4c68-b732-b3f1e62c53bf)|定義組態區段的命名空間。|
