## <a name="example"></a><span data-ttu-id="1fc2f-246">範例</span><span class="sxs-lookup"><span data-stu-id="1fc2f-246">Example</span></span>  
 <span data-ttu-id="1fc2f-247">下列範例會使用`AppContextSwitchOverrides`項目來定義單一應用程式相容性參數， `Switch.System.Globalization.NoAsyncCurrentCulture`，可避免文化特性中的非同步方法呼叫的執行緒之間流動。</span><span class="sxs-lookup"><span data-stu-id="1fc2f-247">The following example uses the `AppContextSwitchOverrides` element to define a single application  compatibility switch, `Switch.System.Globalization.NoAsyncCurrentCulture`, that prevents culture from flowing across threads in asynchronous method calls.</span></span>

```xml  
   <runtime>  
      <AppContextSwitchOverrides value="Switch.System.Globalization.NoAsyncCurrentCulture=true" />  
   </runtime>  
</configuration>  
```  
  
 <span data-ttu-id="1fc2f-248">下列範例會使用`AppContextSwitchOverrides`項目來定義兩個應用程式相容性參數，`Switch.System.Globalization.NoAsyncCurrentCulture`和`Switch.System.IO.BlockLongPaths`。</span><span class="sxs-lookup"><span data-stu-id="1fc2f-248">The following example uses the `AppContextSwitchOverrides` element to define two application  compatibility switches, `Switch.System.Globalization.NoAsyncCurrentCulture` and `Switch.System.IO.BlockLongPaths`.</span></span> <span data-ttu-id="1fc2f-249">請注意分號分隔的兩個名稱/值組。</span><span class="sxs-lookup"><span data-stu-id="1fc2f-249">Note that a semicolon separates the two name/value pairs.</span></span>

```xml  
    </runtime>
```  
