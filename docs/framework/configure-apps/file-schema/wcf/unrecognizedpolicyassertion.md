# &lt;unrecognizedPolicyAssertion&gt;
表示指定原則判斷提示的繫結項目。  這個項目沒有屬性，並呈現為空白 switch。  
  
## 語法  
  
```  
  
<unrecognizedPolicyAssertion/>  
```  
  
## 類型  
 `Type`  
  
## 屬性和項目  
 下列章節說明屬性、子項目和父項目。  
  
### 屬性  
 無。  
  
### 子項目  
 無  
  
### 父項目  
  
|項目|描述|  
|--------|--------|  
|[\<繫結\>](../../../../../docs/framework/misc/binding.md)|定義自訂繫結的所有繫結功能。|
