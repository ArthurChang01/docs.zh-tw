# <a name="unrecognizedpolicyassertion"></a>\<unrecognizedPolicyAssertion>
表示指定原則判斷提示的繫結項目。 這個項目沒有屬性，並呈現為空白 switch。  
  
 \<system.serviceModel>  
\<bindings>  
\<customBinding>  
\<系結 >  
\<unrecognizedPolicyAssertion>  
  
## <a name="syntax"></a>語法  
  
```xml  
<unrecognizedPolicyAssertion />
```  
  
## <a name="type"></a>類型  
 `Type`

 下列各節描述屬性、子項目和父項目。

### <a name="child-elements"></a>子元素  
 無
