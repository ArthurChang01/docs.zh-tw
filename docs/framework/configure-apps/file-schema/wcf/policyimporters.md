# <a name="policyimporters"></a>\<policyImporters>
這個組態項目會指定所有原則匯入工具，這個匯入工具控制匯入有關繫結的自訂原則判斷提示。 每個子項目是 <`policyImporter`> 用來搜尋有關繫結功能，自訂原則判斷提示，以及附加可實作判斷提示所需要的功能的自訂繫結項目。
