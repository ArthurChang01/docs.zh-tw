 <span data-ttu-id="127fb-160">如需使用此設定元素的詳細範例，請參閱[服務的審核行為](../../../wcf/samples/service-auditing-behavior.md)。</span><span class="sxs-lookup"><span data-stu-id="127fb-160">For a detailed example of using this configuration element, see [Service Auditing Behavior](../../../wcf/samples/service-auditing-behavior.md).</span></span>
