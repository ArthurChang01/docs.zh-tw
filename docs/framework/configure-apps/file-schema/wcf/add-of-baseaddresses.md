# &lt;baseAddresses&gt; 的 &lt;add&gt;
表示組態項目，其中指定由服務主機使用的基底位址。  
  
## 語法  
  
```  
  
<add baseAddress="string" />  
```  
  
## 類型  
 `Type`  
  
## 屬性和項目  
 下列章節說明屬性、子項目和父項目。  
  
### 屬性  
  
|屬性|描述|  
|--------|--------|  
|`baseAddress`|字串，指定服務主機所使用的基底位址。|  
  
### 子項目  
 無。  
  
### 父項目  
  
|項目|描述|  
|--------|--------|  
|[\<baseAddresses\>](../../../../../docs/framework/configure-apps/file-schema/wcf/baseaddresses.md)|`baseAddress` 項目的集合。|
