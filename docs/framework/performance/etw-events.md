# .NET Framework 中的 ETW 事件
Windows 事件追蹤 \(ETW\) 是一套由 Windows 作業系統所提供的高效能、低額外負荷且可擴充的追蹤系統。  它補充了 .NET Framework 所提供的程式碼剖析和偵錯支援不足之處，而且可用來疑難排解各種案例。  如需 ETW 的一般資訊，請參閱 MSDN Library 中的 [改善 ETW 的偵錯和效能優化](http://go.microsoft.com/fwlink/?LinkID=161142) 。  
  
 在 .NET Framework 中，ETW 事件追蹤適用於 Common Language Runtime \(CLR\)、[工作平行程式庫](../../../docs/standard/parallel-programming/task-parallel-library-tpl.md)和[Parallel LINQ \(PLINQ\)](../../../docs/standard/parallel-programming/parallel-linq-plinq.md)。
